# Open Collective 官方声明 - OCF 解散

> 来源：[https://blog.opencollective.com/open-collective-official-statement-ocf-dissolution/](https://blog.opencollective.com/open-collective-official-statement-ocf-dissolution/)

亲爱的 Open Collective 社区，

我们怀着悲痛的心情得知 Open Collective Foundation 决定于 12 月 31 日解散。他们的公告全文，对我们来说也是个意外，可以在[这里](https://docs.google.com/document/d/1odj1zfwqZgIyGOeBhv5QZxYHn422VOMdo32i3T3CPps/edit?usp=sharing&ref=blog.opencollective.com)找到，并且他们还创建了[常见问题解答](https://docs.opencollective.foundation/?ref=blog.opencollective.com)。我们仍在努力应对这个决定的现实后果，它已经在 Open Collective 生态系统和社区中引起了连锁反应。

**实体澄清**

[Open Collective. Inc (OC)](https://opencollective.com/opencollective?ref=blog.opencollective.com) 是一家盈利实体。我们负责构建 Open Collective 平台，许多财务主办机构都在其上运作。

Open Collective 的创始人也创办并支持了几个非营利组织的早期发展：

+   [Open Collective Foundation](https://opencollective.com/foundation?ref=blog.opencollective.com) (OCF) 是一个总部位于美国的 501(c)(3) 财务主办机构，托管慈善项目，并已选择解散。

+   [Open Source Collective](https://opencollective.com/opensource?ref=blog.opencollective.com) (OSC) 是一个总部位于美国的 501(c)(6) 财务主办机构，只托管与开源软件相关的项目，并且继续良好运营。

+   [Open Collective Europe](https://opencollective.com/europe?ref=blog.opencollective.com) (OCE) 是一个总部位于欧洲的组织，能够在美国拥有 501(c)(3) 的等同性，主要托管与欧洲有根或有关的开源、团结、艺术和文化项目，并继续良好运营。

这些实体都有类似的名称和相似的起源，由 Open Collective 的创始人发起并在其生态系统中孵化，但它们是**独立的非营利组织，拥有自己的预算、账户、员工、董事会和使命。** 它们与 Open Collective 有单独的商业关系。

近年来，Open Collective 生态系统经历了快速增长。尽管上述四个组织在历史上是紧密合作伙伴，随着它们的成长，它们也越来越彼此独立，并在功能上完全分离，尽管它们在名称和品牌上有相似之处。因此，**Open Collective Foundation 完全独立做出了这个决定。**

**Open Collective 对这一情况的理解**

在过去的一年中，OCF让我们意识到增加了技术需求，我们通过新功能和其他形式的支持做出了响应。尽管我们共同努力，但似乎OCF未能找到可行和可持续的前进路径。关于OCF决定解散的具体细节我们并不清楚。我们必须相信OCF有充分的理由这样做，但我们仍然深感遗憾。

Open Collective Foundation是使用Open Collective平台的最大非营利组织之一，作为一个组织，Open Collective将需要根据这些变化进行演变，但首先，我们将尽我们所能尽快为所有受影响的OCF集体纠正这一情况。

**从OCF过渡**

*请确保在每个步骤中与OCF沟通。详细信息请参阅[*他们的文档*](https://docs.opencollective.foundation/leaving-ocf?ref=blog.opencollective.com)*。*

我们正在尽快努力理解如何支持由OCF托管并为它们服务的众多集体。根据OCF的[公告](https://docs.google.com/document/d/1odj1zfwqZgIyGOeBhv5QZxYHn422VOMdo32i3T3CPps/edit?ref=blog.opencollective.com)，所有**OCF**集体都需要：

+   减少并关闭他们的集体

+   减少并转移他们的集体至另一财政支持者，或

+   将他们的集体和资金转移至另一慈善机构。

明确指出，**这仅适用于由Open Collective Foundation托管的集体。** 平台上大多数开源项目由另一组织Open Source Collective托管，其财务状况良好。

**如果您由OCF托管**，请注意以下日期：

• 最后一天接受资金/接受捐赠：2024年3月15日

• 集体能够有员工的最后一天：2024年6月30日

• 最后一天支出或转移资金：2024年9月30日

尽管Open Collective平台上没有OCF的1:1替代方案，但以下可能是OCF集体前进的潜在途径：

+   [Open Collective Europe](https://opencollective.com/europe?ref=blog.opencollective.com)是一家欧洲慈善机构，类似于美国的501(c)(3)。他们主要支持与欧洲根源或联系紧密的开源、团结、艺术和文化项目。（请参考OCE的声明[这里](https://opencollective.com/europe/updates/regarding-the-announcement-to-dissolve-open-collective-foundation-us?ref=blog.opencollective.com)，并阅读关于他们的提供[这里](https://opencollective.com/europe/updates/what-is-open-collective-europe?ref=blog.opencollective.com)。）

+   [开源集体](https://opencollective.com/opensource?ref=blog.opencollective.com) - 如果你的集体在开源社区工作，可能有资格由他们托管。但需要注意，作为501(c)(6)组织，可能会有来自特定捐赠者的资金机会限制，这些捐赠者仅限于资助501(c)(3)实体。此外，从501(c)(3)财政赞助主机转移资金也可能会受到限制。（请参考OSC的声明[此处](https://opencollective.slack.com/archives/C0RD4S62E/p1709098296992259?ref=blog.opencollective.com)以及关于过渡的详细信息[这里](https://docs.oscollective.org/readme/moving-from-ocf-or-another-501c3?ref=blog.opencollective.com)。）

+   Open Collective平台上的另一个财政赞助主机。我们目前正在整理一份OCF集体的选项列表，将在本博客上分享。这可能让你在Open Collective上保留集体档案，并更轻松地保留定期捐款。（我们正在努力开发工具和功能来帮助过渡。）

+   这份由New Economy Coalition创建的[电子表格](https://airtable.com/appQVOmXysA863DDc/shrdjThde0Xteoabu/tblwopS490Qylth3g/viwi6ALAi325gE0Gb?ref=blog.opencollective.com)包含了多个与使命相关的财政赞助人以及有关财政赞助的资源。

+   National Network of Fiscal Sponsors维护着他们成员的[数据库](https://www.fiscalsponsors.org/member-directory?ref=blog.opencollective.com)。

+   一个位于美国的财政赞助主机[目录](https://fiscalsponsordirectory.org/?page_id=1330&ref=blog.opencollective.com)。

+   [Open Collective Foundation](https://opencollective.com/foundation?ref=blog.opencollective.com) 提供了支持 - “许多财政赞助人有一些特定要求，某些集体可能无法满足。如果你在寻找新的支持来源时遇到困难，可以通过发送电子邮件至[generalinquiries@opencollective.org](mailto:generalinquiries@opencollective.org)来寻求我们的帮助，我们将与你一起探讨可行的选项。”

+   [成为独立集体](https://docs.opencollective.com/help/independent-collectives/about-independent-collectives?ref=blog.opencollective.com) - 这需要你获得自己的法律地位和银行账户，并遵守会计和法律义务。这样可以让你在Open Collective上保留集体档案，并更轻松地保留定期捐款。（我们正在努力开发工具和功能来帮助过渡。）

**无论选择哪个选项，都需要与Open Collective Foundation协调。** 我们理解，对OCF的各个集体来说，这种过渡并不简单或直接，但如果你受到影响，我们将尽力提供帮助。

我们的支持团队随时准备为你提供支持，并回答任何问题 - [support@opencollective.com](support@opencollective.com)。

随着我们获取更多信息和情况的进展，我们将继续添加到此公告中。

衷心地，

开放集体团队
