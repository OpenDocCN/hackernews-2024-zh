# Ofsted检查员在IT系统崩溃时‘捏造’学校绩效的证据 | Ofsted | 卫报

> 来源：[https://www.theguardian.com/education/2024/feb/03/ofsted-inspectors-make-up-evidence-about-a-schools-performance-when-it-fails](https://www.theguardian.com/education/2024/feb/03/ofsted-inspectors-make-up-evidence-about-a-schools-performance-when-it-fails)

《观察家》的调查发现，由于他们用于记录检查的计算机系统有时会崩溃并擦除所有数据，Ofsted检查员被迫“捏造”证据。

自2018年起，[检查员在笔记本电脑](https://educationinspection.blog.gov.uk/2018/09/07/ofsted-is-going-digital/)或平板电脑上进行实时记录，他们在采访工作人员和观察课堂后使用了一种新的电子证据收集（EEG）数字平台。然而，我们的调查发现，这项技术从一开始就存在严重问题，有时会意外崩溃并丢失所有采访笔记，甚至是整整一天的证据，因此检查员不得不凭记忆重新补充这些笔记，而没有告知学校。

*观察者*在匿名条件下采访了几位现任或最近的检查员，他们表示这些问题多年来一直很“普遍”，Ofsted的高级领导人知道这一点，但“涉及到很多掩盖行为”。

在这篇报道最初发布后，Ofsted的一位发言人告诉*观察者*，首席检查员马丁·奥利弗爵士“正在启动快速审查，以确保EEG和对检查员的指导是可靠的”。发言人补充说，如果学校或检查员有任何担忧，“我们希望直接听到它们，以便我们能做出适当的回应”。

国家教育联盟的总书记丹尼尔·凯贝德表示：“这与[邮政局丑闻](https://www.theguardian.com/uk-news/post-office-horizon-scandal)有相似之处。检查员编造丢失的证据，或者记住他们能记住的，这是令人憎恶的行为。学校不允许这样做。” 他认为，如果这种做法普遍存在，“过去五年进行的每一次检查都应该被忽略”。

“这与邮政局丑闻有相似之处”：国家教育联盟总书记丹尼尔·凯贝德。摄影：PA Images/Alamy

Ofsted的发言人表示：“我们非常重视我们用于做出检查评级的过程。” 他们强调称，所有的评级都是由检查团队提供充分证据支持的，并且还由Ofsted的高级工作人员进行质量保证。

发言人将EEG工具描述为“仅用于记录书面证据的简单平台”，并表示：“所有检查员都已经收到明确的指导，如果无法访问系统该怎么做。”

一位现任检查员告诉*观察者*：“在与学校的会议中，你可能突然丢失所有的证据。建议是你不能表现出任何不满，保持不动声色。”

他回忆起坐在旅馆里尝试在系统关闭后输入一天的证据，结果一切都丢失了：“你已经筋疲力尽，因为检查的一天又长又紧张，而且你还要阅读明天的文件。”

总部的同事们曾经想要对他刚给予一所学校的“杰出”评分进行“严格的审查”。他说：“他们要求提供证据，但却没有。”“涉及到很多掩盖行为。”

第二位前检查员描述了他坐在那里看着一场漫长面试的证据“逐渐从屏幕上消失，就像班克斯的作品自毁一样”。他说：“在这种情况下很难避免透露出任何身体语言线索。” 他补充道：“当时的指示是随便编造。”

他补充道：“我确切知道 Ofsted 的高级领导人员知道这些问题。”

Ofsted 去年对 EEG 进行了调查，检查员表示确认了严重问题，但没有人意识到有任何重大变化作出响应。

教育提供者在很大程度上完全没有意识到这些问题。然而，《观察者报》看到了 Ofsted 对学校关于证据丢失的投诉的书面回复。回复中写道：“你们声称在最后的会议上，主检员宣称：‘一切都没了’，指的是她电脑上的证据。”

回应还指出，草稿报告中包括一所学校的座右铭，“实际上并不是你们的校训”，并且学校对证据基础“缺乏信心”。

Ofsted 否认了这一点，称主检员记录了最后会议的记录消失了，但所有其他证据都已记录并“经过增强的质量保证”。《观察者报》了解到，学校从未看到这些证据。

一位前 Ofsted 主检员表示：“我在一次与协调员的会议中突然失去了一切，这是一次对某一特定学科教学的深度审查的开端。主检员在晚上回到酒店后也失去了一切，而团队中的新检查员也是如此。”

三名检查员报告称，他们或同事开始在 Word 中做笔记，因为他们不信任系统，但如果他们不使用 EEG，Ofsted 就会威胁采取纪律行动。

朱莉·普莱斯·格林肖（Julie Price Grimshaw），一位曾在 Ofsted 担任检查员现在为学校提供建议的人，说在她服务的铅笔和纸时代，“很多检查员”告诉她系统崩溃时他们丢失了证据。

她说：“显然，大量内容，甚至可能是整个证据基础，都消失在虚空中，再也没有被看到。”

Ofsted 表示，如果 EEG 系统不工作，检查员被建议用“合理的替代方式”记录他们的证据，并“在最早的机会将其转移到系统中”。Ofsted 的发言人补充道：“如果在质量保证过程中系统上没有足够的证据支持判断，检查员将被指示返回学校收集和记录进一步的证据。”
