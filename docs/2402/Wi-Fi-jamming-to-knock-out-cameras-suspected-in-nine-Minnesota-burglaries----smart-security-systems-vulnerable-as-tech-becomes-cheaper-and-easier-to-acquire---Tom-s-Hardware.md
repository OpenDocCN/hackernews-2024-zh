# 使用 Wi-Fi 干扰器摧毁摄像头，怀疑在明尼苏达州发生的九起盗窃案件中存在风险 —— 随着技术变得更便宜更易获得，智能安全系统也更易受攻击 | Tom's Hardware

> 来源：[https://www.tomshardware.com/networking/wi-fi-jamming-to-knock-out-cameras-suspected-in-nine-minnesota-burglaries-smart-security-systems-vulnerable-as-tech-becomes-cheaper-and-easier-to-acquire](https://www.tomshardware.com/networking/wi-fi-jamming-to-knock-out-cameras-suspected-in-nine-minnesota-burglaries-smart-security-systems-vulnerable-as-tech-becomes-cheaper-and-easier-to-acquire)

在明尼苏达州埃迪纳市，一名连环入室盗贼被怀疑使用[Wi-Fi 干扰器](https://www.kare11.com/article/news/local/edina-burglars-could-be-using-wifi-jammers/89-838f08f6-8e13-4577-8a88-628d757207a2)来摧毁[连接的安全摄像头](https://www.tomshardware.com/how-to/raspberry-pi-security-camera)，然后才进行偷窃并带走受害者珍贵的财物。明尼苏达州通常不被视为技术热点，所以读者不应对在美国出现使用 Wi-Fi 干扰器协助盗窃的报道感到惊讶，这种情况已有几年历史了。PSA：即使是[罪犯也在使用技术](https://www.tomshardware.com/news/raspberry-pi-used-to-rob-atm)，而且现在越来越多，所以房主应考虑采取应对措施。

埃迪纳警方怀疑过去六个月发生的九起入室盗窃案件中，使用了 Wi-Fi 干扰器以确保调查人员无法获取到涉案视频证据。盗贼或盗窃团伙的作案模式被认为大致如下：

+   在富裕地区找到目标房屋

+   盗贼会精心观察房屋

+   盗贼避免直接对抗，通常会等待房屋空无一人时才行动

+   抓住房屋无人的机会，盗贼会部署 Wi-Fi 干扰器

+   “保险柜、珠宝和其他高端设计物品”通常会被带走

一位由 KARE11 采访的[安全](https://www.tomshardware.com/tag/security)专家解释说，这些干扰器通常只是混淆无线设备，而不是阻断信号。据新闻网站报道，它们通过过载无线流量“使真实流量无法通过”。

(Image credit: Ring)

## 干扰无线安全设备是一个逐渐增长的趋势

检索关于使用 Wi-Fi 干扰器进行盗窃的新闻报道，会发现大量早期的证据支持这一做法。关于一家 Ring 社区帖子提到其无线门铃在经历了 Wi-Fi 退认证攻击后失去了一个门廊小偷的案例，该帖子于 2020 年 1 月发布。还很容易找到关于盗贼在 2021 年、2022 年和 2023 年使用 Wi-Fi 干扰技术的报道，而这类报道随着时间的推移变得越来越频繁。

我们提到Ring是推广视频门铃以解决多个家庭安全问题的公司之一，但其他来自Blink（[亚马逊](https://www.tomshardware.com/tag/amazon)）和Nest（[谷歌](https://www.tomshardware.com/tag/google)）的无线[智能家居](https://www.tomshardware.com/tag/smart-home)安全产品也会受到无线信号干扰的影响。

担心的是，2024年对潜在的小偷来说，Wi-Fi干扰几乎是一种微不足道的活动。KARE11指出，他们可以非常容易和便宜地在线购买到干扰器，价格从$40到$1,000不等。在美国使用干扰器是非法的，但它们在线购买非常简单。

（图片来源：eBay无线干扰器列表）

在我们离开之前，对于那些现在对他们家庭安全系统中的无线组件的效力感到担忧的人，有几点建议。首先，物理连接一些允许有线连接和本地存储录像的设备。其次，利用[智能家居技术](https://www.tomshardware.com/news/samsung-ai-npus-bixby-appliances-2024)，使其看起来有人在家。你的设备还可能具有在信号/连接中断时发送警报的功能，调整这些设置可能是值得的。
