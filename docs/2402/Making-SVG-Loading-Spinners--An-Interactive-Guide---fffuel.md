# 制作 SVG 加载旋转器：交互式指南

**SVG 旋转器**：你随处可见它们，给我们提供了视觉提示，当元素正在加载或数据正在获取时。它们实用、可伸缩，并且可以增强用户体验。本指南将指导你如何仅使用一些简单的 SVG 属性，如 *stroke-dasharray* 和 *stroke-dashoffset* 来实现它们。

✨🔗 深入了解不同的 SVG 属性，查看[SVG 参考](/sssvg/)。
