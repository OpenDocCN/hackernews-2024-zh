# 有人的道义责任是向世界泄露《Coyote vs Acme》——事后

> 来源：[https://aftermath.site/coyote-vs-acme-fuck-david-zaslav](https://aftermath.site/coyote-vs-acme-fuck-david-zaslav)

上周，[The Wrap的一篇优秀报道揭示](https://www.thewrap.com/coyote-vs-acme-update-offers-warner-bros/)，尽管最初收回了决定，华纳兄弟确实打算把*《Coyote vs Acme》*——一部资助、拍摄并完成的电影——扔进垃圾桶。不是因为它糟糕，而是为了在税收上节省一些钱。

这不是最近第一次发生这种事情——从迪士尼的电视节目到*蝙蝠女侠*电影都遭遇了同样的命运——但这一次刺痛了一些人。迪士尼和HBO移除观众已经有机会看过的电视节目是一回事，而*蝙蝠女侠*（据报道也非常糟糕）是另一回事。

但是《*Coyote vs Acme*》还没有被*任何人*看过。这部电影——由戴夫·格林执导，*五月的* 萨米·伯奇编剧，由（包括）詹姆斯·冈和定义华纳兄弟整个存在的角色主演——已经付款，完成，准备好被观看，现在它有很大的机会永远被删除，只是为了节省一些钱。

我们发现自己处于一个极为反乌托邦的情境中，艺术只是为了被喂入粉碎机来服务股东，而这一点已经在其他地方有了广泛的报道，所以我不打算在这里深入讨论决策的政治和经济学问题。

如果你在华纳兄弟工作，或者与华纳兄弟有任何关联，并且有任何接触这部电影的机会，那么在为时已晚之前，向世界泄露它是你的道义责任。如果华纳兄弟不愿意发布它，并据报道阻止其他公司发布它，那又有什么关系呢？

（**请注意**：我不是律师或执法人员，这不构成实际建议）。

正如我们*先前所述*，[没有所有权就没有盗版](https://aftermath.site/there-is-no-piracy-without-ownership)。如果我们无法通过合法手段购买或租赁这个东西，而且很可能永远也无法这样做，那么泄漏这部电影并将其放在种子网站上也不应被视为偷窃。据所有报道，华纳首席执行官大卫·扎斯拉夫很高兴将这部电影丢进垃圾箱，而没有售出一张电影票或数字租赁，因此无论我们是否免费观看，都不会在财务上有任何影响。

泄漏这部电影*会*做的事情是拯救数百人投入了几个月甚至*数年*心血的艺术品。这不应该是一种罪行。这是公共服务。
