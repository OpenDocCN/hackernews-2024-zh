# Malcolm Gaskill · 打在膝盖：意大利，1943

> 来源：[https://www.lrb.co.uk/the-paper/v46/n03/malcolm-gaskill/strike-at-the-knee](https://www.lrb.co.uk/the-paper/v46/n03/malcolm-gaskill/strike-at-the-knee)

1943年圣诞前夜，奥尔托纳。几天前还是一个繁荣的亚得里亚海港口，现在却已成为废墟。原本居住着一万人口的地方已经空无一人。取而代之的是两个精锐部队——德国伞兵团守卫着城镇的残骸，以及第一加拿大步兵师，他们接到了占领该城的命令。这里不会有节日的休战。整天盟军的炮弹炸毁了一个又一个房屋，谢尔曼坦克把敌人驱赶到更深的瓦砾中。许多人死去，尸体散落在泥泞的街道上。在一个晚间的间歇中，西格弗里德·贝尔和另一名无线电操作员在他们躲避的房间里拉起扶手椅，加热他们在地下室里找到的酒和糖。晚些时候，他们每人收到了一个包裹，里面有一个小圆面包、巧克力、杏仁和橙子，分享着家乡的记忆，尽力庆祝他们能够过的最好的圣诞节。奥尔托纳很快就会被称为意大利的斯大林格勒，但它也有点像帕赛恩代尔。

前一场昂贵的消耗性战斗正是那些计划盟军入侵意大利的人所想要避免的。速度和果断将是关键：在大陆建立滩头、发挥优势、联结兵力并开辟通向罗马的道路。盟军部队不能陷入泥潭，陷入僵局。然而，这涉及到一些神奇的想法。甚至支撑战略的假设也是矛盾的。入侵的主要目的是为了转移德军的兵力和物资，以帮助斯大林，同时也是为了从法国北部帮助计划中的入侵。但即使希特勒通过布伦纳峡谷向南增兵，盟军徒劳地希望他会在罗马北部的防御线停下，他们明白他们没有足够的力量与集结在一起的德国军队正面交锋。

随着盟军学会在自己的能力范围内作战，降雨如期而至，搅动起*strade bianche*，使前进速度减慢。习惯于在沙漠中奔驰的英国第八军团士兵，如今每天仅能前进几百码。他们大部分时间都在防空洞里度过，因为供应线路被湿天气中断和桥梁被炸毁，导致他们又饥又寒。美国战地记者厄尼·派尔说：“没人有权因为在通往罗马的道路上的进展缓慢而不耐烦，没有见过那些泥浆、黑暗的天空、那些令人生畏的山脊和幽灵般的云彩的人。”一个美国士兵，他的意大利父亲曾对永恒之城赞叹不已，现在失望了：“我们除了去另一个到处爬满虱子的该死的泥潭外，一无所获。”每一个山村的每一寸土地都必须为之战斗。士兵们挖掘阵地，在前方的地点与敌人交火，并等待他们采取行动。

盟军将领们的冲动性逐渐被这一现实所磨砺。他们的军队很快面对了另一道防线——古斯塔夫线——横贯意大利，从西部的加里利亚诺河到东部的奥尔托纳，穿过阿彭宁山脉，横跨通往罗马的北部道路。1944年，他们还要突破更多的防线，经历更多的战斗和流血。英国步兵的伤亡比例比一战时更高。第三冷流守卫连队，一个有845人的营，1943年夏秋季失去了一百人；到12月中旬，只剩下598人——实力减少了30%。美国师也忍受着类似的屠杀。到年底，美国第五军和英联邦部队分别遭受了大约三万人的伤亡。在“血腥十二月”的奥尔托纳，近1400名加拿大人丧生，占该国意大利战役的四分之一损失。西格弗里德·贝尔幸存了下来，但他的八百多名德国同志没有。

1942年10月在埃尔阿拉曼胜利和九个月后的胡斯基行动成功大大提高了联军的期望。但西部沙漠大部分是平坦开阔的地形，在那里的战斗基本上符合将军们在地图上设计的计划。在胡斯基行动中，盟军对抗了四个德军师的两支军队，而一个师只相当于一个军的四分之一大小。在意大利，不仅国土多山，而且在入侵前夕，这两支盟军的一部分仍停留在北非和西西里。此外，面对的敌人比以往更为强大：18个师的德军。此外，意大利有4400万人口，他们的家园和生活都处于战火之中。然而，盟军被一种有利的偏见鼓舞着，认为意大利战争是意大利的错，因此，尽管令人遗憾，因与德国盟友的战斗造成的附带损害是可以接受的。“这些意大利人简直就是一群败类，”蒙哥马利对他的士兵讲道，“他们只是躺在他们的葡萄园和柠檬树之间，生儿育女。他们太多了。这就是问题所在。他们太多了。”这些话让士兵们欢呼起来。

意大利入侵决策是在1943年一月的卡萨布兰卡会议上做出的，罗斯福同意了丘吉尔进一步在地中海战争中取得胜利的愿望。美国更倾向于尽快入侵诺曼底，同意了有限的意大利战役，只要诺曼底行动保持优先。这就是为什么意大利的军队资源匮乏 - 不仅是人力，还有用于水陆两栖突击的船只和登陆艇。此外，回顾美国内战期间的条件，罗斯福坚持“无条件投降”，这让丘吉尔感到不安，但对斯大林来说很合适，因为这排除了盟军将轴心国作为反苏联盟友的可能性（在迅速接近的冷战中），即使这也阻止了斯大林达成自己的和平协议。美国的心态是重量级选手在几轮比赛后展现出来的：跨过英吉利海峡，进入鲁尔和萨尔兰，然后向柏林前进。英国人在敦刻尔克和托布鲁克战役中受到伤害，并且被弗兰德斯的幽灵所困扰，更为谨慎：一个轻量级的挑战者在搜索弱点。丘吉尔称意大利为“鳄鱼的软肚子”，并在攻克西西里后大胆地提出：“我们为什么要像从脚踝开始的收割虫一样爬上去呢？我们宁愿直接打击膝盖。”

在意大利的脚趾地区（Operation Baytown）于9月3日开始登陆，意图是将德军力量从萨莱尔诺（Operation Avalanche）主要入侵地点，即提尔利亚海岸270英里以北的地方吸引开来。即使不是膝盖，至少也是在脚背和小腿之间，距那不勒斯仅四十英里，那是丘吉尔心中所向往的战利品。从一开始，盟军部队就陷入鳄鱼猛咬的夹击之中。9月4日，特别突袭中队（原1特种空勤团）在巴尼亚拉的一次袭击展示了在意大利的情况下，一个简单计划如何迅速崩溃。他们登陆错误的地方，然后在攀登悬崖公路时在火力下暴露无遗。这是在意大利大陆的第一次接触，尽管特别突袭中队最终占领了巴尼亚拉，但战役的一个定义性特征已经显现出来：地形有利于高地上的防御者，因此军事胜利所需的传统3:1的兵力比需要更高。

六天后，证实了在萨莱尔诺东南的巴蒂帕利亚，Avalanche行动已经展开。试图与第2苏格兰近卫队会合以包围德军阵地，第3冷流卫队被协调的迫击炮、机枪和狙击手火力困住长达48小时。苏格兰近卫队打破僵局的尝试是一场灾难：其中一个连队被击溃，另外两个连队的大部分军官都被击毙。两周后，冷流卫队发现自己在一个树木丛生的山丘顶附近遭到火力横扫。麻疹使人头昏脑胀的克里斯托弗·布尔蒂尔中尉目睹了从熊熊燃烧的火中跳出的人们；先前看不见的壕沟中扔出的棒榴弹；以及令人作呕的肉搏战场面。当布尔蒂尔和他的战友撤退时，天开始下起了大雨。他们挖了狐狸洞，陷入相对安全的泥潭中。

意大利战役的艰苦与怪诞常常足以使人铭记，但它常常被忽视，被轻视为从罗默尔战败到诺曼底登陆之间的填充物。甚至在战争期间，美国人起初也将其视为次要战场，而在英国，流传着一种神话，认为在意大利作战的士兵轻松自在：根据一首流行歌曲，他们是“D-Day的逃兵”。科尼利厄斯·瑞安的书《最长的一天》（1959年）及其启发的电影，以及之后的《拯救大兵瑞恩》和迷你系列《兄弟连》，将诺曼底定格为一个主导叙述的序幕，像一场追求般注定要以柏林为终点的故事。好像即使我们愿意也无法同时考虑意大利。詹姆斯·荷兰抵制了这种趋势，他之前的作品——关于西西里岛和1944-45年大陆战役等等——提高了人们对意大利战事发生及其原因的认识。这从未是一个未被讲述的故事，但绝对是一个被低估的故事，尤其是1943年的情况。正如荷兰所观察的，我们记得卡西诺和安齐奥，但不记得奥尔托纳，一个悲剧性的加拿大记忆之地，堪与1917年的维米峡岭相提并论。

荷兰细致地将战争日记、作战报告、信件、日记和回忆录中的故事串联起来，从烟雾弥漫的政客的冷静预测到战斗的恐怖幻境。如果一枚炮弹被发射，荷兰会尽力向我们展示它落在哪里——以及落在谁身上。而我们越接近战争的前线，文化和政治的差异就越能溶解成普世的人类情感。德国人也学会了在狭窄的山谷中部署坦克的难度；在攀登山丘时所造成的致命暴露；被掩体半埋时的恐惧，在没有足够空中掩护或防空火力支援的情况下遭到战斗机轰炸；以及古老的军事格言——计划不及接触时的残酷现实。

《野蛮风暴》是人道主义且毫不畏缩的，如同一场悲剧，情节由人物推动。哈罗德·亚历山大将军，荷兰旨在恢复其声誉，被战地记者艾伦·摩尔黑德赞赏地描述为：“他似乎具备了在自己处于火线之下、四周充斥着最令人担忧和混乱信息时，仍能清晰而全面地看待事物的罕见才能。”马克·克拉克，另一位被后人嘲笑的将军，也得到了改观。罗默尔在北意大利指挥部队，才华横溢却目光短浅；阿尔伯特·凯瑟林，他在南方的对应人物，是更好的战略家——但最终两人都没有处在应有的位置以防止德军撤退至阿尔卑斯山区。

军事历史长期以来一直是学术界的局外人，被大多数大学课程排斥，我想是因为其在从业者和读者以及主题上过于男性化、苍白、陈旧。它可能带有民族主义气息，男性为帝国主义事业牺牲的自豪感，并且通常有点*纳粹大结构*和“他的罂粟花在哪？”。左翼学者也可能因传统军事历史对叙事和迷人的单一视角的偏爱而感到厌弃。在其纯粹的操作和战略化外观中，军事历史通常枯燥乏味，积累了大量铁路迷般的细节。所有这些可能解释了为什么英国的课程，包括许多硕士课程，将其与国际关系结合起来，这不仅增加了知识深度，也许还增加了其可尊重性。

书店中的军事历史部分仍然相当丰富。许多书名构成了一种战争色情，为某些读者提供专业和强烈的满足感：纳粹党卫军的军团之类的东西。但也有许多‘新军事历史’，借鉴于20世纪的‘新社会历史’（注入了社会学）和‘新文化历史’（人类学）。从步兵到高级军官，服役人员不仅仅是打仗，而是体验战争，而且也有女性，有时候处于中心舞台，尽管更多时候是在背景中。我在学校记得的小杰作有马丁·米德尔布鲁克的*索姆河战役首日*和约翰·基根的*战斗的面孔*；而在本世纪，安东尼·毕沃、马克斯·黑斯廷斯和霍兰德则引领了英国队伍。战争历史现在在学术界显然更受欢迎：伦敦国王学院拥有一个非常有声誉的战争研究部门。这种上下文化、细腻和富有同情心的军事历史被男女同样地实践，基本上对旧的辉煌免疫，而是致力于揭露战争的多种哀愁。

虽然霍兰德对第二次世界大战的硬件设备充满热情，他与喜剧演员艾尔·默里（*我们有办法让你说话*）主持的播客也因对斯派特火和谢尔曼坦克的热爱而蓬勃发展。然而，在他的写作中，他紧紧把握住并展示了军事技术的重要性。生命取决于制空权、坦克装甲的厚度、美国突击队员在敌军阵地上方悄然指挥海军炮火的技能。 "弗里茨X" 是一种无线控制炸弹，它在飞行到位置后再俯冲击中舰船，这是一个显著的德国发明，不仅因其技术创新而引人瞩目，而且作为一个无情的生死裁判。 霍兰德简洁明了地剥离了他的散文，以便战斗中的疯狂时刻迎面而来：“混乱，感官被狠狠地打击。噪音。颤抖的地面。人们倒下。在这种情况下很难继续前进。” 机枪像电锯一样开火，信号弹噼啪作响，火箭呼啸而过，炮弹尖啸并吸入空气。 炮弹片——在此霍兰德引用厄尼·派尔的话说——“叮当作响地落在我们周围的岩石上，发出一种清脆的金属声音”。 我们被提示想象火药味、橡胶、柴油、烧焦的肉味。

机械化战争似乎仍然如此不自然，以至于溢出到离奇之境，难以用传统手段描述。超现实主义充斥其中，正如保罗·富塞尔在《大战与现代记忆》中所捕捉到的那样——霍兰德注意到1940年代也有类似的荒诞闪光。一个战争震惊的士兵对他的德国俘虏们喊道：“天哪，你们都长着英国人的脸！” 艾伦·穆尔黑德在土耳其渡轮上与阿拉伯人挤在一起的“疯狂之旅”，完全出自伊夫林·沃夫的《荣誉之剑》三部曲。但大多数时候，超现实主义是可怕的：那些“几分钟前还是这些碎片化的人类遗骸曾经是活生生的、呼吸的年轻人”的不可理解；突然无头的司机；当驮驴踩到地雷时，携带他的伤员的双腿被砍断；坐在篱笆上微笑的死去的德国士兵；月光下尸体的面孔“看起来是绿色的，非人类的”。此外，还有暴力与同情并存的并置：为一个垂死的德国人送去朗姆酒，为英国囚犯送去一口格拉帕酒，男人们安慰垂死的敌人——这些令人惊讶的讽刺性临时圣像，如《西线无战事》中著名的弹坑场景。美与恐惧的赞赏也同时出现：明信片式的别墅营地，海滩远足时男人们“像鱼一样嬉戏，像兴高采烈的孩子一样欢乐”；闪烁的紫色山峰如梦般消逝；日出时的山谷宁静。

尽管被战争的奇遇困扰，人们也学会了如何生存——主要是如何迅速和准确地行动——以及他们在极端情况下的能力。皇家爱尔兰轻步兵团的劳里·弗兰克林-维尔上尉在特尔莫利的首次交战中感到宽慰，他并没有恐慌，并且能够清晰地思考如何带领他的士兵。同时，勇气是资本，战斗中的人们总是在消耗它；当他们精疲力尽时，他们垮掉了。（这是查尔斯·莫兰的见解，他是第一次世界大战的医疗官，在本书中出现为丘吉尔的医生，1943年12月，首相遭受心脏病发作时也有所体会。）在弗兰克林-维尔的一个士兵在莫罗河的山脊上被杀后，他最好的朋友“有点发疯，跳了起来，开始乱喊乱射”，直到他的一个同志将他击昏并把他送往救护站。在奥尔托纳的一所房子中被围困的一名德国军官突然放下了机枪，向无线电操作员西格弗里德·贝尔要了一支香烟，然后漫步到广场上被击倒。

另一个经常遇到的军事历史和战争史的区别在于，后者对内部和政治事务的关注，以及敌对行动对平民生活的影响。*The Savage Storm* 并非仅仅发生在意大利：就像它的前作 *Italy’s Sorrow*（2008）一样，它讲述的是意大利，即使它侧重于盟军和德军，而非游击队员和法西斯党之间的内战。意大利被迫退出战争，由于罗马的轰炸和西西里的征服，带来了许多新问题。如果要将意大利转变为反纳粹盟友，需要高风险的边缘政策和精确的时机。即使在意大利别无选择而必须退出战争时，其秘密和平谈判代表们对抵押国家即时未来的更多保证需求，超出了盟军能够诚实承诺的范围。对安全的渴望使意大利人易受欺骗。在1943年7月推翻墨索里尼之后，盟军向新任首相巴多里奥施压，同意接受他们的“短期条款”来达成停战，但没有提及无条件投降和其他严峻的政治条件——这些条件由拖延的“长期条款”施加。巴多里奥面临两难局面，无法再与盟军作战，但在他的国家充斥着准备为罗马南部领土而战的德国人时，他也很难召集足够的信任来加入他们。

平民承受了这些决策的大部分压力。空中和地面战争摧毁了一个贫穷的人口，严重考验着这场战争的道德性。进入城镇的联军士兵震惊地发现，卫生条件原始或不存在，街道和建筑物肮脏不堪，到处是垃圾和苍蝇。皮包骨的孩子们身上长满了疮痍，四处乞讨；而同样饥饿的老妇人则无助地坐在人行道上。来自东本恩的皇家炮兵中士杰克·沃德看到了“我见过的最糟糕的景象：一个怀抱婴儿的人看起来就像是一块皮肉和骨头，头上却是一个老人的脑袋。” 军官们和士兵们从假期、杂志和电影中所了解的风景如画的意大利，在他们所到之处无不留下伤痕：弹坑、弃置的装备、膨胀的死牛、破碎的橄榄树、焦黑的草垛、匆忙挖掘的浅浅坟墓。

许多平民直接卷入了战斗中，比如24岁的卡拉·卡波尼在环绕斗兽场的街道上躲避坦克炮弹，或者圣皮耶特罗的马塞拉家族，被误认为是*装甲步兵*而被美军击倒。在埃博利，一些被严重烧伤的人被放在马厩里，并由一名医学生照料。在特尔莫利，一整个家庭被炸弹从他们家门口冲出，只剩下一个男孩在四处奔跑，内脏暴露在外，直到一名SAS战士逮住他，并用手枪了结他的性命。撤退的德军布设了地雷并设置了诡计多端的陷阱，用三脚架构成恶魔的花园，或者将绊线绑在被抛弃的物体上，比如一支铅笔，可能会被一个好奇的美国士兵捡起，或者像经常发生的那样，一个孩子。

财产的毁坏是末日般的，其中大部分是由盟军造成的。1943年8月1日，对罗马的空袭造成了一千名平民死亡，两倍于此的人受伤。两周后，在米兰又有一千人死亡，其中包括圣母玛利亚德莱格拉齐教堂，3000座建筑物被夷为平地或受损。教堂食堂的一面墙奇迹般地留了下来，上面是达·芬奇的《最后的晚餐》。那年那不勒斯被轰炸了180次，给城市居民带来了难以置信的痛苦。在无数城镇和村庄中，当家庭目睹时，盟军的坦克会摧毁房屋，如果他们的火线被遮挡或怀疑敌人藏在后面。步兵短缺增加了盟军对炮兵的依赖，以在他们进入之前摧毁城镇。奥尔托纳的毁坏部分是因为德军炸毁街道，以便将前进的小队引入交叉火力，但也因为加拿大人打通了墙壁——这种被称为“钻洞”的做法——以便他们可以在房屋之间穿行，而不暴露自己于危险的街头战斗。加拿大炮手轰击圣托马斯大教堂的圆顶，以防止其用作观察哨，他们被命令摧毁大教堂，但幸运地是避免了这一点。当时担任情报官的作家诺曼·刘易斯责怪第五军指挥官克拉克将军破坏了巴蒂帕利亚，他认为那像是一个意大利的格尔尼卡。

路易斯被驻扎在那不勒斯，那里经历了断电、缺水、面包骚乱和斑疹伤寒以及天花的流行。这座城市自1943年10月起由盟军控制，但还有许多城市等待解放。经过一个寒冷的冬季后，英美军队在1944年大部分时间内向北推进，占领了罗马、安科纳、佛罗伦萨，并在随后的春季突破了哥特防线。到1945年4月底，博洛尼亚、热那亚、米兰、都灵和威尼斯都已经沦陷，德军投降了。解放的喜悦往往被创伤所掩盖。占领那不勒斯和其他困境城市的盟军开始对意大利人表现出同情。现在，他们的看法变得柔和了。意识到他们为了自由和民主所造成的破坏，一些人开始怀疑自己的正义性，并对普通人——被掠夺的佃农和遭轰炸的工厂工人、饥饿的流浪儿和被迫卖淫的母亲们——心存感激，将侵略者视为解放者、朋友，甚至替代儿子。许多意大利人似乎在政治上持中立态度，对法西斯主义冷漠或敌视，对其口号无动于衷，其中最虚伪的是：“战争使有勇气面对它的人获得尊贵的标志。”墨索里尼的赌博没有带来承诺的任何回报，甚至胜利者也不认为战争中有尊贵，除非是在意大利人民的坚韧和尊严中。
