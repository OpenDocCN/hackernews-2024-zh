# 《Planet 9可能藏身的最后一处地方 - Universe Today》

> 来源：[https://www.universetoday.com/165774/theres-one-last-place-planet-9-could-be-hiding/](https://www.universetoday.com/165774/theres-one-last-place-planet-9-could-be-hiding/)

最近提交给《天文学杂志》的研究继续寻找神秘的第九行星（也称为[行星X](https://science.nasa.gov/solar-system/planet-x/)），这是一个假设的行星，可能绕太阳系外围运行，远超矮行星冥王星的轨道。该研究的目标是缩小第九行星可能的位置，并有望帮助研究人员更好地理解我们太阳系的组成、形成和演化过程。那么，这项研究背后缩小潜在的第九行星位置的动机是什么？

[迈克·布朗博士](https://mikebrown.caltech.edu/)，加州理工学院的理查德和芭芭拉·罗森伯格天文学教授和研究的首席作者，在接受*Universe Today*采访时表示：“我们正在继续系统地覆盖我们预测第九行星可能存在的所有天空区域。利用Pan-STARRS的数据使我们能够覆盖迄今最大的区域。”

[Pan-STARRS](https://www2.ifa.hawaii.edu/research/Pan-STARRS.shtml)，全称为全景调查望远镜与快速响应系统，是位于哈雷阿卡拉天文台的合作天文观测系统，由夏威夷大学天文学研究所运营，望远镜建设由美国空军资助。研究人员利用第二阶段数据发布（DR2）的数据，旨在根据过去的研究结果缩小第九行星可能位置的范围。

[https://www.youtube.com/embed/tpLzRWqTlVc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/tpLzRWqTlVc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

最终，团队通过排除先前研究计算出的大约78%的可能位置，缩小了第九行星可能的位置范围。此外，研究人员还提供了第九行星的大约半长轴（以天文单位（AU）衡量）和地球质量大小的新估计，分别为500和6.6。那么，这项研究的最显著结果是什么，目前正在进行或计划进行哪些后续研究？

“虽然我很想说最重要的结果是找到第九行星，但我们并没有，” 布朗博士告诉*宇宙今日*。“所以相反，这意味着我们已经显著缩小了搜索范围。我们现在已经调查了大约80%我们认为第九行星可能存在的区域。”

[https://www.youtube.com/embed/2UZKME9WP9M?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/2UZKME9WP9M?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

就后续研究而言，布朗博士告诉*宇宙今日*，“我认为大型普查望远镜（LSST）是发现第九行星最有可能的项目。一旦它在一两年内上线，它将快速覆盖大部分搜索空间，并且如果第九行星存在，会找到它。”

LSST代表[空间与时间遗产调查](https://kipac.stanford.edu/research/projects/vera-rubin-observatorys-legacy-survey-space-and-time)，是一项天文调查计划，目前计划为期10年，旨在研究南半球，并将在智利的维拉·C·鲁宾天文台进行，该天文台目前正在建设中。LSST的目标包括研究识别近地小行星（NEAs）和太阳系内小行星，以及深空研究。这些包括调查暗物质和暗能量的特性以及银河系的演化。但找到第九行星的重要性是什么呢？

[https://www.youtube.com/embed/vBlFFEdfVVQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/vBlFFEdfVVQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

布朗博士告诉*宇宙今日*，“这将成为我们太阳系第五大行星，唯一一个质量介于地球和天王星之间的行星。这样的行星在其他恒星周围很常见，而我们突然有机会在我们自己的太阳系中研究一颗。”

科学家们在1846年发现了海王星后不久开始假设第九行星的存在，包括D. Kirkwood在1880年写的一篇[回忆录](https://articles.adsabs.harvard.edu/pdf/1880Obs.....3..439K)，以及后来美国天文学家克莱德·汤博的[1946年论文](https://articles.adsabs.harvard.edu/pdf/1946ASPL....5...73T)，他于1930年发现冥王星。更近期的研究包括2016年和2017年的研究，这些研究提供了第九行星存在的证据，前者由布朗博士联合撰写。这项最新的研究标志着对第九行星位置进行最完整的调查，布朗博士长期以来一直相信其存在，并告诉*宇宙今日*：“有太多单独的迹象表明第九行星存在。太阳系如果没有第九行星是很难理解的。”

[https://www.youtube.com/embed/n8UNkvIa_zc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/n8UNkvIa_zc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

他继续告诉*宇宙今日*：“...第九行星解释了关于外太阳系物体轨道的许多事情，否则这些都无法解释，并且每个都需要一种单独的解释。轨道方向的聚集是最好的例子，但还有许多物体的大近日点距离，高度倾斜甚至逆行的物体的存在，以及非常偏心的轨道的丰富度，这些轨道跨越了海王星的轨道。在太阳系中，这些都不应该发生，但所有这些都可以轻松地解释为第九行星的效应。”

第九行星是否存在，未来几十年我们将在何处找到它？只有时间能告诉我们，这也是为什么我们要进行科学研究！
