# 欧盟正在调查 Microsoft 对人工智能初创公司 Mistral 的 1.5 亿欧元的投资 • The Register

> 来源：[https://www.theregister.com/2024/02/28/eu_microsoft_mistral/](https://www.theregister.com/2024/02/28/eu_microsoft_mistral/)

欧盟委员会正在调查微软对法国初创公司 Mistral 的 1.5 亿欧元（1.63 亿美元）投资，此举恰逢 Mistral 发布与 OpenAI 的 ChatGPT 竞争的大型语言模型之后。

Mistral 成立于 2023 年，由来自 Google DeepMind 和 Meta 的研究人员创建，[Mistral](https://www.theregister.com/2023/06/15/mistral_ai/) 已从 Andreessen Horowitz 和 Lightspeed 等投资者那里筹集了超过 3.85 亿欧元（4.171 亿美元）。该初创公司今天估值约 18 亿欧元（20 亿美元）。

Mistral 最新推出了其最强大的大型语言模型 [Mistral Large](https://mistral.ai/news/mistral-large/)，并发布了一个网页应用，供网民使用该模型驱动的聊天机器人进行实验。此外，它还发布了一个更小的模型 Mistral Small，正如其名称所示，经过优化，速度更快、更紧凑。Mistral 之前还分享过其他模型，比如其七十亿参数的 Mistral-7B，发布于 [九月份](https://mistral.ai/news/announcing-mistral-7b/)。

为了让开发者更多地访问其技术，Mistral 还透露与 Microsoft 合作，将其开放和商业模型引入 Azure 平台。

作为回报，微软将通过其云计算平台为 Mistral 提供硬件和基础设施，用于训练和运行其未来的系统。Mistral 的联合创始人兼 CEO Arthur Mensch 在 [微软的声明中](https://azure.microsoft.com/en-us/blog/microsoft-and-mistral-ai-announce-new-partnership-to-accelerate-ai-innovation-and-introduce-mistral-large-first-on-azure/) 热情地支持该交易，表明双方对“人工智能行业的影响性进展和为全球客户和合作伙伴提供无与伦比的价值”有着共同的热情。

云提供商投资人工智能初创公司并涉及云基础设施访问并不少见。微软已向 OpenAI 投资了 130 亿美元，而 Google 和 Amazon Web Services 也大量投资 Anthropic。

但 Mistral 的交易引起了欧盟监管机构的关注。

这些交易对双方都是双赢。托管最新的大型语言模型能推动云平台的业务增长，而初创公司则需要大规模的 GPU 集群来开发新模型。然而，欧盟委员会担心人工智能与大科技公司的合并可能会巩固权力并扼杀创新，这可能使技术更难以监管，同时也会使较小的公司难以竞争。

欧盟委员会发言人莱娅·祖伯告诉Politico："委员会正在调查大型数字市场参与者与生成AI开发者和供应商之间达成的协议。在这种情况下，我们已收到提到的协议，我们将对其进行分析。"

上个月，欧盟监管机构对微软在OpenAI的股份发起了调查。欧盟委员会负责竞争政策的执行副总裁玛格丽特·维斯塔格尔在一份声明中宣称："我们邀请企业和专家告诉我们他们在这些行业中可能感知到的任何竞争问题，同时也密切监控AI合作伙伴关系，确保它们不会不当地扭曲市场动态。"

委员会已经就AI法案的措辞达成一致——这是一个全面的计划，旨在根据每个AI代表的风险制定规则。像Mistral Large或ChatGPT这样的生成AI模型被认为是"高影响通用"系统，可能构成"系统性风险"。开发者将被要求公开其训练数据的来源，并设计防范措施，以防止它们生成令人担忧的内容。

*The Register*已要求Microsoft和Mistral发表评论。
