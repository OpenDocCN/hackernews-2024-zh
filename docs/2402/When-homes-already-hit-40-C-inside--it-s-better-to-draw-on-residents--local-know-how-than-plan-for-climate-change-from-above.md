# 当住宅内温度已达40°C时，最好利用居民的本地知识，而不是从上而下规划应对气候变化。

> 来源：[https://theconversation.com/when-homes-already-hit-40-c-inside-its-better-to-draw-on-residents-local-know-how-than-plan-for-climate-change-from-above-221870](https://theconversation.com/when-homes-already-hit-40-c-inside-its-better-to-draw-on-residents-local-know-how-than-plan-for-climate-change-from-above-221870)

气候变化推动的极端天气更严重地影响[低收入社区](https://theconversation.com/climate-change-hits-low-income-earners-harder-and-poor-housing-in-hotter-cities-is-a-disastrous-combination-180960)。原因包括住房质量差和缺乏安全舒适的公共空间。这使得“气候适应能力”成为西悉尼等快速增长地区的[政府](https://wsroc.com.au/heat-smart-western-sydney-community)，[城市规划者](https://theconversation.com/what-is-climate-ready-infrastructure-some-cities-are-starting-to-adapt-91784)和[应急服务](https://www.redcross.org.au/emergencies/climate-ready-communities/)的紧迫问题。

我们与西悉尼的多元文化居民和社会住房提供者合作，探讨他们如何适应[日益加剧的高温](https://theconversation.com/why-western-sydney-is-feeling-the-heat-from-climate-change-more-than-the-rest-of-the-city-201477)。居民在室内外安装了热数据记录器。

去年夏天相对温和，但我们记录到一些住宅内温度高达40°C。回顾2019年的热浪，一位居民说：“草地上的泥土开裂，几乎能扭到脚踝。”

我们将这些数据与居民和社会住房提供者告诉我们的关于管理热浪以及需要改进的信息进行了关联。不同文化群体采用了不同的策略。通过该项目，居民分享了大量关于如何应对不断变化的气候极端的集体知识。

## 空调有其局限性。

官方对气候极端事件的响应通常依赖于[室内避难](https://basc.pnnl.gov/resource-guides/creating-cool-room-extreme-heat-events)。这些“最后手段”避难所大多依赖于可靠的电力供应，在热浪期间电力可能会中断。

有努力在其他地方（但不包括澳大利亚）制定“[被动生存能力](https://www.buildinggreen.com/op-ed/passive-survivability)”建筑法规。其目的是确保在热浪期间（或寒冷时期）即使断电数天，房屋仍能保持适度凉爽（或温暖）。

我们承认空调对于老年人和那些[健康状况](https://www.abc.net.au/news/2024-02-13/disability-pensioner-social-housing-no-air-conditioning-heatwave/103456146)欠佳的人群至关重要，但我们不希望放弃外出！

在户外，例如为无家可归者提供[临时制冷枢纽](https://theconversation.com/australias-first-mobile-cooling-hub-is-ready-for-searing-heat-this-summer-and-people-who-are-homeless-helped-design-it-218829)，这是一种[富有同情心的](https://www.sydney.edu.au/medicine-health/our-research/research-centres/heat-and-health-research-incubator/pop-up-mobile-cooling-hubs.html?#uniqueId_muBzCemD_5_button)方法。尽管重要，但这类方法仅止于“应对”。

也存在将城市最贫困居民视为无力改变自身境况的赤字叙事风险。我们的基于优势的行动研究方法寻求那些在社区中已有的集体知识和实践的替代解决方案。

## 研究是如何进行的？

[我们的项目](https://www.westernsydney.edu.au/ics/projects/living_with_urban_heat_becoming_climate-ready_in_social_housing)，在社会住房中生活与城市热的项目：成为气候准备的一部分，是更广泛研究计划[冷却共同体](https://www.coolingthecommons.com/)的一部分。它的焦点是共享空间和知识在设计气候适应性城市中的作用。

我们采用参与式设计方法。通过[与那些已经熟悉他们所在地点和社区的人们合作](https://www.tandfonline.com/doi/full/10.1080/15710880701875068)来制定适应策略。

作为第一步，为了更好地掌握每个地点的微气候情况，居民在他们的家中安装了数据记录器。数据显示，位置、城市密度的程度和住房类型影响了居民的热体验。

例如，在温莎，极端天气在家中感受更加明显。去年夏天，温莎和里士满的记录器记录了超过30°C的69天。平均而言，室内温度比室外高出6°C，并在四次场合达到40°C。

在Riverwood和Parramatta向东更远的地方记录的温度较低。然而，对于项目研究员塞巴斯蒂安·普福奇来说，这些数据也突显了城市热岛效应。在Riverwood，白天和夜晚的平均温度分别为25.8°C和25.4°C，因为砖石表面会保持热量。

我们将这些数据与居民和社会住房供应商告诉我们的关于如何管理各自地方的热和舒适度的信息进行了关联。

## 居民如何应对酷热？

在各地的双语设计工作坊上，居民群体之间的访谈主题得以分享。

觉得“我退缩”的居民觉得自己被困而不是在安全的环境中。

“取暖”意味着使用冰块、水喷雾、床单和毛巾来冷却空间和身体。中国居民使用米粥等食物来降温。居民在炎热的日子里还依靠房屋提供者和邻居关心他们的健康状况。

有车的居民“追逐空气”。这意味着在空调空间之间移动：朋友家、咖啡店和超市。

没有车的居民利用公共图书馆等凉爽场所，可以通过公共交通抵达。其他几十年来在该地区生活的家庭利用他们的本地知识追逐“迪怀医生”和其他本地微风，还有[在河里坐着](https://theconversation.com/when-the-heat-hits-inland-waters-look-inviting-heres-how-we-can-help-people-swim-safely-at-natural-swimming-spots-219333)。

居民通常会回到一整天都被热浪蒸发的家中。

他们有巧妙的方法通过窗户、门和风扇来使空气流动。在各个群体中，“制造空气”是一个重要的模式。

对身体舒适来说，空气流动和较低温度同样重要，尤其是对于在炎热中呼吸困难的人。正如一位参与者所说：“卧室里很闷。有时真的很难……我感觉不能开窗，因为气味和噪音。”

居民还制定了“规则”来管理家中的热量。这些规则从特定时间打开和关闭门窗，保持灯光熄灭，避免烘烤，到[空调配给](https://apo.org.au/node/119256)不等。

这些主题的分享使各群体受益匪浅。例如，大多数不开车的中国社区居民从未想过“追逐空气”。另一方面，利用米粥感觉凉爽对其他人来说是新鲜事物。

## 集体适应效果最佳。

在每个社区中，分享这些方法引发了关于更集体的适应形式的广泛讨论，包括在建筑和自然环境中的共享空间和实践。

这项研究引发了一些问题。例如，空调需要的密闭与许多居民视为健康的新鲜空气流动之间存在紧张关系。这对于[冷却中心蓝图](https://www.tandfonline.com/doi/full/10.1080/15710880701875068)和社会住房未来可能有什么影响，特别是在安全需求常常意味着封闭开口和上锁门的地方？

气候准备并不意味着加固不足的技术解决方案让我们困在里面，或者勉强的解决方案。这些将我们缩小到哲学家乔治奥·阿甘本所说的“[裸生命](https://www.oxfordreference.com/display/10.1093/oi/authority.20110803095446660)”状态，这种状态排除了过上美好生活的可能性。情况未必如此。

我们的研究正在试验适应性做法，利用对凉爽空间（无论是自然还是建筑的）的本地知识，并在文化之间分享这些做法。这表明我们可以将气候准备重新想象为繁荣社区的一部分。
