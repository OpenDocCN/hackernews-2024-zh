# 英国推出的生育治疗比较工具，涵盖IVF数据 | IVF | 卫报

> 来源：[https://www.theguardian.com/society/2024/jan/30/uk-fertility-treatment-comparison-tool-dashboard-ivf](https://www.theguardian.com/society/2024/jan/30/uk-fertility-treatment-comparison-tool-dashboard-ivf)

寻求试管婴儿等生育治疗的人现在可以访问一个在线工具，比较其地区与英国其他地区的国民健康服务（[NHS](https://www.theguardian.com/society/nhs)）资助情况。

The [dashboard](https://www.hfea.gov.uk/about-us/hfea-dashboard/)，由监管机构人类生育与胚胎学权威（HFEA）推出，被认为是世界首创，允许有意进行IVF和ICSI等过程的人士以及研究人员探索从1991年以来的数据，涵盖了665,690名患者。

除了其他功能外，仪表板的用户可以按年份、种族和原籍国筛选出34,725名新卵子或14,665名新精子捐献者；探索每年冷冻或解冻卵子的患者数量及其平均年龄；查看每年多胎率的趋势；以及探索[试管婴儿](https://www.theguardian.com/society/ivf)成功率随年龄的差异。

该仪表板还突出显示了一些关键统计数据，例如，接受国民健康服务(NHS)资助的周期百分比从2009年的38%下降到2021年的26%，而每次移植的胚胎出生率从1991年的7%上升到2021年的23%。

彼得·汤普森是英国生殖和胚胎管理局的首席执行官，他表示每年在英国进行的生育治疗超过70,000例。

他说：“随着过去30年生育领域的进展，改善我们如何呈现和分享数据的机会也在增加。我们的新数据仪表板标志着信息透明度更大的下一步。”

“对于研究人员来说，以前要找出英国生育治疗的重要数据意味着逐个提交查询。仪表板的推出将确保我们的大量数据能够被用来获得进一步的洞见，这可能反过来改善所有人的结果。”

亚当·巴伦教授是利兹教学医院国家健康服务信托的生殖医学顾问，他对数据的可用性和英国生殖和胚胎管理局的透明方法表示欢迎，并指出英国生殖和胚胎管理局的数据是全球试管婴儿信息的最大库。

他说：“新的仪表板使研究人员能够访问数据以进行研究，患者能够获取信息，从而更好地指导他们的生育旅程，并解密一些治疗结果统计背后的复杂性。”

斯图尔特·拉弗里，伦敦大学学院医院 NHS 基金会信托的生殖医学顾问，表示：“英国监管机构的这个新仪表板应该受到患者和诊所的欢迎。它的图形和表格格式以简单易懂的方式呈现了非常复杂的信息。

“这展示了我们如何利用世界上最全面的试管婴儿数据库中的信息，以造福今天寻求生育治疗的患者。”

“虽然我会建议我的患者使用这个仪表板，但我也会建议最好通过与您的生育医生进行个性化咨询来了解您自己的个人情况和成功的可能性。”
