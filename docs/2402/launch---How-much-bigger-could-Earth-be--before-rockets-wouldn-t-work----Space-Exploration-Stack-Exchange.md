# 发射 - 地球变得多大后火箭才无法运行？-空间探索堆栈交换

> 来源：[https://space.stackexchange.com/questions/14383/how-much-bigger-could-earth-be-before-rockets-wouldnt-work](https://space.stackexchange.com/questions/14383/how-much-bigger-could-earth-be-before-rockets-wouldnt-work)

首先，让我们看看[火箭方程](https://en.wikipedia.org/wiki/Tsiolkovsky_rocket_equation)：

$$\Delta v=\ln \left(\frac{m_0}{m_f}\right)v_e$$

这就说明了一个火箭可以改变其速度（Δv）的幅度。在更重的地球上，达到更高速度以实现最小轨道的要求将会增加（对于恒定密度，与半径成正比）。

如何增加火箭的Δv以保持速度？我们可以增加引擎的排气速度$v_e$，但化学发动机的截止速度约为5000m/s。我们可以做的另一件事是增加火箭的质量比$\left(\frac{m_0}{m_f}\right)$。这也有问题，因为我们实际上不能把燃料箱做成肥皂泡。分级是剩下的选择，你可以将一个大火箭放在一个小火箭下面，以获得更多的速度变化。这样你就会因指数性的开销而得到线性的收益。

举个例子，土星五号火箭进入了低地球轨道（~9000 m/s），向月球发送了一个有效负载（3120 m/s），服务模块将整个组合减缓到了低月轨道（820 m/s），最后登月舱着陆并再次起飞（2*1720 m/s）。此时服务模块里仍有一些未使用的燃料，所以我们将土星五号/阿波罗的总Δv称作17km/s。这少于2倍半径的地球所需的要求。阿波罗计划成本很高[需要引用]，因此在一个2倍地球的国家尝试进入轨道之前可能需要一段时间。正如你所言，限制因素是荒谬的低有效载荷比。

另一个考虑因素是增加的表面重力（在恒定密度下，与直径成线性关系）。这要求火箭具有更高的推重比，这将增加干重，减少可能的Δv。（它也增加了重力损失，但这主要是由于行星的低标高减少了阻力损失）。

最终，重力如此之大，以至于即使是最强大的发动机也无法将自身从地面上升离。这至少是一个明确的限制。

**更多的理论考虑，Δv需求是否真的有一个有限的极限？**

令人惊讶的是，这并不是。还记得我之前说过的关于级联的话吗：“你正在为指数级的开销获得线性的利益”。但我们可以花多少我们可以支出！考虑以下情景：我们在火箭底部添加越来越多的级，每一个级的质量与它上面的所有级别相同。然后每个级别的燃烧都给出了相同的质量比，因此每个级别都提供了相同数量的Δv。要增加10倍的数量，您需要增加10个翻倍的级别。要增加100倍的数量，您需要100次加倍。质量增长非常快，即使翻倍10次也是超过一千倍。但我们为什么要停止 :)

**但是我们*真的*可以继续无限地添加指数级更大的级吗？**

过一段时间，其他问题会出现。例如：火箭很长很细，以最小化阻力。这种形状对于非常大的火箭来说是无法保持的。原因并不是[立方-方关系定律](https://en.wikipedia.org/wiki/Square-cube_law)。保持相同的尺寸比例，高度加倍的火箭质量增加了8倍。但是火箭的底面积只增加了4倍。这意味着每个单位的面积必须支持更多的质量。迟早，即使是最强的材料也必须放弃，你必须放弃传统的火箭形状，选择一个更宽的底座。这会大大增加阻力！像这样的问题将继续出现：

“更多的质量意味着更多的问题，指数级的质量意味着指数级的问题。”

**总结：**

现代设计，比土星五号火箭更大的火箭，通过增加推重比的改进，可能在地球的两倍半径，八倍质量下成功进入轨道。这是一个可行性限制，那些更大得荒谬的火箭可能会有额外的几公里每秒的Δv，但这并不会很大改变数字。理论上，火箭可以成长直到阻力阻止它们，或者发动机不能再抬起它们自己。

或许你在某个时候想要利用行星上的资源发射一枚单独的火箭到轨道。
