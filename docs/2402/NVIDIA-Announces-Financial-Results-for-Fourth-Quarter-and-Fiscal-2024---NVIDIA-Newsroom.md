# NVIDIA公布了2024财年第四季度和全年财务业绩 | NVIDIA新闻室

> 来源：[https://nvidianews.nvidia.com/news/nvidia-announces-financial-results-for-fourth-quarter-and-fiscal-2024](https://nvidianews.nvidia.com/news/nvidia-announces-financial-results-for-fourth-quarter-and-fiscal-2024)

+   创纪录的季度收入达到221亿美元，比上一季度增长22%，比去年同期增长265%。

+   创纪录的数据中心季度收入达到184亿美元，比上一季度增长27%，比去年同期增长409%。

+   全年创纪录的收入达到609亿美元，增长了126%。

NVIDIA（纳斯达克代码：NVDA）今天公布了截至2024年1月28日的第四季度收入，达到221亿美元，比上一季度增长22%，比去年同期增长265%。

本季度，按照通用会计准则计算的每股摊薄收益为4.93美元，比上一季度增长33%，比去年同期增长765%。非通用会计准则计算的每股摊薄收益为5.16美元，比上一季度增长28%，比去年同期增长486%。

2024财年，收入达到609亿美元，同比增长126%。按照通用会计准则计算的每股摊薄收益为11.93美元，比去年同期增长586%。非通用会计准则计算的每股摊薄收益为12.96美元，比去年同期增长288%。

“加速计算和生成式人工智能已经到达转折点。全球范围内公司、行业和国家的需求正在激增。” NVIDIA创始人兼首席执行官黄仁勋表示。

“我们的数据中心平台由日益多样的驱动因素驱动 —— 大型云服务提供商和GPU专业化公司对数据处理、训练和推理的需求，以及企业软件和消费互联网公司的需求。汽车、金融服务和医疗保健等垂直行业现在已经达到数十亿美元的水平。”

“NVIDIA RTX，不到六年的时间，已经成为生成式人工智能的重要PC平台，受到1亿游戏玩家和创作者的喜爱。未来一年将带来重大的新产品周期，引领行业前进。请加入我们参加下个月的GTC，我们和我们丰富的生态系统将展示前景广阔的未来。” 他补充道。

NVIDIA将于2024年3月27日向所有登记股东支付每股0.04美元的季度现金股息，登记日为2024年3月6日。

**展望**

NVIDIA 2025财年第一季度的展望如下：

+   预计收入将为240亿美元，上下波动2%。

+   预计GAAP和非GAAP毛利率分别为76.3%和77.0%，上下波动50个基点。

+   预计GAAP和非GAAP运营费用分别约为35亿美元和25亿美元。

+   预计GAAP和非GAAP其他收入和支出将约为250亿美元，不包括非关联投资的收益和损失。

**非GAAP措施**

为了补充按照通用会计准则（GAAP）编制的NVIDIA简明合并财务报表，公司使用了某些财务表现的非GAAP措施。这些非GAAP措施包括非GAAP毛利润、非GAAP毛利率、非GAAP营业费用、非GAAP营业利润、非GAAP其他收入（费用）、净额、非GAAP净利润、每股摊薄非GAAP净利润或收益以及自由现金流。为了使NVIDIA的投资者更好地能够比较其当前结果与以往时期的结果，公司展示了按照GAAP和非GAAP财务措施的调和表。这些调和表调整了相关的GAAP财务措施，以排除收购终止费用、股权酬劳费用、收购相关和其他费用、知识产权相关费用、其他、非关联投资的收益和损失、与债务折扣摊销相关的利息费用，以及适用的这些项目的税务影响。自由现金流的计算方法是按照GAAP提供的经营活动现金流净额减去与固定资产和无形资产相关的购买及主要支付和固定资产及无形资产的本金支付。NVIDIA认为其非GAAP财务措施的呈现增强了用户对公司历史财务表现的整体理解。公司的非GAAP财务措施的呈现并不意味着单独考虑或替代按照GAAP编制的财务结果，公司的非GAAP措施可能与其他公司使用的非GAAP措施不同。
