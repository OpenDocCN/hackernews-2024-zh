# 软件如何工作 | 无淀书社

> 来源：[https://nostarch.com/howsoftwareworks](https://nostarch.com/howsoftwareworks)

"对于一些相当复杂的工具，这是我见过的最好的解释之一...读者会说‘啊哈！’"

**—IT World**

"结构清晰，文笔优美...对于我们视而不见的这些应用程序内部运作机制的高度愉悦的阅读体验。"

**—BCS, 计算机协会**

"作者用简单的语言和简单的图表解释概念。有好奇心的读者将能够‘揭开魔术’或解开软件如何工作的神秘面纱。"
