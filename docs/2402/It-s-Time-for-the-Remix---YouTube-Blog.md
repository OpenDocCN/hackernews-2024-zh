# 是时候进行混音了 - YouTube博客

> 来源：[https://blog.youtube/news-and-events/youtube-shorts-remix/](https://blog.youtube/news-and-events/youtube-shorts-remix/)

想象一下…那是一个周五早晨，你惊喜地从你最喜欢的艺人[Maggie Rogers](https://www.youtube.com/watch?v=JHh3w5xgSkc)那里收到一首新歌！那个周日，你和朋友们一起观看大赛，突然之间，Beyoncé发布了她即将发行的专辑的两首新歌，[TEXAS HOLD ‘EM](https://www.youtube.com/watch?v=238Z4YaAr1g)和[16 CARRIAGES](https://www.youtube.com/watch?v=hhKNjTb6U1Y)。几天后，新晋格莱美奖得主KAROL G发布了[Contigo](https://www.youtube.com/watch?app=desktop&v=NCmp7AV8O9s&feature=youtu.be&ab_channel=KarolGVEVO)，这是一首爱情大作，与Tiesto合作，正好赶上情人节。在每一个情况下，你会做什么？？立刻打开YouTube应用，观看官方音乐视频、视觉效果和Shorts！

但仅仅观看视频还不够 – 你想要参与到围绕这些歌曲建立的热潮中，并成为音乐运动的一部分。现在，通过YouTube Shorts的重新混音，你可以从一开始就参与其中。

这是如何做到的：直接从YouTube视频上，点击“Remix”，选择四个选项进行混音：Sound、Green Screen、Cut和Collab。选择其中一项或尝试不同Shorts中的所有选项 – 创作的可能性是无限的：

+   -   **Sound**：仅使用视频中的声音制作你的Short，为即将到来的纳什维尔之行打造完美的配乐。

+   -   **Collab**：与视频同时创建一个Short，这样你和朋友可以和艺人一起表演舞蹈。

+   -   **Green Screen**：将视频用作Short的背景，这样你可以录制实时反应，第一次听到时的情景。

+   -   **Cut**：无法忘怀音乐视频中的特定场景？剪辑出那5秒钟的片段，添加到你的Short中，让你随时可以重温。

为什么这么独特？在YouTube上，你可以重复观看音乐视频，查看其他粉丝根据同一首歌制作的Shorts，发现你最喜欢艺人的深度目录曲目，并通过重新混音将它们重塑为你自己的作品。**所有这些都可以在一个地方完成，而且这个地方只有YouTube！**
