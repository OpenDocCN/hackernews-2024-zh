# 1950年，一架载有44名乘客的美国军用运输机在育空无影无踪 | CBC 纪录片

> 来源：[https://www.cbc.ca/documentaries/in-1950-a-u-s-troop-plane-carrying-44-passengers-vanished-without-a-trace-in-the-yukon-1.7063704](https://www.cbc.ca/documentaries/in-1950-a-u-s-troop-plane-carrying-44-passengers-vanished-without-a-trace-in-the-yukon-1.7063704)

1950年1月26日，一架载有44人的美国空军运输机离开安克雷奇前往蒙大拿。它在严寒中起飞，剩余的日光时间极少。

道格拉斯 C-54 天航者 #2469 的机组人员应在每30分钟内报告航线情况。当飞机飞越育空时，他们通过无线电向 Snag 小前哨报告说机翼上结冰，但其他一切正常。

随后，天航者神秘失踪。直至今日，飞机及其乘客的任何迹象都未曾被找到。

[*天航者沉没*](https://gem.cbc.ca/skymaster-down/s01e01)是一个数十年来依然困扰着44名失踪人员家属的谜团。

飞机失踪后，美国空军立即展开了大规模的搜救行动。在1950年的冬天，全球各地的报纸都报道了天航者及其乘客的事迹，以及绝望的救援行动。育空大雪覆盖，气温急剧下降。美国和加拿大的飞机沿着飞行路径飞行网格图，机上的观察员透过结霜的窗户观察。情况混乱 — 有四架搜救飞机坠毁。

一则报纸剪报展示了迈克·蒂西克（Mike Tisik）的父母、兄弟姐妹，他是失踪的C-54天航者的副驾驶员，他们正在收听无线电报告，希望听到蒂西克平安的消息。（《天航者沉没》）

几周后，所有美国军用飞机被重新调往阿拉斯加湾寻找一架携带核弹头的失踪轰炸机。与天航者一同失踪的44人家属收到了死亡证明。美国空军再也没有返回育空进行搜寻。

道格拉斯 C-54 天航者（Douglas C-54 Skymaster）装备四台发动机，翼展100英尺，尾翼高26英尺。即使在广袤的育空荒野中，要隐藏一架如此巨大的飞机也很困难。它是在湖中吗？在山谷底部？是否有可能偏离航线，并撞山而毁？

在该地区有500多起已知的飞机失事记录。所有飞机均已定位、参观和核实。但天航者依然顽固地失踪。

[*天航者沉没*](https://gem.cbc.ca/media/skymaster-down/s01e01)讲述了受害者家属的故事，并带领观众北上，即使今天，一群勇敢的育空居民每年夏季仍在努力，希望能为那些家庭带来了结，终于找到天航者的踪迹。

朱迪·杰克逊（Judy Jackson）的父亲是这架失踪飞机上的无线电操作员。（《天航者沉没》）

想象一下，如果一场神秘的飞机失事深深地影响了你的家庭故事——几代人一直在思考*我们是否会找到真相？*不知道成了一种折磨。

[*天行者之谜*](https://gem.cbc.ca/skymaster-down/s01e01) 是一个几乎被时间遗忘的故事；这是一个几乎没有人知道的传说。它是加拿大历史上最伟大的未解之谜之一，背景是世界上最壮丽和致命的地貌之一，以及一群引人入胜且令人心碎的人物。

### 1950年，一架美军运输机在育空上空失踪。至今未被找到

一架失踪的飞机。44名失踪的人。一个令人难以置信的未解之谜。在CBC Gem上观看纪录片《天行者之谜》。
