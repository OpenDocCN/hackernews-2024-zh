# VR色情对于Apple Vision Pro用户不可行，因为苹果已经阻止了这一功能 - NotebookCheck.net新闻

> 来源：[https://www.notebookcheck.net/VR-porn-a-no-go-for-Apple-Vision-Pro-users-as-capability-blocked-by-Apple.801282.0.html](https://www.notebookcheck.net/VR-porn-a-no-go-for-Apple-Vision-Pro-users-as-capability-blocked-by-Apple.801282.0.html)

Apple已经禁用了Safari支持其新的[Vision Pro](Apple-Vision-Pro-weight-has-been-revealed-and-is-almost-as-heavy-as-a-12-9-inch-iPad-Pro.794956.0.html)头显上观看VR色情的能力。虽然苹果一直以来都有一个不允许iPhone和iPad上有色情内容的长期政策，但它从未试图阻止用户通过网络访问他们设备上的任何选择内容。当然，目前，它在Mac上的控制更少，就像其他PC一样，允许用户安装他们自己的应用程序并以任何方式查看任何内容。

尽管Vision Pro用户不能通过Safari观看色情内容，但Apple已经阻止了在沉浸式VR模式下观看色情内容的能力，这一功能在其他制造商的VR头显上并未被阻止。这意味着Vision Pro是世界上第一个无法播放空间色情内容的“空间计算机”。用户仍然可以在“平面”屏幕上观看常规色情内容，但这可能会让喜欢沉浸式色情的人感到失望 - 在多个方面。

对于苹果爱好者的一个变通方法是自己弄一个廉价的[Google Cardboard](Google-Cardboard-open-sourced-as-a-lifeline-for-mobile-VR.441427.0.html)或类似的老式手机VR观看器，然后像VR色情的早期一样使用他们的iPhone。然而，Vision Pro的高视觉保真度无疑会使其成为一个超越级的体验。历史上，色情行业一直是推动新技术主流采用的重要驱动力，然而，苹果似乎准备将该市场部分让给其他头显制造商。

对于这个问题的Reddit帖子下的评论值得一看。

**更新：** Vision Pro用户可以启用沉浸式网络内容 - 包括VR色情 - 然而，苹果已经使这个过程变得非常复杂，需要9个步骤来激活Safari的Vision Pro WebXR API。苹果目前将这一功能视为“实验性的”，这对于一个AR/VR头显在推出初期内容有限来说似乎是一种不寻常的疏忽。这表明，未来可能会默认启用此功能，但目前没有公开的时间表。考虑到苹果的“家庭友好”公共形象，苹果可能会永久地将这一功能隐藏起来。
