# ‘你可以改变视角’: 旋转的房子在新西兰出售 | 新西兰 | 卫报

> 来源：[https://www.theguardian.com/world/2024/feb/20/new-zealand-rotating-house-sale-auckland-maraeta-hillside-valuation-pictures](https://www.theguardian.com/world/2024/feb/20/new-zealand-rotating-house-sale-auckland-maraeta-hillside-valuation-pictures)

一栋位于[新西兰](https://www.theguardian.com/world/newzealand)的房子，可以像旋转木马一样持续旋转，自其主人35年前设计并建造以来首次挂牌出售。

这座建筑被昵称为‘灯塔’，坐落在奥克兰沿海的玛雷塔山坡上，位于一个直径两米的圆柱形钢基座上，据信是该国唯一的这种建筑。一次完整的旋转需要33分钟。

“优点在于你可以改变视野，可以调整阳光，可以躲开风……如果有风暴在夜间来临，你不想让卧室的窗户受到影响，你就可以转动它，”他周二告诉卫报。

“几周前我坐在那里，阳光照进来照在了电视上，所以我起身调整了一下房子。”

Dunick和他的同事花了五年时间设计了一个系统，确保所有服务，如水、电和污水管道都是独立的，并且可以随着房子移动而旋转 - 这是当地政府在批准建造之前要求的。

最终结果是“你一生中见过的最简单的系统”，Dunick说道，并补充说，它可以被改编成允许房屋上下升降，保护它们免受洪水或野火等自然灾害的影响。

这栋房子可以转向大海或丛林。照片：Wide Photography

Carolyn Hanson来自苏富比国际房地产，称这处价值超过NZ$1m的房产将通过招标程序出售。自上周挂牌出售以来，房子就被参观者涌入，Hanson说道：“当地人对此很感兴趣，现在我们也开始得到国际关注。”

现年80岁的Dunick计划于今年晚些时候从他目前在澳大利亚的家搬回新西兰，并希望在新西兰的北部乡村进行半退休。出售他喜爱的房子是苦乐参半的。

“我对此感到非常高兴，这是一个了不起的成就……而且领先于时代，”他说道。

“但是[新业主]可以期待更好的生活方式……你可以根据自己的需求调整房子，这太神奇了。”
