这里的微妙之处实际上在于允许/可用的操作是什么。

在没有任何其他操作的情况下，每一个 $2^{32}$ 的双射都是同构的，因此它们并不是唯一的。

对于有序 ("<") 的情况，存在一个同构：0 -> int32.min，1 -> int32.min+1，…… 这个是唯一的。

如果你添加任何其他内容（例如 "== 0" 或 "+"），它们就不再是同构的了。

二进制补码是一个双射，但对于大多数操作集，它不是同构的。
