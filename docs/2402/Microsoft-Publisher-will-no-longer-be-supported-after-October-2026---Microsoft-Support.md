# Microsoft Publisher将在2026年10月之后不再受支持 - Microsoft支持

> 来源：[https://support.microsoft.com/en-us/office/microsoft-publisher-will-no-longer-be-supported-after-october-2026-ee6302a2-4bc7-4841-babf-8e9be3acbfd7](https://support.microsoft.com/en-us/office/microsoft-publisher-will-no-longer-be-supported-after-october-2026-ee6302a2-4bc7-4841-babf-8e9be3acbfd7)

在Microsoft，我们致力于为客户提供在工作和生活中所需的工具。为了专注于新的好处，我们偶尔会移除功能和产品。

在2026年10月，Microsoft Publisher将达到生命周期终点。此后，它将不再包含在Microsoft 365中，现有的本地套件也将不再获得支持。在此之前，将继续支持Publisher，并且用户可以期待与今天相同的体验。

许多常见的Publisher场景，包括创建专业品牌模板、信封和标签打印，以及制作定制日历、名片和程序等，已经可以在其他Microsoft 365应用程序中找到，如Word和PowerPoint。您可以在[Microsoft Create](https://create.microsoft.com/en-us?ocid=oo_create_visitors_templates_publisher)上找到各种可定制的模板。

随着我们向Microsoft Publisher的退役迈进，我们正在探索通过诸如Microsoft Word、PowerPoint和Designer等应用实现常见Publisher场景的现代方法。我们将在有更多信息时进行更新。

*您可以继续使用当前功能的Publisher直到2026年10月。

永久版本Publisher的支持将在2026年10月结束，届时Office LTSC 2021也将终止支持。从那时起，Microsoft 365客户将无法访问Publisher。

随着日期的临近，我们将提供更新。* **不需要采取任何行动。  这些信息是为了让您有足够的时间计划，我们将在日期临近时提供更多更新。**
