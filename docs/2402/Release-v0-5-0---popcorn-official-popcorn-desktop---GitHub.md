1.  [v0.5.0](/popcorn-official/popcorn-desktop/releases/tag/v0.5.0)

**New Features:**

+   更新NW.js运行时至0.82.0（此版本中还支持0.44.5作为选项）

+   为Apple Silicon添加macOS构建（仅限NW.js 0.82.0）

+   添加工作中的动漫选项卡

+   添加观看过的选项卡

+   添加种子盒选项，以便在下载完成后退出应用程序

+   添加VLC flatpack外部播放器支持

+   添加电影/系列UI透明度选项

+   添加新的主题Dutchy's Dark Orange

+   切换到新的OpenSubtitles REST API

+   更新WebTorrent至1.9.7

**Bug Fixes:**

+   修复WebTorrent导致高CPU/内存使用的Bug

+   修复由于损坏的书签条目而导致列表无法加载的问题

+   修复标题翻译问题

+   修复当影片预告链接损坏时切换到默认Chromium播放器的错误

+   修复未设置名称属性的磁力链接无法保存的错误

+   修复没有来源链接时缺少提供者图标的问题

+   修复系列海报缩放功能

+   修复亮度和色调滤镜实现

+   修复mpv外部播放器的标题显示问题

**其他：**

+   更新构建系统

+   清理过时/不必要的代码

+   更新Torrent Collection提供者

+   更新种子追踪器

+   更新各种模块/依赖项

+   各种其他小修复和优化</turbo-frame> </main>
