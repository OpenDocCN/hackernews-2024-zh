# 英国长达数周的大选结束 | 今日历史

> 来源：[https://www.historytoday.com/archive/history-matters/end-britains-weeks-long-general-elections](https://www.historytoday.com/archive/history-matters/end-britains-weeks-long-general-elections)

“谈到要点时”，罗伊·詹金斯写道，“首相的最清楚的特权之一就是选择选举日期。” 首相们在做出决定时非常纠结。但在1918年之前，政府只确定了解散日期：选择各个选区的提名和投票日期的责任是由各个选区的返回官员负责的。

18世纪的大选可能需要长达两个月才能完成。每个选区不仅按照自己的时间表工作，而且在1785年之前，开放投票的天数没有实际限制。在1784年威斯敏斯特选区的选举中花了47天后，最终规定了15天的限制。在候选人必须支付所有选举成本的时代，更长的投票时间是富有的候选人让对手无法承受而认输的一种承认的方法。
