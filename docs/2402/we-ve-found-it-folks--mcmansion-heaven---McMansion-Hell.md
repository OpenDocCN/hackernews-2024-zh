# 我们找到了它，朋友们：麦克曼斯莫天堂

大家好。很高兴为大家带来我见过的最棒的房子。这是一位真正的远见者的房子。一个真正的临时工。一个真正的窗户风格的先驱。这所房子位于阿拉巴马州。建于1980年，造价约500万美元。它物有所值。也许更多。

现在，我知道你在想什么：“得了吧，凯特，这有点怪诞，但肯定不是麦克曼斯莫天堂。这绝对是一个在世俗界中的房子。炼狱。麦克曼斯莫炼狱。”好吧，现在让我成为你的但丁，年轻的朝圣者。欢迎。欢迎，欢迎，欢迎。

很难找到一个拥有一切的房子。一座房子自愿地将自己定位于后现代主义，但仍然无法放下前一时代精神病的最怪异时刻，布鲁斯·戈夫（Bruce Goff）和地球船，用汽车挡风玻璃建造的公社房屋，七十年代的迷幻嬉皮裂缝时刻。它是一切。它拥有一切。它是主题公园，是高科技。它是文艺复兴（以圣安东尼奥河滨道的意义来说）。它是中世纪的。也许是最伟大的拼凑品，紧贴山脉一侧，险象环生地俯瞰着一大片水域。看看它。就看看。

里面是白色的。这使它如梦似幻，几乎仁慈。它很明亮，因为这里是麦克曼斯莫天堂，而灰色则是麦克曼斯莫地狱。这所房子有着80年代乐观主义的压倒性光泽。在这所房子里，信用违约掉期还没有发明，但却可能存在。

对我来说，要我说“可卡因”这个词是需要付出很大代价的，因为我认为这是一个低级笑话。但这个例子让它变得似乎有可能，不是贬低的方式，而是解放的方式，感官的方式。有人创造了这个房子，为了有一种特定的体验，一种特定的感觉。它拥有真正幻想的元素，主题的。它的房间并非意在成为一个统一的组合，而是一系列场景，极不同的空间时刻，压缩，扩展，明亮，紧密。

然后还有这个厨房，出于某种原因。或者你这么认为。内部设计试图隐藏的一切，即房子有多么异想天开，它并未完全做到，因为这里的选择仍然奢华、放纵，尽管以更熟悉的方式。

很少有内部环境让人真正需要戴墨镜。为了服务透明度而创造的环境必须在某种程度上阻止元素过深渗透，同时保留它们的可取之处。我不认为有建筑师设计了这座房子。建筑师本应该能够获得专门为此目的设计的产品。建造这座房子的人可以访问建筑目录，但并非那些用于最高端或结构最复杂的项目。这里的定制性体现在材料的组合上，并通过这种方式将它们延伸到想象力的极限。借用查尔斯·詹克斯（Charles Jencks）的话来说，临时性是一个完美的描述。这是一种可得性和冒险的建筑。

一个小插曲。我们在外面。这座房子的后部没有外部景观，因为从其深处的瘦小草坪上获取一个外部视角是不可能的。这个空间的意图是为了达到同样的目的，即看向房子本身，正如从房子向外界眺望一样。

生活在城市中，我常常思考展示主义。生活在城市中本质上是展示主义的。房子是一种可透视的可见表面；当我穿着T恤赶去客厅关灯睡觉时，完全有可能被某人看到他们不该看到的一瞥。但这个空间只在一种暴露的意义上是展示主义，然而这种暴露不可能没有站点的保护，没有与每一双眼睛的距离。在这方面，双重自由得以确保。窗户暗示了可能看到的可能性。但没有人看到。

这座房子的核心蕴藏着一种奇特的概念混合。迪士尼世界舞台上的后现代古典主义柱子。俗语的未经雕琢的边缘。整个结构还带有一种有机主义的倾向，更接近戈夫（Goff）而非高迪（Gaudí），在这里我们看到房子最有机的形式之一，如单块石或贝壳般的梳妆台与镜子和白色的发光假象混合在一起。一个背光的洞穴，既原始又表演性，实质上是豪华浴室的辩证法。

然而，我们的大杂院天堂仍然是一座大杂院。它仍然是故意标示财富的积累，非常符合引发羡慕的次要目的，一个没有太多内在统一性的豪宅。高尔夫的存在，木材的存在，带有男性和家长权力象征的存在，伴随着奢华的暗流，进一步印证了这一点。大杂院可以向艺术形式迈进，但在达到上帝所在之处之前，还有许多层次要攀登。
