# coffeestains – 为文档添加咖啡污渍

这个包为 LaTeX 提供了一个长期缺失的基本功能：直接在页面上添加咖啡污渍，而不是手工添加，可以节省大量时间。

[下载](https://mirrors.ctan.org/graphics/pgf/contrib/coffeestains.zip) 这个包的所有内容，打包成一个zip文件（493.0k）。
