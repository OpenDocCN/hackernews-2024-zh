# 马克·扎克伯格称苹果的DMA规则“如此繁琐”，他怀疑任何开发者都不会选择加入 | TechCrunch

> 来源：[https://techcrunch.com/2024/02/01/mark-zuckerberg-calls-apples-dma-rules-so-onerous-he-doubts-any-developer-will-opt-in/](https://techcrunch.com/2024/02/01/mark-zuckerberg-calls-apples-dma-rules-so-onerous-he-doubts-any-developer-will-opt-in/)

Meta首席执行官马克·扎克伯格也加入了批评苹果对[欧盟新数字市场法（DMA）规定](https://techcrunch.com/2024/01/25/apple-dma-changes/)的声音，该法规强制苹果开放其App Store，并允许开发者使用自己的支付系统，等等。在Meta今天下午的Q4盈利电话会议上，扎克伯格回应了投资者的问题，询问Meta对DMA的看法，他表示，苹果的新规则“如此繁琐”，以至于他会对任何开发者采用这些规则感到惊讶。

新规定旨在通过允许其他公司运行自己的应用商店并收取自己的费用来增加应用经济中的竞争，理论上这将使它们能够绕过苹果的佣金和费用。但是，苹果对DMA的遵守减少了佣金，却增加了其他新费用，包括新的“核心技术费”，这些费用将适用于任何采纳DMA规则的开发者，无论他们的应用在何处发布，包括在App Store上。否则，苹果表示开发者可以选择继续使用现有的佣金结构，根据应用的覆盖范围和其他因素，苹果将抽取应用内购买收入的15%至30%。

如果DMA写得更加严密，Meta本可以利用新法律推出自己的应用商店。例如，它曾考虑过提供Facebook Games商店的想法，但是苹果的规则迫使Facebook在应用中删除任何实际的游戏功能。Meta最终放弃了，在其推出两年后[关闭了该应用](https://techcrunch.com/2022/08/30/meta-shutting-down-facebook-gaming-app/)。

从扎克伯格今天的评论来看，Meta似乎没有任何打算重新推出该功能。

“我不认为苹果的事情对我们会有任何不同，因为我认为他们的实施方式，我会非常惊讶，如果有任何开发者选择进入他们拥有的另一种应用商店，”扎克伯格告诉投资者。“他们使它如此繁琐，而且我认为，与欧盟法规的初衷如此不符，以至于我认为任何人，包括我们自己，都很难真正认真对待他们在那里做的事情。”

Meta加入了批评苹果遵守DMA的其他科技公司的行列，包括[Epic Games](https://techcrunch.com/2024/01/25/epic-games-ceo-calls-out-apples-dma-rules-as-malicious-compliance-and-full-of-junk-fees/)、[Spotify](https://techcrunch.com/2024/01/26/spotify-calls-apples-dma-compliance-plan-extortion-and-a-complete-and-total-farce/)、[Mozilla](https://techcrunch.com/2024/01/26/apple-dma-webkit/)和[Microsoft](https://www.theverge.com/2024/1/29/24054943/microsoft-apple-app-store-dma-rule-change-response)。Epic Games曾因反垄断问题起诉苹果并大部分失败，称苹果的DMA规则是“恶意遵从”且充满“垃圾费用”。Spotify则称之为“勒索”和“彻头彻尾的闹剧”，而Microsoft则表示这是“朝错误方向迈出的一步”。另一位苹果的显赫批评者[Match表示尚未决定](https://techcrunch.com/2024/01/31/match-group-says-it-hasnt-yet-decided-if-it-will-opt-into-apples-new-dma-rules/)是否会接受新的DMA规则，因为他们仍在评估中。
