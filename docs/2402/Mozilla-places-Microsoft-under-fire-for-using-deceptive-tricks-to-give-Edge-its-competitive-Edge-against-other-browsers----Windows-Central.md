# Mozilla因Microsoft使用欺骗性技巧来帮助Edge在Windows上与其他浏览器竞争而对其进行抨击。 | Windows Central

> 来源：[https://www.windowscentral.com/software-apps/browsing/hi-microsoft-please-stop-using-harmful-designs-and-deceptive-tactics-to-give-edge-the-competitive-advantage-over-other-browsers-on-windows-says-mozilla](https://www.windowscentral.com/software-apps/browsing/hi-microsoft-please-stop-using-harmful-designs-and-deceptive-tactics-to-give-edge-the-competitive-advantage-over-other-browsers-on-windows-says-mozilla)

## 需要了解的事情

+   Mozilla委托的新研究揭示了Microsoft如何利用有害的设计和欺骗性策略来推广其Edge浏览器给Windows用户。

+   报告详细描述了这家科技巨头如何利用误导性的UI和广告等手段，阻止用户将其他浏览器设为Windows上的默认浏览器。

+   Mozilla指出，这使得Google Chrome和Firefox等其他浏览器处于不利地位，并呼吁监管机构调查此事。

* * *

即使到了2024年，浏览器战争依然存在。用户对他们设备上使用的浏览器有各种各样的兴趣和需求。[Google Chrome](https://www.windowscentral.com/tag/google-chrome)和[Microsoft Edge](https://www.windowscentral.com/tag/microsoft-edge)可以说是用户中最受欢迎的浏览器，但这是自愿还是设计的结果？

Mozilla的人士明确表示事情是后者，进一步将责任推给了Microsoft。该公司指出用户无法自由选择在Windows设备上使用任何浏览器作为默认浏览器（[链接](https://research.mozilla.org/browser-competition/over-the-edge-the-use-of-design-tactics-to-undermine-browser-choice/)）。

Mozilla委托Harry Brignull和Cennydd Bowles以及研究人员和专家调查此事。事实证明，Microsoft可能正在通过阻止在Windows上选择默认浏览器来玩弄手段（[via TechRadar](https://www.techradar.com/computing/browsers/mozilla-blows-the-whistle-on-microsoft-over-the-edge-report-accuses-windows-11-maker-of-browser-bias-and-deceptive-tactics)）。

在报告“[Over the Edge: The Use of Design Tactics to Undermine Browser Choice](https://research.mozilla.org/files/2024/01/Over-the-Edge-Report-January-2024.pdf)”中，研究人员指出Microsoft如何将其Edge浏览器放在其操作系统的核心位置，使其竞争对手处于劣势。

报告还概述了这家科技巨头利用的欺骗性策略，包括误导性的UI和广告、被字眼和呈现为系统警告的警示通知等。

去年，微软、谷歌和苹果被欧盟委员会列为[监管门户企业之一](https://www.windowscentral.com/microsoft/microsoft-added-to-eu-gatekeeper-list-but-bing-and-edge-are-under-investigation)。微软因其Windows操作系统和LinkedIn入选，谷歌则因其Chrome浏览器和搜索引擎而入选。委员会要求这些门户企业在六个月内遵守DMA规定；违规将面临巨额罚款和处罚。DMA要求之一是要求列入名单的公司使其服务具备互操作性。

有趣的是，微软的Bing最初被列为门户服务，但该公司辩称[搜索引擎未达到此类别的门槛](https://www.windowscentral.com/software-apps/bing/microsoft-argues-bing-isnt-big-enough-to-make-eus-gatekeeper-list-says-report)。委员会对此事进行了调查，并最近得出结论，[Bing和Edge在数字市场上并不占主导地位](https://www.windowscentral.com/microsoft/microsoft-edge-and-bing-might-get-a-free-pass-from-europes-dma-since-they-arent-dominant-enough-in-digital-markets)，因此免除了DMA的规定。

Mozilla希望对微软Edge的DMA规定能给Firefox一些喘息空间，并在公平竞争的平台上有机会竞争。但实际情况似乎已经失去了其竞争优势。微软的CEO萨提亚·纳德拉对搜索表达了相同的看法，还补充说[谷歌对Bing不公平](https://www.windowscentral.com/software-apps/bing/google-doesnt-play-fair-with-bing-says-microsoft-ceo-satya-nadella)。

## 这对微软而言只是同一脚本，只是不同的一出戏。

（图片来源：Future）

这不是我们第一次了解到微软强行推广其浏览器和搜索引擎的情况。去年末，一些用户在使用微软Edge下载谷歌Chrome时，被要求填写调查问卷，详细说明[为什么想尝试其他浏览器](https://www.windowscentral.com/software-apps/browsing/edge-begs-for-answers-microsoft-polling-users-who-download-chrome)。

我们怎么能忘记微软故意在谷歌Chrome下载页面上投放广告，或许是为了阻止用户过渡？事实上，Windows 11和10预装的微软Edge并没有改善情况。

Mozilla在[深度报告](https://research.mozilla.org/files/2024/01/Over-the-Edge-Report-January-2024.pdf)中进一步阐述其观点，并恳请监管机构审视此事以促进健康竞争。
