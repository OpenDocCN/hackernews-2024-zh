# Think Python，第3版

*Think Python*是一本面向从未编程过的人们或者曾经尝试过但遇到困难的人们的Python入门书籍。

[这是Green Tea Press的书籍页面](https://greenteapress.com/wp/think-python-3rd-edition/)。

对于第三版来说，最大的改变是：

+   现在整本书都在Jupyter笔记本中，所以你可以在一个地方阅读文本、运行代码并完成练习。使用以下链接，你可以在Colab上运行笔记本，这样你就无需安装任何东西就可以开始了。

+   文本进行了大幅修订，并且有几章重新排序。现在有更多的练习题，我认为其中很多都更好。

+   每章末尾都有关于使用工具如ChatGPT和Colab AI进一步学习和获取练习帮助的建议。

本书预计将于2024年6月由O’Reilly Media出版。您现在可以从[O’Reilly Media](https://www.oreilly.com/library/view/think-python/9781098155421/)或[Amazon](https://www.amazon.com/_/dp/1098155432?smid=ATVPDKIKX0DER&_encoding=UTF8&tag=oreilly20-20&_encoding=UTF8&tag=greenteapre01-20&linkCode=ur2&linkId=e2a529f94920295d27ec8a06e757dc7c&camp=1789&creative=9325)预订，或者如果您想支持本地书店，可以在[Bookshop.org](https://bookshop.org/a/98697/9781098155438)购买。

## 教师资源

如果您在使用本书进行教学，这里有一些可能对您有用的资源。

+   您可以从[此GitHub存储库](https://github.com/AllenDowney/ThinkPythonSolutions/tree/v3/soln)下载带有解决方案的笔记本。

+   每章的测验以及整本书的总结测验，可根据请求提供。

+   *使用Jupyter进行教学与学习*是一本在线书籍，提供了如何有效使用Jupyter的建议。您可以[在这里阅读这本书](https://jupyter4edu.github.io/jupyter-edu-book)。

+   在课堂上使用笔记本的最佳方法之一是现场编码，即教师编写代码，学生们在自己的笔记本中跟随。想要了解更多关于现场编码以及教授编程的其他优秀建议，我推荐由The Carpentries提供的教师培训，[您可以在这里阅读](https://carpentries.github.io/instructor-training)。
