# 今天是33年前的今天：祝《[旅鼠](https://en.wikipedia.org/wiki/Lemmings_(video_game))》生日快乐！- The Scottish Games Network

> 来源：[https://scottishgames.net/2024/02/14/it-was-33-years-ago-today-happy-birthday-lemmings/](https://scottishgames.net/2024/02/14/it-was-33-years-ago-today-happy-birthday-lemmings/)

今天，2024年2月14日，标志着《旅鼠》诞生33周年，这款游戏不仅仅是娱乐，更成为了文化标志和苏格兰蓬勃发展游戏产业的催化剂。但在绿发大军全球入侵屏幕之前，让我们回到1991年，追溯它的非凡历程。

由DMA设计团队（现在是Rockstar North）的头脑孕育而生，这家位于邓迪的小工作室创造了《旅鼠》，这是一个革命性的概念。与其毁灭一切不同，你的任务是拯救这些坠落的啮齿动物的生命。

然而，DMA的天才在于他们的执行力。凭借迷人的角色设计（在惊人的小尺度上）、令人上瘾的解谜机制，以及DMA即将成为拙劣幽默的一丝不苟，他们将一个复杂的概念转变为任何人都能掌握的游戏。

*旅鼠*提供了一个简单的前提：通过分配角色如阻挡者、攀爬者、建筑者和漂浮者，引导一定数量的旅鼠到达出口。但简单背后隐藏着深度。每个关卡都提出了独特的挑战，需要战略思维、快速反应和一丝试错式的血腥死亡。

成功来得很快。*旅鼠*征服了游戏机和电脑，销售了超过1500万份副本，并成为1991年英国最畅销的游戏。奖项和荣誉铺天盖地，但也许最重要的影响是对苏格兰游戏产业本身的影响。《旅鼠》让DMA设计团队名扬四海，吸引了人才和投资，并激发了世界上第一个游戏学位。

苏格兰游戏网络采访了几位原团队成员，询问他们对游戏影响的看法：

#### 原创小东西被喷得稀烂的动画的创作者迈克·戴利说：

> 我对旅鼠的遗产感到不可思议。无论我走到哪里，都有老粉丝和新粉丝喜爱这款游戏。凭借其风格和易于接近性，它不仅娱乐了大家，还让家庭更加紧密，因为孩子们与不玩游戏的父母和祖父母一起玩耍。我经常收到人们的来信，告诉我他们与从未对游戏感兴趣的亲戚一起玩游戏的快乐回忆，能够与他们分享自己的爱好，对他们来说意义重大。
> 
> 即使是现在，在展会上，33年后，你仍然会看到一些人装扮成旅鼠，并表达对游戏、音乐、音效、角色的喜爱 - 或者是他们在游戏中毫无用处，但却喜欢简单地将它们炸掉！
> 
> 旋风旅鼠仍然是我参与其中最自豪的游戏，在第一人称射击游戏的世界中，它如今与年轻人和老年人一样受欢迎

#### 罗素·凯告诉我们，他的梦想是将这些游戏带给新一代：

> 33年前，我们发布了一款至今仍受欢迎的游戏，这非常令人满足，如果当时告诉我们这一点，我认为没有人会相信。多年来，我们对这个系列情感上有所起伏，但它在我们心中占据着特殊的位置，我个人希望能更新角色和系列，但索尼却嫉妒地控制着权利，有机会看到旋风旅鼠如何在现代游戏世界中发挥作用将是非常棒的！

旋风旅鼠的影响远超过了苏格兰。可以说它使解谜游戏变得流行，启发了诸如*神奇机器*之类的游戏。它对物理学和用户生成内容的重视奠定了未来沙盒游戏的基础（甚至可能是*Minecraft*…？）

此外，它的幽默和难忘的角色巩固了DMA Design在创新、惊喜和引人入胜的游戏玩法方面的声誉，为未来的经典作品如*侠盗猎车手*、常被忽视（但更加疯狂）的*坦克战术*、文化经典*灭绝之日*和被低估的*野性金属乡村*铺平了道路。

如今，旋风旅鼠仍然是一款受人喜爱的经典解谜游戏，重新在各种平台上发布，并激励着新一代设计师。但它的遗产远不止是怀旧的像素记忆。邓迪市中心设有一组亲爱的[旋风旅鼠雕像系列](https://scottishgames.net/2013/10/14/lets-go-lemmings-in-the-real-world/)，努力工作，在俯瞰泰河的花园门口攀爬和架桥。

在2011年的20周年纪念日上，[一块牌匾揭幕了](https://scottishgames.net/2021/02/15/happy-birthday-lemmings-30-today/)，位于城市珀斯路的底部，纪念DMA的第一个办公室，游戏最初诞生的地方。
