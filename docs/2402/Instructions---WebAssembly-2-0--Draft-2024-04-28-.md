# 指令 — WebAssembly 2.0（草案 2024-04-28）

> 来源：[https://webassembly.github.io/spec/core/binary/instructions.html](https://webassembly.github.io/spec/core/binary/instructions.html)

# 指令

[指令](../syntax/instructions.html#syntax-instr)由*操作码*编码。每个操作码由一个字节表示，并在存在时跟随指令的直接参数。唯一的例外是[结构化控制指令](#binary-instr-control)，它们由几个操作码框定其嵌套指令序列。

注意

编码指令时保留了字节码范围中的间隙，用于未来扩展。

## 控制指令

[控制指令](../syntax/instructions.html#syntax-instr-control)有不同的编码方式。对于结构化指令，形成嵌套块的指令序列以明确的操作码 \(\href{../syntax/instructions.html#syntax-instr-control}{\mathsf{end}}\) 和 \(\href{../syntax/instructions.html#syntax-instr-control}{\mathsf{else}}\) 结束。
