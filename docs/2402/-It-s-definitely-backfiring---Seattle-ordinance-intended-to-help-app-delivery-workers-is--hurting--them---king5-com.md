# “它绝对是适得其反的”：西雅图旨在帮助应用程序送货员的法令正在“伤害”他们 | king5.com

> 来源：[https://www.king5.com/article/news/local/seattle-ordinance-intended-app-delivery-workers-hurting-them/281-9516c79c-3161-41f3-a662-798b9db16d3f](https://www.king5.com/article/news/local/seattle-ordinance-intended-app-delivery-workers-hurting-them/281-9516c79c-3161-41f3-a662-798b9db16d3f)

一名送货员表示，新的法令正在伤害司机和顾客。

西雅图 — 根据几名司机的说法，旨在给予食品配送应用程序司机更高生活工资的西雅图市新法令正在“适得其反”。

你可能已经注意到了Doordash和Uber Eats订单上的新的5美元费用，但这不仅仅让沮丧的顾客[删除他们的应用，正如我们报道的那样](/article/money/food-delivery-fee-angering-seattle-app-users/281-45019904-27a4-4e9a-9cd1-b7ee4bbdb9b8)。我们现在了解到，这个法令旨在帮助的人们正在受到伤害。

根据我们听到的几位司机的说法，自1月13日以来，曾被认为是这些应用程序上的“热点”的地方感觉有点冷。其中包括Gary Lardizabal，西雅图长期从事基于应用程序的食品配送司机。

“在法令颁布之前的周日，” Lardizabal 说。“你知道，我们会考虑早餐。今天，我甚至没有碰它。他们不会下单。这绝对是适得其反的。”

自从法令上个月生效以来，Mia Shagen 表示她的送货机会大大减少了。

“我一无所获，” Shagen 说。“我不会坐在这里几个小时就为了一个订单。”

即使在通常繁忙的送货日，也可以看到一群司机在高密度的餐厅区域一起等待。

“它不是在下午2:30或3:00停止，而是在下午一点左右，有时甚至更早，” Shagen 说。“所以确实在下午一点左右，突然间就没有订单了。”

[支付法案](https://www.seattle.gov/council/issues/past-issues/payup)，正如城市所称，旨在通过使他们有权获得“最低工资”，或者换句话说，根据每个报价工作的时间和行驶的里程来改善零工工作者的工资。

[正如我们报道的，Doordash表示](https://about.doordash.com/en-us/news/responding-to-seattles-new-delivery-laws)他们的Dashers将获得更多报酬：至少每小时26.40美元，不包括小费，在他们的估计中。

“他们没有说全面，” Shagen 说。“假设你一直在工作，那么是的，你会赚那么多钱。但现在情况并非如此。因为人们不再像以前那样经常下单了。小费在下降，因为他们认为我们赚了这么多钱。”

一名司机分享了他去年这个星期的收入：931美元。但这个星期，他只赚了464.81美元。

Lardizabal 说，他们的“衣食父母”经常是靠近亚马逊的南湖联合。但是 KING 5 在上周日访问该地区时与几名无聊的送货员进行了几次对话，他们再次强调自己的工资已被削减。

此外，Lardizabal 和其他人告诉 KING 5 他们认为现在他们正在与更多的司机竞争。

“所有在汽车、飞机、火车、汽车、摩托车上的人都聚集在这座城市，” Lardizabal 说道。“我们在奋斗。而且我们真的没有拿到每小时 $26。”

最近市长布鲁斯·哈雷尔在一份声明中 [称赞了该法律的实施，](https://news.seattle.gov/2024/01/12/office-of-labor-standards-marks-effective-date-of-app-based-worker-minimum-payment-ordinance-and-extended-coverage-of-app-based-worker-paid-sick-and-safe-time-ordinance-with-seattle-center-event/)将兼职工人描述为“西雅图经济的关键”。

最初支持该法案的西雅图市议员 Lisa Herbold 和 Andrew Lewis 都已不再任职。

去年春天，Herbold 在一条推文中表示：“这些公司和其他公司支付工人合理工资并提供基本的职场保护已经迟到了。”
