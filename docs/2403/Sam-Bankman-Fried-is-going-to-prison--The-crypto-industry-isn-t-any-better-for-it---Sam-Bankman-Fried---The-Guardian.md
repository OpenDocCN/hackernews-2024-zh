# -   Sam Bankman-Fried 将入狱。加密行业也未因此变得更好 | Sam Bankman-Fried | 卫报

> 来源：[https://www.theguardian.com/global/commentisfree/2024/mar/28/sam-bankman-fried-prison-crypto-regulation](https://www.theguardian.com/global/commentisfree/2024/mar/28/sam-bankman-fried-prison-crypto-regulation)

-   加密货币行业弥漫着一种如释重负的情绪。传教士们宣扬着这个好消息，行业已经摆脱了像 Sam Bankman-Fried、[Alex Mashinsky](https://www.theguardian.com/technology/2022/jul/15/celsius-network-crypto-firm-deficit-bankruptcy-funds)、[Do Kwon](https://www.theguardian.com/world/2023/mar/23/man-suspected-of-being-crypto-fugitive-do-kwon-arrested-in-montenegro) 和 Changpeng Zhao 这样的人物。他们宣称，加密货币终于可以从炼狱般的“狂野西部”时代中脱颖而出，成为一个受到监管者和投机者双重祝福的金融界的尊贵行业。

那种欢欣鼓舞的态度促成了加密货币价格的飙升，在[周四宣判至25年监禁](https://www.theguardian.com/business/2024/mar/28/sam-bankman-fried-sentenced-prison-ftx-fraud)之前的几周，这些价格超过了先前的历史最高点。

这种解脱是不合理的，同样不合理的是对加密货币行业将变得更好、更不同寄予的信任。自 Bankman-Fried 被定罪以来，没有任何改变能阻止新一轮像上次一样毁灭性的加密狂潮。

## 加密货币之王的垮台

Bankman-Fried，曾经的加密货币大亨，背后是[FTX](https://www.theguardian.com/business/ftx)交易所和Alameda Research交易公司，他在十一月初因七项欺诈和串谋罪名被定罪。他的调查和审判，以及他曾经的数十亿美元企业的破产程序，帮助揭示了一个从始创以来基本上逃避监管和监督的行业中不法行为的阴暗面。尽管FTX的商业行为震惊了那些习惯于传统金融更严格监管世界的人，审判证词和破产申请书，以及其他失败的加密货币企业的类似申请书，突显了在加密货币界发生的事情是这个行业的常态。

FTX 和 Alameda Research 由同一些人运营，甚至在同一个办公室工作，尽管明显的利益冲突源于分享交易所、市场制造商和资产保管者的角色。原本应该被保护的客户资金与公司资产混在一起，随后在从巴哈马房地产到超级碗广告再到个人朋友经营的公司的投资等各种事物上被任意挥霍。员工在个人与职业之间模糊界限，Bankman-Fried 的长期离合女友 Caroline Ellison 也担任其顶级副手之一。她后来证实，他们动荡不安的关系导致了公司内部混乱，并促使她决定成为FTX欺诈案的共谋者之一（她也因认罪七项指控而等待判刑）。另一家加密货币公司的CEO [作证](https://www.coindesk.com/policy/2023/10/13/sbf-trial-crypto-lender-blockfi-believed-alameda-was-solvent-given-balance-sheet-it-was-shown-ceo-testifies/)，他的全部尽职调查流程基于未经审计的财务报表。他称这是公司加密对手的常态，“因为审计问题”。这导致了他的公司破产。

## 缺席的监管者

[山姆·班克曼-弗里德](https://www.theguardian.com/business/sam-bankman-fried)及其诸如他之辈在2022年的衰落，并非源于积极主动的监管机构通过打击广泛的不当行为来保护投资者，而是来自建立在脆弱、高风险贷款和虚幻代币上的加密货币公司的巨大崩溃。这些倒闭对那些相信将资金投入加密货币是值得的人来说是毁灭性的。这更像是赌博。不少人认为持有其加密代币的公司受到类似传统银行的监管。当这些代币消失时，许多人对监管机构缺乏保护感到震惊，甚至有些人惊讶地发现，在银行倒闭事件中可以保护他们的存款保险却不存在。

与其保护投资者免受这些掠夺性加密货币计划的侵害，金融监管机构和执法者只有在捡拾碎片并检查数百万人被摧毁的投资时才介入。这些机构一直在赶进展，迟迟地调查和起诉像班克曼-弗里德或他的加密货币竞争对手常鹏赵这样的人，FTX和币安的问题在膨胀到巨大规模后才开始处理。

## 加密货币的波峰和波谷将会持续。

幸存的公司虽然对该行业新获得的合法性喜出望外，却无法指出任何阻止这个多事之秋重演的改变。

我们以前就见过这种循环。比特币的早期热情在2014年达到高峰，随后由于Mt Gox加密货币交易所的爆炸性增长和灾难性崩溃而暴跌，该交易所因盗窃和管理失误而损失了数十万个比特币。十年后，大多数在Mt Gox交易所存有代币的人仍在等待看到对今天将价值数十亿美元的资产进行任何补偿。2017年的另一个加密货币泡沫以监管机构对“首次代币发行（ICO）”变得明智，充斥欺诈的策略而结束崩溃，允许公司避开一般伴随着从公众寻求投资的披露和监管。监管机构和执法机构在2017年的ICO骗子时代仍在提起执行行动，当加密货币在2020年和2021年进入下一轮狂热时，这些机构已经在忙于起诉前几年的不当行为，而不是关注当今的情况。

监管机构试图加强对该行业的投资者保护，但加密行业一直激烈反对。尽管该行业宣称希望更多的监管，但其一直在试图追赶过去的灾难，浪费宝贵的时间。立法者得到了强大的资源充足的加密游说团体的支持，他们帮助阻碍了规则的改变，这些规则可能帮助机构采取更积极的预防措施。至于新法规方面，由于行业利益的压力，立法者一直陷入僵局，试图将当前的宽松监管状态通过豁免和漏洞进行编码。加密行业认为这将允许持续的“创新”，尽管迄今为止该行业几乎没有创新之处，除了找到新的、有创新性的方法来欺骗人们钱财。

## 像SBF一样的人还会出现。

银行曼·弗里德曼很可能会因他策划的欺诈行为在监狱里度过大量时间。但如果不能改变行业的运作方式，也没有监管机构来阻止在其存在的现在十五年里所暴露的滥用和贪婪行为，我们将注定历史将不断重演。更多的银行曼·弗里德曼会出现，被轻松赚钱的希望所吸引，以及很少可能出现后果的情况。有多少像他这样的人能够逃避惩罚甚至审查呢？

加密货币的支持者将从普通民众相信这个行业的问题已经随着几个骗子被抓而解决的故事中获益良多。当他们说出所有的坏分子已经被铲除时，请问：实际上有什么改变？采取了哪些新的保护措施来防止同样类型的骗局再次出现，欺骗新的受害者？

监管机构和立法者未能采取任何措施积极保护公众，同时允许加密公司进行广告宣传和招聘新客户，这些客户看起来更有可能成为下一个加密百万富翁，而不是成为又一个失败的受害者。在多少人失去多少钱之前，我们才会停止相信这个行业的谎言，这个行业曾经依靠人们的信任和对财务奇迹的希望，却一次又一次地让他们失望？

Bankman-Fried 要进监狱，但情况没有改变。

#### Sam Bankman-Fried的兴衰

展示</summary>

2019年5月

创办FTX

Sam Bankman-Fried与前谷歌员工Gary Wang创办加密交易所FTX。Bankman-Fried年仅20多岁，在离开交易公司Jane Street Capital后不久。

2019年11月

FTX获得10亿美元投资

加密货币交易所币安的前CEO赵长鹏以10亿美元的价格投资FTX，换取公司20%的股份。

2021年9月

Bankman-Fried和FTX搬到巴哈马

Bankman-Fried于2021年将公司总部从香港搬迁至巴哈马，购买了数百万美元的房地产并在该国设立了居所。

2021年8月

合作伙伴关系和重大投资

Bankman-Fried与FTX达成公司首个重大合作伙伴关系的协议，与梅赛德斯的一级方程式车队签订赞助协议。与此同时，FTX通过一轮融资筹集了数亿美元，并估值达到250亿美元。

2022年1月

FTX估值飙升至320亿美元

Bankman-Fried在C轮融资中筹集了4亿美元，公司估值膨胀至320亿美元。

2022年2月

FTX推出超级碗广告

Bankman-Fried投入数百万美元在超级碗广告中邀请Larry David，FTX试图走向主流市场。

2022年4月

加密巴哈马将SBF推上聚光灯下

FTX举办了名为Crypto Bahamas的活动，邀请了Gisele Bündchen和Tom Brady等名人。Bankman-Fried与前世界领导人比尔·克林顿和托尼·布莱尔一起登台。

2022年10月

Sam Bankman-Fried的竞选捐赠者身份

Bankman-Fried投入约4000万美元支持各种政治竞选活动，成为华盛顿新兴的权力角色。

2022年11月

FTX财务状况动荡的报告浮出水面

加密货币新闻网站CoinDesk发表文章显示，FTX的财务状况可能比之前所知更为岌岌可危，引发了媒体的大量报道和投资者的担忧。客户对存款的挤兑导致交易所危机四伏。

2022年11月

Bankman-Fried辞职，FTX破产

Bankman-Fried在一周混乱和资金失踪后辞职。

2022年11月

Bankman-Fried被逮捕

当局在巴哈马逮捕Bankman-Fried，并将其引渡至美国。他被控阴谋和欺诈罪。

2023年8月

取保候审取消，被送进监狱

在其前FTX执行官Caroline Ellison的日记泄露给《纽约时报》后，Bankman-Fried的保释被撤销。他被送往监狱等待审判。

2023年10月

开庭审理开始

曼哈顿联邦法院刑事审判开始，前亲密盟友和FTX高管作证反对Bankman-Fried。

2023年11月

Bankman-Fried被判有罪

陪审团认定 Bankman-Fried 在所有七项串谋和电信诈骗指控中有罪。他面临数十年的监禁可能。

2024年3月28日

Bankman-Fried 被判处25年徒刑。

在曼哈顿联邦法院，Sam Bankman-Fried 被判处25年徒刑，并被命令没收110亿美元的资产。
