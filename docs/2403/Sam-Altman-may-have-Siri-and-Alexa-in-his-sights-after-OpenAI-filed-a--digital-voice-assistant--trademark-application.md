# 萨姆·奥尔特曼可能在OpenAI提交“数字语音助手”商标申请之后，将注意力转向Siri和Alexa。

> 来源：[https://www.businessinsider.nl/sam-altman-may-have-siri-and-alexa-in-his-sights-after-openai-filed-a-digital-voice-assistant-trademark-application/](https://www.businessinsider.nl/sam-altman-may-have-siri-and-alexa-in-his-sights-after-openai-filed-a-digital-voice-assistant-trademark-application/)

+   OpenAI提交了一个“语音引擎”的商标申请，并建立“数字语音助手”。

+   该申请是在萨姆·奥尔特曼暗示即将发布的GPT-5之前发布的一天。

+   OpenAI还在十月份为未来的模型如GPT-6和GPT-7提交了商标申请。

[萨姆·奥尔特曼](https://www.businessinsider.com/sam-altman-chatgpt-openai-ceo-career-net-worth-ycombinator-prepper-2023-1)可能正努力挑战苹果的Siri和亚马逊的Alex语音助手。

OpenAI已经[提交](https://tsdr.uspto.gov/#caseNumber=98456635&caseSearchType=US_APPLICATION&caseType=DEFAULT&searchType=statusSearch)了建立“数字语音助手”和“语音引擎”的商标申请，这表明它可能即将发布新产品。

该申请是在周二向美国专利商标局提交的，即在萨姆·奥尔特曼接受采访并宣布OpenAI在其即将发布的GPT-5之前有“许多其他重要事情要发布”的一天之后。

这些功能可能不会变成现实，因为公司经常为从未付诸实施的想法提交商标申请。然而，根据[之前的Business Insider报道](https://www.businessinsider.com/openai-launch-better-gpt-5-chatbot-2024-3)。

在[采访](https://www.youtube.com/watch?v=jvqFAi7vkBc)中，与播客主持人Lex Fridman的采访中，奥尔特曼说：“今年我们将发布一个惊人的新模型。我不知道我们会叫什么名字。在接下来的几个月里，我们将发布许多不同的东西，我认为它们会非常酷。”

他补充道：“在我们讨论是否要推出一个类似于GPT-5的模型、名字是不是这个、比预期的GPT-5稍微差一点或者好一点之前，我认为我们有很多其他重要的事情要先发布。”

公司目前尚未向用户提供数字语音助手，尽管它有一个将语音转换为文本的API称为TTS，根据其[网站](https://platform.openai.com/docs/models/tts)。它还有一个称为Whisper的通用语音识别模型。

OpenAI还在十月份为其未来的模型，包括[GPT-6](https://tsdr.uspto.gov/#caseNumber=98233550&caseSearchType=US_APPLICATION&caseType=DEFAULT&searchType=statusSearch)和[GPT-7](https://tsdr.uspto.gov/#caseNumber=98233552&caseSearchType=US_APPLICATION&caseType=DEFAULT&searchType=statusSearch)提交了商标申请。

GPT-6的申请包括模拟对话，分享数据集以进行机器学习、预测分析，以及"分析能够学习分析、分类并对数据作出响应的算法"。

GPT-7商标申请涵盖使用人工智能进行音乐生成、将文本和数据文件转换为软件代码，以及创建和生成软件代码的软件。这两个申请仍在审查中。

尽管如此，专利局可能不会授予这些商标，因为在二月份，它[拒绝了](https://tsdr.uspto.gov/documentviewer?caseId=sn97733259&docId=FREF20240206125856&linkId=1#docIndex=0&page=1)OpenAI尝试注册“GPT”的商标。该机构表示，这是一个“广泛使用的首字母缩写”，仅仅是描述性的，因为它代表“生成预训练转换器”。

"语音工程师"商标申请正在接受审查，旨在涵盖以下10个领域的软件：

+   构建数字语音助手。

+   语音和语音识别，处理语音命令，以及文本与语音之间的转换。

+   处理语音命令，以及文本与语音之间的转换。

+   自动语音和语音识别与生成。

+   基于自然语言提示、文本、语音、视觉提示、图像和/或视频生成语音和音频输出。

+   对用户提示生成音频和/或语音。

+   基于机器学习的自然语言和语音处理。

+   多语言语音识别、翻译和转录。

+   使用人工智能进行自动文本转语音和文本转音频转换。

+   语音服务交付的开发。

OpenAI并未立即回应《商业内幕》的置评请求。
