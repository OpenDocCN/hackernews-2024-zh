# 比两极分化更强大的唯一力量？房价上涨 - 大西洋月刊

> 来源：[https://www.theatlantic.com/ideas/archive/2024/03/yimby-housing-bipartisan-zoning/677704/](https://www.theatlantic.com/ideas/archive/2024/03/yimby-housing-bipartisan-zoning/677704/)

*2024年3月11日下午1:25更新*

上周，在乔·拜登发表国情咨文前的几天，有传言称他将大力推动扩大国家住房供应的可能性，这一可能性让推动全国各地社区增加建设的邻里之间活跃组织（YIMBY）活动人士感到担忧。美国的政治两极分化已经变得如此严重，以至于赢得总统支持可能适得其反。演讲当天早晨，白宫公布了[一些温和的提议](https://www.whitehouse.gov/briefing-room/statements-releases/2024/03/07/fact-sheet-president-biden-announces-plan-to-lower-housing-costs-for-working-families/#:~:text=The%20President%20is%20proposing%20that,assist%20nearly%20380%2C0000%20households.)，但幸运的是，拜登本人对此只作了简短而温和的评论，而这些评论不可避免地被更具争议性的问题所掩盖。

“许多YIMBY成员松了一口气，因为拜登没有把问题极端化，”加州YIMBY倡导团体的CEO布莱恩·汉伦告诉我，引用了关于政治科学研究的[研究](https://onlinelibrary.wiley.com/doi/abs/10.1111/j.1540-5907.2011.00541.x)表明，如果总统在某个问题上公开表态强硬，来自另一党的人们更不可能支持这一立场。

包括移民改革、疫苗接种、乌克兰资助以及最近的体外受精等许多事业，尽管在意识形态上有支持者，但仍然陷入了红蓝两党的两极分化之中，阻碍了可能解决美国重大问题的两党立法。在国会，立法者们通常跨越党派界限合作，但有一个条件：所讨论的问题不能太具政治敏感性，也就是说，选民们*真正*关心的问题。这就是为什么国会可以通过关于补贴半导体的法律，却无法通过解决移民问题的法律。

对于希望实现他们优先事项的团体来说，问题在于如何在不陷入两极分化的漩涡中获得足够的关注。在这方面，没有任何一个运动比YIMBY更加摇摆不定，他们通常支持放宽住房发展限制。

他们的运动诞生于旧金山，一个在那里最后一位共和党选举官员于2014年[离任](http://sfstandard.com/2023/11/27/how-san-francisco-republican-party-died/)，曾在湾区快速交通董事会任职。但随着住房经济可负担性危机蔓延到全国各地的城市和州，活动家、律师、选举官员和政策专家的网络也随之扩展。这一运动不仅在旧金山和俄勒冈州波特兰等自由派聚集地扎根，也在盐湖城和蒙大拿州怀特菲什等地扎根。

随着时间的推移，YIMBY策略已经从逐个游说市议会和城镇规划委员会转变为推动州长和立法机构放宽整体州范围内的区域规划规则和其他住房供应限制。亨利·霍纳罗夫，一个支持住房团体的松散全国联盟的主任告诉我，没有一个州能够在没有两党支持的情况下通过支持住房的立法。即使在坚定的蓝色加利福尼亚州，民主党的YIMBY也需要共和党的支持者。有益的是，使住房更加可负担的举措吸引了注重公平的左派，而市场私有化的解除管制则吸引了热衷于财产权的保守派。

许多支持住房的倡导者最大的担忧是他们的问题会被卷入总统选举的交火之中。这种情况以前曾发生过。

2020年，当时的总统唐纳德·特朗普试图在选民中激活NIMBY主义。在2020年8月，他与其住房和城市发展部长联合在*《华尔街日报》* [评论](https://www.wsj.com/articles/well-protect-americas-suburbs-11597608133) 中警告说，“左派试图通过推动‘高密度住房’来剥夺你们的‘美国梦’”。他在推特上进一步升级了这些攻击，[在竞选活动中](https://www.politico.com/news/2020/10/01/how-white-grievance-politics-informs-trumps-campaign-playbook-424590)，甚至在辩论舞台上（[YouTube链接](https://www.youtube.com/watch?v=Vi-c594fLzI)）也是如此。然而，共和党人和民主党人并没有将自己归为NIMBY和YIMBY阵营，至少部分原因是新闻媒体和选民都过于关注COVID-19，以至于住房政策鲜有关注。

[杰西卡·特朗斯廷：特朗普误解了郊区选民的本质](https://www.theatlantic.com/ideas/archive/2020/09/nimby-homeowners-dont-need-trumps-help/616195/)

过去四年来，随着经济可负担性危机加剧，YIMBY们已经占据了优势地位。在保守的蒙大拿州，反加利福尼亚州的信息促使立法者通过了支持发展的法案；在华盛顿州，出于可负担性和种族公平的名义通过了雄心勃勃的提案。但成员们在两边都面临着放弃的压力。他们能坚持多久呢？

YIMBY运动之所以保持跨党派性质，是因为它是分散的。但这个团体定期聚集在一个被有趣地称为“YIMBYtown”的全国性会议上，这是一个罕见的地方，您可能会在同一个房间里找到社会主义者、中间派经济学家和支持特朗普的当选官员，共同追求同一目标。但当我参加今年在德克萨斯州奥斯汀举行的活动时，这个联盟中出现了一些裂痕。

最爆炸性的时刻发生在关于德克萨斯州住房经济性的小组讨论上。台上是来自进步非营利组织德克萨斯苹果籽的官员布伦南·格里芬。他身旁是保守派人士：一侧是曼哈顿研究所保守派导演兼主张清除无家可归者营地的格洛克法官；另一侧是共和党州代表科迪·瓦苏特和保守派德克萨斯公共政策基金会诉讼总监查斯·韦尔登。

会议一开始就遭到了抗议者的截击：“为什么我们要相信这里的任何人关心奥斯汀的经济适用房？”名叫辛西娅·瓦斯奎斯的抗议者问道。她和其他几个人开始在活动室里走动，发放传单，指责主持人和一位小组成员“刑事化无家可归者”，并指责四位本地进步市议会成员与保守派有联系。他们遭遇了一些嘲笑，如“哦，天啊”和“又来了”的嘟囔声在房间里传开。

[Reihan Salam: 为什么YIMBY的正义回击会事与愿违](https://www.theatlantic.com/ideas/archive/2023/07/yimby-california-social-justice-kahlenberg/674714/https:/www.theatlantic.com/ideas/archive/2023/07/yimby-california-social-justice-kahlenberg/674714/)

当我与抗议者交谈时，他们提到的担忧包括，如果东奥斯汀的开发压力增加，低收入房主的财产税会上涨，以及当地YIMBY支持的最近提升区划提案中缺乏负担得起的住房法规。但他们显然也对会议仅仅宣称要致力于住房经济性的事实感到愤怒。

抗议者们自己肯定不会认为自己是YIMBY的一员，但一些会议参与者显然受到了他们的影响。就像2020年特朗普从右翼威胁这个联盟一样，奥斯汀的抗议活动指向了可能更大的威胁：联盟可能会从左翼开始瓦解。在抗议之后的问答环节中，一位观众要求小组成员讨论许多社区低收入居民表达的被迫搬离的担忧。“那些尖叫和嚷嚷的人，他们有真正的恐惧，”北卡罗来纳州Wake County唤醒行动组织的执行董事丹泽尔·伯恩赛德说，引起了大部分人群的掌声。

会后，特区城市主义团体“Greater Greater Washington”的区域政策主任丹·里德（Dan Reed）发表了一篇简短的[博客文章](https://ggwash.org/view/92602/do-something-the-week-of-february-26-2024)，表达了对蒙大拿州共和党州长格雷格·吉安福特（Greg Gianforte）扮演的突出角色的疑虑，并敦促同行的住房倡导者们做得更好。尽管YIMBY在蒙大拿州赢得了胜利，里德指出，州议员们却把一位跨性别同事赶出了会场。里德认为，突出像吉安福特这样的保守派人士，“强化了”运动只是“为了白人自由主义兄弟”的感知。

然而，当我在中断的小组讨论后与伯恩赛德交谈时，他对与保守派在住房政策上合作并没有任何疑虑。“我在南方，红色州，”他说。“这可以是一个两党制的问题。”对于一个曾经在保守州努力推动进步政策变革的前牧师来说，妥协是一种生活方式。“我一直在寻找欢迎邻居的共同的、神学上的、哲学上的立场，”伯恩赛德解释道。“即使那个邻居看起来不像我。即使那个邻居的想法不同于我……我有我的花坛，你有你的花坛，但为了我们变成一块完整的地毯，我们必须找到一些在中间编织的地方。”

[杰鲁撒冷·戴姆萨斯（Jerusalem Demsas）：科罗拉多解决住房危机的巧妙想法](https://www.theatlantic.com/magazine/archive/2023/07/local-government-power-nimby-denver/674164/)

一个有助于团结意识形态多样的亲住房运动的因素是，当住房价格飙升时，社区的每个人都会遭受损失。查看Zillow是一种无党派活动。使得这个联盟保持团结的另一个因素是，实际上它几乎不算是一个联盟。YIMBY们在各自州和城市的背景下工作。没有国家级团体规定他们支持的法案或发送的信息。

北卡罗来纳州的伯恩赛德（Burnside）所在的团体是一个日益壮大的全国联合会的一部分，名为“Welcoming Neighbors Network”。这个团体有意保持低调，同时根据需要将当地独立团体与研究、组织和政策援助联系起来。网络主管霍诺罗夫（Honorof）告诉我，无论成员们与右翼还是左翼的盟友一起在住房问题上工作，“都非常明确地理解我们不会谈论*任何*其他事情”。

这种方法之所以可行，是因为两件事情可以同时成立：住房很重要，但住房并不是一切。

克里斯蒂安·索洛里奥是亚利桑那州的一位34岁的进步派前州代表，白天是建筑师，业余时间担任两个亲住房组织的董事会成员。当我们在奥斯汀坐在一起时，他正在等待关于[亚利桑那州初始住房法案](https://www.azleg.gov/legtext/56leg/2R/bills/HB2570P.htm)的州参议院投票的更新情况，该法案将阻止人口超过70,000的城市要求大型土地面积和大型建筑面积，并防止施加其他使房屋更加昂贵的规定。

这项法案得到了两党支持和两党反对。在众议院通过时，有15名民主党人和18名共和党人投赞成票，而有13名民主党人和13名共和党人投反对票。参议院的情况类似。索洛里奥回忆说，该法案在众议院通过的同时，民主党州长凯蒂·霍布斯已经承诺否决了一项[激烈的党派边境法案](https://azmirror.com/2024/02/28/skipping-public-hearings-republicans-rammed-through-a-bill-to-enforce-federal-immigration-law/)。民主党议员们是如何从辩论最具分裂性的边境州问题，转向横跨党派，与他们的对手一同投票支持由共和党主导的支持住房的法案的？因为这就是政治运作的方式。

这并不意味着这项法案会成为法律。[霍布斯告诉记者](https://www.thecentersquare.com/arizona/article_a2942252-dca0-11ee-aaa2-537fdce28690.html)，她仍在考虑是否签署亚利桑那州初始住房法案，指出她更喜欢得到地方政府支持的立法，而这项法案却遭到了地方政府游说团体的反对。无论如何，政治代价很低。在像亚利桑那这样分裂的州份，上一届州长选举是在霍布斯和右翼激进分子卡里·莱克之间进行的，没有人会因为区划政策而改变选票。

即使是坚定的YIMBY支持者也不会。“我是民主党人，我投票选了州长，”索洛里奥告诉我。“如果她最终成为我们州最大的NIMBY，我仍然会支持她的连任，因为区划，尽管我是州内最大的区划改革倡导者之一……对我来说，还不足以改变我的选票。”
