# CS 6120：高级编译器：自主学习在线课程

CS 6120是由[Adrian Sampson](https://www.cs.cornell.edu/~asampson/)提供的博士级别的[Cornell CS](https://www.cs.cornell.edu/)课程，关于编程语言实现。它涵盖了通用编译器主题，如中间表示、数据流和“经典”优化，以及更多研究性的主题，如并行化、即时编译和垃圾回收。工作内容包括阅读论文和开源编码任务，使用[LLVM](https://llvm.org/)和[专为本课程设计的教育IR](https://capra.cs.cornell.edu/bril/)。

这一页列出了在你的想象力大学修读课程的课程表，获得四个想象力学分（不计分）。课程设置了线性时间表，穿插着需要阅读的论文。每个课程包括视频和书面笔记，有些还有*实现任务*需要你完成。任务都是开放式的，程度不一，旨在通过将抽象概念转化为实际代码来巩固你对这些概念的理解。顺序建议交替观看视频和阅读论文。

与“真正”的CS 6120不同之处在于，你可以忽略任务截止日期，无法参与我们在Zulip上的讨论串。真正的6120还包括学期末的课程项目 —— 在自主学习版本中，你的学期末任务是通过编译器的魔法改变世界。

由于教学录像制作者是一个新手，请原谅早期课程中的制作价值。CS 6120是[开源项目，托管在GitHub上](https://github.com/sampsyo/cs6120)，如果发现问题，请提交错误报告。
