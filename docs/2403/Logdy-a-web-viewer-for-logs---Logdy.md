# Logdy 日志的网页查看器 | Logdy

> 来源：[https://logdy.dev/](https://logdy.dev/)

通过 web 浏览器 UI 和低代码

超级增强终端日志

节省搜索和浏览日志时间的 90%。

```
$ curl https://logdy.dev/install.sh | sh
# Install, then pipe logs directly
$ tail -f file.log | logdy
$ [Logdy] Enter Web UI at http://localhost:8080 
```
