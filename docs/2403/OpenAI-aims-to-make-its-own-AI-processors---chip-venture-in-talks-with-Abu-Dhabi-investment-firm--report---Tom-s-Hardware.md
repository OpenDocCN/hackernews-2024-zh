# OpenAI旨在打造自己的AI处理器 — 与阿布扎比投资公司谈判进行芯片合资：报告 | Tom's Hardware

> 来源：[https://www.tomshardware.com/tech-industry/artificial-intelligence/openai-aims-to-make-its-own-ai-processors-chip-venture-in-talks-with-abu-dhabi-investment-firm-report](https://www.tomshardware.com/tech-industry/artificial-intelligence/openai-aims-to-make-its-own-ai-processors-chip-venture-in-talks-with-abu-dhabi-investment-firm-report)

阿布扎比新成立的投资公司MGX正在谈判，以资助Sam Altman的[雄心勃勃计划](https://www.tomshardware.com/tech-industry/artificial-intelligence/openais-sam-altman-raises-billions-to-build-chip-empire-report)，为AI处理器建造过剩的半导体制造能力，据[《金融时报》](https://www.ft.com/content/d018067f-20e7-49eb-83dc-ebb8b1aad1a5?s=31)报道。如果谈判成功，此举将符合阿联酋成为全球人工智能行业关键参与者的战略。

[OpenAI](https://www.tomshardware.com/tech-industry/artificial-intelligence/openais-sam-altman-discusses-chip-collab-with-samsung-and-sk-hynix-report)推动开发自己的半导体是为了减少对Nvidia等AI GPU（如[H100](https://www.tomshardware.com/news/nvidia-hopper-h100-80gb-price-revealed)）的依赖。与此同时，除了开发AI处理器外，[Sam Altman](https://www.tomshardware.com/news/openai-ousts-ceo-sam-altmanhttps://www.tomshardware.com/tech-industry/artificial-intelligence/jim-keller-responds-to-sam-altmans-plan-to-raise-dollar7-billion-to-make-ai-chips)还希望确保它们稳定供应给[OpenAI](https://www.tomshardware.com/tech-industry/artificial-intelligence/openai-and-microsoft-being-sued-by-the-new-york-times-over-copilot-and-chatgpt-copyright-infringement)及潜在的其他公司。在近年来目睹了两次主要半导体供应危机（首先是汽车和个人电脑行业，然后是人工智能行业），他似乎认为传统的芯片合同制造商没有足够的制造能力来满足全球对AI芯片的需求。因此，他据称希望建立更多的晶圆厂。

OpenAI 的首席执行官，Sam Altman，据报道估计新建半导体制造设施及其支持基础设施的成本可能高达[$7 trillion](https://www.tomshardware.com/tech-industry/artificial-intelligence/openai-ceo-sam-altman-seeks-dollar5-to-dollar7-trillion-to-build-a-network-of-fabs-for-ai-chips)。虽然尚不清楚芯片项目是OpenAI的项目还是Sam Altman的项目（很可能是后者），但首席执行官一直在全球各地[与芯片制造商进行交流](https://www.tomshardware.com/tech-industry/artificial-intelligence/openais-sam-altman-discusses-chip-collab-with-samsung-and-sk-hynix-report)，并寻求国家支持，包括与新加坡淡马锡的讨论，因为传统的风险投资者不太可能投资如此庞大的金额。

由阿联酋发起的以人工智能为重点的基金MGX由国家安全顾问Sheikh Tahnoon bin Zayed al-Nahyan主持。该基金旨在利用阿布扎比的财政资源和政治支持，使其成为人工智能发展的中心枢纽。阿联酋成为全球人工智能强国的雄心在于吸引人工智能行业的领军人物，包括Elon Musk，他已表示有意与这个海湾国家合作进行其人工智能项目xAI。

Sheikh Tahnoon 不仅仅是MGX的主席，他还领导着来自阿联酋的以人工智能为重点的控股公司G42，由阿布扎比主权投资基金穆巴达拉支持。G42已与人工智能行业的主要参与者建立了合作伙伴关系，包括OpenAI，[Microsoft](https://www.tomshardware.com/tag/microsoft)，以及[Cerebras](https://www.tomshardware.com/tech-industry/artificial-intelligence/cerebras-launches-900000-core-125-petaflops-wafer-scale-processor-for-ai-theoretically-equivalent-to-about-62-nvidia-h100-gpus)。与G42和穆巴达拉的合作成立了MGX，这是阿布扎比在人工智能领域的又一战略举措。

阿联酋对[人工智能](https://www.tomshardware.com/tech-industry/artificial-intelligence/cerebras-launches-900000-core-125-petaflops-wafer-scale-processor-for-ai-theoretically-equivalent-to-about-62-nvidia-h100-gpus)的积极态度体现在其早期采用的人工智能倡议，例如2017年任命世界首位人工智能部长和2019年开设的人工智能研究生院。这些努力显示了该国在推进人工智能行业地位方面的承诺。

然而，阿联酋的人工智能野心出现在复杂的地缘政治背景下，例如美中贸易紧张局势，远超出贸易范围，涉及国家安全。面对来自美国的担忧，G42不得不重新评估其与中国公司（包括[Huawei](https://www.tomshardware.com/pc-components/cpus/intel-staves-off-amd-and-china-critics-to-keep-exclusive-export-license-to-huawei-intel-sells-90-of-cpus-used-in-the-companys-laptops)）的合作伙伴关系。
