# 推出Zapier Central：与AI机器人紧密合作

Zapier，以[自动化工作流程](https://productivity.directory/category/workflow-automation)而闻名于小型企业，已收购了Vowel的团队，他们是一款AI驱动视频会议工具的创造者。通过这次收购，Vowel的CEO安德鲁·伯曼将担任Zapier的AI主管，专注于提升其AI自动化功能。

# 什么是Zapier Central？

[Zapier Central](https://zapier.com/central)将AI智能与每天受到超过2.2百万家企业信任的平台相结合，创造出更像团队成员而非简单软件的机器人。

通过Central，在不编写代码的情况下自动化后台任务，通过教授机器人特定行为来实现。轻松连接超过6,000个应用程序，让Central处理动作设置。获取来自电子表格或Google文档等各平台的实时数据，并直接与机器人合作完成跨连接应用程序的任务。发送电子邮件或更新电子表格等命令成为向您的机器人发出的简单请求。

[Zapier](https://productivity.directory/zapier)通过收购以其AI视频会议技术著称的公司Vowel，扩展了其AI能力。这包括引入Vowel的CEO安德鲁·伯曼，以加深Zapier在AI方面的专业知识。这一战略举措旨在增强自动化服务，并加速Zapier Central的开发，反映了他们对AI在未来工作场所中关键角色的信念，使客户能够更高效地进行自动化。
