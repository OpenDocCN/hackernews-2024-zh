# 欧洲委员会使用Microsoft 365侵犯了欧盟机构和机构的数据保护法律 | 欧洲数据保护监督官员

> 来源：[https://www.edps.europa.eu/press-publications/press-news/press-releases/2024/european-commissions-use-microsoft-365-infringes-data-protection-law-eu-institutions-and-bodies_en](https://www.edps.europa.eu/press-publications/press-news/press-releases/2024/european-commissions-use-microsoft-365-infringes-data-protection-law-eu-institutions-and-bodies_en)

**根据其调查结果，欧洲数据保护监督官员（EDPS）发现欧洲委员会（委员会）在使用Microsoft 365时侵犯了多项关键数据保护规定。在其决定中，EDPS对委员会施加了纠正措施**。

EDPS发现委员会**违反了《2018/1725/EU条例》的多项规定**，即欧盟机构、机构、办事处和机构（EUIs）的数据保护法律，包括关于个人数据转移到欧盟/欧洲经济区（EEA）以外地区的规定。特别是，委员会未能提供适当的保障措施，以确保将个人数据转移到欧盟/欧洲经济区之外的地区时，能够获得与在欧盟/欧洲经济区内所保证的基本等同的保护水平。此外，在其与Microsoft的合同中，**委员会没有充分说明在使用Microsoft 365时将收集哪些类型的个人数据，以及为哪些明确和特定的目的收集**。**委员会作为数据控制者的侵犯行为还涉及由其代表进行的数据处理，包括个人数据的转移**。

**EDPS Wojciech Wiewiórowski表示：** *“确保个人数据的任何处理活动，无论在欧盟/欧洲经济区内外进行，包括在云服务环境中进行时，都必须伴随着强大的数据保护措施和保障措施，是欧盟机构、机构、办事处和机构（EUIs）的责任。这是为了确保根据《2018/1725/EU条例》的要求，个人信息得到保护，每当他们的数据被EUI或代表EUI处理时。”*

因此，EDPS决定于**2024年12月9日**命令委员会**暂停由其使用Microsoft 365引起的所有数据流**，包括流向Microsoft及其位于未受欧盟/欧洲经济区（EEA）充分保障决定的国家的关联公司和子处理者。EDPS还决定命令委员会**使由其使用Microsoft 365引起的处理操作符合《2018/1725/EU条例》**。**委员会必须在2024年12月9日之前展示符合这两项命令的情况**。

欧洲数据保护监督员（EDPS）认为，**它强制执行的纠正措施（[详见附件获取详细摘录](https://www.edps.europa.eu/system/files/2024-03/EDPS-2024-05-European-Commission_s-use-of-M365-infringes-data-protection-rules-for-EU-institutions-and-bodies_EN.pdf)）在发现的侵权行为的严重性和持续时间的情况下是适当的、必要的**和**成比例的**。

发现的许多侵权行为涉及委员会或其代表使用Microsoft 365进行的所有处理操作，并影响大量个人。

EDPS还考虑到需要**不损害委员会履行公共利益任务或行使委员会赋予的官方权力的能力，以及需要允许委员会适当时间来实施预期的相关数据流暂停，并使数据处理符合条例（EU）2018/1725的规定。**

EDPS在其2024年3月8日的决定中强加的措施不影响EDPS可能采取的任何其他或进一步行动。

**[EDPS在其决定中发现的侵权和强制执行的纠正措施的详细信息可见附件](https://www.edps.europa.eu/system/files/2024-03/EDPS-2024-05-European-Commission_s-use-of-M365-infringes-data-protection-rules-for-EU-institutions-and-bodies_EN.pdf)。**

背景信息

[欧盟机构、机构、办事处和机构的数据保护规则](https://european-union.europa.eu/institutions-law-budget/institutions-and-bodies/search-all-eu-institutions-and-bodies_en)，以及欧洲数据保护监督员（EDPS）的职责，详见[**条例（EU）2018/1725**](https://eur-lex.europa.eu/legal-content/EN/TXT/HTML/?uri=CELEX:32018R1725)。

**Wojciech Wiewiórowski**（EDPS）由欧洲议会和理事会共同决定任命，任期五年，自2019年12月6日起。

**关于EDPS对欧盟委员会使用Microsoft 365的调查：** 这项调查是在[*Schrems II*判决](https://curia.europa.eu/juris/liste.jsf?language=en&jur=C%252CT%252CF&num=C-311/18)后的[2021年5月开展的](https://edps.europa.eu/press-publications/press-news/press-releases/2021/edps-opens-two-investigations-following-schrems_en)，目的是验证委员会是否遵守EDPS先前就欧盟机构和机构使用Microsoft产品和服务发布的[建议](https://edps.europa.eu/data-protection/our-work/publications/investigations/outcome-own-initiative-investigation-eu_en)。此调查是EDPS在2022年协调执行行动中的一部分。欲了解更多信息，请阅读[EDPB对2022年协调执行行动使用基于云的服务的报告](https://edpb.europa.eu/our-work-tools/our-documents/report/coordinated-enforcement-action-use-cloud-based-services-public_en)。

**关于** [**EDPS调查**](https://www.edps.europa.eu/data-protection/our-work/our-work-by-type/investigations_en) **：** 欲了解更多关于EDPS调查流程的信息，请访问[EDPS调查政策](https://edps.europa.eu/system/files/2023-01/2023-01-30-edps_investigation-public_policy_en.pdf)，[EDPS调查资料](https://edps.europa.eu/system/files/2023-01/30-01-23-edps-investigation_infographic_factsheet_en.pdf) 在EDPS网站上。
