# '如果发生任何事情，这不是自杀'：波音告密者去世前的预言

> 来源：[https://abcnews4.com/news/local/if-anything-happens-its-not-suicide-boeing-whistleblowers-prediction-before-death-south-carolina-abc-news-4-2024](https://abcnews4.com/news/local/if-anything-happens-its-not-suicide-boeing-whistleblowers-prediction-before-death-south-carolina-abc-news-4-2024)

南卡罗来纳州查尔斯顿县（WCIV）— 约翰·巴内特的一位密友说，他预测他可能会死去，可能会有一个关于他自杀的故事浮出水面。

但当时，他告诉她不要相信。

“我知道他不会自杀，”是Barnett的朋友Jennifer说。“不可能。”

Jennifer说他们谈到了这种情况正在发生。然而，现在，他的话看起来像是他直接告诉她不要相信的预兆。

“我认识约翰，因为他妈妈和我的妈妈是好朋友，”Jennifer说。“多年来，我们在一起过生日、庆祝和其他聚会。我们一直聚在一起聊天。”

当Jennifer有一天需要帮助时，Barnett来看她。他们谈到了他即将在查尔斯顿的传票。Jennifer知道Barnett对波音提起了一个极其具有破坏性的投诉。他说，这家航空巨头在他揭露其不安全做法后对他进行了报复。

他担任质量经理超过30年。最近退休并搬回路易斯安那州照顾他的母亲。

“他并不担心安全问题，因为我问过他，”Jennifer说。“我说，‘你不害怕吗？’他说，‘不，我不害怕，但如果我出了什么事，不是自杀。’”

Jennifer补充道：“我知道他不会自杀。不可能。他太热爱生活了。他太爱他的家人了。他太爱他的兄弟们了，不会让他们经历现在的这些事情。”

Jennifer说她认为有人“不喜欢他说的话”，想要“让他闭嘴”，而不让这事情有回溯性。

“这就是为什么他们要让它看起来像是自杀，”Jennifer说。

上次詹妮弗见到巴尼特是在她父亲二月底的葬礼上。他是其中一个抬棺者。有时家人和朋友会用他的中间名字——米奇来称呼他。

“我认为每个人都不敢相信，都不敢相信这是真的，”詹妮弗说。“不管他们说什么，我知道米奇没有做过那种事。”

巴尼特去世并不意味着案件不会继续进行。

他的律师表示他们仍准备在六月庭审。

新闻4频道在巴尼特去世后联系了波音公司。他们提供了以下声明：

“我们对巴尼特先生的去世感到悲伤，我们的思念与他的家人和朋友同在。”
