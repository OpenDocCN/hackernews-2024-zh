# 直接提交正式在12个试点州开放，随着初期评价的积极反馈，符合条件的纳税人可以在线免费向美国国内税务局提交税款 | 美国国内税务局

> 来源：[https://www.irs.gov/newsroom/direct-file-officially-opens-in-12-pilot-states-following-positive-early-reviews-eligible-taxpayers-can-file-online-directly-with-the-irs-for-free](https://www.irs.gov/newsroom/direct-file-officially-opens-in-12-pilot-states-following-positive-early-reviews-eligible-taxpayers-can-file-online-directly-with-the-irs-for-free)

## 鼓励纳税人在4月15日截止日期之前尝试这一创新的新选项

IR-2024-68, March 12, 2024

华盛顿 — 美国国内税务局今天宣布了创新的[直接提交试点项目](https://directfile.irs.gov/ "IRS Direct File")的全面推出，并鼓励12个州的符合条件的纳税人尝试这一新服务，以免费在线直接向美国国内税务局提交他们的纳税申报表。

该试点项目已经在测试模式下进行了数周，允许早期用户免费在线提交他们的2023年联邦税款，并直接向美国国内税务局提交。数千名纳税人成功使用了该系统，并且早期用户对这一新选项的评价非常积极。

在初步直接提交项目成功之后，IRS 正将试点项目从测试阶段推向全面阶段，允许12个州的所有符合条件的纳税人（约1900万人）随时使用系统。纳税人可以登录[直接提交](https://directfile.irs.gov/ "IRS Direct File")开始申报，并在4月份申报期限之前的任何时候完成申报。

“直接提交项目的初步结果显示，纳税人喜欢这一工具的简便和便利性，而随着试点项目的全面推出，更多纳税人将有机会使用这一免费选择，” 美国国内税务局局长丹尼·韦弗尔（Danny Werfel）表示。“随着税务截止日期的临近，扩展直接提交项目将为更多纳税人提供一种与美国国内税务局直接免费联系的途径，并为我们提供更多有价值的信息来评估此试点项目。对于尚未在这12个州提交税款的纳税人，我们鼓励他们访问 IRS.gov 网站，查看直接提交项目是否适合他们。”

直接提交是纳税人提交税款的众多选择之一，包括税务软件和税务专业人士。在试点项目中，IRS 简化了处理较为简单的纳税人直接向机构提交税款的方式。

“政府跨部门专家团队开发并测试了直接提交（Direct File）试点项目，旨在为纳税人提供一种简便、准确且免费的在线报税方式，直接与美国国内税务局（IRS）联系。”Werfel 补充道。“我们的目标是通过直接提交试点项目帮助人们尽可能轻松快速地履行税务义务。我们从一开始就与纳税人合作开发了直接提交项目，并将继续与纳税人沟通，了解他们对未来数字服务的期望。”

Direct File引导纳税人通过国家税法的复杂性，轻松完成申报，并准确计算他们的退款或欠税。Direct File设计易于纳税人使用。在每个步骤中，它都会展示计算背后的工作，让用户可以确信税表是完整和准确的。一旦纳税人开始在[Direct File](https://directfile.irs.gov/ "IRS Direct File")网站上申报，他们可以暂停，并在4月申报截止日期之前随时安全登录完成。

用户还可以通过Direct File的实时聊天功能获得特殊的IRS客户服务代表的支持。像其他电子申报选项一样，Direct File允许纳税人在选择直接存款时通常在21天内获得退款。

“我们从使用它进行测试的成千上万的纳税人那里得到了很好的初步反馈，”Werfel说。“我们听到的许多纳税人使用Direct File在不到30分钟内就提交了税表，并称其为一个简单、免费的税务申报体验。”

### 谁有资格？

满足特定要求的12个试点州的纳税人可以在4月税务截止日期之前使用Direct File。试点州包括：

+   亚利桑那州

+   加利福尼亚州

+   佛罗里达州

+   马萨诸塞州

+   内华达州

+   新罕布什尔州

+   纽约州

+   南达科他州

+   田纳西州

+   德克萨斯州

+   华盛顿州

+   怀俄明州

完成联邦税表后，拥有州所得税的纳税人在亚利桑那州、加利福尼亚州、马萨诸塞州和纽约州将被引导使用州政府提供的工具来完成州税申报。

国税局设计了这个试点，以遵循推出新技术平台的最佳实践，先小规模启动，确保其正常工作，然后再逐步扩展。该试点故意限制在相对简单的税务情况范围内。

Direct File试点是适用于以下类别的纳税人的一个选项：

+   报告从工作中获得的收入，生成W-2表格的收入；包括拥有多个W-2工资工作的纳税人；

+   主张已赚取收入税收抵免、儿童税收抵免和其他受抚养人抵免；

+   主张标准扣除和教育支出及学生贷款利息扣除；

+   在2023年整个日历年度内居住在同一州。

有兴趣的纳税人可以访问 [directfile.irs.gov](https://directfile.irs.gov/ "IRS Direct File")，在那里他们可以确定是否符合资格。使用Direct File需要通过ID.me进行身份验证。一旦身份验证通过，并且他们安全登录到Direct File，他们将直接向国税局提供税务信息，而不是第三方。

对于访问Direct File但不符合试点条件的纳税人，将被引导至[IRS.gov的免费文件](/filing/free-file-do-your-federal-taxes-for-free "免费申报您的税款")。现在进入第22个申报季节，全国各地的纳税人可以访问由IRS Free File信任合作伙伴提供的免费软件产品。通过这种公私合作伙伴关系，税务准备和申报软件提供商向符合条件的纳税人提供其在线产品。今年，八个私营部门Free File合作伙伴为2023年的调整总收入（AGI）为79,000美元或以下的任何纳税人提供在线引导税务软件产品。从IRS Free File开始可免费获得在线产品。

### 2024年申报季节的Direct File试点

IRS推出了2024年申报季节的Direct File试点。通货膨胀削减法案要求IRS研究并评估创建直接电子申报工具的兴趣和可行性，以便纳税人可以准备并申报他们的联邦所得税申报。IRS委托进行了一项独立研究，显示出对这样一个系统的广泛兴趣，IRS在2023年5月的[直接文件报告提交给国会](/newsroom/irs-submits-direct-file-report-to-congress-treasury-department-directs-pilot-to-evaluate-key-issues "IRS提交直接文件报告给国会；财政部指示评估关键问题")中详细说明了这一点。

不久之后，根据财政部的指示，IRS组建了一个由税务专家、技术人员、工程师和来自各个政府部门的策略家组成的团队，以建立直接文件系统。IRS与美国数字服务和总务管理局的技术办公室18F密切合作，共同建立和测试了Direct File。

初步测试始于2024年2月初，涉及少数联邦和州政府雇员，随后是较短的开放时间窗口，供更多纳税人开始申报。3月4日，IRS宣布Direct File将进入最终测试阶段，其中包括一定数量的可用空间，并每天增加更多空间。Direct File现在向所有符合条件的纳税人在试点州开放，让他们决定是否选择在线免费向IRS申报2023年联邦税款。
