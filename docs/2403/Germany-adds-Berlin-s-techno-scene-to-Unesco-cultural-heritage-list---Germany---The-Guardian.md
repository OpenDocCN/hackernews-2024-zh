# Germany adds Berlin’s techno scene to Unesco cultural heritage list | Germany | The Guardian

> 来源：[https://www.theguardian.com/world/2024/mar/15/berlins-techno-scene-added-to-unesco-intangible-cultural-heritage-list](https://www.theguardian.com/world/2024/mar/15/berlins-techno-scene-added-to-unesco-intangible-cultural-heritage-list)

德国文化部和联合国教科文组织委员会已将柏林的技派场景列入该国非物质文化遗产名录，以表彰这一场景对城市文化身份的贡献。

柏林的Clubcommission，柏林技派俱乐部和音乐家的网络，[描述](https://twitter.com/clubcommission/status/1767979957173580132)此举为“柏林技派制作人、艺术家、俱乐部经营者和活动组织者再次创造了一个里程碑”。

Lutz Leichsenring，Clubcommission董事会的执行成员，[告诉德国广播公司DW](https://www.dw.com/en/berlin-techno-added-to-unesco-cultural-heritage-list/a-68515354)：“此决定将有助于确保俱乐部文化被认可为值得保护和支持的宝贵部门。”

十多年来，一直有一场运动，旨在将技派文化和音乐添加到德国的名录中，由非营利组织Rave the Planet主导，该组织支持电子音乐文化。

“祝贺所有为柏林技派文化塑造和贡献的文化创作者，”该组织在社交媒体上的声明中说道。“这是整个文化的重要里程碑，我们的喜悦无法用言语表达。”

Rave the Planet在2022年11月提交了将技派列入名录的申请。

更多传统文化活动通常被授予非物质文化遗产地位，如马拉维的Mwinoghe舞蹈或斯洛伐克的风笛文化。然而，最近联合国教科文组织名录中加入牙买加雷鬼和印度庞大的康布梅拉节等非物质文化遗产，促使柏林的技派社区领导者们倡议他们的场景被包括在德国的登记册中，这与联合国名录是分开的。

柏林总部的DJ和音乐制作人Peter Kirn表示，技派是城市的基本组成部分。他在2021年告诉观察家：“在其他城市，人们不会接受那种非常硬、奇怪且充满合成器和极端变形鼓机的音乐。你不能在俱乐部的高峰时段播放那种音乐，更不用说午餐时了。而在这里，播放这种音乐是完全可以接受的。”

“技派已经成为被边缘化人群的庇护所，对柏林的吸引力很自然，因为对比起其他不那么宽容的地方，柏林更加宽容。”

电子音乐场景是德国非物质文化遗产名录的六个新入选项目之一；其他项目包括果酒和登山。在巴伐利亚州举行的一场名为*基希塞奥纳·佩尔赫滕劳夫*的游行，参与者身着毛茸茸的怪物服装，也被列入了名录。

这篇文章的标题和内容已于2024年3月15日进行了修改。德国联合国教科文组织委员会将电子音乐场景列入了国家非物质文化遗产名录，而不是早期版本中所指的全球非物质文化遗产名录。
