# 开放数据描述语言（OpenDDL）

开放数据描述语言（OpenDDL）是一种通用的、人类可读的、强类型的数据语言，用于信息交换。它可用于各种应用程序，从简单的配置文件到以可编辑格式在多个程序之间交换复杂信息。它最初作为 [开放游戏引擎交换格式](http://opengex.org/) 的基础而开发。
