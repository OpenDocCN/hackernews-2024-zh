# 观点：关于 UFO 的实际隐藏真相 | CNN

> 来源：[https://edition.cnn.com/2024/01/26/opinions/ufos-actual-truth-bergen-german/index.html](https://edition.cnn.com/2024/01/26/opinions/ufos-actual-truth-bergen-german/index.html)

主编的说明：*彼得·伯根是 CNN 的国家安全分析员，新美国副总裁，亚利桑那州立大学的实践教授，Audible 节目“[*In the Room*](https://audible.com/intheroom)”的主持人，也可在[*Apple*](https://podcasts.apple.com/us/podcast/in-the-room-with-peter-bergen/id1685702585)和[*Spotify*](https://open.spotify.com/show/7ltl6QQBRzovzdQ7MHo6XB?si=ab3e17fb90d44577)上找到。埃里克·德尔曼是“In the Room”的资深制片人。此评论中表达的观点属于他们自己。在 CNN 查看更多[*观点*](http://www.cnn.com/opinions)。

CNN —

一名前五角大楼官员 — 他说，出于对真相的责任感 — 公开了一个爆炸性指控。面对一群电视摄像机和国会议员，[这名官员声称](https://www.congress.gov/118/meeting/house/116282/documents/HHRG-118-GO06-Transcript-20230726.pdf)美国政府几十年来一直对坠毁的外星飞船保密。

这听起来像是好莱坞电影的情节。但去年，美国人在新闻中看到了这一切发生。前五角大楼官员戴维·格鲁斯曾是空军情报官员。他告诉[国会委员会](https://www.cnn.com/videos/us/2023/07/26/ufo-whistleblower-military-pilots-david-grusch-cprog-orig-ht-js.cnn)，他得知五角大楼进行了长达数十年的“坠毁飞碟回收和逆向工程”计划。格鲁斯还表示，在飞船坠毁现场发现的遗骸是“[非人类生物](https://www.npr.org/2023/07/27/1190390376/ufo-hearing-non-human-%20biologics-uaps)”。

没错。在[C-SPAN](https://www.c-span.org/video/?529499-1/hearing-unidentified-aerial-phenomena)上播出的最疯狂的事情，是坠毁的外星飞船和死去的外星人，这些记录就在[国会记录](https://www.govinfo.gov/content/pkg/CHRG-118hhrg53022/html/CHRG-118hhrg53022.htm)中。有人应该调查一下，对吧？

结果发现，某人已经开始调查了。在2022年，五角大楼任命了一位名叫肖恩·柯克帕特里克的资深科学家和情报官员，建立了一个新的办公室，负责调查美军对 UFO 目击事件。由美国国防部命名为全域异常解决办公室，柯克帕特里克告诉我们他的[团队调查了 UFO 事件并采访了](https://www.defense.gov/News/News-Stories/Article/Article/3249317/dod-office-moving-ahead-in-mission-to-identify-anomalous-%20phenomena/)声称了解与 UFO 相遇情况的美国军人。

柯克帕特里克最近从五角大楼的工作中退休，并且为我们的Audible播客“[在房间里](https://www.audible.com/pd/Episode-38-Is-The-Pentagon-Really-Hiding-Crashed-Alien-Spaceships-Podcast/B0BZ29R9CH?ref_pageloadid=not_applicable&ref=a_pd_In-the_c3_lAsin_0_0&pf_rd_p=625c212d-b95a-47db-8d56-d35a359de6e9&pf_rd_r=YBQ3ZG874HRVM1S1S6TD&pageLoadId=wH034t3pndDW1uCB&creativeId=6c5159d7-bb10-4fb4-929b-ae2b2f5000c7)”。柯克帕特里克及其团队调查了自1940年代罗斯威尔事件以来的每一起美国政府UFO目击事件，并将调查结果整理在一份周五公开发布的五角大楼[报告中](https://media.defense.gov/2024/Mar/08/2003409233/-1/-1/0/DOPSR-CLEARED-508-COMPLIANT-HRRV1-08-MAR-2024-FINAL.PDF)。该报告驳斥了关于地球上外星人访问和任何涉及其掩盖的多重主张。

在他最详尽的媒体采访中，柯克帕特里克详细阐述了一个令人信服的论点，即围绕涉及政府掩盖外星人相关UFO的传闻多年的故事很可能主要受到美国政府内部真正信奉者或与其有密切联系的人的推动。

自从术语“[飞碟](https://airandspace.si.edu/stories/editorial/1947-year-flying-saucer)”首次被创造以来，关于UFO的阴谋论思维大多源于人们偶然看到高度机密的美国飞行器，并且渴望找到答案。当政府没有提供答案时，公众的想象力便会膨胀。

但实际上，柯克帕特里克表示，他的调查发现大多数UFO目击事件都是关于美国政府需要保密的先进技术，其他国家利用来监视美国的飞行器，或者是善意的民用无人机和气球。

“关于所有（UFO报告中的）…约有2至5%…我们会称之为真正的异常现象，”柯克帕特里克说道。他认为，这个小比例的解释很可能是地球上真实存在的。

这就是柯克帕特里克及其团队解释罗斯威尔事件的方式，该事件在UFO传说中占据重要地位。因为在1947年，美国军方新闻发布声称在新墨西哥的罗斯威尔陆军空军基地附近坠毁了一架飞碟。

一天后，军方撤回了这个报道，并称坠毁的物体是一个气象气球。报纸们[发布了最初的飞碟头条新闻，随后是官方的辟谣](https://www.washingtonpost.com/history/2022/07/08/roswell-flying-saucer-ufo/)，对这个案件的兴趣大大降低。直到1980年，一对UFO研究人员出版了一本书，声称从罗斯威尔的残骸中恢复了外星人尸体，并且美国政府掩盖了证据。

Kirkpatrick 表示，他的办公室深入调查了罗斯威尔事件，发现在1940年代末至1950年代初，罗斯威尔空军基地附近发生了许多事情。那时候有一个名为[摩格尔计划](https://www.nytimes.com/1994/09/18/us/wreckage-in-the-desert-was-odd-but-not-alien.html)的间谍项目，发射了一系列奇形怪状的金属气球。这些气球设计用来监视苏联的核试验，是高度机密的。

与此同时，美国军方也正在进行[高空气球试验，携带装有传感器并装在[人形袋中以防止风雨侵袭的试验假人](https://media.defense.gov/2010/Oct/27/2001330219/-1/-1/0/AFD-101027-030.pdf)。附近还发生了至少一起军用飞机坠毁事故，造成11人死亡。

与早期政府调查相呼应，Kirkpatrick 及其团队得出结论，坠毁的摩格尔气球、回收操作以及寻回坠毁试验假人的行动，以及那次真实飞机坠毁的烧焦残骸，很可能合成了一个关于坠毁外星飞船的虚假叙事。

Kirkpatrick 还提出了一个令人信服的案例，认为类似的事情今天也在发生。他说，现在起飞的新技术可以解释从2000年代初开始的大量现代 UFO 目击事件。而这不仅仅是秘密政府技术。许多观察者在看到最新[无人机](https://www.mdpi.com/2504-446X/6/9/260)和[奇形怪状的气球](https://www.cnn.com/2023/02/09/politics/spy-balloon-technology/index.html)时感到困惑。

“更有可能的是什么？” Kirkpatrick 问道。“是有一种最先进的技术正在佛罗里达州商业化，而你却不知道，还是我们有外星人？” 他说。“当你向他们展示，这里有一家佛罗里达州的公司正好生产你描述的东西时，他们的回应是，不，不，不，一定是外星人，而你们在掩盖真相。”

尽管如此，UFO 仍然是一个真正的国家安全问题，主要是因为它们是[飞行危险](https://www.dni.gov/files/ODNI/documents/assessments/Prelimary-Assessment-UAP-20210625.pdf)。正如 Kirkpatrick [所说的](https://www.grc.nasa.gov/www/k-12/airplane/mach.html)，“飞行速度超过音速的军事飞行员，如果撞上带有索具的气球，就会撕掉一只机翼。”

自[2020年](https://www.cnn.com/2020/05/13/politics/navy-ufo-incident-reports/index.html)以来，五角大楼已经标准化、去除污名化并增加了对美国军方 UFO 报告的数量。柯克帕特里克说，这就是去年首次发现美国政府严肃对待 UFO 问题政策的原因，而这个事件还遭到了[广泛嘲笑](https://youtu.be/ZNHGHJItFKw?feature=shared&t=90)。

面对实际证据，为什么政府内外的人们还要推广不受支持的想法，认为[美国政府隐瞒外星入侵者](https://news.gallup.com/poll/266441/americans-skeptical-ufos-say-government-knows.aspx)？

“真正的信徒不仅仅在政府之外，许多人在政府内部也是。”柯克帕特里克告诉我们，包括已故的内华达州民主党参议员哈里·里德在内。里德的长期朋友罗伯特·比格洛，内华达州的亿万富翁，也是比格洛航天公司的所有者，两人都对不明飞行物有浓厚兴趣。柯克帕特里克说：“哈里·里德参议员是一个真正的信徒，认为‘嘿，政府在对国会监督方面隐瞒了这件事。’”

2007年，[里德参议员获得资金](https://www.politico.com/magazine/story/2017/12/16/pentagon-ufo-search-harry-reid-216111/)用于美国国防情报局的一个项目，该项目向他的朋友比格洛的航空航天公司支付了2200万美元 — 公司用这笔钱进行了对超自然现象的调查。比格洛的团队还调查了美国军方人员关于不明飞行物的目击事件，并提议建立实验室研究外星飞船据称的物理残留物。在2017年5月的“60分钟”节目中，[比格洛](https://www.cbsnews.com/news/bigelow-aerospace-founder-says-commercial-world-will-lead-in-space/)表示，他“绝对相信”外星人存在，并且 UFO 曾访问过地球。

里德在[2021年告诉内华达州的一名记者](https://www.8newsnow.com/news/i-team-exclusive-sen-reid-discusses-ufo-study/)，尽管这是一个调查 UFO 的秘密项目，但比格洛并没有从中受益于“一些内幕交易……它是公开竞标的。”里德还告诉[《纽约时报》](https://www.nytimes.com/2017/12/16/us/politics/pentagon-program-ufo-harry-reid.html)，“我对推动这个项目并不感到尴尬、羞愧或抱歉……我认为这是我在国会服务中做的一件好事之一。”

然而，柯克帕特里克指出，“这一切事实上并没有体现在任何外星飞船的证据上。”但是有关这些秘密项目的故事在五角大楼内部传播开来，并得到了 embellished，并且偶尔得到了那些听到有关看到或抓到看似科幻技术或飞机的传言的军事人员的支持

柯克帕特里克称，他的调查人员最终追溯到少于十几个人的这场绝密“电话游戏”。

“这一切都可以追溯到同一群核心人物，”柯克帕特里克说。这既极为奇怪又具有丰富的讽刺意味。因为几十年来，UFO的真正信徒们一直告诉我们，美国政府存在着掩盖外星人证据的阴谋。但是——如果你相信柯克帕特里克——更加平凡的事实是，这些故事是由一群政府内外的UFO真正信徒炒作起来的。

### 获取我们的免费周报。

遗憾的是，对于所有UFO爱好者来说，这可能是柯克帕特里克向国会提交的报告中最重要的收获，预计本月晚些时候将会发表。许多[外部人士](https://reason.com/2022/11/15/the-military-ufo-complex/)长期以来一直[推测](https://www.newyorker.com/magazine/2021/05/10/how-the-pentagon-started-taking-ufos-seriously)，五角大楼的关于外星人的项目是否一无所获，或者可疑地自我延续。

但现在，五角大楼内部高度可信的人士——拥有非常高级别的安全许可——终于说，我们查看了关于据称的外星人UFO的每一份秘密证据。据我们所知，这完全是人类的行为。

尽管柯克帕特里克承认，对于那些真正相信地球上存在外星访问的人来说，几乎没有什么能够说服他们改变看法：“没有任何我能做到、说到或提供证据的事情会让真正的信徒们转变……这是一种超越批判性思维和理性思考的宗教信仰。”
