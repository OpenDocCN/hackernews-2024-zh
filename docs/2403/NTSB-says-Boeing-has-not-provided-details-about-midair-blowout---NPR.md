# NTSB称波音未提供关于半空中爆炸事件的细节：NPR

华盛顿 — 超过[两个月后门插销板爆炸](https://www.npr.org/2024/01/06/1223296736/boeing-737-max-9-planes-grounded)出波音737 Max 9飞机在半空中之后，最高联邦安全调查员表示，波音仍未提供关键信息，以揭示问题的根源。

国家运输安全委员会主席Jennifer Homendy [在周三告诉参议院商业委员会](https://www.commerce.senate.gov/2024/3/national-transportation-safety-board-investigations-report)，波音公司未透露是谁在其靠近西雅图的工厂未能正确重新安装门插销。

“两个月后我们仍然没有这些信息，这太荒谬了，”Homendy说。

NTSB 在其上个月的[初步报告](https://www.npr.org/2024/02/06/1229528737/ntsb-boeing-737-max-9-alaska-airlines-door-plug-missing-bolts)中表示，去年飞机离开波音工厂时，应该固定门插销的四个关键螺栓不见了。报告发现，在飞机组装过程中，为了修补机身上误钻的铆钉，门插销被打开。

但是Homendy表示，NTSB仍无法确定是谁打开和关闭了门插销。

“波音未能提供我们多次请求的文件和信息，”Homendy告诉委员会。

“你是在告诉我们，即使两个月后你们仍然不知道是谁打开了门插销？”，委员会的高级成员、得克萨斯州共和党参议员Ted Cruz问道。

“没错，参议员。我们不知道，”Homendy回答道。“并不是因为我们没有尝试。”

Homendy查看了电子邮件和短信后表示，调查人员认为门插销工作发生在去年9月中旬的两天。

美国国家运输安全委员会（NTSB）已要求波音提供执行时间及执行人员的文件。但霍曼迪说，波音告诉调查人员他们“找不到”这些文件。

调查人员还在寻找25名波音员工的名单，他们是负责开启和关闭门插销的团队成员。但霍曼迪表示，迄今为止，飞机制造商尚未提供这些名字。

在这张美国国家运输安全委员会（NTSB）发布的照片中，2024年1月7日，俄勒冈州波特兰市，塑料覆盖了阿拉斯加航空1282号航班的机身插销区外部。在1月5日从波特兰起飞后10分钟，737 Max 9机身后部的一块门大小的区域脱落。**NTSB via Getty Images** ****hide caption****

NTSB via Getty Images

波音发言人在一封发给NPR的电子邮件声明中对霍曼迪的说法提出了异议。“自阿拉斯加航空1282号航班事故发生的第一刻起，我们就积极透明地全力支持NTSB的调查，”波音的康纳·格林伍德说。

“在调查初期，我们向NTSB提供了波音员工的名单，包括门专家，我们认为他们可能拥有相关信息。最近，我们已提供了737门组全体成员的完整名单，以回应最近的请求，”格林伍德说。

在本周三的听证会上，人们对公司的不满似乎跨越了党派界限。

委员会主席、华盛顿州民主党参议员玛丽亚·坎特威尔（Maria Cantwell）[称波音的缺乏合作“令人失望至极。”](https://www.commerce.senate.gov/2024/3/cantwell-to-boeing-where-are-the-documents)

克鲁兹参议员称对调查人员的公司反应“无法接受。”他要求NTSB在一周内报告，看看波音是否同意共享调查人员请求的姓名和文件。

这个委员会的参议员们并没有忘记，在[2018年和2019年的两起737 Max 8飞机失事造成346人死亡](https://www.npr.org/2020/09/16/913426448/congressional-inquiry-faults-boeing-and-faa-failures-for-deadly-737-max-plane-cr)后，波音最初是如何推卸责任的。

霍曼迪说，目前正在寻找的记录可能不存在。如果不存在，这将对公司的质量控制实践提出严重质疑，她说。

“我们被告知，他们有一套程序用于记录工作执行时间，包括门插销的开启、关闭或移除时间，”她说。“我们无法验证这一点。没有这些信息，对波音的质量保证、质量管理、安全管理系统提出了担忧。”

Boeing也承认了NTSB正在寻找的文件可能不存在的可能性。

“如果门插销拆除未经记录，那么将没有文档可供共享，”格林伍德在声明中说道。“我们将继续全力配合并透明地与NTSB的调查合作。”

[联邦航空管理局的监管人员也表示关注](https://www.npr.org/2024/03/04/1235826355/faa-audit-boeing-737-max-8-9-quality-control-failures)他们称之为波音及其供应商存在的“系统性质量控制问题”。他们已经给予公司直到五月底提出解决这些问题的计划。
