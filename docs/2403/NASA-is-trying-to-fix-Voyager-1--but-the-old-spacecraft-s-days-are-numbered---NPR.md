# NASA正在尝试修复旅行者1号，但这艘老航天器的时日已经不多了：NPR

> 来源：[https://www.npr.org/2024/03/06/1236033493/nasas-voyager-1-spacecraft-is-talking-nonsense-its-friends-on-earth-are-worried](https://www.npr.org/2024/03/06/1236033493/nasas-voyager-1-spacecraft-is-talking-nonsense-its-friends-on-earth-are-worried)

[斯塔马提奥斯“汤姆”·克里米吉斯](https://voyager.jpl.nasa.gov/mission/science/institutions/)上次亲眼见到旅行者1号太空探测器是在1977年夏天，就在它从佛罗里达州卡纳维拉尔角发射前不久。

如今，旅行者1号已经飞行超过150亿英里，越过许多人认为的太阳系边界。尽管如此，克里米吉斯负责的舱内仪器仍然运转良好。

“我是世界上最吃惊的人，”克里米吉斯说道——毕竟，这艘航天器最初的任务只是前往木星和土星，预计只能持续大约四年。

然而，如今，每当他想起旅行者1号时，他也感到另一种情绪。

“坦率地说，我非常担心，”他说。

从11月中旬以来，旅行者1号太空船一直在向地球发送一些毫无意义的消息。仿佛这艘年迈的航天器遭遇了某种中风，干扰了它的通信能力。

“它基本上停止以有条理的方式与我们通信，”NASA喷气推进实验室的[Suzanne Dodd](https://www.jpl.nasa.gov/who-we-are/executive-council/suzanne-dodd-director-for-the-interplanetary-network-directorate)说道，她自2010年以来一直是[旅行者](https://voyager.jpl.nasa.gov/)星际任务的项目经理。“这是一个严重的问题。”

现在，旅行者1号不再以二进制代码发送消息回家，而是只发送交替的1和0。Dodd的团队已经尝试了通常的重置方法，但没有成功。

看起来舱内计算机出现了问题，它负责接收数据并打包发送回地球。Dodd说，与解锁你的汽车的钥匙远远不同，所有这些计算机技术都是原始的。

"你按开车门的按钮，那比旅行者航天器的计算能力都要强大，" 她说道。"他们依然在飞行，已经飞行了46多年，这真是不可思议。"

航天者1号及其孪生航天者2号已经超过了许多设计和建造它们的人。所以，为了试图修复航天者1号目前的困扰，道德团队的十几个人不得不研究泛黄的文件和老式打印件。

“他们正在努力进入原始开发者的思维，弄清他们为什么设计某样东西以及我们可能尝试什么可能给我们一些解答，关于航天器出了什么问题，”道德说。

她说他们已经列出了可能的修复清单。随着时间的推移，他们可能会开始向航天者1号发送更为大胆和风险的指令。

“我们未来要做的事情可能更具挑战性，因为你无法确切地知道它是否会执行正确 — 或者你可能会做一些你不想做的事情，无意间地，”道德说。

[琳达·斯皮尔克](https://science.jpl.nasa.gov/people/spilker/)在NASA喷气推进实验室担任航天者任务的项目科学家，她说，当她去工作时，看到“所有这些电路图都挂在墙上，上面贴着便签。而这些人只是在努力排除故障，你知道，处理60年代和70年代的技术。”

“我持谨慎乐观态度，”她说。“这里有很多创造性。”

然而，这是一个繁琐的过程，可能需要几周，甚至几个月。航天者1号距离如此之远，信号传输需要几乎一整天的时间，然后再等待一整天才能收到回应。

“我们会继续努力，”道德说，“但这不会很快。”

与此同时，航天者1号的故障让天文学家斯特拉·奥克（[斯特拉·奥克](https://astro.cornell.edu/stella-koch-ocker)）等研究人员感到沮丧，她是加州理工学院和卡内基天文台的天文学家。

“自从这个异常开始以来，我们就没有得到科学数据，”奥克说，“这意味着我们不知道飞船所处环境的样子。”

那个星际环境并不只是空荡荡的黑暗，她说。它包含像气体、尘埃和宇宙射线这样的物质。只有孪生航天者探测器足够远，可以取样这些宇宙炖菜。

“我真正感兴趣的科学研究实际上只有航天者1号能够完成，”奥克说，因为航天者2号尽管在其先进年龄中通常健康，但无法进行她需要的特定测量。

即使NASA的专家和顾问们设法提出了一个能让航天者1号恢复正常的奇迹计划，它的时间也在流逝。

两艘航天者探测器由钚提供动力，但这种动力系统最终将耗尽。任务负责人已经[关闭了加热器](https://www.jpl.nasa.gov/news/nasas-voyager-will-do-more-science-with-new-power-strategy)并采取其他措施来节约能量并延长航天者探测器的使用寿命。

"我很长一段时间的座右铭是五十年或者失败," Krimigis博士笑道，"但我们似乎正在接近那个目标。"

几年之内，电力供应的逐渐减少将迫使管理人员逐个关闭科学仪器。最后一个仪器可能会一直运行到2030年左右。

当电力耗尽并且探测器毫无生命迹象时，Krimigis博士说这两个传奇航天探测器基本上将成为"太空垃圾"。

"说出这个真让我伤心," 他说。虽然Krimigis博士参与了前往每个行星的太空任务，他说旅行者计划在他心中占有特殊的位置。

Spilker女士指出，每个航天器将继续向外移动，并携带一份[金唱片](https://voyager.jpl.nasa.gov/golden-record/)，其中记录了多种语言的问候语，以及地球的声音。

"科学任务将会结束。但旅行者的一部分和我们的一部分将会继续在星际空间中存在," Spilker女士说道，指出金唱片"甚至可能在我们知道的人类灭绝之后继续存在。"

Krimigis博士却怀疑任何外星人会偶然发现**旅行者**探测器并进行倾听。

"太空是空无一物的," 他说，"旅行者探测器撞上行星的可能性可能几乎是微乎其微的。"

旅行者1号将需要大约40000年才能接近另一颗恒星；它将会在NASA所称的"北斗座小熊座一个不知名的恒星"附近的1.7光年内靠近。

旅行者探测器的时限即将到期，科学家们已经起草了一个新的[任务](https://www.npr.org/2021/10/26/1048358753/if-nasa-green-lights-this-interstellar-mission-it-could-last-100-years)，如果NASA资助并发射成功，将会把另一个探测器送往更远的星际空间。

"如果发生，它将会在2030年代发射," Ocker先生说道，"并且在仅仅50年内就会到达比旅行者1号更远的距离。"
