# 如何让谷歌显示更准确的搜索结果 | Lifehacker

> 来源：[https://lifehacker.com/tech/how-to-get-more-accurate-google-search-results](https://lifehacker.com/tech/how-to-get-more-accurate-google-search-results)

* * *

谷歌在显示有用的搜索结果方面变得更差，我知道[不止我一个人注意到](https://mashable.com/article/google-search-low-quality-research "(在新窗口中打开)")。例如，最近我搜索了一篇Lifehacker的帖子，其中包含“kelvin”一词，这并不是一个常见的术语。但谷歌没有列出少数提到它的文章（包括我在寻找的[连接专栏](https://lifehacker.com/entertainment/nyt-connections-answer-today-march-2-2024)），而是决定给我展示“K”，温度单位的科学缩写的结果页面。因此，给我展示了一大堆关于如何在401(k)账户中省钱的文章。

这些偏离目标的搜索结果变得越来越普遍，我们只能把*部分*责任归咎于结果中充斥着人工智能生成的垃圾和人类生成的SEO诱饵。我无法修复人们（和机器人）所写的内容，也无法深入谷歌的排名算法来调整页面顺序。但是我可以告诉你如何让谷歌真正搜索你输入的内容。

## 如何使谷歌搜索你想要的词汇，而不是同义词

这种情况之所以发生的原因是：谷歌试图通过搜索与您搜索的词汇相似的词来帮助您（更加愤世嫉俗地说，它们可能试图通过将您的兴趣导入广告商想要销售给您的东西来帮助您）。大部分时间这是有用的；例如，搜索复数“跑鞋”也会找到名为“跑鞋”的页面。

最近，这些调整似乎变得更加广泛，给我们带来了偏离目标的搜索结果，比如我之前提到的，我在寻找一个温度单位，结果却得到了关于退休基金的建议。以前有一个简单的解决办法：你把你要搜索的词用引号括起来。但这最近已经不太可靠了（尽管有时似乎还有效）。

输入**verbatim**选项。这不是新事物，但鉴于谷歌对我们搜索查询的不断调整，它变得更加重要。

作者：Google/Beth Skwarecki

## 如何在谷歌搜索中开启verbatim模式

在搜索结果页面上，寻找一个标签栏，上面写着**Tools**。它可能位于屏幕的最右侧。

+   在桌面上，看到页面的最右侧，靠在任何按钮的右侧，提示相关搜索或不同类别的搜索（图片，购物等）。

+   在移动设备上，在搜索框下方你会看到那些“图片、购物…”类别。把手指放在这些词上并向左滑动；在排行末尾的最右边，你会看到**搜索工具**。

+   点击或触摸**工具**或**搜索工具**后，你会看到一个标有**所有结果**的下拉菜单。点击或触摸它，你会看到**逐字搜索**选项。

作者：Google/Beth Skwarecki

在**工具**下的**逐字搜索**功能是搜索你实际输入的关键。我们不能保证你会着陆在质量如何的页面上，但至少你会得到更精准的选择。
