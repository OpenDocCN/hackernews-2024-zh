## HotDoc 文档

Hotdoc 是一个文档框架。它提供一个接口供扩展插入，并提供一些基础对象（格式化程序，...）

Hotdoc 随附一组扩展，执行各种任务，例如使用 `clang` 解析 C 代码并提取符号，解析 `gobject-introspection`（gir）文件，使用 `prism` 高亮代码片段的语法等。
