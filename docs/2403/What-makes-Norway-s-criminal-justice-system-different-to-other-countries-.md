# 挪威刑事司法系统与其他国家有何不同？

> 来源：[https://www.thelocal.no/20210429/what-makes-norways-criminal-justice-system-different-to-other-countries](https://www.thelocal.no/20210429/what-makes-norways-criminal-justice-system-different-to-other-countries)

**“正常性”与挪威刑事司法系统的原则**

挪威的刑事司法系统在根本层面上与其他国家不同，这是一位专家向《本地新闻》透露的观点。

“系统的哲学，尤其是监狱系统，非常注重康复，” 奥斯陆大学法学院犯罪学和社会学系的教授卡特亚·弗兰科说。

挪威的刑事司法和监狱系统的主要原则是帮助罪犯重新融入社会，而不仅仅是惩罚他们。

“在挪威，康复可能比大多数欧洲国家，尤其是英国，更为重要，” 弗兰科说。

在挪威，入狱被视为犯罪行为的足够惩罚。

[挪威矫正服务总局](https://www.kriminalomsorgen.no/information-in-english.265199.no.html)概括其主要信念：“惩罚是剥夺自由；判决法院未取消其他任何权利。因此，被判犯罪的罪犯拥有与挪威其他居民相同的所有权利。没有人应该在不必要的情况下受到比社区安全所需更严格的条件下服刑。”

### 广告

<googlead-js :data="{&quot;current_editorial&quot;:&quot;no&quot;,&quot;name&quot;:&quot;mpu_2&quot;,&quot;size&quot;:&quot;[[300,250]]&quot;,&quot;css_class&quot;:&quot;&quot;}">这一总结被称为“正常性原则”，是挪威司法系统的核心原则。

"这一原则至少有两个方面：其中一个——对我们来说至关重要的一个方面——是即使你被判了刑，你仍然是一个公民，" 挪威矫正服务总局局长玛丽安娜·沃兰在《司法趋势》网站上说。

"正常性原则的另一部分是，在执行刑罚期间，生活应尽可能与无刑罚时的生活相似，" 沃兰说道。“因此，当我们谈论监狱时，我们应努力使监狱中的日常生活和常规生活尽可能与社会外部生活相似。”

并非所有在挪威被判犯罪的人最终都会入狱，但如果他们入狱，刑期可能会比其他国家的相同犯罪刑期短（请参见页面底部）。

挪威当局增加了电子监控的使用，这意味着刑罚可以在家中执行。

“毫不夸张地说，将挪威与其他刑罚体系，例如美国，区分开来的一些重要差异包括更短和更多样的刑期以及更短的整体最高刑罚，”奥斯陆大学刑法与法律社会学系博士候选人Rose Boyle说道。

**挪威的监狱是什么样的？**

斯堪的纳维亚监狱通常被描述为比其他地方的监狱更加“人道主义”和福利导向。

### 广告

<googlead-js :data="{&quot;current_editorial&quot;:&quot;no&quot;,&quot;name&quot;:&quot;mpu_3&quot;,&quot;size&quot;:&quot;[[300,250]]&quot;,&quot;css_class&quot;:&quot;&quot;}">特别是在挪威。挪威的监狱更被视为康复设施而非拘留中心。[负责看管囚犯的官员](https://www.kriminalomsorgen.no/informasjon-paa-engelsk.536003.no.html#:~:text=In%20total%2C%20Norway%20has%20a,of%20these%20are%20high%2Dsecurity.)必须完成为期两年的培训项目，学习心理学、犯罪学、法律、人权和伦理等多个学科。监狱内工作的官员未装备武器，而且几乎一半的员工是女性。

“看守和囚犯一直在一起参与各种活动。他们一起吃饭、打排球、进行休闲活动，这使我们能够真正与囚犯互动、与他们交谈并激励他们。”监狱长Are Høidal告诉BBC。

“囚犯是人类。他们犯了错误，必须受到惩罚，但他们仍然是人类，”他补充道。

[在监狱期间](https://www.sixnorwegianprisons.com/experiences)，挪威的囚犯可以选择继续接受教育，并参与由教师主持的工作坊和课程学习。

在监狱提供的福利服务，如继续教育和医疗保健，由挪威国家保险计划管理。这是为了确保囚犯在这些福利方面与生活在外面的人享有同等质量和访问权利。

请注意，这只是一种选择。一些囚犯选择以更为隐蔽的方式服刑。

“监狱中的日常生活理想上尽可能反映出与监狱外生活相似的存在方式，”犯罪学博士生Boyle说道。

“囚犯被期望工作或接受教育，可以参与各种项目，可以享受健康服务，并且有权在选举中投票，这只是几个例子。他们享有与其他公民相同的权利和责任，”她补充道。

**“正常性”有效吗？**

挪威不会为无故付出这些努力和成本。

“正常性原则”的总体目标及其康复追求是降低再犯率，使前囚犯不再进监狱。

"这都是我们主要任务的一部分，而不是一种天真的方法。我们了解我们所处理的人，但我们相信应用正常化原则可以减少累犯率（重新犯罪）"，监狱服务主任沃兰[告诉《司法趋势》。](https://justice-trends.press/full-rights-citizens-the-principle-of-normality-in-norwegian-prisons/)

<googlead-js :data="{&quot;current_editorial&quot;:&quot;no&quot;,&quot;name&quot;:&quot;mpu_4&quot;,&quot;size&quot;:&quot;[[300,250]]&quot;,&quot;css_class&quot;:&quot;&quot;}>"我们在这里不仅是为了执行羁押令和判决，还要以一种可以防止累犯的方式来执行"，她说。

有统计数据显示这些努力是值得的。

根据2019年BBC的报道，英国的重新犯罪率在仅一年后接近50％。在美国，这一比例高达70％。但在挪威，这一数字已经降至仅20％。

前囚犯在挪威可能更有可能避免再次入狱，但这并不意味着他们在离开监狱后的生活轻松。

"公共话语中可能对前囚犯带来的危险关注较少。然而，对大多数人来说，重新融入社会仍然很困难。对许多人来说，他们必须向国家偿还债务是一个很大的障碍"，奥斯陆大学刑事学专家弗兰科说。

经济压力是一个真正的问题。[八成囚犯中](https://www.nrk.no/norge/8-av-10-innsatte-i-norske-fengsler-har-omfattende-gjeld-1.13256674)有大额债务。这些类型的债务包括未支付的索赔、罚款、高额消费贷款和信用卡债务。

应当指出，顺利重新融入社会并不总是最终目标。

"这种哲学对于挪威公民和外国犯罪者来说也不同"，弗兰科指出。

“大约30％的囚犯是外国公民，对他们而言，重点是在服刑后将他们送出国门，而不是重新融入社会” ，她说。

当然，挪威的这一制度并不是普遍被认可的。

"这种'斯堪的纳维亚例外主义'在现实中存在到什么程度，以及北欧国家监狱系统与世界其他地区有何不同，这是监狱学者们正在进行的辩论之一"，奥斯陆大学的博伊尔说。

许多人认为该系统对于暴力等严重犯罪的施害者来说过于舒适和宽容，而其他人则对确保人道主义监禁期间所作的努力感到骄傲，并认为这是挪威犯罪率低的重要原因之一。

在挪威，将刑事司法系统作为随便聊聊的话题是社会上可以接受的，因为这个话题不太容易引起很大的紧张感。

“在媒体上，犯罪司法问题并不像某些其他国家那样热议，也不像移民问题那样”，Franko说道。“公众似乎有一种公开一致的表象。”

**监狱率和判决**

在2020年，挪威的监狱囚犯总数为[2,653](https://www.prisonstudies.org/country/norway)，官方占用率为76%。共有58所监狱，仅有0.3%的囚犯年龄低于18岁。

在所有欧洲国家中，[监禁率](https://www.coe.int/en/web/prison/home/-/asset_publisher/ky2olXXXogcx/content/publication-of-the-latest-european-data-on-prison-populations?inheritRedirect=false&redirect=https%3A%2F%2Fwww.coe.int%2Fen%2Fweb%2Fprison%2Fhome%3Fp_p_id%3D101_INSTANCE_ky2olXXXogcx%26p_p_lifecycle%3D0%26p_p_state%3Dnormal%26p_p_mode%3Dview%26p_p_col_id%3Dcolumn-1%26p_p_col_count%3D4)自2013年以来一直在下降。挪威与芬兰、荷兰和冰岛一样，是欧洲监禁率最低的国家之一。

然而，相对于其他北欧国家，挪威的监禁率相对人口更高，这在很大程度上是由于严格的禁毒法律所致。这是由[研究网站](https://forskning.no/kriminalitet/norge-har-flest-fanger-i-norden/1274053)指出的。

挪威的最高[监禁刑期](https://www.kriminalomsorgen.no/informasjon-paa-engelsk.536003.no.html#)被认为是21年，尽管新刑法为种族灭绝罪、反人类罪和战争罪相关罪行规定了最高30年的刑期。

最高刑期为21年，但挪威最严厉的惩罚是*forvaring*或“特别拘留”。[*Forvaring*是无限期的](https://22julisenteret.no/ressurs/22-juli-saken-forvaring/)，可以在判决后延长。它适用于被判有严重犯罪行为并被认定对社会构成有害风险的个人。[ ](https://22julisenteret.no/ressurs/22-juli-saken-forvaring/)

挪威的平均监禁期约为8个月。几乎90%的囚犯在监禁不足一年后即获释。
