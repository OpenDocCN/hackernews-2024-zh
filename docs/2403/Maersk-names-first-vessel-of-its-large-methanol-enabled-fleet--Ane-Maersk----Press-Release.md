# 马士基航运宣布其大型甲醇船队的首艘船只命名为“安妮·马士基”

2024年1月26日

**韩国乌山** – 世界首艘大型甲醇船用集装箱船在韩国乌山现代重工业（HD HHI）的船坞举行的仪式上，以“安妮·马士基”之名命名。该船以安妮·马士基·麦金尼·乌格拉（Ane Mærsk Mc-Kinney Uggla）为名，她是A.P.穆勒基金会和A.P.穆勒控股的主席。安妮的长孙女担任教母，并通过在船艏处打破香槟酒瓶来为船艇命名。

“安妮·马士基”是马士基公司2024年至2025年间交付的18艘大型甲醇船队的首艘船只。这也是全球第二艘采用甲醇动力的集装箱船。从二月初起，它将投入AE7航线服务，连接亚洲和欧洲，标志着公司在领先低排放航运解决方案上的重要里程碑。新系列船只具有行业首创的创新设计，桥梁和船员住宿设施位于船艇前部，确保燃油高效操作。

> 这一系列船只将对我们在行业领先的气候野心上取得的进展产生深远影响。这不仅是我们承诺更可持续行业的视觉和操作证明，也标志着我们向越来越多旨在减少供应链排放的企业提供的扩展服务。通过“安妮·马士基”及其姐妹船只，我们正在扩展我们的服务。

“安妮·马士基”将以绿色甲醇进行首航，马士基公司继续努力于2024-2025年甲醇动力船队的采购和供应解决方案。

¹ 马士基将“绿色燃料”定义为其整个生命周期内温室气体排放低至非常低的燃料，与化石燃料相比。不同的绿色燃料通过其生产途径实现不同的生命周期减排。在此，我们所说的“低”是指与化石燃料相比生命周期温室气体减排65-80%的燃料，例如某些生物柴油。而“非常低”是指与化石燃料相比生命周期温室气体减排80-95%的燃料。
