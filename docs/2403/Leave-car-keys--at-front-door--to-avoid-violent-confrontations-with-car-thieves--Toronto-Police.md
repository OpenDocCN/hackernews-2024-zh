# 把车钥匙“放在门口”以避免与汽车小偷的暴力冲突：多伦多警方

> 来源：[https://toronto.citynews.ca/2024/03/14/leave-car-keys-at-front-door-to-avoid-violent-confrontations-with-car-thieves-toronto-police/](https://toronto.citynews.ca/2024/03/14/leave-car-keys-at-front-door-to-avoid-violent-confrontations-with-car-thieves-toronto-police/)

多伦多警方因其许多人认为的对继续困扰大多伦多地区的汽车盗窃问题的漠不关心的态度，在线上遭到了严厉批评。

上个月在Etobicoke举行的安全会议上，警官马可·里奇亚迪建议居民把他们的钥匙离开他们的车的地方，放在一个方便小偷的法拉第袋里，以减少暴力冲突的风险。

“为了防止在家中被攻击的可能性，请将您的钥匙放在前门，因为他们会闯入您的家偷车，他们不想要其他东西。”

“他们抓到的许多人身上带着枪，而且不是玩具枪，”他恐吓地补充道。“是真枪。它们上膛了。”

这些建议在X（以前称为Twitter）上引发了愤怒的反应。

“这太离谱了，”一位X用户写道。“多伦多警方建议公众把钥匙放在门口。他们之所以这样建议是因为入侵者主要是进入房屋偷车。为什么不把钥匙挂在门外？或者更好的办法是把它们留在车里。”

“这就像失败的国家级疯狂，”另一位补充道。“嘿，伙计们，你们介意让人们更容易偷你们的车，以便他们不会闯入你们的房子吗？谢谢。”

多伦多警方周四在X上趋势上升，有成千上万条关于这个话题的推文，大多数表达了愤怒。

统计数据显示，2023年大多伦多地区的汽车盗窃率比前一年增加了近25%。

更令人不安的是，多伦多警方称，2023年家庭入侵和入室盗窃以及汽车盗窃上升了400%。

在周三的一份新闻稿中，多伦多警方为担心的市民提供了更多建议。

“最近的社区会议上，一名警官建议人们把车辆的钥匙放在前门的法拉第袋里，”新闻稿中提到。

“虽然出于善意，还有其他防止汽车盗窃动机家庭入侵的方法……警方担心暴力升级，使用各种武器和枪支来偷车，包括在家庭入侵期间。”

*多伦多警方建议以下措施：*

+   如果可能，把车停在车库里

+   确保您的车道照明良好，并整晚保持外部灯光亮着

+   安装安全摄像头是一项资产

+   如果可能，安装家庭安全系统——在家时激活待命模式，外出时激活离开模式

+   考虑安装连接到您手机的运动检测警报系统

+   在玻璃窗户和门上安装安全膜

+   在您的门上安装多点锁

+   锁好后院的大门和地面窗户

+   不要在社交媒体上发布你将要度假的消息。

+   报告邻里中可疑的车辆/人员给警察。

约克区警察甚至送出了门挡给一些高风险区域的居民，以防止窃贼通过寻找钥匙来踢开前门。

“所以已经到了他们只是随意把这些东西发给居民的地步，对我来说，这表明这是非常高风险的，”Ally Galinski说道。

Galinski说，警方还递送了一本小册子，提供了防止车辆被盗的建议，包括投资报警系统、安全摄像头和运动传感器灯，并避免在社交媒体上发布度假计划。

她说她很感谢这些建议，但希望警方能对这个问题采取更积极的措施。

“我很高兴他们采取了主动措施...但至少在高风险区域，也许可以增加巡逻？”

皮尔地区警察在周四更新了X的情况，概述了他们应对不断增长的问题的努力，并向公民提供了建议。

与多伦多警察不同，皮尔警察建议将“钥匙离开门口和入口处”。
