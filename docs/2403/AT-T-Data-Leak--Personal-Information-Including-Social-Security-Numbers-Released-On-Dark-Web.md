# AT&T数据泄露：个人信息——包括社保号——在暗网上泄露

> 来源：[https://www.forbes.com/sites/tylerroush/2024/03/30/att-says-personal-information-from-73-million-customers-leaked-on-the-dark-web-including-social-security-numbers/](https://www.forbes.com/sites/tylerroush/2024/03/30/att-says-personal-information-from-73-million-customers-leaked-on-the-dark-web-including-social-security-numbers/)

## 要点

个人信息——包括大约7300万名AT&T客户的社保号——本月早些时候在暗网上泄露，公司周六表示，尽管数据来源仍在调查中，但尚不清楚泄露是来自公司内部还是其供应商之一。

<fbs-accordion class="expandable" current="-1">数据泄露包括客户的社保号等信息，公司... [+] 表示。</fbs-accordion> Getty Images

## 关键事实

对泄露的初步分析显示，数据集似乎来自2019年或更早，影响约760万当前账户持有者和6540万前账户持有者，AT&T [说](https://www.prnewswire.com/news-releases/att-addresses-recent-data-set-released-on-the-dark-web-302103937.html "https://www.prnewswire.com/news-releases/att-addresses-recent-data-set-released-on-the-dark-web-302103937.html")。

数据集“包括个人信息，如社保号码”，AT&T表示，数据在大约两周前泄露到暗网。

AT&T表示尚无证据显示未经授权访问其系统，并且泄露尚未对公司运营产生“实质性影响”。

泄露后，公司重置了数百万客户账户密码，Bloomberg [报道](https://www.bloomberg.com/news/articles/2024-03-30/at-t-t-says-data-from-73-million-accounts-leaked-on-dark-web "https://www.bloomberg.com/news/articles/2024-03-30/at-t-t-says-data-from-73-million-accounts-leaked-on-dark-web")。

AT&T将“主动”与受泄露影响的人员沟通，并根据情况提供信用，公司表示。

## 旁支

上个月，AT&T 客户遭遇了广泛的服务中断，超过 70,000 名客户在德克萨斯州、佛罗里达州、加利福尼亚州和其他州失去了手机服务。 国家安全委员会发言人约翰·柯比（John Kirby）表示，国土安全部和联邦调查局都协助公司调查了这次故障，并随后确认没有证据表明这是一次网络攻击。 AT&T [向](https://www.forbes.com/sites/zacharyfolk/2024/02/25/att-will-pay-customers-5-credit-after-massive-outage/?sh=6b95e4b912b7 "https://www.forbes.com/sites/zacharyfolk/2024/02/25/att-will-pay-customers-5-credit-after-massive-outage/?sh=6b95e4b912b7") 受影响的客户提供了一笔 $5 的信用，公司称这是“全天服务的平均成本”，而内部对事件的审查则指向了一个内部错误。

## 令人惊讶的事实

其他通信公司在过去一年中受到数据泄露的影响。 Verizon [通知](https://apps.web.maine.gov/online/aeviewer/ME/40/65b9290a-b22e-4ae7-93e7-5acb84357297.shtml "https://apps.web.maine.gov/online/aeviewer/ME/40/65b9290a-b22e-4ae7-93e7-5acb84357297.shtml") 缅因州总检察长办公室，透露在去年九月份的内部数据泄露中，超过 63,000 名员工的个人信息遭到暴露。一项内部调查确定，一名 Verizon 员工未经授权获得了包含员工姓名、地址、出生日期和社保号码等个人信息的文件。 T-Mobile [向](https://apps.web.maine.gov/online/aeviewer/ME/40/ea3bf342-eca7-4833-b128-7b09f6893ac4.shtml "https://apps.web.maine.gov/online/aeviewer/ME/40/ea3bf342-eca7-4833-b128-7b09f6893ac4.shtml") 缅因州总检察长办公室报告称，836 名客户受到数据泄露的影响，尽管未访问个人财务信息。这起泄露是几个月前报道的另一起泄露事件的后续，该事件[影响了](https://www.theverge.com/2023/1/20/23563825/tmobile-data-breach-api-customer-accounts-hacker-security "https://www.theverge.com/2023/1/20/23563825/tmobile-data-breach-api-customer-accounts-hacker-security")约 37 百万客户。
