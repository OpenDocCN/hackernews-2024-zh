# STMicro的STM32系列通过新的边缘AI、支持3D的STM32MP芯片迈向64位时代 - Hackster.io

> 来源：[https://www.hackster.io/news/stmicro-s-stm32-family-makes-the-64-bit-jump-with-new-edge-ai-3d-capable-stm32mp-parts-2a470d8eb2cb](https://www.hackster.io/news/stmicro-s-stm32-family-makes-the-64-bit-jump-with-new-edge-ai-3d-capable-stm32mp-parts-2a470d8eb2cb)

STMicroelectronics宣布推出第二代STM32微处理器，承诺提升性能并加速边缘人工智能（AI）和机器学习工作负载，同时首次在该系列历史上使用64位处理器核心。

“我们的嵌入式MPU解决了推动更多工作负载和更高要求到智能设备（通常部署在IoT边缘）的趋势，以实现更快的响应和提高效率。” STMicro公司的Stephane Henry如此描述公司的新处理器。

STMicro宣布了STM32系列的新产品，它们是公司迄今为止性能最强的产品。(📷: STMicroelectronics)

"我们今天宣布的新STM32MP2设备," Henry 继续说道, "延续了性能发展轨迹，引入了我们最强大的处理引擎，现在增加了边缘AI，并由STM32生态系统支持，加速产品开发。"

新芯片系列的第一个惊喜是：尽管保留了STM32的名称，却迈向了64位处理器核心。STM32MP2芯片是公司首次采用Arm Cortex-A35核心，运行速度高达1.5GHz，同时配备运行速度高达400MHz的32位Arm Cortex-M33微控制器核心，配备具有3D加速和支持1080p显示的独立图形处理单元（GPU）、视频处理单元（VPU）和神经处理单元（NPU），旨在加速设备边缘AI和机器学习工作。

新产品包括最多两个64位Arm Cortex-A35核心，运行速度高达1.5GHz，但仍保留“STM32”名称。(📷: STMicroelectronics)

STMicro声称，这款NPU能够提供高达1.35万亿次每秒（TOPS）的计算能力，工作负载可以在需要时在Cortex-A35、Cortex-M33、GPU、VPU或NPU核心上运行。然而，并非所有型号都配备：仅STM32MP257、STM32MP255和STM32MP23x芯片包括AI协处理器和3D GPU；其余系列则仅配备一到两个Cortex-A35核心和单个Cortex-M33。

STMicro对这些部件的工业认证非常重视：公司表示这些芯片可以24小时不间断运行10年，其结温额定为-40°C至125°C（-40°F至257°F）。新部件还具备IoT平台安全评估标准（SESIP）第3级认证，并已准备好获得美国网络信任标志，这一标志是在去年7月宣布的[（https://www.hackster.io/news/the-us-cyber-trust-mark-aims-to-label-internet-of-things-products-built-with-security-in-mind-166c82e1da9a）](https://www.hackster.io/news/the-us-cyber-trust-mark-aims-to-label-internet-of-things-products-built-with-security-in-mind-166c82e1da9a)，以及公司最近公布的32位STM32WBA55和STM32WBA54芯片[（https://www.hackster.io/news/stmicro-goes-concurrent-with-multi-protocol-ble-zigbee-thread-and-matter-stm32wba55-54-chips-f5feff3995e0）](https://www.hackster.io/news/stmicro-goes-concurrent-with-multi-protocol-ble-zigbee-thread-and-matter-stm32wba55-54-chips-f5feff3995e0)。

这个系列中的一些（但不是全部）型号包括支持3D能力的GPU和一个1.35 TOPS神经网络协处理器。（📷：STMicroelectronics）

这些部件的其他特性包括通过两端口交换机实现最多三个千兆以太网端口，支持时间敏感网络（TSN）、PCI Express Gen. 2、USB 3.0，以及最多三个CAN-FD总线，RGB、LVDS和MIPI显示串行接口（DSI）视频输出，配备GPU的模型支持H.264视频加速，以及一个MIPI摄像头串行接口2（CSI-2）输入并带有图像信号处理器。

尽管STMicro现在正在发布这些部件，但是，他们还没有完全准备好接受订单：这些部件预计将在今年6月进入大规模生产阶段，随后会推出开发板。更多信息请访问[STMicro的网站](https://www.st.com/content/st_com/en/campaigns/microprocessor-stm32mp2.html)。
