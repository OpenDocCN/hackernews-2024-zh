# 美国国立卫生研究院研究提供了对慢性疲劳综合征的期待已久的洞见 | 神经学 | JAMA | JAMA Network

> 来源：[https://jamanetwork.com/journals/jama/fullarticle/2816614](https://jamanetwork.com/journals/jama/fullarticle/2816614)

医学界长期以来一直认为，慢性疲劳综合征（ME/CFS）缺乏合法性。现在，来自美国国立卫生研究院（NIH）及其合作伙伴的75名多学科科学家和临床医生组成的团队完成了一项针对后感染性ME/CFS患者精心挑选群体的广泛住院研究，该研究发表在《自然通讯》上。
