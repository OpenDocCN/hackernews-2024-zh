# 波音公司是如何输掉的一面

> 来源：[https://backofmind.substack.com/p/how-the-wrong-side-won-at-boeing](https://backofmind.substack.com/p/how-the-wrong-side-won-at-boeing)

当我在听[Odd Lots](https://www.bloomberg.com/news/articles/2024-01-19/what-s-wrong-with-boeing-ba-how-its-corporate-culture-changed-over-the-years) 关于波音的播客时，这些想法袭击了我，我强烈推荐这个。波音737MAX是[不负责任机器](https://www.amazon.co.uk/Unaccountability-Machine-Systems-Terrible-Decisions-ebook/dp/B0CGFWBFD6)（现在预订！或从[各种替代品](https://profilebooks.com/work/the-unaccountability-machine/)购买，如果你不喜欢亚马逊！）的核心案例研究之一，因为它是一个决策系统的生动例子，产生了一个可怕的结果，而没有任何可识别的自然人承担责任。

这本书还有一章讨论了[作为精神监狱的会计系统](https://backofmind.substack.com/p/every-accounting-system-is-a-mental)，但我从未想过把这两件事联系起来，可以说我本应该这样做，因为我认为波音公司出现问题的方式与此有很大关系。所以，现在文本已经经历了“现在只有微小的变动”、“现在只有不影响分页的变动”、“只有错别字”、“不再有变动”和“我们是认真的，如果你要求更多变动，请不要回复此邮件”的阶段，我想用‘Stack重新写一下，专注于这个具体问题而不是一般的叙述线索。

《Odd Lots》节目，我个人认为是正确的（也就是说，当我在书中阅读时，它对我来说似乎是令人信服的[链接](https://www.amazon.co.uk/Flying-Blind-Tragedy-Fall-Boeing/dp/0241455596/ref=asc_df_0241455596/?tag=googshopuk-21&linkCode=df0&hvadid=570519535670&hvpos=&hvnetw=g&hvrand=5831130990039913929&hvpone=&hvptwo=&hvqmt=&hvdev=c&hvdvcmdl=&hvlocint=&hvlocphy=9045342&hvtargid=pla-1637642458063&psc=1&mcid=c266d24ddc823e4386022f89745d23c4&th=1&psc=1），然后当作者在播客上重复同样的观点时，它再次似乎是令人信服的）把1997年与麦道公司的合并作为波音公司历史的转折点。这导致了从波音历史上“工程文化”（以做到正确和做必要的事情为基础）到更符合杰克·韦尔奇/股东价值精神的文化的彻底转变，专注于成本控制和投资回报。

不过有点奇怪。 波音公司是收购者和更大的公司； 麦克唐纳实际上并不是处于那么好的状态； 它有一个良好的国防业务，但民用飞机业务不佳； 事实上，波音的一个吸引力之一是，麦克唐纳有多余的工厂产能，可以用来加速其自己日益增长的订单。 正如您所期望的那样，作为一家沿财务路线运作的公司，它也相当负债累累。 那么为什么成为了麦克唐纳的文化成为了主导？

让我们退一步讨论经济学。 约瑟夫·熊彼特（Joseph Schumpeter）指出了他称之为“里卡多谬误”的东西。 这是经济学家倾向于建立一个理论模型（从太多现实世界的细节中抽象出来），解决这个模型，然后行事，好像他们已经解决了现实世界的问题。 在某种程度上，这并不特别是经济学家的特有现象——建模的本质是这样的，（正如我在[一段时间前提到的](https://backofmind.substack.com/p/let-us-be-led-by-our-thoughts)）已经制作了现实的表现，特别是为了减轻其复杂性并使问题可控，然后您不再回去减少其复杂性。

但是要记住，里卡多谬误与经济学家做的另一件事交互作用； 他们收集数据。 数据收集几乎从不是中立的活动； 它在一个理论框架内进行。 这意味着，如果收集、分类和制表数据的系统是由具有特定模型的人设计的，那么数据很可能会支持该模型。 模型的一切都将经过良好验证，基于数据，以实证为基础等等。 一切不属于模型的东西将是玩忽职守的、主观的、“难以量化”的和其他“很可能是特别辩护和虚构”的同义词。 在书中，我有一个名为“里卡多赢得争论的方式”的小标题，这就是他们如何做到的——他们收集数据。

重返主题，理解的关键在于**财务账目是企业的模型**。它们包含许多假设，其中可能最少被分析但最重要的是，“财政年度是该过程的有意义时间段”。账目中显示的是可以用数字支持的事物。其他事物则不会出现在账目中，因此也不会。 （或者在最好的情况下，只能用临时的、未经审计的数字支持，每个人都会怀疑）。

我认为这是波音公司出了什么问题的深层原因之一；麦克唐纳-道格拉斯（McDonnell-Douglas）的高管们能够用一堆硬数据支持他们的商业案例。而波音的传统高管则继续谈论文化、最佳实践和各种听起来软绵绵的事情，这些很难用模型表达。理查德学派赢得了这场争论，而灾难性的决定原来是在没有任何人意识到他们在做决策时做出的，当他们决定将财务报告系统作为管理工具时。

在《The Brompton》中，我所写的几乎都是在转述威尔·巴特勒-亚当斯（Will Butler-Adams）的话，描述了他与公司创始人就对公司改变的不同解释而发生的争论。在我的笔记中，有一句话深深地印在我的脑海中：“理解数字必须源于对业务的理解。人们在试图反过来做时往往会出错。”但这确实是他们经常做的事情。
