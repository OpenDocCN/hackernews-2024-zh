# 警方使用GPS追踪飞镖以避免危险追逐

> 来源：[https://www.thedrive.com/news/police-tag-fleeing-cars-with-gps-tracking-darts-to-avoid-dangerous-pursuits](https://www.thedrive.com/news/police-tag-fleeing-cars-with-gps-tracking-darts-to-avoid-dangerous-pursuits)

间谍电影中最常见的情节之一是主角能够从远处偷偷地用跟踪飞镖标记嫌疑人。这样，他们可以秘密地追踪坏人的行踪。现在，这不再是间谍电影的把戏，而是现实，因为警察部门已经开始用GPS追踪器标记[逃逸车辆](https://www.thedrive.com/news/this-high-speed-chase-of-a-u-haul-gmc-savana-is-pure-chaos)，从而消除了高速追逐的需要。

纽约Old Westbury警察局正在使用车载发射器，这些发射器发射带有热激活粘胶的泡沫弹丸，用于追击车辆逃逸。这些弹丸内置无线GPS追踪系统，允许警方跟踪汽车的位置。这些发射器和飞镖由[StarChase](https://www.starchase.com/products/vehicle-mounted-gps-launcher/)制造，他们还向纽约市警察局等部门租赁手持装置。这一理念的目标是消除危险且经常将公众置于危险中的车辆追逐行动。

“警方极力反对进行警车追逐行动，”据Old Westbury警察局长Stuart Cameron称，来自[*Gothamist*](https://gothamist.com/news/traffic-jams-airtags-and-gps-darts-are-all-ways-ny-nj-police-are-trying-to-stop-car-theft)。“统计数据说明一切……嫌犯受伤，警官受伤，甚至公众成员在这些追逐中受到杀害和受伤的风险。”

这正是追踪器旨在预防的情况。*AP新闻通过YouTube*

这些发射器安装在警车前部，因此警察只需追击逃逸车辆足够长的时间以标记它，然后就可以后退并跟踪其位置。压缩空气发射这些飞镖，据说它们以约30英里/小时的速度飞行，并且粘附在上面而不留下任何损害。

不仅在长岛，这些追踪飞镖也在其他地方使用。密歇根、田纳西、德克萨斯、华盛顿州的警察部门都已经开始使用它们。然而，这些追踪器在民用车辆上的使用引发了对它们是否符合宪法的质疑，一些人认为它们违反了第四修正案的权利。在2012年的最高法院案件*美国诉琼斯案*中，一致裁定未经授权对汽车进行GPS追踪构成非法搜查，违反第四修正案。然而，美国公民自由联盟（ACLU）表示，关于GPS追踪飞镖并不存在问题，"假设它们被用在大家可能想象的方式"，用ACLU高级政策分析师[杰伊·斯坦利](https://www.aclu.org/news/national-security/gps-bullets-allow-police-shoot-tracker-car)的话来说。换句话说，如果在与逃逸车辆进行追逐时发射飞镖，只要追踪器在捕捉到车辆后立即移除，其使用很可能是合法的。

GPS追踪飞镖并不是什么新鲜事物，但它们在执法部门中的普及程度正在增加。一方面，警察以这种方式追踪车辆的做法有点反乌托邦，但另一方面，如果正确使用，它们有潜力避免危险的碰撞。
