# 8年后：一位世界围棋冠军对阿尔法围棋的反思

> 来源：[https://blog.google/around-the-globe/google-asia/8-years-later-a-world-go-champions-reflections-on-alphago/](https://blog.google/around-the-globe/google-asia/8-years-later-a-world-go-champions-reflections-on-alphago/)

*编辑注：李世石是一名职业围棋选手，曾在2016年公开对阵谷歌DeepMind的人工智能系统阿尔法围棋的比赛中成名。*

作为一名世界围棋冠军，我之前只与其他人类玩家对弈过。这一切在2016年改变，当我与[阿尔法围棋](https://deepmind.google/technologies/alphago/)，谷歌的人工智能系统进行了一场五番对局。我承认，我低估了人工智能的力量，最终只赢得了我们五场比赛中的一场。

当我首次收到谷歌邀请与[阿尔法围棋对弈](https://blog.google/technology/ai/alphagos-ultimate-challenge/)时，并未意识到这是多么严肃的事情。我认为这将是一场轻松的胜利，而这只是一次随意的实验。但一旦这场比赛公开，我意识到这有多么重大。

我仍然记得当初看到阿尔法围棋（AlphaGo）出色表现时的惊叹，它的棋招如何出奇制胜。围棋是一款极度复杂的战略游戏 — 著名的比国际象棋更为复杂，共有1,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000种可能的棋盘配置。这意味着阿尔法围棋必须具备创造力，而不仅仅是计算能力。

后来，我了解到科学家们预测，人工智能在另一个十年内不会达到这种能力。今年标志着我与阿尔法围棋对局八周年纪念。在这八年中，人工智能以令人难以置信的速度发展。全球围棋选手现在已经利用阿尔法围棋尝试在这个古老的游戏中发现新的棋招、新的策略和新的思想。
