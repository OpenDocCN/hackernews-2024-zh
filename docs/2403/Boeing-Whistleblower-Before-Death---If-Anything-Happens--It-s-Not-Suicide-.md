# 波音公司举报人去世前：“如果发生什么事，那不是自杀”

> 来源：[https://futurism.com/the-byte/boeing-whistleblower-warning-not-suicide](https://futurism.com/the-byte/boeing-whistleblower-warning-not-suicide)

## "我知道他没有自杀。"

## 奇怪的原因

随着波音公司因其[飞机](https://futurism.com/the-byte/jet-tire-falls-off)的[反复](https://futurism.com/the-byte/plane-wing-breaking-footage)[故障](https://futurism.com/the-byte/boeing-plane-fire)而继续登上新闻头条，一名前员工的死亡引发的后续报道继续增加 —— 这些新报告使验尸官最初的自杀结论更加复杂化。

在南卡罗来纳州查尔斯顿的 [*ABC4 News*](https://abcnews4.com/news/local/if-anything-happens-its-not-suicide-boeing-whistleblowers-prediction-before-death-south-carolina-abc-news-4-2024) 的一次采访中，波音公司举报人约翰·巴奈特的朋友说，他警告她说可能会发生什么事情。

"我说，'你不害怕吗？'" 女士只提供了她的名字詹妮弗告诉当地广播公司。"他说，'不，我不害怕，但如果发生什么事，这不是自杀。'"

詹妮弗说，巴奈特在针对波音公司的证词前说了这些话。当时，他提到公司因他提出安全问题而报复他 —— 这也确实是他的职业安全与健康管理局（OSHA）[投诉](https://www.corporatecrimereporter.com/news/200/john-barnett-on-why-he-wont-fly-on-a-boeing-787-dreamliner/)的主题，导致他现在未完成的证词。

这名女士居住在举报人的家乡路易斯安那州，他最近几年搬回那里照顾年迈的母亲。她说，那句神秘的警告自巴奈特去世以来一直困扰着她，而查尔斯顿的验尸官说，他是自杀身亡。

"我知道他没有自杀，" 詹妮弗告诉 *ABC4*。"没可能。他太爱生活了。他太爱家人了。他太爱他的兄弟们了，不会让他们经历现在的这些痛苦。"

## 传闻

看来，她也不是唯一一个持有这种看法的人。

巴奈特去世后，《未来主义》的一份声明中，他的律师们表示，他们也没有看到任何迹象表明这位举报人可能计划自杀，并且在他的证词即将结束时，他看起来“精神状态良好”。

长期服务于波音公司质量控制的经理的亲近者并不完全同意这种看法。

在[《西雅图时报》的采访](https://www.seattletimes.com/business/boeing-787-whistleblower-found-dead-in-apparent-suicide/)中，巴内特的侄女凯特琳·吉尔斯皮说，她的“有趣叔叔”在最近几个月因前雇主因安全问题引起新闻而辞职，并且变得“焦虑和沮丧”。

“他因为波音的事情斗争很多，”告密者的侄女说。“这对他造成了重大影响。”

尽管查尔斯顿县验尸官对死因的初步尸检报告显示是自我注射的枪伤，并且当地警方根据[该市的《Live 5 News》报道](https://www.live5news.com/2024/03/12/brave-honest-man-boeing-whistleblowers-attorneys-release-statement-his-death/)发现了“某种笔记”，但当局表示他们仍在积极调查此案，并等待官方验尸官的裁定。

正如在其他[可疑的](https://www.theguardian.com/politics/2013/jul/16/david-kelly-death-10-years-on) [告密者](https://www.cbsnews.com/sacramento/news/dhs-whistleblower-philip-haney-death-ruled-suicide/) [自杀](https://foreignpolicy.com/2022/05/17/robert-mcfarlane-death-national-security-advisor-iran-contra/)案件中一样，任何人几乎不可能确切知道巴内特去世时发生了什么事情 —— 但由于有人发表了这个关于巴内特的新主张，情况变得更加复杂。
