# 新的转折点：我们将如何工作的重要变化

> 来源：[https://inflection.ai/the-new-inflection](https://inflection.ai/the-new-inflection)

从成立之初，Inflection 就秉承着一个简单的使命：为每个人创建个人智能。为此，我们训练了世界上最优秀的LLM之一，并创建了我们的第一个个人AI，Pi。Pi结合了非凡的情商和行业领先的智商，目前每周为数百万人提供服务。

作为一个AI工作室，我们长期计划将我们的技术提供给开发人员和企业。在过去的一年里，我们听到很多次，人们无法用公开可用的模型复制Pi独特的对话风格，并希望能够获得我们的模型和优化基础设施的访问权限。这为Inflection提供了巨大的机遇。

我们未来的计划是专注于我们的AI工作室业务，为商业客户精心打造、测试和优化定制的生成AI模型。我们在训练、调整和改进大型AI模型方面的成功，使我们成为全球企业的AI平台的独特优势所在。

作为其中的一部分，我们非常高兴地宣布，我们现在将在Microsoft Azure上托管Inflection-2.5，帮助我们将其交到全球创作者手中。我们还将确保它很快在其他云托管平台上推出。API本身今天还不可用，但将很快启动并运行。要注册获取早期访问权限并帮助我们测试，请在此处[注册](https://docs.google.com/forms/d/e/1FAIpQLScM9Iz1KzaRlfgDrYrldoPDnXbhO5LW3-hqmQCd56YpheEN7g/viewform)。通过API访问和与伟大客户的选择性高级合作伙伴关系，我们的AI现在可以扩展到更大的新用户群体，同时帮助将尖端AI能力交到数千名开发人员手中。

我们对API的这一新的强调也伴随着公司的重要变化。今天，我们还宣布，我们三位联合创始人中的两位，穆斯塔法和卡伦，将离开Inflection，创办Microsoft AI，这是微软的一个新部门，将汇集他们的消费者AI努力，以及Copilot、Bing和Edge。我们对他们在将Inflection推向这一阶段的所有出色工作表示感激，并祝愿他们在新的篇章中好运。

我们非常高兴地欢迎新任CEO，[肖恩·怀特](https://www.linkedin.com/in/seanwhite/)，他在技术、研究和商业的前沿领域拥有数十年的经验。他是一位具有远见的领导者，准备带领Inflection迈向新时代。我们的第三位联合创始人雷德·霍夫曼将继续担任董事会成员，并对打造人人都能使用的个人智能技术迈出下一步充满期待。

我们为我们在Pi上取得的成就感到非常自豪。服务不会立即进行任何更改，我们致力于确保用户将来可以持续访问出色的人工智能体验。我们的隐私和数据政策仍然保持不变，以确保用户的保护。与往常一样，在没有用户明确同意的情况下，不会与任何第三方共享数据。
