# Nvidia CEO 预测编程的终结 —— 詹森·黄称AI将完成工作，所以孩子们无需学习 | TechRadar

> 来源：[https://www.techradar.com/pro/nvidia-ceo-predicts-the-death-of-coding-jensen-huang-says-ai-will-do-the-work-so-kids-dont-need-to-learn](https://www.techradar.com/pro/nvidia-ceo-predicts-the-death-of-coding-jensen-huang-says-ai-will-do-the-work-so-kids-dont-need-to-learn)

[英伟达](https://www.techradar.com/sg/tag/nvidia) CEO 詹森·黄再次宣布编程的终结，但这一次是在一个潜在更具影响力的观众面前。

在迪拜世界政府峰会上发表讲话时，黄认为由于人工智能的快速发展，学习编程不再是那些希望进入科技行业的人的首要任务。

学习编程长期以来一直是许多行业领导者推动的关键技能，但黄先生的最新叙述似乎打破了这一传统。

## 编程已经是老生常谈，所以专注于农业

多亏了生成AI领域的进展，自然语言处理显示出成为编程未来的潜力，消除了年轻专业人士需要花费数百小时学习特定编程语言的必要性。

“我们的任务是创造计算技术，使得没有人需要编程。并且编程语言是人类的语言，现在全世界每个人都是程序员。这是人工智能的奇迹，”黄在峰会上说道。

学习编程的时间应该用来投资于农业、生物学、制造业和教育等行业的专业知识，Nvidia 的负责人指出。对于编程来说并非全是末日之音，因为仍然需要一些技能来知道何时何地使用AI编程。

黄说，技能提升是未来的方向，技能提升将提供如何和何时使用AI编程的知识。他进一步指出，自然语言处理将进步到只需他们的母语来编程的地步。
