# 丹麦将建造世界上最强大的AI超级计算机之一，加速解决社会挑战 - 诺和诺德基金会

> 来源：[https://novonordiskfonden.dk/en/news/denmark-to-build-one-of-the-worlds-most-powerful-ai-supercomputers-accelerating-solutions-to-societal-challenges/](https://novonordiskfonden.dk/en/news/denmark-to-build-one-of-the-worlds-most-powerful-ai-supercomputers-accelerating-solutions-to-societal-challenges/)

### 丹麦正与英伟达合作建立一个国家级AI创新中心，该中心将容纳世界上最强大的AI超级计算机之一。其目标是加速从医疗保健和生命科学到绿色转型等领域的研究和创新，支持开发解决全球最大问题的创新解决方案。

该合作由诺和诺德基金会和丹麦出口和投资基金（EIFO）领导。丹麦公共和私营部门的研究人员将能够获取优化用于大规模项目的世界领先NVIDIA AI超级计算机以及世界领先的NVIDIA软件平台、培训和专业知识。

“突破性的科学发现基于数据，AI现在为我们提供了一个前所未有的机会，加速人类和地球健康等领域的研究，” 诺和诺德基金会首席执行官 Mads Krogsgaard Thomsen 表示。“药物发现、疾病诊断和治疗以及复杂的生命科学挑战等领域，都是极端AI计算能力能够推动社会正面转型的示例。与英伟达的合作及由此产生的国家AI创新中心，可以帮助丹麦杰出的研究人员和创新者迈向新的高度。”

“在当前的地缘政治氛围下，加强我们的战略立场非常重要，” 丹麦工业、商业和金融事务部部长 Morten Bødskov 表示。“我们的政府致力于提升丹麦在尖端技术和无与伦比的数据资源方面的全球地位。我很高兴看到丹麦的公共和私营部门都在投资像超级计算机这样的转型技术，这无疑将提升我们在世界舞台上的竞争优势。”

Eviden，作为Atos集团公司和欧洲领先的先进计算领域的公司，将于2024年晚些时候交付、安装和配置名为Gefion的超级计算机，并在启动阶段提供支持。该中心——名为丹麦AI创新中心——预计将在年底前准备好进行试点项目。诺和诺德基金会已承诺向中心的初期成本投入约6亿丹麦克朗。EIFO已贡献1亿丹麦克朗。

**开启进展之门**  丹麦的新超级计算机是一台大规模[NVIDIA DGX SuperPOD](https://www.nvidia.com/en-us/data-center/dgx-superpod/)，由[NVIDIA H100 Tensor Core GPUs](https://www.nvidia.com/en-us/data-center/h100/)提供动力，并利用[NVIDIA Quantum-2 InfiniBand](https://www.nvidia.com/en-us/networking/quantum2/)进行互连。丹麦目前还没有GPU加速的超级计算机。在规划阶段，丹麦各界相关利益攸关者包括大学和部委，已经被征询意见，并且同意这是阻碍丹麦蓬勃发展的AI研究生态系统的主要障碍。

这台超级计算机和与 NVIDIA 的合作将有助于让丹麦在无数领域追求利用AI作为有价值工具的大型项目，以及在AI研究本身。在部分旗舰项目领域中，研究人员将能够与NVIDIA的专家团队合作开发解决复杂问题的解决方案。其中包括使用[NVIDIA BioNeMo](https://www.nvidia.com/en-us/clara/bionemo/)进行制药和生物技术的研究，包括蛋白质设计；加速绿色转型；以及使用[NVIDIA CUDA Quantum](https://developer.nvidia.com/cuda-quantum)进行容错（即无错误）的量子计算的研究，这是一个开源的混合量子计算平台。

“AI和加速计算将推动科学创新和发现取得重大进展，”NVIDIA的HPC和量子计算总监Tim Costa说。“由NVIDIA DGX SuperPOD 提供支持的Gefion超级计算机将成为丹麦研究人员在量子计算、生命科学和绿色转型等关键领域取得新发现的必不可少的工具。”

**‘开拓性倡议’**  该基金会成立了一家公开有限公司——丹麦AI创新中心A/S，这家公司将拥有和运营这台超级计算机，并将所有收入再投资于该倡议。EIFO是少数股东，持有15%的股份。

“这个中心将对未来数年内的基于AI的研究和商业发展产生巨大影响，” EIFO的CEO Peder Lundquist说。“这是一项对丹麦经济和社会产生更广泛影响的投资，EIFO非常高兴成为这项开拓性倡议的一部分。EIFO已经是深度技术的重要投资者，与Novo Nordisk Foundation的合作与我们在AI、量子计算、绿色转型、生命科学以及研究商业化方面的战略重点是完美契合的。”

数字房地产全球数据中心提供商 Digital Realty 将在其位于丹麦的一家AI-ready设施中托管这台超级计算机。这个数据中心采用可持续设计和建造，并且运行于100%可再生能源之上。

这台超级计算机将拥有最高级别的安全性，并支持丹麦的数据主权。一旦完全投入运行，它将能够支持涉及敏感数据的项目。中心不会永久存储任何数据，中心的用户将始终完全控制他们的数据。

正在开发差异化的付费模型，旨在平衡商业和学术使用，以满足中心当前和未来的成本。

**更多信息**  这台新超级计算机是由191个[NVIDIA DGX SuperPOD](https://www.nvidia.com/en-us/data-center/dgx-superpod/)系统组成的大规模系统，每个系统都是[NVIDIA](https://www.nvidia.com/en-us/data-center/dgx-h100/) [DGX H100系统](https://www.nvidia.com/en-us/data-center/dgx-h100/)，共计1528个[NVIDIA H100 Tensor Core GPU](https://www.nvidia.com/en-us/data-center/h100/)和382个[Intel Xeon Platinum CPU](https://www.intel.com/content/www/us/en/products/sku/231730/intel-xeon-platinum-8480c-processor-105m-cache-2-00-ghz/specifications.html)，通过[NVIDIA Quantum-2 InfiniBand](https://www.nvidia.com/en-us/networking/quantum2/)网络平台连接，具有

网络计算。该系统还将使用最先进的NVIDIA软件平台，如[NVIDIA CUDA Quantum](https://developer.nvidia.com/cuda-quantum)，具备强大的仿真工具和能力，可编程混合CPU、GPU和量子处理单元（QPU）系统。

H100 GPU的Hopper架构 - 以美国计算机科学先驱格雷斯·霍珀（Grace Hopper）命名 - 旨在解决世界上最具挑战性的AI问题，在大型语言模型如GPT-3的训练任务上比其前身快四倍。

GPU最初用于游戏和图形相关工作，在过去十年中由于深度学习和生成式AI模型的出现而迅速增长，这些单元在这些任务上比通用CPU运行得更快更高效。

[Eviden](https://eviden.com/)，欧洲高性能计算（HPC）的领导者，将把他们在HPC整合和安装方面的丰富经验和专业知识带到这个项目中。

[Digital Realty](https://www.digitalrealty.com/)致力于设计和建造可持续的数据中心，最小化对社区的影响。他们已实现他们整个欧洲和北美共置组合的100%可再生能源。Digital Realty在全球数据中心平台上超过16年的操作卓越纪录 - 超过99.999%（数据行业的测量标准通常称为五个9）的正常运行时间 - 并已承诺大量投资以继续建设和保护他们的全球数据中心足迹，让客户可以放心他们的数据的安全性和合规性。
