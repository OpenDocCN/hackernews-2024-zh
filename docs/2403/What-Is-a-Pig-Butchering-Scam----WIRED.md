# 什么是猪屠宰诈骗？| WIRED

> 来源：[https://www.wired.com/story/what-is-pig-butchering-scam/](https://www.wired.com/story/what-is-pig-butchering-scam/)

在中国，尽管进行猪屠宰欺诈需要大量的沟通和与受害者的关系建设，研究人员称，犯罪集团开发了脚本和操作手册，使他们能够将这项工作大规模地转移给经验不足的骗子或甚至是被迫劳工，他们是[人口贩运的受害者](https://www.vice.com/en/article/n7zb5d/pig-butchering-scam-cambodia-trafficking)。

"我们已经可以看到对受骗者和被迫劳工的损害和人道成本，" Rexxfield Cyber Investigations的创始人迈克尔·罗伯茨说。他一直在与猪屠宰攻击的受害者合作。"这就是为什么我们需要开始教育人们关于这种威胁，以便我们能够打破这一周期，减少对这些绑架和强迫劳工的需求。"

这个概念类似于勒索软件攻击和数字勒索，法律执法机构鼓励受害者不要支付黑客的赎金要求，以防止他们继续尝试。

自2021年起，中国政府开始[打击](https://www.reuters.com/world/china/china-arrests-over-1100-suspects-crackdown-crypto-related-money-laundering-2021-06-10/)加密货币欺诈，但犯罪分子已将他们的猪屠宰操作转移到包括柬埔寨、老挝、马来西亚和印度尼西亚在内的东南亚国家。全球各国政府已[越来越多地](https://www.fbi.gov/contact-us/field-offices/portland/news/press-releases/fbi-oregon-tech-tuesday-building-a-digital-defense-against-a-new-cryptocurrency-scam-pig-butchering)对这一威胁提出[警告](https://www.ic3.gov/Media/Y2022/PSA221003)。2021年，FBI的互联网犯罪投诉中心收到了与猪屠宰诈骗有关的超过4,300份投诉，总损失超过4.29亿美元。11月底，美国司法部[宣布](https://www.justice.gov/usao-edva/pr/court-authorizes-seizure-domains-used-furtherance-cryptocurrency-pig-butchering-scheme)，他们在2022年查封了用于猪屠宰诈骗的七个域名。

"在这种计划中，诈骗分子冒充加密货币中极为成功的交易者，诱使受害者进行所谓的加密货币投资，提供虚假回报以鼓励更多投资，" FBI在十月份的警报中[说](https://www.ic3.gov/Media/Y2022/PSA221003)。

政府官员和研究人员强调，公众教育是帮助人们避免成为猪屠宰诈骗受害者的关键组成部分。如果人们了解诈骗的显著特征并理解其背后的概念，他们就不太可能陷入其中。他们说，挑战在于接触更广泛的公众，让了解猪屠宰的人将信息传递给他们的家庭和社交圈子中的其他人。

就像浪漫骗局和其他高度个人化和剥削性攻击一样，研究人员说，猪屠宰诈骗不仅在财务上对受害者造成巨大心理负担。同时，使用[强迫劳动](https://www.propublica.org/article/human-traffickers-force-victims-into-cyberscamming)来执行猪屠宰计划增加了一层创伤，并增加了解决这一威胁的紧迫性。

“从受害者那里听到的一些故事——它会把你吞噬掉，”长期从事商业电子邮件欺诈和猪屠宰研究的首席威胁顾问Ronnie Tokazowski说。“它真的会让你非常糟糕。”
