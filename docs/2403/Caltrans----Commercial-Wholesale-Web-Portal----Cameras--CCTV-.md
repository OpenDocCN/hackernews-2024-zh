# 摄像头

## 描述

下面提供的闭路电视 (CCTV) 文件描述了每个位于加州州际公路网络上的Caltrans CCTV的位置和状态。

文件格式包括CSV、JSON、TXT和XML。每种文件格式包含相同的数据集。这些文件可供集成到您的应用程序中，并通过HTTPS协议提供。使用这些数据不收费。

## 使用条件

使用这些数据集之前，请阅读[使用条件](https://dot.ca.gov/conditions-of-use)。

加州交通摄像头视频和静态图像不予保留或归档。

## 公告

+   2020年10月20日：2020年11月9日，加州交通局将从HTTP协议迁移Caltrans批发网站门户版本2 (CWWP2) 到HTTPS协议。HTTP将在2020年11月9日之前可用，并且之后将被重定向到HTTPS。在2020年11月9日之后才能使用HTTPS。请根据需要更新使用CWWP2的客户端和应用程序以适应此重要更改。

## 数据格式和文件布局

提供四种文件格式的数据，以便轻松集成到您的应用程序中。以下是文件格式的布局：

+   [CSV](https://cwwp2.dot.ca.gov/documentation/cctv/cctv-csv-layout-example.htm) - 逗号分隔值

+   [JSON](https://cwwp2.dot.ca.gov/documentation/cctv/cctv-json-layout-example.htm) - JavaScript对象表示法

+   [TXT](https://cwwp2.dot.ca.gov/documentation/cctv/cctv-txt-layout-example.htm) - 文本文件，分隔符值为"ÿ"或0xFF

+   [XML](https://cwwp2.dot.ca.gov/documentation/cctv/cctv-xml-layout-example.htm) - 可扩展标记语言
