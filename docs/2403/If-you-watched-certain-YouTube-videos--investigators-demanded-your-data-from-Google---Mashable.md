# 如果你观看了某些YouTube视频，调查人员要求Google提供你的数据 | Mashable

> 来源：[https://mashable.com/article/google-ordered-to-hand-over-viewer-data-privacy-concerns](https://mashable.com/article/google-ordered-to-hand-over-viewer-data-privacy-concerns)

如果你曾经半开玩笑地想过你的搜索或观看历史是否会让你“被列入某种名单”，你的担忧可能不无道理。

在现在公开的法庭文件中，由[*Forbes*](https://www.forbes.com/sites/thomasbrewster/2024/03/22/feds-ordered-google-to-unmask-certain-youtube-users-critics-say-its-terrifying/?sh=68be66e1ca71 "(opens in a new window)")审查，Google被[命令透露](https://www.engadget.com/authorities-reportedly-ordered-google-to-reveal-the-identities-of-some-youtube-videos-viewers-140018019.html "(opens in a new window)")观看特定YouTube视频的账户和IP地址的姓名、地址、电话号码和用户活动，这是联邦调查员进行的一项更大刑事调查的一部分。

这些视频是由便衣警察发送给一个疑似的加密货币洗钱者，在与比特币交易员的对话中，调查人员向其发送了关于通过无人机和增强现实软件进行地图制作的公开YouTube教程的链接，*Forbes*详细描述。这些视频被观看超过30,000次，据推测，观看者中有成千上万与案件无关的用户。

YouTube的母公司[Google](https://mashable.com/category/google)被联邦调查员命令在2023年1月1日至1月8日期间悄悄地透露所有此类观看者数据，但*Forbes*无法确认Google是否已经遵守。

根据隐私专家的说法，强制的数据检索本身就令人担忧。联邦调查员辩称，这一请求在法律上是合理的，因为这些数据“将对正在进行的刑事调查具有相关性和实质性作用，包括通过提供有关施事者的身份信息，引用了全国其他警方使用的理由”。在新罕布什尔州的一个案例中，警方在调查实时通过YouTube直播的炸弹威胁时，也要求类似的数据。法庭特别要求在直播流中的特定时间戳查看观看信息。

“在所有执法要求中，我们都有严格的流程，旨在保护用户的隐私和宪法权利，同时支持执法部门的重要工作，” Google发言人Matt Bryant告诉*Forbes*。 “我们会根据法律有效性审查每一个需求，与正在发展的案例法一致，并经常反对对用户数据的过度广泛或其他不合适的要求，包括完全反对某些要求。”

然而，隐私专家对法院命令所创造的先例感到担忧，引用对第一和第四修正案保护的担忧。"[监控技术监督项目](https://www.stopspying.org/ "(在新窗口中打开)")的执行主任阿尔伯特·福克斯-卡恩告诉该出版物：“这是一个令人不安的趋势的最新章节，在这个趋势中，我们看到政府机构越来越多地将搜查令转变为数字网罗。”“这是违宪的，令人恐惧的，而且每天都在发生。”

多年来，倡导者一直呼吁谷歌在其数据共享政策上更加透明，担心持续的抗议者被公开逮捕和全州范围内对堕胎的刑事化的情况加剧。

十二月，[谷歌更新了其隐私政策](https://mashable.com/article/google-blocks-geofence-location-data-warrants)，允许用户直接将他们的位置数据保存到设备而不是云端，并缩短了此类存储的保留时间 — 新政策间接阻碍了长期以来使用[谷歌位置数据进行调查的漏洞](https://www.washingtonpost.com/technology/2023/10/24/google-privacy-police-geofence/ "(在新窗口中打开)")，这种方法被用于定位嫌疑人。

在过去的一年中，谷歌因此类担忧而被带上法庭，包括围绕[关键词搜索令的合宪性](https://www.brookings.edu/articles/keyword-search-warrants-and-the-fourth-amendment/ "(在新窗口中打开)")的两个州最高法院案件，这些令强迫网站交出个人的互联网搜索数据。
