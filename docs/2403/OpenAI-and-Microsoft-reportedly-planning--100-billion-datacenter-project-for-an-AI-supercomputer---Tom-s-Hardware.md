# OpenAI 和 Microsoft 据报道计划建设一座价值 1000 亿美元的数据中心项目，用于 AI 超级计算机 | Tom's Hardware

> 来源：[https://www.tomshardware.com/tech-industry/artificial-intelligence/openai-and-microsoft-reportedly-planning-dollar100-billion-datacenter-project-for-an-ai-supercomputer](https://www.tomshardware.com/tech-industry/artificial-intelligence/openai-and-microsoft-reportedly-planning-dollar100-billion-datacenter-project-for-an-ai-supercomputer)

[微软](https://www.tomshardware.com/tag/microsoft) 和 OpenAI 据说正在合作建设一个庞大的数据中心，用于容纳一台以 AI 为中心的超级计算机，配备数百万个 GPU。[*The Information*](https://www.theinformation.com/articles/microsoft-and-openai-plot-100-billion-stargate-ai-supercomputer?rc=bdqvyp) 报道称，该项目的成本可能“超过 1150 亿美元”，而目前在 OpenAI 内部被称为“Stargate”的超级计算机将设在美国。

报告称，Microsoft 将承担数据中心的费用，其成本可能比今天一些最大的运营中心“昂贵 100 倍”。Stargate 将成为两家公司希望在未来六年内建造的一系列数据中心项目中最大的一个，并且高管们希望在 2028 年之前让其投入运行。

根据报告称，OpenAI 和 Microsoft 正在分阶段建设这些超级计算机，而 Stargate 将是第五阶段系统。*The Information* 的消息来源称，第四阶段系统成本较低，可能最早在 2026 年推出，并可能计划在威斯康星州的 Mt. Pleasant 启动。该系统可能需要大量的电力（“至少数千兆瓦”），因此 Microsoft 和 OpenAI 正在考虑替代电源，如核能。

消息来源称，这种规模的数据中心将是一个挑战，部分原因是现有设计需要“将更多的 GPU 放入单个机架中，以提高芯片的效率和性能。” 这意味着也需要设计新的方法来保持一切的冷却。

听起来这些公司还可能利用设计阶段来摆脱对 Nvidia 的依赖。报告声称，OpenAI 希望避免在 Stargate 中使用 Nvidia 的 InfiniBand 网线，尽管 Microsoft 在当前项目中使用它们。OpenAI 声称宁愿使用以太网网线。

目前还有许多事情有待确定，因此价格和计划似乎都可能发生变化，目前尚不清楚何时会最终敲定详情。*The Information* 还指出，尚未确定这台计算机将位于何处，以及它是否将建造在单个数据中心或“在多个靠近的数据中心中”。

今年早些时候，有报道称 OpenAI CEO Sam Altman 有建立 AI 芯片的雄心，并寻求筹集多达 $7 万亿美元来建造生产基地。去年，微软推出了专门用于 AI 项目的 128 核 Arm 数据中心 CPU 和 Maia 100 GPU。还有报道称微软为 AI 数据中心[开发了自己的网络设备](https://www.tomshardware.com/desktops/servers/microsoft-develops-its-own-networking-gear-for-ai-datacenters-report)。随着 AI 技术的普及，Nvidia 的 GPU 需求量大增，因此微软和 OpenAI 这样的公司希望拥有其他选择是合情合理的。

“我们始终在计划下一代基础设施创新，以继续推动 AI 能力的前沿，” 微软首席传播官 Frank Shaw 告诉《The Information》，尽管他显然没有直接评论超级计算计划。

[微软已投入数十亿美元](https://www.tomshardware.com/news/microsoft-openai-investment-chatgpt-azure)于与 OpenAI 的合作中，主要是用于运行其模型的计算能力。如果 Stargate 或类似项目实现，这种合作将随着投资的增加而日益深入和复杂化。
