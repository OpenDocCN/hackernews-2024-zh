# EPA 禁止石棉，数十年来仍在部分国家使用 | 美联社

> 来源：[https://apnews.com/article/epa-asbestos-cancer-brakes-biden-72b0fa8b36adedaff6000034d35c2acd](https://apnews.com/article/epa-asbestos-cancer-brakes-biden-72b0fa8b36adedaff6000034d35c2acd)

华盛顿（美联社）—环境保护署周一宣布全面禁止[石棉](https://apnews.com/hub/asbestos-exposure)，这种致癌物每年导致数万美国人死亡，但仍在某些氯漂白剂、制动片等产品中使用。

[最终规则](https://www.epa.gov/newsreleases/biden-harris-administration-finalizes-ban-ongoing-uses-asbestos-protect-people-cancer)标志着环境保护局在2016年一项里程碑法律下的大规模扩展，该法律彻底改革了管理成千上万种日常产品中毒性化学品的法规，从家庭清洁剂到服装和家具。

新规定将禁止蛇纹石石棉，这是美国唯一正在使用的石棉形式。这种物质被发现存在于产品如制动衬板和垫圈中，并被用于制造氯漂白剂和氢氧化钠，也称为苛性钠，包括用于水净化的部分产品。

EPA 管理员[迈克尔·雷根](https://apnews.com/hub/michael-s-regan)称最终规则是保护公共健康的重大步骤。

“今天的禁令，环保署终于关闭了一种如此危险的化学品的大门，该化学品已经在50多个国家被禁止使用，”Regan 说。“这一历史性禁令超过30年的努力，并且要感谢国会在2016年修正《毒性物质控制法案》时所做的修订，这是美国主要的化学品使用法律。”

曝露于石棉环境已知可导致肺癌、间皮瘤和其他癌症，并且每年与超过 40,000 名美国人的死亡有关。终止石棉的持续使用是推动[拜登总统的癌症登月计划](https://nam12.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.whitehouse.gov%2Fcancermoonshot%2F&data=05%7C02%7Cmdaly%40ap.org%7C45bf7215e9e1425f634c08dc4758f20a%7Ce442e1abfd6b4ba3abf3b020eb50df37%7C1%7C0%7C638463694592445278%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C0%7C%7C%7C&sdata=jivGxATy67H%2FVp2rbpQSJnLBt4fk%2Fm8i%2F16FSMZGfbg%3D&reserved=0)，这是一个美国政府全方位结束癌症的倡议，Regan 表示。

“科学已经明确：石棉是一种已知的致癌物质，严重影响公共健康。这项行动只是我们努力保护所有美国家庭、工人和社区免受有毒化学品侵害的开始，”Regan 表示。

2016年的法律授权对成千上万种常见产品中发现的毒性化学品制定新规，包括几十年来已知会导致癌症的石棉和三氯乙烯等物质，这些化学品在联邦法律下一直未受到充分管制。这项法律称为“弗兰克·劳滕伯格化学品安全法案”，旨在清理各州对化学品的规定混乱，并更新1976年未改变40年的毒性物质控制法案。

美国环保署于1989年禁止使用石棉，但该规定在1991年的上诉法院判决中被大幅推翻，削弱了根据TSCA法案下美国环保署管理石棉或其他现有化学品健康风险的权力。2016年的法律要求美国环保署评估化学品，并制定保护措施来应对不合理风险。

石棉曾广泛用于家庭隔热材料和其他产品，但已在50多个国家被禁止使用，几十年来在美国的使用量一直在下降。目前在美国知道有进口、加工或分销的石棉仅有白石棉，主要从巴西和俄罗斯进口。它被氯碱工业使用，生产漂白剂、烧碱和其他产品。

历史上含有白石棉的大多数消费产品已停止生产。

虽然氯是水处理中常用的消毒剂，但美国仅有八家仍在使用石棉隔膜生产氯和氢氧化钠的氯碱厂。这些厂大多位于路易斯安那州和德克萨斯州。

美国环保署表示，石棉隔膜的使用量已经在下降，现在仅占美国氯碱生产的不到三分之一。

美国环保署的规定将禁止石棉用于氯碱的进口，该规定发布后将逐步实施禁止氯碱使用，预计过渡期将为五年或更长，以提供“合理的过渡时期”。

对大多数其他用途的石棉禁令将在两年内生效。

代表350个公共污水处理机构的全国清洁水协会在一份声明中表示支持“美国环保署努力摆脱石棉产品”，并将与该机构合作跟踪规则的实施情况。在最终规则宣布之前，该协会警告称，对石棉的立即禁令几乎肯定会导致水处理部门使用的氯和其他消毒和处理化学品短缺和价格上涨。

化学工业最大的游说团体美国化学理事会表示，需要15年的过渡期来避免氯和氢氧化钠供应的重大中断。

禁止在油田制动块、售后市场汽车制动器及衬板以及其他垫圈中使用石棉将在六个月内生效。

根据EPA规定，含石棉的板状垫片可在南卡罗来纳州萨凡纳河站的美国能源部中使用到2037年，以确保核材料的安全处置可以按计划继续进行。

环境工作组的高级副总裁Scott Faber，这个推动禁止石棉的倡导组织，对EPA的行动表示赞赏。

“长期以来，污染者一直被允许制造、使用和释放石棉和PFAS等有毒物质，而不考虑我们的健康，”Faber说。“多亏了拜登政府EPA的领导，这些日子终于结束了。”

另外，EPA还在评估石棉在老建筑物（包括学校和工业场所）中的所谓“遗留用途”，以确定可能存在的公共健康风险。预计将在年底前完成最终风险评估。

俄勒冈州民主党参议员Jeff Merkley表示，美国在石棉问题上“终于开始赶上”世界其他国家。“然而，在淘汰其他危险的石棉纤维方面，这不可能是路的终点，”他说。“在为我们的健康提供更强保护方面，国会有责任发挥作用。”
