# 2024 年财务报告和筹款活动

Zig 软件基金会是一家 501(c)(3)非营利组织，我很自豪地说，它**非常高效地利用了金钱资源**。与我们许多同行不同，我们的主要开支是直接支付给贡献者的费用，用于改进 Zig 项目。

不要轻易相信我的话 - 我们来看一些数字。

## 2023 年支出

| 支出名称 | 2023 年支出 | 描述 |
| --- | --- | --- |
| 承包商 | $308,102.61 | 直接支付给在 Zig 上工作的贡献者，费率为每小时$60。 |

| 员工 | $102,000.00 | ZSF 只有一名员工，即您亲爱的安德鲁·凯利，担任首席软件工程师。我作为 Zig 软件基金会主席的职责完全是义务性工作；我没有因担任董事会成员而获得补偿。ZSF 董事会的另外两名成员[决定了我的薪水](https://docs.google.com/document/d/1wPQtJxIgCo7SReJev-H7OZGgYt0k8LqECc4G4QYT0aE)，为每年$159,790，与当时纽约市首席软件工程师的中位数薪资相匹配。

我从未收到过全额薪水。撰写本文时，我接受的年薪为$108,000（税前），并将其余款项捐赠给 ZSF。过去，我捐赠了更大比例的款项。我认为将来能够得到全额补偿将是件好事。|

**我们在 2023 年花费了 92%的资金用于支付贡献者的时间。**

目前为止一切顺利。你可以看到我们一直在努力花费我们尊贵捐赠者的资金来推进使命宣言。

然而，如果我们观察 2023 年捐款趋势，我们会发现总体上有一个缓慢的下降。这可能是因为我们在*要钱*方面偷懒了。

## 捐款的请求

我们有非常有才华的 Zig 核心团队成员想要续签合同，还有其他人对首次获得有偿工作感兴趣。

为了做到这一点，**我们需要更多的定期捐款**。我个人不喜欢要钱，但考虑到我们的用户和贡献者的利益，不这样做是不负责任的。

**如果你能的话，请注册月度捐款**。我们首选的捐款方式是[通过 Every.org](https://www.every.org/zig-software-foundation-inc/f/help-zig-stay-indepe)。作为一家同样是 501(c)(3) 非营利组织，他们无缝管理礼品收据，并且不像 GitHub 目前正在做的那样转向人工智能，这实在让我们感到恐慌。

| 公司 | 联系我们，以每月捐款为交换，在 ziglang.org 上放置你的标志。 |
| --- | --- |
| 员工 | 请检查你的公司是否匹配对慈善机构的捐款，比如 Zig 软件基金会。这个 2 倍乘数产生了巨大的影响。我们已经在系统中了。 |
| 风险投资家 | 我们知道有一些初创公司押注 Zig 作为他们构建明天关键基础设施的语言和工具链。帮助 Zig 软件基金会更快地达到 v1.0 是你可以做出的最有效的资本使用之一，以提升你的投资组合。 |
| 个人 | 你能捐出每月 $5-10 吗？这是我们最喜欢的捐赠方式，因为它有助于使 ZSF 的收入多样化，使我们免受任何单一方的不当影响。 |

非常感谢所有慷慨捐款支持我们事业的人。我们共同为用户服务！
