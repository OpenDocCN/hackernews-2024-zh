# 卡特·冯 D 赢得了关于未经许可使用迈尔斯·戴维斯照片纹身的审判

在洛杉矶市中心的陪审团只用了几个小时就决定，被指称侵权的纹身与照片并不具有实质相似性。

洛杉矶（CN）——知名纹身艺术家卡特·冯 D 在一场版权侵权审判中取得了胜利，该案涉及她未经许可在一位朋友的手臂上纹下迈尔斯·戴维斯的照片。

上周五，洛杉矶市中心的陪审团只用了几个小时就驳回了摄影师杰夫·塞德里克的索赔。他起诉冯 D，因为她没有就 1989 年他拍摄的爵士乐传奇人物举起食指示意“安静”的照片向他寻求许可。

陪审团得出结论，这个纹身与原始照片并不实质相似，并且任何在社交媒体上展示冯 D 在纹身上工作的照片都是合理使用。

罗伯特·艾伦，塞德里克的律师，表示裁决似乎是仓促的决定，他们将提出上诉。

“实质相似性的问题本不应该提交给陪审团，”艾伦说。“那本应该作为法律问题来决定。”

塞德里克曾要求约 4.5 万美元的补偿金，以赔偿故意侵权，包括纹身和艺术家在社交媒体上展示她在纹身上工作的帖子，或者约 15 万美元的法定赔偿金。

艾伦说，这个案子从来不是关于金钱，而是关于保护所有视觉艺术家的权利。他说，如果纹身被认为与塞德里克的照片实质相似，那么没有一个可见艺术家的知识产权是安全的。

凯瑟琳·冯·德拉谢恩伯格，艺名卡特·冯 D，通过她在真人秀节目《迈阿密纹身》和《洛杉矶纹身》中的露面而声名鹊起，后者在好莱坞的她的 High Voltage Tattoo 店拍摄。现在是一位全职妈妈的冯 D 在周三作证说，她已经有十多年没有为任何人纹过纹身，只为朋友免费做过工作。

“我很高兴这件事结束了，”冯 D 在裁决后说道。“如果我们没有反击，对已经处境艰难的行业会造成很大的伤害。”

迈尔斯·戴维斯的纹身是给冯 D 的朋友布雷克·法默的礼物，他是一位灯光技术员，曾在 2017 年为她的化妆品业务工作过一些拍摄。在与自己也是小号手的法默交谈后，了解到迈尔斯·戴维斯对他有多重要，她提出为音乐家制作纹身。

Von D 告诉陪审团，纹身界没有人会为他们创作的参考照片获得许可证。她坚持认为她对迈尔斯·戴维斯的使用是“公平使用”，因为这是她对图像的诠释，并且与 Sedlik 的作品完全不同的目的。

Farmer 向 Von D 提供了成为版权诉讼主题的照片。然而，Sedlik 在未经许可使用他的作品方面非常严格；他经常扫描互联网以找到侵权者。他作证说，在 2014 年，他追踪到另一位纹身艺术家在社交媒体上发布了一张基于同一张迈尔斯·戴维斯照片的纹身。

那位艺术家在 Sedlik 联系他后得到了一个免费的事后许可证，同意免除 5000 美元的许可费作为“专业礼貌”，因为，他作证说，该艺术家为没有事先寻求许可证而道歉并表现出悔意。

Von D 在判决后表示，她可能再也不会创作纹身了，因为她的心被这次事件击碎了。她补充道，她可能会为在审判中作证的 Farmer 例外，因为这起诉讼污染了他的迈尔斯·戴维斯纹身。
