# 引入一个新的 Copilot 密钥，开启人工智能 Windows PC 的一年 | Windows Experience 博客

> 来源：[https://blogs.windows.com/windowsexperience/2024/01/04/introducing-a-new-copilot-key-to-kick-off-the-year-of-ai-powered-windows-pcs/](https://blogs.windows.com/windowsexperience/2024/01/04/introducing-a-new-copilot-key-to-kick-off-the-year-of-ai-powered-windows-pcs/)

今天，我们很高兴迈出下一个重要的步骤，并向 Windows 11 PC 引入一个新的 Copilot 密钥。 在这个新的一年里，我们将迈向一个更个人化和智能化的计算未来的重大转变，其中人工智能将被无缝地融入 Windows，从系统到硅再到硬件。 这不仅会简化人们的计算体验，还会增强它，使 2024 年成为人工智能 PC 的年份。

由人工智能创新推动的平台转变持续快速而强劲地增长，从根本上改变了我们在工作和生活中使用技术的方式。 在微软，我们致力于赋予个人和组织在这个人工智能新时代中适应和蓬勃发展的能力。

[在过去的一年里](https://blogs.microsoft.com/blog/2023/12/05/celebrating-the-first-year-of-copilot-with-significant-new-innovations/)，我们一直站在这一转变的最前沿，通过 Copilot，您的日常人工智能伴侣，创新并改进我们的产品体验。 从 [用 Copilot 在必应中改变搜索方式](https://blogs.microsoft.com/blog/2023/02/07/reinventing-search-with-a-new-ai-powered-microsoft-bing-and-edge-your-copilot-for-the-web/)，以及 [使用 Copilot 提高 Microsoft 365 的生产力](https://blogs.microsoft.com/blog/2023/03/16/introducing-microsoft-365-copilot-your-copilot-for-work/)，再到 [在 Windows 中重新想象人们如何完成任务](https://blogs.windows.com/windowsexperience/2023/09/26/the-most-personal-windows-11-experience-begins-rolling-out-today/)，我们倾听了反馈并加倍努力创建一个每天都能帮助人们的体验。 我们还从我们的硅合作伙伴 [AMD](https://www.amd.com/en/newsroom/press-releases/2023-12-6-amd-extends-mobile-pc-leadership-with-amd-ryzen-8.html)，[英特尔](https://www.intel.com/content/www/us/en/newsroom/news/ai-everywhere-core-ultra-5th-gen-xeon-news.html#gs.1s9vqy) 和 [高通](https://www.qualcomm.com/news/releases/2023/10/qualcomm-brings-record-breaking-generative-ai-for-devices-at-sna) 那里看到了令人难以置信的势头，他们都向世界展示了他们的最新硅创新，为 Windows PC 上的新 AI 体验打开了新的可能性。 我们正在共同建立新的系统架构，以推动新的 Windows AI 体验，将 GPU、CPU、NPU 和云相结合。

我们被人们在过去一年中分享的 Copilot 如何启发和赋予他们力量的故事所激励，我们知道我们的工作永远没有完成，我们会不懈努力，不断为我们的客户创新。

Copilot 键的引入标志着近三十年来 Windows PC 键盘的第一次重大变化。我们相信它将使人们更容易参与人工智能转型。Copilot 键与 Windows 键一样成为 PC 键盘的核心部分，当按下时，新的键将调用 Windows 体验中的 Copilot，使其在日常中与 Copilot 交互更加流畅。近 30 年前，我们引入了 Windows 键到 PC 键盘，使全世界的人们都能与 Windows 互动。我们将其视为我们在 Windows 旅程中的又一个变革时刻，Copilot 将成为 PC 上人工智能世界的入口点。

在接下来的几天里，包括 CES 在内，您将开始在我们的生态合作伙伴的许多新的 Windows 11 PC 上看到 Copilot 键，可用性将从本月末开始延续到春季，包括即将推出的 Surface 设备。

我们对我们的旅程迈出了下一步感到兴奋。现在开始使用 Copilot 从未有过如此好的时机，[无论是将您的想法变成歌曲](https://blogs.bing.com/search/december-2023/Turn-your-ideas-into-songs-with-Suno-on-Microsoft-Copilot)，创建美丽的图像和精致的草稿，还是调整您的 PC 设置，Copilot 都是您工作和生活中的日常人工智能伴侣。要了解更多信息，请[访问此链接](https://www.microsoft.com/en-us/copilot)。

当我们踏上新的一年时，我们充满了乐观和激情。我们将继续打造 Windows，使其成为最佳人工智能体验的目的地。这将需要一个操作系统来模糊本地和云处理之间的界线。前方的一年将是非同寻常的！

*免责声明：*

***Copilot 功能交付和可用性的时间因市场和设备而异。需要 Microsoft 帐户登录。* *当设备上不可用或未启用 Windows 上的 Copilot 时，按下 Copilot 键将启动 Windows 搜索。**
