# 三星预计芯片销售疲软，利润下降85%

> 来源：[https://koreajoongangdaily.joins.com/news/2024-01-09/business/industry/Samsung-forecasts-85-drop-in-profit-for-last-year-amid-chip-slump/1954130](https://koreajoongangdaily.joins.com/news/2024-01-09/business/industry/Samsung-forecasts-85-drop-in-profit-for-last-year-amid-chip-slump/1954130)

# Samsung forecasts 85% drop in profit as chip sales falter

一名工人在三星电子总部所在地京畿道水原市的公司标志旁给花坛浇水。[路透社]

三星电子预计去年营业利润同比下降84.9%，因半导体行业的严重周期性下滑侵蚀了其盈利能力。

总部位于京畿道水原市的科技巨头周二在初步业绩报告中宣布，预计将发布6.5万亿韩元（50亿美元）的营业利润，这个数字是自2008年以来最低的，低于市场普遍预期的7.5万亿韩元。

公司预计年销售额为258.2万亿韩元，同比下降14.6%。这一预测低于市场预估的262万亿韩元。

预测初步预测未包括详细的细分。

但银行证券（KB Securities）分析师金东元估计，三星的存储芯片业务全年产生了合计15万亿韩元的营业亏损，尽管智能手机和面板制造业务的利润帮助公司保持盈利。

韩国国内外的专家和市场分析师均认为，去年的当前经济低迷已触底，并且预计今年在先进人工智能领域的强劲需求将会反弹。

“存储芯片业务在2023年产生了15万亿韩元的营业亏损，但很可能会在2024年通过增加总额29万亿韩元而录得14万亿韩元的利润。”金教授说。

市场变化反映了三星在去年第四季度减少库存和增加发货方面取得的进展。

动态随机存取存储器和NAND闪存芯片的出货量根据位增长增加了30%，而库存水平已比去年年末下降了50%以上。据金教授称，与去年年初相比，存货水平在去年底的时候已经下降了50%以上。

仅在第四季度，三星电子的营业利润就下降了35%，至2.8万亿韩元，而销售额下降了4.9%，至67万亿韩元，据初步业绩结果显示。

经济证券（Mirae Asset Securities）分析师金永勋估计，三星的半导体业务在这个季度录得了2.2万亿韩元的营业亏损，而智能手机和网络设备部门则录得了2.5万亿韩元的利润。面板制造业务带来了1.9万亿韩元的利润，而消费电子业务则带来了4000亿韩元的利润。这是根据Mirae Asset Securities的金氏所言。

他指出，芯片行业的复苏速度较预期缓慢，因为发货量和价格的增长主要集中在少数几款性能出色的半导体产品上，而整体电子设备的需求仍然疲软。

“虽然发货量和平均售价有所改善，但规模较预期偏弱，因为一批高端产品先行发货，” 金氏说。

最终结果将在本月底公布。
