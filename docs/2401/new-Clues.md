# 新线索

> 来源：[https://www.cluetrain.com/newclues/](https://www.cluetrain.com/newclues/)

曾经我们年轻时在花园里……互联网是我们，连接着我们。

互联网不是由铜线、光纤、无线电波，甚至管道构成的。

我们用来连接互联网的设备并不是互联网。

Verizon、Comcast、AT&T、Deutsche Telekom 和中国电信不拥有互联网。Facebook、Google 和亚马逊也不是互联网的君主，他们的手下或算法也不是。地球上的政府和他们的贸易协会没有[网络用户的同意](http://consentofthenetworked.com/)来作为主权者横行于网络之上。

我们共同拥有并且不归任何人所有的是互联网。

从我们和我们在其上构建的东西中，互联网获得了所有的价值。

网络是我们的，由我们创造，为我们服务。

互联网是我们的。

互联网什么也不是，也没有任何目的。

互联网不是一个东西，就像重力不是一个东西一样。两者都把我们聚集在一起。

互联网根本就不是一样东西。[在其基础上](http://www.worldofends.com/)，互联网是一系列协议，我们中间那些怪胎（愿他们的名字永存）称之为“协议”，但在当今的气氛下，我们可能称之为“戒律”。

其中最重要的是：你的网络应该在不偏不倚地移动所有数据包到达它们的目的地，而不会基于来源、源、内容或意图而产生偏见或延迟。

因此，这第一个戒律向每一种想法、应用、业务、探索、恶习以及任何东西开放了互联网。

自语言以来，没有一种工具具有如此广泛的用途。

这意味着互联网不是为了特定的任何事物。不是为了社交网络，不是为了文件，不是为了广告，不是为了商业，不是为了教育，不是为了色情，不是为了任何事物。它专门为一切而设计。

[优化](http://books.google.com/books?id=9lTtWQlLkwwC&pg=PA56&lpg=PA56&dq=optimizing+de-optimizing+david+reed&source=bl&ots=_mqGSLoqSV&sig=TWO4uUTWkzGPDBGmwyS0uqBz0NA&hl=en&sa=X&ei=Nm5lVN-hJtbdsASnzoKQAQ&ved=0CCAQ6AEwAA#v=onepage&q=optimizing%20de-optimizing%20david%20reed&f=false) 互联网为某一目的进行优化，却对其他所有目的进行了非优化。

互联网就像重力一样，吸引力不加区分。它将我们所有人，善良的和邪恶的，都吸引在一起。

网络不是内容。

互联网上有很多好内容。但是天呐，互联网并不是由内容构成的。

一个少年的第一首诗，长久保守的秘密的愉悦释放，一个抖颤的手画的精美草图，一个厌恶人民声音的政权下的博客帖子——这些人都不是坐下来写内容的。

我们用带引号的单词“内容”了吗？我们觉得很肮脏。

互联网不是一个媒介。

网络不是一个媒介，就像对话不是一个媒介一样。

在网络上，我们就是媒介。我们是移动消息的人。每当我们发帖、转推、发送邮件中的链接或在社交网络上发布链接时，我们都这样做。

与媒介不同，你和我在传递的信息上留下了指纹，有时甚至咬痕。我们告诉人们我们为什么要发送它。我们与之争论。我们加入一个笑话。我们剪掉我们不喜欢的部分。我们让这些消息成为我们自己的。

每当我们通过网络传递一条消息，它都带着我们的一点点痕迹。

只有当我们以人类关心某种无数方式中的一种方式关心某事时，我们才会将一条消息传递到这个“媒介”中。

关心——重要性——是互联网的动力。

互联网为我们提供了一个共同的地方，在这里我们可以做自己，与其他人一起欣赏我们的差异。

没有人拥有那个地方。每个人都可以使用它。任何人都可以改进它。

这就是开放互联网。有些战争的起因比这更小。
