# “不可能”创建像ChatGPT这样的AI工具，OpenAI表示 | OpenAI | 卫报

> 来源：[https://www.theguardian.com/technology/2024/jan/08/ai-tools-chatgpt-copyrighted-material-openai](https://www.theguardian.com/technology/2024/jan/08/ai-tools-chatgpt-copyrighted-material-openai)

开发者OpenAI表示，如果没有访问版权资料，将无法创建像[其开创性的聊天机器人](https://www.theguardian.com/technology/2023/feb/02/chatgpt-100-million-users-open-ai-fastest-growing-app)ChatGPT这样的工具，因为对人工智能公司使用的内容施加的压力越来越大。

诸如ChatGPT之类的聊天机器人以及稳定扩散等图像生成器是通过从互联网上获取的大量数据进行“训练”的，其中很大一部分受版权保护 - 这是对某人的作品未经许可而使用的[法律保护](https://www.gov.uk/copyright)。

上个月，[纽约时报起诉OpenAI和微软](https://www.theguardian.com/media/2023/dec/27/new-york-times-openai-microsoft-lawsuit)，微软是OpenAI的主要投资者并在其产品中使用其工具，指控它们“非法使用”其作品来创建产品。

在一份[提交给上议院通信和数字选择委员会的文件](https://committees.parliament.uk/writtenevidence/126981/pdf/)中，OpenAI表示，它无法训练大型语言模型[如其GPT-4模型](https://www.theguardian.com/technology/2023/mar/14/chat-gpt-4-new-model)——ChatGPT背后的技术——没有访问版权作品。

“因为今天的版权几乎涵盖了各种人类表达形式，包括博客文章、照片、论坛帖子、软件代码片段和政府文件，所以要在不使用版权材料的情况下训练今天领先的人工智能模型是不可能的，”OpenAI在其提交的文件中表示，首次由《每日电讯报》报道。

它补充说，将训练材料限制在已过版权的书籍和图纸上将产生不完善的人工智能系统：“将训练数据限制在一百多年前创作的公共领域书籍和图纸上可能会产生有趣的实验，但不会提供符合今天公民需求的人工智能系统。”

在周一发布在其网站上的一篇[博客文章](https://openai.com/blog/openai-and-journalism)中回应纽约时报的诉讼，OpenAI表示：“我们支持新闻报道，与新闻机构合作，认为纽约时报的诉讼毫无根据。”

公司之前表示尊重“内容创作者和所有者的权利”。人工智能公司使用版权材料的辩护倾向于依赖“公平使用”法律原则，该原则允许在某些情况下使用内容而无需寻求所有者的许可。在其提交文件中，OpenAI表示其认为“从法律上讲，版权法并不禁止训练”。

纽约时报的诉讼是对 OpenAI 提出的众多其他法律投诉的跟进。约翰·格里沙姆（John Grisham）、乔迪·皮科特（Jodi Picoult）和乔治·R·R·马丁（George RR Martin）等 17 名作者在 9 月对 OpenAI 提起诉讼，指控其存在“大规模系统性盗窃”。

通讯推广后

拥有全球最大照片库之一的盖蒂图片公司正在美国和英格兰威尔士起诉 Stable Diffusion 的创始人 Stability AI，指控其涉嫌侵犯版权。在美国，包括环球音乐在内的一组音乐出版商正在起诉 Anthropic，即亚马逊支持的 [Claude chatbot](https://www.theguardian.com/technology/2023/jul/12/claude-2-anthropic-launches-chatbot-rival-chatgpt) 背后的公司，指控其滥用“不计其数”的受版权保护的歌词来训练其模型。

在其提交给上议院的文件中，针对关于人工智能安全性的问题，OpenAI 表示支持对其安全措施进行独立分析。该提交表示支持对人工智能系统进行“红队测试”，即第三方研究人员通过模拟恶意行为者的行为来测试产品的安全性。

OpenAI 是同意在其部署之前和之后与政府合作进行 [最强大模型的安全测试](https://www.theguardian.com/technology/2023/nov/02/top-tech-firms-to-let-governments-vet-ai-tools-sunak-says-at-safety-summit) 的公司之一，此前一年在英国举行的全球安全峰会上达成了一项协议。
