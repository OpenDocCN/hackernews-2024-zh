# 数学填字游戏：在线数字填字拼图

> 来源：[https://scientific.place/math-crossword/](https://scientific.place/math-crossword/)

一款简单的益智游戏，测试并提高你的基本数学技能。将数字拖放到正确的位置，以进入下一个级别。适合学生、儿童和成人。

方程式的方向是什么？

方程式从上到下、从左到右。对涉及减法和除法的方程式很重要。

有多少个级别？

几乎有无限多个级别！没错。完成一个拼图后，系统会即时为你生成一个新的随机拼图。
