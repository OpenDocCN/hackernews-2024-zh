从左边选择一个平台。

### 源代码

Anki 的源代码可在 GitHub 上获得。在 docs/ 文件夹中有构建的说明。如果你在构建系统方面遇到问题，请告诉我们，但请注意期望你能自行解决基本问题。如果你没有编程经验，请使用打包版本。

如果在运行源代码的 Anki 时遇到 bug，请检查是否可以在打包版本中重现它们，因为 Qt 版本或系统中的其他库可能是一些问题的原因。

### 开发
