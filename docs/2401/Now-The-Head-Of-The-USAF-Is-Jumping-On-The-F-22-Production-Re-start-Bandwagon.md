# 现在，美国空军的头目也加入了重新启动F-22生产的行列。

> 来源：[https://www.thedrive.com/the-war-zone/3687/now-the-head-of-the-usaf-is-jumping-on-the-f-22-production-re-start-bandwagon](https://www.thedrive.com/the-war-zone/3687/now-the-head-of-the-usaf-is-jumping-on-the-f-22-production-re-start-bandwagon)

“我认为这不是一个疯狂的想法，我是说F-22的成功以及飞行员和机组的能力都是非常杰出的。我认为已经证明了这种飞机正是每个人都希望的那种……我们正在以新的不同方式使用它，它取得了令人瞩目的成功，它的潜力真的非常非常惊人。因此，回过头来看，并且肯定提出这样的想法：嗯，你能再建造更多吗？这不是一个疯狂的想法。”

*洛克希德·马丁*

在五角大楼内，关于建造更多猛禽战机的话题在过去十年中一直是禁忌，自国防部在2000年代后期围绕F-35计划形成合围以来。追求与F-35的存在竞争的任何事物都被视为异端邪说。

结束F-22生产的讨论在国防部长罗伯特·盖茨、空军参谋长迈克尔·莫斯利和空军部长迈克尔·韦恩之间是一个尤为突出的痛点。莫斯利和韦恩最终被革职，至少部分原因是因为他们反对盖茨推动仅在187架生产型号结束猛禽生产的举动，这一举动在某种程度上得到了布什政府的暗示支持，并得到了奥巴马政府的全力支持。

*洛克希德·马丁*

美国空军即使在增购更多F-22战机方面改变了态度，这可能意味着美国空军F-35A总购买量的下降，由1763架飞机减少。这两种飞机的制造商洛克希德·马丁公司表示，这一举措将导致更高的F-35单位成本，并可能使该计划陷入死亡螺旋。（是的，洛克希德不会因为他们制造了这两种战机而受益。）

F-35代表着历史上最大规模的国防采购项目，这将为21世纪后半叶提供大量资金流。相比之下，即使追求高度升级版本的飞机，生产几百架F-22也只是一笔收入的微不足道的一部分。

“[46个州生产](http://www.reuters.com/article/us-usa-fighter-f35-insight-idUSBRE92E10R20130315)”的F-35并不是唯一阻碍F-22生产线重新启动的计划。其他像B-21轰炸机这样的重点项目也是障碍之一。甚至美国空军的财政难以持续的愿望清单上都有一个白日梦般的第六代战斗机，当你正在生产升级版F-22时，这将更加难以推动。

*美国空军*

这也值得注意的是，韦尔什在离任美国空军首席的职务时发表了这一声明。真正重要的是他的继任者大卫·戈德费恩将军的想法。到目前为止，他对于其他关键的美国空军武器系统，比如与ISIS作战中非常重要的A-10，发表了正确的言论，他在 *[Military Times](http://www.airforcetimes.com/story/military/2016/04/26/goldfein-nominated-next-chief-staff/83542108/)* 中表示：

"当我们做出淘汰A-10的决定时，我们在ISIL组织出现之前做出了这些决定。当时我们并不在伊拉克，我们正在从阿富汗撤军。我们在讨论淘汰A-10的时间段内，并没有一个重新崛起的俄罗斯，所以当假设发生变化且不实现时，我们必须足够灵活以做出调整。"

*Lockheed Martin* 是指美国的一家公司，专门从事航空航天和国防方面的业务。
