# 方块小子

> 来源：[`azich.org/blockdude/`](https://azich.org/blockdude/)

触摸、向左、向右、向上或向下拖动，然后释放以移动角色。

**左手势**

: 向左移动并面向左

**右手势**

: 向右移动并面向右

*以下内容适用于你的角色面朝的方向*。

**下手势**

: 拾取或放下方块

**上手势**

: 向上移动
