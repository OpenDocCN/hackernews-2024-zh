# FTC禁止X-Mode出售手机位置数据，并要求公司删除已收集的数据 | TechCrunch

> 来源：[https://techcrunch.com/2024/01/09/ftc-x-mode-ban-delete-location-data/](https://techcrunch.com/2024/01/09/ftc-x-mode-ban-delete-location-data/)

美国联邦贸易委员会已禁止数据经纪人X-Mode Social共享或出售用户的敏感位置数据，[联邦监管机构于周二表示](https://www.ftc.gov/news-events/news/press-releases/2024/01/ftc-order-prohibits-data-broker-x-mode-social-outlogic-selling-sensitive-location-data)。

这一首例协议禁止了现在称为Outlogic的X-Mode共享和出售用户的敏感信息。协议还要求数据经纪人删除或销毁其先前收集的所有位置数据，以及该数据产生的任何产品，除非该公司获得了消费者的同意或确保数据已被去标识化。

X-Mode购买并出售从普通手机应用程序中收集的位置数据的访问权限。虽然X-Mode只是数十亿美元数据经纪行业中的众多组织之一，但其因[向美国政府和军方承包商出售美国人过去活动的商业位置数据](https://www.nytimes.com/2021/01/22/us/politics/dia-surveillance-data.html)而受到审查，并因[向军方承包商出售美国人的商业位置数据](https://www.vice.com/en/article/jgqm5x/us-military-location-data-xmode-locate-x)而受到审查。

不久之后，苹果和谷歌告知开发者[从他们的应用中移除X-Mode](https://www.wsj.com/articles/apple-and-google-to-stop-x-mode-from-collecting-location-data-from-users-phones-11607549061)或面临被应用商店禁止的风险。

FTC声称，X-Mode出售的精准位置数据可用于追踪人们对敏感位置的访问，例如医疗和生殖健康诊所、宗教场所和家庭暴力庇护所的访问。

监管机构还声称，数据经纪人未能从其出售给第三方的原始位置数据中删除敏感位置，并未实施“合理或适当的保障措施”来防止此精准位置数据的下游使用。对于至少一份合同，FTC表示，X-Mode向一家未透露名称的私人临床研究公司提供了有关曾在俄亥俄州哥伦布市一定地理区域内访问某些医疗设施、药店或专科输液中心的消费者信息。

FTC表示，X-Mode还未能确保其自有应用程序——Drunk Mode和Walk Against Humanity的用户充分了解其精准位置数据的使用方式。

“通过X-Mode/Outlogic出售的位置数据所揭示的信息不仅侵犯了消费者的隐私，而且使他们面临潜在的歧视、身体暴力、情感困扰和其他危害，”FTC在一份声明中表示。

“地理位置数据不仅可以揭示一个人居住的地方和与之相处的人，还可以揭示他们寻求的医疗治疗和他们信仰的地方等信息，”FTC 主席 Lina M. Khan 说道。“FTC 对 X-Mode 的行动表明，企业无权随意市场和出售美国人的敏感位置数据。”

“通过首次禁止敏感位置数据的使用和出售，FTC 继续着对入侵式数据经纪商和不受检查的企业监视的重要工作，”Khan 表示。

根据 FTC 的命令，X-Mode 还必须实施程序，确保其位置数据的接收方不将数据与为 LGBTQIA+ 人群提供服务的位置相关联，为消费者提供一种简单的方式撤回其位置数据的收集和使用的同意，并建立和实施一项全面的隐私计划，以保护消费者个人信息的隐私。

由代表 Outlogic 的公关公司 Broadsheet 发给 TechCrunch 的声明写道：“我们不同意 FTC 新闻稿的含义。经过长时间的调查，FTC 没有发现任何数据滥用的情况，也没有提出任何这样的指控。自成立以来，X-Mode 对所有数据客户都实施了严格的合同条款，禁止他们将其数据与诸如医疗保健设施之类的敏感位置相关联。遵守 FTC 新引入的政策将通过实施额外的技术流程来确保，并不需要对业务或产品进行任何重大变更。”

首先披露 X-Mode 向美国军方承包商出售位置数据的参议员 Ron Wyden 对 FTC 的调查结果表示：“我赞扬 FTC 采取了强硬行动，追究这个不正当的位置数据经纪商对美国人位置数据出售的责任。”
