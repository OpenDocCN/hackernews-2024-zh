# 英特尔的下一代 Arrow Lake 处理器可能没有超线程核心 — 泄漏指向 24 个 CPU 核心，DDR5-6400 支持和新的 800 系列芯片组 | Tom's Hardware

> 来源：[`www.tomshardware.com/pc-components/cpus/intels-next-gen-arrow-lake-cpus-might-come-without-hyperthreaded-cores-leak-points-to-24-cpu-cores-ddr5-6400-support-and-a-new-800-series-chipset`](https://www.tomshardware.com/pc-components/cpus/intels-next-gen-arrow-lake-cpus-might-come-without-hyperthreaded-cores-leak-points-to-24-cpu-cores-ddr5-6400-support-and-a-new-800-series-chipset)

Intel 的下一代桌面处理器，代号为 Arrow Lake-S，根据一份据称由著名硬件爆料人[@YuuKi_AnS](https://twitter.com/yuuki_ans/status/1748640540084912422)发布的英特尔文件摘录，与现有的第 14 代酷睿“猛禽湖刷新”处理器相比，不会增加核心数量（该爆料人已删除帖子，但[VideoCardz](https://videocardz.com/newz/leaked-documents-list-intel-arrow-lake-s-with-8p16e-cores-125w-tdp-full-800-series-chipset-details)已重新发布）。令人惊讶的是，这些文件指出英特尔的超线程将不会出现在新处理器中，这或许证实了长期以来的传言。该消息来源在泄露英特尔文件方面有着无可挑剔的记录，但请注意，这些规格说明可能会因为它们概述了 Alpha 前硅片而发生变化。

文件显示，英特尔计划在其 Arrow Lake-S 桌面处理器系列（采用 LGA-1851 封装）中提供三种类型的硅片，包括高性能（P）和高效率（E）核心，包括三种芯片配置：8P + 16E 核心，6P + 16E 核心和 6P + 8E 核心。

假设信息准确，英特尔的下一代桌面处理器不仅不会增加核心数量，与猛禽湖刷新-S 处理器相比，也将不再配备超线程 P 核心，这是英特尔现有处理器阵容的一个特点。长期以来一直有传言称英特尔将采用一种新的方法来处理其 P 核心，放弃超线程，这允许两个线程在单个 CPU 核心上运行，而采用一种[在专利中概述的新方法](https://www.freepatentsonline.com/y2023/0168898.html)。此列表肯定会为这些预测增加实质性的依据，但这可能仅仅是因为这是非常早期的 Alpha 前硅片。最好对此持保留态度；文件指出 P 核心在 BIOS 中被禁用，因为存在初始化问题，所以一切皆有可能。

Arrow Lake 处理器将采用新的微架构和更快的 DDR5-6400 内存。值得注意的是，这个系列还将终止英特尔对 DDR4 内存的支持，这意味着全球两大 x86 芯片制造商都将全面采用 DDR5。

当我们谈论英特尔的 Arrow Lake 产品时，我们应该记住英特尔打算用来制造这些产品的工艺技术：英特尔 20A 工艺技术（2 纳米级）。这个制造节点引入了 RibbonFET 全围栅极晶体管和 PowerVia 背面供电网络。这两项创新可以使英特尔优化功耗，提高性能并增加晶体管密度。这将如何影响插座式英特尔 Arrow Lake 处理器的性能尚待观察，但由于工艺节点带来的创新，新的 CPU 显然将比现有产品具有一些优势。

此外，下一代台式机 CPU 设计支持 16 条 PCIe 通道用于图形卡，以及两条 x4 通道用于固态硬盘。与此同时，整个平台支持以 UHB20 速率的 DisplayPort 输出和 Thunderbolt 4 连接器，假设主板配备适当的海登桥重新计时器（这并不便宜）。

就芯片组方面而言，英特尔的 800 系列平台控制器中枢几乎支持所有可以想象的东西，包括 NVMe、SATA、PCIe 和 USB 3.2 Gen2x2。
