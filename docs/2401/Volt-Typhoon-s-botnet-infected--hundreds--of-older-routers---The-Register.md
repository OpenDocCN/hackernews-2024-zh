# 电压台风的僵尸网络感染了‘数百’台旧路由器 • The Register

> 来源：[`www.theregister.com/2024/01/31/volt_typhoon_botnet/`](https://www.theregister.com/2024/01/31/volt_typhoon_botnet/)

中国的电压台风间谍感染了“数百”台过时的思科和网件设备，以便指示这些设备攻击美国的关键基础设施设施，司法部表示。

周二[消息爆出](https://www.theregister.com/2024/01/30/fbi_china_volt/)，联邦政府封锁了设立在美国境内末日、基于美国的小型办公室/家庭办公室路由器上的恶意僵尸网络。现在有关 FBI 小组如何渗透该操作并在远程擦除 KV 僵尸之前收集关键数据的更多细节已经公布，根据去年上个月 FBI 在南部德克萨斯州法院提交并今天发布的四个搜查令([5018](https://lnks.gd/l/eyJhbGciOiJIUzI1NiJ9.eyJidWxsZXRpbl9saW5rX2lkIjoxMDIsInVyaSI6ImJwMjpjbGljayIsInVybCI6Imh0dHBzOi8vd3d3Lmp1c3RpY2UuZ292L29wYS9tZWRpYS8xMzM2NDIxL2RsP2lubGluZT0mdXRtX21lZGl1bT1lbWFpbCZ1dG1fc291cmNlPWdvdmRlbGl2ZXJ5IiwiYnVsbGV0aW5faWQiOiIyMDI0MDEzMS44OTQzMDk5MSJ9.4LRHVZyUDYfMnRWC7s_-GUsAOhl4SewCG7GXGj4tAX8/s/3078670442/br/236285728656-l)、[5530](https://lnks.gd/l/eyJhbGciOiJIUzI1NiJ9.eyJidWxsZXRpbl9saW5rX2lkIjoxMDMsInVyaSI6ImJwMjpjbGljayIsInVybCI6Imh0dHBzOi8vd3d3Lmp1c3RpY2UuZ292L29wYS9tZWRpYS8xMzM2NDE2L2RsP2lubGluZT0mdXRtX21lZGl1bT1lbWFpbCZ1dG1fc291cmNlPWdvdmRlbGl2ZXJ5IiwiYnVsbGV0aW5faWQiOiIyMDI0MDEzMS44OTQzMDk5MSJ9.8vmYc17z664gSNfykf8ncM2yRyRspFYF6SCIHiTZhb8/s/3078670442/br/236285728656-l)、[5451](https://lnks.gd/l/eyJhbGciOiJIUzI1NiJ9.eyJidWxsZXRpbl9saW5rX2lkIjoxMDQsInVyaSI6ImJwMjpjbGljayIsInVybCI6Imh0dHBzOi8vd3d3Lmp1c3RpY2UuZ292L29wYS9tZWRpYS8xMzM2NDA2L2RsP2lubGluZT0mdXRtX21lZGl1bT1lbWFpbCZ1dG1fc291cmNlPWdvdmRlbGl2ZXJ5IiwiYnVsbGV0aW5faWQiOiIyMDI0MDEzMS44OTQzMDk5MSJ9.d-xmdG8esN2ZFRwqLXHNvwIo6ZpVS-2HQ5js3wwfJo8/s/3078670442/br/236285728656-l) 和 [5432](https://lnks.gd/l/eyJhbGciOiJIUzI1NiJ9.eyJidWxsZXRpbl9saW5rX2lkIjoxMDUsInVyaSI6ImJwMjpjbGljayIsInVybCI6Imh0dHBzOi8vd3d3Lmp1c3RpY2UuZ292L29wYS9tZWRpYS8xMzM2NDExL2RsP2lubGluZT0mdXRtX21lZGl1bT1lbWFpbCZ1dG1fc291cmNlPWdvdmRlbGl2ZXJ5IiwiYnVsbGV0aW5faWQiOiIyMDI0MDEzMS44OTQzMDk5MSJ9.xD5SrU9r17bUh7t5QaI4TWGGTTTlIfn6D8a3QecwFJY/s/3078670442/br/236285728656-l))。"

"中国的黑客正在瞄准美国的民用关键基础设施，为了在冲突发生时对美国公民和社区造成真实世界的伤害，" FBI 局长克里斯托弗·雷在一份[声明](https://www.justice.gov/usao-sdtx/pr/us-government-disrupts-botnet-peoples-republic-china-used-conceal-hacking-critical)中说道。"电压台风恶意软件使中国得以在瞄准我们的通信、能源、交通和水利部门时隐藏行踪。"

联邦政府声称，中央王国的网络间谍向易受攻击的路由器下载了虚拟私人网络模块，并建立了加密通信渠道，远程控制僵尸网络，并有可能命令设备执行攻击并隐藏他们的活动。具体来说：据称 Volt Typhoon 使用美国的路由器和 IP 地址 [瞄准](https://www.theregister.com/2024/01/31/critical_infrastructure_hacking/) 美国的关键基础设施。

搜查令允许执法部门远程安装软件到路由器上，以搜索、然后查扣或复制有关非法活动的信息，然后从受感染设备中清除恶意软件。

为了做到这一点——并且限制联邦政府的搜索范围仅限于感染了远程控制僵尸网络恶意软件的路由器——根据搜查令，FBI 向受感染的路由器发送了特定的 KV 僵尸网络命令，以收集关于这些节点的“非内容信息”。

这包括被感染的路由器用于与其他节点通信的 IP 地址、端口号，以及每个节点的父节点使用的 IP 地址和端口号，以及命令和控制节点的数据。

“未感染 KV 僵尸网络恶意软件的路由器将不会接收或响应此命令”，法庭文件声称。

联邦政府以及五眼联盟国家的外国机构 [于 2023 年 5 月首次发出警告](https://www.theregister.com/2023/05/25/china_volt_typhoon_attacks/) 关于这种威胁。

此外，美国政府的网络安全机构和 FBI [发布了警报](https://www.cisa.gov/news-events/alerts/2024/01/31/cisa-and-fbi-release-secure-design-alert-urging-manufacturers-eliminate-defects-soho-routers) 敦促制造商消除 SOHO 路由器 Web 管理界面中的缺陷。据联邦政府称，这包括自动更新功能、仅在局域网端口上定位 Web 管理界面，以及需要明确的手动覆盖来删除安全设置。
