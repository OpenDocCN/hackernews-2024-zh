# 公平和善良并非软弱 - 杰瑞·麦格芬

> 来源：[https://gerrymcgovern.com/fairness-and-kindness-are-not-weaknesses/](https://gerrymcgovern.com/fairness-and-kindness-are-not-weaknesses/)

您离睡在街头要比睡在亿万富翁的豪宅近得多。而对着渴望睡在亿万富翁的豪宅中的嫉妒之情正在摧毁我们的环境。我们已经允许贪婪和残忍统治，而他们无尽的贪婪和残忍将不可避免地导致环境破坏。

亿万富翁排放的温室气体比普通人多一百万倍。二十家大型科技公司统治着世界，而每年，125名超富豪个人排放的温室气体相当于8500万辆汽车。7700万人排放的气候变化产生的排放量比最贫穷的50亿人多，而超级精英的排放量比与1.5°C升温限制兼容的CO2多77倍。到2019年，美国的顶级1%排放的排放量比最贫困的50%多，全球顶级1%的人排放的CO2是最底层1%的1000倍。顶级1%的精英每年排放110吨CO2，而最贫穷的50%只有1.5吨。在1995年至2021年间，最富有的1%占据了38%的总财富增长，而最底层50%只占了2.3%。60%的新财富被顶级1%的精英占据。如果世界上最富有的10%被迫将他们的CO2水平削减到欧盟平均水平，全球CO2排放量可以减少32％。

马里居民的平均碳排放量与英国的水壶相当，尼日利亚居民的平均碳排放量与美国的冰箱相当。生活在撒哈拉以南非洲的普通人每年排放约0.6吨二氧化碳，而居住在美国的人平均排放14.5吨。为了防止全球升温超过1.5°C，我们的平均排放量不应超过每人每年两吨二氧化碳。全球仅有1%的人口年收入超过58000美元，而每年19300美元的收入使您成为全球最富有的10%。

比尔·盖茨每年排放7500吨CO2，而罗曼·阿布拉莫维奇排放32000吨。根据心理研究，亿万富翁是权谋、精神病和自恋的，是恶毒的自我推销者，情感冷淡、不停地撒谎和高度攻击性。他们喜欢私人喷气机，其销售额在2023年达到历史最高水平。私人喷气机数量在20年内翻了一番多。 

美利坚合众国仅占世界人口的5%，却造成了超过四分之一的环境破坏。美国最富有的10%人口每年每人排放56.5吨CO2，这比欧洲最富有的10%人口的排放量高出一倍多，几乎是中国最富有的10%人口排放量的两倍。美国是有史以来最大的排放国和环境破坏者。美国负责40%的气候崩溃。欧盟负责29%。全球北部负责92%的过量全球碳排放。

最贫困的50%人口仅造成12%的CO2排放，但由于气候变化将经历75%的相对收入损失，同时仅拥有世界财富的2%。与此同时，最富有的10%人口造成48%的总CO2排放，拥有76%的一切，并且由于他们造成的破坏，只会经历3%的相对收入损失。我们为什么要让贪婪和残酷的人破坏我们的环境？

全球能源消耗几乎是满足所有基本需求所需能源的三倍。我们必须将无谓需求的能源与真正需求的能源分开。我们是要开着两吨重的SUV驶向世界末日，一边吸烟电子烟一边制作TikTok视频吗？全球最富有的1%人口消耗的能源量相当于为17亿人提供体面生活所需的能源。如果中产阶级放弃他们的无谓需求，50亿人可以过上有尊严、可持续的生活。

总之，破坏环境的是富人而不是穷人。如果你有足够的特权来阅读这篇文章，那么你——就像我一样——目前是问题的一部分，而不是解决方案的一部分。解决方案是什么？富人和中产阶级人士大幅减少消费，并在可持续环境的框架内为所有人建立公平的生活标准。
