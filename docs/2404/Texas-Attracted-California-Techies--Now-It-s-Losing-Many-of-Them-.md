# 德克萨斯州吸引了加州的技术人才。现在它正在失去其中许多人。

> 来源：[https://www.texasmonthly.com/news-politics/austin-texas-tech-bust-oracle-tesla/](https://www.texasmonthly.com/news-politics/austin-texas-tech-bust-oracle-tesla/)

回顾2020年的美好岁月，那是我们所有人都怀念的一年，德克萨斯州和加利福尼亚州之间持续战争中开启了一个新的火点。技术人员开始离开“税加利福尼亚”，大量迁移到独行星州。敌方主要报纸《洛杉矶时报》[担心](https://www.latimes.com/politics/story/2022-02-09/silicon-valleys-tech-monopoly-is-over) 硅谷的“垄断”已经结束，并猜测奥斯汀是否是“未来”。州长格雷格·阿伯特 [宣布](https://twitter.com/GregAbbott_TX/status/1337518818013405190) 德克萨斯州真正是“商业、就业和机会的乐土”。

在2020年关于奥斯汀崛起的一系列报道中，始终提到两个关键证据。当年，科技巨头甲骨文将其总部搬到了奥斯汀，此前已在城市南岸的城市湖畔建立了一个庞大的园区；而[埃隆·马斯克](https://www.texasmonthly.com/news-politics/elon-musk-tesla-electricity-power-grid/)则在城市外建立了一个巨大的赛博卡车工厂。奥斯汀当地政府帮助甲骨文[获得了有价值的湖滨房地产](https://www.dallasnews.com/business/2024/04/02/elon-musk-boosts-headcount-by-86-at-biggest-site-of-his-texas-empire/#:~:text=with%2022%2C777%20surpassed%20grocery%20store,number%20of%20workers%20in%20Austin.&text=Elon%20Musk%20is%20now%20officially,Y%20SUVs%20and%20angular%20Cybertrucks.)，并向特斯拉提供了约6000万美元的税收减免，包括来自[历史上困难重重的](https://www.austinchronicle.com/news/2023-08-04/the-state-wants-to-give-more-schools-f-ratings-to-what-end/) 德尔瓦雷尔学区的5000万美元。新设施受到州官员的欢迎，成为“德克萨斯奇迹”依然存在的证据。阿伯特州长去年自豪地宣称奥斯汀是“全球领先科技公司的首选目的地”。

本周出现了一个重大情节转折：甲骨文宣布将总部迁至纳什维尔，而特斯拉——首都最大的私营雇主——在一份令人失望的财报后宣布将裁减其奥斯汀工厂近2,700名工人。德克萨斯并非真正有错。制造商商业软件的甲骨文称纳什维尔作为美国医疗保健行业中心的实力，虽然公司从城市和田纳西州获得近25亿美元的税收减免和激励措施，显然也有帮助。与此同时，特斯拉由于Cybertruck出现重大质量问题而全国裁员，导致其奥斯汀生产设施的未来岌岌可危。据报道，该城市在汽车制造方面的首秀是一款显然会在雨中生锈的车辆。马斯克曾承诺的工厂综合体最近利用新的州法律豁免自己免受奥斯汀的环境法规约束。

然而，尽管严峻的财务现实解释了总部迁移和裁员，这些消息也表明了文化变革的时刻：奥斯汀在2020年突然间变得火爆，但至少在某种程度上，现在已经不再如此。在2020年的疯狂夏天中，科技传教士乔·朗斯代尔写道，德克萨斯是人类自由斗争的新前线，并誓言在奥斯汀外建立一个新城市，作为像其他几个世纪以来来到该州的乌托邦者一样的颠覆者的避风港。但事实证明，在COVID-19大流行后，现状更具实用性。当面对面工作恢复时，与帕洛阿尔托的交往对科技领袖们变得更加宝贵。尽管朗斯代尔仍然留在那里，但许多新到达者开始返回家园。

在十二月，或许并非巧合，正值奥斯汀历史上最炎热的夏季之后，《TechCrunch》报道说，初创公司正在逃离这座城市，称其“[失去了吸引力](https://techcrunch.com/2023/12/07/is-the-texas-boom-town-of-austin-losing-its-luster/)”。毫不奇怪，一些在疫情期间搬到这里的加利福尼亚人意识到，他们用伊甸园般的天气换来了每年110度的夏季和无所得税的情况，他们决定所得税并不那么糟糕。这在科技行业的财务状况中有所体现。风险投资家在2021年为奥斯汀的初创公司投资了**67.5亿美元**，但到了2023年，他们只投资了**38亿美元**。 （在整个行业普遍紧张的背景下，帕洛阿尔托的融资也有所下降，但湾区依然是霸主，2023年那里的公司筹集了超过**600亿美元**的投资。）

甲骨文在不到四年后将总部从德克萨斯搬出，这提醒我们，试图套住全球公司就像试图与风搏斗，这个世纪初曾经大肆宣扬的经济发展激励措施只能起到有限的作用。奥斯汀很可能会继续增长，但这特定炒作周期的结束恰逢该州政治身份的转变。

* * *

**在本世纪的头两个十年里**，作为这个州政客解释的，作为德克萨斯人的含义很大程度上是与加利福尼亚竞争的一种感觉。州的“胜利”通常是通过德克萨斯能够从黄金州诱使多少新居民和公司搬到这里来进行计数的。

罗伯特·派瑞州长是与这场竞赛相关的主要人物：他不停地吹嘘他与来自收入更高、税收更重的州的CEO达成的交易，甚至还吹嘘从旧金山到达拉斯的[U-Haul租车价格的上涨](https://www.politifact.com/factchecks/2014/mar/12/rick-perry/rick-perry-correct-trucks-cost-half-much-rent-goin/)。他将德克萨斯经济的增长归功于他长达14年的任期中的主要成就。阿伯特接过了接力棒，参议员泰德·克鲁兹也试图分享这份荣耀。这个州的政治文化是围绕着不惜一切吸引新移民，即使这个努力有点荒谬。

2014年，时髦的Sriracha辣椒酱生产商Huy Fong Foods在洛杉矶附近的加州伊文戴尔市与当地政府发生了麻烦。问题在于，居民称工厂的烤辣椒烟雾令他们受尽折磨，市政府起诉该公司试图让其控制排放。（该公司否认责任，并在其总部外悬挂了一幅横幅，上面写着“这里不制造催泪瓦斯”，但空气质量监管机构承诺展开调查。）“这种事情不会在德克萨斯发生”，当时的德克萨斯州代表杰森·维拉巴如此说道，他是来自达拉斯地区的温和派共和党人。他率领一个德克萨斯州立法者代表团——两名州代表、一名州参议员，以及州长办公室、总检察长办公室和德克萨斯农业部门的官员——请求Huy Fong将其总部迁至德克萨斯的某个地方，那里的居民会感激不尽。

代表团未能成功：工厂的所有者利用德克萨斯人与加州当局谈判，并把两组人都打发走了。然而，德克萨斯州几乎每个全州政客都认为这种努力是英勇的，即使只是戏剧化的：州的角色是鼓励新企业和新移民来到这里，即使这些移民引发了[讨厌的笑话](https://www.austinmonthly.com/the-story-of-the-sassy-shirt/)。

这种信念已经消散。德克萨斯州右翼改变了州官员对新移民和经济增长前景的看法。这种转变在大流行爆发初期加速，当时德克萨斯州政客更常表达对加利福尼亚人和外地人的敌意。共和党人现在承诺保护德克萨斯免受加利福尼亚人的侵害，与赢得他们的欢迎一样频繁。佩里在今年春季的一次集会上指出，德克萨斯人经常接近他，责怪他——即使是开玩笑——因为现在他们不得不应对所有外地人。上周五，由来自[康涅狄格州的外来者](https://www.texasmonthly.com/news-politics/matt-rinaldi-texas-gop/)管理的德克萨斯共和党发送了一封筹款呼吁信，开头宣布“乔·拜登正在向德克萨斯州涌入移民和加利福尼亚人”。

当代共和党领袖，尤其是来自巴尔的摩的副州长丹·帕特里克，更不倾向于将经济增长视为一个目标。该州共和党越来越怀疑商业利益，这些利益往往支持温和派共和党人和对文化战争持谨慎态度。当里克·佩里离任时，立法机构瘫痪了他最珍视的经济发展项目：专款基金。

最近几届会议上，围绕2001年创建的第313章税收激励计划的未来展开了长时间的争斗，该计划授权对大企业进行财产税减免以促使其迁至此处。议会允许该计划在2022年到期，然后以一种新的、规模缩小的形式将其恢复，该形式今年生效。虽然阿伯特将半导体厂迁往纽约的损失归咎于该计划的失效，帕特里克对第313章的终结感到相当自豪，他称其被“滥用”。

得克萨斯州议会的情绪不仅仅改变了。虽然许多得克萨斯人不喜欢加利福尼亚人，但那些将这个词用作咒骂的人通常是在描述所有新来者，就像那些称棕色移民为“墨西哥人”一样。尽管没有民意调查可以告诉你这一点，尽管那些对新来者感到愤怒的人可能自己并不会做出这种联系，但可以很有理由认为对拉丁美洲移民和美国外来者的恐惧源于一种普遍的经济不稳定感。这个世纪以来，德克萨斯州整体上变得更加富裕。但这种财富分配不均。州内城市的生活成本飞涨：根据联邦储备数据，与洛杉矶相比，奥斯汀的房价在本世纪上涨得更快。在过去的十年里，DFW地区的房价翻了一番多，而中位收入仅增长了45%。对于想要成为房主的人来说，芝加哥现在是一个更好的选择。

与此同时，得克萨斯州并不是一个低税收、低服务的州，这是常见的看法。事实上，它是一个高税收、低服务的州：我们可能没有所得税，但至少一项研究发现，我们在全国十大总税负中名列前茅，房产税占据了大部分差距。然而，随着州预算的增长，州政府服务的质量并没有相应提高。老年得克萨斯人感到在他们已经生活了几十年的城市里压力很大。年轻的得克萨斯人上学时往往接受不到标准的教育，接受不到标准的医疗保健，然后无法在他们出生的城市中买得起房子。得克萨斯州的政治越来越集中于管理由此产生的怨恨，而最简单的方法就是责怪外来者。

但这让州领导人陷入了困境。该州的经济模型依赖于廉价的移民劳动力、外州的技术工人和新资本的常规注入。移民者获得学士学位的可能性是本地德克萨斯人的两倍。他们住在由非法移民建造的房屋里，吃着由非法移民服务的餐馆里的食物。由此带来的增长对每个人都有好处，但对某些人的好处更大。德克萨斯大学奥斯汀分校公共事务学院的教授史蒂文·佩迪戈在2021年告诉《德克萨斯月刊》，“所有的船都会上浮，但并不是所有的船都能上浮得足够高、足够快，以应对生活成本的上涨”。所以即使在好时期，你也会看到日益增长的怨恨。由于人口激增而导致的问题在预算盈余与预算盈余之间的犹豫中没有得到解决。

更聪明的反移民保守派提出，通过利用外国劳动力解决美国的技术劳动力短缺问题，会抑制本国人才培养的进程。我们之所以引进医生和工程师，是因为这比培养我们自己的人更便宜。不管这是不是事实，德克萨斯州正以类似的方式对待“加利福尼亚人”。该州在发展自己的劳动力方面做得很糟糕。我们为什么要费心培养，当我们可以从其他州挖来工人呢？

但是没有繁荣是永久的，甲骨文的离开提醒了我们，人们可以双向流动。当涌入减少时会发生什么情况？我们将只剩下自己萎缩的增长机制？嗯，我们可以像美国人一直做的那样。我们可以把U-Haul挂在Cybertruck上，踏上旅途。
