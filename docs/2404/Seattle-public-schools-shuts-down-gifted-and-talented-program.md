# 西雅图公立学校关闭了天才和才能项目

> 来源：[https://nypost.com/2024/04/03/us-news/seattle-public-schools-shuts-down-gifted-and-talented-program/](https://nypost.com/2024/04/03/us-news/seattle-public-schools-shuts-down-gifted-and-talented-program/)

西雅图公立学校正在拆除其天才和才能项目，管理员们认为该项目过度饱和了白人和亚裔学生，而更倾向于一个更“包容、公平和具有文化敏感性”的项目。

由于种族不平等问题，[学区注意到，](https://www.seattleschools.org/departments/advanced-learning/services-and-programs/service-delivery/)西雅图学区在2021-22学年开始逐步取消其高才能群体学校和高级学生课堂。

到2027-28学年，该项目将完全停止存在，到2024-25学年，每所学校都将有一种新的适合所有人的丰富模型。

“这个项目并没有消失，而是变得更好了”，学校官员在学区网站上表示。

“它将更具包容性、公平性和文化敏感性。

“特别是历史上被排除在外的学生现在将有与其他学生相同的服务机会，并获得他们需要的支持和丰富的成长机会。”

西雅图公立学校正在拆除其天才和才能项目，以支持更“包容、公平和具有文化敏感性”的项目。Google St View

目前的丰富课程仅允许在标准化考试中排名前2%的学生加入高才能群体，接受丰富的学习。

然后，学生将被分到三所小学、五所中学和三所高中之一。

但在2020年，西雅图学校董事会投票决定终止该项目，此前[2018年的一项调查](https://www.parentmap.com/article/equity-teams-seattle-hcc-racial-disparity-gifted)发现，高才能群体中的学生中，13%是多种族，11.8%是亚裔，3.7%是西班牙裔，仅有1.6%是黑人。

根据新的项目，被称为高才能邻里学校模型，教师将需要为他们所有的20到30名学生设计个性化的学习计划。Louis-Photo – stock.adobe.com

近70%的学生是白人。

“数据显示，在我们的城市中……主要是白人儿童比其他文化和种族更有天赋，我们知道这绝对不是真的”，当时该区的学生支持服务主任卡里·汉森告诉Parent Map。

在新计划下，被称为高能力邻里学校模型，教师将需要为他们所有20到30名学生设计个性化学习计划 —— 他们认为，由于学区面临1.04亿美元的预算赤字，他们没有时间或资源来完成这项任务，根据[《西雅图时报》报道](https://www.seattletimes.com/education-lab/why-seattle-public-schools-is-closing-its-highly-capable-cohort-program/)。

学区表示正在努力为教师提供课程和教学指导，以确保其顺利实施，但根据2020年的估计，全面富 enrichment-for-all 计划将使学区在前三年内花费110万美元。

一位教师表示，她担心在新计划下，教授具有不同能力的学生数学会变得更加困难，并且全班教学方法无法充分准备学生参加高级教学数学和科学课程。

家长们还表达了他们的担忧，担心新模式可能导致孩子们被忽视。

当学校董事会决定解散该计划时，副主席Chandra Hampson指责黑人学生家长在这个同伴团体中“把一个非常小的社区当作象征”，西雅图公立学校

“我觉得也许极端情况下的孩子们都将受到忽视，” Erika Ruberry对《西雅图时报》说。

Karen Stukovsky，有三个孩子在天才计划中的母亲，补充道，每位教师“只能做到有限的差异化教学”。

“你们有些孩子可能几乎看不懂，有些孩子却在一年级或幼儿园时就读《哈利·波特》，” 她说道。

“你们打算怎么样既让这些孩子达到年级水平，又能挑战那些已经远远超过年级水平的孩子？”

一些黑人学生的家长甚至反对结束这一计划。

“我要求您考虑一下，如果您批准这项新计划，会对已经在HCC计划中的少数族裔造成多大的不公平，” 一位父亲在2020年学校董事会会议上说道，根据[《The Stranger》报道](https://www.thestranger.com/slog/2020/01/24/42658513/seattle-school-board-takes-steps-to-dismantle-gifted-program)。

“这个计划对黑人儿童，尤其是黑人男孩，有益多过对他们同龄人。”

但是当时的学校董事会副主席Chandra Hampson回击说：“在现有的同伴团体中，这是一种将一个非常小的有色社区象征化的高超技巧。”

然而，在过去几年里，越来越多的少数族裔学生加入了高能力同伴团体的行列。

在2022-23学年，学生中有52%是白人，16%是亚裔，3.4%是黑人，据《西雅图时报》报道。

新项目的支持者表示，这将创建更强的社区意识，因为所有学生来自同一地理区域。

“他们带来了他们的家庭经验和文化，这真的很独特”，View Ridge小学校长Rina Geoghagan告诉《西雅图时报》。

“会完美吗？不会。但任何时候发生变化，都不会是完美的。”
