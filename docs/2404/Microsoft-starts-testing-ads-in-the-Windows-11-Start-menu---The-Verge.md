# 微软开始在 Windows 11 的开始菜单中测试广告 - The Verge

> 来源：[https://www.theverge.com/2024/4/12/24128640/microsoft-windows-11-start-menu-ads-app-recommendations](https://www.theverge.com/2024/4/12/24128640/microsoft-windows-11-start-menu-ads-app-recommendations)

微软表示，它正在开始在 Windows 11 的开始菜单内测试广告。软件制造商将使用开始菜单中的“推荐”部分（通常显示文件推荐）来建议来自 Microsoft Store 的应用程序。

“这将仅适用于位于美国的 Windows Insider 测试人员的 Beta 通道，并且不适用于商业设备（由组织管理的设备），” [微软在一篇博客文章中](https://blogs.windows.com/windows-insider/2024/04/12/announcing-windows-11-insider-preview-build-22635-3495-beta-channel/) 表示。

应用推广可以在 Windows 11 的设置部分禁用，但微软似乎会默认启用这些功能。微软正在寻求关于这些变化的反馈，因此如果足够多的反馈表明这些广告不会受欢迎，公司可能会决定在 Windows 11 的开发版本中取消这些广告。

*广告是微软用红框突出显示的那一个。*

图片：Microsoft

微软去年开始在 Windows 11 的文件资源管理器内测试[广告](/2022/3/15/22979251/microsoft-file-explorer-ads-windows-11-testing)，然后在 Windows 11 的测试版本中禁用了该实验。微软在 Windows 内进行广告实验已超过 10 年。在 Windows 10 锁屏和开始菜单中已经有了宣传位，所以在 Windows 11 中看到它们出现并不令人意外。

现在，轮到 Windows 的测试人员提交反馈了，以查看微软是否可能被迫在发布的 Windows 11 版本中不包含这些广告。
