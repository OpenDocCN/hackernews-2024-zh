# Holodeck：语言引导下的3D具体化人工智能环境生成

> 来源：[https://yueyang1996.github.io/holodeck/](https://yueyang1996.github.io/holodeck/)

3D模拟环境在具体化人工智能中扮演了关键角色，但它们的创建需要专业知识和大量手动工作，限制了它们的多样性和范围。为了减轻这一限制，我们提出了Holodeck，这是一个系统，可以完全自动地生成与用户提供的提示相匹配的3D环境。Holodeck能够生成多样化的场景，如游戏厅、水疗中心和博物馆，并能根据风格调整设计，捕捉复杂查询的语义，比如“带猫的研究员公寓”和“星球大战迷教授的办公室”。Holodeck利用大型语言模型（GPT-4）来获取关于场景外观的常识性知识，并使用Objaverse的大量3D资产来装饰场景，填充各种物体。为了解决正确定位物体的挑战，我们提示GPT-4生成物体之间的空间关系约束，并优化布局以满足这些约束。我们的大规模人类评估显示，评注者更倾向于Holodeck而不是在住宅场景中手动设计的程序化基准，并且Holodeck能够为多种场景类型生成高质量的输出。我们还展示了Holodeck在具体化人工智能中的一个激动人心的应用，即训练代理程序在像音乐室和托儿所这样的新场景中导航，而无需人工构建的数据，这是在开发通用目的的具体化代理中的重要一步。
