# Pitcairn岛移民

> 来源：[https://www.immigration.pn](https://www.immigration.pn)

Pitcairn岛最著名的可能是它位于南太平洋的偏远岛屿，1790年，HMAV Bounty号的叛变者在从塔希提岛航行几天后与他们的塔希提伴侣最终在此找到了避难所。

很少有人知道Pitcairn岛是英国的海外领土，是由四个小而多样化、独特的岛屿组成的一个大型专属经济区，其原始水域是许多特有物种的栖息地，并且自2016年9月起受到世界上最大的海洋保护区之一（836,000平方公里）的保护。如今，岛上的文化融合了英国、波利尼西亚和更广泛的太平洋根源，这是其历史和当代联系的遗产。

Pitcairn是Pitcairn群岛唯一有人居住的岛屿。它位于塔希提岛东南2170公里、复活节岛西1570公里和新西兰东北5310公里处。该群岛的其他岛屿，Henderson、Ducie和Oeno，支持脆弱的生态系统和独特的栖息地。Henderson岛自1988年以来一直是联合国教科文组织世界遗产地。所有四个岛屿都属于亚热带气候。特别是Pitcairn岛，拥有肥沃的火山土壤和茂密的植被。

这些岛屿只能通过海路进入。每季度供应船将必需品和材料运送到岛上。供应船还提供Pitcairn与法属波利尼西亚的Mangareva之间的客运服务。查看船运时间表[这里](https://www.visitpitcairn.pn/shipping-schedule)

Henderson、Ducie或Oeno没有定期的客运服务。这意味着Pitcairn群岛是世界上最偏远的群岛之一。
