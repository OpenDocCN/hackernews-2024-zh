# OfOne 创始软件工程师 | Y Combinator

> 来源：[https://www.ycombinator.com/companies/ofone/jobs/u2E2fCX-founding-software-engineer](https://www.ycombinator.com/companies/ofone/jobs/u2E2fCX-founding-software-engineer)

**关于OfOne：**

在 OfOne，我们通过语音AI重塑了驾驶通道体验。我们已经帮助了排名前50的餐厅连锁更高效地运营，并提升了他们的客户体验。

我们正在寻找一位准备进入语音AI领域的创始软件工程师。我们提供一个机会，可以开发触及数百万人、推动数十亿经济影响并处于实时AI应用前沿的软件。

我们行动迅速，信奉协作胜过竞争，并得到了像SignalFire、Correlation Ventures和Y Combinator等主要风险投资的支持。

如果这听起来适合你，请加入我们在旧金山市中心的办公室，让我们一起打造伟大的事业。

**职责：**

+   设计、开发和维护我们自动化订单平台的可扩展高性能后端系统

+   与我们的团队密切合作，确保我们的后端与我们的ML模型和客户设备的无缝集成

+   监控并优化我们的后端系统在生产环境中的性能

+   构建并维护与第三方餐厅软件系统（如POS、忠诚度、支付网关和客户数据平台）的集成

+   实施最佳实践以确保可靠、安全和可维护的系统设计、代码质量和测试

**要求：**

+   计算机科学、工程学或相关领域的学士或硕士学位

+   4+年在生产环境中开发、实施和维护后端基础设施的实际经验

+   有构建和部署可扩展高性能后端系统的成功经验记录

+   有使用AWS和API设计等云基础设施和部署技术的经验

+   出色的问题解决能力和适应快速发展的初创环境的能力

+   强大的沟通能力和与团队协作的能力

+   愿意跨越我们整个产品进行问题解决

+   能够主动识别问题并找到解决方案

**附加要求：**

+   有音频处理经验

+   从零开始建立后端系统的经验

+   有构建和维护与多个第三方API集成的系统经验，特别是销售点系统

+   有 Kotlin 或 Java 的经验

+   有使用 Docker 和 Kubernetes 等容器化工具的经验

**我们提供的：**

+   有竞争力的薪水和股权奖励

+   全面的福利包括健康、牙科和视觉保险

+   在快速、创新的初创环境中，建立在尖端技术基础上的成长和晋升机会

+   对一个渴望创新的行业产生有意义影响的机会

+   免费午餐
