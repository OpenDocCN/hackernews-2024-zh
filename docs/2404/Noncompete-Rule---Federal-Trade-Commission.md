# 不竞争规则 | 联邦贸易委员会

> 来源：[https://www.ftc.gov/legal-library/browse/rules/noncompete-rule](https://www.ftc.gov/legal-library/browse/rules/noncompete-rule)

根据本部分使用：

*商业实体* 意味着合伙企业，公司，协会，有限责任公司或其他法律实体，或其部门或子公司。

*雇佣* 意味着为某人工作。

*不竞争条款* 意味着：

(1) 禁止工作者不得，对工作者进行处罚，或者阻止工作者从事以下行为的雇佣条件或条款：

(i) 寻求或接受在美国与其他人的工作，此类工作将在包括该条款或条件的雇佣结束后开始；或

(ii) 在包括该条款或条件的雇佣结束后，在美国经营业务。

(2) 对于本第 910 部分的目的，雇佣条件或条款包括但不限于合同条款或工作场所政策，无论是书面还是口头。

*官员* 意味着总裁，副总裁，秘书，财务总监，会计主管，以及任何经常执行与任何商业实体（无论是否已注册）对应职能的自然人。

*个人* 意味着任何自然人，合伙企业，公司，协会或其他在委员会管辖范围内的法律实体，包括根据州法行事或依法行事的任何人。

*决策权* 意味着对业务实体或共同企业重要方面的政策决策拥有最终权力，并不包括仅限于向此类政策决策提供建议或施加影响的权力，或仅对共同企业的子公司具有最终决策权的权力。

*决策职位* 指商业实体的总裁，首席执行官或等同职位，任何其他具有决策权的商业实体官员，或者对商业实体具有类似于具有决策权的官员的决策权的其他自然人。作为共同企业一部分的子公司或关联公司的官员，若对共同企业具有决策权，则可以视为具有决策职位，适用于本段。即使一个自然人对共同企业没有决策权，即使该人对子公司或关联公司具有决策权，也不会被视为具有决策职位。

*前一年* 意味着一个人在以下时间段中的选择：最近的52周年，最近的日历年，最近的财政年度或最近的雇佣周年。

*高级主管* 指一名工作者：

(1) 曾担任决策职位；和

(2) 收到来自雇佣者的雇佣报酬：

(i) 在前一年的总年薪至少为$151,164；或

(ii) 如果工人在前一年仅部分时间内受雇，则年化后的总薪酬至少为$151,164；或者

(iii) 如果工人在前一年离职，且工人在离职前一年年化后的总薪酬至少为$151,164，并且工人受到非竞争条款的约束。

*总年薪* 基于工人在前一年内的收入。总年薪可能包括在该52周期间内获得的工资、佣金、非自主分红和其他非自主薪酬。总年薪不包括膳食、住宿和其他设施的费用，如29 CFR 541.606所定义，并且不包括医疗保险费用、人寿保险费用、退休计划的贡献以及类似边际福利的费用。

*工人* 指的是不考虑工人的头衔或工人在任何其他州或联邦法律下的地位，无论其是否为雇员、独立承包商、实习生、外部人员、志愿者、学徒或向人提供服务的个体经营者，已经工作或曾经工作的自然人。该术语包括为特许经营者或特许经营者工作的自然人，但不包括特许经营者在特许经营者-特许经营者关系背景下的特许经营者。

(a) *不公平的竞争手段*—(1) *高管以外的工人*。关于高管以外的工人，以下行为被视为不公平的竞争手段：

(i) 签订或试图签订非竞争条款；

(ii) 强制执行或试图强制执行非竞争条款；或

(iii) 声称该工人受制于非竞争条款。

(2) *高管*。关于高管，以下行为被视为不公平的竞争手段：

(i) 签订或试图签订非竞争条款；

(ii) 在生效日期后签订或试图签订非竞争条款；或

(iii) 声称该高管受制于非竞争条款，如果非竞争条款是在生效日期后签订的。

(b) *现有非竞争条款的通知要求*—(1) *要求通知*。对于每个现有非竞争条款，根据本节(a)(1)(ii)的规定，不得强制执行或试图强制执行的，与工人签订了非竞争条款的人必须在生效日期之前向工人提供清晰和显著的通知，告知工人的非竞争条款不会被强制执行，也不能在法律上对工人进行强制执行。

(2) *通知形式*。根据本节(b)(1)的规定，向工人提供的通知必须：

(i) 确定与工人签订非竞争条款的人员；

(ii) 可以使用纸张递送给工人，或通过邮寄至工人的最后已知个人街道地址，或通过电子邮件发送至属于工人的电子邮件地址，包括工人当前的工作电子邮件地址或最后已知的个人电子邮件地址，或通过短信发送至工人的移动电话号码。

(3) *例外*。如果有责任根据本节(b)(1)段提供通知的人员没有街道地址、电子邮件地址或移动电话号码的记录，该人员对于该工人不受本节(b)(1)段的通知要求的豁免。

(4) *模板语言*。为本节(b)(1)段目的，以下模板语言构成通知工人其非竞争条款在法律上不可执行且不会对工人生效。

(5) *安全港*。根据本节(b)(4)段提供通知的人员符合本节(b)(1)段的要求。

(6) *其他语言的选择性通知*。除了在本节(b)(1)段要求的通知以英语提供外，允许以其他语言（或多种语言）提供该通知，或包括指向其他语言翻译的互联网链接。如果在本节(b)(6)段下提供选择性通知，可使用委员会提供的本节(b)(4)段模板语言的任何翻译。

(a) *真实销售业务*。本910部分的要求不适用于在真实销售企业实体、个人对企业实体的所有权利益或企业实体的全部或实质全部经营资产进行非竞争条款的情况。

(b) *现有诉因*。本910部分的要求不适用于非竞争条款相关的诉因在有效日期之前已产生的情况。

(c) *善意*。在有充分理由相信本910部分不适用的情况下，强制执行或试图强制执行非竞争条款或就非竞争条款作出陈述不构成不公平竞争方法。

(a) 本第910部分不会被解释为废止或豁免任何人遵守适用于非竞争条款的任何州法规、法令、命令或解释，包括但不限于州反垄断和消费者保护法律以及州普通法，但是本第910部分超越这些法律的范围，仅在这些法律本应允许或授权某人从事违反§ 910.2(a)中的不正当竞争行为或与§ 910.2(b)中的通知要求冲突的行为时，超越其范围。

(b) 除了在本节(a)项下被超越的法律外，本第910部分的任何条款均不得被解释为改变、限制或影响州总检察长或任何其他监管或执法机构或实体的权力，或者影响任何人根据任何州法规、法令、命令或解释提起索赔或监管行动的权利，包括但不限于州反垄断和消费者保护法律以及州普通法。

如果本第910部分的任何条款因其条款或在适用于任何人或情形时被视为无效或不可执行，或者在进一步机构行动待定期间被暂停，应解释为继续给予法律允许的最大效果，并且该无效性不会影响该条款对其他人或情形的适用，也不会影响其他条款的有效性或适用性。如果本部分的任何条款或适用被视为无效或不可执行，该条款或适用应可分割自本第910部分，并且不会影响其余部分的有效性。

本第910部分自最终规则发布之日起120天生效。
