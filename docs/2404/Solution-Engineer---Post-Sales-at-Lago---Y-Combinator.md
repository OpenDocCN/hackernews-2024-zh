### 团队及你的角色

+   作为我们增长的一部分，我们正在寻找一位负责领导我们售后客户体验的人，监督我们产品的实施、日常运行和潜在的客户账户扩展。我们的客户基本上包括欧美的软件、金融科技和基础设施公司，例如[Mistral.ai](http://Mistral.ai)，[Together.ai](http://Together.ai)，[Swan.io](http://Swan.io)。

+   这个角色要么是在巴黎第十区我们华丽的新办公室内工作，要么是远程工作，时区需与中欧时间（CET）相差+2/-2小时。不符合这些条件的申请将被自动归档。

+   今天我们是一个由12人组成的团队，未来几个月将扩展至20人团队，我们专注于客户体验、产品和开源。

**你的角色包括：**

+   深入了解客户、他们的业务目标、工作流程和目标，以尽可能有效地展示价值

+   利用深厚的技术专业知识，从端到端驱动和项目管理客户实施

+   通过为多个利益相关者提供一致、清晰的沟通，在实施项目的整个过程中提供卓越的客户体验

+   区分客户反馈中的“必须有”和“好有”的要求和反馈

+   理解复杂和/或复杂的业务需求，并为各种受众翻译它们

+   为了内外部理解而记录实施项目计划。定义项目里程碑并按时完成；清楚地沟通依赖关系和所有权（向不同观众更新状态，明确设置内外部优先事项）

+   辨识并减轻项目风险，如范围膨胀、利益相关者参与、客户IT资源、内部积压等；适时向内部和/或客户高管升级

+   与技术和运营团队成员建立信任、理解和影响力，他们为实施我们的产品贡献力量

+   利用这些“实时”案例建立可重复的操作手册（例如，学习、框架、最佳实践），以支持我们的增长

+   通过技术软件工程背景培训

+   对于确定实施或迁移范围时所需的准确性充满热情，同时擅长领导谈判：具备战略思维、耐心和情商。解决复杂且模糊的跨部门项目将是日常工作内容。

对学习领域特定概念的技术性细节充满热情，例如计费、支付、集成，甚至数据摄取，开源。

**了解我们的思维和工作方式**

Lago 是帮助工程师构建更好盈利系统的开源平台。他们使软件团队能够构建可定制和可扩展的使用计量、订阅管理、计费、发票和支付系统。

Lago 是Github上该领域的领导者（5000多星），在ProductHunt上赢得了“月度产品”，并被Mistral、Together、Swan、Juni、Fintecture、JuliaHub、Regate、Flipside Crypto、Unifonic、[Vida.id](http://Vida.id)、[Bureau.id](http://Bureau.id)等领先公司使用。

Lago 的创始团队之前曾构建并扩展了金融科技独角兽 Qonto 的自制盈利系统，从预发布到D轮融资。

### 我们的价值观

+   雄心：我们志向远大，因为否则，这有什么乐趣呢？

+   进步：我们更关心毅力、陡峭的学习曲线、韧性和创造力，而不是当前状态或你来自哪里。

+   谦逊的自信：我们知道自己的优势，始终追求卓越，但我们也知道还有很多我们不知道的东西。

+   支持前行：我们都是从某个地方开始的，当我们需要帮助时，有人伸出援手，我们不会忘记这一点，并强化这个积极循环。

+   不过分认真：我们雄心勃勃，不断学习和努力奋斗，但归根结底，我们的健康（身体和心理）以及我们所爱的人才是最重要的。
