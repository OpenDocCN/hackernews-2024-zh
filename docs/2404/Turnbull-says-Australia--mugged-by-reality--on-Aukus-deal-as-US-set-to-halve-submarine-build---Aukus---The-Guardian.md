# Turnbull says Australia ‘mugged by reality’ on Aukus deal as US set to halve submarine build | Aukus | The Guardian

> 来源：[https://www.theguardian.com/world/2024/mar/13/turnbull-says-australia-mugged-by-reality-on-aukus-deal-as-us-set-to-halve-submarine-build](https://www.theguardian.com/world/2024/mar/13/turnbull-says-australia-mugged-by-reality-on-aukus-deal-as-us-set-to-halve-submarine-build)

前总理马尔科姆·特恩布尔在美国宣布明年将减半潜艇建造数量后表示，澳大利亚在[Aukus](https://www.theguardian.com/world/aukus)潜艇协议上已被“现实所打击”，这使得协议的澳大利亚端面临疑问。

由于美国总统乔·拜登继续面对敌对的国会，五角大楼的预算草案请求包括仅建造一艘弗吉尼亚级核潜艇，用于2025年。

根据[Aukus](https://www.theguardian.com/world/aukus)协议，生产计划将加速，以确保澳大利亚在2030年代至少能从美国获得三艘弗吉尼亚级潜艇。这是为了填补在2040年代由阿德莱德建造的核潜艇服役前的“能力差距”。

总理安东尼·阿尔巴内塞（Anthony Albanese）淡化了美国预算公告的影响，坚称“我们的计划非常明确”。

“我们与美国和英国达成了一项协议，”阿尔巴内塞在达尔文对记者说。“那项立法在去年通过了美国国会。这是经过大量努力的成果。”

国防部长理查德·马尔斯（Richard Marles）早些时候表示，美国依然致力于这一协议。

*“*当我们接近澳库斯一周年时，澳大利亚、美国和英国在我们去年三月宣布的路径上保持坚定承诺，这将使澳大利亚获得常规武装的核潜艇，”他说。

“所有三方澳库斯伙伴正在加紧工作，整合我们的工业基础，实现我们国家之间的这一历史性倡议。”

绿党参议员大卫·舒布里奇（David Shoebridge），自始至终对澳库斯协议持批评态度，表示美国的预算公告是澳库斯协议的开始末期。

“当美国通过法律设   “当美国[通过法案设立澳库斯](https://www.theguardian.com/world/2023/dec/14/us-congress-passes-bill-allowing-sale-of-aukus-nuclear-submarines-to-australia)时，他们设置了杀死开关，其中一个允许美国在转移潜艇时，如果这样做会‘削弱美国的海底能力’，就不进行转移。”他说在X上。

美国的预算确实包括增加潜艇工业基地支出，这是奥库斯协议第一支柱的关键组成部分，因为它为未来几年增加生产奠定了基础。

但是特恩布尔，作为[法国潜艇协议](https://parlinfo.aph.gov.au/parlInfo/search/display/display.w3p;query=Id:%22media/pressrel/5005951%22?fbclid=IwAR21AMD3WoHLZ0ZDNaf0e4I2_BtzvM4naeaif6DD4MLUhgTf6aQYHL4FKFc)的缔造者之一，该协议被莫里森政府[不体面地抛弃](https://www.theguardian.com/world/2021/oct/31/macron-accuses-australian-pm-of-lying-over-submarine-deal)以支持奥库斯协议，表示澳大利亚现在对美国在其国防战略的一个关键部分完全依赖。

他表示，美国需要增加潜艇生产以满足自身需求，然后才能将潜艇转交给澳大利亚，但现在他们只生产约所需数量的一半，并因劳动力短缺而难以维护他们手中的潜艇。

*“*那对澳大利亚意味着什么？这意味着因为莫里森政府，由阿尔巴尼斯接管，基本上已经[放弃了我们的主权](https://www.theguardian.com/world/2023/feb/02/malcolm-turnbull-says-labor-has-failed-to-answer-if-aukus-deal-compromises-australian-sovereignty)在潜艇方面，我们现在完全依赖于美国的行动是否现在给予我们潜艇，”他告诉ABC电台。

“实际情况是，美国人不会通过向澳大利亚提供或出售潜艇来加剧他们已有的潜艇赤字，而Aukus立法实际上已经明确规定了这一点。”

澳大利亚工党基层活动人士团体“反对战争工党”希望阿尔巴尼斯政府冻结向美国支付的Aukus款项，以免“为美国海军工业船坞买单”。

反对战争工党的全国召集人马库斯·斯特罗姆表示，澳大利亚纳税人不应该为美国的船坞买单。

“我们有责任立即支付30亿美元的预付款，以便获得可能永远不会到货并在交付时毫无用处的潜艇。”他说。

“这届工党政府成功废除了斯科特·莫里森的税收计划。继续执行他的战争计划为何如此愚蠢？”

尽管五角大楼试图向澳大利亚保证其潜艇生产将在2028年前恢复正轨，[唐纳德·特朗普重返白宫的潜在威胁](https://www.theguardian.com/commentisfree/2024/feb/17/us-presidential-election-2024-donald-trump-reelection-australia-us-relationship)引发了进一步的担忧，[这笔交易可能会泡汤](https://www.theguardian.com/australia-news/2023/nov/01/almost-40-think-australia-should-dump-us-alliance-if-donald-trump-returns-as-president-poll-finds)。

“在 Aukus 的第一支柱上，我们与美国海军的需求实际上存在冲突，你和我一样知道，当美国政府需要在美国海军和澳大利亚海军之间做出选择时，他们总是会支持自己的。” 特恩布尔说。

马尔斯曾经 [否认过 Aukus 会侵蚀澳大利亚的主权](https://www.theguardian.com/australia-news/2023/feb/08/richard-marles-insists-aukus-submarine-deal-will-not-erode-australias-military-sovereignty)。在去年的一次议会演讲中，马尔斯表示，澳大利亚将“始终独立、主权地决定我们如何运用我们的能力”。
