# 美国参议院将投票表决一项被批评为“像东德秘密警察”的监听法案 | WIRED

> 来源：[https://www.wired.com/story/senate-section-702-ecsr-provision/](https://www.wired.com/story/senate-section-702-ecsr-provision/)

美国参议院本周将就一项立法进行投票，该立法至少在未来两年可能大幅扩展美国政府可以强迫窃听美国人而无需许可的企业范围。

一些国家顶尖法律专家就[一个有争议的美国间谍计划](https://www.wired.com/story/section-702-safe-act-compromise/)辩称，名为《改革情报和保护美国法案》（RISAA）的立法将增强美国政府的间谍权力，迫使各种新企业秘密窃听美国人的海外电话、短信和电子邮件。

这些专家包括少数有幸出现在美国政府秘密监视法院前的律师。

第702条款计划授权根据《外国情报监视法案》（FISA）进行，已经在十多年前成立，合法化了政府迫使主要电信公司在2001年9月11日恐怖袭击后窃听海外电话的做法。

一方面，政府声称该计划旨在专门针对身处国外的外国公民；另一方面，政府竭力捍卫其能力，可以获取对美国人电子邮件和电话对话的监听，通常是数年之后，且与最初发出监听指令的原因无关。

第702条款计划通过强迫美国政府定义的“电子通信服务提供商”——传统上如AT&T和Google等电话和电子邮件提供商——合作的方式来运作。众议院情报委员会成员，今天其领导人在国会中主要充当美国情报界的游说者，一直在努力扩大该术语的定义，使政府能够强迫新类别的企业代表政府窃听。

马克·兹威林格，一位曾两次出现在FISA法院审查的私人律师，[上周写道](https://www.zwillgen.com/law-enforcement/fisa-702-reauthorization-amendments-second-time-not-charm/)，RISAA立法扩展了“电子通信服务提供商”（ECSR）的定义，以 [包括数据中心](https://www.nytimes.com/2024/04/16/us/fisa-surveillance-bill-program.html) 和商业房东——他说，这些企业“仅仅是在物理空间中有接触通信设备”。根据兹威林格的说法，RISAA还可能囊括任何“能够访问这些设施和设备的人员，包括递送人员、清洁承包商和公用事业提供商。”

今年早些时候，Zwillinger批评了ECSR条款，导致众议院立法者修改文本，明确排除了某些类型的企业，包括酒店。

Zwillinger在回应中指出，这些排除的需要足以证明该文本过于宽泛；这些例外仅仅证明了规则的存在：“从起草者感到有必要排除的普通场所，如老年中心、酒店和咖啡店，可以明显看出新定义的广度。”他写道。“但是对于这些具体的例外情况，新定义的范围将覆盖它们——以及许多未获得特定豁免的企业。”

上周，这项分析迅速充斥了国会山的收件箱，一些国会助手和隐私专家悄悄地称ECSR条款为“斯塔西修正案”，这是对东德秘密警察力量的称谓，他们以渗透行业和强迫德国公民相互监视而臭名昭著。
