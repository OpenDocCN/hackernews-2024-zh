# NASA的LRO在与韩国的Danuri月球轨道器擦肩而过时，找到了一个拍照机会 - NASA

> 来源：[https://www.nasa.gov/missions/lro/nasas-lro-finds-photo-op-as-it-zips-past-skoreas-danuri-moon-orbiter/](https://www.nasa.gov/missions/lro/nasas-lro-finds-photo-op-as-it-zips-past-skoreas-danuri-moon-orbiter/)

*编辑注：此功能已于2024年4月9日更新，以更正航天器间的速度差异为每小时11,500公里。*

NASA的LRO（[月球勘测轨道飞行器](https://www.nasa.gov/lro)），已围绕并研究月球15年，上个月捕捉到了韩国航空航天研究院的Danuri月球轨道器的几张图像。两个航天器几乎平行轨道运行，在2024年3月5日至6日之间相向而过。

LRO的窄角相机（被称为“[LROC](https://www.lroc.asu.edu/)”摄像机套件中的一部分）在三次轨道上抓拍到了这里展示的图像。

由于两个航天器之间的快速相对速度（约每小时7,200英里或11,500公里），马里兰州格林贝尔特的NASA戈达德航天飞行中心的LRO操作团队需要精确的时间来将LROC指向正确的位置，在正确的时间捕捉Danuri的一瞥，这是韩国第一颗月球航天器。Danuri自2022年12月以来一直处于月球轨道上。尽管LRO的相机曝光时间非常短，仅为0.338毫秒，但由于两个航天器之间的高速相对运动，Danuri仍然出现了向相反方向拉长至其大小10倍的模糊现象。

*LRO is managed by [NASA's Goddard Space Flight Center](https://www.nasa.gov/goddard)，位于马里兰州格林贝尔特，为位于华盛顿NASA总部的科学任务司管理。LRO于2009年6月18日发射，其七台强大仪器已收集了大量数据，为我们了解月球做出了不可估量的贡献。NASA正与商业和国际合作伙伴一起返回月球，以扩展人类在空间中的存在，并带回新的知识和机遇。*
