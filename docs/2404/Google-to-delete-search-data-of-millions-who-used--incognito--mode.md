# 谷歌将删除数百万使用“无痕”模式的用户的搜索数据

> 来源：[https://text.npr.org/1242019127](https://text.npr.org/1242019127)

# 谷歌将删除数百万使用“无痕”模式的用户的搜索数据

作者：Bobby Allyn

周一，2024年4月1日 • 下午3:11

谷歌将销毁使用其Chrome浏览器“无痕”模式的数百万人的私密浏览历史，作为周一在联邦法院提出的[和解协议](https://www.documentcloud.org/documents/24527110-google-unopposed-settlement)的一部分，该案件涉及公司对网络活动的秘密跟踪。

多年来，谷歌只是通知Chrome互联网浏览器的用户，“您已进入无痕模式”，“现在您可以私密浏览”，当打开这个据说无法被追踪的浏览选项时，却没有说明公司在收集哪些数据。

然而，根据[2020年的一起集体诉讼案](https://www.documentcloud.org/documents/24527422-amended-google-complaint)，这家科技巨头继续通过广告工具获取互联网上使用无痕模式浏览的用户的搜索数据，收集数百万人的“潜在尴尬”搜索记录。随后，谷歌利用这些数据来衡量网站流量并销售广告。

“谷歌已经成为一个详细和广泛的信息宝库，乔治·奥威尔都无法梦见”，原告律师马克·毛和其他原告律师写道。

在此诉讼未决期间，谷歌将无痕模式的闪屏界面更改为声明，即网站、雇主、学校和互联网服务提供商可以查看无痕模式下的浏览活动。但根据协议，谷歌将不得不声明公司本身也可以在无痕模式下跟踪浏览活动。

此外，当用户使用无痕模式时，谷歌默认会阻止第三方公司追踪所谓的**cookies**，这是广告商获取个人搜索历史信息的方式。

类成员包括数千万使用无痕模式浏览过的人，他们将不会获得任何经济赔偿。但根据协议，个人用户可以在加州州法院起诉谷歌，以追回因秘密数据跟踪而损失的资金。

谷歌发言人何塞·卡斯塔涅达在一份声明中表示，公司“很高兴删除从未与个人关联并且从未用于任何形式个性化的旧技术数据”。他指出，诉讼要求5亿美元的赔偿，但作为提议和解的一部分，谷歌不会作出任何支付。

这笔交易避免了可能长期且昂贵的审判，同时谷歌正面对司法部提起的针对公司[垄断在线搜索](/2023/09/12/1198558372/doj-google-monopoly-antitrust-trial-search-engine)的诉讼以及针对谷歌[广告业务](/2023/01/24/1151055903/doj-files-second-antitrust-suit-against-google-seeks-to-break-up-its-ad-business)的另一项诉讼。

### 谷歌员工向管理层抱怨隐身模式的问题。

诉讼揭示，Google在同一用户配置文件中保存了标准和隐身浏览历史。然后，该数据用于个性化广告投放。

根据诉讼，"即使用户在 '隐私浏览模式' 下浏览互联网，Google仍在追踪他们。" "Google的追踪发生并继续发生，无论用户的在线活动多么敏感或个人化。"

代表起诉Google的消费者的律师引用了公司内部的电子邮件，显示员工向管理层抱怨隐身模式未能达到其名义上的隐私保护效果。

"我们需要停止称其为隐身模式，不再使用间谍图标，" 一名工程师在2018年写信给谷歌同事。

另一位谷歌员工建议将隐身模式的启动页面更改为："你并未受到Google的保护。"

诉讼提到了Google市场总监Lorraine Twohill发送给CEO Sundar Pichai的一封电子邮件，暗示公司在有关浏览工具的宣传中误导公众。

"我们在市场上推广隐身模式时受到一定限制，因为它并非真正私密，因此需要非常模糊、避重就轻的措辞，这几乎是更具破坏性的言辞。" Twohill在2019年写信给Pichai。

美国地方法官Yvonne Gonzalez Rogers最初设定了案件的二月审判日期，但在双方达成协议之后，这一日期被搁置。

交易仍需获得Gonzalez Rogers的最终批准。
