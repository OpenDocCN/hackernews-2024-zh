# 更安全的台锯可能会被法律要求，可能会防止断指：NPR

> 来源：[https://www.npr.org/2024/04/02/1241148577/table-saw-injuries-safety-sawstop-cpsc](https://www.npr.org/2024/04/02/1241148577/table-saw-injuries-safety-sawstop-cpsc)

大约20年前的一天，Tom Noffsinger经历了每个木工人最糟糕的噩梦：在结束一天的最后一次切割之前，他在台锯上切到了，结果送进了急诊室。那天下午，他早上就开始在他的工作室里工作。

"我有点疲倦。我应该早点停下来，" Noffsinger说。"我把手伸进了锯片里，几乎砍掉了我的拇指。"

台锯被普遍认为是最危险的电动工具，并且每年大约有[30,000](https://www.federalregister.gov/documents/2017/05/12/2017-09098/safety-standard-addressing-blade-contact-injuries-on-table-saws#:~:text=In%202015%2C%20there%20were%20an,contact%20with%20the%20saw%20blade.)次因刀片接触而需要医疗治疗，其中约有4,000例导致截肢，这可能对一些专业木工和承包商是毁灭性的。消费品安全委员会称，当一个人住院治疗时，每次台锯伤害的社会成本超过50万美元，同时还考虑到收入损失、痛苦和痛苦。

相比之下，Noffsinger算是幸运的。虽然他需要14针缝合，但在他家附近的罗利市医院的医生们成功保住了他的拇指。接下来进行了重建手术。即便如此，多年后他仍然经常感到疼痛。

木工对诺夫辛格几乎是终生的热情，他对电动工具也并不陌生。在他事故发生前，他曾在当地的木工店看到一种新型、更安全的台锯演示。该产品以SawStop的名义推广，设计初衷是在接触到肉体时[迅速停止并回缩旋转的刀片](https://www.youtube.com/watch?v=SYLAi4jwXcs)，速度足以将潜在的重大伤害变成不过是一道轻伤。诺夫辛格的台锯并未安装这种高科技安全功能，因为制造商并未要求包含此项功能。

但情况可能即将改变。联邦消费品安全委员会（CPSC）似乎准备要求所有在美国销售的新台锯安装SawStop型安全刹车。这一举措将是该机构多年来努力和失败的结果。

制造商一直在反对一项新规定，称其会提高消费者购买台锯的价格。安全倡导者将其比作汽车上的安全气囊，并认为其益处大于成本。

多年来，委员会中的共和党人一直支持电动工具行业反对进一步的法规。但随着拜登政府新任命的委员，委员会中的支持者似乎占据了多数。今年10月，消费品安全委员会（CPSC）[投票](https://www.federalregister.gov/documents/2023/12/11/2023-27133/safety-standard-addressing-blade-contact-injuries-on-table-saws-notice-of-extension-of-comment#:~:text=On%20October%2018%2C%202023%2C%20the,closes%20on%20January%202%2C%202024.)推进此项授权，预计将在今年晚些时候获得批准。

“我们提出的规定旨在预防成千上万例医疗治疗的台锯伤害，”CPSC委员理查德·特朗卡·小说家说。“这是我非常支持的事情。”

### 支持者表示，新标准早就应该有了。

前临时CPSC主席罗伯特·阿德勒表示，要求刀片刹车的标准“早就应该有了。”每天在美国有超过10人因此类台锯而截肢，“这一数字令人震惊，当你考虑到这一点时，”他说。“我非常高兴看到现在这一变化非常可能发生。”

2009年由巴拉克·奥巴马总统任命并在委员会任职12年的阿德勒，是新台锯安全标准争斗中的老兵。他称没有在台锯上要求这种功能是他在委员会任职期间感到的“最大的挫败”。他说这是因为台锯是大多数美国人购买的最危险的工具。

SawStop的竞争对手由[电动工具协会](https://www.powertoolinstitute.com/)代表，该贸易团体包括博世、德瓦特、米尔沃基等大型电动工具制造商，以及一些较为不知名的品牌。该团体认为新的安全规定会是一种过度干预。

“小型制造商可能会倒闭，”Power Tool Institute 执行经理 Susan Orenga 在二月份新规公听会上表示。她说，要求使用安全刹车会使台锯的成本过高。“台锯的销售将减少，导致失业，政府可能正在创造垄断。”

该行业长期以来一直声称，由于 SawStop 拥有围绕安全技术的专利，该公司将不当受益于政府强制实施的标准。但在 Orenga 发言的同一听证会上，SawStop 承诺允许制造商制造更安全的台锯，无论这些专利如何。

### 台锯的安全性需付出代价

安全刹车会给锯床的价格增加多少还不清楚。入门级的 SawStop 零售价为 $899。一台没有安全技术的可比锯床则便宜几百美元。

但随着较大竞争对手的规模经济，价格差异可能会在未来变得更小。

SawStop 的零售价格比竞争对手高出数百美元，具体取决于制造商和台锯类型。与大型连锁商店销售的廉价品牌不同，SawStop 属于高端市场。 **NPR 的 Cornell Watson** ****hide caption****

自 [SawStop 2004 年进入市场](https://www.sawstop.com/company/#:~:text=The%20technology%20was%20invented%20by,SawStop%20Table%20Saw%20was%20sold.) 以来，该公司在美国销售了数万台台锯，据估计，这挽救了数万名专业和业余木工免受伤害。

SawStop 的关键在于其 [主动伤害缓解](https://www.sawstop.com/service-tips/active-injury-mitigation-aim-system-and-brake-cartridge-replacement-for-the-compact-table-saw-cts/) (AIM) 系统，该系统通过锯片传递一个小电荷，因为皮肤导电，系统可以感应到锯片是否被触摸。基本上，木材不导电，但人类导电。当手接触 SawStop 上的锯片时，这会触发刹车停止锯片旋转。这一过程发生得非常迅速，没有足够时间造成严重伤害。

自 2004 年在 [NPR](https://www.npr.org/2004/12/07/4182602/table-saw-technology-aims-to-save-fingers) 上首次听到 SawStop 技术以来，国家消费者联盟的执行主任 Sally Greenberg 就对台锯安全感兴趣。像 Adler 一样，她对新安全标准的缓慢推进感到沮丧。

"这是一类产品，在这种情况下可以做到100%安全，但因为行业拖延、抵制以及在国会和机构中的游说影响力，你会看到两步前进，一步后退的局面。"

直到最近，SawStop的竞争对手大多被德国TTS Tooltechnic Systems持有的一系列专利阻止开发AIM型技术。TTS在2017年收购了SawStop。但是在第一台SawStop销售20年后，大多数这些专利现已过期。

### SawStop誓言为竞争对手释放关键专利。

然而，一个关键专利——"840"专利——直到2033年才到期。为了防止潜在竞争对手，它广泛描述了AIM技术。在二月份CPSC听证会上，TTS Tooltechnic Systems North America的CEO Matt Howard宣布，如果新的安全标准被采纳，公司将把"840"专利"奉献给公众"。Howard表示，这将释放竞争对手开发他们自己的安全装置或简单地复制SawStop的技术。在听证会上，他挑战他们"加入游戏"。

Howard的让步是在SawStop和更大的电动工具公司之间多年的积怨之后做出的。在创办SawStop之前，其技术发明人[Steve Gass](https://www.npr.org/2011/05/25/136617222/advocates-urge-lawmakers-to-make-table-saws-safer)——他本人是一名专利律师——曾试图让制造商授权他的想法。但没有人愿意接受。多年后，当Bosch Power Tools开始销售一种带有自己版本的伤害缓解系统的锯子时，SawStop [赢得了一场专利侵权诉讼](https://www.sawstop.com/news/sawstop-sues-bosch-to-protect-inventions/)。随后，TTS同意让Bosch销售这种锯子，但Bosch从未重新引入到美国市场。

这起诉讼被行业引用来支持其主张，即任何开发类似安全功能的举措都将遭到SawStop和TTS的激烈抵制。

还有其他行业反对意见。Orenga指出，制造商已经遵守了一个自愿标准，要求刀片罩和防反弹功能，旨在防止刀片抓住木头并将其猛烈抛回操作者。

### "脆弱、功能欠佳的防护罩"起不了什么作用。

但根据CPSC的说法，台锯用户经常会"[移除模块化刀片防护罩](https://www.federalregister.gov/documents/2023/11/01/2023-23898/safety-standard-addressing-blade-contact-injuries-on-table-saws)，"通常是因为"提高可见度"——换句话说，因为他们无法轻易看到他们试图进行的切割。

因此，美国消费品安全委员会（CPSC）表示，自2010年行业采纳了改进刀片保护罩和其他安全功能的自愿要求以来，刀片接触伤害的数量没有明显变化。简而言之，自愿标准“不能充分减少受伤风险”，特鲁姆卡说，这就是为什么委员会正在推动强制性标准。

吉姆·汉密尔顿，在YouTube上主持一档受欢迎的[木工频道](https://www.youtube.com/@StumpyNubs)，表示如果木工们始终使用刀片保护罩，大多数台锯伤害是可以避免的。“可悲的是，围绕许多电动工具，包括台锯，形成了一种文化，认为安全装置是不必要或者妨碍使用的，”他说，并指出即使是“经验丰富的工人，包括那些在其行业的最高水平工作的人，每天都会严重受伤。”

汉密尔顿说，制造商包括“脆弱、功能欠佳的保护罩”，实际上鼓励用户将其移除，这使情况变得更糟。

### 台锯会导致一种“蒸发”类型的伤害。

理查德·博多尔，一位驻圣地亚哥的整形外科医生，对锯片可能造成的手部灾难性伤害非常了解。

他最清楚地记得发生在大约25年前的一起事故，在SawStops上市之前。当他一夜间进行手术以重新植回一根被切断的手指时，急诊室打来电话，说博多尔的同事 — 一位资深外科医生和导师 — 又有一个“四指植回”案例转介。起初，博多尔以为他的同事只是询问另一个病人的情况。他很快意识到，受伤的是那位外科医生本人。

那位外科医生在使用台锯时，手套被锯片卷入并拉住了他的手。博多尔说，在术前，受伤的外科医生出奇地镇定，两人讨论了重建这名男子残破手部的复杂手术过程。

在谈到他每根撕裂的手指时，受伤的外科医生运用自己的专业知识进行重建。“‘我认为这根手指会好起来。现在，我有点担心这个家伙。不过，我觉得这个小家伙可能没救了，’”博多尔回忆道，引用他们的对话。

博多尔说，经过长时间的康复，这名男子最终能够恢复手术工作。但是，他说，这类锯伤尤其具有挑战性，很难修复。他解释说，与由锋利的烹饪刀造成的干净截肢不同，台锯刀片实际上会蒸发组织。“这是一种蒸发性的伤害，”他说，并补充说，重新植回通常需要数小时的细致显微外科手术。

但并非所有人都相信仅仅通过新的安全标准就能防止这种灾难性的伤害。戴尔·尤恩图南在明尼苏达州迪尔河拥有一家承包公司，已经建造房屋超过40年。“在我经营这么多年的生意中，我们从未有人在台锯上受伤过，”他说。

“如果被强制实施，你会发现人们会一直使用他们的旧锯子，” 朱恩特宁说。“而且，你知道，这就是我会说会有更多的人在旧锯上受伤的时候。”

北卡罗来纳州的业余木工 Noffsinger 说，尽管他受伤了，但他不确定在所有锯子上强制实施新的安全技术是否是个好主意。

尽管如此，当他从急诊室回到家时，几乎切断拇指的锯片，他的妻子站在那里，“双手叉腰”，他说道。“她说，‘你会买那个SawStop的东西。’ 所以我就这么做了。”
