# -   不要表现得像是名人

-   有关休闲活动的建议。

-   你不是名人。你所做或创造的任何事物可能都不会引起多少关注，所以停止为一个不存在的观众进行优化，而是专注于让你享受活动的方面。

想尝试一种手工艺或艺术爱好吗？专注于掌握技能，并享受它带来的多样性。你不需要打造个人风格。没有人会在意。想要摄影并认为黑白照片很酷？太好了！你不需要创建一个围绕你情绪化黑白照片的Instagram品牌。很可能你会对此感到厌倦，想尝试不同类型的摄影，这也很好。你不是安塞尔·亚当斯，如果你的“风格”东一榔头西一棒子也没人会在意。

-   想要开发一个应用程序或网站，但不喜欢设计过程吗？那就把它设计得丑一点。谁在乎！设计是为了观众，而你没有观众。功能性现在更重要。也许会有设计师注意到，并希望帮助你改进它，但在那之前，为你那些糟糕的UI感到自豪吧。

-   博客写作既有趣又治疗性。语法和编辑不是最重要的。只要你的思路连贯，不要太担心写作错误或自我过滤。只需使用Grammarly来修复初级错误，然后继续前进。这更多关乎写作过程而不是最终产品。

-   任何活动中最严重的事情就是幻想如何从中赚钱。这是把重点放错地方、把乐趣吸干的最快方式。很可能你会立即停止这项活动，所以把赚钱计划留给工作吧。

-   最终，找到一些你喜欢做的事情，并因为你喜欢而去做它。如果必须的话，为自己设定一些目标，但绝不为了你的“观众”。
