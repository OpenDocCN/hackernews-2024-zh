# 《The Atlantic：Neuralink所缺少的东西》

> 来源：[https://www.theatlantic.com/technology/archive/2024/04/neuralink-bci-implant/677996/](https://www.theatlantic.com/technology/archive/2024/04/neuralink-bci-implant/677996/)

直到最近，在整个人类历史上，真正的赛博格人数仅为约70人。Ian Burkhart一直在计算这个数字，因为他就是其中之一——其大脑直接连接到计算机的人。

Burkhart在一次游泳事故中因一波把他冲到沙洲上并伤害了他的脊髓后成为四肢瘫痪患者。后来，他能够通过一项研究获得植入物，使他能够暂时恢复一只手的一些运动能力。七年半来，他一直使用这个设备——一个嵌入他运动皮层的电极阵列，将信号传输到计算机，然后激活绕着他手臂的电极。Burkhart现在领导着BCI Pioneers Coalition，这是一个为其他志愿者将他们的大脑用于推动脑-计算机界面技术或BCI技术边界的少数残疾人的组织。

上个月，Burkhart 和其他可能数百万人一起观看了最新的赛博格的首次亮相。[在发布在 X 上的视频中](https://twitter.com/neuralink/status/1770563939413496146)，埃隆·马斯克的BCI公司Neuralink的第一个人类受试者似乎通过大脑植入物控制了一台笔记本电脑。Neuralink尚未公布其研究成果，也没有回应请求进行评论，但该设备的工作原理可能是这样的：受试者，一位名叫Noland Arbaugh的29岁瘫痪患者，通过思考特定事物（比如移动计算机屏幕上的光标或移动他的手）产生一种神经活动模式。然后，植入物将这种神经信号模式传输到计算机，其中一种AI算法将其解释为移动光标的命令。因为据称该植入物允许用户几乎用思维控制计算机，马斯克将该设备命名为“心灵感应”。

Burkhart看着Arbaugh无需手动，通过计算机象棋来混合认可和沮丧，因为演示清楚地为投资者和Musk的粉丝准备，而不是像他这样的残疾人士。众所周知，Musk的真正目标是为普通消费者创造BCI设备，而不仅仅是让我们能够移动光标；他设想人类未来可以直接从计算机获取知识，以“[实现与人工智能的共生](https://www.axios.com/2018/11/26/elon-musk-artificial-intelligence-neuralink)”。这个梦想在[伦理上充满了争议](https://www.nature.com/articles/d41586-019-02214-2)——例如，当你的思想被专有算法增强时，隐私问题就很棘手——但它离实现还有很长的路要走。研究人员在某种程度上[成功实现了](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC8138057/)与老鼠的双向信息传输，但没有人确定老鼠对此有何感受，或者他们是否愿意在商场的一个小卖部里为此支付。

然而，一个更为谨慎、可行的神经假体的愿景，可以让残疾人士轻松使用计算机，*是可以实现的*。问题在于我们的社会结构是否准备跟上我们先进的科学步伐。

BCI技术已经花费数十年的时间才达到这一点——科学家们手工制作原型，而那些既不能移动也不能说话的志愿者们则努力控制它们。将大脑与计算机连接的最基本挑战之一是材料的不兼容性。虽然计算机由硅和铜制成，但大脑却不是。它们的一致性有些类似于木薯布丁；它们会晃动。大脑在学习时不断变化，并且往往会在侵入物周围形成瘢痕组织。你不能简单地把一根电线插进去。

不同的开发者尝试了不同的解决方案来解决这个问题。Neuralink正在研究通过灵活的丝状物质无声地穿过大脑组织。Precision Neuroscience由部分前Neuralink科学家成立，正在尝试一种覆盖电极的Saran Wrap，它可以贴在大脑表面或滑入褶皱中。然后是犹他阵列，这是一种广泛使用的模型，看起来有点像带有[硅鞭毛垫](https://blackrockneurotech.com/wp-content/uploads/2023/04/Utah-Array-NeuroPort-2.jpg)的发刷。这就是Burkhart在2021年之前脑中所植入的，直到他所参与的研究[失去了资金支持](https://www.technologyreview.com/2023/05/26/1073655/how-it-feels-to-have-a-life-changing-brain-implant-removed/)，他决定将植入物取出。他担心外科医生可能不得不“连同一些大脑组织一起取出”。幸运的是，他告诉我，它“并没有太多抵抗”。

一旦植入物放置好了，个体神经元的微小信号——以微伏测量——必须被放大、数字化并传输，最好是通过无线和不显眼的单元。这是第二个问题。问题三是解码这些信号。我们对大脑如何与自身交流没有真正的了解，因此机器学习算法必须采用蛮力方法，找出神经活动中的模式，并学会将其与植入物患者试图让计算机执行的任务相关联。

这些问题没有一个是微不足道的，但在过去30年的BCI研究中，它们已经得到了大幅解决。至少有六家不同的公司正在测试应用，例如桌面界面（例如帮助阿尔鲍奇下棋的那种界面）、[机械假肢](https://www.wired.com/story/this-brain-controlled-robotic-arm-can-twist-grasp-and-feel/)和外骨骼，甚至是[语音假肢](https://www.wired.com/story/the-long-search-for-a-computer-that-speaks-your-mind/)，这些假肢可以传达思想。对于所有这些概念验证设备现在都已经存在。

但这仅仅带我们来到第四个问题——这与工程无关，可能比其他所有问题更难解决。这个问题是Precision公司首席科学官本·拉波波特（Ben Rapoport）向我描述的“科学产品化”。这里是工程成功遇到政治和经济障碍的地方。要推出甚至是基本的点按式医用BCI界面，开发者不仅需要赢得FDA的批准，还需要赢得“付款方”的认可：医疗保险、医疗补助和私人保险公司。这是成败关键：即使是医疗设备，即使再巧妙，如果保险不覆盖，也无法传递给消费者。很少有人能自掏腰包支付这些费用，这意味着潜在消费者的数量太少，以至于生产不具盈利性。

其他设备已经克服了这个障碍——例如耳蜗植入物、深部脑刺激设备、起搏器——如果保险公司认为BCI植入物值得花费，那么BCI植入物很可能也会加入到这个列表中。一方面，保险公司可能会认为BCI设备并非严格必需医疗——正如伯克哈特所说，它们是“增强生活”的而不是“维持生命”的——但另一方面，如果它们的实施可以节省其他更昂贵支持类型的费用，保险公司很可能会认为它们是成本效益高的。

尽管如此，脑植入物能做到的和能替代的有限。 最能从BCI设备中受益的人群，像Arbaugh和Burkhart这样有重大运动障碍的人，仍然依赖人力劳动来完成许多事情，如起床、洗澡、穿衣和进食。 这种劳动的成本很容易高达六位数，并且通常不会由私人健康保险公司报销。 对于大多数人来说，唯一覆盖此类护理的保险公司是[Medicaid](https://crsreports.congress.gov/product/pdf/IF/IF10343)，在大多数州都对受益人的[收入](https://www.kff.org/affordable-care-act/state-indicator/medicaid-income-eligibility-limits-for-adults-as-a-percent-of-the-federal-poverty-level/)和[资产](https://choicemutual.com/blog/medicaid-asset-limits-by-state/)设有严格的限制。

Burkhart居住的俄亥俄州，其Medicaid受益人不能拥有超过$2,000的资产，也不能每月收入超过$943，否则将失去覆盖。 （一项豁免计划提高了某些人的月收入上限至$2,829。） 然而，他们必须赚到的工资来支付支出和家庭护理的费用远远超过大多数工作的报酬。 “许多人没有机会迈出如此巨大的一步，”Burkhart说道。“这个系统被设置成迫使你生活在贫困中。”

除了他在BCI Pioneers Coalition的工作外，Burkhart还领导一家非营利性[基金会](https://www.ianburkhartfoundation.org/)，为帮助残疾人筹款，以支付一些保险不覆盖的费用。但正如他告诉我，“这些费用远远不足以支付获取BCI或类似设备的费用。” “我们做了很多淋浴椅。或者是车辆的手动控制。”

从20世纪末开始，简单的开关设备开始使重度运动障碍的人们能够访问计算机。 因此，许多过去可能会[被收容](https://www.nytimes.com/2003/11/23/magazine/the-disability-gulag.html)的人，例如那些无法说话或大部分身体无法动弹的人，现在能够沟通并使用互联网。 相比之下，BCI有可能比开关访问技术更加强大，后者与之相比较慢且不稳定。 然而，接受第一代医疗植入物的人们可能会发现自己处于与现在使用开关技术的人们相同的境地：在实质上，为了获得维持生命的服务，他们被迫失业、贫困，甚至[单身](https://www.medicaidplanningassistance.org/question/what-is-the-medicaid-marriage-penalty/)。

Musk可能是对的，我们很快就会到达BCI技术实用和普及的时刻。但现在，我们在如何分配诸如医疗保健等资源上还没有社会共识，许多残疾人仍然缺乏访问社会所必需的基本支持。这些是仅靠技术无法解决的问题。
