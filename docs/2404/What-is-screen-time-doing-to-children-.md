# 什么是屏幕时间对孩子们的影响？

> 来源：[https://www.economist.com/science-and-technology/2024/04/17/what-is-screen-time-doing-to-children](https://www.economist.com/science-and-technology/2024/04/17/what-is-screen-time-doing-to-children)

两个月前，黛西·格林韦尔（Daisy Greenwell）和克莱尔·费恩哈夫（Clare Fernyhough）成立了一个WhatsApp群组，讨论如何阻止他们年幼的孩子要求拥有智能手机。在她们在Instagram上发布了她们的计划后，其他家长也想要加入。现在，她们的“无智能手机童年”（Smartphone-Free Childhood）群组拥有超过6万名关注者，讨论如何让孩子远离这些“恶魔设备”——而这些讨论自然地也是在她们自己的智能手机上进行的。

这个位于英国的组织并不是唯一关注儿童屏幕时间的团体。上个月，佛罗里达州通过了一项法律，禁止14岁以下的儿童使用社交媒体。据报道，英国政府正考虑禁止向16岁以下的青少年销售手机。担忧的核心在于乔纳森·海德特最近出版的一本书《焦虑的一代》，书中认为智能手机，尤其是通过它们访问的社交网络，正在导致一种恶性的“重塑童年”的现象。
