# 威尼斯测试入城费以减少人群：NPR

> 来源：[https://www.npr.org/2024/04/25/1247162570/venice-entry-fee-day-trippers-tourists](https://www.npr.org/2024/04/25/1247162570/venice-entry-fee-day-trippers-tourists)

意大利威尼斯——在全球媒体的注视下，[脆弱的泻湖城市](https://apnews.com/article/venice-unesco-world-heritage-downgrade-threat-45a7e0d51069c9f06e79b69bd8bd1ed3)威尼斯周四启动了一项试点计划，向日间游客收取5欧元（约合5.35美元）的入城费，当局希望这一举措能够减少高峰日的游客到访，使这座城市对居住在此的居民更加宜居。

抵达威尼斯主火车站的游客被迎接的大型标志列出了[截至7月的29个日期](https://apnews.com/article/venice-italy-tourism-24732367323640a7975ae77a038d093b)，这是该计划测试阶段的安排，同时为游客和居民、学生及工作者分配了不同的入口。

“我们需要在游客和居民之间找到新的平衡”，该市的首席旅游官Simone Venturini表示。“当然，我们需要保护居民的空间，我们需要阻止某些特定日子的日间游客到来。”

然而，并非所有居民都认为新系统在[阻止大规模旅游](https://apnews.com/article/virus-outbreak-ap-top-news-international-news-changing-economy-italy-d95a5c8ce216791243e3ec2189536140)方面有效，他们坚持认为只有人口的复兴才能恢复这座城市的平衡，狭窄的小巷和水上巴士经常因游客而拥挤。

数百名威尼斯居民对这一计划表示抗议，他们欢快地游行穿过城市的主公交车站，携带写着“反对门票，支持服务和住房”的横幅。抗议者与身着防暴装备的警察发生了短暂的冲突，后来改变方向，通过另一座桥由便装警察护送进入了城市。抗议活动在一个广场上和平地结束了。

抵达主车站的游客不仅遇到了几乎和执勤人员一样多的记者，还有礼貌地指导任何不了解新要求的人下载[QR码](https://cda.ve.it/en/)以支付费用的过程。

首次来威尼斯游览的罗马游客阿丽安娜·塞西莉亚表示，她觉得在自己的祖国内必须付费进入一个城市，并通过专门的入口进入是"奇怪"的。她和她的男朋友住在附近的特雷维索，并且按要求下载了QR码，但当她第一次看到威尼斯运河的景色时，她还是被入口标志和她的男朋友告诉她要出示门票弄得措手不及。

在入口的另一侧，穿着黄色背心的工作人员在火车站进行随机检查。违规者将面临50到300欧元（53到320美元）的罚款，但官员表示正在根据情况进行启动。

此要求仅适用于早上8:30至下午4:00之间到达的人员。在这些时间之外，入口是免费且无需检查的。

威尼斯长期以来一直饱受过度旅游之苦，官员们希望这个试点项目能够帮助提供更准确的数据以更好地管理这一现象。

城市可以追踪酒店游客的数量，去年达到460万人次，比疫情前高峰时期减少了16%。但作为威尼斯人群中的大多数，日间游客的数量直到最近才能估算出来。

在疫情期间设置的智能控制室利用手机数据追踪到达者，粗略确认了每年2500万到3000万人次的预疫情估计，该城市的首席经济官员米歇尔·祖因表示。这包括日间游客和过夜客。

但祖因表示数据还不完整。

"显然，我们将从日间游客支付的贡献中获得更可靠的数据"，他说。

文图里尼表示，当日间游客数量达到3万到4万人时，城市就感到压力。在高峰日，当地警方会为行人设置单向交通，以保持人群流动。

反对日间游客税的居民坚持认为，解决威尼斯问题的方法是增加居民人口和他们所需的服务，限制短期租赁，以提供更多的住房，并吸引从大陆回来的家庭。

去年，威尼斯迎来一个显著的里程碑，游客床位数量首次超过了官方居民人数，现在历史中心及其风景如画的运河地区的官方居民人数不到5万人。

"对一个城市收费入城将不会减少即使一个游客的数量，" 组织了一场反对措施的抗议活动的活跃分子汤马索·卡恰里说。

"你买地铁票，去博物馆，去游乐园。你不会买门票进入一个城市。这是这个市政管理项目的最后象征性步骤，目的是把居民赶出威尼斯，" 他说。

威尼斯官员预计周四付费日游客到达数量将达到约10,000人。超过70,000人已下载了一个免责的QR码，包括在威尼托地区工作或是居住的居民。威尼斯的酒店，包括玛格赫拉或梅斯特等大陆区的酒店，都应提供一张包含住宿信息的QR码，其中包括酒店税。

旅游官员文图里尼表示，威尼斯试点项目引起了其他饱受大规模旅游之苦的地方的浓厚兴趣，包括其他意大利艺术城市以及国外的市政当局，如西班牙的巴塞罗那和荷兰的阿姆斯特丹。

但居住在威尼斯已有30年的玛丽娜·罗迪诺并不认为这个费用是万灵药。她居住在靠近著名里亚尔托桥的住宅楼里，曾经住满家庭的邻近公寓现在都成了短期出租公寓。

街角的肉铺已经关门。尽管如此，她指出新的入场费要求仍将允许年轻人在晚上涌入城市参加传统的开胃酒会，这可能会变得喧闹起来。

她正在发放“威尼斯，开放城市”的模拟欧盟护照，强调了新系统的讽刺意味，并挑战其在意大利宪法中保障公民“在国家任何地方自由移动或居住”的合法性。

"这不是一个自然绿洲。这不是一个博物馆。它不是庞贝。这是一个城市，我们需要为了住户重返家园，商店重新开业而进行斗争。这才是对抗这种狂野旅游的方法，" 罗迪诺说。
