# 谷歌员工举行静坐抗议公司与以色列合作 - 华盛顿邮报

> 来源：[https://www.washingtonpost.com/technology/2024/04/16/google-sit-in-employee-protest-nimbus-israel/](https://www.washingtonpost.com/technology/2024/04/16/google-sit-in-employee-protest-nimbus-israel/)

在旧金山，数名谷歌员工周二晚在公司位于纽约市和加利福尼亚州圣尼瓦尔的办公室内举行静坐抗议，抗议这家科技巨头与以色列政府的合作，使科技公司内部有关加沙战争和美国公司是否应向以色列出售技术的冲突升级。
