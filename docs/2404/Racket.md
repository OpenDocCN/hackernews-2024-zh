# Racket

> 来源：[https://racket-lang.org/](https://racket-lang.org/)

Racket 配备了一套全面的库：[跨平台 GUI 工具箱](https://docs.racket-lang.org/gui/)，[Web 服务器](https://docs.racket-lang.org/web-server/index.html?q=%23lang%20web)，等等。 [数千个附加包](https://pkgs.racket-lang.org) 只需[一个命令](https://docs.racket-lang.org/pkg/getting-started.html?q=raco%20pkg#%28part._installing-packages%29)即可获得：3D 图形，蓝牙套接字连接器，颜色映射，数据结构，教育软件，游戏，量子随机数生成器，科学模拟，Web 脚本测试等等。

宏与这些工具一起使用。 左侧的示例 <abc class="disappear-if-smartphone">上方</abc> <abc class="smartphone-only">上述</abc> 展示了一个小型猜数字游戏的实现。 它是用 Racket 的 GUI 方言实现的，并展示了多种语言特性。
