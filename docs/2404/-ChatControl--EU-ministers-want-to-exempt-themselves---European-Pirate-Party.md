# #ChatControl: 欧盟部长希望豁免自己 | 欧洲海盗党

> 来源：[https://european-pirateparty.eu/chatcontrol-eu-ministers-want-to-exempt-themselves/](https://european-pirateparty.eu/chatcontrol-eu-ministers-want-to-exempt-themselves/)

根据法国新闻机构Contexte泄露的最新草案文本显示，欧盟内政部长们希望豁免情报机构、警察和军方职员的专业账户不受计划中的聊天和信息扫描影响（第1条（2a））。该法规还不适用于“机密信息”如专业秘密（第1条（2b））。欧盟各国政府反对新的欧盟儿童保护中心支持他们在预防儿童性虐待方面的作用，并制定最佳实践预防倡议（第43（8）条）。

“欧盟内政部长们希望豁免警察、士兵、情报人员甚至他们自己免受聊天控制扫描的事实证明，他们确切地知道他们想向我们公民释放的窥探算法是多么不可靠和危险，” 海盗党欧洲议会议员帕特里克·布莱尔评论道。“他们似乎担心，即使与儿童性虐待无关的军事机密也可能随时传送到美国。政府通讯的保密性当然很重要，但同样适用于保护商业以及包括受虐待受害者本人在内的公民通讯的安全空间和治疗。我们知道，今天自愿窥探算法泄露的大多数聊天与警察无关，例如家庭照片或自愿性情色短信。令人愤慨的是，欧盟内政部长们自己不愿承受他们强加给我们的数字通信隐私和安全加密破坏的后果。”

“专业秘密不受聊天控制影响的承诺是一场用段落编织的谎言。没有任何供应商或算法能知道或确定聊天是否与医生、治疗师、律师、辩护律师等进行，从而使其免于聊天控制。聊天控制不可避免地威胁到发送用于医疗目的的亲密照片和为保护受虐待受害者而发送的审判文件的泄露。”

“这对于儿童保护的官方目标是一种嘲弄，因为欧盟内政部长们拒绝开发预防儿童性虐待最佳实践。这个法案的目的显然是中国式的大规模监控，而不是更好地保护我们的孩子。真正的儿童保护需要系统的科学评估和跨学科预防方案的实施，以及欧洲范围内针对儿童虐待的刑事调查标准和指南，包括受害者的识别和必要的技术手段。欧盟内政部长们没有计划实施这些措施。”

欧盟政府希望在六月初通过聊天控制法案。
