# 法官宣布无罪Backpage联合创始人Michael Lacey的大部分指控

> 来源：[https://reason.com/2024/04/25/judge-acquits-backpage-co-founder-michael-lacey-on-most-counts/](https://reason.com/2024/04/25/judge-acquits-backpage-co-founder-michael-lacey-on-most-counts/)

联邦法官[宣布](https://reason.com/wp-content/uploads/2024/04/usa-v-lacey-order-of-acquittal.pdf)Backpage联合创始人Michael Lacey数十项指控无罪，包括联邦检察官计划今年晚些时候对Lacey重新审判的大部分指控。美国地区法官Diane Humetewa还对前Backpage高管Jed Brunst和Scott Spear多项陪审团去年秋季判他们有罪的指控作出了无罪判决。

"在将记录以有利于政府的最光明的方式观看后，法庭认为无法支持Lacey先生的第19至51项指控和Lacey、Brunst和Spear先生的第66至99项指控的定罪，" Humetewa得出结论。

今年11月，[陪审团判定Lacey有罪](https://reason.com/2023/11/29/the-backpage-defendants-never-stood-a-chance/)仅在86项指控中的一项，而且还有一项无罪。陪审团对其他84项指控未能达成一致意见，包括所有Lacey主动促进卖淫或参与旨在通过他与长期报纸合作伙伴James Larkin创立的在线分类广告网站促进卖淫的指控。（Larkin在去年夏天的审判前几天[自杀身亡](https://reason.com/2023/08/02/backpage-publisher-alt-weekly-entrepreneur-and-free-speech-warrior-james-larkin-has-died/)。）

尽管已经对同一指控进行了两次审判（第一次是在2021年，[宣布审判无效](https://reason.com/2021/09/14/biased-testimony-in-backpage-trial-triggers-more-calls-for-a-mistrial/)，因为检察官及其证人无法停止谈论性交易，尽管被告没有面临性交易指控）。

现在，Humetewa法官已经判Lacey在其余84项指控中的53项无罪。此外，Humetewa法官还对Backpage的前执行副总裁Spear有10项被陪审团认定有罪的指控作出了无罪判决，并对前首席财务官Brunst有18项被判有罪的指控作出了无罪判决。

另外两名被告被陪审团对所有指控作出无罪判决。

Lacey、Spear和Brunst可能仍面临严重的问题。

Lacey仍有可能在今年晚些时候重新审判其余的指控，包括一项共谋违反联邦旅行法促进卖淫的指控，17项违反旅行法促进卖淫的指控，以及几项不同的洗钱指控。

而在6月17日，Lacey将因为陪审团认定他有罪的一项指控——国际洗钱隐瞒罪——而被判刑。这可能导致他在联邦监狱里被判最长20年的刑期。Lacey计划就这一项定罪上诉，看起来他成功的机会很大，因为他所谓的“隐瞒”的钱款已经按照所有规定向联邦政府报告。但在上诉过程中，他仍可能面临监禁。

Brunst预计将与Lacey一起在6月份被判刑，尽管Humetewa无罪释放他，他仍面临14项指控的判刑。而Spear将在7月9日被判刑，仍面临29项指控的判刑。
