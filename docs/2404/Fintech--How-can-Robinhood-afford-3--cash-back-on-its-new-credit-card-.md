# 金融科技：罗宾汉如何负担起其新信用卡的3%现金返还？

罗宾汉，总部位于门洛帕克的公司，以推广免佣交易而闻名 — 还有[迷因股票狂潮](https://techcrunch.com/2022/06/27/robinhood-report-meme-stock-gamestop/) — 以其新的罗宾汉金卡进入信用卡领域。

信用卡的推出不到一年时间，罗宾汉以9500万美元现金收购了信用卡公司X1。[我们去年对此写了一篇详细的文章](https://lex.substack.com/p/fintech-robinhood-acquires-x1-smart); 简言之，X1自称是“有史以来最聪明的信用卡”，是一种无年费、基于奖励的信用卡，提供现代数字界面、创新的基于收入的核批技术以及易于取消订阅和单次使用的虚拟卡功能。它更像是一个“软件应用”而不是“卡”。

Robinhood计划对资产采取的具体措施，特别是它将为市场带来的好处，目前仍未知晓。根据迄今发布的细节，我们认为这种兴奋是正当的。一些关键亮点：

尽管旅行的5%返现非常有竞争力，但无限制的3%所有购买返现设立了新的市场标准。尽管有些卡在特定类别提供更高的奖励，但这些福利通常受到类别限制和年度收入上限的限制。

这在当今市场上应该是基本要求，但添加持卡人可能会令人震惊地痛苦。此外，许多发卡机构不提供用户的详细支出细分。这将家庭信用额度视为软件组件，我们喜欢这种方法。

这是一个很棒的功能，对许多消费者卡来说仍然缺失，尽管新的商业卡已经加入了。正如我们在X1上提到的那样，这允许取消订阅并增加安全性。

+   **作为Visa Signature卡的持卡人，用户自动享有旅行中断保护、延长保修保护、自动租车碰撞豁免等服务。**

虽然不及Visa的顶级产品Visa Infinite，但与Visa Classic和Visa Gold等其他Visa类别相比，仍然是一个优质选择。

现在，几个警告。

这项费用与所得价值相比微不足道。只要您每月的支出达到166美元（166*3% = 5美元），您就能支付月费用。

想要在Robinhood生态系统之外使用返现的用户将不得不想办法将现金转移到支票账户或储蓄账户。显然，该计划旨在成为他们核心经纪活动的客户获取成本，这是其收入来源成熟的地方。虽然表面上这似乎是一项赔钱的举措，但了解信用卡背后的经济模式很重要。以下是影响因素：

+   **利息收入**：信用卡持卡人在未偿账户余额上支付的利息所积累的收入；

+   **交换费**：通常被称为刷卡费，每次使用信用卡进行交易时，卡网络都会向商家收取这笔费用。

+   **年费**：为拥有某些高端信用卡的持卡人收取的费用，以补偿这些卡片具有的额外好处。

+   **其他费用**：此类费用包括一系列收费，包括滞纳金、余额转账费、现金前额费等。

为了获得利润，这些收入来源必须覆盖奖励计划和无法收回的债务所带来的成本。如果管理得当，信用卡就是赚钱的机器。但是，至关重要的是要认识到产生的收入是分配给生态系统中各种利益相关者的。发行商，也被称为发卡行（例如，大通、花旗、巴克莱）获得最大份额，通常获得交换费和利息收入的大部分，因为它们是发放信贷的实体。信用卡网络（例如，Visa、Mastercard）获得更少部分的交换费。然而，由于网络参与的交易数量大大增加，这些费用真的不少——商家在2023年共支付给万事达卡和Visa的手续费总额超过了1000亿美元（https://merchantspaymentscoalition.com/visa-mastercard-credit-card-swipe-fees-hit-record-100-billion-first-time-2023-underscoring-need）。

对于Robinhood来说，该计划的确切经济模型更加复杂。Robinhood并不直接发行信用卡，而是依赖Coastal Community Bank作为发行方。然而，与通常的工作模式不同的是，似乎Robinhood仍然将承担发卡和同意覆盖某些不良贷款的风险。考虑到这一点，尽管确切的收入份额没有披露，但双方的关系性质很可能意味着交换费和利息收入将以某种方式在两个实体之间分配。

信用卡推出是Robinhood持续推动增长的一部分，其产品包括24小时交易服务、IRA账户以及扩展到英国市场。

Robinhood正在执行“占领并扩展”战略，吸引顾客，然后向他们交叉销售金融科技捆绑服务以增加ARPU。这有几个好处：

+   **增加客户价值**：使公司能够从每个客户身上提取多种产品的最大价值。

+   **客户保持**：从同一公司获得多种产品或服务的客户更融入其生态系统，使他们更“黏性”，不太可能转向竞争对手。

+   **降低获取成本**：获取新客户是昂贵的，涉及营销、销售努力和入职流程。相比之下，向已经熟悉并信任机构的现有客户销售，成本要低得多。

+   **增强客户洞察**：让客户使用多种产品使公司能够收集更多关于他们的数据，从而创建更有效的营销和风险管理。

+   **收入多元化**：交易收入可能是周期性的，而信用卡收入则更为稳定，在利率较高时可能更为有利可图。

罗宾汉需要从每位客户身上获得最大的价值，因为自2021年以来他们的月活跃用户（MAU）一直在下降。该公司在2021年拥有超过2000万用户，而2023年显示约1000万用户 —— 减少了50%。

尽管总体月活跃用户数正在下降，金牌会员正在增加，2023年增加了28万人，2024年前两个月增加了9万人。定期订阅收入是对市场波动的解药，可能是罗宾汉未来发展的最关键因素。正如首席财务官杰森·沃尼克在公司2023年第四季度收益网络研讨会上所讨论的，除了产生每年8500万美元的会员费收入外，金牌会员以更高的速度采用产品，并持有大约是普通用户的8倍资产，即$40,000。

将信用卡引入这一组合中对订户来说是一个引人注目的建议。金卡与金牌会员之间的联系将加速会员增长，从会员费中获得更多收入。

然而，一个关键挑战仍然存在 —— 罗宾汉如何有效地向其生态系统中的这些新成员进行交叉销售。信用卡的吸引力可能会吸引一种与现有金牌会员不同的人群，这可能会复杂化交叉销售的努力。新的信用卡用户会带来$40,000到交易平台吗？罗宾汉似乎愿意在一段时间内返还3%现金以测试这一假设。
