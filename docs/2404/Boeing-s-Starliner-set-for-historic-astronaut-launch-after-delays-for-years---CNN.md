# [波音的星际客船](https://www.cnn.com/2024/04/25/world/boeing-starliner-launch-spacex-delays-scn/index.html)在历经多年延误后，终于准备进行历史性的宇航员发射。

> 来源：[https://www.cnn.com/2024/04/25/world/boeing-starliner-launch-spacex-delays-scn/index.html](https://www.cnn.com/2024/04/25/world/boeing-starliner-launch-spacex-delays-scn/index.html)

*订阅CNN的奇思妙想科学新闻简报。*[*通过有关神奇发现、科学进展及更多内容的新闻探索宇宙*](https://www.cnn.com/newsletters/wonder-theory?source=nl-acq_article)*。*

CNN  —

经历多年的延误和测试飞行中的一系列挫折后，波音的星际客船最终将进行首次载人发射。

该任务终于接近其历史性的宇航员发射尝试，NASA官员已经批准在美国东部时间周一晚上10:34进行发射。星际客船将搭载NASA的苏尼·威廉姆斯和布奇·威尔莫尔前往[国际空间站](https://www.cnn.com/2024/02/13/world/mira-robot-first-surgery-in-space-scn/index.html)，这可能标志着备受折磨的波音计划的一个重大而又期待已久的胜利。

“设计和开发是艰难的——尤其是对于载人航天器。” 波音副总裁兼星际客船项目经理马克·纳皮在星期四的新闻发布会上表示。“沿途遇到了许多意外，我们不得不克服。……这确实使团队变得非常强大。我为他们克服了每一个问题并带领我们走到了这一步感到非常骄傲。”

如果成功，星际客船将与[SpaceX的Crew Dragon飞船](https://www.cnn.com/2023/09/03/world/spacex-nasa-crew-6-undock-return-scn/index.html)一起定期飞往空间站，保持轨道站的宇航员人数充足，这些宇航员来自NASA及其合作伙伴空间机构。

美国航天局早就期待已久的场景——龙飞船和星际客船定期飞行。

“这是历史性时刻，” 美国国家航空航天局局长比尔·尼尔森在3月22日的[新闻发布会](https://www.youtube.com/watch?v=7L8wOkah06c)中谈到即将进行的星际客船任务时说。“我们现在正处在太空探索的黄金时代。”

SpaceX和波音在NASA的[商业载人计划](https://www.nasa.gov/humans-in-space/commercial-space/commercial-crew-program/commercial-crew-program-overview/)下开发了各自的飞行器，这是与私营行业承包商的合作伙伴关系。从一开始，航天局就旨在让两家公司同时运行。Crew Dragon和Starliner飞船将各自作为对方的备份，即使一艘飞船因技术问题或其他挫折停飞，宇航员也能继续飞行。

然而，最初，NASA并未预料到SpaceX的龙飞船将在波音的星际客船进行首次载人测试飞行前独立运行近四年。

在项目最早期，NASA在2014年分别[授予了SpaceX和波音合同](https://www.nasa.gov/news-release/nasa-chooses-american-companies-to-transport-u-s-astronauts-to-international-space-station/)，当时NASA更偏向波音 —— 这是NASA从20世纪中期开始的紧密合作伙伴 —— 而不是相对年轻和变幻莫测的后起之秀SpaceX，这也是联邦机构的看法。

### 波音、SpaceX和NASA的愿景

即使在2016年，NASA仍计划着其进程，希望[星际客机能够比龙飞船更早到达发射台](https://www.cnn.com/2019/11/16/tech/spacex-boeing-nasa-oig-scn/index.html)。

但是到了2020年，波音和SpaceX之间的竞争明显出现了转折。[星际客机的飞行测试遭遇了许多问题](https://www.cnn.com/2020/04/06/tech/boeing-starliner-spacecraft-flight-test-repeat-scn/index.html)，导致NASA和波音官员匆忙查明问题所在。由于一些软件问题，包括宇宙飞船内部时钟的问题，星际客机在那次任务中未能与空间站对接，时钟偏差了[11小时](https://www.cnn.com/2019/12/22/tech/boeing-starliner-landing-deorbit-scn/index.html)。

与此同时，SpaceX在2020年5月创造了历史，发射了其演示2号测试飞行，搭载了宇航员鲍勃·贝恩肯和道格·赫利进行了为期两个月的国际空间站任务。

自那时起，SpaceX的龙飞船已经开始进行例行航行，运送NASA的宇航员甚至付费客户和旅游者。该宇宙飞船现在已经进行了13次载人任务。

然而，波音在过去几年中一直在应对一系列挑战，包括2022年在[其第二次无人飞行测试期间发现的问题清单](https://www.cnn.com/2023/06/01/world/boeing-mission-to-send-nasa-astronauts-to-space-station-faces-more-lengthy-delays/index.html)。波音的商用飞机部门还面临一系列丑闻，包括[737 Max危机](https://www.cnn.com/travel/boeing-737-max-passenger-boycott/index.html)以及今年1月阿拉斯加航空公司飞行中[飞机门窗盖脱落问题](https://www.cnn.com/2024/01/08/us/alaska-airlines-plug-door-found-investigation-monday/index.html)，这些事件都严重损害了公司的品牌形象。

NASA官员甚至在2020年某个时候承认，他们已经更加关注SpaceX及其非正统的方法，而对波音星际客机的问题却未被重视。

“也许我们在这个过程中没有像应该的那样有足够的人员参与其中，”NASA商业乘务计划经理史蒂夫·斯蒂奇在2020年7月的新闻发布会上说。

“当一个供应商（SpaceX）比另一个采用更新的方法时，人们往往更倾向于花更多时间研究这种更新的方法，也许我们没有花足够的时间去研究（波音的）更传统的方法。”

波音的航天部门与其商业航空团队分开运营，NASA和美国航空航天巨头的官员们通常试图强调这一区别。

NASA官员还明确表示，他们与波音的合作比以往任何时候都更紧密，波音已经在即将到来的Starliner飞行前实施了一些修复工作，地面上的人员正在波音设施内监督这些修复工作。

纳皮周四表示：“这对NASA来说是一项重要的能力。我们签约去做这件事，我们会成功地完成它。”“我不认为这对波音来说有多重要，我更多地考虑这对该计划的重要性。”

尽管如此，波音和NASA还有一长串问题需要解决。

例如，在2022年的最后一次飞行测试中，工程师们发现，Starliner降落伞的悬挂线的故障阈值比最初预期的要低。

今年早些时候，NASA和波音的工程师为这个问题测试了一个解决方案，但降落伞仍然是他们在发射前进行最后一分钟检查时关注的重点，斯蒂奇周四表示。

据纳皮称，一些用于保护布线束的胶带也被发现具有易燃性，波音不得不移除并更换大约一英里的材料。

波音甚至可能需要对宇宙飞船的一些阀门进行重新设计，因为存在腐蚀问题。然而，该升级预计最早要到2025年的第二次载人飞行才能完成。

在五月的首次载人飞行中，波音将使用一种“完全可以接受的缓解措施”，这应该可以防止阀门粘住，纳皮在三月份表示。

### Starliner和安全问题

尽管通向发射场的道路很长，Starliner首次载人任务的中心人物 — 威廉姆斯和威尔莫尔，两位长期在NASA工作的宇航员 — 在抵达发射场时表示他们与以往一样充满信心。

“我们希望公众认为这很容易，但实际上并非如此 — 它非常困难，”威尔莫尔在周四抵达佛罗里达Starliner发射场后说。“如果我们没有准备好，我们就不会在这里。我们已经准备好了，宇宙飞船也准备好了，团队也准备好了。”

威尔莫尔在三月的一个新闻发布会上提到，他并不指望Starliner宇宙飞船会出现任何“故障模式”。

“但是如果发生了什么事情——因为我们都是人类，我们不能完美地建造事物——如果发生了什么事情，我们有几种降级模式，”他在[新闻发布会](https://www.youtube.com/watch?v=WEjMIv6291g)中说道，这些模式赋予了宇航员在计划出现问题时能够更多地手动控制飞船的能力。

威廉姆斯在三月的一次新闻活动中表示：“如果我们对这艘飞船以及我们控制它的能力没有信心，我们就不会坐在这里，并告诉我们的家人我们对此感到有信心。”

她在佛罗里达州的周四新闻简报中补充道：“我对我们的能力和飞船的能力不仅充满信心，还对我们的任务控制团队充满信心，他们已经为这个挑战做好了准备。”
