# 一个男子的艺术仙境，在租住的公寓秘密创造，获得了保护地位 | CNN

> 来源：[https://www.cnn.com/2024/04/07/style/rental-apartment-artwork-listed-gbr-scli-intl-gbr/index.html](https://www.cnn.com/2024/04/07/style/rental-apartment-artwork-listed-gbr-scli-intl-gbr/index.html)

伦敦 CNN —

一个租赁房产被这位生活了三十年的古怪[艺术家](https://cnn.com/2024/04/01/style/janet-sobel-abstract-surrealist-untold-art-history/index.html)秘密改造后，五年后英国政府正式宣布其[受保护](https://cnn.com/2024/03/21/asia/philippines-chocolate-hills-bohol-resort-controversy-intl-hnk/index.html)。他去世后。

不久之后，罗恩·吉廷斯于2019年9月去世，享年79岁，他的家人访问了他自上世纪80年代中期以来一直居住的公寓，位于利物浦西北郊区的伯肯黑德。

他们在里面发现的东西让他们完全惊叹。吉廷斯几乎在他的家中的每一个可用表面上装饰了装饰品，并绘制了多幅描绘历史场景的壁画。这些艺术品包括描绘古埃及和乔治亚时期英格兰的绘画，以及形状为狮子咆哮头和巨型米诺陶洋的壁炉，甚至还有一个罗马面包烤箱。

这一发现尤其令人惊讶，因为吉廷斯虽然未接受过正规艺术培训，却大多数时候不鼓励人们来参观他的家，一个位于维多利亚双层公寓一楼的公寓，包括三个主要房间、走廊、厨房和浴室。

根据他的侄女简·威廉姆斯所说，他去世后，这一切变得清晰了。

“这栋房子简直乱成一团，甚至无法通过门口进入，”她在电话采访中告诉CNN。

威廉姆斯说她的叔叔“浮华”和“非常古怪”，经常穿着戏服出现在镇上，搜寻一些他会装载到购物车里的东西，例如水泥袋。

“这就是家人没有经常看到他的原因之一，”威廉姆斯说道，她补充说她最后一次见到他是在他去世前一年。

“他的行为非常具有挑战性，但我们仍然爱他并且非常敬重他。”

公寓的每个房间都堆满了袋子、盒子、书籍、艺术材料、食品包装以及许多手写笔记，其中一些是用代码写的。

不久之后，威廉姆斯和她的伴侣克里斯·蒂斯代尔，两人都是艺术家，并经营着一个移动展览空间，发起了一个运动来拯救“罗恩的地方”，得到了朋友、亲戚、艺术家及其他文化遗产界人士的支持。

他们成立了威勒尔艺术与文化社区土地信托（WACCLT），通过网站[Ron’s Place](https://ronsplace.co.uk/)发起了众筹运动，申请了列名地位，并在去年最终购买了这栋建筑物。

现在，这处物业已经被英国文化、传媒和体育部授予了“二级”保护级别，根据英国国家遗产机构历史英格兰的建议。

英国历史英格兰的一位发言人在给CNN的电子邮件中解释了这一登记的重要性，称：“将Ron's Place列入英格兰国家遗产名单使我们能够突出和庆祝这个非凡地方的重要性，并帮助确保对其未来的任何改动不会导致其重要性的丧失。”

“Ron肯定会欢欣鼓舞的，”Williams说道，她的叔叔去世后。

“他是一种表演艺术家，他打造的公寓就像他的舞台和私人世界。”

“他确实有一些心理健康问题，经历了一段时间的不稳定。我认为正是他的艺术作品和创造力支持着他继续前行。”

据Williams称，该物业目前正在进行工作以“稳定和保护”艺术品，同时信托基金还希望改造建筑中的其他公寓，打造一个“全面的艺术与创造之家”。

空间随后将供其他艺术家使用，而进一步的计划可能还包括开发花园和地窖。

“这件事非常令人感动，因为在他去世后我整理他的东西时，我发现了这张给我的明信片，上面写着‘我真的很想给你看看我做的东西’，但地址写错了，我从未收到过。现在我几乎可以弥补当时的遗憾了。”她说道。

Ron’s Place是英格兰第一个获得保护地位的离群艺术典范，根据历史英格兰的说法。

根据泰特美术馆的描述，国际公认的离群艺术被定义为“具有天真质感的艺术，通常由未接受过艺术家培训或在传统艺术结构中工作的人创作。”它并非为观众而创作，通常是为了艺术家自己。

历史英格兰在新闻稿中表示：“此次登记认可Ron的作品为英格兰大型离群艺术的典范，这是一种由个人愿景驱动的创造现象，通常在无形训练和独立于主流艺术界影响之外进行，往往以强迫性方式完成。”

皮尔普乐队的主唱兼Ron’s Place保护运动的赞助人贾维斯·考克尔也是Gittins作品的众多粉丝之一。

考克尔在历史英格兰发布的一份声明中对消息做出了反应，他也是Ron’s Place的赞助人之一，称：“这个星球上很少有人知道Ron's Place是一个非常特别的地方——从现在开始它是正式的：Ron's Place已获得列名地位！

“英格兰北部一个独特绅士的工作已经得到了全国甚至全球的认可。哈利路亚！！”

Williams对她叔叔的成就感到自豪，并对他对未来几代人的影响充满希望。

“他鼓励我认为创意和非传统的做法是完全正常的，我为此感激不尽，”她说。“他从未真正坚持一份工作，长期以来也很少有稳定的收入，但他确实拥有丰富的想象力和创造力。我们希望与他人分享这一点，并鼓励他们发挥创意和智慧，看到周围他人的潜力，以及你能从中创造出什么。”
