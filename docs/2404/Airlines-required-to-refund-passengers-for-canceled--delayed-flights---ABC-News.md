# 航空公司必须为取消和延误航班的乘客退款 - ABC新闻

> 来源：[https://abcnews.go.com/Politics/airlines-give-automatic-refunds-canceled-flights-delayed-3/story?id=109573733](https://abcnews.go.com/Politics/airlines-give-automatic-refunds-canceled-flights-delayed-3/story?id=109573733)

对于航空旅客来说是个好消息：交通部周三宣布，将推出新规定，要求航空公司自动为取消和严重延误的航班乘客提供现金退款。

"这对美国的飞行公众来说是一个重要的日子，" 交通部长皮特·布蒂吉格在周三上午的新闻发布会上说道。布蒂吉格表示，新规定 —— 包括要求及时退款 —— 是该部门历史上乘客权利扩展的最大规模。

根据新的交通部规定，航空公司不再决定延误多久后才发放退款。该机构表示，涵盖的延误时间为国内航班超过三小时和国际航班超过六小时。

这包括从航空公司、旅行代理以及Expedia和Travelocity等第三方网站购买的机票。

交通部规定，如果航班取消或显著改变，乘客将“有权获得退款，而且不接受提供的替代交通或旅行积分”。

一名人士在弗吉尼亚州阿灵顿的罗纳德·里根华盛顿国家机场的航站楼内行走，周三，2023年1月11日。

如果您的行李丢失并且在12小时内未送达，交通部还将要求航空公司给予现金退款。

根据交通部的新规定，退款必须在七天内发放，必须以现金形式发放，除非乘客选择其他形式的补偿。当消费者有权获得现金时，航空公司不再以券或积分形式发放退款。

航空公司将有六个月时间来遵守新规定。

美国交通部长皮特·布蒂吉格在2024年4月24日在里根国家机场举行的新闻发布会上发表讲话。

ABC新闻, POOL

"乘客在航空公司欠款时应获得退款，无需头疼或讨价还价，" 皮特·布蒂吉格在一份声明中说道。

交通部表示，它还在制定关于家庭座位费用、增强轮椅乘客的安全和尊严旅行权利以及要求航空公司在航班因其原因延误或取消时提供补偿和服务的规定。

布蒂吉格表示，交通部还在保护航空公司乘客免受隐藏费用的惊吓 —— 他估计这一举措每年将为美国人节省数十亿美元。

[运输部规定](https://www.transportation.gov/briefing-room/biden-harris-administration-announces-final-rule-requiring-automatic-refunds-airline)包括乘客将获得为未提供的额外服务（如Wi-Fi、座位选择或机上娱乐）支付的退款。

这些规定是在该机构向[西南航空公司处以创纪录的1.4亿美元罚款](https://abcnews.go.com/International/southwest-airlines-fined-record-140-million-dot-2022/story?id=105733507)，因其在2022年假日旅行季节的[运营崩溃](https://abcnews.go.com/Business/stranded-southwest-customers-details-exhaustive-efforts-home-amid/story?id=95848764)之后发布的。

布蒂吉格说，西南航空公司的罚款为航空公司和乘客权利设定了“新标准”。

“明确地说，我们希望航空业蓬勃发展。这就是为什么我们在帮助他们度过疫情期间投入了如此多资源，老实说，这也是我们在乘客保护方面如此严格的原因。”他说。

布蒂吉格重申，退款要求已经是航空公司的标准，但新的运输部规定将追责航空公司，确保乘客得到“应得的退款”。

“航空公司对我们提高标准并不热衷，”布蒂吉格说，并补充道，“我知道他们能够适应这一点。”

美国航空公司协会代表该国主要的客运和货运航空公司告诉ABC新闻，其成员公司“提供各种选择，包括全额退款机票。”他说消费者在“第一次搜索结果时可以选择最符合其需求的退款机票选项及相关条款。”

该组织表示，美国最大的11家航空公司在2020年至2023年间为客户退还了430亿美元，仅去年一年就退还了近110亿美元。
