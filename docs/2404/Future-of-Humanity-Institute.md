# 未来人类研究所

> 来源：[https://www.futureofhumanityinstitute.org](https://www.futureofhumanityinstitute.org)

**（2005–2024）**

成立于2005年，最初为期3年，未来人类研究所是牛津大学的跨学科研究小组。 由Nick Bostrom教授创立，汇集了来自哲学、计算机科学、数学和经济学等学科的精选研究人员，研究人类文明的大局问题，试图保护这些问题免受普通学术压力的影响，并创造有利于创意和知识进步的组织文化。

在其19年的存在期间，未来人类研究所的团队做出了一系列研究贡献，这些贡献有助于改变我们关于未来的对话，并促成了几个新领域和范式的形成。 FHI参与了广泛的思想发展，包括存在风险、有效利他主义、长期主义、人工智能对齐、人工智能治理、全球灾难风险、宏大未来、信息危害、单方主义者的诅咒和道德不确定性等。 它还在人类学、人类增强伦理学、系统风险建模、预测和预测市场、寻找外星智能以及关键未来技术的属性和战略影响方面做出了重要工作。 其中一个主要贡献在于表明，对人类未来的大局问题进行严格研究是可能的。

随着时间的推移，FHI在哲学系（研究所的组织性归属地）面临越来越多的行政阻力。 从2020年开始，学院对筹款和招聘实施了冻结。 到2023年底，哲学系决定不再续签剩余FHI员工的合同。 于2024年4月16日，该研究所关闭。

在其19年的历程中，FHI激发了一个充满活力的组织生态系统的出现，这些组织可以探索FHI调查的那些问题。 FHI的校友将继续在牛津以及世界其他地方研究这些问题。 曾经在单个哲学系边缘艰难生存的主题，现在被领先的人工智能实验室、政府机构、非营利组织和专门的学术研究中心追求（许多正在创建中）。
