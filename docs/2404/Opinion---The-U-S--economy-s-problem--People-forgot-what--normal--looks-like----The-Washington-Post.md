# 美国经济的问题？人们忘记了“正常”是什么样子。 - 华盛顿邮报

> 来源：[https://www.washingtonpost.com/opinions/2023/12/02/us-economy-2024-recovery-normal/](https://www.washingtonpost.com/opinions/2023/12/02/us-economy-2024-recovery-normal/)

美国经济今年是一个奇迹般的一年。通胀率大幅下降，而没有引发经济衰退。许多[专家说](https://www.washingtonpost.com/business/2023/08/13/larry-summers-biden-inflation-projections/?itid=lk_inline_manual_2)这种情况不可能发生，除非普遍裁员和经济衰退。截至目前，今年已经新增了240万个工作岗位，经济增长加快，第三季度年化增长率超过5%。这些好消息也推动了[股市的反弹](https://www.cnn.com/2023/11/28/investing/premarket-stocks-markets-2024/index.html#:~:text=The%20S%26P%20500%20currently%20sits,was%20reached%20in%20January%202022)。民意调查显示，人们对当前经济持悲观态度，但他们的行动却与言辞不符。今年出现了[消费热潮](https://www.nytimes.com/2023/11/20/upshot/economy-voters-poll.html)，美国人继续大量消费服装、音乐会和度假。从多个方面来看，今年经济终于回归到接近正常状态。但许多人似乎已经忘记了经历了几年的创伤之后正常的模样。
