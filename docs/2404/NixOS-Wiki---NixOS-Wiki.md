# NixOS Wiki - NixOS Wiki

> 来源：[https://wiki.nixos.org/wiki/NixOS_Wiki](https://wiki.nixos.org/wiki/NixOS_Wiki)

这是官方的 NixOS Wiki。它涵盖了 [NixOS 手册](https://nixos.org/nixos/manual) 未涉及的主题：NixOS 用户指南、配置示例和故障排除技巧。

NixOS 是基于 Nix 包管理器和 Nixpkgs 软件包仓库的 Linux 操作系统。

新手可以阅读[Nix 核心生态系统](/wiki/Nix_Ecosystem "Nix Ecosystem")概述文章，以获取全面的入门指导。如需更详细的介绍，请参阅：

如果你希望在其他 Linux 发行版或 Mac 上单独尝试 Nix，请从 [nix.dev](https://nix.dev) 入手。

很多人使用[Home manager](/wiki/Home_manager "Home manager")来管理他们的主目录内容，许多人使用[Flakes](/wiki/Flakes "Flakes")代替 NixOS 的半年发布通道。这两者增加了复杂性，你可能甚至不需要它们。从 NixOS 开始，只有在你对 NixOS 感到自信并理解它们的重要性后，才将 Home Manager 和/或 Flakes 集成到你的设置中。
