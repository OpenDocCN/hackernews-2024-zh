# 英格丽德·罗贝因斯的《限定主义》为设定财富上限提出了论据 - 大西洋月刊

> 来源：[https://www.theatlantic.com/family/archive/2024/04/ingrid-robeyns-limitarianism-makes-case-capping-wealth/677925/](https://www.theatlantic.com/family/archive/2024/04/ingrid-robeyns-limitarianism-makes-case-capping-wealth/677925/)

二月份，现年93岁的露丝·戈茨曼（Ruth Gottesman），前阿尔伯特·爱因斯坦医学院教授，也是金融家大卫·戈茨曼的遗孀，宣布向她的母校捐赠10亿美元。捐款附带的指示是：这笔钱将永久用于布朗克斯地区的学生免费教育。

这笔捐款受到了赞扬，当然，除了其规模之外，还因其人道主义色彩。正如《纽约时报》专栏作家吉尼亚·贝拉法特所[称](https://www.nytimes.com/2024/02/29/nyregion/billion-dollar-gift-einstein-medical-college.html?searchResultPosition=2)，戈茨曼的慷慨“传达了一个亿万富翁如何过上最好的生活的信息——不需要改造火星，不需要参加彩虹之旅，也不需要试图秘密控制哈佛大学”。

资助一所经费不足的医学院显然比再买一艘超级游艇更明智。但同样令人震惊的是，一个如此具有社会影响力的决定，解决成千上万潜在医生的债务负担，竟然取决于一个人的心血，而且这个人有足够的资源独自实施这样的政策，无需任何其他人的参与或批准。

[阅读：杰夫·贝佐斯的主张](https://www.theatlantic.com/magazine/archive/2019/11/what-jeff-bezos-wants/598363/)

戈茨曼的财富相对较为适中，比起日益增长的超级富豪群体，她估计的30亿美元甚至都不足以进入*福布斯*排行榜的前100位，而在那里，像埃隆·马斯克、杰夫·贝佐斯和伯纳德·阿诺等人则在12位数中游戏。如果戈茨曼有能力改变这么多人的生活，那么那些拥有她10到50倍财富的人的力量，对于人们的理解来说难以置信。也许我们应该问问，她——或者任何其他人——是否应该拥有那么多财富。

荷兰哲学家英格丽德·罗贝因斯写道：“很长一段时间，我觉得个人积累如此巨大财富存在问题，但我无法准确表达为什么。”“经过十年分析和辩论极端财富，我逐渐确信我们必须创造一个没有超级富豪存在的世界——任何一个人的财富都必须设有上限。我称之为*限定主义*。”在她的同名书籍中，罗贝因斯阐述了限定主义的理由，同时颠覆了常见的代理权、所有权以及财富*真正*意味着什么的观念。

极端财富使穷人继续贫穷，她认为这扩大了不平等。超级富豪通过其巨大的政治影响力破坏了民主，并通过奢华的生活方式破坏了气候。其中一些财富是通过可疑手段获取的——来自剥削性商业实践，或是逃避税款，甚至是公然偷窃。罗贝恩斯认为，没有人应该拥有如此过度的财富，道德和心理上，人们没有这些财富会更好，并且社会过剩的资源有更好的用途——比如消除贫困或改善基础设施。即使是出于良好意图的慈善行为也无法弥补这些缺陷：它不能代替运作良好、资金充足的政府——而这种政府往往在富人积累财富的过程中被削弱。

政府及其公民决定了任何财富的命运。“把任何一位千万富翁或亿万富翁放在荒岛上，”罗贝恩斯写道，“他们仍然拥有与之前相同的所有才能和个人特质。他们能变得多富？显然不会很富。”

[阅读：取消亿万富翁](https://www.theatlantic.com/ideas/archive/2019/10/cancel-billionaires/599587/)

任何个人的财富都依赖于社会资源、努力和合作。然而，如今，即使是亿万富翁也是利用更广泛社会的资源来积累财富——从顾客、员工和公共基础设施中获取利润；受到政府监管和国际协议的保护——他们能够单方面决定按照自己的意愿塑造社会，而这个社会却很少能够提供任何输入。

罗贝恩斯提出了个人财富的两个上限。她认为，大多数拥有稳固社会保障网的国家应该将1000万欧元（约合1080万美元）的上限纳入其社会和财政体系中。作为伦理指南，个人应该将自己限制在100万（在不那么安全的美国可能是500万美元，在那里，一张不及时的医院账单就足以使一个家庭破产）。她也感慨地指出，这两个提议的数字也比一些哲学家理想中的更不严格：例如，在《法律》中，柏拉图认为最富有的人不应该比最贫穷的人拥有的财产多四倍。

数字在某种程度上是随意的且依赖于上下文，但精确的金额不如在社会上认可的上限起到重要作用——在成为合理富有和非道德超级富有之间画一条线。在某一点后，额外的钱对个人带来的边际效用递减——相反，罗宾斯建议，这些多余的资金应用于解决社会上最迫切和未解决的需求，“重新分配给那些非常贫困的人或用于资助造福我们所有人的公共物品。”

当然，任何限制很可能在没有对我们的税收和治理框架进行重大改变的情况下难以实施。[批评者](https://www.heritage.org/taxes/commentary/democrats-proposed-wealth-tax-spells-doom-entrepreneurs-and-economic-growth)称，这样的政策几乎不可能实施——即使可以，对潜在财富设定上限也会失去动力，抑制创新。此外，其他人可能会对个人所赚取的东西（尽管如罗宾斯所言，没有人在真空中“赚取”任何东西）进行分配，这可能会让人感到威胁。政府能否像我们期望的那样有效地使用这些钱？我们真的信任我们的同胞做出明智的决策吗？

但是，罗宾斯对“极端财富的案例”吸引人之处并不在于实施的复杂性。相反，它挑战了人们对所有权以及许多人如何衡量自身价值的常常未经审视的信念。限制论质疑了个人财富是否真正属于个人。

罗宾斯认为，绝大多数极端财富来自偶然因素：富裕的父母，恰到好处的遗产，或者像智力和创造力这样不应得的天赋。虽然利用自己的资产和动力是公平的——有些人比其他人工作更努力，在限制论体制下，不平等不会被消除——但这样做不应导致今天我们看到的巨大差距。

[阅读：美国的财富已经破碎](https://www.theatlantic.com/family/archive/2019/07/the-wealth-gap-taints-americas-success-stories/593719/)

然而，意识到运气在我们成功中所起的作用有多大，以及多少财富是不应得的，可能会带来动荡，这也是为什么最幸运的人可能会[完全避免谈论钱](https://press.princeton.edu/books/hardcover/9780691165509/uneasy-street)。他们经常重复表明，[只需一点努力](https://www.nbcnews.com/pop-culture/celebrity/kim-kardashian-told-women-get-work-people-are-saying-s-hypocritical-rcna19547)，[一种狂热的早晨日常](https://www.ft.com/content/1a4d33e1-43fc-4387-a816-dd081ee53175)，和紧紧抓住我们的靴带，任何劣势都可以克服。

毕竟，美国长期以来理想化自力更生的人，无论他是粗犷的边界人还是卡内基式的大亨。有些美国人认为，对个人经济自由的限制是对[政治自由的威胁](https://www.newyorker.com/magazine/2023/07/24/the-rise-and-fall-of-neoliberalism)，认为[政府天生浪费](https://www.pewresearch.org/politics/2019/12/17/views-of-government-and-the-nation/)，并且我们可以通过我们取得的成就来定义我们的价值。

美国只有极少数人能达到500万美元的净值门槛——事实上，今天美国的平均年薪是[$59,384](https://www.usatoday.com/money/blueprint/business/hr-payroll/average-salary-us/)。（相比之下，亿万富翁杰夫·贝索斯的净资产每分钟增加大约[$59,000](https://finbold.com/how-much-money-does-jeff-bezos-make-a-second/)。）限制主义政策不会实质性影响大多数人的财富。但将过多的财富用于亲社会目标——用于支付更强大的社会安全网和更好的公共资源，缓解气候变化，或者结束饥饿——将有助于每个人感到更安全。

尽管如此，设定财富上限的想法在直觉上令人不安，因为这与美国文化中一些最深层次的信念相矛盾。如果极端财富不再是向往的目标，那我们还应该追求什么？如果失去了物质利益的可能性，我们会失去所有的抱负吗？如果我们所拥有的并非自己赚来的，我们如何衡量自己的价值？

罗贝恩斯的书在挑起这种不适感中，及时地增加了对极端财富的讨论，颠覆了某种程度上可能使人更容易接受的（远没有那么戏剧性的）转变。人们普遍认为美国的不平等程度太高。年轻一代对资本主义的看法已经不如他们的父母那样[积极](https://www.pewresearch.org/politics/2022/09/19/modest-declines-in-positive-views-of-socialism-and-capitalism-in-u-s/#:~:text=While%20younger%20adults%20are%20more,of%20those%2065%20and%20older.)，并且更支持重新分配政策。乔·拜登总统的[最新预算提案](https://home.treasury.gov/system/files/131/General-Explanations-FY2025.pdf)要求对拥有超过1亿美元财富的美国人征收25%的最低所得税，部分用于资助社会保障和医疗保险等政策，旨在缩小收入不平等。

政策变革是必要的，但更重要的是心态的改变。“我们不仅需要制度设计和财政选择；我们还需要发展一套文化根植的公共价值观，在这里物质利益不是主要激励”，罗贝恩斯写道。“我们必须重新平衡我们对社会的看法，以及我们作为人类的看法。”

一个有财富限制的社会将不得不培养新的愿望。现在正是开始梦想新梦想的好时机。
