# 大学生应该更多地学习 - 马修·伊格莱西亚斯（Matthew Yglesias）

> 来源：[https://www.slowboring.com/p/college-students-should-study-more](https://www.slowboring.com/p/college-students-should-study-more)

在去年一系列大制作电影中，亚历山大·佩恩（Alexander Payne）的静默时期悲剧喜剧《The Holdovers》被认为是有点轻微的，尽管它受到好评。但尽管它没有任何字面上的核爆炸或戏剧性的政治演讲，该片却探讨了我们时代一个重要的社会主题之一：精英教育的价值主张的结构性转变，每当我看到新的校园争议时我都会想到。

这并不完全是电影的情节（这里没有剧透），而是其背景。保罗·吉亚玛蒂（Paul Giamatti）扮演的保罗·亨厄姆（Paul Hunham）是1970年代虚构的巴顿学院（Barton Academy）的一名教师。巴顿是一所精英男子寄宿学校，尽管学生大多非常富裕，但住宿条件并不特别好。

这类学校的伦理之一是学生们生活在相当简朴的条件下，远离父母豪华的生活方式。而教授古代历史的亨厄姆是一位特别严格的老派教师。他对纪律和学习有着高标准。他布置了大量阅读任务，期望学生完成，并如果他们没有完成就给予他们差评。他希望接受差评的学生承担后果。在他对自己和他的工作的理解中，这是精英教育机构的角色：富裕的父母聘请巴顿来让他们的孩子接受严格的训练，因为他们认为这对学生的长远发展有好处。学校提供一种服务，而他们提供的服务的一部分是严酷性。

至少按照亨厄姆的说法，这些是巴顿学院建立的价值观，也是雇佣他的前任校长相信的价值观。

但是亨厄姆与*新任*校长对立，后者认为他的工作是让富有和有影响力的父母满意，这意味着要做他们想要的事情，而不是让他们心爱的孩子挂科。新校长的方法虽然不值得赞扬，但似乎是对市场的分析正确评估 — 巴顿是一所需要捐助者和付费顾客的私立学校。也许曾经有过捐助者和顾客想要亨厄姆所提供的东西的时候，但现在不再如此。在电影中，这与美国精英阶层性格的衰落在主题上有关联。一场设在巴顿礼拜堂的场景横扫过许多毕业于巴顿的校友纪念碑，他们在美国历史的各种战争中丧生，包括许多在世界大战中丧生的校友。但只有一名巴顿学生在越南战争中丧生，一位黑人孩子，因为他的母亲在学校的餐厅工作而获得奖学金。我们得知，这位孩子想上大学，但没有钱，所以他参军了。

这不是电影的“主题”。但这是背景，是50年前精英教育机构面临的交织交压。

我认为理解当代教育话语的关键是理解这些压力是如何在更加竞争的入学条件中解决的，但对行为和学术表现的标准要低得多。

这种转变的一个主要结果是，当代大学生真的不花太多时间在课业上。不幸的是，在大萧条的低谷期间，当年轻人面临严重经济问题时，这个问题引起了很多关注，这使得问题变得复杂。例如，当《纽约时报》在减少学习努力的问题上进行“辩论空间”时，安雅·卡门茨的回应集中在那些需要**兼职赚钱**的学生。但即使你只看**全日制大学生**，他们每天在课堂内外的教育上也只花不到三个小时。

对学术的忽视产生了影响。2011年，[理查德·阿伦和约西帕·罗克萨出版了《学术漂流：大学校园中的有限学习》](https://www.amazon.com/Academically-Adrift-Limited-Learning-Campuses/dp/0226028569)，当时引起了很大反响，但现在很少被讨论。他们使用了来自大学学习评估的数据，并展示了尽管一些学生在大学学到了很多并显著提高了学术技能，但有很大一部分学生没有。他们指出，实际的学习量因学院专业而异。他们的数据显示，理科/数学专业的学生学到的最多，其次是人文/社会科学专业，然后是工程专业。但是，健康、传播、教育/社会工作和商业学生基本上什么都没学到。

这再次出现在大萧条的背景下，因此很多讨论围绕着进步人士争论这些问题是**青年失业问题的一种分散注意力**。

那时我倾向于同意，并且我认为[阿鲁姆和罗克萨在2014年的后续著作中犯了一个错误](https://www.vox.com/2014/9/4/6101987/aspiring-adults-adrift-roksa-arum-millennials-academically-adrift)，特别是把当时年轻的千禧一代的经济困境归咎于他们的大学教育不足，而不是宏观经济环境。过去十年来，随着宏观经济状况的改善，[千禧一代已经赶上](https://economistwritingeveryday.com/2024/01/24/young-people-have-a-lot-more-wealth-than-we-thought/)。这只是说，阿鲁姆和罗克萨的批评者是正确的 —— 这并不能解释宏观经济结果的大部分内容。尽管如此，考虑到高等教育在美国社会中的角色，他们描绘的许多学生不学习的学校画面仍然很重要。他们表明，学习缺乏与不花时间在学习上相关联，而[菲利普·巴布科克和明迪·马克斯已经表明](https://www.nber.org/papers/w15954)，几十年来，学生们在学习上花费的时间越来越少 —— “1961年全日制学生每周分配40小时用于上课和学习，而到2003年，他们每周大约只投入27小时。”

有很多事情可以尝试解释为什么会这样。但攻克课本的一个主要原因是取得好成绩，随着时间的推移，取得好成绩变得越来越容易，这意味着没有足够的理由去用功了。

当我在哈佛大学在劳伦斯·萨默斯短命的校长任内时读大学时，他大力推动逆转一代人过去发生的成绩膨胀趋势。如果你具体查看我在2000年代初上学时的情况，[你会看到它起作用了](https://www.chartr.co/stories/2023-12-08-2-grade-inflation-at-harvard)。我们这届学生是唯一一批记录显示成绩略有下降的学生。

我要指出，萨默斯几乎什么都没做。他只是用清晰而有力的语言表示，他认为评分过于松懈了。这意味着那些私下里有些狠心的教授们感觉到，如果他们变得严厉一点，他们会得到支持。我曾参加过一个大班课，一位杰出的教授从她的助教的评分堆中拿出几篇论文（包括我的一篇），当着所有人的面重新打分为较低分数。

萨默斯最终因其他争议失去了那份工作，他的继任者没有继续这一努力。因此，早期的零星增长只是图表上的一个怪异现象，是规则的例外。

我认为这是一个重要的例外，因为它表明逆转是可能的。

一些教授强烈同意Summers的观点，并在从管理员那里得到绿灯后立即开始发放更严厉的成绩。如果他们的老板希望他们更加严格，助教们也乐意配合。大多数教授我想不会对此有特别强烈的感觉，只是希望大致上保持平均水平，所以当异常值的行为开始改变时，平均教授的行为也会跟着改变。当然，人们会抱怨。Summers时代的新生儿式镇压正在把A-变成B+，B+变成B。这导致学生们抱怨，但最终，为了恢复老派学术价值观，学校将需要分发C和D等级，这将使学生面临真正的负面后果，如失去奖学金、被开除或进入看起来像真正失败的工作市场。然后你会遇到Hunham所面对的问题：这是学生和他们的父母想要的吗？

正如Catherine Rampell在[2010](https://archive.nytimes.com/economix.blogs.nytimes.com/2010/04/19/want-a-higher-g-p-a-go-to-a-private-college/)和[2011](https://archive.nytimes.com/economix.blogs.nytimes.com/2011/07/14/the-history-of-college-grade-inflation/)年所写，私立学院和大学显然率先推动了成绩通胀的浪潮，而服务大多数学生的公立机构紧随其后。

因为选择效应在教育中非常重要，竞争的主要基础是吸引最高质量的申请者。申请者在某种程度上希望有一个愉快的消费体验。这意味着为学生提供大量非常慷慨评分的课程路径，其中可能会正式分配大量工作，但很容易逃避做这些工作。

众所周知，这在所有课程和学科中并不均衡。

如果你上数学或科学课，你将会得到问题集。如果你不能正确回答问题，你将会得到一个糟糕的分数。根据你的背景和天赋，你可能需要非常努力地才能正确回答问题。话虽如此，我喜欢Hunham的例子，因为它提醒我们人文科学科目并不 *本质上* 容易。你可以给古代文明课的学生一张测试，要求他们准确复制关于伯罗奔尼撒战争的大量详细信息，如果他们掌握不了，就会得到低分。

在传统的文科教育中，你可能期望人们学习用希腊语和拉丁语阅读甚至作文，这似乎是件难事。我在学校曾有一个老师让我们背诵诗歌。这是一个几乎任何人*都能*做到的伟大例子 —— 诗歌的历史起源之一是，出于某种原因，我们的大脑更容易记住它们 —— 但这需要有意义的努力。直到今天，我仍能背诵[《春与秋》](https://www.poetryfoundation.org/poems/44400/spring-and-fall)或[《冰激凌皇帝》](https://www.poetryfoundation.org/poems/45234/the-emperor-of-ice-cream)，但当我上那节课时，这需要了很多时间和练习。

然而，情况的发展是，STEM部门倾向于拥有大量的资助金，不太愿意应付让人讨厌的本科生，因此他们在课程设置上采取了严格的措施来淘汰较弱的学生。

然而，人文学科部门必须通过学生兴趣来证明其相对有限的预算，这意味着他们需要尝试提供人们想要的东西，也就是简单的课程。不一定*必须*如此。人们付费参加团体健身课程，整个目的是让教练挑战学生，推动他们比平时更加努力地工作。如果评分更加严格，学生会学到更多，并且如果他们更加努力工作，他们也会更加努力，“我希望我的孩子在学校学到很多东西”并不是一个特别奇怪的想法。

在过去的50年里，学生及其父母的方向并不是这样。

我认为，美国本科生学习不认真导致严重经济问题的想法已经被事件充分驳斥了。但我仍然认为这是解开许多校园争议的一把万能钥匙。

机构在设定明确使命时运行更加高效。使命有助于界定机构本身重要的事物与机构内工作人员关心的事物之间的界限。它还有助于结构化分歧。如果两个有着截然不同想法的人至少能就他们试图实现的目标达成一致，那就为某种解决方案奠定了基础。使命也是管理多样性的良好方法。有着完全不同背景和价值观的人们可以找到抛开分歧、朝着共同目标合作的方式，*只要*他们有一个共同的目标。

大学本科项目的明确目标似乎与教育学生有关。

但是，大学似乎越来越不愿意要求学生达到高标准的工作。市场已经决定，客户大多对同龄人的影响和愉快时光更感兴趣，学校更善于选择而不是教育也是受到奖励的。这导致学生有大量的空闲时间，他们用各种方式度过 — 我开始写博客，米兰即将成为 [耶鲁日报的评论编辑](https://twitter.com/milansingh03/status/1782131078523728305)，玛雅为 [Crimson 写了一系列优秀的专栏](https://www.thecrimson.com/writer/1219782/Maya_A._Bodnick/)，其他人参与了喧嚣的活动主义，也可能是 [最常见的活动是聚会](https://www.amazon.com/Paying-Party-College-Maintains-Inequality/dp/0674088026) — 而教职员的主要工作是研究。然而，当争议爆发时，确实没有一个教育卓越（甚至是足够）的组织原则可以呼吁帮助找出应该做什么。

即使这并不是经济危机，我也认为这是令人遗憾的。如果没有其他理由，这实际上是很多人时间的巨大浪费，以及一定数量的金钱的非常可观的浪费。

虽然如此，我怀疑，如果年轻人在进入管理职位之前接受一些更深入的智力锻炼，对社会可能是有益的。显然，不能指望每个人都知道德利安同盟是什么，并不是一个危机。但是，通过努力工作和掌握一系列知识，确实有一种内在的启迪意义。这本是我们曾经聘请高等教育来完成的任务，如果没有这些，我们都会有些茫然。
