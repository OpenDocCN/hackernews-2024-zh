# 让纽约地铁更安全的一件科技产品 – The Markup

> 来源：[https://themarkup.org/hello-world/2024/04/06/the-one-piece-of-tech-that-would-make-the-new-york-city-subway-much-safer](https://themarkup.org/hello-world/2024/04/06/the-one-piece-of-tech-that-would-make-the-new-york-city-subway-much-safer)

世界各地好，我是 Ese。

我经常乘坐纽约市地铁：这是我与家人的通勤交通，市中心博物馆之旅和与朋友的晚餐，以及跨区约会和购物之旅的交通工具。

非常幸运的是，我在地下旅行中从未成为暴力犯罪的受害者，这或许只是一些案例数据，对于那些关注我们地铁新闻的人来说可能意义不大。仅在过去的一个月里，地铁已经发生了野蛮的高峰期[枪击](https://www.nytimes.com/2024/03/14/nyregion/subway-shooting-brooklyn.html)，[多起](https://nypost.com/2024/03/26/us-news/convicted-killer-busted-in-nyc-subway-stabbing-after-getting-off-parole-cops/) [刺伤](https://www.cbsnews.com/newyork/news/upper-west-side-stabbing-near-subway-station/) [事件](https://apnews.com/article/nyc-subway-conductor-neck-slashed-brooklyn-b0bd79435518fe378b0aea768d6f62aa)，以及其他随机袭击。

这些可怕的事件引发了官方的激烈反应：上个月初，卡西·霍楚尔州长[下令调动750名国民警卫队士兵](https://www.reuters.com/world/us/new-york-deploy-750-national-guard-soldiers-check-bags-subway-2024-03-06/)在高流量车站检查行李。市长埃里克·亚当斯还在系统中增加了[额外的800名警察](https://apnews.com/article/new-york-subway-fare-evasion-policing-crackdown-7c004e19a41a4739af12103366486082)。

当然，在这里需要强调的是，尽管安全感至关重要，根据现有证据，地铁系统相对来说是安全的。每天有超过[三百万名乘客](https://new.mta.info/agency/new-york-city-transit/subway-bus-ridership-2022)乘坐列车，绝大多数人没有遇到问题。根据2022年中期数据的《纽约时报》最新分析，地铁每百万次乘车约有一起暴力犯罪。自那时起，《时报》[报道称](https://www.nytimes.com/2024/03/06/nyregion/nyc-subway-crime.html?searchResultPosition=7)，“整体犯罪率下降，乘客量增加，使得成为暴力犯罪受害者的可能性更加微小。”

甚至市长亚当斯和纽约市警察局也报告说地铁系统正在变得更安全。上周三，纽约市警察局[宣布地铁犯罪减少](https://www.nyc.gov/site/nypd/news/pr0101/nypd-citywide-crime-statistics-march-first-quarter-2024)，称2024年第一季度总体公共交通犯罪减少了近24%。官员们将报告的事件减少归因于额外的警员在公共交通系统中的工作。

“看到我们辛勤工作的实际成果非常令人鼓舞——我们正在进行的投资显然正在产生回报，”纽约市警察局局长爱德温·卡班在宣布中说。“我们发誓将紧密关注犯罪的驱动因素，以改善每个车站、每列火车的交通安全和安全感，无论是白天还是夜晚。这是纽约人期望和应得的。”

其中一部分投资包括亚当斯对打击犯罪技术的采纳。上周，亚当斯宣布该市将在地铁中试行枪支探测器——是的，很像金属探测器，但专门用于检测枪支。这项试验是亚当斯探索地铁安全技术的最新尝试：去年九月，亚当斯在时代广场与Knightscope公司宣布了一个自主安全机器人的试点项目，时代广场是地铁472个车站中最繁忙的一个。这款机器人重达400多磅，配备摄像头记录图像和视频。它还有一个呼叫按钮用于报告事件。

除了像机器人和枪支探测器这样的打击犯罪技术，我认为有一种技术可以让地铁更安全：在站台边缘设置屏障。

就在上周，一名男子在哈林区被[推到轨道](https://www.nytimes.com/2024/03/26/nyregion/subway-shove-manhattan.html)前，被即将驶来的4号列车撞上。警方称这次致命袭击是无端的。本周，我从一位同事那里听说，又有一名人员在当天早些时候被[火车撞击](https://pix11.com/news/local-news/manhattan/person-struck-by-train-at-28th-street-station-trains-delayed-mta/)。

我想：在2024年，为什么官员们不投资于技术，使得将某人推到火车前或防止在轨道上自杀变得困难？

### 边境巡逻的未来：AI 一直在监视

人权倡导者警告称，依赖AI监控边境可能存在算法偏见、法律违规和其他严重后果。

[2024年3月22日11:00 ET](https://themarkup.org/news/2024/03/22/the-future-of-border-patrol-ai-is-always-watching)

值得肯定的是，由纽约州控制的MTA在2020年2月发布了一份[庞大的3920页报告](https://new.mta.info/document/73241)，内容涵盖了其多年来关于站台屏幕门可行性研究。结果表明，站台宽度、通行无障碍空间、多种列车车型、限制高度以及站台边缘的结构完整性等因素，都使得在这样一个古老系统中全面推广站台屏幕门变得复杂。机构估计，系统范围内推行站台屏幕门项目的可行性为27%，即472个车站中的128个。

价格也是一个挑战。机构发现，仅在这128个车站安装全高闸门的总成本将超过70亿美元，而半高闸门将花费65.3亿美元。一旦安装完成，机构每年将不得不花费超过1.19亿美元来维护这些门。

目前，市内最接近地铁闸门的设施是金属屏幕，这些屏幕是今年早些时候在曼哈顿的一个车站[安装的](https://new.mta.info/press-release/mta-installs-platform-barriers-191-st-1-station)，它们被螺栓固定在地面，但并非机械操作：它们不会滑动打开或关闭，因此人仍然可能通过站台边缘的开放空间掉落或被推出去。不幸的是，在可能有一天我们将拥有枪支探测器和巨型机器人巡逻地下的时间线上，这些钢铁围栏似乎是我们遏制可预防的地铁死亡的最佳工具。
