# 参议院在该监视工具到期后的片刻通过了续期法案

> 来源：[https://www.nbcnews.com/politics/congress/senate-renews-fisa-section-702-spying-privacy-rcna148394](https://www.nbcnews.com/politics/congress/senate-renews-fisa-section-702-spying-privacy-rcna148394)

华盛顿 — 参议院投票重新授权了一项被美国政府视为打击恐怖主义至关重要的强大监视工具，此前击败了左右两派公民自由倡导者限制其权力的努力。

以60-34的投票结果通过法案，将其送交支持者总统乔·拜登。该立法将《外国情报监视法》第702条款（FISA）延长两年。

最终投票在参议院打败了来自进步和保守派参议员的六项修正案，他们认为间谍权力过于广泛，并要求保护美国公民的公民自由和隐私。拜登政府和FISA支持者曾警告称，即使短暂失效也可能对情报收集过程产生不利影响。

参议员们在午夜截止时间错过重新授权《外国情报监视法》第702条款的时刻，但几分钟后投票重新授权。如果采纳了任何修正案，法案将被送回众议院，可能导致法律长时间失效。

“及时雨，这里在参议院中取得了两党合作的胜利，” 纽约州民主党籍参议院多数党领袖查克·舒默说。

“这并不容易，人们有很多不同的看法，但我们都知道一件事：让FISA失效将是危险的。它是我们国家安全的重要组成部分，可以阻止恐怖行为、毒品走私和暴力极端主义，”舒默在参议院发言厅说。“感谢我在两党中的参议院同僚们的共同努力完成这项工作。”

众议院在共和党反对声音稍弱后，上周通过了为期两年的FISA续期法案。在拒绝要求搜索收集的数据中美国人通信需要授权的修正案之后，参议员们通过推迟数天的修正案投票推动了对该法案的修改。

这项法案的通过发生在美国情报界与一个不寻常的进步派和保守派公民自由倡导者联盟之间激烈的斗争之后，他们认为这些权力范围过大，侵犯了美国人的隐私。

俄勒冈州民主党参议员、情报委员会成员和隐私保护的坚决支持者罗恩·怀登说：“人们理解这个法案的广泛影响是很重要的。”“最后一刻插入了一些东西，基本上会强迫像是电缆人这样的人为政府从事间谍活动。他们将强迫这个人去做，并且没有上诉的机会。”

华盛顿州民主党参议员、临时议长帕蒂·默里与舒默和拜登少数意见不合，反对该法案，她说：“我对FISA第702条授权权力的扩展存在严重担忧，因为这可能会导致法律的滥用和误用，侵犯在美国境内的美国人权利。”

弗吉尼亚州民主党参议员、情报委员会主席马克·沃纳对众议院修改的FISA重新授权法案及其他批评进行了反驳，认为其“只是针对一个重大情报空白的狭窄焦点”，但怀登等成员担心可能会被滥用。

沃纳在周三的参议院发言中说：“与一些人所说的相反，该法明确排除了咖啡店、酒吧、餐馆、住宅、酒店、图书馆、娱乐设施以及一系列类似场所。”“与一些批评者坚持的观点完全不同，它绝对不会允许美国政府强迫例如弗吉尼亚北部某办公楼的清洁工为情报界从事间谍活动。”

沃纳表示，如果允许外国情报监视法案（FISA）到期，美国可能会进入“未知领域”，因为与政府合作提供情报的公司可能会在没有重新授权的情况下停止合作。

德克萨斯州共和党参议员约翰·科尔宁称，“总统每日简报中有60%是由702衍生材料组成，所以这绝对至关重要。”

美国司法部长梅里克·加兰称第702条“不可或缺”。

他说：“第702条的这一次重新授权授予了美国继续收集关于位于美国境外的非美国人的外国情报信息的权力，同时还将美国司法部已采纳的重要改革固化，以确保保护美国人的隐私和公民自由。”
