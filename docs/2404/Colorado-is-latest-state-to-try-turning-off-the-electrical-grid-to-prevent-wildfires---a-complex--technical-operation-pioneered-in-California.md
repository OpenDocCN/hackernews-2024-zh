# 科罗拉多州是最新尝试关闭电网以防止野火的州，这是一项复杂的技术操作，最早在加利福尼亚州开创。

> 来源：[https://theconversation.com/colorado-is-latest-state-to-try-turning-off-the-electrical-grid-to-prevent-wildfires-a-complex-technical-operation-pioneered-in-california-227639](https://theconversation.com/colorado-is-latest-state-to-try-turning-off-the-electrical-grid-to-prevent-wildfires-a-complex-technical-operation-pioneered-in-california-227639)

美国电网是[迄今为止建造的最大和最复杂的机器](https://nap.nationalacademies.org/catalog/11735/the-grid-a-journey-through-the-heart-of-our-electrified)。它也在[老化](https://www.energy.gov/gdo/articles/what-does-it-take-modernize-us-electric-grid)并受到[气候驱动灾害](https://www.wilsonquarterly.com/quarterly/strategic-competition/climate-change-is-overpowering-americas-electric-grid)（如野火、飓风和热浪）日益增加的压力。

在过去十年中，电网在多个州的野火中发挥了作用，包括加利福尼亚、夏威夷、俄勒冈和明尼苏达。当风速高且湿度低时，如架空电力线路等电气基础设施可能会碰撞植被或其他构件，引发火灾，随后被强风蔓延。

在极端条件下，电力公司可能选择关闭其服务区内的电网部分，以减少野火风险。这些停电被称为公共安全停电，主要发生在加利福尼亚州，那里的野火在近几十年来变得[更大更破坏性](https://www.epa.gov/climate-indicators/climate-change-indicators-wildfires)。

2024年4月5日至6日，科罗拉多州的Xcel Energy进行了该州的[首次公共安全停电](https://co.my.xcelenergy.com/s/about/newsroom/press-release/xcel-energy-update-on-public-safety-power-shutoffs-and-weather-related-outages-i-MCVMKSLT6RTZFMLHPG24IPW6YNEU)，在一场强风暴前夕为[成千上万的客户](https://www.denverpost.com/2024/04/06/xcel-power-shut-off-wind-fire-danger-boulder-gilpin-larimer-douglas-broomfield-jefferson/)切断了电力。公共官员和Xcel的客户抱怨称，他们[未收到足够的警告或解释](https://boulderreportinglab.org/2024/04/09/why-did-your-neighbor-have-power-and-you-didnt-boulders-grid-complexity-exacerbates-confusion-over-xcel-energys-wildfire-safety-outages/?)。科罗拉多州州长贾里德·波利斯已指示州监管机构[调查此事件](https://www.westword.com/news/colorado-governor-wants-state-to-investigate-xcel-wind-storm-outages-20354563)，并提出未来改进程序的建议。

科罗拉多居民描述了四月停电对他们造成的影响。

我是一名[电气和计算机工程师](https://scholar.google.com/citations?user=TAOTdN4AAAAJ&hl=en)，研究电网的高效运行。我也住在博尔德县，这里在2021年一场风驱动的野火中[毁坏了1000多栋房屋](https://research.noaa.gov/2024/01/08/looking-back-at-colorados-marshall-fire/)。随着[气候驱动的极端天气](https://theconversation.com/wildfire-risk-is-soaring-for-low-income-elderly-and-other-vulnerable-populations-in-california-washington-and-oregon-213455)对老化电网组件的压力，这些停电可能会在更多地方变得更为普遍。因此，理解和评估公用事业的公共安全停电计划变得尤为重要。

## 一个复杂的系统

断电和重新通电——在公用事业术语中称为去电和送电——不仅仅是翻开一个开关那么简单。电网是一个复杂的系统，实时在发电厂输送的电力和向客户提供的电力之间保持平衡。

电力线路没有能力储存电力以供以后使用——它必须立即使用。这使得电网不同于其他关键基础设施。水网可以在管道中保留水，道路可以支持车辆静止，即使交通无法流动。

由于电网无法储存电力，去电和送电必须分阶段进行，以确保电网需求端的变化不会对供给端适应能力造成过快的冲击。发电厂无法立即启动：[有些发电厂可以在几分钟内启动，而其他需要几个小时](https://www.eia.gov/todayinenergy/detail.php?id=45956)，这取决于它们的年龄、设计和燃料类型。它们通常也不能在瞬间将发电能力从10%提升到90%。

当公用事业在停电后恢复电力时，会产生大量瞬时的功率使用峰值，因为许多家庭的设备重新开启。电网还需要时间来适应这一需求的激增。公用事业可能会对电力线路进行目视检查，以检查是否存在损坏，然后再恢复供电。

[本地电网的复杂性](https://boulderreportinglab.org/2024/04/09/why-did-your-neighbor-have-power-and-you-didnt-boulders-grid-complexity-exacerbates-confusion-over-xcel-energys-wildfire-safety-outages/?utm_medium=email)可能会让人觉得这些停电是不均匀或任意实施的。为您家提供电力的配电变压器可能与为您对面邻居提供电力的变压器完全电气隔离，这会使您的街区陷入黑暗，而隔壁街区的灯光却依然亮着。

公用事业可能会因为网络中某处的风险而切断电力，这是客户看不见的。对于电力切断发生的区域进行精确调节并非总是可能的。

## 制定停电计划

加利福尼亚州一直是公共安全停电的试验田。该州公用事业委员会[在2012年批准了这些措施](https://www.cpuc.ca.gov/psps/)作为减少电力设备引发的野火潜在影响的一种方式，并颁布了指导公用事业何时可以切断电力线路的法规。该法规要求电力公司提前通知客户，并在每次计划停电后的10天内提交详细报告。

在过去的十年中，数百万加利福尼亚人经历过这些停电。2023年PSE Healthy Energy独立研究所的一项审查发现，受影响最严重的县，主要位于南加利福尼亚州，平均每年[有四到五次公共安全停电](https://www.psehealthyenergy.org/mapping-public-safety-power-shutoffs/)，平均持续时间为48小时。

俄勒冈州也[在2022年采纳了公共安全停电规则](https://ktvz.com/news/oregon-northwest/2022/05/11/oregon-puc-adopts-power-safety-shutoff-rules-as-wildfire-season-approaches/)，但许多其他司法管辖区尚未制定正式的协议。这些关键决策目前由公用事业公司负责。制定针对公共安全停电的复杂计划需要对公用事业公司管辖区的广泛了解，并准确评估哪些区域是高风险区域。

但是，公用事业公司并不总是在低压配电网（将电力输送到社区的部分）拥有丰富的传感器，并且可能缺乏对单个组件的极端细粒度控制。事实上，不计划的停电通常必须[手动报告](https://co.my.xcelenergy.com/s/outage-safety/report-outage)。许多公用事业公司正在安装智能电表，其中一些可以[自动报告停电情况](https://electricenergyonline.com/energy/magazine/572/article/smart-grid-calling-do-you-know-where-your-outages-are-tonight-.htm)给公用事业公司。

电力供应商还采取其他措施来加强他们的电网以防灾害，比如[埋设电力线路](https://theconversation.com/can-burying-power-lines-protect-storm-wracked-electric-grids-not-always-167018)。但根据地点不同，这种策略对低压配电网每英里的成本可能高达100万美元，对高压输电线路每英里的成本可能高达[数百万美元](https://arpa-e.energy.gov/sites/default/files/02%20Philseok%20Kim%20UG-Workshop%20introduction.pdf)。这些项目需要复杂的成本效益计算，并且最终的成本将会传递给电力消费者。

这段南加州爱迪生公司的视频解释了该公司如何确定是否需要进行公共安全停电以及如何管理这些行动。

## 为公共安全断电做好准备

对于一些人来说，停电只是一个小小的不便。但对于其他人来说，如果他们需要电力来保持重要药物的冷藏、给[便携式氧浓缩器](https://www.oxygenconcentratorstore.com/blog/oxygen-therapy-safety-what-to-do-if-you-lose-electricity)充电或者维持家庭温度在安全范围内，这可能意味着生死攸关。

公用事业公司应在公共安全断电的每一步骤中与客户保持联系，从提醒他们为关键设备充电并采取[其他准备步骤](https://www.cpi.coop/wp-content/uploads/2023/05/PSPS-Emergency-Preparedness-Checklist-Updated-05-23.pdf)开始。他们还应将需要依赖电力的医疗条件或其他问题的人员移出停电区域。

为了使这种合作成为可能，公用事业公司需要更深入地了解配电网的状态，更详细的系统模型以及改进的测量和传感策略，这些策略可以帮助他们对火灾风险进行准确和细致的评估。

最后，公用事业公司需要使他们的决策更加透明。没有人喜欢停电，因此对于电费支付者来说，理解停电发生的原因及方式以及公用事业公司为尽可能安全可靠地提供电力所做的努力至关重要。
