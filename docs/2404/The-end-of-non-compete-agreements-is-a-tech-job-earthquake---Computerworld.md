# 竞业禁止协议的终结对科技行业是一场地震 – Computerworld

> 来源：[https://www.computerworld.com/article/2095092/the-end-of-non-compete-agreements-is-a-tech-job-earthquake.html](https://www.computerworld.com/article/2095092/the-end-of-non-compete-agreements-is-a-tech-job-earthquake.html)

美国联邦贸易委员会（FTC）认为，要求低端工作人员签订竞业禁止协议是一种过度，但也不应限制有价值的员工。毕竟，该机构称，“商业秘密法和保密协议（NDA）都为雇主提供了确立的手段来保护专有和其他敏感信息。研究人员估计，超过95%的签有竞业禁止条款的工人已经签有NDA。”

此外，正如FTC主席Lina M. Khan所说：“竞业禁止条款压低工资，压制新想法，并且从美国经济中夺走了活力，包括一旦禁止竞业禁止协议后每年将会创立的8500多个新创企业。”

我不知道取消竞业禁止协议是否能做到这一切。但我知道，在我数十年来撰写科技报道的过程中，我看到竞业禁止协议成为了科技行业最优秀员工、帮助台技术人员甚至是保持办公室清洁的人员头上的铁链。
