# 基于科学 - 人类是否能够利用当前技术显著改变月球的轨道？ - Worldbuilding Stack Exchange

> 来源：[https://worldbuilding.stackexchange.com/questions/47410/could-humans-alter-the-moons-orbit-significantly-with-current-technology](https://worldbuilding.stackexchange.com/questions/47410/could-humans-alter-the-moons-orbit-significantly-with-current-technology)

**达蒙的答案**接近我心目中的想法，这在“正统”的或者说传统思想的大部分中从未被提到。这是我第一个答案中的第二点。如果你在向地球方向添加了10厘米/秒或0.1米/秒的矢量，你可能认为这很小，但在千年的时间尺度内将会产生毁灭性的影响，这更符合我们的时间尺度。

人们习惯于说“嗯..月球的质量大约是10^22千克，等等它的速度是多少啊，大约10^3米/秒，所以你需要为月球的动能添加约10^28焦耳，对吗？

WRONG! 当你反复引述“教授”的答案而没有学术上认真研究细节时，就会出现这种情况。（我自己也经常这样）你只需在这样的方向上给月球加上10^22或10^21焦耳的动能，使月球获得大约1米/秒或0.1米/秒的横向速度矢量。这将足以在100到1000年的过程中，将月球从地球的当前距离朝着该侧向移动。

有些人可能会说：现在如果我有无限的金钱和时间，当然可以。讽刺地指出，这可能在理论上是可能的，但在实际上却不太可能，因为这将需要大量的时间和金钱，对吧？

VERY WRONG AGAIN! 非常错误！非常感谢你。俄罗斯和美国仅在两十年内开发的核武库“兆吨数”约为12,000兆吨，或约为5x10^19焦耳。但是，这是半个世纪前的军备竞赛。现在整个地球都在建造核武器，并且掌握了更好的技术，意味着建造特定破坏力所需的时间更短，弹头也更强大。

如果再次爆发军备竞赛，确保，“超级大国”（美国、俄罗斯、中国、英国、加拿大、澳大利亚、法国、韩国、伊朗、巴基斯坦、印度等）将在不到一个世纪的时间内建造10^22“焦耳的毁灭力量”，对其预算几乎没有明显影响，或者他们可以在150或200年内建造同样的力量，看似很多但在整体范围内微不足道。

所以，我们能否建造足够的核武器来改变月球或其他天体的轨道？当然，这是一个相当大的挑战，但在当前技术能力范围内是可以做到的。

P.S. 通过爆炸释放的 10^22 焦耳不一定意味着会转化为某个方向的速度。在计算距离和其他参数时需要超级计算机的帮助，这样我们才能最大限度地利用这种能量作为运动。
