# 赫兹因特斯拉购买电动车耗资巨大，决定再次放弃更多电动车

> 来源：[https://qz.com/hertz-ev-sales-tesla-rental-cars-1851438100](https://qz.com/hertz-ev-sales-tesla-rental-cars-1851438100)

赫兹正在淘汰其更多电动车，称其不可靠且比汽油车更昂贵，并宣布2024年第一季度亏损3.92亿美元。

Nvidia的收益令人震惊，股价飙升

总部位于佛罗里达州埃斯特罗的租车公司表示，计划在今年放弃比[最初计划的更多](https://qz.com/hertz-rental-car-electric-vehicle-ev-reduce-fleet-cost-1851230776)1万辆电动车。现在，赫兹打算在2024年出售3万辆电动车，之前因车队折旧而[在上季度担负了1.95亿美元的损失](https://ir.hertz.com/news-releases/news-release-details/hertz-reports-first-quarter-2024-results)。加上2024年最后一个季度2.45亿美元的减值，赫兹在[前首席执行官史蒂芬·谢尔领导的电动车失败赌局上已经亏损了4.4亿美元](https://qz.com/hertz-ceo-stephen-scherr-resigns-ev-electric-vehicles-1851341889)。

这一轮新的出售使赫兹的电动车队只剩下3万辆。

“本季度的车队和直接运营成本对业绩产生了影响，” [首席执行官吉尔·韦斯特](https://newsroom.hertz.com/news-releases/news-release-details/hertz-announces-appointment-gil-west-chief-executive-officer)，一位曾在达美航空和通用汽车的[Cruise部门](https://qz.com/gm-cruise-general-motors-autonomous-driving-vehicles-1851397984)担任高管，并于4月1日加入赫兹的公司声明中说道。“我们正在解决这两个问题 - 在接受的资本成本下达到正确的车辆供应，同时提高生产力并降低运营成本。”

赫兹股票在周五上午交易中下跌超过3%。今年股价已下跌近56%，使其市值降至13.8亿美元。

租车公司表示，其电动车队不仅是财务负担，还给消费者带来了负担。韦斯特在周四的收益电话会议上告诉分析师，许多租车者不熟悉如何给电动车充电或正确使用它们，但由于可供选择的车辆有限，他们还是选择了这样做。

赫兹的电动车问题出现在其转向[从特斯拉购买10万辆电动车](https://www.bloomberg.com/news/articles/2021-10-25/hertz-said-to-order-100-000-teslas-in-car-rental-market-shake-up?sref=P6Q0mxvj)以及[与Polestar签署的暂停协议](https://qz.com/hertz-electric-vehicle-pause-polestar-1851226693)后的几年。它还与[Uber合作增加了5万辆电动车](https://www.uber.com/newsroom/hertz-update/)到该公司的网络中。到2024年底，赫兹希望其车队四分之一是电动车。

但到去年年底，由于电动车（EVs），该公司遭受了重大打击。特斯拉的迅速降价 — 旨在应对销售增长放缓 — 损害了赫兹车队的价值。而赫兹表示，修复特斯拉的费用是[同类汽油车的两倍](https://www.usatoday.com/story/money/cars/2024/01/11/hertz-selling-ev-fleet/72194050007/#:~:text=Repairs%20strain%20the%20bottom%20line&text=According%20to%20Johann%20Rawlinson%2C%20Hertz,weighed%20significantly%20on%20Q3%20earnings.)。
