# 勒芒2024

> 来源：[https://world.hey.com/dhh/le-mans-2024-1f87d732](https://world.hey.com/dhh/le-mans-2024-1f87d732)

这将是我第十一次尝试。我第一次出现在勒芒的起跑线上是在2012年 —— 大约是我第一次驾驶真正的赛车五年之后，甚至比我将参加世界顶级耐力赛作为终极目标的时间还短。但今年几乎没有发生这种情况。

看，赛车运动依赖于资金和才华的奇特混合。通常仅仅擅长驾驶是不足以获得在竞争激烈的赛车中的座位的。在大多数参赛队伍中，特别是在耐力赛车运动中，要么是由充满激情的私人赞助者资助，他们带来了使赛车运动变为可能的预算。即使在一级方程式赛车中也是如此的动态的一部分。

我带了一些钱和一些才华，但从来没有足够的能力单枪匹马地让一个配得上勒芒参赛资格的项目发生。我必须与其他带来预算的车手组队合作，还有一些带来更多才华的车手，以及一些只带来巅峰才华的车手。哦，还要找一个最好有几个赞助商来填补差距的车队。这总是一个舞蹈。

而今年，看起来这场舞蹈似乎不会结束。进行了大量的对话，提出了许多想法，但没有合同，没有签名。直到几周前，一位车手突然退出了他们的合同，然后，就有了在欧洲勒芒耐力赛和勒芒24小时赛中与Nielsen Racing一起参赛的机会。

[为我提供](https://sportscar365.com/lemans/elms/heinemeier-hansson-completes-nielsen-lmp2-lineup/)

填写。很好！

但现在是赛车运动的一个奇怪时期。一方面，从厂商参与度高的角度来看，这可能是有史以来最激动人心的勒芒24小时赛。今年六月在法国，我们将看到法拉利、保时捷、兰博基尼、丰田、凯迪拉克、宝马、标致和阿尔派之间的竞争。难怪这场比赛从去年十一月起就已经售罄。

另一方面，似乎没有人知道汽车的未来会是什么样子。一切都会电动化吗？不久之后，人工智能会开车带领我们吗？谁将收获这一转型的成果：硅谷的软件制造商还是中国的电池制造商？这会让欧洲的历史悠久的汽车品牌处于何种境地？

你可能会认为对未来的这种不确定性会使得汽车制造商在大预算赛车项目上花费大量资金的意愿降低。然而，无论是在一级方程式赛车还是在耐力赛车运动中，我们都达到了历史上的最高水平。这很奇怪。

话虽如此，我并不是在抱怨。能参与到如此黄金时代的赛车运动中，真是令人惊讶。作为一名业余选手，能够参与我们运动的超级碗、温布尔登或世界杯决赛，与世界上最优秀的车手竞争，这是多么荣幸啊。

甚至更重要的是，因为我知道这一切不会永远持续。我第一次在勒芒参加比赛时已经32岁了。现在我44岁了。赛车手在40岁后不会变得更好，这只是事实。

实际上，大多数人在那时已经退役了。42岁的费尔南多·阿隆索是例外，证明了这个规则。所以我一直在等着我的低谷随时可能到来。要么圈速开始下降，先是一点点，然后是很多。要么事故率增加。对抗自然是毫无意义的，这种情况注定会发生。

但也许正因为我没有那种顶峰的天赋，我能再多坚持一点时间。在不可避免的衰退来临之前，能有几场比赛处于巅峰状态。

我说这话并不感到恐惧。我接受生命循环的和其无法避免的衰退，以及生命的终结。没有人能永远在任何事情上保持最佳状态，尤其是在身体层面上，而我们中没有人能活着走出这一切。如果说有什么的话，那就是知道这一切——无论是赛车、编程还是创业——都不是永恒的，这让人感到一种解脱。

也许一旦你有了孩子，这种存在主义的药丸就更容易吞服了。看着我的三个儿子在卡丁车上变得更快，不管他们最终是否会转为赛车手，这种感觉就像是谚语说的“老人种树，知道树荫下不会坐着”。

现在我们真的偏离了赛道，但这也许是拥有孩子最有益的方面之一。这种生物学上深刻接受终局的感觉。我不必活到永远，因为我所属的人类血脉已经有数百万年的历史，将会继续下去。无论他们将来是否会参加赛车。

但这让我想起了我最喜欢的快速驾驶赛车的其中一个部分：没有时间进行存在主义的思考！你的所有注意力必须集中在为下一个弯道做出完美的操作，否则你就会失去十分之一秒，或者犯下一个让你和赛车撞向墙壁的错误。为了全神贯注于当前任务——快速行驶，避免撞车，其他所有事情都必须消失。
