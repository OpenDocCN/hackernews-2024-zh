# Shortbread的创始工程师 | Y Combinator

> 来源：[https://www.ycombinator.com/companies/shortbread/jobs/8jV0RY9-founding-engineer](https://www.ycombinator.com/companies/shortbread/jobs/8jV0RY9-founding-engineer)

**这是旧金山的混合面试角色**！您必须能够在旧金山当地工作。

**关于我们**

[Shortbread](https://shortbread.ai/)是由AI漫画工作室提供支持的漫画Netflix。

传统上，漫画由多位艺术家手工绘制。Shortbread Studio加速了这一过程20倍，使任何人都能在几分钟内将故事创意变成高质量的漫画系列。

在Shortbread应用上，创作者可以发布和销售他们的漫画系列。我们专注于为成熟观众提供心动的罗曼史故事。在您的指尖上尽情享受视觉上令人惊叹的《五十度灰》漫画。

我们是一支由Webtoon艺术家、MIT和伯克利工程师组成的团队，得到了Y Combinator的支持。我们热爱视觉故事，认为每个人都应该有能力创作自己的蝙蝠侠、灌篮高手和水手月。我们正在寻找一位卓越的创始工程师与我们共同打造漫画的未来。

**为什么加入我们？**

+   我们正在改变80亿美元的网络漫画行业。我们的技术使任何拥有优秀故事的人可以以前所未有的速度专业地创作网络漫画。

+   我们的工作涉及想象如何应用前沿的生成式AI研究和技术来构建令人难以置信的界面。这是一个开放的、深具创造性和技术性的问题领域，结合了艺术、AI和工程。

+   我们是一支不到10人的小团队。您将作为第一批工程师之一加入，立即拥有所有权、影响力，并直接与创始人合作。

+   有竞争力的薪水、股权和医疗福利

**要求：**

+   有2年以上全栈开发经验

+   在多种浏览器上测试、监控和优化解决方案和性能。

+   在前端，我们使用React、Typescript和TailwindCSS

+   在后端：NodeJS、ExpressJS、Flask

+   除了上述现代框架外，还需熟悉HTML、CSS、JavaScript。

+   具备设计、构建和维护高效、可重用、可靠的前端和后端代码的经验。

+   有使用NoSQL数据库（Firestore）、开发API和Web服务的经验。

+   有云计算服务（GCP、AWS）、容器（Docker）以及大规模构建分布式系统的经验

+   出色的英语书面和口头沟通能力。

**额外加分项：**

+   对Webtoons、漫画、艺术、故事讲述或其他视觉艺术形式充满热情

+   有图形/视觉设计经验

+   有稳定扩散A1111和生成式AI技术的经验

+   在Flutter中进行移动开发的经验
