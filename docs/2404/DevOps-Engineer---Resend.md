# DevOps 工程师 · Resend

> 来源：[https://resend.com/careers/devops-engineer](https://resend.com/careers/devops-engineer)

作为基础设施提供者，DevOps 工程师至关重要，以确保通过 Resend 发送的数百万个关键任务的电子邮件被准确送达到最终用户。

您将花费时间改进可靠性、正常运行时间、安全性、扩展性和性能。您将负责配置数据中心、扩展数据库、增强可观察性、响应事件以及优化灾难恢复计划。

## 在这个角色中，你将...

+   **提高多云提供商如 AWS 的高可用性**

+   使用像 Terraform 这样的 IAC 工具 **配置数据中心**

+   **扩展数据库** 如 Postgres 和 Clickhouse 以确保其性能

+   **通过使用像 Grafana 和 Prometheus 这样的监控工具增强可观察性**

+   通过配置 ETL 和数据仓库如 Snowflake **促进运营**

+   **优化灾难恢复** 计划以确保恢复迅速可靠

+   **响应事件** 并协助处理、解决或升级问题

+   **维护 CI/CD 流水线** 以优化安全最佳实践

## 如果你...

+   具有 **5 年的 DevOps 工程经验**

+   或者 **具有 5 年的后端工程经验**，专注于云基础设施

+   精通 **书面和口语英语**

+   能够 **持续在美洲时区工作**

+   **你热爱帮助** 其他开发人员

## 如果你还...

+   了解电子邮件最佳实践或 **愿意深入了解此领域**

+   熟悉 **Node.js 和无服务器架构**

+   曾使用过类似 **AWS SQS 和 SNS** 的服务

## 加入团队的意义：

+   自主权去“快速上线”

+   100% 远程团队，灵活的工作时间表

+   现代技术栈（Next.js, Raycast, Retool, Notion, 等等）

+   诚实而不自我膨胀的团队

+   问题和解决方案的所有权

您可以在 [resend.com/about](/about) 了解更多关于我们文化的信息。
