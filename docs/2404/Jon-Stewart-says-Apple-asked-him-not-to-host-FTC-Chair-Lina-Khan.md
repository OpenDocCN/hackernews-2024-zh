# 乔恩·斯图尔特说苹果要求他不要主持FTC主席莉娜·卡恩

> 来源：[https://www.engadget.com/jon-stewart-says-apple-asked-him-not-to-host-ftc-chair-lina-khan-090249490.html](https://www.engadget.com/jon-stewart-says-apple-asked-him-not-to-host-ftc-chair-lina-khan-090249490.html)

乔恩·斯图尔特昨天在他每周的《每日秀》节目中[主持](https://www.youtube.com/watch?v=oaDTiWaYfcM)联邦贸易委员会主席莉娜·卡恩，但斯图尔特自己的揭示与卡恩的一样有趣。在这次对话中，斯图尔特承认，苹果要求他不要在播客上主持卡恩，这是他当时的[*《乔恩·斯图尔特的问题》*](https://www.engadget.com/jon-stewart-apple-tv-plus-221022043.html)苹果电视+节目的延伸。

"我想请你上播客，但苹果要求我们不要这么做，" 斯图尔特告诉卡恩。"他们确实说过，‘请不要和她交谈。’"

实际上，整集节目似乎有一个“苹果不让我们做的事情”主题。在卡恩采访之前，斯图尔特做了一个名为“AI的虚假承诺”的[片段](https://www.youtube.com/watch?v=20TAkcy3aBY)，有效地揭穿了AI领袖们的利他主义主张，并提出AI严格是为了取代人类员工而设计。

"他们甚至不允许我们做第一幕关于AI的那个愚蠢的事情，" 他对卡恩说。"这是什么敏感性？为什么他们如此害怕在公共领域开展这些对话？"

"我认为这只是当你把如此大量的权力和决策集中在少数几家公司时，会出现的危险，" 卡恩回答道。

[*《乔恩·斯图尔特的问题》*](https://www.engadget.com/jon-stewarts-apple-tv-show-reportedly-ends-following-clash-over-ai-and-china-080117666.html)在第三季之前[突然取消](https://www.engadget.com/jon-stewarts-apple-tv-show-reportedly-ends-following-clash-over-ai-and-china-080117666.html)，据报道是因为关于AI和中国片段的冲突。这促使美国议员[质疑苹果](https://www.engadget.com/lawmakers-question-apple-over-cancellation-of-jon-stewarts-show-192316298.html)，想知道这个决定是否与对中国的可能批评有关。

尽管表示苹果有权播放任何内容，"外国权力的强制手段不应该直接或间接地影响这些决定，" 双方委员会写道。（苹果对此是否有回应尚未发布。）

斯图尔特没有说AI和卡恩采访的问题是他的节目被取消的原因，但这确实表明苹果在直接涉及其问题时主张了编辑影响。

在其他部分中，汗谈到了FTC对亚马逊的诉讼，指称FTC称该公司通过非法手段维持垄断地位（高额卖家费用，不明确的广告）。他们还触及了FTC对Facebook的诉讼，科技公司通过人工智能勾结，企业合并，过高的药品价格等问题。
