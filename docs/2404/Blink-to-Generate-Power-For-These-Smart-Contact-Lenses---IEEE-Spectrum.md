# 《眨眼即可为智能隐形眼镜产生电力 - IEEE Spectrum》

> 来源：[https://spectrum.ieee.org/power-smart-contact-lenses](https://spectrum.ieee.org/power-smart-contact-lenses)

智能隐形眼镜的潜在用途引人注目且多样化。将镜片放在眼睛上，监测健康指标如血糖水平；为眼部疾病接收定向药物传递；体验增强现实并通过信息展示在你的视野中阅读新闻更新。

但是眼睛对电子设计来说是一个相当大的挑战：角膜是人类组织中神经密度最高的之一，其敏感性是我们皮肤的300到600倍。研究人员已经开发出了小型、柔软的芯片，但电源却证明更加困难，因为大电池和电线显然在这里行不通。现有的应用提供的解决方案不够理想，如隔夜感应充电和其他依赖某种类型外部电池的设计。

现在，来自[犹他大学](https://www.ece.utah.edu/)的团队表示，他们已经开发出了更好的解决方案：一款专为眼部技术设计的全能量混合生成单元。

在[发表于3月13日的*Small*期刊上的一篇论文](https://onlinelibrary.wiley.com/doi/10.1002/smll.202401068)中，研究人员描述了他们如何建造该设备，结合了柔性硅太阳能电池与一种将眼泪转化为能量的新设备。该系统能可靠地供应足够的电力来操作智能隐形眼镜和其他眼部设备。

这是从单独电池进行无线电能传输的重大改进，据在犹他大学攻读博士期间参与该系统开发的[Erfan Pourshaban](https://www.linkedin.com/in/erfan-pourshaban-ph-d-09401a12b/)说。

“大多数当前的研究包括一种无线电能传输，其中一个天线在镜片上，另一个在设备上，它们之间的距离只能约为一厘米，非常非常小，”现任德州仪器的工艺整合工程师[Pourshaban](https://www.linkedin.com/in/erfan-pourshaban-ph-d-09401a12b/)说。“它必须直接在你的视线中，这是不切实际的。我们认为，如果你真的不能佩戴它，那么智能隐形眼镜的实际技术再好也没有用。我们必须建造一个能够独立运行的单元。”

## 光伏和金属-空气电池

**Pourshaban和他的同事们制造了定制零件来创建这个电源包。第一步是制造迷你化、柔性的硅太阳能电池，能捕捉来自太阳以及人造光源如灯的光线。团队连接了八个微小的（1.5 x 1.5 x 0.1毫米）刚性晶体电池，并用聚合物封装成柔性光伏系统。**

**下半部分是一个眨眼激活的系统，其功能类似于金属空气电池。佩戴者的天然眼泪——更确切地说是其中的电解质——作为生物燃料发电。**

**光伏和电池在眨眼的不同阶段激活。Erfan Pourshaban**

**收获实际上是眨眼的瞬间发生的：当眼睛完全张开时，收割者停止工作。然后当眼睛开始眨眼时，眼泪中的电解质与镁阳极相遇，引发氧化反应并产生电子。最后，眼泪中的电解质与阳极和铂阴极接触，通过进一步在阳极表面的氧化和阴极表面的氧还原来产生更多能量。电极通过眼睑的运动和眼泪的持续更新保持清洁。**

**通过一体化的 11 毫法拉德超级电容器的功率管理电路将这两个“半部分”的能量生成结合起来——将电压转换为直流电，提升电压，并最终以稳定的 3.3 伏特输出约 150 微瓦的电力——不需要天线、外部电池包或特殊的充电盒。**

**“这个系统是智能隐形眼镜领域的一项令人兴奋的进展”，未参与研究的[魏高](https://www.eas.caltech.edu/people/weigao)，一位[生物传感器专家](https://www.gao.caltech.edu/)，加州理工学院医学工程助理教授表示。他指出，智能隐形眼镜的前景受阻主要因为这些设备需要可持续的电源。”**

**这种新的电源不仅“用户友好”，他指出，还是一种“创新的双模式方法”，利用两种能量收集模式使其持续发生——无论眼睛是开着还是闭着。**

**“这个设备可靠的电力输出能够支持广泛的应用，包括可穿戴生物传感器和直接在眼部环境中使用的电动响应药物传递系统”，Gao 补充道。**

## **青光眼及其他应用**

**Pourshaban 补充道，明显的消费者应用包括，如镜片能够向跑步者显示心率、速度和燃烧的卡路里。零售商可以通过跟踪购物者如何扫描货架和选择商品来获得宝贵的见解。他说，商业化潜力是显著且多样化的。**

**不过，Pourshaban 或许对监测眼部健康的潜在应用最为兴奋，从普通的状况如[老花眼](https://www.mayoclinic.org/diseases-conditions/presbyopia/symptoms-causes/syc-20363328)——中年开始可能出现的远视，到更为隐匿的疾病，包括[青光眼](https://www.nei.nih.gov/learn-about-eye-health/eye-conditions-and-diseases/glaucoma)。**

**“青光眼被称为视力的悄然窃贼：没有疼痛，可能没有任何你能察觉到的症状，然后突然间，如果眼压在您看眼科医生之间变得过大，您可能会不可逆地失去视力，” Pourshaban说。“如果我们能够实时检测这些和其他疾病的预警信号并迅速干预，可以改变一个人的生活轨迹。”**
