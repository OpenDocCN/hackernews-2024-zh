# 瑞典高速公路上抓到了睡着的特斯拉司机 - 行驶了25英里后 | Vi Bilägare

> 来源：[https://www.vibilagare.se/english/sleeping-tesla-driver-caught-swedish-highway-after-25-miles](https://www.vibilagare.se/english/sleeping-tesla-driver-caught-swedish-highway-after-25-miles)

*要阅读本文的瑞典版本，请[*点击这里*](https://www.vibilagare.se/nyheter/teslaforare-somnade-pa-e4-bilen-korde-vidare)*。*

**事件发生**在瑞典E4高速公路上的一月份。这名工作在汽车运输公司的司机，在斯德哥尔摩南部的林雪平外突然在2019年驾驶特斯拉Model X时睡着了。

尽管其名称如此，特斯拉的自动驾驶系统并不使车辆完全自主。司机必须保持警觉并注意道路和其他道路使用者。

另一名也拥有特斯拉的司机驶近正在超车道上行驶而没有超车的Model X。这名特斯拉司机切换到右车道并试图超越Model X时，他注意到驾驶员正在睡觉。

警方赶到并试图多次唤醒司机，但他在“车自行驶”时继续睡觉。最后，司机醒来，特斯拉突然向右转。警车司机不得不急转弯和刹车，以避免碰撞。
