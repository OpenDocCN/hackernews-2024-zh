# Cally：无障碍声明

> 来源：[https://simonwillison.net/2024/Apr/2/cally-accessibility-statement/](https://simonwillison.net/2024/Apr/2/cally-accessibility-statement/)

[Cally：无障碍声明](https://wicky.nillia.ms/cally/accessibility/)（[通过](https://twitter.com/wickynilliams/status/1775208741392052299 "@wickynilliams"））Cally 是一个新颖的开源日期（和日期范围）选择器 Web 组件，由尼克·威廉姆斯（Nick Williams）开发。

它与框架无关，重量不到 9KB，但其最佳功能是这个详细的文档页面，涵盖其无障碍故事，包括如何在 JAWS、NVDA 和 VoiceOver 中进行测试。

我希望看到其他开源的 JavaScript 库效仿这个例子。
