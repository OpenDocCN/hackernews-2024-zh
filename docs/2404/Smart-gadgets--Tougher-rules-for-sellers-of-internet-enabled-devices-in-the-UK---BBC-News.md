# 英国网络连接设备销售商的更严格规定 - BBC News

> 来源：[https://www.bbc.co.uk/news/business-68917837](https://www.bbc.co.uk/news/business-68917837)

来自国家网络安全中心的Sarah Lyons表示，制造这些产品的公司需要承担责任。

“企业在通过确保其制造、进口或分销的智能产品提供持续保护来保护公众方面起着重要作用，而这部里程碑式法案将帮助消费者对他们购买的产品的安全性做出明智的决策，”她说。

Ken Munro是Pen Test Partners的安全研究员，该公司专门进行对智能设备的道德黑客攻击，他将这项新法律描述为“朝着正确方向迈出的一步”。

“它有实际效力，这点我非常喜欢，”他说。

他说，过去制造商往往很容易在推出新款产品时终止对旧产品的支持，因此对消费者来说，能够比较所购买产品的支持年限将会非常有用。

较长的支持周期表明制造商通常认真对待网络安全问题，他说。

“我认为市场底端的一些设备制造商可能只是敷衍了事，仅仅做到了最低限度以确保其产品的安全性。”

消费者团体Which?的政策与倡导主任Rocio Concha表示，新法律将为消费者提供“重要的保护”。

但他补充说，产品安全与标准办公室应准备好“对违反法律的制造商采取强有力的执法行动”。
