# 巴尔的摩基桥：据消息来源称，船舶存在明显的电力问题 | 美联社

> 来源：[https://apnews.com/article/baltimore-bridge-collapse-fbi-investiagation-58188d524035c756872603055f309c78](https://apnews.com/article/baltimore-bridge-collapse-fbi-investiagation-58188d524035c756872603055f309c78)

巴尔的摩（美联社）——据了解情况的人士透露，造成[巴尔的摩桥梁坍塌](https://apnews.com/article/baltimore-bridge-collapse-53169b379820032f832de4016c655d1b)的大型集装箱船在离港前数小时遇到明显的电力问题，当天FBI探员登上该船，展开对导致灾难的情况进行刑事调查。

达利号于3月26日清晨从巴尔的摩港口出发，装载货物前往斯里兰卡。在进入开阔水域之前失去动力，并撞击了弗朗西斯·斯科特·基桥的支撑柱，导致跨度坍塌进入帕塔普斯科河，致使[六名道路维修工作人员](https://apnews.com/article/baltimore-bridge-collapse-honduras-mexico-guatamala-victims-ac79dd7413b948c635549ef1845c6d22)坠入水中身亡。目前仍有两名遇难者下落不明。

当局周一晚间宣布，在水下残骸中的一辆施工车辆上找到了第四具遗体。官方称，根据家属的要求，未透露该人的身份。

在[该船](https://apnews.com/article/baltimore-bridge-collapse-cargo-ship-dali-09aeffc6fa81f3069d4ba226def90555)停靠巴尔的摩期间，据知情人士透露，在部分冷藏集装箱上触发了警报，表明电力供应不稳定，该知情人士未被授权公开发表评论，并在匿名条件下向美联社发表了讲话。

国家运输安全委员会的官员表示，他们的调查将包括对船舶在启航前是否出现电力问题的调查。

董事会主席Jennifer Homendy上周表示，调查重点是[船舶的电力系统](https://apnews.com/article/baltimore-bridge-collapse-investigation-85bd81d1dfefc6efc5bc9a2e22d6999a)，船只在事故前曾经历电力问题，视频显示船灯闪烁。

Homendy表示，从船舶的航行数据记录器中获取的信息相对基本，“因此引擎室的信息将对我们有巨大帮助。”

现在FBI正在进行一项刑事调查，重点是调查导致桥梁坍塌的情况以及是否遵守了所有联邦法律，据熟悉此事的不同人士透露。此人未被授权公开讨论调查细节，并在匿名条件下向美联社发表了讲话。

FBI特工星期一在货轮上进行了法庭授权的执法活动，该机构在一份声明中表示。声明没有详细说明，并表示不会进一步评论此次由《华盛顿邮报》首次报道的调查。

与此同时，市长布兰登·斯科特星期一发布声明，宣布与两家律师事务所合作，“启动法律行动，追究责任人的责任”，并减少对巴尔的摩市民的伤害。他表示，市政府需要迅速行动以保护自身利益。

市长斯科特表示市政府“将采取果断行动，追究责任造成基桥悲剧的所有单位的责任，包括货轮达利的所有者、经营者和制造商，该货轮在失去动力并偏离航道约半小时后开始其航程。”

达利号由新加坡的Synergy Marine Group管理，由Grace Ocean Private Ltd所有。丹麦航运巨头马士基租用了达利号。

Synergy and Grace Ocean [提交了法院诉状](https://apnews.com/article/baltimore-bridge-collapse-liability-litigation-16de885948e769566e7181ffa35b1753)，试图限制它们在美国海事法律诉讼中的法律责任——这是一个例行程序。它们的[联合提交](https://storage.courtlistener.com/recap/gov.uscourts.mdd.556480/gov.uscourts.mdd.556480.1.0.pdf)旨在将公司的责任限制在大约4360万美元左右。它估计船只本身的价值高达9000万美元，并且应该获得超过110万美元的货运收入。该估算还扣除了两笔主要费用：至少2800万美元的修复费用和至少1950万美元的打捞费用。

“由于此次事件的严重性，各个政府机构正在进行调查，我们正在全力配合，” Synergy发言人达雷尔·威尔逊星期一在一份声明中说。“出于对这些调查和未来法律程序的尊重，目前进一步评论将不合适。”

这些公司根据1851年海事法的规定提交了他们的请愿书，允许它们寻求将其责任限制在船只残骸价值之内，事故发生后。

一些遇难者的律师和幸存的工人星期一辩称，拥有和管理该船的公司正在利用一项“陈旧的法律”试图保护它们的资产。

“想象一下，当悲痛的家庭正在筹备葬礼时，船东却在法庭上，” 律师L·克里斯·斯图尔特在巴尔的摩举行的新闻发布会上说。

斯图尔特说，在倒塌发生前的片刻，道路施工人员“绝对没有任何警告”，尽管船长的最后时刻求救电话让附近的警察停止交通通过跨度。三名工人的尸体仍然失踪，施工人员继续进行从河中移除大块钢铁的危险工作。

Julio Cervantes在从桥上坠落后幸存，并通过摇下工作车辆的车窗和顶着寒冷的水域搏斗，虽然不会游泳，但幸存于危险之中，律师们说。他抓住残骸直到被救出。

“这一切都是可以避免的，” 斯图尔特说道。“这就是我们被委托调查并找出发生了什么，并为这些家庭发声的原因。”

此次调查发生在对[数千座美国桥梁安全](https://apnews.com/article/infrastructure-poor-bridges-collapse-biden-699d6c5e7aee9af73a12c7f7d2297ed1)的担忧之际，并且在数十艘河上驳船脱离控制后撞击了匹兹堡的封闭桥墩的几天之后。
