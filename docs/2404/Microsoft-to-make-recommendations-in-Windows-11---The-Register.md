# Microsoft 将在 Windows 11 中推荐应用 • The Register

> 来源：[https://www.theregister.com/2024/04/15/microsoft_recommendations_windows_11/](https://www.theregister.com/2024/04/15/microsoft_recommendations_windows_11/)

Microsoft 将在 Windows 11 开始菜单中尝试“推荐” – Microsoft Store 中应用的广告，但首先只针对一小部分美国 Beta 频道 Windows Insider 用户。

这个 Windows 巨头以在其旗舰操作系统中展示广告而闻名。在 2022 年，公司发布了一个版本，该版本在[文件资源管理器中显示广告](https://www.theregister.com/2022/03/15/file_explorer_ad/)，随后又撤回了这一功能。Windows 10 长期以来一直展示广告，因此 Microsoft 在 Windows 11 用户界面中利用空间展示“推荐”也并不意外。

然而，这会让那些将“让开始菜单再次伟大”解读为改善其功能的承诺的用户感到沮丧，而不只是添加一些人们可能认为是有帮助的建议。

这些[推荐](https://blogs.windows.com/windows-insider/2024/04/12/announcing-windows-11-insider-preview-build-22635-3495-beta-channel/)可以关闭，并且仅限于已选择获取最新操作系统更新的一部分美国 Beta 频道 Windows Insider 用户。

Microsoft 指出：“我们定期尝试新的体验和概念，这些可能永远不会随 Windows Insiders 发布。”但它在 Beta 频道中的出现表明，这一功能未来有可能被更广泛地发布。

Windows Insider 计划中有四个通道。Canary 频道允许用户在开发周期早期预览平台变更，而 Dev 频道则针对爱好者，被 Microsoft 用来“孵化新的想法”。

Microsoft 建议“早期采用者”使用 Beta 频道。接下来是 Release Preview 频道，商业用户可以在更广泛发布之前预览修复和更新。

关于 Beta 频道，Microsoft [表示](https://blogs.windows.com/windows-insider/2023/03/06/whats-coming-for-the-windows-insider-program-in-2023/)：“您的反馈在这里具有最大影响力”，并要求接收新体验的 Insider 用户“告诉我们您的想法”。

到目前为止，反应并不积极。开源桌面软件 KDE 迅速发布了一条[宣传帖](https://floss.social/@kde/112263362860298416)，其中写着： “看！没有广告！”

建议出台时，微软正加紧 [尝试](https://www.theregister.com/2023/03/01/microsoft_windows_11_crash/) 阻止 [多款](https://www.reddit.com/r/pcmasterrace/comments/1bw6clb/microsoft_seem_to_be_blocking_startallback_and/) 桌面定制工具，如 ExplorerPatcher 和 StartAllBack. ®
