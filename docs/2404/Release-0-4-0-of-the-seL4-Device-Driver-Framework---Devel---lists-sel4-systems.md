# seL4 设备驱动框架 0.4.0 版本发布 - Devel - lists.sel4.systems

> 来源：[https://lists.sel4.systems/hyperkitty/list/devel@sel4.systems/thread/6QKUK5C5PNN6CUR2XEUR2SSFLTCTTXLL/](https://lists.sel4.systems/hyperkitty/list/devel@sel4.systems/thread/6QKUK5C5PNN6CUR2XEUR2SSFLTCTTXLL/)

大家好。

这是一条消息，告知大家 seL4 设备驱动框架（sDDF）的版本 0.4.0 已经发布。

sDDF 旨在为 seL4 编写设备驱动程序提供一组接口、库和工具，允许安全地访问设备且开销低。一些人可能在以前的 seL4 峰会讲座中了解过 sDDF。

自从 2023 年峰会的预发布以来，我们在网络子系统上进行了重大改进，通常是合理化接口并进一步提高性能，以太网设备级接口现在已经很成熟。我们还为串行、I2C、块、音频和图形设备类引入了初步规范和原型实现。

您可以在这里找到有关发布及相关文档的信息：[https://github.com/au-ts/sddf/releases/tag/0.4.0](https://github.com/au-ts/sddf/releases/tag/0.4.0)

感谢，伊万。
