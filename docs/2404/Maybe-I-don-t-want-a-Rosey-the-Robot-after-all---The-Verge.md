# 或许我终究不想要一个罗茜机器人 - The Verge

> 来源：[https://www.theverge.com/2024/4/19/24134905/boston-dynamics-atlas-robot-smart-home-robot](https://www.theverge.com/2024/4/19/24134905/boston-dynamics-atlas-robot-smart-home-robot)

作为上个世纪八十年代的孩子，我对智能家居的看法主要受到了这样一个观念的影响：总有一天，我们会拥有像罗茜机器人一样的机器人在家里自由活动 — 扫灰尘、准备晚餐，甚至把洗碗机中的碟子卸下来。（最后一点是必须的；我们已经足够聪明地发明了洗碗机器人，难道不能再发明一个能把碟子卸下来的吗？）

然而，在本周发布的[Boston Dynamics最新机器人Atlas](/2024/4/17/24133145/boston-dynamics-resurrects-atlas-humanoid-robot-electric-new)亮相后，我童年的梦想正在迅速变成智能家居的噩梦。尽管*杰森一家*的家庭机器人管家有一种迷人的魅力，穿着褶边围裙更是别有一番风味，但我们离在家中拥有人形机器人越来越近，它们看起来却越来越令人恐惧。倒不是因为它们的外表 — 我完全可以想象Atlas穿着围裙的样子 — 而更多是因为它们所代表的意义。

Atlas是波士顿动力公司推出的一款新型全电动人形机器人，具备双足运动能力。尽管该公司的Atlas计划的下一代是为商业用途设计的，但他们宣称它能够完成“对我们人类来说太危险、太艰难或太枯燥肮脏”的工作。其中三项确实适用于家庭生活。

Atlas似乎是故意设计成令人恐惧的形象 — 或者至少与可爱差得很远 — 但这里展示的技术很容易让人们将其与创造出家用人形机器人联系起来。Nvidia正在研究这个问题，最近宣布推出[基金会模型GR00T用于人形机器人](https://nvidianews.nvidia.com/news/foundation-model-isaac-robotics-platform)。Nvidia的创始人兼CEO黄仁勋在推出时表示：“为通用人形机器人建立基础模型是当今AI中最令人兴奋的问题之一。”“全球领先的机器人技术专家正积极探索让人类迈向人工智能的巨大飞跃。”

作为上述80年代的孩子，人工智能机器人是噩梦的源头。*终结者*系列在我这一代人的心理中根深蒂固地灌输了对机器人起义的恐惧，而事情看起来并没有变得更加愉快。

然而，抛开流行文化，很明显，机器人在我们家中有一席之地。问题是，我们应该努力开发一个全能、双足、人类化的机器人来接管日常家务吗？我越想越多——越有越多机器人在我家里漫步——我认为答案是否定的。我们不需要一个能理解我们说话并能复制我们动作的机器人；我们需要的是能做好一项工作（或者也许是两项相关工作）的机器人。

我对家用机器人比大多数人都有更多的经验。[亚马逊的Astro机器人在我家里滚动](/23141966/amazon-astro-robot-review)了两周。我玩过一个ElliQ [伴侣机器人](https://elliq.com/)，测试过数十款[机器人吸尘器和拖地器](/22997597/best-robot-vacuum-cleaner)，目前我家里有两台[机器人割草机在花园巡逻](/2024/3/19/24105936/segway-navimow-i-robot-mower-price-release-date-specs)。更别提我家里到处都是各种智能音箱，内部集成人工智能，包括一个[带有机械臂的屏幕的亚马逊Echo Show 10（2021款）](/22297810/amazon-echo-show-10-2021-review)，当你和它说话时可以转向你（是的，这有点吓人）。如果机器人起义有可能发生，我可能会是第一个倒霉的。

当机器人完成它们的工作时，它们非常有用。当它们出故障时，它们可能会造成严重破坏。我曾经有一台机器人割草机砍倒了我丈夫最喜欢的植物，一个机器人吸尘器撞倒了一把椅子，引发了一连串的多米诺效应，最终导致了一扇破碎的窗户。在你家中自由移动的未监督机械设备伴随着后果，你给它们的责任和附加物越少，当事情出错时后果就越不灾难化，它们设计的工作就越能好好地完成。

一个可以折叠衣服的机器人，一个可以自动清理的洗碗机，一个可以炒大蒜和洋葱的炉灶，这些是我想看到的创新。一些[已经尝试过](/2018/1/10/16865506/laundroid-laundry-folding-machine-foldimate-ces-2018)，其他的则像*Samantha在《女巫俏佳娃》中的厨房的事物。所有这些看起来比起家里漫步的六英尺高的机器人更安全、更简单，最终更舒适。

舒适，因为当我自动清理的洗碗机坏了，我可以负责地回收它并得到一个新的。当我的人形机器人管家完成了它的固件更新生命周期，我将不得不让它退休。

阅读有关Atlas前身已达到其用途终点，已被[波士顿动力公司大堂的一个装饰物](https://www.nytimes.com/2024/04/17/technology/humanoid-robot-boston-dynamics-atlas-retires.html)，我感到了一丝悲伤。这让我想起了Kazuo Ishiguro出色小说[*《克拉拉与太阳》*](https://www.amazon.com/Klara-Sun-novel-Kazuo-Ishiguro/dp/059331817X?tag=theverge02-20)中令人心碎的最后时刻。

将家电拟人化 — 赋予它们人类特征和类人智能 — 带来了一系列关于意识本质和人类边界的复杂挑战。但当涉及到洗碗机时，这并不是我想要处理的事情。
