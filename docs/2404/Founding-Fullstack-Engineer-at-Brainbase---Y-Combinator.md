# Brainbase创始全栈工程师 | Y Combinator

> 来源：[https://www.ycombinator.com/companies/brainbase/jobs/qNs5JRo-founding-fullstack-engineer](https://www.ycombinator.com/companies/brainbase/jobs/qNs5JRo-founding-fullstack-engineer)

### 关于Brainbase

*Brainbase对AI工作者的意义，就像Retool对内部工具一样。*

**很快每家公司都会有一个人工智能团队与他们的人类团队一起工作。** 在Brainbase，我们让企业创建自己的企业级人工智能工作者。从中小型企业到政府机构，选择我们创建从外呼销售电话到客户支持等各种工作者。

Brainbase建立在我们的创始人在哈佛攻读博士期间开发的一种全新类型的AI代理之上，该代理允许在更长的时间范围内进行更复杂的工作流，使我们能够将整个工作负载从人类员工转移到AI工作者身上。

我们公司成立于2024年1月，**我们已经实现了超过100K的年收入**，并且已经从YC、500 Global和其他顶级公司筹集了种子轮融资，参与者包括Youtube和Hubspot的创始人，以及许多顶级YC校友。

### 关于这个角色

作为我们创始团队的一员，您将在企业中发挥高度的作用：发布新功能，与用户交流并塑造公司文化，以便未来的团队成员参与其中。

你将与我们的CEO和创始团队一对一合作，主要根据客户反馈进行迭代，帮助基础设施扩展（云和本地），并改进我们的基础专有代理系统。

### 谁适合？

如果你具备以下条件，我们认为你应该申请：

+   **Resourceful –** 你以少量资源（时间、金钱或计算能力）完成大量工作的能力为人所知

+   **坚韧不拔 –** 当面临困难时，你不会轻易放弃，而是在找到更好的方法后才会停下来

+   **Decisive –** 在压力和不完整数据下能做出明智的决策，并且进入下一个任务（没有犹豫）

+   **L̶i̶k̶e̶ Love difficult problems –** 你喜欢解决难题，比如拼图、国际象棋或者任何你一开始不擅长的事情

+   **Fun –** 即使在困难和艰苦时期，你也能微笑、开玩笑、享受自己，并且能够在事情明显不顺利时振奋他人的情绪

### NSFABIQ - 并非经常问但重要的问题

+   **🧠 为什么我应该加入Brainbase而不是自己创办一家新公司？**

    如果您被接受加入我们的团队，毫无疑问我们相信您有能力经营自己的公司。事实上，您可以确信我们对您充满信心，您很可能在某个时刻创办自己的独角兽公司。但是创业（众所周知）是困难的，通常最好先了解一个优秀公司的运作方式，然后再开始自己的创业。尤其是在当前人工智能蓬勃发展的时代，您确实想要看看是什么让一个优秀的人工智能公司与普通的区别。

    作为我们团队的创始成员，您将体验到创始人的所有优点（自主性、与客户交流、交付、文化创建），而没有任何开销（筹款、运营、税收）。最终，如果您选择离开开始自己的旅程，我们将尽一切可能支持您。

+   **🧠 我想加入，但我认为我不是创始工程师的材料**

    如果您有疑问，并满足上述必备条件，您应该申请。我们非常欢迎优秀的人才以任何形式加入我们的团队，也许我们会在您身上看到您自己尚未发现的优点。

+   **🧠 申请过程是什么样的？**

    我们根据您的简历和以往经验的情况变化申请流程，但您应该预期（1）一次筛选电话，（2）技术面试（在家完成或Zoom进行），以及（3）在办公室进行一周的有偿试用。
