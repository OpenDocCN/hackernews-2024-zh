# FAA：再入车辆必须在发射前获得许可证 • The Register

> 来源：[https://www.theregister.com/2024/04/22/faa_now_requires_reentry_vehicles/](https://www.theregister.com/2024/04/22/faa_now_requires_reentry_vehicles/)

美国联邦航空管理局正在更新其发射许可要求：如果你要发射一个设计用于再入的东西，你也需要相应的许可证。在发射前。

看起来这是对去年情况的回应，瓦尔达航天工业公司发射其W-1任务时没有许可证将舱返回到美国土地。

[通知](https://www.federalregister.gov/documents/2024/04/17/2024-08156/launch-of-a-reentry-vehicle-as-a-payload-that-requires-a-reentry-authorization-to-return-to-earth)，自2024年4月17日生效，要求如果有效载荷是再入车辆，则必须在发射前获得再入授权。

FAA对于未经再入授权即发射的担忧源于公共安全，并承认一旦发射后再入车辆最终会返回地球。其任务可能受推进剂、部件故障或其他因素的限制，但除非其轨道高度足够避免担忧，否则它最终将再次进入大气层。

FAA解释说：“因此，在发射前评估再入的安全性至关重要。”

去年，基于火箭实验室光子卫星总线的瓦尔达W-1卫星进入了轨道。尽管FAA批准了发射，但未授权返回美国。

结果是，该任务最终超出了最初计划的范围，尽管未达到[宇航年](https://twitter.com/VardaSpace/status/1702776131819094397)可维持在轨的时间。

公司在社交媒体上[强调](https://twitter.com/VardaSpace/status/1781041900717285455)遵循了规定，并表示：“去年，FAA正式书面许可Varda发射W-1号卫星，并且Varda符合所有发射要求。”

"瓦尔达将继续与FAA和其他联邦监管机构合作，因为他们的再入操作政策持续演变。"

瓦尔达的发言人告诉《The Register》：“今年初FAA发布许可后，我们的飞行验证再入系统安全且成功降落在犹他州试验和训练范围”，并补充道：“我们预计今年晚些时候将发射我们的下一个任务。”

[SpaceNews报导](https://spacenews.com/faa-to-require-reentry-vehicles-licensed-before-launch/)称，FAA商业空间运输副管理员Kelvin Coleman表示，更新与Varda的经验有关。他承认，由于时间紧迫，Varda获准在没有再入许可的情况下发射，但指出这种情况不太可能再次允许。®
