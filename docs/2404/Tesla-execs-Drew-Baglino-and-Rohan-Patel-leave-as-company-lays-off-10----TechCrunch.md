# 特斯拉高管德鲁·巴格利诺（Drew Baglino）和罗汉·帕特尔（Rohan Patel）在公司裁员10%的同时离职 | TechCrunch

> 来源：[https://techcrunch.com/2024/04/15/tesla-layoffs-workers-sales-cybertruck-cuts/](https://techcrunch.com/2024/04/15/tesla-layoffs-workers-sales-cybertruck-cuts/)

两位备受关注的高管在特斯拉裁员数千名员工的同一天离职，TechCrunch 已确认。

特斯拉动力总成和能源高级副总裁德鲁·巴格利诺（Drew Baglino）和公共政策与业务发展副总裁罗汉·帕特尔（Rohan Patel）已离职，帕特尔告诉 TechCrunch。帕特尔称，他决定离职是因为公司发生了“大的整体变化”，他拒绝具体说明。他们的离职首次被 [彭博社](https://www.bloomberg.com/news/articles/2024-04-15/tesla-executive-baglino-leaves-as-musk-loses-another-top-deputy) 报道。

“对我来说是好事，希望对特斯拉也是如此，” 帕特尔在给 TechCrunch 的消息中说道。“与业界最佳的政策/商业发展团队合作，取得了超乎我想象的成就，经历了一段了不起的时光。” 巴格利诺在 X 平台的一篇帖子中[表示](https://twitter.com/baglino/status/1779872449628426739)离职是一个“艰难的决定”，在特斯拉工作了18年之后。特斯拉 CEO 埃隆·马斯克在他的社交媒体平台 X 上的帖子中[感谢](https://twitter.com/elonmusk/status/1779885461290287117)他们两位在公司的时间。

他们的离职正值特斯拉裁员数千名员工以削减成本和提高生产力之际，据马斯克向员工发出的内部电子邮件称，这一消息首次被 [Electrek](https://electrek.co/2024/04/15/tesla-lays-off-more-than-10-of-its-global-workforce/) 和 [彭博新闻](https://www.bloomberg.com/news/articles/2024-04-15/tesla-to-cut-over-10-of-its-global-workforce-electrek-reports) 报道。这家电动汽车制造商正裁减其全球总员工数的“超过10%”，马斯克在电子邮件中称。特斯拉在2023年结束时的员工人数超过14万人，这意味着裁员可能会影响超过14,000人。

在特斯拉报告了三年来首次的 [年销量同比下降](https://techcrunch.com/2024/04/02/tesla-sales-drop-first-quarter-estimates-deliveries/) 后两周，马斯克宣布了裁员计划，这是在电动车需求普遍降温的背景下发生的。该公司已警告投资者，2024年的销售增长可能明显低于每年50%的目标。同时，特斯拉产品周期有所改变，昂贵的赛博卡车刚刚开始生产，而受欢迎的 Model Y 已进入第四个年头，没有任何重大更新。

“在我们为公司的下一个增长阶段做准备时，非常重要的是审视公司的每个方面，以降低成本并提高生产力”，马斯克在邮件中说道。他说，特斯拉的增长导致了“在某些领域的角色和工作职能重复”。

“作为这一努力的一部分，我们对组织进行了彻底审查，并做出了全球范围内减少超过10%员工人数的艰难决定。没有比这更让我痛恨的事情了，但必须这样做。这将使我们变得精益、创新并渴望下一个增长阶段周期”，马斯克写道。

巴格利诺的离职是该公司过去一年内第二位高管离职，此前特斯拉首席财务官扎卡里·柯克霍恩在[2023年8月](https://techcrunch.com/2023/08/07/tesla-cfo-master-of-coin-zachary-kirkhorn-steps-down/)辞职。巴格利诺负责公司的动力系统和电池技术工程。他也成为马斯克社交媒体平台上反应最快的高管之一。帕特尔曾在担任奥巴马总统特别助理和气候能源顾问多年，并且最近几个月与特斯拉的客户和粉丝们保持密切互动。

特斯拉在2023年运输了[a record](https://techcrunch.com/2024/01/24/tesla-says-ev-sales-growth-may-be-notably-lower-in-2024/)180万辆电动汽车。但由于高利率和全球竞争加剧的影响，该公司大部分时间都在削减其最受欢迎车型的价格。据报道，该公司还[dropped](https://techcrunch.com/2024/04/05/tesla-abandons-low-cost-ev-robotaxi/) — 或至少推迟 — 计划建造一款起售价约为25,000美元的低成本电动汽车，而是选择使用正在开发中的基础平台来支持马斯克声称将于8月8日首次亮相的所谓[robotaxi](https://techcrunch.com/2024/04/05/elon-musk-says-hell-unveil-a-tesla-robotaxi-on-august-8/)。
