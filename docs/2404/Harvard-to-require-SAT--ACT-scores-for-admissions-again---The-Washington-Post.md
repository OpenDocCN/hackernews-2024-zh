# 哈佛要求 SAT 和 ACT 成绩再次用于入学申请 - 华盛顿邮报

> 来源：[https://www.washingtonpost.com/education/2024/04/11/harvard-reinstates-sat-act-admissions-requirement/](https://www.washingtonpost.com/education/2024/04/11/harvard-reinstates-sat-act-admissions-requirement/)

哈佛大学将再次要求申请者提交标准化考试成绩，成为最新一所在 [疫情期间](https://www.washingtonpost.com/coronavirus/?itid=lk_inline_manual_2) 将选择性要求改为强制的高度选择性学校。

加州理工学院也宣布，从即日起，申请本科入学者需要提交 SAT 或 ACT 成绩。
