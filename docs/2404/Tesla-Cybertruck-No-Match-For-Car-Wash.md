# Tesla Cybertruck 洗车毫无抵抗力

> 来源：[https://jalopnik.com/tesla-cybertruck-no-match-for-car-wash-1851417011](https://jalopnik.com/tesla-cybertruck-no-match-for-car-wash-1851417011)

仅仅几个月后，并拥有几千英里的[拥有权](https://jalopnik.com/tesla-cybertruck-is-significantly-worse-at-towing-than-1851406117)，Tik Tok 用户 @captian.ad 的特斯拉 Cybertruck 效果上变成了[几千磅的废纸](https://jalopnik.com/tesla-cybertruck-teardown-finds-massive-battery-pack-lo-1851400405)数小时。将卡车带到海滩并在洗车后停在车库后，[它决定停止工作了一段时间](https://www.tiktok.com/@captian.ad/video/7357905192454606123?is_from_webapp=1&sender_device=pc)。显示屏，运行卡车的所有功能，变黑，[完全不响应](https://jalopnik.com/tesla-cybertruck-owners-forum-is-already-full-of-tales-1851388334)，即使执行工厂预定的重启程序。这不太好。

特斯拉的 Cybertruck 终于抵达

<track kind="captions" label="English" src="https://kinja.com/api/videoupload/caption/21637.vtt" srclang="en">

向特斯拉提交故障单以重新启动卡车后，Cybertrucker 上床睡觉，并于[第二天早上醒来发现卡车基本正常运作](https://www.tiktok.com/@captian.ad/video/7358236474321505578)。与特斯拉的电话确认，卡车需要完全重启，需要坐上五个小时才能完成。从他启动重新启动方法——按住两个方向盘按钮的时刻算起，卡车显然在整夜中一直在进行重新启动，直到半夜某个时刻。

有趣的是，用户并未提及特斯拉是否能够解释卡车为何停止工作，这是由洗车引起的还是其他原因。Cybertruck 的用户手册确实警告不要在直射阳光下洗车，并特别提到，在洗车前必须将卡车切换到“洗车模式”，以避免损坏车辆的部件。

如果你正在驾驶特斯拉的 Cybertruck，也许不要马上洗它。你永远不知道什么时候会因为卡车自己花了大约五个小时左右来解决问题而被困。
