# 日全食期间Airbnb的预订达到了天文数字的水平。

> 来源：[https://www.economist.com/business/2024/04/06/airbnb-bookings-for-the-solar-eclipse-reach-astronomical-levels](https://www.economist.com/business/2024/04/06/airbnb-bookings-for-the-solar-eclipse-reach-astronomical-levels)

直到4月8日早晨，月球才开始移动到地球和太阳之间。但本月的[全日食](https://www.economist.com/the-economist-explains/2024/04/02/why-are-total-solar-eclipses-so-rare)，从太平洋开始，横跨北美并结束于大西洋，其商业影响已经显而易见。根据旅行数据公司AirDNA的Jamie Lane称，4月的一个典型星期日晚上，位于或接近日食路径的Airbnb或Vrbo上约30%的短租房屋已被占用。日全食带内92%的房源已预订至4月7日。而这一约180公里宽带之外的几个城镇的住房需求几乎没有变化。

这次日食可以在包括达拉斯、印第安纳波利斯、克利夫兰、布法罗和蒙特利尔在内的一些大城市或中大型城市看到。酒店数据提供商CoStar估计，这些地方的入住率从12个百分点（蒙特利尔）到67个百分点（印第安纳波利斯）不等。剩余的房间似乎只能以高价出售。在日全食前，《纽约时报》报道称，其路径上仍有空房的Super 8汽车旅馆中，近一半的房间至少收费是标准价格的两倍。
