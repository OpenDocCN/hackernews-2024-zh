# 未来的 Roku 电视可能在暂停时播放定制广告 • The Register

> 来源：[https://www.theregister.com/2024/04/17/roku_tv_ad_patent/](https://www.theregister.com/2024/04/17/roku_tv_ad_patent/)

更新：未来的 Roku 电视会在你暂停视频时展示定制广告吗？我们希望不会，但是……

Roku 提交的专利申请，以美国 20230388589A1 号文档注册，并命名为“HDMI 定制广告插入”，详细描述了一个相当复杂的系统，最终在你的电视或其他显示设备上在检测到播放暂停时展示广告。

提出的专利于去年年底发布，并现在被发现，描述了一种显示设备，可以在识别到 HDMI 连接的媒体设备中视频内容暂停后展示广告。

关键是，它表示显示和媒体硬件可以是同一设备，即：像 Roku 的智能电视那样。这种设计有两种看法：独立电视、监视器和投影仪在暂停期间展示自己的广告，以及集成智能显示器采取相同做法。

如果使用的显示器或电视采用这项技术，这显然也意味着广告可以在游戏过程中或通过PC播放的视频中间播放。

Roku 甚至贴心地在专利申请中附上了一个流程图，详细说明了广告播放的所有情况。这些情况不仅包括检测暂停按钮的按压，还包括电视根据各种标准推断出暂停的情况。

这些标准包括检查是否正在创建新帧，是否有任何音频，甚至屏幕上是否有暂停符号。在这些情况下，电视或显示器将开始播放广告，直接覆盖屏幕上的任何内容。

该专利未明确指定这些广告将采用静态图像、音频还是视频形式，但 Roku 指出，广告将基于屏幕上播放的内容进行投放。专利详细说明了分析内容以确定展示哪种广告的多种方法，如视频元数据、水印和音频。检查源设备甚至正在运行的软件应用程序也是可选的，也许在涉及游戏主机或其他特定设备名称的设备时会派上用场。

草案专利在讨论内容识别时从未提及人工智能，因此其检测可能相当基础。但是，由于电视可能必须与某些外部服务器通信以获取广告，该过程中任何基于人工智能的部分可能都是在云端完成，远离寻求专利保护的设计。

尽管智能电视当前并不会进行这种广告注入，技术上已经是可能的。早在2015年，三星不小心让其澳大利亚电视播放了[静音百事可乐广告](https://www.theregister.com/2015/02/11/samsung_fixes_advert_interruption_problem/)，不断在通过 Plex Media Server 播放的电影上播放。尽管广告显然是一个错误，但这表明三星正在研发类似 Roku 专利中概述的功能。

当然，这一切都只是在专利申请阶段，可能永远不会发生，许多专利最终都未能成为实际产品。我们已经要求 Roku 就这一功能是否可能实现发表评论。®

### 更新于 2130 UTC 补充。

Roku 的发言人已经与我们联系，以确保他们目前没有计划商业化他们费力申请专利的设计。

“这不是一项已经发放的专利，目前没有计划在我们的产品中加入 HDMI 广告插入功能，”说话者告诉我们。

“就像许多其他科技公司的专利一样，Roku 的专利申请经常描述团队在 Roku 探索的技术，而这些技术中很多并不会成为我们销售的产品的一部分。”
