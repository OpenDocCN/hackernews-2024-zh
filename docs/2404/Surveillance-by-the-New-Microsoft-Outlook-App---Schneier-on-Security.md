## 新版 Microsoft Outlook 应用的监控

ProtonMail 的人们指责 Microsoft 新版 Outlook for Windows 应用 [大规模进行监控](https://proton.me/blog/outlook-is-microsofts-new-data-collection-service) 其用户。它与广告商分享了大量数据：

> 窗口通知用户，Microsoft 和 801 家第三方利用其数据进行多种目的，包括：
> 
> +   存储和/或访问用户设备上的信息
> +   
> +   开发和改进产品
> +   
> +   个性化广告和内容
> +   
> +   测量广告和内容
> +   
> +   推断观众见解
> +   
> +   获取精确的地理位置数据
> +   
> +   通过设备扫描识别用户
