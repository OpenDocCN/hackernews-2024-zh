# 普通显示器上的计算机生成全息图

> 来源：[https://opg.optica.org/ol/abstract.cfm?uri=ol-49-8-1876](https://opg.optica.org/ol/abstract.cfm?uri=ol-49-8-1876)

## 摘要

我们提出了一种使用手机屏幕发出的非相干光进行计算机生成全息图（CGH）的方法。在这种方法中，我们假设全息级联，其中第一个全息图是显示在手机屏幕上的彩色图像。通过求解关于非相干光传播的逆问题合成全息级联。我们演示了使用由 iPhone 和空间光调制器组成的双层全息级联来复制三维彩色图像。</da_html>
