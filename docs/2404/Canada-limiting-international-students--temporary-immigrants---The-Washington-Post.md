# 加拿大限制国际学生和临时移民 - 华盛顿邮报

> 来源：[https://www.washingtonpost.com/world/2024/04/23/canada-immigration-international-students/](https://www.washingtonpost.com/world/2024/04/23/canada-immigration-international-students/)

安大略省基奇纳市 — 加拿大对移民的广泛支持，加拿大总理贾斯廷·特鲁多曾表示这是为了对抗人口老龄化和低生育率，这使得该国与众不同。

加拿大是美国最接近的邻国，其增长速度超过了其七国集团同行以及印度等发展中和更肥沃的国家。2023年，[这里的人口增加了超过120万人](https://www150.statcan.gc.ca/n1/daily-quotidien/240327/dq240327c-eng.htm)，比前一年增长了3.2% — 自1957年以来的最高年度增长率。约98%来自移民。
