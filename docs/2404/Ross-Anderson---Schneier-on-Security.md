# 罗斯·安德森 - 施奈尔安全网站

> 来源：[https://www.schneier.com/blog/archives/2024/03/ross-anderson.html](https://www.schneier.com/blog/archives/2024/03/ross-anderson.html)

## 罗斯·安德森

[罗斯·安德森](https://www.cl.cam.ac.uk/~rja14/)在剑桥的家中[周四晚上](https://www.lightbluetouchpaper.org/2024/03/29/rip-ross-anderson/)意外去世。

我记不清我第一次见罗斯是什么时候了。当然，那是在我们创建[安全与人类行为](https://www.schneier.com/blog/archives/2023/06/security-and-human-behavior-shb-2023.html)研讨会之前的事，时间早在2008年之前。在我们创建[经济与信息安全研讨会](https://econinfosec.org/)之前的事，时间早在2001年之前。（好吧，两个都是他创建的——我有所帮助。）在我们[写过关于](https://www.schneier.com/academic/archives/1997/04/the_risks_of_key_rec.html)密钥托管系统问题之前，时间早在1998年之前。他曾邀请我参加剑桥大学牛顿研究所的六个月密码学驻留项目（我错失了全程）——那是在1996年。

我记得我参加了1993年12月的第一届[快速软件加密](https://iacr.org/meetings/fse/)研讨会，这也是他创建的另一个会议。在那里，我介绍了[Blowfish](https://www.schneier.com/academic/archives/1994/09/description_of_a_new.html)加密算法。从书架上拿下一本旧的《[应用密码学](https://www.amazon.com/Applied-Cryptography-Protocols-Algorithms-Source/dp/0471597562)》第一版（蓝色封面的那本），我看到他在致谢中的名字。这意味着在1993年初的某个时候——可能是在挪威洛夫图斯举行的欧洲密码术会议上——我作为一名未发表的书籍作者，仅写过几篇密码学文章给*Dr. Dobb’s Journal*，向他请求阅读并评论我的书稿。他答应了。这意味着我邮寄了一份纸质副本给他。他阅读了它。然后用邮票在信封上亲笔写下了评论。因为那时我们就是这么做的。

我认识Ross已经三十多年了，既是同事也是朋友。他热情、聪明、有主见、口才流利、性情乖戾但又善良。随便拿起他的任何学术论文——有[很多](https://www.cl.cam.ac.uk/~rja14/#Research)——你很可能会找到至少一个意想不到的洞见。他是一名密码学家和安全工程师，但也是一位非常全面的学者。他在1990年代发表了有关块密码破译的文章，去年又发表了关于大语言模型安全性的文章。他如麻地创办了各种会议。他的巨著书籍，《[安全工程](https://www.cl.cam.ac.uk/~rja14/book.html)》，现已进入第三版，是关于网络安全及相关主题的全面巨著。 （还要注意同一页上的他十五讲视频系列。如果你从未听过Ross讲课，你将有所收获。）他是[第一个](https://www.acsac.org/2001/papers/110.pdf)理解安全问题通常实际上是[经济问题](https://www.cl.cam.ac.uk/~rja14/econsec.html)的人。他是第一个做出这类联系的人。他反对监视和后门，并支持学术自由。他既不容忍政府也不容忍企业世界的愚蠢行为。

他在致谢中列出他是我从[*超越恐惧*](https://www.schneier.com/books/beyond-fear/)开始的每一本书的读者。最近几年，我们每年都会见面几次：在这个或那个研讨会或活动上。我上次见到他是去年6月，在[Pittsburgh的SHB 2023](https://www.heinz.cmu.edu/~acquisti/SHB2023/index.htm)上。我们正在[Alessandro Acquisti](https://www.heinz.cmu.edu/~acquisti/)的屋顶露台上吃晚餐，庆祝另一个成功的研讨会。他原本打算参加我在12月的[重新想象民主研讨会](https://www.schneier.com/blog/archives/2024/01/second-interdisciplinary-workshop-on-reimagining-democracy.html)，但他在最后一刻取消了。 （他把他打算发表的演讲稿发给了我。我会考虑发布它。）他去世的前一天，我们正在讨论如何安排今年的[SHB研讨会](https://www.schneier.com/shb/2024/)的所有注册人员。每次我们交谈时，我都从他那里学到了一些东西。而我并不是唯一一个。

我对他的妻子Shireen和他的家人深感同情。我们失去了他，实在是太早了。
