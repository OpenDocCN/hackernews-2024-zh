# Infinitown | 由Little Workshop开发的WebGL实验

> 来源：[https://demos.littleworkshop.fr/infinitown](https://demos.littleworkshop.fr/infinitown)

这个WebGL实验尝试创建一个过程化的城市，让人感觉生动且观看起来有趣。

首先，我们生成一个随机城市街区的有限网格。然后，通过一些技巧，视角围绕这个网格，从而产生了无尽城市景观的幻觉。

使用的工具：Three.js, Blender, Unity.

模型由VenCreations提供。
