# EURISKO lives | Paraprogramming Dispatches

> 来源：[https://blog.funcall.org//lisp/2024/03/22/eurisko-lives/](https://blog.funcall.org//lisp/2024/03/22/eurisko-lives/)

当我几年前写关于 EURISKO 的文章时[/lisp/2018/11/03/am-eurisko-lenat-documents/](/lisp/2018/11/03/am-eurisko-lenat-documents/)，几乎没有人预料到会有后续。这个系统是一个尘封的传奇，有些愤世嫉俗的人怀疑它是否真实存在过。

然而，Lenat 去年八月的去世解锁了他的 [SAILDART 档案](https://www.saildart.org/%5B*,DBL%5D/) 账户。这导致了 *WhiteFlame* 发现了 [令人激动的发现](https://white-flame.com/am-eurisko.html)，包括 AM 和 EURISKO 的源代码。在进一步的发展中，*seveno4* 已经成功地 [将 EURISKO 适配运行](https://github.com/seveno4/EURISKO) 在 Medley Interlisp 上。

在我之前对发现系统的想法感到惊叹之前，我甚至从未考虑过自己可能会运行 EURISKO。这真是一个印第安纳·琼斯发现失落方舟的时刻。然而，这个非常低概率的事件确实发生了，正如下面的视频中所记录的那样。请倒带到 8:20 看 Medley 运行的情况。
