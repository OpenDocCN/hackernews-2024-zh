# 计算机图形学中的代数几何

詹姆斯·F·布林（James F. Blinn） 计算机图形学家用多项式模型形状。简单的线性多项式生成平面多边形；高阶多项式生成曲线和曲面。为了开发建模和渲染这些形状的算法，我们必须深入理解多项式系数如何与生成的几何形状相关联。各种几何问题的答案用原多项式系数构建新的多项式表达。问题在于，对于相当简单的形状，这些新多项式可能会变得极其复杂。例如，三次曲线具有双重点的条件导致一个测试表达式，其是三次系数的12次多项式，这个测试表达式有超过10000个术语！幸运的是，解决这个问题有更好的方法。我所做的研究建立在一种图形符号技术之上，这种技术最初是由西尔维斯特和克利福德在100多年前提出的。我已经将经典代数几何的许多结果翻译成这种符号。这导致更好地可视化多项式形状与其系数之间的关系，并产生了新的解决和分解多项式的算法。我的最终目标是一个新的目录，描述多项式系数与其形状之间所有有趣的代数关系，以一种使其处理可以通过现代GPU等并行处理器准确计算的方式，描述阶数为4或5的曲线和曲面。

数学家研究的代数几何通常涉及非常抽象和一般的问题，我承认我自己并不完全理解。相反，我的演讲将集中在我认为对计算机图形学最相关的代数几何子集上。它们只需要简单的线性代数和齐次坐标的知识。我仍在研究一些主题，因此演示将展示未解答的问题以及答案。

Springer 1977\.  Jim Blinn在1968年在密歇根大学本科期间制作了他的第一批计算机生成图片。从1974年到1977年，他在犹他大学攻读研究生期间，进行了逼真渲染方面的研究。这些研究成果已经成为今天计算机动画系统中的标准技术。它们包括逼真的镜面光照模型、凹凸映射和环境/反射映射。1977年，他获得了博士学位，并加入了喷气推进实验室，为到木星、土星和天王星的各种太空任务制作计算机图形动画。这些动画作品在许多新闻广播中展示，作为任务新闻报道的一部分，并为今天行业中许多人首次接触计算机动画提供了机会。在JPL期间，他还为PBS系列节目《宇宙》和安娜伯格/CPB资助的项目“机械宇宙”制作了动画，后者是一套52部分的大学物理课程电视课程。在这些制作过程中，他开发了几种其他标准的计算机图形技术，包括云模拟和一种被称为“blobbies”或“metaballs”的建模技术。从1989年到1995年，他在加州理工学院工作，制作动画来教授“项目数学！”的高中数学课程。从1987年到2007年，他在IEEE计算机图形和应用期刊上有一个名为*Jim Blinn's Corner*的常规专栏，描述了计算机图形中使用的数学技术。这些内容已经汇集成了三本书。从1995年到2009年，他在微软研究院担任图形研究员，开发了一种新的数学符号方案，大大简化了曲线和曲面的代数描述和操作。他目前已经退休。
