# 供应商锁定削弱了英国政府的谈判能力 • The Register

> 来源：[https://www.theregister.com/2024/04/04/uk_cddo_admits_cloud_spending_lock_issues_exclusive/](https://www.theregister.com/2024/04/04/uk_cddo_admits_cloud_spending_lock_issues_exclusive/)

独家报道，英国政府承认供应商锁定限制了其对数十亿英镑云基础设施支出的谈判能力。

白厅内流传的由 *The Register* 报道的文件来自内阁办公室中央数字与数据办公室，称“英国政府在其部门间云采纳和管理的当前方法面临数个挑战”，这些挑战综合起来导致“风险集中和供应商锁定，从而抑制了英国政府对云供应商的谈判能力”。

该文件还指出，如果英国政府在过去十年花费数百亿英镑在云服务上的做法不作出改变，“AWS和Azure在英国政府云服务中的现有主导地位将继续存在。”

无所作为将意味着“政府在定价和产品选择上几乎没有任何影响力。”

“这条路线预测了未来十年内，公共部门可能面临其无法协商有利条款的结局，导致根深蒂固的供应商锁定和[英国监管机构]竞争与市场管理局的潜在监管审查。”

这份文件在“英国公共部门云市场”的标题下流传，作者是CDDO顾问Chris Nesbitt-Smith，由CDDO首席技术架构师Edward McCutcheon和CDDO首席技术官David Knott赞助。

提到的供应商锁定问题是在与主要云供应商签署的多项重大协议之后提出的。例如，全球最大的云服务提供商AWS与内政部签署了一项高达[£450.3 million ($569.7 million)的协议](https://www.theregister.com/2023/12/01/uk_home_office_aws/)，这是在[单一政府价值协议（OGVA）](https://www.theregister.com/2020/11/02/aws_signs_memorandum_of_understanding/)的框架下进行的。其他协议还与税务机关HMRC和工作与养老金部门签署，AWS在OGVA 2.0的框架下共计约9亿英镑（约合11.38亿美元）的交易额，并将持续至2026年10月。

曾有[建议](https://www.publictechnology.net/2021/05/13/business-and-industry/excl-government-agreement-offers-public-sector-20-discount-aws-hosting/) OGVA 在其云服务交易中实现了18%的基线节省。

根据公共部门支出研究公司Tussell的数据，AWS在2022/23财年从英国政府收取的直接公共部门收入比前一年增加了76%。

文档引述了政府当前在云采纳方面面临的挑战，包括采纳云服务所需的时间过长，部门通常需要六至九个月的时间，成本通常超过£500k来激活和部署它们。其方法也受到不一致的云环境的阻碍，导致"缺乏标准化方法"，从而在部门之间产生"云架构、安全和合规性的不一致性和重复发明"，引发了在数据共享协议中理解等效控制的风险、低效和挑战。

文件列出了英国政府在云采纳中的另外四个挑战。文件提出了一个解决方案，涉及与主要云服务提供商合作的"英国公共部门云市场服务"。它将提供"一致安全、良好架构的云环境，可以快速部署"。

文件称："该服务承诺为所有部门创造公平竞争的条件，确保遵循最佳安全和合规实践，并使政府能够更有效地与云供应商进行谈判，从而达成更好的交易和节约成本。"

然而，目前尚不清楚CDDO在没有内阁办公室内的执行与冠军商业服务(Crown Commercial Service)的帮助下将如何执行这样的策略。

实际上，文件指出，如果CCS"无法创建一个符合要求且不会阻碍预批准支出的商业实施"，那么采纳提议方法成功的关键风险是存在的。

一位接近安排的专家告诉《注册》杂志说："如果政府在质疑其谈判力量，那么也许你应该相信他们。"

专家表示，AWS的合同使得政府对云平台的承诺增加了四倍，任何即将上任的政府几乎无法退出这些交易。英国将在明年一月底之前举行大选，目前工党在民意调查中领先。

专家指出，AWS的交易结构是这样的，AWS为预付款提供更大的折扣，承诺年度最低消费。

参与谈判的相关人士表示，提议的策略缺乏详细信息以及如何影响由CCS和各政府部门领导的采购策略的计划，这些部门都有其自己成熟的商业团队。

AWS 的一位发言人告诉我们："使用 AWS 的政府部门不仅享受高达 60% 的成本节省，还支持英国境内的许多小型公司生态系统，这些公司提供与 AWS 产品和服务相辅相成，帮助客户充分利用 AWS 的优势。

"亚马逊认识到作为公共部门供应商的地位是一种特权，必须通过我们的服务质量和为英国纳税人带来的物有所值来不断重新赢得。我们知道他们只会在我们能够在这两个方面做到的时候继续作为客户。"

*The Register* 已要求英国内阁办公室、Microsoft Azure 和 Google Cloud 发表评论。
