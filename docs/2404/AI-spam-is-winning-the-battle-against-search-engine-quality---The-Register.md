# AI垃圾信息正在赢得与搜索引擎质量的战斗 • The Register

> 来源：[https://www.theregister.com/2024/04/13/google_ai_spam/](https://www.theregister.com/2024/04/13/google_ai_spam/)

采访 我们知道，谷歌搜索结果正在受到AI垃圾内容泛滥的影响，这家网络巨头试图遏制机器生成的废话的努力并没有太大帮助。

它太糟糕了，以至于AI内容检测平台Originality.ai的创始人兼CEO乔恩·吉尔汉姆告诉我们，谷歌正在输掉[它对所有这些垃圾信息的战争](https://www.theregister.com/2023/02/09/google_reverses_ai_page_spam/)，在其搜索结果中出现了[垃圾](https://www.theregister.com/2024/01/17/google_search_results_spam/)、[欺诈](https://www.theregister.com/2024/03/25/googles_aipowered_search_results_are/)内容。您可以在下方看到我们对乔恩的完整采访。

[Youtube视频](https://www.youtube.com/watch?v=2fZMOIBfEwk)

“现在显而易见的是，没有一个在谷歌上进行垃圾信息投放的人不是通过AI在做这件事，”吉尔汉姆告诉*The Register*。“并非所有AI内容都是垃圾，但我认为现在所有的垃圾内容都是AI生成的。”

吉尔汉姆的团队每月制作报告，跟踪AI生成内容在谷歌网页搜索结果中的出现程度。据称，截至上个月，大约有10%的谷歌搜索结果指向AI内容，这是在谷歌承诺下架大量推动此类垃圾信息的网站之后的结果。

“谷歌采取了这些手动操作来试图赢得一场战斗，但他们似乎仍在努力应对其算法被AI内容淹没的问题，”吉尔汉姆告诉我们。

随着AI内容的增加，人们也担心我们可能会陷入[模型崩溃](https://www.theregister.com/2024/01/26/what_is_model_collapse/)的境地，因为AI摄入其他由AI生成的材料，最终会产生其他低质量合成数据的风险。Gillham称，他的AI内容识别技术，已经被用来扫描机器生成的信息数据集，可以提供帮助，但这并非完全解决方案。
