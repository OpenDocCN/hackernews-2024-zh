# 《互联网档案馆刚刚备份了整个加勒比岛屿 | WIRED》

> 来源：[https://www.wired.com/story/internet-archive-backed-up-aruba-caribbean-island/](https://www.wired.com/story/internet-archive-backed-up-aruba-caribbean-island/)

阿鲁巴的殖民历史也意味着文件散落各处。“我们的收藏被分散了，”阿鲁巴国家档案馆的档案保存和管理负责人埃德里克·克罗斯说道。在世界各地，包括荷兰、西班牙、美国以及库拉索等其他岛屿，都有文献需要扫描。Scholing指出，建立在线查找文档的中心对于位于国外的研究人员尤其有帮助，他们不再需要前往阿鲁巴亲自挖掘档案。

让一个国家将这种项目外包给外国非营利组织是不寻常的。“在理想世界里，每个国家图书馆都能有足够的资金聘请一支优秀的团队，”滑铁卢大学历史教授伊恩·米利根说，他正在撰写一本关于互联网档案馆起源的书籍，他并没有参与阿鲁巴项目。“然而政府通常没有这样的条件。”

互联网档案馆以前从未担任过任何国家整个收藏的保管人，尽管它曾与世界各地许多国家和地区的图书馆合作过。早在2011年，它与印度尼西亚的巴厘文化办公室合作，[保护](https://www.wired.com/2012/08/brewster-kahle/)当时该办公室描述的“巴厘90%的文献”。（这现在构成了互联网档案馆的[巴厘数字图书馆](https://archive.org/details/Bali)收藏。）

阿鲁巴的档案馆希望其他国家能够效仿其数字化步伐。“这是一个非常可行的模式，可以应用于许多小岛屿、发展中国家，甚至是经济有限的大国，”Scholing说道。

与互联网档案馆合作看起来是一个对现金紧缩的档案馆管理员显而易见的解决方案。潜在的合作伙伴们确实需要考虑，依赖另一个国家的私人组织意味着什么，这个组织有其自身的挑战。

“当我们思考数字保存时，我们经常考虑技术挑战，”滑铁卢的米利根说。“但我认为最大的挑战是社会挑战，人类挑战。你如何建立一个可以在50年后依然存在的组织？”

他认为互联网档案馆具有非常“可持续的结构”，在未来保护方面。但这并不意味着它完全无懈可击。档案馆目前面临着一系列严重的法律挑战，包括[一场诉讼](https://www.reuters.com/legal/music-labels-sue-internet-archive-over-digitized-record-collection-2023-08-12/)，来自包括环球音乐集团、Capitol和索尼在内的主要唱片公司，这构成了一种生存威胁——这些唱片公司要求赔偿金额可能超过4亿美元。

这还加剧了与[出版公司](https://www.wired.com/story/internet-archive-copyright-library/)就在疫情期间建立的数字借书馆的持续争议。尽管其数字化能力远远超过许多国家，但互联网档案馆在版权持有者和技术公司之间日益激烈的战场上的位置意味着其未来也很不稳定。

互联网档案馆认为阿鲁巴的支持尤为及时。“看到阿鲁巴国家继续添加材料并上传内容的能力真是非常有力量，特别是在我们面临这种局面时，”弗里兰德说。“我们将长期以来都在这样做。”
