# Llama-3可能刚刚击败了专有AI模型 · Kadoa · AI Web Scraper

> 来源：[https://www.kadoa.com/blog/llama3-killed-proprietary-models](https://www.kadoa.com/blog/llama3-killed-proprietary-models)

Meta仅在三天前发布了Llama-3，但它已经感觉到开源模型终于在专有模型中拉近了差距的拐点。初步的基准测试显示，Llama-3 70B在许多任务中与GPT-4非常接近：

更强大的Llama-3 400B+模型目前仍在训练中，并有可能在发布后超越GPT-4和Opus。

### Meta vs OpenAI

有人推测，Meta从一开始的目标就是通过释放强大的开放模型来颠覆竞争格局，避免在AI竞赛中落后于OpenAI，采取了["焦土政策"](https://en.wikipedia.org/wiki/Scorched_earth)。  

Meta可能在计算资源和人才方面胜过OpenAI：

+   OpenAI的估计收入为20亿美元，可能还没有盈利。Meta在2023年创造了1,340亿美元的收入和390亿美元的利润。

+   目前，Meta的计算资源可能已经超过OpenAI。

+   开源可能会吸引更好的人才和研究人员。

一个可能的结果是微软收购OpenAI以赶超Meta。谷歌也在向开放模型空间进军，并拥有与Meta相似的能力。看到它们的定位将会很有趣。

### 胜利者：开发者和AI产品创业公司

我最近写了一篇关于[现在构建AI初创公司的兴奋感](https://www.kadoa.com/blog/why-building-an-ai-startup-feels-amazing)，因为随着每个主要模型的进展，你的产品会自动得到改进。随着Llama-3的发布，开发者的机会更加广阔：

+   现在不再有供应商锁定。

+   现在，开发者不再只是包装专有API端点，他们现在可以以非常具有成本效益和性能的方式深度集成AI到他们的产品中。已经有超过800个[llama-3模型变体在Hugging Face上](https://huggingface.co/models?sort=modified&search=llama3)，看起来每个人都可以为他们的用例、语言或行业进行精细调整。

+   更快、更便宜的硬件：Groq现在可以以极低的成本每秒生成800个llama-3令牌。在不久的将来，我们将会看到价格低廉的近即时LLM响应。

开源的视觉和视频多模型仍然需要赶上，但我预计这很快就会发生。

Llama-3的发布标志着AI民主化的重要里程碑，但现在宣布专有模型的死亡可能还为时过早。谁知道，也许GPT-5会让我们惊讶，并超越我们对变形金刚模型能做什么的想象。
