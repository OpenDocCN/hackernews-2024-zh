# 增长是一种心灵癌症 – Manu

> 来源：[https://manuelmoreale.com/growth-is-a-mind-cancer](https://manuelmoreale.com/growth-is-a-mind-cancer)

<主要>

我在以稍微模糊的兴趣关注苹果目前卷入的各种法律斗争。阅读他们对欧盟数字市场法案的回应让我感到悲哀。不是为了公司本身。我对公司本身完全无所谓。也不为那些经营该公司的人。我感到悲哀是因为无休止增长的追求就像一种心灵癌症。它消耗并分散了每个人的注意力。如果你是一名工匠，创造令人惊叹的物品是你的最终目标。理想情况下，你希望这些物品永远持久。即使它们不能，你也希望做得如此出色，以至于一旦某物损坏无法修复，人们会再次来找你，请你做出新的东西，而不是从别人那里购买。

苹果制造了令人惊艳的产品。我买了我正在打字的这台笔记本电脑9年前。它仍然很好用。当然，与我的新机器相比速度慢了点，但我可以用它来打电话和写博客。这很棒。我喜欢它。2015年我很高兴把钱给了苹果。但你知道谁不高兴吗？苹果。苹果不高兴我2015年买了一台笔记本电脑，而这台电脑仍然在9年后运行良好。而且他们也不高兴我4年多前买了一部手机，这部手机仍然能够完成我需要做的一切事情。因为他们需要赚钱。更多的钱。这里没有终点。*"更多"*没有终点。在某一点上，像苹果这样的公司最终会没有人愿意购买他们的产品。因为期望人们每年升级手机、笔记本电脑、屏幕、手表、平板电脑、虚拟滑雪护目镜是不合理的。那么他们会怎么做呢？他们转向服务。音乐、电影、游戏、健身、存储。无所不包。但这些也不能永远增长。因为猜猜看？还有其他公司在做同样的事情。

但是他们不能停止。他们是一家上市公司。如果他们的增长不够，这意味着他们失败了。忘记他们生产的能够持续数十年而没有问题的令人惊艳的产品吧。忘记他们是一家近三万亿美元的公司。如果他们的增长不够，股价就会下跌，这对他们来说不是好事。因为记住，这里没有终点。他们不能仅仅对自己的规模感到满意。他们不能满足于雇佣成千上万聪明人并创造惊人产品的想法。不，他们必须继续增长。而迟早，这种心灵癌症会变得恶性。

别误会，这不仅仅是苹果的问题。这是任何大公司的问题。这是每个无法接受已经达到终点的人的问题。

科里·多克托罗曾经有名地创造了“糟糕化”这个词来描述在线服务随着时间变得糟糕的悲哀趋势。我认为这不仅仅是在线服务的问题。这是一个与无止境增长追求有关的社会问题。如果你仔细想想，这是一个深刻的人类问题。这就是当你无法说停止时会发生的事情。不管你做什么，无论是积极的还是消极的，如果你不能说停止，坏事就会发生。试着去跑步，永远不停。或者试着喝水，永远不停。

但这是我们的错。作为一个社会的错。我们庆祝当苹果成为首个万亿美元的公司，但我们不庆祝有人说：“*你知道吗？我觉得我够了*”。
