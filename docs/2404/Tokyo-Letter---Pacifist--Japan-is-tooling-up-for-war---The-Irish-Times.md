# 《都柏林时报》东京信：‘和平主义’日本正为战争做准备

> 来源：[https://www.irishtimes.com/world/asia-pacific/2024/04/16/tokyo-letter-pacifist-japan-is-tooling-up-for-war/](https://www.irishtimes.com/world/asia-pacific/2024/04/16/tokyo-letter-pacifist-japan-is-tooling-up-for-war/)

本月，[日本](https://www.irishtimes.com/tags/japan/)从广岛县庆良市的港口升级了其加贺号战舰。[改装后的舰艇](https://www.newsweek.com/japan-js-kaga-helicopter-destroyer-aircraft-carrier-upgrade-china-us-alliance-1888740)现在是一艘事实上的航空母舰，能够容纳多达十二架美国制造的F-35战斗机。

对于那些敏感于象征意义的人来说，这一切都意味深长。原日本海军舰队的旗舰加贺号曾在1941年发动了对珍珠港的袭击，导致美国卷入第二次世界大战。四年后，太平洋战区的冲突在广岛和长崎的摧毁中达到高潮。

乌雷是日本海军野心的终结之地，阿马吉号战舰在1945年的美军空袭中沉没于海底。加贺的姐妹舰出云号也在升级，成为自战后首艘能够操作固定翼飞机的日本海军舰艇。

日本的“和平主义”宪法是在那场战争的废墟上由占领军编写的，尽管名义上禁止其通过战争解决争端，但上周，日本首相岸田文雄发出了他对战后时代结束的最明确信号。

在向美国立法者的演讲中，[岸田文雄](https://japan.kantei.go.jp/101_kishida/statement/202404/11speech.html)表示，日本准备帮助美国分担保护“自由世界”的负担。他说，日本与其盟友站在一艘名为“自由与民主”的船上，“为能成为你们的同伴而感到自豪”。

“我察觉到一种美国人中存在的自我怀疑情绪，不确定你们在世界上应扮演何种角色”，岸田在国会的联席会议上说道。

“这种自我怀疑正发生在我们世界处于历史转折点之际。后冷战时代已经过去，我们现在正处于将定义人类下一阶段历史的拐点之中”，他说。

“我们已经在甲板上，我们已经在执行任务。我们已经做好必要的准备。我在这里要说，日本已经与美国肩并肩站在一起。你们并不孤单。我们与你们同在。”

美国驻日本大使拉姆·伊曼纽尔最近称，岸田的演讲标志着日本军事姿态进入了一个“深刻转变和转型”的时代。

“[伊曼纽尔表示](https://www.youtube.com/watch?v=Ft8KsZn1HN8)，受到中国和俄罗斯的存在威胁驱动，东京已经承诺在2027年将国防预算加倍至国内生产总值的2%。”

他指出，这不仅使其领先于许多北约国家；它将使日本在美国和中国之后成为“世界第三大国防预算”。

今年，日本自卫队与美军首次将[中国列为假想敌](https://english.kyodonews.net/news/2024/02/47012e046105-china-named-as-hypothetical-enemy-for-1st-time-in-japan-us-exercise.html#google_vignette)。

另一项首次，日本向处于冲突中的国家乌克兰发送了军事装备。并且，日本通过将美国设计的爱国者导弹退还给美国，放宽了其防务装备转让的原则。

“没人曾想到”这样深刻的变化是可能的，Emanuel 说。“事实上，每个人都预测了相反的结果。”

并非每个人都欢迎这些变化。CNN警告称，由于相互不信任的驱使，日本、中国和美国正在亚洲推动军备竞赛，存在失控风险。

“全球三大核大国和一个快速发展中的国家，全球三大经济体和几十年来的联盟都在争夺世界上一些最具争议的陆地和海域的优势”，[分析师布拉德·伦登写道](https://edition.cnn.com/2023/01/15/asia/asia-nuclear-arms-race-analysis-intl-hnk-ml/index.html)。

然而，推动日本放弃其被日本最畅销报纸《读卖新闻》称为“完全不切实际、乌托邦式和平主义”的呼声越来越高。

在这种背景下，改装后的加贺号和出云号的重要性显而易见。航空母舰，不同于其以前的直升机母舰身份，意味着日本在本国海岸之外进行力量投射。

《读卖新闻》的社论称，俄罗斯入侵乌克兰已经成为“改变游戏规则的事件”。 [如此说《读卖新闻》的社论](https://japannews.yomiuri.co.jp/editorial/insights-world/20230210-90089/)。

“迄今为止，日本处理其安全和防务政策，就像把一只手绑在背后一样。如果日本经济继续高速增长并且美国实力强大，这种方法本是可以接受的。”

“那个时代已经过去了。”
