# ‎CodeBrew - 适用于 Java 的 IDE 在 App Store 上的评价

> 来源：[https://apps.apple.com/de/app/codebrew-ide-for-java/id6475267297?l=en-GB](https://apps.apple.com/de/app/codebrew-ide-for-java/id6475267297?l=en-GB)
> 
> 对于初学 Java 的人来说，尤其是在 iPad 上学习的话，这款应用非常适合。它非常易于使用，非常适合初学者。这款应用的一个特别之处在于，您可以在没有互联网的情况下使用它制作具有按钮和菜单（图形用户界面或 GUI）的应用程序。这是并非所有类似应用都能做到的。它还提供了示例来帮助您理解如何操作。
> 
> 由于这款应用相对较新，存在一些小问题，比如有时无法正确显示错误。但总体而言，它表现还不错。如果能增加更多的设计选项，并且允许用户自定义快捷方式来修复错误（调试），那将会更好。另外，添加一个快速切换已打开标签页的功能，比如在顶部显示它们，将会使您在项目的不同部分工作变得更加轻松快捷。
