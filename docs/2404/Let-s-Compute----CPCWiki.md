# Let's Compute! - CPCWiki

> 来源：[https://www.cpcwiki.eu/index.php/Let%27s_Compute!](https://www.cpcwiki.eu/index.php/Let%27s_Compute!)

Let's Compute! 是一本针对 Electron、BBC Micro、Commodore、Atari ST、Spectrum、Archimedes 和 Amstrad CPC 的英国月刊（尽管严格来说，第一期没有 Amstrad CPC 内容 - 从第二期开始有）。该杂志售价为 99 便士，仅持续了 12 期（1990 年 8 月至 1991 年 7 月）。

由 Database Publications 生产，主编是 Derek Meakin（来自《Computing with the Amstrad》）。最后两期由 Europress Publications 发布。

这本杂志在其 48 页中尽可能涵盖了各种内容 - 游戏评测、新闻、漫画/笑话、读者问题和输入式程序。有时输入式程序会针对特定平台，其他时候会提供你需要修改的行数，以使其在你的平台上运行。
