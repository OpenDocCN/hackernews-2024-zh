# 日本的月球着陆器在经历了另一个月球夜之后成功度过 • The Register

> 来源：[https://www.theregister.com/2024/04/25/slim_another_lunar_night/](https://www.theregister.com/2024/04/25/slim_another_lunar_night/)

日本的月球智能着陆器（SLIM）再次苏醒，已经度过了三个月球夜晚。

在社交媒体上，来自着陆器X账户的[帖子](https://twitter.com/SLIM_JAXA/status/1783330118683050427)确认，SLIM再次蔑视了一切可能性，并使用其导航摄像头拍摄了月球表面的照片。

SLIM在经历了第二个月球夜后[a few weeks ago](https://www.theregister.com/2024/04/02/jaxa_slim_awakens/)苏醒。然而，遥测显示部分电子设备（温度传感器）和电池单元出现了[malfunctioning](https://twitter.com/SLIM_JAXA/status/1773162603075821881)，登陆器能否度过第三个月球夜的机会似乎渺茫。

尽管缺乏保持电子设备温暖的加热器，但SLIM再次在月球表面活跃起来。

通常设计用于在极端寒冷条件下生存的航天器都会安装加热器以保持电子设备的温暖。月球表面的温度可以低至−150°C，而在某些陨石坑中的温度甚至可以低至−247°C。

[根据](https://twitter.com/SLIM_JAXA/status/1783330326850503075)SLIM在社交媒体上的信息：“即使在月球上度过了三个夜晚，SLIM的主要功能仍然保持完好，这在设计时是没有预料到的！

“SLIM的状态将被监测，以阐明由于昼夜环境而导致的恶化区域，以及较不易受影响的区域。”

SLIM原本只设计在月球表面的首个月球日结束前运行。由于着陆时登陆器倾斜，无法实现所有预期目标，所以任务最初被宣布为“最低成功”。

尽管如此，SLIM超出了所有耐久性的预期，在第三个月球夜后再次活跃。

JAXA并未说明SLIM是否能够度过第四个月球夜，或者登陆器电子设备的状态。四月初开始，关于第三个夜晚能否存活的希望，受到了“传感器和其他功能逐渐丧失”的警告的限制。

正如我们在早些时候的报道中所说：“祝愿你能够迎来尽可能多的月球日出。” ®
