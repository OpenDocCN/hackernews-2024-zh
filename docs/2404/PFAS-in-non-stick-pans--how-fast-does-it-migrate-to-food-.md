# 不粘锅中的PFAS：它迁移到食物有多快？

> 来源：[https://molecularspec.substack.com/p/pfas-in-non-stick-pans-how-fast-does](https://molecularspec.substack.com/p/pfas-in-non-stick-pans-how-fast-does)

*这是一篇我深入探讨关于污染和毒性问题的博客，我个人对此感兴趣，目标是为像我这样的非化学专家解释许多新闻中这些化合物背后的实际数据。每篇博客都基于二手研究，主要来源于通过Google Scholar和已知学术期刊找到的发表文章，以及我在化学分析工作中形成的经验。*

1.  三篇研究论文评估了不粘锅烹饪中PFAS传递的广泛范围。

1.  数据表明，每天用不粘锅烹饪4至70餐可能会使你进入有毒区域。

1.  一些次要见解是：PFAS传递似乎随着同一锅的重复使用而增加。而食物中的盐似乎加速了PFAS的传递。

PFAS是一个大约包含约5000种分子（并不断增长）的家族，被称为“永久化学品”，由氟和碳组成，通常具有极长的存留期。其中一些化合物属于PFAS，如PFOA和PFOS，但还有许多其他化合物。其中极少数化合物已经彻底研究了其对人体影响，但我们知道的是：

+   它们需要很长时间才能[分解](https://chemtrust.org/wp-content/uploads/PFAS_Brief_CHEMTrust_2019.pdf)，并且有在体内积累的趋势，这本身就是一个风险因素。

+   [CDC](https://www.atsdr.cdc.gov/pfas/health-effects/us-population.html)的数据显示，一旦关于特定化合物PFOS/PFOA的监管措施生效，需要10-20年时间让人群从血液中清除这些化合物。

+   存在[PFOA暴露可能与癌症和其他疾病风险增加的可能联系](http://www.c-8medicalmonitoringprogram.com/)。

+   因此，PFOA已经从PTFE（通常被称为聚四氟乙烯，商标为特氟龙）中淘汰，PTFE是一种塑料变体。

+   即使从体内清除，这些化合物也相当不易分解，很可能在土壤中积累。

更多关于PFAS结构、风险和背景的内容可以在我早前的帖子[关于PFAS和防水衣物](https://molecularspec.substack.com/p/water-repellent-clothing-and-pfas)中找到。

我最近开始使用瓷涂铸铁锅做饭。但哦，我确实怀念那些好旧的不粘锅——尽管有瓷涂，但在铸铁锅里做煎蛋却难以做到不粘。

因此，让我们通过数据透视实际看一看PFAS和不粘煎锅。

##### 假设

在这一部分，我在以下假设上有重要依赖：

1.  安全暴露限量。回顾我之前的帖子中的研究[（链接）](https://molecularspec.substack.com/p/water-repellent-clothing-and-pfas)，让我们以0.070 mg/day = 70 μg/day的PFOS量作为人体暴露的关注水平。这实际上是基于CDC报告，引用了小鼠研究，以体重和在该水平下引起“免疫抑制”为基础。虽然不是直接适用于人类，但在这个假设上更保守为好。

1.  从锅具到食品的转移速率。我使用了三篇不同的论文，每篇的迁移数量都不同，以便更好地了解情况。

1.  今天我们做了300g / 315ml的番茄酱，这被假定为一个平均份量大小。

##### 转移速率：论文1

[一篇论文](https://wjarr.com/content/impact-household-cooking-release-fluorinated-compounds-pfoa-and-pfos-tefal-coated-cookware)研究了使用不粘锅具进行烹饪，特别是来自特福尔公司的产品，发现以下来自锅具到食品的迁移：

+   从不粘锅表面到加盐番茄酱的PFOS迁移量为18至60 ng/g。

+   当不使用盐时，迁移量减少了15-25%。让我们使用加盐的数字，因为我们通常会经常使用它。

+   随着每次后续烹饪的增加，迁移水平也在增加。18 ng/g代表第一次，而在第10次尝试时达到60 ng/g。据推测，随着时间的推移，有一个增加的趋势而不是减少。

今天我们在家做番茄酱（[来源](https://commons.wikimedia.org/wiki/File:Pr%C3%A9paration_des_lasagnes_%28mars_2020%29_-_%28003%29_m%C3%A9langer_avec_la_sauce_tomate.jpg)）

通过眼测一个平均食物份量大小，让我们以300g平均熟食为例（[来源](https://ootabox.com/how-many-grams-does-a-typical-home-food-weigh-for-example-if-i-serve-myself-a-plate-of-my-home-food-does-it-weigh-around-100-gms-or-350-gms/)）。因此，在第10次尝试中，使用PTFE涂层的炊具，番茄酱的平均餐将含有300g x 60ng = 18,000ng = 18μg的PFOS。

好哇，那可不少。

因此，每天做4顿这样的饭菜将等于18 x 4 = 72μg，这超过了PFAS相关化合物的70 μg/day安全限量。这已经是4餐导致“免疫抑制”。

##### 转移速率：论文2

让我们以第二个假设为例。下表2中的[论文](https://www.mdpi.com/2304-8158/10/7/1443#B30-foods-10-01443)引用了PFODA（其他一种PFAS化合物之一）向类食品样品的迁移速率为3.05 μg/L：

+   所以300g的番茄酱大约是315ml的体积（基于这个在线[计算器](https://coolconversion.com/cooking-weight-volume/100~gram~of~tomato+sauce~to~ml)）

+   这意味着我们的番茄酱中大约有0.96μg ~ 1μg的PFODA。好的，比之前的研究中的18μg少得多。

+   在这种情况下，我们需要每天做70餐类似的菜肴，才能超过安全暴露限量（假设与PFOS相同的限量）。

##### 转移速率：3

[另一篇文章](https://www.sciencedirect.com/science/article/pii/S0039914023007506) 研究了各种食品制备托盘（而不是煎锅），发现了PFOA的迁移量在3.15至22.3 微克/千克之间。这种变化是由托盘的制造商引起的。让我们估算两个极端：

+   在安全端，每公斤体重3.15微克将相当于我们300克番茄酱的0.9微克PFOA。这可能不是问题。

+   在更高端，每公斤22.3微克将等于大约6.7微克的PFOA，用于300克番茄酱。因此，我们每天需要大约10顿饭才能进入风险区。所以这不太可能，但也没有远离风险区，让人完全放心。

##### 总结

根据不同的研究来源，每天使用非粘/PFAS涂层的炊具进行4到70次餐饮会使我们进入健康和毒性危险区域。有证据表明，每次重复使用炒锅时，PFAS转移到食物中的量会增加。

PFOS/PFOA现在还在今天的炊具中吗？我不确定这是否仍然存在，因为这些特定的化合物与健康风险紧密相关。

[EPA](https://www.epa.gov/pfas/our-current-understanding-human-health-and-environmental-risks-pfas) 表示，“PFOA和PFOS近年来已被替代”。这是基于生产商和EPA之间的自愿协议。尽管我在线上找到了一些关于这是否在行业内属实的问题（[一个倡导组织的新闻稿](https://www.ewg.org/news-insights/news/why-are-dupont-and-chemours-still-discharging-most-notorious-forever-chemical)）。

原来欧盟的法规变动是最近才有的，2019年开始生效（[来源](https://verive.eu/what-is-pfoa-and-why-is-it-banned-in-europe/#:~:text=The%20limit%20for%20PFOA%20present,paper%20and%20fibre%2Dbased%20packaging.)），将PFOA含量限制在0.025 毫克/千克，并且PFOA相关化合物的最大浓度为1毫克/千克。这涉及到整体产品，例如煎锅的构成，而不是从中迁移。

让我们来审视一下：

+   在一个直径为12英寸（30厘米）的特氟龙不粘锅中，重量约为3.4磅（1.5公斤）（[来源](https://blacksmithskillets.com/weights)）。

+   因此，欧盟的限值将是每个30厘米特氟龙锅的PFOA相关化合物最高达1.5毫克。

+   鉴于每日的安全暴露限制为70微克，该锅每天需要将其PFOA重量的高达5%转移到食物中才能有毒。

+   这感觉像是很多，这意味着在20天内涂层中就没有PFOA了，据推测这意味着不粘性能的丧失。这不是我在实践中观察到的情况。

+   所有这些都假设是符合法规的炊具。

让我们尝试根据这种锅的典型寿命来评估一下：

+   同一种特氟龙锅出厂时含有1.5毫克PFOA。

+   假设在1年内我们没有任何涂层剩余（例如，如[这篇Reddit帖子](https://www.reddit.com/r/Cooking/comments/sirtqp/every_nonstick_pan_i_get_loses_its_nonstick/)所述），假设实际上我们已经耗尽了PFOA。

+   假设我们每天都用这个平底锅做饭（这可能是极端情况），并且所有的PFOA都转移到了食物中（一些肯定会通过洗涤流入污水），我们每天会释放约4μg。

+   这是危险暴露限值的17.5倍。

我尝试以不同的方式核实了这些数字。总之，研究论文评估表明，每天使用不粘锅烹饪4到70顿饭才会变得有毒。这是一个很大的变化范围，这种范围是否实际上更广泛，尤其是在更有毒的一侧，这给人们带来了怀疑。

最后，约翰·奥利弗几年前做了一段关于PFAS的节目。约翰·奥利弗实际上在第15分钟左右说，如果不过热或刮伤，不粘煎锅并不是问题，但没有进一步详细说明。
