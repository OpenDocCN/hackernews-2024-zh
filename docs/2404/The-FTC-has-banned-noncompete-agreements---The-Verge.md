# FTC已禁止非竞争协议 - The Verge

> 来源：[https://www.theverge.com/2024/4/23/24138559/ftc-noncompete-agreement-ban](https://www.theverge.com/2024/4/23/24138559/ftc-noncompete-agreement-ban)

联邦贸易委员会已经表决通过全国性禁止非竞争协议，称其为“不公平的竞争手段”。

试图阻止员工为竞争企业工作或成立竞争企业的非竞争协议在科技领域特别普遍，我们见过像亚马逊这样的公司为仓库工人实施，然后撤销非竞争协议，还有Acer甚至因前首席执行官涉嫌违反非竞争政策而起诉其成为联想的顾问。

这一变动将迫使公司撤销现有的非竞争协议，并通知员工有关此变动的情况。现有的高级执行官的非竞争协议可以继续有效，但公司不能进入或执行新的协议。（FTC将高级执行官定义为年收入超过151,164美元并参与“决策制定”的工作者。）

FTC发现有多种替代非竞争协议的选择，包括商业秘密法和保密协议。FTC主席莉娜·卡恩在一份声明中表示：“非竞争条款会压低工资，抑制新想法，剥夺美国经济的活力。”

FTC首次提出在2023年1月禁止非竞争协议，并估计目前约有3000万工人被锁定在此类协议中。该机构表示，该禁令将每年创造超过8500家新企业，同时降低医疗保健成本并提高工人的薪酬。新规将在其在《联邦纪事》上发布后的120天内生效。
