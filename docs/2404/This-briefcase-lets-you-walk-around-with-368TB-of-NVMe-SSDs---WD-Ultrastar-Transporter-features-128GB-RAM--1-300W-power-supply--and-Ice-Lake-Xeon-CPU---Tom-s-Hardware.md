# 这个公文包让您可以携带368TB的NVMe SSDs四年，WD Ultrastar Transporter具有128GB RAM，1300W电源和冰湖Xeon CPU | Tom's Hardware

> 来源：[https://www.tomshardware.com/desktops/servers/this-briefcase-lets-you-walk-around-with-368tb-of-nvme-ssds](https://www.tomshardware.com/desktops/servers/this-briefcase-lets-you-walk-around-with-368tb-of-nvme-ssds)

Western Digital，以其硬盘和固态硬盘而闻名，推出了一款在技术意义上不是存储设备的新产品。这款产品名为Ultrastar Transporter，是一款全新的Western Digital制造的旅行箱，内置惊人的368TB NVMe闪存存储。这个箱子符合TAA标准，配备了一台12核的Intel Ice Lake服务器，内置所有固态硬盘和双200Gb以太网端口。

Ultrastar Transporter公文包解决了服务器和边缘位置之间高性能、高容量数据传输的问题。Western Digital报告称，对于物理存储传输的需求很高，特别是需要花费数天甚至数周才能通过互联网传输的大量数据。WD的新旅行箱通过传统的物理运输方式解决了这个问题，允许用户通过这种方式传输大量数据。

一些例子包括需要在一个地点处理高容量数据捕捉的工作负载，还有多数据中心工作流的普及，需要在多个位置保存大型数据集，并快速传输到这些位置，速度比原始互联网带宽提供的更快。

Ultrastar Transporter有一个机架式服务器，容纳所有368TB的NVMe闪存存储以适应所有存储需求。该服务器配备了一台12核Intel [Ice Lake](https://www.tomshardware.com/reviews/intel-10nm-ice-lake-test-benchmarks,6257.html) Xeon 4310 CPU，128GB DDR4 RDIMM内存，1300W钛金电源，以及双2TB SN740 NVMe SSD在Raid 1中作为引导驱动器。I/O包括双200GB以太网端口，双USB 3.0端口，一个DB9串口和一个VGA端口。主板上还有7个PCIe Gen4插槽，包括四个x16插槽和三个x8插槽。这些PCIe插槽连接系统的所有368TB的SSD存储。

这个箱子本身看起来像是一个传统的携行箱。外壳似乎是硬塑料，箱唇填充了金属支撑轨。箱子配备了两个拉锁机制和一个用于安装[安全锁](https://www.tomshardware.com/tag/security)的机制。你可以通过两个轮子滚动箱子或者另一个手柄像提包一样携带箱子。
