# 窥视历史的窗户：布拉格投窗事件（1419年-1997年）— 公共领域评论

> 来源：[https://publicdomainreview.org/essay/windows-onto-history/](https://publicdomainreview.org/essay/windows-onto-history/)

[](#p-1-1)

两名天主教议员，雅罗斯拉夫·波里塔·马丁尼采和维勒姆·斯拉瓦塔·赫卢姆，因公开否认鲁道夫二世1609年颁布的宽容宪章而臭名昭著。当一个在克洛斯特格拉布和布劳瑙关闭的礼拜堂中无法容纳的新教集会吸引了大量人群时，马丁尼采和斯拉瓦塔开始担心起来。当选为新教信仰捍卫者的亨德里赫·马蒂亚斯·图恩伯爵呼吁执行他们时，他们意识到为时已晚。天主教议员们玩火自焚；愤怒的市民情绪急剧升温。一大群人跟随新教贵族进入哈拉丘尼城堡。他们涌入城堡内部和上层楼，将马丁尼采和斯拉瓦塔困在塔室中。他们对议员们大声呵斥、威胁，然后抓住他们，首先将马丁尼采从窗户推了出去。与此同时，斯拉瓦塔抓住窗台，请求圣母祈求帮助。最后一次推搡，人群将他也从城堡窗户中驱逐出去。只有他们的秘书留下。他因恐惧而颤抖，紧紧依附着知名的新教贵族约阿希姆·冯·施利克寻求保护，但如同后来被称为投窗者一样，他们将他从施利克身边拉开，也扔出了窗外。
