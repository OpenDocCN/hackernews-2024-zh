# 第一个已知的分子分形是一种自然的数学奇迹

> 来源：[https://newatlas.com/biology/fractal-molecule-natural-mathematical-marvel/](https://newatlas.com/biology/fractal-molecule-natural-mathematical-marvel/)

[分形](https://newatlas.com/materials/fractal-void-cube-shock-dissipating-armor/)对于数学爱好者来说是一种迷人的模式类型，具有重复的、人造外观的结构。现在，科学家们发现了已知的第一个分子分形蛋白质，看起来是一种进化意外。

[分形](https://en.wikipedia.org/wiki/Fractal)对于数学家来说很难定义，但通常它们是由更小的结构组成的几何形状，这些结构本身类似于整体。在实践中，这意味着如果你放大分形的一部分，你会看到类似的结构 - 如果你再放大那部分的一部分，你会看到类似的结构，依此类推，基本上到无穷远。

在自然界中，分形特征出现在像雪花、闪电和河流网络等事物中。分子似乎是你期望找到它们的理想地方，因为它们可以自我排列成各种有趣的形状，但在所有现有的分子目录中，从未出现过规则的分形（即在不同尺度上几乎完全相同的那种）。

但现在，马克斯·普朗克研究所和菲利普大学的科学家们发现了第一个规则的分子分形。这是一种由一种蓝藻菌株使用的酶来生产柠檬酸，它被发现自然地组装成一个特定的分形模式，称为[Sierpiński三角形](https://en.wikipedia.org/wiki/Sierpi%C5%84ski_triangle)。

基本上，从一个单独的等边三角形开始。然后从中心打出一个倒置的三角形，这样你就有三个较小的三角形组成一个较大的三角形，中间有一个三角形的空洞（这个符号可能对《塞尔达传说》的粉丝来说很熟悉）。然后从每个较小的实心三角形中打出一个倒置的三角形，这样你就得到了第二种形状的三个，或者原始形状的九个，组成一个较大的三角形。你可以无限地继续这样做，每个较小的三角形都可以这样 - 这就是Sierpiński三角形分形图案。

“我们完全偶然地发现了这种结构，当我们第一次用电子显微镜拍摄它的图像时，我们几乎无法相信我们所看到的。”研究的第一作者弗朗茨斯卡·森德克尔说，“蛋白质形成这些美丽的三角形，随着分形的增长，我们看到它们中间有越来越大的三角形空洞，这完全不同于我们以往见过的任何蛋白质组装。”

为了弄清楚这种蛋白质如何形成如此引人注目的结构，研究人员使用电子显微镜更详细地分析了它，并发现其自组装不对称。在分形的各个位置，不同的蛋白链发现有略微不同的相互作用，形成了这一引人注目的结构。

那么，这种数学奇迹在细菌的生物学中是否有特定目的？为了找出答案，团队基因改造了这种微生物，使蛋白质不再形成分形结构，但细菌仍能正常产生柠檬酸。

这启发了研究人员调查另一种可能性：这种结构相对容易进化，所以它仅仅是偶然发生，无需具有特定目的。他们使用祖先序列重构技术追溯了数百万年来其可能的进化过程，然后生化地产生了那些古老的蛋白质。

令人惊讶的是，他们发现这种分形结构出现得相当迅速，仅经历了几次突变。但它很快就在其他蓝藻细菌系中失去了，直到最近才在一种细菌中保留下来。

“尽管我们永远无法完全确定过去发生事情的原因，但这个特定案例确实具备看似复杂的生物结构的所有特征，这个结构似乎只是因为进化非常容易而毫无理由地突然出现”，研究的高级作者Georg Hochberg如是说。

对于那些试图在几乎感觉人工的事物中寻找意义的人来说，这可能并不是特别令人满意的解释，但好的一面是，这确实表明可能还有许多分子分形等待被发现。

这项研究发表在期刊*[自然](https://www.nature.com/articles/s41586-024-07287-2)*中。
