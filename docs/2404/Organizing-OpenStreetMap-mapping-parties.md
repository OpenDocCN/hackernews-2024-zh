# 组织 OpenStreetMap 映射派对

> 来源：[https://contrapunctus.codeberg.page/blog/mapping-party-tips.html](https://contrapunctus.codeberg.page/blog/mapping-party-tips.html)

卷尺可用于绘制多种事物，例如[最大宽度：物理](https://wiki.openstreetmap.org/wiki/Key:maxwidth:physical) 用于障碍物和[车道宽度](https://wiki.openstreetmap.org/wiki/Key:width:carriageway) 用于街道。⁵ 特别重要的是当您意识到 OpenStreetMap 没有“自行车/摩托车/汽车/卡车是否可以[物理上]通过这条街道？”的概念时 - 您只能说车辆是否*合法*允许。这是非常合理的（因为这些车辆没有通用的标准宽度），但这意味着您必须测量街道的宽度才能回答这个问题。

裁缝卷尺非常适合绘制小长度，如[路缘](https://wiki.openstreetmap.org/wiki/Tag:barrier%3Dkerb)高度、[路障](https://wiki.openstreetmap.org/wiki/Tag:barrier%3Dbollard)间隙、[行人门](https://wiki.openstreetmap.org/wiki/Tag:barrier%3Dwicket_gate)的宽度等。（这些都用`maxwidth:physical`标记。）

金属卷尺（由木工和管道工使用）可以完成所有这些工作，还可以测量较宽的距离，如窄街道的宽度。质量好的卷尺足够坚硬，可以在[不折叠的情况下延伸相当长的距离](https://www.youtube.com/watch?v=6517aSbzlyE)，这样您就可以在测量时不需要别人扶持另一端。⁶ 这个距离被称为卷尺的“突出长度”。

测量师用的胶带（可以长达100米或更长）可以用于上述所有情况，还可以用于绘制较大街道、建筑物占地面积、室内房间、边界等的宽度。

激光测距仪（例如[Bosch GLM 50-23G](https://www.bosch-pt.co.in/in/en/products/glm-50-23-g-0601072VK0)，我有一个）是上述方法的更昂贵的替代品。它允许在难以到达的地方进行测量，例如测量高度限制器、门、桥梁的`maxheight:physical`。一些设备具有防尘和/或防水等级（IP），可以在尘土和/或潮湿环境中使用。其中一些还配备角度传感器，用于准确的距离测量和测量道路、轮椅坡道、台阶等的坡度。（现代智能手机也可以使用免费软件如[Bubble](https://f-droid.org/packages/org.woheller69.level/)进行测量。）
