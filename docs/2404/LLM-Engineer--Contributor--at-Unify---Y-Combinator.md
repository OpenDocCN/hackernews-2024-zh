# Unify的LLM工程师（贡献者）| Y Combinator

> 来源：[https://www.ycombinator.com/companies/unify/jobs/XoPGc3D-llm-engineer-contributor](https://www.ycombinator.com/companies/unify/jobs/XoPGc3D-llm-engineer-contributor)

**关于Unify：**

我们的使命是统一支离破碎的LLM领域，使其对所有人变得更快、更便宜和更简单。我们是一个雄心勃勃的初创公司，拥有遍布每个大洲的全球团队。我们的神经路由器自动选择最佳的LLM来处理每一个提示，我们需要帮助来展示这一技术能解锁的潜力！✨

**角色概述：**

我们正在寻找热情的LLM工程师加入我们的团队。对于那些希望深化他们在机器学习领域的专业知识并贡献于激动人心的AI项目的人来说，这是一个绝佳的机会 🤖

**我们提供什么：**

导师指导：从我们经验丰富的团队中获取见解和知识。加入每周会议和与其他贡献者讨论想法并培养学习社区的网络研讨会。与我们的团队进行技术指导和绩效评估的会议 📈

证书：为你的贡献和技能发展赢得认可🏆

赏金计划：为表现优异的顶尖贡献者提供财务激励，以鼓励突出贡献 💸

**承诺：**

作为贡献者，我们鼓励你在我们的各种预定义的应用LLM项目和竞赛中工作，可以是独自进行，也可以是与其他贡献者一起。在每个项目结束时，我们将其添加到我们的网站，并在我们所有的社交媒体平台上宣传你的优秀工作🔥

**加入我们：**

如果你对不断发展的LLM领域充满热情，并且渴望与一支敬业和充满激情的团队一起贡献，那么请申请吧！🧑‍💻
