# Twitter 的消亡：如何一个对社交媒体的‘乌托邦愿景’变成‘毒瘤’的历程 | X | 卫报

> 来源：[https://www.theguardian.com/technology/2024/apr/28/the-demise-of-twitter-how-a-utopian-vision-for-social-media-became-a-toxic-mess](https://www.theguardian.com/technology/2024/apr/28/the-demise-of-twitter-how-a-utopian-vision-for-social-media-became-a-toxic-mess)

如果说有什么是 Twitter 消亡的象征，那就是奥普拉·温弗瑞(Oprah Winfrey)的账户的崛起和停滞。

对于该平台来说，这是“一个突破性时刻”，昆士兰科技大学数字媒体研究中心教授阿克塞尔·布伦斯如是说。

“那真的是数字完全起飞的时刻。”

如今，奥普拉仍然在现已更名为X的网站上拥有账号，拥有4170万粉丝。但自2022年11月，也就是埃隆·马斯克收购该网站后的一个月，她只发布过[一次](https://twitter.com/Oprah/status/1615852138063618048) - 在2023年1月，她告诉切尔西·克林顿，她因克林顿在活动中意外穿着不同的黑色鞋子而“真的笑得很大声 😂”。

但随着关于社交媒体平台是否有责任阻止暴力或极端内容传播的争论不断加剧，另一个问题也浮出水面：Twitter/X现在到底是什么？

一度被视为新闻周期和政治辩论中不可或缺的站点，现在却越来越多地被曾经如此热衷的人们所抛弃了，这是怎么回事？

## 开始时：‘一个乌托邦的愿景’

在 Twitter 的早期年份，它设立了崇高的目标，一位不愿透露姓名的前 Twitter 澳大利亚员工表示。

“我觉得当时肯定是乌托邦的愿景。像许多创始人一样，他们真的认为自己是颠覆者，为真正的公共讨论创造了空间，” 她说道。“我觉得人们当时真的很享受这一点 —— 它是一个快速发展、创新的平台，你可以得到突发新闻，你可以关注和连接那些你真正敬仰的人。即使在早期，它总是有一些毒池，但并不完全是那样。”

“它曾经有社交含金量，” 她说。“还记得每个人都迷恋蓝色勾号的时候吗……那些没有的人假装不在乎？”

准确的月活跃用户数量不可用，但是尽管 Twitter/X 从未像 Facebook、WhatsApp、Instagram 或 TikTok 那样广泛吸引大众，多年来它在新闻和政治领域产生了非常大的影响。

“它有一个非常特定且有限的受众群体，” Bruns 说。“但你可以在 Twitter 上接触到记者、政治家、活动家、各种形式的专家……通常是其他线上和线下社区中有影响力的人士。”

墨尔本斯威本科技大学媒体与传播高级讲师贝琳达·巴尼特说：“它成为了一个真正把自己绝对置于新闻周期中心的公司。实质上，它成为了特别是记者无法舍弃的工具。”

这在一定程度上是因为 Twitter 的功能 —— 特别是 @ 提及和标签 —— 使其非常适合发布突发新闻。

[相关：Elon Musk 的辩论与悉尼教堂刺杀视频无关于言论自由，而是为了刺激他的追随者 | 贝琳达·巴尼特](https://www.theguardian.com/commentisfree/2024/apr/23/elon-musk-battle-over-the-sydney-church-stabbing-video-is-not-about-freedom-of-speech-its-to-titillate-his-followers)

例如，在日本，Twitter之所以变得如此之大，部分原因是因为2011年，当国家遭受灾难性海啸时，人们正是用它来沟通和组织，前Twitter员工说。

“它成为人们的真正生命线，是人们得救的途径，”她说。

2021年的皮尤研究发现，69%的美国Twitter用户称他们从该网站获取新闻，46%称该网站增加了他们对当前事件的理解，30%称它使他们在政治上更加积极参与。

新闻爆发功能并非没有问题。Bruns说，尽管平台的即时性给了异见者和公民记者发声的机会，使其在阿拉伯之春等起义中至关重要，但也允许政客绕过传统的新闻媒介。

“有很多政客实际上停止向记者提供采访，因为这样他们也不得不接受批评性问题，基本上只在Twitter上发布他们的公告。”

Barnet说，关于误导信息和恶意攻击问题，一直存在问题，但公司采取了措施试图对抗一些最严重的影响，通过实施她所称的“三大支柱”：用户蓝勾验证，审查政策和信任与安全团队。

“所有这些因素共同作用，使其在新闻事件中相对可靠，这也是人们流连其间的原因。在旧Twitter上，虽然会出现误导性信息迅速传播的情况，但它们通常会在传播到任何地方之前就被封杀。”她说。

## 现在：马斯克的狂野西部

所有这三根支柱在2022年末马斯克收购该平台后迅速解体，Barnet说。

信任与安全团队是在马斯克收购该公司后的疯狂几周里被解雇的一员，他在第一天走进总部时拿着一个陶瓷水槽。一段马斯克进场的视频被贴在网站上，标题是：“让这个水槽进入”。

许多因违反其在线规则而被封禁的人，包括唐纳德·特朗普，他们的账户得到了恢复（尽管特朗普的账户后来再次被封禁）。

验证流程发生了巨大变化。现在，人们不再因为是公众人物或为认可的新闻网站工作而被授予蓝勾，而是可以购买。

审查的方法也发生了变化。马斯克与澳大利亚政府的争执揭示了他对X的愿景，他将其视为言论自由的堡垒。

“他们非常不愿意进行任何形式的调节，”布伦斯说。“在某种程度上，这代表了美国在言论自由问题上的更广泛看法，认为这是一种绝对的好事。而在澳大利亚、欧洲和其他许多地方，更多的是需要平衡言论自由的权利和免受有害言论侵害的权利。对于许多在其他方面相当自由的美国人来说，这听起来像是审查制度。”

具有讽刺意味的是，X已经[暂停了批评马斯克的人的账号](https://www.theguardian.com/technology/2022/dec/15/twitter-suspends-accounts-journalists-musk)，包括几位2022年批评他的CNN、纽约时报和华盛顿邮报的知名记者的账号。与此同时，他还[封禁了一些追踪他私人飞机行踪的账号](https://www.npr.org/2022/12/14/1142967190/twitter-elon-musks-private-jet)，使用的是公开的数据。

“埃隆希望两者兼得，”巴尼特说。“他希望它成为最初的推特，这对新闻周期确实至关重要”，但同时也“剥夺了推特多年来制定的支柱和流程，这些流程有助于一个能找到事实的社区。”

“我认为它正在变成一个有毒的混乱，”巴尼特说。

## 未来：一个无法控制的地方。

研究显示，根据皮尤中心的调查，在马斯克收购推特后的头几个月里，60%的美国推特用户都暂时离开了这个平台，有四分之一的受访者表示他们在一年内可能完全不再使用这个网站。

即使是最多产的推特用户也在减少使用该平台，他们每月发布的推文数量下降了25%。

是否这种趋势继续下去是一个更难回答的问题，部分原因是在马斯克的领导下，研究社交媒体的研究人员的成本变得非常昂贵，以至于他们无法继续他们的工作。

多年来，推特向学术研究人员和私营部门提供了应用程序接口（API），但需付费。大约一年前，访问这些API的费用飙升。

Related:[令人惊讶！言论绝对主义者埃隆·马斯克并不真正热爱言论自由](https://www.theguardian.com/commentisfree/2024/apr/06/elon-musk-free-speech-lawsuits)

皮尤研究中心数据实验室主任亚伦·史密斯表示，他们在多年来在 Twitter 上已经开展了“相当丰富的工作”，但自从访问推文的费用上涨——他说，现在访问 API 的年费“超过了我们团队几年的整体研究预算”——之后，他们已经无法继续对该平台进行研究。

Bruns 说学术界也面临同样的困境。“你无法进行任何特别探索性的研究，寻找平台上的仇恨言论机器人或虚假信息。实际上，[X] 已经让自己的价格超出了市场。”

他说这是一种遗憾，因为以前在 Twitter 上的学术研究曾帮助平台识别和清理仇恨言论和虚假信息的小团体，现在这些将更难以监管。

“它已经开始转变成为更类似于……像 Gab 或 Parler，甚至是 [特朗普的] Truth Social 这样的平台，你会看到极右的人在狂热地相互赞同，狂热地对其他人进行仇视。”

即使这位前员工也已经停用了她的账号。“我觉得现在它真的是一个非常危险的空间，它已经无法控制了，”她说。

“有时我会想念它。我一直认为它对记者和公民记者来说是一个了不起的新闻源……我不知道，我发现自己面对突发新闻时不知所措。留下了一个空洞。我希望有人会尝试填补这个空缺。”
