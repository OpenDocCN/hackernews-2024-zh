# 冰川正在融化，但留下的尘土可能会帮助拯救我们 - 华盛顿邮报

> 来源：[https://www.washingtonpost.com/climate-solutions/2024/04/06/glacier-greenland-rock-flour/](https://www.washingtonpost.com/climate-solutions/2024/04/06/glacier-greenland-rock-flour/)

米尼克·罗辛长大在格陵兰冰川流出的细泥周围。直到他自己的女儿长大并且已经二十多岁时，他才意识到它有多么特别。

在格陵兰农村度假期间，那里没有电，她正在一个乳蓝色的峡湾里捞冰来喝金汤力时，这些泥土紧紧地抓住了她的脚，以至于她不得不放弃其中的一只靴子。
