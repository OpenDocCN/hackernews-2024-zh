# 因果性、相互作用和复杂性

> 来源：[https://www.alexahn.com/2024/04/causality-interaction-and-complexity.html](https://www.alexahn.com/2024/04/causality-interaction-and-complexity.html)

在高度混沌的系统中，如高温气体中，使用原子因果模型并非理想。相反，有效的因果模型是近似每个原子与每个其他原子相互作用的程度。如果我们增加温度，则每个原子相互作用的原子数应增加。随着温度降低，每个原子相互作用的原子数应减少。

如果我们随机采样任何一个原子，则平均而言，该原子应与一组特定大小的原子相互作用。我们不是以条件概率和因果关系为思考方式，而是以一组相互连接的事件为思考方式。这是因为以线性方式分析混沌系统并不是计算有效的。

我们可以将同样的推理方法应用于采样。如果一个系统有特定的采样率，系统的输入将根据采样率进行批处理。换句话说，系统无法辨别单个采样内事件的顺序，只能在不同采样之间进行。事件的耦合（根据采样率）并非出于计算效率考虑，而是系统的特性之一。

奇怪的是，关于因果关系的主要推理方式是通过原子蕴涵，比如 p 蕴涵 q。如果我们降低一个混沌系统的温度，使得每个原子仅与另一个元素相互作用，我们不就达到了相同的效果，只是没有了身份？或者，如果我们增加系统的采样率，以至于每个事件都可以表示为一个原子样本，我们不也达到了同样的效果？

看起来，因果性是两个系统具有不同熵级别或不同采样率的结果，因为熵级别或采样率的差异允许捕捉模式。如果两个系统具有相同的熵级别，则彼此无法内省，因为它们对彼此看起来相同。同样地，如果两个系统具有相同的采样率，则在一个样本内事件的顺序无法解码。但是，如果两个系统具有不同的熵级别，则熵较低的系统将对熵较高的系统显得更有秩序。同样地，如果两个系统具有不同的采样率，则具有较高采样率的系统将对具有较低采样率的系统看起来更有秩序。由于这种差异，可以构建与事件的因果模型，这些事件是由于熵导致的互动连接或由于采样而批处理。
