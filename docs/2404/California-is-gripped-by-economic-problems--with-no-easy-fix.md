# 加州正陷入经济问题之中，问题棘手且难以解决。

> 来源：[https://www.economist.com/united-states/2024/03/31/california-is-gripped-by-interlocking-economic-problems-with-no-easy-solution](https://www.economist.com/united-states/2024/03/31/california-is-gripped-by-interlocking-economic-problems-with-no-easy-solution)

美国许多最进步的政策，从刑事司法到[车辆排放](https://www.economist.com/the-economist-explains/2022/08/27/how-revolutionary-is-californias-ban-on-petrol-powered-cars)，都起源于加州，它在右翼政治家眼中扮演着独特的沙袋角色。每隔几年，声明加州是个失败的州或者加州梦正在变成噩梦，就成了一种时尚。这种言论常常言过其实：纯粹从经济实力上看，加州仍然是美国最强大的州。但尽管它在创新方面的持续[成就](https://www.economist.com/finance-and-economics/2024/02/12/san-franciscos-surprising-comeback)（尤其是在人工智能方面），加州似乎再次步入其周期性的艰难时期。
