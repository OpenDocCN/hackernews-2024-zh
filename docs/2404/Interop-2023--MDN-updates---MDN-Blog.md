# Interop 2023: MDN 更新 | MDN Blog

> 来源：[https://developer.mozilla.org/en-US/blog/interop2023-mdn-doc-updates/](https://developer.mozilla.org/en-US/blog/interop2023-mdn-doc-updates/)

[Interop 项目](https://github.com/web-platform-tests/interop#the-interop-project)是一个基准和测试套件，用于跟踪不同浏览器之间的 Web 功能互操作性。然而，Interop 不仅仅是测试；它是浏览器供应商之间的联合倡议，旨在确定哪些重要功能可以从跨浏览器的一致性中受益，并在一年的时间内推动开发工作。

在过去的三年里，开发人员一直致力于共同提高整体的 Interop 基准分数。像往年一样，[Interop 2024](https://wpt.fyi/interop-2024) 仪表板显示了浏览器供应商已同意集体努力的功能以及在不同浏览器间的网页功能兼容性评分进展。

这项倡议的主要目标是确保开发人员可以为所有人创建无缝的网络体验，无论他们选择的浏览器是什么。当开发人员可以建立在一致的网络平台上时，每个使用网络的人在他们的日常在线活动中都会受益。

> 网络平台的一个关键优势是它是由标准定义的，而不是由单一实现的代码定义。这创造了一个共享平台，不受特定硬件、公司或商业模式的限制。 - [Interop 2022 公告, Mozilla Hacks](https://hacks.mozilla.org/2022/03/interop-2022/)
