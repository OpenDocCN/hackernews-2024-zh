# 美国法院阻止EPA下令在塑料容器中消除PFAS - 新引导

> 来源：[https://www.thenewlede.org/2024/03/us-court-blocks-epa-order-to-eliminate-pfas-in-plastic-containers/](https://www.thenewlede.org/2024/03/us-court-blocks-epa-order-to-eliminate-pfas-in-plastic-containers/)

美国上诉法院撤销了美国环境保护署（EPA）下令一家公司停止生产塑料容器，这些容器在杀虫剂、家用清洁剂、调味品和许多其他产品中渗出有毒的全氟和聚氟烷烃物质（PFAS）的行动。

美国第五巡回上诉法院于3月21日[裁定](https://peer.org/wp-content/uploads/2024/03/3_22_24-Fifth-Circuit-Opinion.pdf)，EPA在去年12月向德克萨斯州的Inhance发布命令，禁止该公司通过其氟化工艺为容器生产或处理PFAS时，超出了其法定职权。

EPA的行动是在机构确定氟化过程中产生的某些PFAS类型“具有高度毒性并带来无法通过禁止生产以外的方式防止的不合理风险”之后采取的。

该机构根据《有毒物质控制法》（TSCA）发布了这些命令，该法授权EPA对进入美国的化学品进行监管和筛查。

但Inhance质疑EPA的行动，认为EPA错误地试图根据TSCA的一个条款对公司采取行动，而Inhance的工艺并不构成新用途。

法院认同Inhance用于防止液体泄漏出其塑料容器的氟化技术（一种产生PFAS化学品的过程）并不算是重大的新用途，因为Inhance自1983年以来一直在使用这一工艺。

法院意见指出，EPA对法律的解释“扭曲了TSCA的框架，违背了常识”。法院表示，EPA可以根据TSCA的另一部分对Inhance采取行动，但必须进行行动的成本效益分析。

当询问其评论时，EPA表示正在审查法院的决定。

Inhance在[声明](https://www.inhancetechnologies.com/news/fifth-circuit-unanimously-rules-in-favor-of-inhance-technologies)中称法院的决定是“决定性胜利”，并表示这将避免对“下游产业”和相关供应链造成的破坏。

考虑到有安全替代品存在，公共雇员环境责任组织（PEER）的科学政策主任Kyla Bennett表示，该公司继续使用其用于高密度聚乙烯（HDPE）容器的产生PFAS的氟化技术是“不道德的”。

“我怀疑会有责任诉讼出现，”她说。“当你有这么多PFAS，尤其是PFOA（全氟辛酸）-渗入产品时，总会有人起诉。”

二月份，PEER和环境健康中心（CEH）[对EPA提起了诉讼](https://www.thenewlede.org/2024/02/epa-is-illegally-concealing-data-about-pfas-in-millions-of-plastic-containers-lawsuit-alleges/)，指控该机构违反了TSCA法案，拒绝提供有关Inhance塑料容器中PFAS化学物质的健康和安全数据，EPA称其为“机密商业信息”。

美国境内的水道和大多数美国人的血液中都被成千上万种PFAS化学物质污染，这些化学物质不会自然降解。根据EPA的研究，无论是在实验动物还是人类中，某些PFAS化学物质的暴露都可能对人体的多个器官和系统造成危害，包括心脏、肝脏、肾脏和免疫系统。([原文链接](https://www.epa.gov/superfund/proposed-designation-perfluorooctanoic-acid-pfoa-and-perfluorooctanesulfonic-acid-pfos))

先前的研究表明，PFAS可以从容器中渗出，包括2023年由圣母大学进行的[研究](https://beyondpesticides.org/dailynewsblog/2023/03/pfas-leaches-into-ketchup-mayo-other-common-foods-elevating-health-hazards/)发现番茄酱和蛋黄酱中含有PFAS，以及2022年EPA的[研究](https://www.thenewlede.org/2022/09/epa-confirms-pfas-leach-from-shipping-containers-into-food-other-products/)发现8种PFAS化学物质从运输容器壁渗出至其液体内。

蚊虫控制公司克拉克蚊虫已开始使用具有非氟隔离保护的容器进行杀虫剂处理。EPA在2021年[审查了这一举措](https://www.thenewlede.org/wp-content/uploads/2024/03/EPA-Clarke-letter.pdf)并确定该公司选择了可接受的替代品。
