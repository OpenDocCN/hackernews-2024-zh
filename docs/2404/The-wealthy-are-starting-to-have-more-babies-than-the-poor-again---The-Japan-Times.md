# 富人开始再次比穷人生更多孩子 - 《日本时报》

> 来源：[https://www.japantimes.co.jp/commentary/2024/04/03/wealthy-having-babies/](https://www.japantimes.co.jp/commentary/2024/04/03/wealthy-having-babies/)

在人类历史的大部分时间（可能包括史前时期），男性更高的社会地位与拥有更多子女有关。

这种正面状态与后代之间的关系似乎在像传说中的千里之父成吉思汗这样的专制统治者身上达到了顶峰 — 一项2003年的基因研究发现，全球每200名男性中就有一人可能携带他的Y染色体。对于女性来说，这种多产的水平当然是不可能的，总体上她们的状态与后代之间的联系较弱，但一般来说，拥有更多资源的人会有更多的子女。

随后，在18世纪，情况开始发生变化。在工业革命前沿国家以及随之而来的生活水平和预期寿命革命中，出生率下降。在这些日益富裕的国家内，富裕和受教育的人士 — 教育成为社会地位的关键标志 — 在减少家庭规模方面处于领先地位。
