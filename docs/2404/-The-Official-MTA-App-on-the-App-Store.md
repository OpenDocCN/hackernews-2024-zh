# ‎App Store 上的[纽约市交通局官方应用](https://apps.apple.com/ca/app/the-official-mta-app/id1297605670)

> 来源：[https://apps.apple.com/ca/app/the-official-mta-app/id1297605670](https://apps.apple.com/ca/app/the-official-mta-app/id1297605670)
> 
> 这款应用非常实用，比我用过的一些公共交通应用更好。它显示所有延误及其原因，每条线路的状态，并显示我附近的每个站点，可选择地铁、公交或轨道，并且你甚至可以同时查看这三种方式。在我看来，这已经足够快速开始一天或前往某处所需的一切。但我没能找到路径选择的功能。例如，如果我想去一个陌生的地方，我必须使用未包含在该应用中的第三方应用。说实话，即使没有那部分内容，这款应用已经拥有你所需的一切，是一个很好的选择。
