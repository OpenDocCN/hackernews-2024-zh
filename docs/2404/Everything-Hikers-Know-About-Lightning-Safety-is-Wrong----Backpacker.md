# 一切远足者关于闪电安全的认知都错了 - Backpacker

> 来源：[https://www.backpacker.com/survival/natural-hazards/lightning/lightning-safety-facts-for-hikers/](https://www.backpacker.com/survival/natural-hazards/lightning/lightning-safety-facts-for-hikers/)

大约十年前，当我第一次在大学户外俱乐部学习风险管理时，我被教导说，在雷暴中假设“闪电姿势”是保护自己的最佳方法（剧透：其实没什么用）。如果你不熟悉这个户外传说，让我来给你提供一个复习：闪电姿势包括蹲在睡垫上，双脚并拢，胸部压在膝盖上，双手压住耳朵。据说这可以减少触电的可能性。

在过去的十年里，许多户外教育团体已经放软了他们的立场，现在说闪电姿势只能*稍微*减少风险，应只作为最后的选择。但很多人，包括国家户外领导学校（NOLS），仍然建议在尝试了一切其他方法后，坐在睡垫上作为帮助隔离你的身体免受地面雷击的最坏情况方式（这并没有帮助）。还有其他人，包括童子军，说[触碰金属](https://www.scouting.org/health-and-safety/gss/gss03/)会增加你被雷击的风险（另一个迷思）。

很少有远足者花时间定期更新他们的知识。因为安全研究真的很烦人。权威地背诵几十年前的格言给我们的朋友和家人听，更有趣（我也有这个罪过）。所以，请允许我提供我们过期的更新。以下是关于野外闪电安全的真相。

**## 闪电姿势实际上是无效的。**

NOLS教导说，闪电姿势是一种最后的选择，可以在远足者尽力寻找掩蔽物之后减轻地面雷击的影响。但据气象学家、闪电安全委员会的安全专家Ron Holle博士说，情况并非总是如此。

“闪电姿势是毫无用处的，”他说。“有很多推测性的建议，像是‘我猜这样或许有用’，但我有一个文件柜，里面有大约5000起因雷击而死伤的案例，而这些‘我猜’的方法一个都不管用。”

[美国国家严重风暴实验室（National Severe Storms Laboratory）](https://www.nssl.noaa.gov/)的研究科学家Kristin Calhoun博士倾向于认同，闪电姿势是无济于事的。

“如果我在开阔的田野上，我不会采取那个姿势，”她说。“我会跑。”她说，找到更低的地方并站在高度一致的树荫下是减少闪电风险的少数措施之一。（NOLS也同意，寻找庇护应该是徒步者在风暴中的首要任务。）她补充说，站着或坐在睡垫上“没有什么区别”她所知道的。最好将垫子放在背包里，集中精力移动到更安全的位置。

## 你可以被距离10英里的风暴击中。

我们中的许多人站在山脊或山顶上，看到远处正在形成的风暴，并告诉自己我们还有时间。卡尔霍恩说，这是一个危险的误解。

“人们通常直到暴雨已经在他们头顶才开始担心风暴，”她解释道。“但是通常是第一道闪电最致命，因为人们还没有采取保护措施。”

**## **站在树下有时比站在开阔地更危险。**

闪电倾向于击中地表上最高的物体。因此，如果你站在一个高高的山脊上，你的头就成了最高点（这并不理想）。但是站在高大或孤立的树下有时可能更糟糕。这是因为直接击中——即人们直接被闪电击中的情况——只占所有与闪电相关的伤害的3到5％。但是地面击中——即人们受到通过地面传导的电击的情况——却占了伤害的50％。

如果你站在树下，可能会比站在开阔地更容易受到潜在的风险。这是因为闪电更有可能击中一棵高100英尺的树木，而不是一个5英尺高的人。如果确实如此，站在树下的远足者会首先受到余震的影响。

“如果闪电击中树木，地面电流会通过树木下行，然后通过地面回来，”卡尔霍恩解释道。如果你远离树干，电流可能会在到达你之前分散。但是如果你就在旁边，你会遭受电击的冲击。卡尔霍恩的建议是：找一个高度一致的较矮的树木丛站在下面。

## 事实上，金属并不会吸引闪电。

“闪电不会被金属吸引，”霍尔说道。“它会穿过金属，但不会被金属吸引。”他说，闪电只会被三样东西吸引：高耸的结构，孤立的结构和尖锐的结构。“无论这些结构是由金属还是木材制成都无所谓。”

如果闪电击中建筑物是因为它很高，金属材料可能会比木头或塑料更快地将闪电导向地面，因此也更强烈。因此，如果你在露营地有一个挂熊食物的杆子，请尽量不要在靠近它的地方搭帐篷。

“金属可以导电，电流可以跳到10到20英尺外的帐篷上去，”卡尔霍恩说道。

**## 蹲下可能有所帮助——但躺下会让情况变得更糟。**

蹲下可以减少你成为景观上最高点的可能性。但是，矛盾的是，躺下比站立更危险。因为当你躺下时，你与地面的接触最大化。这使得地面雷击更有可能影响到你。同时也增加了地面雷击可能通过脚进入你的身体，并从手臂或颈部退出——沿途穿过你的心脏。

## **帐篷不能保护你免受闪电伤害。**

“只有两个真正安全的地方能避雷：一个是有线路或管道的真正建筑物，这会产生[法拉第笼](https://www.wired.com/story/the-physics-of-faraday-cages/)效应；另一个是金属顶的车辆，”气象学家罗恩·霍尔说。然而，帐篷提供不了避难所。

“自2006年以来，更多的人在露营帐篷中被雷击死亡，而不是独自徒步旅行时，”卡尔胡恩说。“你也会看到这些可怕的事故，许多人因为身处被雷击的封闭区域而受伤。”

所以，如果你要搭帐篷，选择一个你愿意在暴风雨中站立的地方：树线以下、远离高山脊，并且最好位于一片一致高度的树木之间。

## **躲在洞穴里是个糟糕的主意。**

如果你在暴风雨中被困并找到一个干燥的洞穴或岩石悬崖，你可能会认为这是一种恩赐。不过，卡尔胡恩说你要再想想。

如果电流沿着悬崖边缘行进并遇到洞穴入口，它不会绕过洞顶到洞底。相反，它会跳到你身上，利用你的身体作为捷径。虽然避雨听起来不错，但最好还是保持移动。

**## **“偶然”被困在暴风雨中是不存在的。**

“你并不是‘被困’在暴风雨中，”霍尔说。“你的决策可能出现了一些错误。”

现代天气预报非常准确，他解释说——只要你查看适合你所在路径或峰顶的天气，而不是最近的城镇天气。如果预报有可能出现风暴，没有理由不注意。“但是一次又一次地，我看到人们计划特定的远足或攀登，他们觉得自己必须在预定的那一天出发，”他说。登顶狂热是一个大问题。霍尔（Holle）怀疑，人们对闪电的尊重现在比10或20年前少。

“这些活动的参与者越来越多，”他说。“新一代人需要提醒注意风险。”

## **附加信息：闪电趣味事实**

闪电可能很危险，但也是大自然最壮观的奇观之一。一旦安全到达，可以坐下来好好欣赏。

**闪电总会引发雷声——即使你听不到它。** 当闪电击中时，闪电周围的空气受热膨胀得非常快，以至于空气分子朝着每个方向迅速射出。这种热导致的冲击波产生了我们所知道的雷声。即使你听不到，它也存在。“热闪电”，例如，不是无声或特别的——它只是[远处的常规闪电](https://www.pnj.com/story/weather/2023/09/12/heat-lightning-without-thunder-fact-or-fiction-explained/70834571007/)。

+   **不同类型的闪电听起来不一样。** Calhoun说，短促的急促爆炸意味着附近发生了快速的云对地闪电，但是长时间的持续隆隆声意味着闪电更水平地弧形运动。隆隆声的每一拍都是领导者从主流中脱离并通过天空向你分叉的声音。

+   **一道闪电可以覆盖整个州。** 地球同步卫星经常测量数百英里长的闪电。2021年，研究人员记录了一次长达[477英里](https://www.nbcnews.com/science/science-news/477-mile-lightning-bolt-spanning-3-states-sets-world-record-rcna14323)并跨越德克萨斯州、路易斯安那州和密西西比州的闪电。

+   火山还能引发闪电。爆发的火山灰云也能积聚电荷。（龙卷风和飓风云层也能）。通常，这种电荷足以在喷发的火山上[引发闪电](https://www.snexplores.org/article/volcanic-eruption-sparked-highest-lightning)。

+   **闪电有助于肥沃土壤。** 闪电以足够的力量划破空气，以至于[分解空气中的氮分子](https://indianapublicmedia.org/amomentofscience/lightening-helps-fertilize-soil.php)。这使得氮原子自由组合氧，形成被称为“硝酸盐”的分子，对土壤和植物健康至关重要。
