# 一年与 Ergodox EZ - Ryan Bigg

> 来源：[https://ryanbigg.com/2024/04/one-year-with-the-ergodox-ez](https://ryanbigg.com/2024/04/one-year-with-the-ergodox-ez)

一年前，我从 [Kinesis Freestyle Edge](https://kinesis-ergo.com/keyboards/freestyle-edge-rgb/) 升级到 [Ergodox EZ](https://ergodox-ez.com)。最初吸引我的是拇指键，以及在键盘周围散布的“备用”键中编程不同的宏功能。

我使用 [Monkeytype](https://monkeytype.com) 来练习键盘打字，那里的统计数据显示，我大约花了一个月的时间，从最初的 40wpm（相当痛苦）提高到了 90wpm。那时我开始真正感觉自己熟悉这个键盘。我现在可以在这个键盘上达到约 130wpm 的速度，这与我在 Freestyle 和笔记本键盘上的速度相同。

我最初购买 Kinesis 键盘是因为使用单一键盘导致肩部持续疼痛。拥有两个键盘，可以将它们保持在略微超出肩宽的距离有助于减少理疗费用。自从使用分离键盘以来，我不再需要去理疗师那里解锁肩部。所以这些键盘已经为自己赚回了成本。

自从购买 Ergodox 以来，我还开始充分利用层次系统。您可以在这里看到完整的布局：

关于 Layer 0 的几点需要注意：

1.  退格键位于通常是大写锁定位置的地方。我对此有很强的感觉 - 所有键盘都应该是标准的。

1.  按住的键：Z 键变为左控制键，斜杠变为右控制键，分号切换到第二层。

1.  反引号和波浪号位于左键盘的左下角。按住此按钮，切换到第一层。

1.  命令、选项和退出键位于拇指键上。没有复杂快捷键的“Emacs 爪子”。

1.  左上角是等号。在编码时，我经常使用它，所以它有特殊优先权。

1.  “Hyper”键是我 [Hammerspoon](https://www.hammerspoon.org/) 快捷键所需的键之一。我用它们快速在应用程序之间切换。

1.  拇指键上有剪切（刀）、粘贴和复制的快捷键。

1.  我可能会将右拇指上的空闲“Tab”键重新映射为其他功能 - 我习惯于使用左上角的 Tab 键。目前我还没有具体的想法，可以将它映射到什么上。

一个敏锐的观察者会注意到这里没有花括号，但我仍然成功地编写了 Ruby 的哈希和 JavaScript 的对象。我将它们保存在第一层，其中“E”和“R”键是我的花括号。如果我按住反引号（左键盘左下角），就会触发层切换到第一层。键盘通过点亮红色 LED 来指示这一点。然后我按下“E”，就会得到一个方括号。如果我再按两次反引号然后按“E”，我就会得到一个花括号。我认为这是键盘的一个很酷的功能之一。您可以配置当按键被单击、双击、长按或单击后长按时的操作。我使用了前三种，但尚未找到单击后长按的用途。

在第一层的右侧面板上是箭头键。我仍然无法在脑海中映射出左右箭头位于左键盘上，而上下箭头位于右键盘上。因此，我在该层上有传统的箭头布局，大致位于我期望的键盘右侧位置。

接下来是第二层，即媒体控制。在我的笔记本键盘上，这些控制功能位于功能键上，总感觉有点难以触及。要进入这一层，我按住分号键，键盘上的绿色 LED 指示已切换。然后我可以执行所有常规的音乐操作，比如播放、暂停、静音等。

出于我完全不明的原因，键盘最多可以有32个不同的层。我无法想象自己会需要比我现在拥有的更多的层。甚至想到需要从第32层切换而不是第28层就让我头痛不已。

最后：关于键盘内部的开关。键盘配备了 Cherry Brown 开关，这是我认为最好的反对在办公室工作的理由。它们声音极大，而且足够僵硬，以至于即使像我这样打字很用力的人也能感觉到必须用力按下。我后来将开关更换为 Cherry Reds，这种开关声音更轻，也不那么僵硬。我认为这可能使我在这个键盘上的准确性和速度提高了5-10%。
