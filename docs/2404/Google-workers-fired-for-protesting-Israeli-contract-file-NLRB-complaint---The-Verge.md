# 谷歌因抗议以色列合同而被解雇的员工提交了NLRB投诉 - The Verge

> 来源：[https://www.theverge.com/2024/4/30/24145680/google-workers-fired-project-nimbus-protest-nlrb-complaint](https://www.theverge.com/2024/4/30/24145680/google-workers-fired-project-nimbus-protest-nlrb-complaint)

与公司内部抗议活动有关而被谷歌开除的软件工程师称，公司是因为他仅仅观看了针对以色列国防合同的示威活动而对他进行报复。

这位前员工（要求匿名）说，他是在午餐时间前往谷歌纽约办公室10楼的休息室查看抗议活动的。

“我到那里时，地板上可能有20多个人坐着。我没有和他们任何一个人交谈，我和站起来的人交谈，他们在分发传单，扮演其他角色，”他说道，补充道，抗议者穿着相同的T恤。

然后，该员工回到他的桌子前，然后在下午5点左右回到抗议现场。“我和他们聊了大概四分钟，就像‘哦，天哪，你们还在这里坐着！情况怎么样？’”他说。然后，他从附近的沙发上结束了工作日。该员工表示，第二天回到谷歌时并未发生任何意外。当晚，在晚餐时，他收到了谷歌发来的解雇邮件。

“我认为这是谷歌打压员工发声的更大背景的一部分，”这位前员工说道，他在谷歌工作了将近三年，是Alphabet Workers Union的领导成员之一。（Alphabet Workers Union是一个[非合同工会](https://collectiveaction.tech/2021/the-abcs-of-googles-new-union/)，意味着它尚未被NLRB承认。）

谷歌最初对九名员工在纽约市和加州Sunnyvale的办公室内占领行为进行了行政休假，以抗议与以色列政府的12亿美元云计算合同“Nimbus项目”有关。那些员工们也因此被[逮捕了](/2024/4/17/24133056/google-protests-project-nimbus-no-tech-for-apartheid)。随后，公司因与抗议活动有关而[解雇了28名员工](/2024/4/17/24133700/google-fires-28-employees-protest-israel-cloud-contract)。在一封内部备忘录中，谷歌全球安全负责人Chris Rackow表示，公司将视情况采取进一步行动。

“我们公司绝大多数员工都是做正确事情的，” Rackow 的声明说道。“如果你是其中少数几个被诱惑认为我们会放任违反我们政策的行为的人，请重新考虑。公司对此非常重视，我们将继续执行我们长期以来的政策，对破坏性行为采取行动，包括终止。”

不到一周后，谷歌又解雇了另外20多名员工，其中一些声称根本没有参加抗议活动。

谷歌发言人贝利·汤姆森在向《The Verge》发表的声明中表示，公司调查了“4月16日我们建筑物内的物理干扰，查看了由受到物理干扰的同事提供的额外细节”，以确定哪些工人参与了其中。

但是被解雇的软件工程师表示，人力资源部门从未联系过他，也未询问他是否真的参与了抗议活动。“他们甚至没有联系过我，”他说。“这完全让我震惊；我完全不知道这将发生。”

这名工人说，当他看抗议活动时，一名保安走近他和休息室里的其他人，要求看他们的谷歌工作证，以确保没有外部参与者。“我甚至没有意识到我不应该给他看我的工作证。他是我工作地方的保安，我没有做错任何事，”这名工人说。

超过50名因谷歌与以色列政府关系抗议而被解雇的工人[向国家劳工关系委员会提交了一项投诉](https://www.documentcloud.org/documents/24628681-google-charge?responsive=1&title=1)，这些工人声称遭到非法报复，并要求重新获得他们的工作岗位，根据No Tech For Apartheid组织发来的电子邮件声明。

“谷歌通过解雇或将他们停职，对大约50名员工采取报复行动，并干涉了他们第七条权利，这是针对他们的保护性集体行动，即参与（或被认为参与）与他们的工作条件直接和明确相关的和平、非干扰性的抗议活动，”投诉书如是说。

No Tech For Apartheid 的发言人Jane Chung [此前告诉《The Verge》](/2024/4/22/24137464/google-has-fired-another-20-employees-since-the-internal-protest-over-the-companys-israeli-defense-c)，这次解雇包括“非参与旁观者”。 谷歌对此有争议。谷歌发言人汤姆森告诉《The Verge》，所有被解雇的工人都“在我们的建筑物内个人和明确地参与了干扰活动”。

这标志着谷歌处理员工异议方式的明显改变。2018年，超过600名谷歌员工[签署了一封公开信](/2018/11/30/18118691/google-dragonfly-employee-activism-listserv-union-strike)，反对“龙飞”计划，该计划旨在为中国建立一个搜索引擎。据 *The Verge* 报道，这份请愿书起初是通过谷歌内部共享的文档发起的，随后的所有步骤也都是使用谷歌产品组织起来的。员工们还[敦促谷歌放弃Project Maven](/2018/6/1/17418406/google-maven-drone-imagery-ai-contract-expire)，即其与美国国防部的合同。同年，超过20000名谷歌员工[举行了一次抗议性的集会](/2018/11/2/18057716/google-walkout-20-thousand-employees-ceo-sundar-pichai-meeting)，抗议公司对高管性骚扰指控的处理方式。

Meredith Whittaker，一名谷歌的项目经理，曾帮助组织2018年的抗议活动，[2019年自愿离开了公司](/2019/7/16/20695964/google-protest-leader-meredith-whittaker-leaves-company)。2019年，员工们还举行了一场坐议会，抗议对那些发声的同事进行的报复行为[/2019/4/30/18524374/google-employee-sit-in-may-1-protest]。

“谷歌对员工在工作场所发声的反应方式完全改变了”，一名被解雇的软件工程师说。“与五年甚至十年前的谷歌相比，这简直是天壤之别。”
