# 在 Deepnote 工作

> 来源：[https://deepnote.com/join-us](https://deepnote.com/join-us)

我们创立 Deepnote 是为了帮助数据团队解决最困难的问题。我们不仅需要更好的算法、更大的数据集和更强大的计算能力。我们需要能帮助我们探索、协作和共享的工具。这些工具目前尚不存在。**我们需要先发明它们。**

数据工作既是科学和创造过程，也是工程过程。它涉及协作、失败、学习，并重新回到起点。[数据专业人士是探索者](https://ieeexplore.ieee.org/document/8103446)。为了使项目成功，我们需要既强大又易于使用的工具。这些工具帮助我们以引人入胜的方式协作和分享工作。工具使得与数据的工作再次充满乐趣。

这就是为什么我们正在打造数据工具的新标准：一个笔记本，集成团队编码、查询、可视化、组织和分享功能于一体。

**我们正在为探索者构建工具。加入我们。**
