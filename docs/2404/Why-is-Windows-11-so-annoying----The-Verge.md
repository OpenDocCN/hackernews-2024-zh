# 为什么Windows 11这么烦人？- The Verge

> 来源：[https://www.theverge.com/2024/4/21/24063379/windows-11-ads-bing-edge-cruft](https://www.theverge.com/2024/4/21/24063379/windows-11-ads-bing-edge-cruft)

几周前，我在我工作用的 MacBook Air 上唯一的外部显示器上放不下了。所以我切换到了我的五年老的Windows台式机，并插上了另外一个显示器。我喜欢它。我的工作效率飙升了。但这也意味着我终于花了大量时间在Windows 11上，而**天哪**，它真的很卡。

与 macOS 和 Linux 相比，Windows 在某些方面做得非常好。首先，所有的游戏都在那里，而且Windows可以在各种硬件上运行而无需大量摆弄。你不必花费至少一千美元购买一个无法升级的机器来使用它。通常情况下，你也不必下载一堆驱动程序或花六个小时在命令行中手动组装该死的操作系统。

但是，除了像“[Windows 11中的记事本终于增加了拼写检查功能](/2024/3/21/24107882/notepad-spellcheck-windows-11-feature)”这样的头条新闻外，还有像“[微软再次在Windows中向Google Chrome插入弹出广告](/2024/3/15/24101887/microsoft-bing-popups-windows-11-google-chrome)”这样的新闻。每当有[Windows子系统用于Linux](/2021/4/21/22396018/microsoft-windows-10-linux-gui-apps-task-manager-app-throttling-features)这样的东西（非常实用），就会有“[微软开始在Windows 11开始菜单中测试广告](/2024/4/12/24128640/microsoft-windows-11-start-menu-ads-app-recommendations)”这样的东西。微软似乎执意将Windows 11填满一些“功能”，这些“功能”会分散你的注意力，或者试图说服或欺骗你使用某个微软产品，而不是你本来打算使用的东西。我已经三四十岁了，我不需要这些。

我是在 Windows 3.1、NT 和 95 上长大的。我在戴尔台式机上完成了大学学业。我在 *MaximumPC* 杂志工作了五年，天哪。我组装过数十台个人电脑。我现在正在我的主力个人电脑上输入这些文字，这是我2019年精心手工组装的迷你ITX游戏设备。我一直在使用Windows操作系统。

过去几年来，我每周花费40多小时在相对安静的 macOS 上工作，下班后尽量少用电脑。因此，即使大约一年前将台式机升级到了 Windows 11，我也没有花太多时间使用它。当我确实使用我的个人电脑时，主要是用于家庭管理或者（很少）玩游戏，因此并没有与操作系统本身有太多互动。我就像是从锅里跳出去的青蛙；我又跳回去结果被烫伤了。

我就像是从锅里跳出去的青蛙；我又跳回去结果被烫伤了。

在某个时刻，[一个按钮出现在我的开始菜单旁边](/2024/2/29/24085496/microsoft-windows-11-update-copilot-widgets-snap-layouts-features)。点击它甚至是悬停在它上面，它会用我从未请求过并且不感兴趣的东西覆盖我的显示器的三分之一。大量的新闻信息。股票价格。天气情况。（天气情况很有用，但我可以在许多地方获取它。）现在在系统托盘中也有一个[按钮](/23899626/copilot-windows-11-how-to)用于[Copilot](/2024/2/21/24078972/microsoft-copilot-pro-ai-hands-on-features)，我的日常AI伙伴，现在在Microsoft产品中普遍存在，与其效用成反比。

*每次我靠近开始按钮时都会很喜欢弹出这个东西。*

自Windows 8以来，开始菜单基本上就是[垃圾](/2016/2/11/10923808/microsoft-windows-start-menu-20-years-visual-history)，但在默认状态下几乎完全没有用处。其中一半是我没有固定或者甚至没有安装的应用程序。我不怪OEM。我就是OEM，我没有把这些东西放在这里。

在过去的几个版本中，Windows 似乎忘记了如何索引我的计算机上的文件。所以如果我尝试按照通常的方式来拉出一个程序、文件或设置 — 通过按下Windows键并开始输入 — 它大多数时候会显示来自Web的结果，这些结果是无用的，因为它正在使用Bing来查找它们。

Microsoft 在支持文档方面也做了*真正*值得注意的事情。这些信息以前是内置到操作系统中的。现在，如果你在显示设置窗口（例如）中，并转到支持部分并点击“设置多个显示器”，它会打开Microsoft Edge — 即使它不是你的默认浏览器 — bing短语“*如何在Windows 11 PC上添加多个显示器 site:microsoft.com*”，并显示一个页面，只有一个结果：一个摘录了相关支持页面的信息框，还有一个链接，用于打开你刚刚离开的确切设置屏幕。

这是a) 疯狂的，b) 仍然比[上次我尝试过的时候](https://bsky.app/profile/grackle.space/post/3k3jloganbr2v)显著改进。这是微软的企业协同效应。当你可以通过单击一个按钮让必应和Edge团队也吃饭的时候，为什么要将所有这些Windows用户保留给自己呢？

Edge 曾经是Chrome的稍微改进版本。现在它塞满了侧边栏和充斥的软件。 （可以说它仍然是Chrome的改进版本。）它一直在要求我把默认搜索引擎改回Bing（我不会的），它的默认主页也是，是的，充满了垃圾。

*又是一天在光环协同地狱中度过。*

为什么世界上最大的技术公司之一会发布一个如此...不靠谱的操作系统？嗯，其中一部分肯定是因为 [在过去的30多年中，每个新版本的操作系统都是在旧版本的基础上构建的](/2024/3/25/24111288/microsoft-format-drive-windows-ui-dialog)。但这并不能真正解释为什么那些曾经运行良好的东西似乎被新系统替换为不工作的新系统，但可能有其他原因。

Windows非常成功。它赚钱。它在 [全球桌面市场中占据了超过70%的份额](https://gs.statcounter.com/os-market-share/desktop/worldwide)。Edge，仍然是 [一个相当不错的浏览器](/23523118/microsoft-edge-browser-chrome-internet-firefox-explorer)，以及搜索引擎Bing，在各自的市场份额中占有较小的份额。微软可以骚扰、 [唠叨](/2023/6/6/23736289/microsoft-bing-chrome-search-fake-ai-chatbot)，或者 [诱使用户切换到Edge](/21310611/microsoft-edge-browser-forced-update-chromium-editorial) 或 [Bing](/2024/3/15/24101887/microsoft-bing-popups-windows-11-google-chrome) 或 [Copilot](/2024/1/12/24035637/microsoft-windows-11-copilot-ai-chatbot-automatically-open-boot-startup) 的每个Windows用户对于微软来说都是一个好事情，因此在尽可能多地增加协同机会方面制作表格是有道理的。

不仅仅是Windows，显然 [每个该死的应用都想在一天内窃取你的注意力](/24119076/ios-android-notifications-focus-modes-scheduled-summary) 无数次。许多廉价手机和Windows电脑预装了由公司付给OEM厂商塞进去的广告软件和垃圾软件。对于Windows用户来说，通常习惯性地清理预装垃圾软件是一个历史悠久的传统。

但曾经，这些垃圾软件是与操作系统本身分开的。三星的安卓版本有很多垃圾软件，但那是三星的版本，不是安卓本身 —— 这就是为什么“干净版安卓”这个词汇在许多手机评论者中很流行，以及为什么像素手机在评论者中获得了比实际销量高得多的好评率的原因。

*Ars Technica* 已经撰写了一篇很好的实用指南，[关于关闭Windows 11中大部分附带内容的方法](https://arstechnica.com/gadgets/2024/02/what-i-do-to-clean-up-a-clean-install-of-windows-11-23h2-and-edge/)。而这不是我第一次这么做。我可以关闭大部分这些垃圾。大多数人永远不会费心去关掉或者不知道如何关掉，或者根本意识不到这是可选的。他们只是学会忽略它们。偶尔，他们可能会点击某些东西，然后微软的某部门就会得到一些钱。
