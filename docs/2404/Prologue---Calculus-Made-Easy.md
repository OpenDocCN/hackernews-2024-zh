# 序言 | 微积分易学网

> 来源：[https://calculusmadeeasy.org/prologue.html](https://calculusmadeeasy.org/prologue.html)

序言

考虑到有多少傻瓜可以计算，令人惊讶的是，任何其他傻瓜学会掌握相同的技巧，竟然被认为是困难或乏味的任务。

一些微积分的技巧相当简单。有些则极其困难。那些编写高等数学教科书的傻瓜们——他们大多数都是聪明的傻瓜——很少费心向你展示那些简单计算的易法。相反，他们似乎希望通过最困难的方式来给你留下深刻的印象，展示他们的极大聪明才智。

作为一个极其愚蠢的人，我曾经不得不摒弃困难，并且现在向我的同类傻瓜们呈现那些并不难的部分。彻底掌握这些，其余的就会随之而来。一个傻瓜能做到的，另一个也能。
