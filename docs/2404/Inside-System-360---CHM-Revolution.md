# System/360内幕 - CHM革命

> 来源：[https://www.computerhistory.org/revolution/mainframe-computers/7/164](https://www.computerhistory.org/revolution/mainframe-computers/7/164)

###### 软件

System/360的软件也很雄心勃勃。一种操作系统的变体，OS/360，应该在所有型号上运行。

但是OS/360很难编写 - 尤其是那些运行多个程序的复杂“多道程序设计”版本 - 并且推迟了。为最小的机器需要创建了三种特殊的操作系统。

###### 硬件

除了兼容性的基本原则外，System/360还融入了许多硬件先进技术，包括非常大的主存储器选项和固体逻辑技术（混合集成电路）。

它还使用了微程序设计，这是一种实现了一些复杂指令的软件，否则需要特殊硬件 - 这对使较小的型号完成所有任务非常关键。
