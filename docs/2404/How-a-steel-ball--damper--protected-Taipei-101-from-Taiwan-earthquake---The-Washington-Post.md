# 一颗钢球“阻尼器”如何保护台北101免受台湾地震的影响 - 华盛顿邮报

> 来源：[https://www.washingtonpost.com/world/2024/04/04/taiwan-earthquake-taipei-101-damper/](https://www.washingtonpost.com/world/2024/04/04/taiwan-earthquake-taipei-101-damper/)

当台湾发生[7.4级地震](https://www.washingtonpost.com/world/2024/04/04/taiwan-earthquake-hualien-update/?itid=lk_inline_manual_2)时，台北101内的人们得到了保护，这座岛屿最高的摩天大楼中心的大黄色摆锤帮助吸收了地震的冲击。

位于大楼顶部多层之间的 **调谐质量阻尼器（tuned mass damper）** 是一颗重达730吨的钢球，游客可以在[公众](https://www.taipei-101.com.tw/en/observatory/feature/scenery)的眼中看到它。这座观景台是首都的热门旅游景点。
