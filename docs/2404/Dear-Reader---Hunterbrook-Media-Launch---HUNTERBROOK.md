# 亲爱的读者 — Hunterbrook Media 启动 - HUNTERBROOK

> 来源：[https://hntrbrk.com/dear-reader/](https://hntrbrk.com/dear-reader/)

亲爱的读者，

Hunterbrook Media 是一个实验性质的项目。

从今天开始，我们将发布来自全球的调查和商业新闻。

在接下来的几周和几个月中，我们的调查将揭露一个支持军政府的化石燃料巨头、在以强迫劳动著称的地区扩展业务的农业巨头，以及一个摧毁生态系统的能源公司。我们相信，我们将报道的许多公司也在向客户、社区、合作伙伴、股东和全世界撒谎。

我们将定期发布全球新闻报道，首先从在蒙古、纳米比亚、巴西、越南、秘鲁、博茨瓦纳和印度的外国记者开始。

**我们从今天开始 [这里的丑闻](https://hntrbrk.com/uwm/) 开始报道。**

美国最大的抵押贷款放款人 United Wholesale Mortgage (UWM) 是基于一场谎言成为了最大的抵押贷款放款人。

在社交媒体帖子、证券交易委员会文件和一则超级碗广告中，UWM 告诉潜在的购房者，其抵押贷款来自“独立”的经纪人。UWM 曾表示，这些经纪人“[为您着想](https://www.youtube.com/watch?v=FibLrSmjjO0&t=21s)”并“[与数十家放款人竞争](https://www.youtube.com/watch?v=FibLrSmjjO0&t=21s)”以找到“最好的交易”。*(更新：在 Hunterbrook Media 发布调查报道数小时后，UWM 关于“独立抵押经纪人”的视频广告从其社交媒体频道中[消失](https://hntrbrk.com/breaking-uwm-videos-advertising-independent-mortgage-brokers-disappear-after-hunterbrook-medias-coverage/)。*

实际上，UWM 利用各种手段“培养‘忠实’经纪人”，正如公司在向投资者的[演示](https://investors.uwm.com/news-and-events/events-and-presentations/default.aspx)中所述。他们用于压榨经纪人的方法包括为那些向 UWM 提供贷款的人在他们的超级碗广告目录中获得更好的位置，以及[起诉](https://nationalmortgageprofessional.com/news/uwm-files-lawsuit-against-florida-brokerage)那些为了从特定竞争对手处获得更好交易而四处寻找的经纪人。

Hunterbrook 的数据科学家、财经记者和一名经过认证的欺诈审计师，与抵押行业专家合作，分析了公开可获得的数据，以确定 UWM 压力活动的影响 — 结果显示 UWM 在其品牌形象与现实之间存在巨大鸿沟。

近年来，UWM 从向公司几乎提供全部业务的经纪人处获得了390亿美元的抵押贷款。总体而言，Hunterbrook 估计，相比于过去四年中经纪人为其客户找到典型贷款的情况，借款人已为 UWM 支付了数亿到数十亿美元不等的闭市成本 — 在控制利率、交易类型和其他变量的情况下。

研究方法论可以在[这里找到](https://hntrbrk.com/how-we-crunched-the-numbers-behind-the-uwm-investigation/)。

诉讼律师和金融教授告诉Hunterbrook，UWM的行为可能构成欺诈，并违反了2008年后为保护借款人而通过的法律。

我们的报道团队由记者、数据科学家、律师和认证欺诈审查员组成，完全依赖公开可用的信息建立了一个庞大的数据库，详细描述了UWM的计划。您可以在[WasIRippedOff.com](http://wasirippedoff.com)搜索您的经纪人的NMLS ID，查看是否支付了过高的UWM费用，并帮助Mat Ishbia去年以创纪录的估值购买了Phoenix Suns。

Ishbia以来自J.P.摩根的44亿美元贷款抵押他的$UWMC股票，以及自2021年通过SPAC在纽约证券交易所上市以来，尽管$UWMC未达到预期，2023年报告亏损7000万美元，但他的家族每年获得超过6亿美元的股息，却成功购买了球队。

我们的报告简单明了：Hunterbrook Media的编辑委员会认为UWM可能涉嫌欺诈，如果经纪人履行其受托责任，UWM可能会失去业务并获得销售利润，还可能面临来自民事、州级和联邦责任的重大法律和监管后果。Hunterbrook Media认为，UWM与其信贷债权人的合同；与其像房利美和房地美这样的抵押贷款购买者的合同；以及CEO拥有NBA球队的股权也可能面临风险。

Hunterbrook在四天内多次寻求UWM及其高级官员的评论，包括发送关于UWM公开信息的基本问题。UWM拒绝直接评论，而是由外部法律事务所发出了一封停止与否的信。UWM的律师声称四天时间不足以进行评论。Hunterbrook的法律顾问回应称“显然足够的时间让UWM指示您和您的公司向Hunterbrook Media发送一封六页的信函，威胁采取针对受保护的第一修正案活动的战略诉讼。”

Hunterbrook Media认为变革始于意识，但并不止于此。责任要求采取行动。为此，我们已经采取了几项行动：

1）Hunterbrook Media与其投资附属公司Hunterbrook Capital分享了其经过合规审查的报告，后者对$RKT进行了多头操作，对$UWMC进行了空头操作，并购买了衍生产品。Hunterbrook Capital支付了对Hunterbrook Media独家首次访问研究的费用，结果Hunterbrook Media可以发布无广告、无付费墙和无捐款的开源内容。

Hunterbrook Media和Hunterbrook Capital都不提供投资建议。交易很难。不要在家里尝试。我们只披露Hunterbrook Capital根据Hunterbrook Media在其报道中综合公开信息所作出的长期和空头股票、期权、货币和商品交易位置。

2) 非营利性的Hunterbrook Foundation签署了一项协议，与国家诉讼公司共享数据分析和研究——并告知该公司我们将于今天发布。我们希望这为购房者提供了一个途径，以从UWM那里追回可能的损失。Hunterbrook Foundation从此类诉讼中获得的收益将用于资助当地新闻。*(更新：[路透社，2024年4月3日](https://www.reuters.com/legal/transactional/mortgage-lender-united-wholesale-sued-by-consumers-alleging-billions-excess-fees-2024-04-03/))*

3) Hunterbrook Media向具有对借款人法定义务的州总检察长及直接负责抵押贷款部门和包括UWM在内的非银行放贷机构的州监管机构发送了详细的数据分析信函。

4) Hunterbrook Media的两名调查作者向证券交易委员会（S.E.C.）提交了告密报告，由前S.E.C.委员代表。其中，马特·泰明还是Hunterbrook Foundation与国家诉讼公司达成协议的一方。

我们希望我们的行动能推动积极变革，并开创一种新的捕捉报告价值的方式，我们认为这种方式长期以来一直不是给那些产生它的组织和人员。

我们的研究由商业记者、开源分析师和前情报人员领导。我们还与经验丰富的金融编辑、独立事实核查员以及由《华尔街日报》前总编辑、ProPublica创始人、《纽约时报》首位公共编辑和出色记者组成的顾问委员会合作。我们的团队页面链接在此。

我们知道这可能不被视为传统新闻报道，一般以冷静客观、依赖内部消息来源和对盈利漠不关心而闻名。我们自豪地充满激情。我们避免与内部人士交谈，而是依赖公开可获取的信息。我们相信，良好的报道可以是良好的商业 —— 当您将见解货币化并将利润与准确性对齐时。

智者尝试过而失败寻找可持续的报道模式。我们知道这可能不会奏效。我们也相信这是值得测试的假设。无论我们是否成功，我们相信这个团队将会发布有意义的报道 — 从今天开始，我们认为这是自金融危机以来最大的抵押贷款丑闻。

我们迫不及待地希望你阅读它。
