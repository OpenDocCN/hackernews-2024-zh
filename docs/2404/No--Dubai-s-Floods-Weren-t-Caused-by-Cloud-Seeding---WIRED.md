# 不，迪拜的洪水不是由云种植引起的 | WIRED

> 来源：[https://www.wired.com/story/dubai-flooding-uae-cloud-seeding-climate-change/](https://www.wired.com/story/dubai-flooding-uae-cloud-seeding-climate-change/)

迪拜被淹没了。强烈的风暴导致阿联酋全境发生[瞬时洪水](https://www.bbc.co.uk/news/world-middle-east-68831408)，在社交媒体上传播着令人震惊的场景：汽车被弃置路边，飞机在淹没的跑道上行驶。迪拜繁忙的国际机场已取消数百个航班，邻国阿曼至少有18人死亡。

新闻报道和社交媒体帖子迅速将责任推到云种植身上。阿联酋长期以来一直有一个计划，试图从通常干旱的地区经过的云层中挤出更多的雨水——它有一支团队的飞行员在经过的风暴中喷洒盐粒子，以促使更多水汽形成。有些人将洪水定位为一则警示故事：当你试图干预自然时，结果会怎样。甚至[彭博社](https://www.bloomberg.com/news/articles/2024-04-16/dubai-grinds-to-standstill-as-cloud-seeding-worsens-flooding)也报道称，云种植加剧了洪水情况。

事实更加复杂。我过去几个月一直在为即将发表的WIRED专题报道阿联酋的云种植，虽然确实阿联酋本周进行了云种植任务——每年超过300次——但说云种植导致洪水有些言过其实。（事实上，正当我们准备在周三早上发布这篇报道时，阿联酋国家气象中心告诉CNBC，他们在周二的风暴来临之前并未对云层进行种植。）

这背后有几个原因。首先：即使对于云种植最乐观的评估也指出，它每年最多只能增加降雨量的25%。换句话说，无论如何都会下雨，如果云种植确实有影响，它只会稍微增加降水量。云种植在温暖气候中的有效性还有待评估，即使它有效果，云种植也不能在空中产生雨水，它只能增强已有天空中的降水量。

其次，种植作业往往在该国东部地区进行，远离迪拜等人口较多的地区。 这主要是由于对航空交通的限制，这意味着在暴风雨到达迪拜时，种植粒子很可能不再活跃。 我与大多数科学家交谈过，他们表示云种植的影响非常小，局部化效果，不太可能导致其他地区的洪水。 但也许最好的证据表明云种植并未参与这些洪水的事实是，整个地区都下雨了。 阿曼没有进行任何云种植，但洪水影响更为严重，造成了一些伤亡。

指责一种可怕的技术很令人兴奋，但洪水的真正原因可能更加平淡：迪拜在应对降雨方面显然设施严重不足。 过去几十年来，这座城市迅速扩展，几乎没有注意像排水沟这样的基础设施，这些基础设施可以帮助它处理突如其来的大水量。 迪拜主要是水泥和玻璃建筑，几乎没有绿地来吸收降雨。 结果就是每次下雨都会一片混乱——虽然公平地说，大多数城市都会在[12 hours](https://edition.cnn.com/2024/04/16/weather/dubai-rain-flooding-climate/index.html)内降下一年的降水量时感到头疼。

然而，气候变化可能也起到了作用。 随着地球变暖，该地区复杂的气候动态正在以可能带来更猛烈暴风雨的方式转变和变化。 世界各地的城市规划师正试图使他们的城市变得“[更具吸水性](https://www.wired.com/story/the-designer-whos-trying-to-transform-your-city-into-a-sponge/)”，以帮助处理突发洪水，并为一年中更干旱的时段保存更多水资源。 而不是利用云种植将天空变成海绵，迪拜最好是尝试将城市变成一个海绵。
