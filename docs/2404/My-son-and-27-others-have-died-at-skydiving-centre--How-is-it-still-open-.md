# 我的儿子和其他27人在跳伞中心丧生。它怎么还开着？

> 来源：[https://www.thetimes.co.uk/article/skydiving-death-center-california-cm26bkd9k](https://www.thetimes.co.uk/article/skydiving-death-center-california-cm26bkd9k)

在加利福尼亚州一架飞机上准备进行跳伞之前，泰勒·特纳拥抱他的母亲并告诉她：“我爱你。”

弗朗辛·特纳听到她18岁儿子的最后一句话是在2016年8月，当时他与一名教练共用的降落伞未能打开时不久去世。

自1985年以来，洛迪跳伞中心已发生28起死亡事件，尽管特纳说她不知道业务的死亡记录，当她儿子的飞机起飞时。

这位60岁的妇女与丈夫托德住在加利福尼亚州圣华金谷，正在竞选更严格的跳伞监管政策，并且无法理解为什么这个中心仍然开放。

“我们假设他们会关闭它，”她告诉
