# Cogram的客户成功工程师 | Y Combinator

> 来源：[https://www.ycombinator.com/companies/cogram/jobs/tB0R5rt-customer-success-engineer](https://www.ycombinator.com/companies/cogram/jobs/tB0R5rt-customer-success-engineer)

地点：远程或德国柏林；时区：灵活

## 关于我们

Cogram正在建立一个能够通过将人工智能语言模型深度整合到团队沟通中来自动化工作的生产力平台。我们的软件支持各种规模的客户，从小型研究实验室、建筑师或非营利组织，到拥有超过10,000名员工的全球咨询公司和《财富》500强公司。

我们得到了Y Combinator和其他世界领先的投资者的支持，并且正在可持续且非常快速地发展。我们相信，一个小而成功的团队可以取得非常大的成绩——尤其是在人工智能时代——并且不断打败比我们大100倍的竞争对手。

我们非常相信远程优先的文化和多元化团队，因此我们鼓励任何人都可以申请——即使您的经验并不符合所有要求。在Cogram，您有前所未有的机会来塑造人与人工智能协作的未来。如果这听起来很有趣，我们很乐意听听您的想法。

## 工作描述

作为Cogram的客户成功工程师，您将帮助确保我们的客户有效地使用我们的产品。这个角色涉及直接与客户沟通以解决技术问题，监控我们系统的运行状况，并直接解决问题或与我们的工程团队协调解决。您将使用我们的云架构来诊断和解决问题，促进系统的高可靠性和客户满意度。

理想的候选人将具有在初创公司建立和扩展客户成功功能的可靠记录。

## 主要职责

+   处理客户支持请求，提供技术支持，并保持良好的客户关系。

+   使用可观察性和报警工具来监控和增强我们的系统性能。

+   在可行时诊断和解决我们代码库中的问题，并在需要时有效打包复杂问题以提供给工程团队。

+   将客户反馈传达给工程团队，帮助改进系统。

+   编写详细的文档和指南，以协助客户进行系统集成和故障排除。

## 资格

+   具有软件开发和系统架构经验的扎实技术背景，特别是在AWS、Grafana、Python、TypeScript和Postgres方面。

+   具有在初创公司建立和扩展客户成功功能的可靠记录。

+   出色的SQL技能，特别是Postgres。

+   出色的解决问题能力，能够在动态和多层面的环境中蓬勃发展。

+   出色的沟通能力。

## 奖励福利

+   完全远程工作环境，每年会在世界各地的不同地方举办三次团队活动

+   具有竞争力的薪酬和慷慨的股票期权计划

+   每年38天带薪休假（26天个人休假，12天德国公共假日）

+   个人发展津贴，帮助你成长

+   根据地点提供可选健康保险

在Cogram，我们寻找具有好奇心、雄心和同情心的人。即使您的经验不完全符合工作描述，我们也鼓励您申请。您的技能和激情将脱颖而出，并使您脱颖而出，尤其是如果您的职业经历曾有过一些非同寻常的转折，或者您曾经克服了个人挑战。我们欢迎多元化的观点和思考严谨、不怕挑战假设的人。加入我们吧。
