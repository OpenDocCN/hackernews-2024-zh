# 前雅虎CEO玛丽莎·梅耶尔谈她的新照片应用，通过人工智能增强 | Semafor

> 来源：[https://www.semafor.com/article/04/05/2024/ex-yahoo-ceo-marissa-mayer-on-her-new-photo-app-souped-up-by-ai](https://www.semafor.com/article/04/05/2024/ex-yahoo-ceo-marissa-mayer-on-her-new-photo-app-souped-up-by-ai)

**Q: 你先从一个联系人应用开始，现在又有了照片应用。你能解释一下策略吗？**

**A:** 照片和联系人息息相关。当你仔细想一想，这是显而易见的，你和最亲近的联系人会花时间在一起，如果你在这个时代和他们一起度过时间，就会有和他们的照片。因此，如果你能看他们的相机胶卷，你就能理解人们的关系。现在有了人工智能的帮助，我们可以对某人的相机胶卷进行精确分析，并大致了解你和谁在什么时候、地点和原因下一起度过时间。

当涉及到保持人与人之间的联系时，这真的非常有用，帮助将那些平淡无奇的日常接触变得神奇，并帮助人们真正感受到彼此的联系。联系人在其中扮演了重要角色。这是基础，也是我们为什么从那里开始的原因。但是照片可能是我们所有人与他人互动中最有意义的方式之一，这是我们建立关系的方式。

**Q: 这对我有共鸣，因为我从未喜欢社交媒体，但我确实想和认识的人分享。您是否看到消费者在后Web 2.0时代这些日子里的品味发生了变化？**

**A:** 是的，也许吧。我也认为人工智能及我们对其期望会影响它。一个思考的方式是，如果Instagram是你编辑、滤镜过的完美照片，你可以推送给所有你认识和不认识的人，而点对点的照片分享则是我拍摄一张照片然后发短信或者直接空投给你，Shine试图在这两者之间找到一席之地。‘好吧，我刚刚和10、20、50个人在一起过。

那天晚上很有趣，但我并不想打扰自己，把自己从当下抽离出来拍照。但我知道我拍了一些照片，他们也拍了一些，让我们把这次旅行、派对、会议、活动的整个感觉汇聚在一起，形成一个大的照片池。这正是我们试图占据的空间。

在这个领域没有太多新的变化。我称之为智能、小规模的分享。早期社交网络和社交媒体试图填补这个空白。但是随后人们加入后，他们发现在那里他们想要分享的人并不多。因此，他们开始寻找谁是有名的，这就是为什么突然之间你开始看到这些名人粉丝计数达到百万级别。

但与此同时，名人固然有趣，但你的朋友、家人和亲密联系人也同样有趣，他们的生活中也有他们想要与你分享的事情。这真的是我们希望参与的空间。我认为 Snapchat 以一种不同的方式展示了这一点 — 你可以以一种不那么完美、稍微临时的方式分享。这种方式真的非常强大。

再加上作为工程师和产品人员，我们寻找日常生活中感觉破碎的东西。还有一点是，你会像，“我知道这张照片被拍了，但我没有它，或者我的朋友忘记把它发给我。”

或者当你在活动中看到有人拍摄第一次舞蹈、孩子吹熄生日蜡烛的场景时，你会有一种感觉，觉得需要拿出手机拍同样的照片。因为你会想，“好吧，我要确保我能拍到它，我只能亲自拍摄。”对我来说，这些工具可以让我们更加身临其境。同时，我们不希望只是简单地移动很多照片，因为有很多重复的照片并没有组织好。

**Q：这就是你们如何使用AI的方式？你们正在挑选最好的？**

**A:** 我谈到了AI如何改变人们的期望。这是其中的一个重要部分。因为现在我们可以将照片分享变得更加有条理，让你得到的实际上是一个有组织的相册，里面都是人们想要分享的，你想看到的东西。而且，如果他们拍了八张照片，他们不确定哪一张眼睛都睁开了，不确定哪一张没有模糊，AI可以做到所有这些。

**Q：还有其他你们如何使用AI的方式吗？**

**A:** 所以我们正在为相册做AI驱动的策划，以理解什么是值得分享的。当你在活动中时，有些东西可能不合逻辑地出现在相册中，而有些东西是合适的。我们在那里使用AI，并且还用AI做近似重复项的分组。我们还在我们的“为您”选项卡中的建议的相册中使用AI。

建议的相册是该应用程序中最强大的功能之一，因为这并不意味着你现在一定要有一个“闪亮相册”。你实际上可以在过去创建“闪亮相册”。我们已经建立了AI算法，可以回顾你相机胶卷的元数据。你拍照时在哪里？什么时候拍的？基本上对它进行分析，并试图理解在你拍摄那些你可能不想分享的无聊、平凡照片的何时何地，以及在你拍摄那些你确实想分享的照片时，你拍摄了什么？所以我们在那里也运用AI。

然后我们注意到的一件事是，这是你在活动中想要做的智能、小规模分享。从邀请开始，你可以自然地引导。所以我们有我们的Shine网站来组织活动。好处是在这种情况下，你认识这群人，你知道他们都会在那里。如果你能够使照片分享连续进行，那就太好了。

如果你已经有了所有的RSVP，这都是Shine的一部分。你自然地加入相册，并且可以享受活动，自然地参与照片分享，接收照片分享。我们的邀请函使用一种不同形式的AI，即生成式AI。我们有基于AI提示的非常惊艳、聪明和定制的邀请函。

**Q: 那些很棒。**

**A:** 其中一些让我们叹为观止。有些让我们大笑出声。它们真的很有趣，为人们举办的活动增添了很多个性。

**Q: 你是一个技术创始人，曾经带领过编程团队。你看到AI工具正在改变软件开发的方式吗？**

**A:** 其中一个最直接的影响在于测试。AI软件显然已经在那里。很多工具在测试工具方面做得非常出色。你总是可以做得更多，你总是可以找到更多关于测试的东西，但我认为有一些编码和把产品引入市场的过程中的元素很快就会由AI主导。一些最终阶段的过程，比如测试和QA，显然会因为AI工具而更好地运行和更有效率地运行。

这并不意味着那些工作完全消失了，而是你不再需要亲自编写测试工具的框架，你可以让工具来为你做大部分的开发工作，这有助于你快速覆盖更多的情况和更多的地面。

**Q: 作为一个从符号系统开始的人，你现在是否对这些模型的架构感到兴奋？**

**A:** 在所有这些中，我发现最令我惊讶和愉悦的是符号系统。斯坦福的符号系统是一个独特的专业。它基本上是具有非常重的计算机科学倾向的认知心理学。但是专业的实际组成是认知心理学：人们如何学习？哲学：他们如何推理？语言学：他们如何表达自己？以及计算机科学：你能创造一个能学习、推理和表达自己的计算机吗？

我在90年代中期对此很着迷。我在斯坦福的两个学位都是在AI专业或AI专业化方向上取得的，分别是我的学士学位和硕士学位。我觉得那非常有趣，有所有这些符号系统。我们看到这样的线索，学习，推理和表达自己。

我从编写第一行代码的那一刻起就热爱计算机科学。我对语言学知之甚少。我明白表达自己对于智能来说很重要，但我的关注点，我也会说整个行业的关注点，在过去的十年或二十年里，更多地集中在学习上。构建模型，强化学习，所有进入现代模型构建的不同技术，然后部署这些模型，给它们新的事物来推理和看它们的表现。这就是所有的精力所在。

而对于我来说，OpenAI 和 ChatGPT 最大的惊喜之一是表达的部分，语言学的部分，在那个符号系统循环中可能被低估了，它捕捉了每个人的想象力，它能够以一种你能理解并且感觉非常智能的方式进行交流和表达。Dall-E 能够绘制那些艰难复杂的画作，一个艺术家可能需要几天甚至几周的时间来表达，这确实激发了想象力。

所以很有趣，因为很长一段时间，我不理解并且在某种程度上忽视了那种表达的部分。而在这个时刻，正是这种表达的部分成为了AI的巨大突破口，并且真正帮助每个人看到它的潜力。

**Q: 当你在 Google 时，所有正在发生的深度学习都是由计算能力驱动的。测试那些在学术界已经开发的理论的能力。现在你有数据，也有计算资源来做到这一点。但是如果你看看现在发生的事情，它完全超过了当时的情况。这让你感到惊讶吗？这些资源用于开发这些技术有多庞大？**

**A:** 当然，我认为他们没有打折扣的一个因素就是数据，对吧？Google 能做到这一点的原因之一就是庞大的数据量，以及计算能力。有时候，倒推连接点比顺向连接它们更容易。我认为当你看一下从拼写检查到谷歌翻译的所有事情时，这种利用大量数据，大量计算来预测接下来会发生什么以及如何构建事物的理念，虽然不是现代的LLM，但基本上是用了大量数据，大量计算来进行预测。

因此，我认为他们今天所做的事情确实令人惊讶。但是当你回顾这些基础知识和谷歌的工作时，我们一直在拼写上使用它，后来我们在不同语言之间使用它。如果你在语言内稍微不同地应用这些相同的技术，你也可以获得令人瞩目的结果。

**Q: 如果你戴上大公司的CEO帽子，你会花数百亿美元来扩展变压器模型，还是用ChatGPT所带来的灵感寻找下一个突破和架构？**

**答：** 这些基础，那些基础模型非常重要。同时，建设它们也是极其昂贵的。因此，你要么必须知道你有自然应用程序将会依赖于那些LLM和你建立的模型，这些应用程序对人们非常有用，因此非常有利可图。要么你必须创建API，让其他人在那些具有同样丰厚利润性质的模型上构建应用程序。

这就是为什么我依然对Google非常看好的原因，因为他们有数据，他们在搜索、电子邮件、地图和文档等许多不同领域都有自然的应用程序。对于他们来说，在这些基础模型上进行投资绝对是有意义的。他们可以在其上构建应用程序和功能，也可以以各种格式和API提供它。

我认为对于一个中小型公司来说，说不，我不会建立一个基础模型是有道理的。我将依赖科技巨头去做这件事，然后我会挑选适合我领域的他们的模型。

**问：所以你不会开发自己的基础模型？**

**答：** 作为一个小型创业公司，这可能并不合理。我也觉得很有能力，我们可以在许多不同的模型中进行选择。即使是我们在这里讨论过的案例，从背后运行建议相册的AI，到精心策划的相册再到驱动我们邀请的生成式AI，我们有许多不同的模型，我们可以去挑选并按需购买它们。

**问：所以你拿这些模型然后进行微调？**

**答：** 没错。

**问：现在作为一个初创公司，正值这项技术变得如此强大和潜在颠覆性的时机，是个好时机吗？**

**答：** 现在是技术领域的好时机，因为大公司正在做有趣的事情，小公司也是如此。在如何应用这项技术、如何思考和使用它方面，整个新的未开发领域正在被开辟。

**问：但你仍然处于苹果这些围墙园区内，也许在较小程度上，Google也是如此。这些限制会对你的能力造成限制。随着最近的反垄断诉讼，你希望打破哪些围墙？**

**答：** 位置、地图，所有地理相关的事物都是我关心的问题，因为我在Google时长时间从事地图工作。我认为位置非常强大：我们去哪里，我们如何度过我们的时间。它确实会影响你的照片，影响在你联系人中的人，或者它应该影响。位置可能是其中的一个重要部分。但我认为Android和Apple这两个平台在粒度、它们真正允许你访问的内容以及使用的便利性方面有所不同。

他们在用户方面有不同的哲学观念，以及他们想要做的授权类型。但如果这两者能够更加标准化，并真正允许开发者访问与平台拥有的相同类型的信息，那将是非常好的。

**Q: 苹果在这里的立场是‘我们在保护我们的用户隐私。’你对此有何看法？**

**A:** 我认为用户的隐私以及由此建立的信任是非常重要的。这对于苹果来说很重要，对于初创公司也是如此。我们知道在我们所做的工作中尤其如此。你的联系人、你的照片，是你手机上最私密的东西之一。因此，我们需要非常尊重隐私。

与此同时，我们已经看到了大量的历史性例子，甚至是现代日常例子，即使是通过一个值得信赖的来源，人们也愿意提供对他们的私人信息的视图，如果这对他们非常有用的话。经典的例子是你的驾驶执照。你为什么要告诉州或政府你住在哪里，你有多重，有多高，你的头发颜色，眼睛颜色？结果证明开车确实非常有用。因此，人们做出这些种类的权衡。

我的浏览器记录了我的历史和我到过的每个地方，这些都是关于我的大量信息，关于我感兴趣的事物，我正在学习的事物。这些都是非常个人的。但后来要是想问‘我当时在哪儿？我在哪儿看到的？我能回到我的历史记录吗？’

如果分享数据帮助我更轻松地获取照片并更频繁地分享照片，减少遗忘，那么我认为很多人会说‘当然’。这是个人的，但就你在哪儿这一点来说，并不是那么个人化。显然，你的手机运营商也有同样的信息，知道你的手机在不同时间在哪里。所以还有其他人可以访问相同的信息，这对你有利，也对你无利。

这是一个明显只会使你受益的案例。因此，总体来看，我认为有些用户想要做出这样的选择。他们不应受到误导。这确实违反了信任。但如果他们想要进行这种权衡，我认为他们应该能够比现在更容易地这样做。

**Q: 在谷歌和其他公司那里，这种折衷意味着你的数据将被货币化。你们提供的是免费模型。那么这是否意味着你们可以选择不将其货币化？我认为你们目前没有进行任何广告。**

**A:** 我们的联系人模型是免费的，但你可以付费让我们将其导出并成为你的 iOS 联系人。因此，如果你想在 iOS 上升级你的联系人，我们将采用我们创建的增强型联系人，进行去重和增强。对于我们的照片应用Shine和我们的事件应用，目前它们都是免费的。在我们公司成立时制定的隐私承诺中，我们不出售你的信息。

我们希望你了解我们拥有什么信息，以及它如何被使用，它应该始终对你有益作为你的数据。你可以随时带走它。当我们看商业模式时，我学会了永不说永远，但我们的倾向和迄今为止的行动更多地是直接面向消费者，并提供人们一次性或持续订阅的服务，最终实现这些服务的货币化。因此，在混合中没有第三方，因为特别是对于某人非常个人化的信息，你不希望第三方参与其中。

**Q: 看起来互联网的商业模式也在有所转变，人们愿意以一种1.0和2.0中不曾有过的方式为这项技术付费。你觉得这是真的吗？**

**A:** 我将给你一个类比和一个故事。当我在Google的时候，我们对个性化搜索非常感兴趣。当你搜索引擎时，我们知道你对汽车感兴趣，而其他人可能对飞机或不同的火车引擎感兴趣，或者其他什么。所以我们想，我们怎样才能更加个性化地为人们提供服务？

所以我们进行了一次收购，那就是一个叫做Kaltix的公司。他们进来了，我们看了各种各样可以用来个性化你的搜索的东西，比如你住在哪里，你是什么专家，就我们能力所及的范围来看。我们看了所有这些不同的因素，然后看了我们整体的搜索幸福感和搜索结果的满意度。你认为哪个表现最好？如果你只能个性化某人的搜索一件事，什么是最有用的？

**Q: 我不知道。告诉我吧。**

**A:** 你刚才进行的搜索。如果我输入"carburetor"，然后输入"engine"，我可能正在寻找汽车引擎，对吧？或者我在寻找与化油器有关的东西。这些类型的事情。所以这很有趣，因为比你住在哪里，比你是什么专家，比你知道什么，更重要的是，如果你知道他们刚才做的搜索，你可以得到最好的结果。

当你开始查看一些订阅和注册服务时，ChatGPT几乎是相同的原理。如果他们知道你刚刚在做的AI任务，他们将能够在你所在的位置与你会面，并帮助你更快地推动它。很有趣的是，即使在我们的邀请中，我们看到人们在做提示时有这种倾向，因为我们正在学习如何帮助他们更好地为他们的邀请做更好的提示。他们甚至期望，比如做一个带更多红色的那个，这种在他们已经处于的位置上建立的迭代概念。

这是人们对这些AI技术和聊天机器人期望的一部分。他们希望能够继续上次的对话或任务。这也是为什么拥有一个账户在世界上所有的意义上都是有道理的，因为你可以在有限的历史记录或根本没有历史记录的情况下表现出色，就像Google在拥有个性化搜索之前多年一样。但是一旦你拥有了历史记录和那种了解某人的长期性，它们就会变得更好。

当今的AI模型与个人信息相结合时，AI将迎来另一个重要的解锁时刻。例如，如果你问ChatGPT：“下个月我该给侄子买什么生日礼物？”它们会问：“他是谁？他住在哪里？他多大了？”

但我认为，一旦你真正开始了解人们的个人信息、个人关系、个人偏好，并将其应用于语言模型或强大的图像模型上，那时将会迎来另一个重大进步，人们每天都会感受到这种解锁的巨大效果。

**Q: 你认为我们是否会达到这样的时刻，即有一个模型能够真正推理，并因此作为我们的代理人并且可靠？**

**A:** 目前的模型在某种程度上有一定的推理能力，但在哲学意义上相当有限，它们的表现已经足够好，以至于我不确定是否会有太大的压力要进行那种类型的推理。随着时间的推移，这是一个非常有趣的智力领域，我确实认为我们将在那里看到研究的进展。但是现在，凭借当前的技术和推理风格，这种因果推理模型可能并不是真正需要的，也不会受到强烈推动。

**Q: 你是否进行了大量的天使投资，如果是的话，最近在初创公司领域有什么令你印象深刻的事情吗？**

**A:** 我确实进行了一些天使投资，但数量相当有限。显然，我现在的大部分时间和资源投入到了Sunshine这些天。几年前，我投资了一家名为Halo AI的公司，他们基本上在进行AI分析，包括放射学图像，最终在癌症等疾病和各种病况的早期检测方面做得非常出色。在我成长的过程中，我长期想成为一名医生。这些医疗进步和医疗应用总是让我感到非常兴奋。

**Q: 确实看起来那个领域非常有前景，药物发现也是。你享受创业生活吗，还是公司CEO的生活更有趣？**

**A：**我喜欢公司的各个阶段。很多年前，我和我妈妈有过一个有趣的对话，她在谈论我的一个朋友刚说的一句话，‘天啊，4岁是我最喜欢的年龄。’ 然后，她说，‘我为她感到遗憾，因为总有一天，她会有一个5岁的孩子。’ 我妈妈说，‘我喜欢我孩子的每个阶段。’ 那是在我还没有孩子，没有结婚之前，我对我妈妈说，‘你知道吗，妈妈，我想我喜欢公司的每个阶段。’ 我说，‘我喜欢公司小的时候，当它只有个位数，双位数，三位数，四位数。我喜欢当我只是一个个人贡献者，我喜欢管理小团队，大团队，以及拥有成千上万个成员的团队。我喜欢高速增长，我甚至喜欢挑战，比如扭转衰退中的公司或产品线的局面。这是你可以致力解决的最富有成就感、最有趣的智力和设计难题之一。我就是喜欢打造东西。这才是公司的真正意义。它关乎打造产品、业务、团队和文化。这个工作以及所有这些的结合方式真的非常迷人。

**Q：你说过你要成为一名医生。从某种意义上说，人体是一个了不起的系统。公司也类似于此吗？**

**A：**也许这是基本的工程思维。我喜欢系统。我喜欢象征性的系统以及它们如何支持现在的人工智能。那就是一个系统，认知系统，身体系统，还有公司系统。我喜欢思考非常复杂的系统，以及如何使它们运行得更好。而产品也是这样，无论是搜索引擎还是像Shine这样的照片分享应用。它们都是复杂而有趣的系统，总体上可以变得更好，更有效，产生更大的影响。
