# 你是你所读的，即使你并不总能记得 - 吉姆·尼尔森的博客

> 来源：[https://blog.jim-nielsen.com/2024/you-are-what-you-read/](https://blog.jim-nielsen.com/2024/you-are-what-you-read/)

这里是戴夫·鲁珀特（[来自我的笔记](https://notes.jim-nielsen.com/#2024-03-22T1029)）：

> 一本书的目标不是达到最后一页，而是拓展你的思维。

我必须不断提醒自己这一点。尤其是在一个重视优化和最大化个人生产力的环境中，似乎如果你不能衡量（更不用说记住）一本书在你生活中的影响，那么它就不值得阅读。

我不相信这一点，但我从来没有找到表达我不相信的原因的话语。戴夫的表达非常贴切。

几天后，我妻子给我发来了拉尔夫·沃尔多·爱默生的这句话：

> 我不能记得我读过的书，就像我吃过的饭菜一样；尽管如此，它们也塑造了我。

是的！

哎呀，优秀的作家在运用文字方面真是妙笔生花，对吧？

爱默生用极其简洁的方式表达了我自己的想法，甚至我自己都无法理解，更别说用言语表达出来了。这让我感到嫉妒。

无论如何，我想把这个写下来以加深记忆。

在在线世界中同样如此：我无法记住我读过的博客文章，就像我吃过的饭菜一样；尽管如此，它们塑造了我。

这是一个很好的提醒，要注意我的内容消费——你就是你所~~吃~~读的，即使你并不总能记得。
