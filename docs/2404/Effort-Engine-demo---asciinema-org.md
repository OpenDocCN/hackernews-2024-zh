# Effort Engine 演示 - asciinema.org

> 来源：[https://asciinema.org/a/piP22yYwcaohu5cA2gyuv1W61](https://asciinema.org/a/piP22yYwcaohu5cA2gyuv1W61)

虽然此网站目前不支持 GIF 转换，但您仍然可以使用 asciinema GIF 生成实用程序自行完成 - [agg](https://docs.asciinema.org/manual/agg/)。

安装后，使用以下命令生成 GIF 图像：

```
agg https://asciinema.org/a/654503 demo.gif
```

或者，如果您已经下载了录制文件：

```
agg demo.cast demo.gif
```

查看 `agg --help` 以获取所有可用选项。您可以更改字体系列和大小，选择颜色主题，调整速度等。

查看 [agg 手册](https://docs.asciinema.org/manual/agg/usage/) 获取完整的使用说明。
