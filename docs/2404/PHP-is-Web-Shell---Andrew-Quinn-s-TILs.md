# PHP 是 Web Shell | 安德鲁·奎因的 TILs

> 来源：[https://hiandrewquinn.github.io/til-site/posts/php-is-web-shell/](https://hiandrewquinn.github.io/til-site/posts/php-is-web-shell/)

在一个由像我现在这样的许多铁杆 Linux 黑客创立和运营的公司工作的更酷的事情之一是，这里有*大量*的 Bash 脚本，积累了大约 25 年左右。尽管有它们的缺点，纯粹的 shell 解决方案仍然是几乎完全不受比特腐败影响的程序的银标准。

但你知道吗？*纯粹的 PHP 也是如此。* 那种从头开始为了尽快制作工作动态网页而设计的粉红色东西。

PHP 是 Web Shell。现在我确信了。

它很糟糕。你认识的每个人都已经转向了更大更好的东西。而你不认识的那些人，即互联网上的其他 80%，仍在使用它来廉价、可靠地完成工作，虽然没有张扬，但确实伴随着大量的服务合同，因为看看本段开头。
