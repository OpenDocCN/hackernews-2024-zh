# 丹尼尔·丹尼特（1942-2024）- Daily Nous

> 来源：[https://dailynous.com/2024/04/19/daniel-dennett-death-1942-2024/](https://dailynous.com/2024/04/19/daniel-dennett-death-1942-2024/)

哲学刚刚失去了其中一个最有影响力和英雄般的声音。丹尼尔·丹尼特是一种日益稀有的多才多艺的人才的成员，而社交媒体、人工智能、宗教教条和限制性政府政策的侵入则使我们人类变得越来越无思想、令人利润和可税收。

他是一个极度罕见的天才，是一个被神经科学家认为是他们中的一员的哲学家。也许，这样的人每隔几个世纪才出现一次，而我们并没有意识到自己有幸与他共生活在一起。如今，丹尼尔·C·丹尼特正加入哲学史上最伟大思想家的长队，他们永远铭刻在我们的思想和记忆中。如果有神存在，那么对于亵渎地在他的X（Twitter）资料上写道他现在任由神的处置的有神论者而言，这不是个好兆头。相反，事情恰恰相反：那位神祇如果知道自己的利益，将不得不小心应付。天堂已经换了新管理，上帝因为两千年来的那些粗心大意的工作而被降职了。我想象他卷起袖子，怒气冲冲地闯入上帝的办公室，说：“让开，蠢货！要解开你自古以来搞得一团糟的乱子，可得需要一个天才！”

作为在哲学界傲视群雄的巨人，无论是古代还是现代，丹尼特的传奇古怪、卓越才智和无与伦比的幽默都让一门学科难以招架。一个巨大的空白突然出现，就像一个超大质量黑洞的中心在银河系中心涌现出来，具有不可满足的饥渴，要吞噬一切。我知道，如果他在这里，他会告诉我们鼓起勇气继续他开始的事业。哲学中有很多工作要做，而丹尼特已经尽其所能，与宗教的教条、偏见和残酷斗争，以及各种形式的无知，以其各种外观伪装成“理论”。不要让实用主义消亡。

通过我们对他的记忆和遗产的集会，并完成他所开始的事业，是最好的纪念我们逝去的冠军的方式。如果存在有神论与无神论的辩论，请提到他的名字。如果有关自由意志或意识的一些胡言乱语，请提到他的名字；淡化无拘无束的自由意志的概念；将意识减少为一系列的模因！在你的文章和书籍中引用他的名字，并继续引用他。让我们建立在他的理论基础上，并使工具主义成为主导观点，不仅在哲学上而且在整个社会中。无论别人怎么说，丹尼特并没有“死去”；他只是以我们的集体思想形式继续存在。他现在比以往任何时候都更加伟大；信奉神的人永远无法摆脱他。保持他活着的最佳方式是建立在他的遗产上。

今晚，我向一个传奇人物敬酒，他的成就需要太多的篇幅才能在这里列举和解释。

丹尼尔·丹尼特（1942–）和工具主义万岁！
