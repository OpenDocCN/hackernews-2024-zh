# OWASP因Wiki Web服务器配置错误披露了此次数据泄露 • The Register

> 来源：[https://www.theregister.com/2024/04/02/owasp_discloses_data_breach/](https://www.theregister.com/2024/04/02/owasp_discloses_data_breach/)

一个配置错误的MediaWiki Web服务器允许数字窥探者访问Open Web应用安全项目（OWASP）基金会成员的简历，这些简历包含其个人详细信息。

根据致力于提高Web应用程序安全性的非营利组织，该组织在接收到“少量”报告请求后，于2月底意识到了配置错误和随后的数据泄露。

"如果您曾经在2006年至2014年期间是OWASP的成员，并作为加入OWASP的一部分提供了您的简历，我们建议您假设您的简历已包含在此次泄露中，" OWASP在其网站上发布的一份[通知](https://owasp.org/blog/2024/03/29/OWASP-data-breach-notification)中这样说。

"我们意识到这次数据泄露的重要性，特别是考虑到OWASP基金会对网络安全的强调。" 它补充道。

简历中包含姓名、电子邮件地址、电话号码、物理地址，“以及其他可识别个人信息”，据称包括人们的就业地点。

尽管好消息是，在大多数情况下，这些简历至少有十年的历史，但这仍然涉及大量个人的详细信息 — OWASP在全球超过250个章节中拥有“成千上万的成员”。

据推测，至少一些人仍然使用2014年的相同标识符，现在这些信息可能被用于身份欺诈和其他不正当目的，如果这些信息落入错误的手中或在暗网数据库中。

如果您使用相同的电子邮件地址或电话号码，例如，OWASP建议在回答或接收未经请求的电子邮件和电话时要保持谨慎。这些可能导致钓鱼尝试和金融犯罪。

据开源社区称，它不再作为会员申请过程的一部分收集简历，现在使用两步验证来保护会员数据。

为了确保不再发生类似事件，OWASP表示禁用了目录浏览，并检查了Web服务器是否存在其他配置和安全问题，并从网站中删除了所有简历。

此外，基金会清除了CloudFlare缓存，并要求从Web存档中删除访问的数据。

尽管OWASP正尝试通过电子邮件通知受影响的个人，但简历的年龄在10到18年之间，使其变得更加困难。

"不过，我们将联系在我们的调查中发现的电子邮件地址，" 它承诺道。 ®
