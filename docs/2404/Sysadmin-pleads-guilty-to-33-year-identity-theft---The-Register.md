# 系统管理员承认了长达33年的身份盗窃罪名 • The Register

> 来源：[https://www.theregister.com/2024/04/03/sysadmin_33_year_id_theft/](https://www.theregister.com/2024/04/03/sysadmin_33_year_id_theft/)

一位爱荷华州系统管理员因涉嫌在长达33年的时间内窃取并冒用前同事的身份而认罪。

根据法院文件，58岁的马修·大卫·凯兰斯还恶意地与当局合作，提供了伪造文件，最终导致他的受害者被监禁，并在精神病院里呆了将近一年半时间。

凯兰斯第一次遇到他的受害者威廉·唐纳德·伍兹，后者在16岁时离家出走，在“大约1988年”（这是法院文件能提供的具体时间）在新墨西哥州阿尔伯克基市一家热狗摊位工作时。

大约两年后的1990年，据说凯兰斯“在生活的各个方面”都冒用了伍兹的身份，购买了各种伪造文件，包括肯塔基州出生证明、美国社会安全号码和I-9表格，后来他用这些文件在2013年作为远程系统管理员在爱荷华州的一家医院找到了工作。

经过医院的背景检查后，他成为了爱荷华大学医院与诊所IT团队的核心成员，并在维护医院系统和基础设施方面扮演了关键角色，根据凯兰斯的[认罪协议](https://regmedia.co.uk/2024/04/03/usa_v_keirans_plea_memorandum.pdf) [PDF]。

在他的十年工作期间，他从工作中赚取了超过70万美元，并在被当局逮捕时已经挣得超过10万美元。

法庭文件没有详细描述真正的伍兹在1980年代末到2010年代间发生了什么，但我们知道凯兰斯在1990年获得了一张假科罗拉多州身份证后，在一家未透露名称的快餐连锁店找到了工作。他还使用伍兹的名字和新发现的身份证在科罗拉多州开设了一个银行账户。

一年后，凯兰斯用两张价值300美元的支票购买了一辆价值600美元的汽车，后来支票被拒付，他开了一小段时间，直到车抛锚，然后就抛弃了。之后，他去了一个ATM取出他剩下的所有钱，然后在科罗拉多州对威廉·伍兹的逮捕令发布之前逃离了。

大约在1994年，凯兰斯仍然使用伍兹的名字，娶了一位女士，并且生了一个孩子，孩子也姓伍兹。

在2016年8月至2022年5月期间，法院文件透露，凯兰斯以伍兹的名义从信用社中取得了八笔不同的贷款，总额超过20万美元。

凯兰斯显然非常喜欢吉普车，因此他用伍兹的名义购买了三辆吉普车 — 2016年的切诺基，2019年的切诺基，最后是2018年的大切诺基。

这名欺诈者还被批准申请了一张信用额度为11000美元的Visa Platinum信用卡，除此之外还有许多其他个人贷款，其中一些以他的吉普车作为抵押。

### 残酷和无能同在

如果還不夠糟糕的話，在2019年，當時無家可歸的伍茲發現以他的名義產生了巨額債務，當時數額約為13萬美元，他去洛杉磯當地的銀行解決問題。

試圖解釋信用額度不是由真正的伍茲（Woods）提取，他要求關閉自己的帳戶，並表示拒絕支付債務。

鑑於帳戶中的大筆款項，真正的伍茲被要求回答一系列安全問題，當然他答錯了，因為這些問題並非他創建的。這促使處理伍茲的銀行出納員通知洛杉磯警察局，而基蘭斯在向當地警察傳真了一系列偽造文件後，將伍茲逮捕，以“證明”他是真正的伍茲。

這一切發生在真正的伍茲出示真實的社會安全卡和加利福尼亞州身份證的情況下。最後，伍茲被逮捕時不是用他自己的名字，而是用基蘭斯的名字。

最糟糕的是，伍茲在總計被羈押428天後，一名加利福尼亞州法官認定他精神不適合接受審判，而伍茲對兩項重罪指控不服罪名，因此被送往一家精神病院共147天並且接受精神藥物治療。從那時起，法官還命令伍茲只能使用他的“真實姓名”馬修·基蘭斯。

在整個這場磨難中，基蘭斯通過在家工作的高收入系統管理員職位從威斯康星州遠程工作中獲得收入，並與當局保持定期聯繫，要求更新伍茲的刑事案件進展。

在他與當局之間發送的眾多消息中，基蘭斯被視為接受案件的現實情況，並且在伍茲的精神健康受到質疑時，他暗示伍茲有可能恢復健康。

2021年，伍茲（Woods）從醫院和監獄釋放，再次無家可歸，花費兩年多時間提交各種報告，說服願意聽取的任何當局，事實上他是身份盜用的受害者。

他的運氣在2023年1月13日轉過來，當他發現基蘭斯（Keirans）工作的地方後，他通知愛荷華州醫院有關這種情況，隨後當地警察部門的高級偵探介入調查。

那位偵探在幾個月的時間內成功揭開了基蘭斯長達33年的計劃，部分利用DNA測試確定基蘭斯不是他之前聲稱的肯塔基州一位老人的兒子。

在與偵探進行的一系列訪談中，基蘭斯最初繼續假冒伍茲，直到DNA證據被呈現給他時，他才表示“我的生活完了”和“一切都沒了”。

Keirans [现在面临](https://www.justice.gov/usao-ndia/pr/former-hospital-administrator-pleads-guilty-identity-theft-scheme-spanned-three)最高可达32年的监禁，最少两年。此外，他还必须支付125万美元的罚款，并在刑期后服役五年的监督释放期。 ®
