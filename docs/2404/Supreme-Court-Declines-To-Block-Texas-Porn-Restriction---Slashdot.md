# 最高法院拒绝阻止德克萨斯州限制色情内容 - Slashdot

> 来源：[https://yro.slashdot.org/story/24/04/30/1913215/supreme-court-declines-to-block-texas-porn-restriction](https://yro.slashdot.org/story/24/04/30/1913215/supreme-court-declines-to-block-texas-porn-restriction)

最高法院在周二

[拒绝阻止](https://www.nbcnews.com/politics/supreme-court/supreme-court-declines-block-texas-pornography-restriction-rcna149877)

基于言论自由的理由，德克萨斯州法律的一项规定旨在防止未成年人在线访问色情内容。根据一份报告：

*大法官们拒绝了自由言论联盟（一家色情产业贸易团体）以及几家公司的请求。挑战者表示，这项2023年法律违反了宪法第一修正案，因为要求使用相关平台的任何人，包括成年人，提交个人信息。

该法律的一项规定，称为H.B. 1181，要求平台通过要求用户提交有关其身份的信息来验证用户的年龄。尽管该法旨在限制儿童访问性内容，但诉讼关注的是这些措施如何影响成年人。"具体而言，该法案要求成年人遵守侵入性的年龄验证措施，这些措施要求通过互联网提交个人身份信息，以便访问包含敏感和亲密内容的网站，" 挑战者在法庭文件中写道。
