# 埃尔德什问题

> 来源：[https://www.erdosproblems.com/](https://www.erdosproblems.com/)

数学问题一直是我数学生涯中的重要组成部分。一个精心选择的问题可以突出某一领域的关键困难，作为衡量该领域进展的基准。它可能像一个'棉花糖'，提供一点短暂的享受。或者像一个'橡树种子'，需要深刻而微妙的新见解，从而可以发展成一棵巨大的橡树。[…]

在本文中，我想描述一系列我认为是我最喜欢的问题。当然，我不能保证它们都是'橡树种子'，但由于许多问题已经阻挠了数十年来最优秀的数学家们的努力（并且经常因其解决方案而获得现金奖励），这可能表明需要新的思想，这些新思想反过来又可以导致更普遍的结果，自然而然地，又引出了新问题。

如此，数学生活的循环将永远持续下去。

保罗·埃尔德什，*我的一些喜爱的问题和结果*，1997
