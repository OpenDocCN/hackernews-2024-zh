# 人工智能如果解决了一切问题，人类还能做些什么？

> 来源：[https://www.economist.com/finance-and-economics/2024/04/09/what-will-humans-do-if-technology-solves-everything](https://www.economist.com/finance-and-economics/2024/04/09/what-will-humans-do-if-technology-solves-everything)

格雷格·伊根（Greg Egan）的小说《全排列城市》中，角色皮尔（Peer）在虚拟现实中实现了永生并完全掌控，却发现自己无比无聊。于是，他改造自己，培养起新的激情。有时他在突破高等数学的界限，下一刻又在创作歌剧。“曾经，他甚至对艾利西亚人（来世）感兴趣。不再了。他更愿意考虑桌腿问题。” 皮尔的反复无常反映了一个更深刻的问题。当技术解决了人类最深刻的问题后，剩下什么可做呢？

尼克·博斯特罗姆（Nick Bostrom）是牛津大学哲学家，他的最新著作《深度乌托邦》提出了一个问题：假设人工智能的发展非常成功，那么人类将会做什么？他的上一本书认为，在未来100年内，由于危险形式的人工智能的发展，人类面临六分之一的灭绝几率。在《深度乌托邦》中，博斯特罗姆提出了一种截然不同的可能性。如果人工智能发展到能够几乎零成本地完成所有经济有价值的工作的程度，会发生什么？在更激进的场景中，甚至像育儿这样原本认为只能由人类完成的任务，也可以由人工智能做得更好。这听起来可能更像反乌托邦而不是乌托邦，但博斯特罗姆提出了不同的看法。
