# ASD-STE100 主页

> 来源：[https://www.asd-ste100.org/](https://www.asd-ste100.org/)

**ASD-STE100** **Simplified Technical English**（简称**STE**）是一种**受控自然语言**，用于编写技术文档。

英语是科学、技术和人类关系的国际语言。它也是航空和国防工业的语言。然而，对于技术文档的读者来说，英语通常不是母语。许多读者对英语了解有限。复杂的句子结构以及许多英语单词具有的多个含义和同义词可能会引起混淆。

STE 是在1970年代末由**欧洲航空工业协会**（AECMA）与**美国航空工业协会**（AIA）合作，应**欧洲航空公司协会**（AEA）的要求创建的。目标是使飞机维护文档对英语知识有限的人更易读。结果发布了**AECMA Simplified English Guide**，于**1986年**成为国际标准：**ASD-STE100 Simplified Technical English**。

今天，STE 的成功已经超出了原本航空维护文档的预期用途，其他行业也在使用它。STE 标准在语言服务、专业翻译和口译以及学术界的兴趣也急剧增加。
