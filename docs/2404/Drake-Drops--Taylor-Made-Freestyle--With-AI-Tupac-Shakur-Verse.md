# 德雷克发布了带有AI图帕克·沙库尔歌词的“泰勒定制即兴”。

> 来源：[https://variety.com/2024/music/news/drake-taylor-made-freestyle-tupac-shakur-taylor-swift-snoop-dogg-1235977178/](https://variety.com/2024/music/news/drake-taylor-made-freestyle-tupac-shakur-taylor-swift-snoop-dogg-1235977178/)

[德雷克与嘻哈社区的一股强大势力之间的矛盾](https://variety.com/2024/music/news/kendrick-lamar-drake-j-cole-beef-explained-1235951318/)愈演愈烈。周五晚（4月19日），这位说唱歌手在社交媒体上发布了一首名为“泰勒定制即兴”（Taylor Made Freestyle）的歌曲，其中使用了来自图帕克·沙库尔（Tupac Shakur）和斯努普·狗狗（Snoop Dogg）的AI人声，在一系列矛盾性记录之间填补了空白，同时等待着[Kendrick Lamar对他新发布的“Push Ups”的回应。](https://variety.com/2024/music/news/drake-new-song-kendrick-lamar-diss-1235970255/)

在这首歌中，一个想象中的沙库尔直接对拉马尔说话，并谈到了他自从上个月发布“像那样”的诗歌以来的沉默，这首诗歌引发了这场争斗。“肯德里克，我们需要你，西海岸的救世主，把你的名字铭刻在一些嘻哈历史上，”一位人工智能沙库尔说唱道。“帮我骂他一声，谈论他喜欢年轻女孩作为我的礼物。”

[德雷克](https://variety.com/t/drake/)继续使用斯努普·狗狗的计算机生成人声直接对拉马尔说话。“全世界都在看这场棋局，但哦，你的棋子用完了，Dot / 你知道OG从来没有怀疑过你 / 但现在看起来你无所适从 / 或者你到底打算做什么。”

许多人推测，德雷克说唱了沙库尔和斯努普·狗狗的歌词，并使用AI改变了它们的音调。但德雷克在歌曲的结尾用自己的词句出现，表示这只是一个暂时的填充，直到他收到拉马尔的回应。“第一首真的只花了我一个小时或两个小时，”他说。在他发布这首歌曲的Instagram上，他写道：“在等你的时候，我猜。”

他还暗示，由于[泰勒·斯威夫特的新专辑《受折磨的诗人部门》](https://variety.com/2024/music/album-reviews/the-tortured-poets-department-review-taylor-swift-album-1235975690/)在文化上的压倒性时刻，以及一切都将被其存在所掩盖，拉马尔正坐在他的回应音轨上。“但现在我们得等一个该死的星期，因为泰勒·斯威夫特是你新的Top，”他说，指的是拉马尔之前的唱片公司Top Dawg Entertainment。“如果你准备发布，她得批准，这个女孩真的要让你表现得好像你不在争吵中。”

所有这一切始于拉马尔在Future和Metro Boomin的合辑专辑《We Don't Trust You》中为“Like That”贡献了一段惊喜的诗句。拉马尔对[J. Cole](https://variety.com/t/j-cole/)将他列入“三巨头”之一感到不满，此事发生在Cole与Drake合作的2023年专辑《For All the Dogs》中的“First Person Shooter”上。

此后的时间里，嘻哈音乐上层社会一片混乱，J. Cole在他的意外发布项目《Might Delete Later》中的“7 Minute Drill”中回应了拉马尔。在这首歌中，Cole试图贬低拉马尔的唱片作品，但很快就撤回并从流媒体平台上移除了这首歌。

上周末，Drake发布了自己的反击曲目《Push Ups》。这首歌最初以未完成的演示版本泄露，随后很快有了工作室版本。[他今天（4月19日）正式在流媒体服务上发布了这首歌。](https://variety.com/2024/music/news/drake-push-ups-kendrick-lamar-official-release-1235976597/)
