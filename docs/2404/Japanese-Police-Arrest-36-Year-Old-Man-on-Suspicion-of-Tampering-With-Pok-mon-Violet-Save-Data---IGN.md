# 日本警方逮捕36岁男子，涉嫌篡改《宝可梦紫罗兰》存档数据 - IGN

> 来源：[https://www.ign.com/articles/japanese-police-arrest-36-year-old-man-on-suspicion-of-tampering-with-pokemon-violet-save-data](https://www.ign.com/articles/japanese-police-arrest-36-year-old-man-on-suspicion-of-tampering-with-pokemon-violet-save-data)

日本警方逮捕了一名36岁男子，涉嫌在网上销售非法修改的宝可梦存档数据——这种做法在该国的[2019年不正当竞争预防法](https://www.japaneselawtranslation.go.jp/en/laws/view/3629)下被禁止。

根据[日本NHK新闻网](https://www3.nhk.or.jp/lnews/kochi/20240410/8010020207.html)的报道，由[Automaton](https://automaton-media.com/en/news/hacked-pokemon-seller-arrested-by-japanese-police/)翻译，该嫌疑人于4月9日被警方逮捕，警方网络巡逻发现证据表明，该人使用特殊工具非法修改了[《宝可梦紫罗兰》](/games/pokemon-violet)的存档数据，以更改宝可梦的技能组合。一名36岁男子因出售非法修改的宝可梦存档数据而被捕。

36岁男子涉嫌接受定制稀有宝可梦订单，并在2022年12月至2023年3月期间，以每次最高13000日元（约84美元）的价格在一个作为视频游戏资产和物品市场的网站上出售被篡改的数据。他还提供了一项交易，即用大约30美元相当的日元创建六只宝可梦。

据报道，这名36岁男子承认了正在进行的调查核心的犯罪事实，并提出他这样做是为了谋生的理由。根据Automaton的报道，嫌疑人可能面临最高五年监禁和/或最高500万日元（超过32600美元）的罚款。

这并非犯罪分子首次试图从Game Freak极受欢迎的特许经营中赚钱。例如，2021年，一起悲剧性的宝可梦卡牌劫案中，一名绝望的男子试图从六层楼上盗取大量交易卡牌的宝藏。

2022年，东京的犯罪分子继续此趋势，进行了一系列高价值的口袋妖怪抢劫，[包括一次盗贼带走价值超过6万美元的卡片](/articles/pokmon-and-other-trading-cards-targets-of-unprecedent-number-of-robberies-in-japan)。值得注意的是，对口袋妖怪的热情 — 以及它可能给你带来的麻烦 — 远远超出了犯罪团伙之外。以今年一月份洛杉矶警察解雇两名警官为例，他们因[忽略前往一个入室盗窃的呼叫而追捕一只卡比兽](/articles/lapd-officers-fired-for-chasing-snorlax-in-pokemon-go-instead-of-robbers)，此事发生在[Pokemon Go](/games/pokemon-go)中。
