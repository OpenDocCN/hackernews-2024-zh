# 再也不必为难地提意见（即使是对你的经理）

> 来源：[https://read.highgrowthengineer.com/p/never-struggle-to-give-feedback-again](https://read.highgrowthengineer.com/p/never-struggle-to-give-feedback-again)

想象一下，你最好的朋友在你讲话时总是打断你。

你会觉得难以对他们说，“嘿，你能停止打断我吗？”

大概不会吧。

那么，为什么你在给同事提意见时感到困难呢？

有很多因素在起作用：

+   他们可能对你有影响力

+   也许你和这个人不太熟

+   你不知道会发生什么——会发生什么……
