# 高级前端工程师

## 关于QuestDB

我们正在建立一个专注于性能和简易性的开源时间序列数据库。

QuestDB被用于实时存储、处理和分析各种行业的时间序列数据，如金融服务、能源、制造业、Web3和空间探索。像空客和雅虎这样的财富500强公司，为大规模、数据密集型的生产系统部署QuestDB，其中一些系统服务接近10亿用户。

我们的[开源代码库](https://github.com/questdb/questdb/)在三年内获得了13k+星标，并且是我们类别中增长最快的。我们是一个以产品为先的公司，拥有大量开发者社区。我们是由领先的风投公司和Y Combinator支持的全球分布的远程优先团队。

## 该角色

作为QuestDB的前端开发人员，您将成为一个小而紧密团队的重要成员，专注于设计和开发用户界面。您将为长期运行编写简单、高质量、可扩展和可维护的代码。您将完全拥有功能和设计变更。

前端领域变化迅速。因此，我们重视编程经验和交付价值的“冲动”，而不是对某些特定框架的了解。

我们希望候选人熟悉以下技能：Typescript、React、NextJS、Jest、Cypress、GitHub Actions。

该角色需要强烈的问题解决能力和对UI/UX组件及所有用户界面的所有权。

## 职责

+   设计、构建和维护所有QuestDB用户界面

+   确保所有QuestDB产品都具有良好的性能和一致的用户体验

+   改进和维护经过测试和自动化的发布

+   确保与后端API的兼容性

+   与社区互动，推广新功能并收集反馈

## 要求

+   5年以上构建Web前端软件的成功经验

+   精通React、Typescript、HTML和CSS

+   熟悉流行的前端代码构建工具

+   出色的沟通技巧

+   强大的团队合作者，乐于在您的专业领域指导同事
