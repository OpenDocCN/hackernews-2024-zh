# 加入视频技术领域最优秀的团队 | Mux

> 来源：[https://mux.com/jobs?j=product-designer](https://mux.com/jobs?j=product-designer)

Mux致力于为开发者提供视频服务。我们的使命是通过解决开发者在视频构建过程中面临的难题来使视频民主化：视频编码和流媒体（Mux Video）、视频监控（Mux Data）等。视频是人们生活中的重要组成部分，我们希望能够帮助改进它。

我们致力于打造一个健康的团队，欢迎不同背景和经验的人加入。我们希望有志于我们的使命、愿意成长、信奉我们的价值观（从“人性化”到“将客户变成粉丝”）并且希望能够使身边的人变得更好的人加入我们。

您将加入一个紧密团结的团队，成员曾在Google、YouTube、Twitch、Zencoder、Fastly等公司积累丰富经验。我们的创始人曾创办并出售了Zencoder，这是云视频技术的早期领导者，并开发了全球最大的HTML5视频播放器Video.js。我们还主办全球顶尖的视频工程师会议Demuxed。

我们得到了像Coatue、Accel、Andreessen Horowitz和Y Combinator这样顶级投资者的支持。您将有机会与众多杰出企业合作，包括数百家初创公司以及Reddit、Vimeo、Robinhood、CBSi、Discovery、PBS和TED等。无论大小客户都喜欢与我们合作，也喜欢我们的团队。

我们正在共同打造一件大事。我们很乐意听取您的意见！
