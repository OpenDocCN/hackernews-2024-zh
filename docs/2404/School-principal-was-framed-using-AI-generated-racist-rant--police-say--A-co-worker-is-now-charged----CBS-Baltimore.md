# 学校校长被指使用 AI 生成的种族主义言论，警方称同事现已被起诉。- CBS Baltimore

> 来源：[https://www.cbsnews.com/baltimore/news/maryland-framed-principal-racist-ai-generated-voice/](https://www.cbsnews.com/baltimore/news/maryland-framed-principal-racist-ai-generated-voice/)

巴尔的摩 - 周四早晨，一名前高中体育总监因涉嫌使用人工智能冒充学校校长的录音而被捕，录音中包括对学校内个人的[种族主义和反犹太主义](https://www.cbsnews.com/baltimore/news/baltimore-co-schools-still-investigating-alleged-racist-recording-by-pikesville-principal-superintendent-says-antisemetic/)评论。

曾任职于派克斯维尔高中的员工 Dazhon Darien 正面临多项指控，包括盗窃、跟踪、干扰学校运作和对证人的报复。

调查人员确认他伪造了校长 Eric Eiswert 的声音，并在一月份在社交媒体上传播了这段音频。根据巴尔的摩县检察官办公室的说法，Darien 的绰号 DJ 在他所声称伪造的音频片段中提到。

巴尔的摩县侦探表示，Darien 制作录音作为对 Eiswert 的报复，后者曾对学校资金潜在管理不善进行调查，巴尔的摩县警察局长 Robert McCullough 于周四表示。

警方和[AI 专家相信](https://www.thebaltimorebanner.com/education/k-12-schools/eric-eiswert-ai-deepfake-YUNO6ITYM5FWZPQAE24RIBV5CQ/)，Eiswert 的声音是模拟的，对黑人学生和周围的犹太社区发表了贬低性的言论。这段音频在社交媒体上广为传播。

巴尔的摩县警方称，Darien 在机场试图登上飞往休斯敦的航班时被捕。据警方称，他当时身上携带有枪支。

"McCullough 说，"由于 Darien 所申报的枪支包装方式被拦截，当警官们调查他的背景时，他们发现他有一个未执行的逮捕令。

## 基于种族的评论

根据指控文件，音频片段中模仿 Eiswert 校长的声音进行了基于种族的评论，包括声称非裔学生无法 "从纸袋中测试出来"。

还有关于犹太人和两名 "本不应被聘用到学校的" 教师的言论。警方表示，录音描绘了 Eiswert 和学校副校长之间的虚假对话。

社交媒体上的视频片段广为传播，对此产生了重大影响。Eiswert 被暂时移出学校，社交媒体上涌现出一波波充满仇恨的信息，而学校也接到了大量电话。

## 录音的真实性受到了质疑

在会议后，艾斯沃特告诉警方，他认为音频片段是使用人工智能伪造的。根据起诉文件，艾斯沃特告诉警方，他认为达里恩负责此事。他说，达里恩“在技术上很精通，相信他对人工智能很熟悉”，并且有关达里恩合同未来学期不会续签的工作表现频繁挑战的讨论。

多位派克斯维尔高中的管理人员接受采访透露，自从达里恩于2023年7月开始担任体育主任以来，他存在多个与工作相关的问题。警方称，达里恩因未经批准解雇学校长期教练而受到谴责。

调查人员称，达里恩在12月和1月多次访问学校网络，并进行了关于OpenAI工具的在线搜索 - 警方将其与付费OpenAI账户联系在一起。

一名由FBI承包的法庭分析师和大学教授对文件进行了音频分析。结果表明，录音包含AI生成内容的痕迹，并在事后通过人工编辑添加了背景噪音以增加真实感，根据起诉文件。加利福尼亚大学伯克利分校的另一位专家也分析了音频，并确认其经过了篡改。

## 学校领导的回应

“正如你可以想象的那样，对于派克斯维尔高中社区、校长艾斯沃特及其家人以及BCPS团队来说，这是一个非常艰难的时期。我们非常自豪学生和员工如何站出来支持彼此，”巴尔的摩县学校超级管理员Myriam Rogers博士说。“正如我当时所说，并且值得重申，录音中的声明不反映我们体制的核心价值观。”

罗杰斯补充说，该县正在根据达里恩的逮捕采取“适当措施”，“包括建议终止他的职务”。
