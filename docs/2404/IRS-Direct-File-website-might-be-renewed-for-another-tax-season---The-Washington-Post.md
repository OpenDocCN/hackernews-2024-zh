# IRS 直接报税网站可能会为另一个税季延长 - 华盛顿邮报

> 来源：[https://www.washingtonpost.com/business/2024/04/26/irs-direct-file/](https://www.washingtonpost.com/business/2024/04/26/irs-direct-file/)

拜登政府宣布，其首个免费报税网站 [its first-of-its-kind free tax filing](https://www.washingtonpost.com/business/2024/04/15/irs-direct-file-reviews/?itid=lk_inline_manual_2) 不仅预算超支，而且获得了用户高度评价。今年春季，官员们将决定是否继续为另一个税季延长这个试验性项目。

这款软件类似于Intuit和H&R Block等公司的商业产品，允许纳税人直接免费向政府报税。专家称，全国推广有望在未来颠覆数十亿美元的税务准备行业；美国人平均每年花费超过200美元使用软件或税务准备人员申报纳税。
