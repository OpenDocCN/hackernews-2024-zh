# 马克·安德森是个疯子

> 来源：[https://gizmodo.com/marc-andreessen-is-a-maniac-1851433847](https://gizmodo.com/marc-andreessen-is-a-maniac-1851433847)

乍一看，马克·安德森似乎拥有各种矛盾的信仰。从远处看，这位风险投资家口中的话语充斥着[标准的右翼自由主义](https://www.salon.com/2014/06/06/techs_toxic_political_culture_the_stealth_libertarianism_of_silicon_valley_bigwigs/)，但他也以某种方式成为[五角大楼的铁杆粉丝](https://www.youtube.com/watch?v=_lQWIFi0D4k)。他是“美国”的[忠实粉丝](https://fortune.com/longform/andreessen-horowitz-beyond-silicon-valley/)，但似乎不太喜欢大多数美国人。他的风险投资公司安德森·霍洛维茨声称美国政府被特殊利益和游说所束缚，但去年他在游说这个政府时却[花费了大笔资金](https://www.forbes.com/sites/davidjeans/2024/04/19/gundo-bros-silicon-valley-military/?sh=150a428852c9)。

尽管安德森在思想上可能表现得四处碰壁，但当你深入他的核心信仰时，它们似乎很简单。他是权力的忠实粉丝。那是某些人的权力，比如像他这样的人。换句话说，富人。我曾经认为安德森有点小丑般的人物——一个如此长时间如此富有，以至于他的大脑已经变得失常。他对投资[糟糕的创意](https://fortune.com/2023/07/13/adam-neumann-why-marc-andreessen-invested-flow-wework/)的热爱使我坚信这一点。然而，现在我觉得“疯子”这个词更合适。他似乎是任何有助于维持或增强美国精英权力积累的事物的狂热信徒，而其他一切则被视为无关紧要。

一篇关于安德森·霍洛维茨（Andreessen Horowitz）相对较新的“美国活力”基金的[最近福布斯报道](https://www.forbes.com/sites/davidjeans/2024/04/19/gundo-bros-silicon-valley-military/?sh=150a428852c9)揭示了一些新的信息。这个基金主要投资国防初创企业，再次引发了人们的关注。对于一个经常抨击[大政府](https://twitter.com/pmarca/status/1775424509509796108)的人来说，投资美国战争机器或许有些禁忌，但他似乎对此毫不在意。报道聚焦于“军火兄弟”，一群爱国软件开发者，他们的创业公司得到了a16z的支持。这些“兄弟”们听起来大多受到20世纪80年代动作电影的影响：

> 他们一边编程一边举铁，每周在海滩上举办篝火晚会，狂饮能量饮料。他们呼吁回归美国硬件建设的根基，位于埃尔塞贡多的先驱们，比如杰克·诺斯罗普和艾伦·洛克希德，建立了美国军火库的支柱。他们还接纳了[有效加速主义](https://12ft.io/proxy?q=https%3A%2F%2Fwww.forbes.com%2Fsites%2Femilybaker-white%2F2023%2F12%2F01%2Fwho-is-basedbeffjezos-the-leader-of-effective-accelerationism-eacc%2F%3Fsh%3D536890067a13)，这一哲学主张不惜一切代价推动技术进步（而且以安德里森为主要支持者）。

A16z的“美国活力”基金乍一看无疑是滑稽的。在[其网站](https://a16z.com/american-dynamism/)上，一个传奇的时间轴声称拥有一系列令人印象深刻的历史事件——莱特兄弟的首次飞行、晶体管的发明、曼哈顿计划、登月——并将它们全部描绘为“美国活力”的一部分“运动”，好像A16z与这些事情有关系一样。“美国活力体现了推动美国前进的创新、进步和韧性精神。这一强大力量体现在技术和创新领域的开创性成就上，塑造着我们的国家和全球格局，”该网站称。一个浏览这些垃圾的网站访客可能会被分心到忘记这个基金正在[推广一大批私人防御、监视和武器承包商](https://a16z.com/american-dynamism-50/)，他们将为自己储存新发展。

从整体来看，安德里森的兴趣具有一定的混乱一致性。他们的态度似乎是：不管什么，只要能加速你的财富和权力——放任市场、技术颠覆、疯狂的人工智能、导弹和炸弹——都是好东西！安德里森去年在其风险投资公司的博客上发表了他的“[技术乐观主义宣言](https://a16z.com/the-techno-optimist-manifesto/)”，这篇“宣言”基本上是对企业贪婪和技术精英积累财富的一种混合体。它读起来像是戈登·吉柯和史蒂夫·乔布斯的主题演讲，还带有一点帕特里克·贝特曼的修辞风格。

鉴于他对生活的漫画般富有-深搜的眼光，安德里森不会对那些不符合他的优生学式抱负的社会群体抱有好感。换句话说，《美国展望》上周刊登了作者里克·珀尔斯坦的[个人散文](https://prospect.org/power/2024-04-24-my-dinner-with-andreessen/)，这篇散文似乎进一步证明了安德里森的绝望的精英主义观点。

据佩尔斯坦称，他曾被邀请参加安德里森2017年一个价值数百万美元的豪宅所举办的一个听起来真是令人难以忍受的派对。在这个事件中，在场的晚宴参与者们烦人的伪智性闲谈之间，据称安德里森关于美国乡村人的话实在是让人印象深刻。以下是佩尔斯坦叙述的交流：

> 我提到了亲情、友情、手艺、记忆、传说、传承的技能以及小镇提供的其他好处：*这些都是使人类成为人类的东西*。我指出他成长的地方一定有值得保存的东西。我敢说，当一个古老的人类社区从地球上消失时，总是值得哀悼的；也许人们不仅仅是生命平衡表上找到适当价格的数字...
> 
> 那就是那位拥有七个壁炉的城堡里的男人说的话。
> 
> *“我很高兴有奥斯康定和电子游戏让那些人安静下来。”*

或者类似的话。佩尔斯坦在安德里森使用的确切措辞上有些含糊不清，声称：

> 我稍作引用，虽然我不能确定这些是他的确切话语。马克，如果你看到了，随时联系我，让我记起来。也许他说的是“安静”，或者“温顺”，或者可能是“无力”。肯定是沿着这些线路说的。

不过，这听起来像是他会说的话。

从他以前的评论中可以看出，安德里森对没有大笔银行账户的人没有太多好感。多年来，他据报道说过一些迷人的话，比如美国中产阶级[是“历史的偶然事件”](https://www.businessinsider.com/andreessen-american-middle-class-an-accident-2014-10)，美国中产阶级“是一个神话”和/或“是一个产物”，并且[它是一个](https://qz.com/36368/eight-things-marc-andreessen-said-to-quartz-that-made-us-sit-up-and-listen)“试验 [已经进行过，而且是一个灾难性的失败。”另一个有趣的事情 [安德里森说过的](https://qz.com/36368/eight-things-marc-andreessen-said-to-quartz-that-made-us-sit-up-and-listen):“我不是说我们应该在美国设立血汗工厂，也不是说我们不应该有环境法规，但在美国大多数州做生意比在世界上许多地方都要困难。”

如果美国中产阶级是一个意外，法规对生意不利，乡村美国人最好的事情是吃止痛药和玩电子游戏，但美国战争机器却完全了不起……你可以很清楚地看出安德里森对大多数美国人的看法以及他认为我们国家应该走向何方。如果你对此不像他那样感到兴奋，也是可以理解的。
