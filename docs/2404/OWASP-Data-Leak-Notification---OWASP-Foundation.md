# OWASP数据泄露通知 | OWASP基金会

> 来源：[https://owasp.org/blog/2024/03/29/OWASP-data-breach-notification](https://owasp.org/blog/2024/03/29/OWASP-data-breach-notification)

![image](img/234bfd18daa6ac177e7546b2f2a47b1d.png)

Andrew van der Stock

2024年3月29日星期五

2024年2月底，在接到几个支持请求后，OWASP基金会意识到OWASP旧Wiki Web服务器的配置错误，导致十年以上的会员简历泄露。

+   **谁受到影响？** 如果您是2006年至2014年间的OWASP会员，并作为加入OWASP的一部分提供了您的简历，我们建议您假设您的简历已包含在此泄露中。

+   **哪些数据被泄露？** 简历包含姓名、电子邮件地址、电话号码、家庭地址以及其他个人可识别信息。

+   **为什么收集这些数据？** OWASP在早期会员过程中收集简历，会员在2006年至2014年期间必须展示与OWASP社区的联系。OWASP不再将简历作为会员入会过程的一部分收集。

+   **OWASP采取了哪些措施来纠正此泄露？** 我们已禁用目录浏览功能，审查了Web服务器和Media Wiki配置以查找其他安全问题，完全删除了Wiki站点上的简历，并清除了CloudFlare缓存以防止进一步访问。最后，我们请求将这些信息从Web存档中删除。

+   **OWASP会通知谁？** 我们非常谨慎地将此问题带给更广泛的公众注意。由于许多受此泄露影响的个人已不再与OWASP有关，并且数据的年代介于十到十八年之间，其中包含的个人详细信息已经明显过时，因此联系变得困难。尽管如此，在我们的调查中发现的电子邮件地址，我们将会进行联系。

+   **OWASP如何保护当前会员数据？** 我们应用现代基于云的安全最佳实践，例如双因素身份验证、最小访问权限和弹性，以保护我们的会员数据。此外，我们仅有意收集最少信息以减少未来潜在的数据丢失。

+   **我认为我受到影响。我需要做什么？** OWASP已经将您的信息从互联网上删除，因此您无需立即采取任何行动。如果面临的信息已过时，则无需采取任何行动。但是，如果信息是当前的，例如包含您的手机号码，请在回复未经请求的电子邮件、邮件或电话时采取常规预防措施。

我们意识到这次泄露的重要性，尤其考虑到OWASP基金会对网络安全的重视。我们向受影响的人道歉，并承诺确保这种情况不再发生。我们正在审查我们的数据保留政策，并将实施额外的安全措施，以防止未来的泄露。

> 修订（2024年4月19日）：这起事件本应被标记为“泄露”，而不是“违规”。由于配置错误，信息暴露在公共互联网上，而非攻击所致。我们不知道是否有人访问了暴露的信息，或者是谁访问了。新闻稿已经修订以澄清这一点。
