# 乌克兰的厂房破坏无人机是一架价值9万美元的运动飞机，由机器人控制。

> 来源：[https://www.forbes.com/sites/davidaxe/2024/04/03/ukraines-new-factory-smashing-drone-is-a-90000-sport-plane-with-a-robot-at-the-controls/](https://www.forbes.com/sites/davidaxe/2024/04/03/ukraines-new-factory-smashing-drone-is-a-90000-sport-plane-with-a-robot-at-the-controls/)

Aeroprakt A-22。

Leighnor Aircraft照片

为了实施[其有史以来最深入的一次打击](https://www.forbes.com/sites/davidaxe/2024/04/02/ukraine-packed-a-cessna-style-plane-with-explosives-added-remote-controls-and-kamikazed-it-into-a-russian-drone-factory-600-miles-away/?sh=5660d84e1dc0 "https://www.forbes.com/sites/davidaxe/2024/04/02/ukraine-packed-a-cessna-style-plane-with-explosives-added-remote-controls-and-kamikazed-it-into-a-russian-drone-factory-600-miles-away/?sh=5660d84e1dc0")，乌克兰政府使用了本地制造的超轻型运动飞机，将其有人控制改为机器人控制，并填充了爆炸物。

<fbs-ad position="top" progressive="" ad-id="article-0-top"></fbs-ad>

[周二对阿拉布加特殊经济区工业园区发动攻击](https://twitter.com/maria_avdv/status/1775405317376913652 "https://twitter.com/maria_avdv/status/1775405317376913652")的应急响应视频显示，距离乌克兰边境600英里的无人机——可能是两架袭击了该园区——基于Aeroprakt A-22。这是一种单发动机高翼运动飞机，可容纳两人。

乌克兰能够将A-22改装成爆炸无人机，极大地暗示了对阿拉布加设施的袭击可能不会是这种新型无人机的最后一次。

这款简单、可靠且不引人注目的A-22，最终适合用作无人机转换。同样重要的是，它是乌克兰制造的，且价格实惠，每架仅售9万美元。

要有个对比：基于A-22的无人机，能够穿越俄罗斯的防空系统行程600英里，以高精度传送数百磅的炸药，其成本仅略高于单一美制Javelin反坦克导弹的成本。乌克兰部队百发百中地使用Javelin导弹。

从生产角度来看，A-22无人机是可扩展的。因此，乌克兰冲突情报团队在分析阿拉布加袭击后表示，“我们预计未来将会有更多的袭击尝试”，据报道，这次袭击导致14人受伤，损坏了无人机工厂或附近的工人宿舍。

A-22是一种中产阶级业余飞行员可能为了飞行而购买的飞机。亚利桑那州的Leighnor飞机公司在美国销售A-22，并在其网站上[吹嘘道](https://www.leighnoraircraft.com/a22ls/ "https://www.leighnoraircraft.com/a22ls/")：“如果您正在寻找一款坚固易于操作、具有惊人的短场性能、但又能以95节以上的速度巡航（在合法情况下）并携带重载的飞机，那么您来对地方了！”

<fbs-ad position="topx" progressive="" ad-id="article-0-topx-1">“控制和安全性”，Leighnor强调道。“即使没有襟翼，失速也是一个非事件……在低速下，控制轻便而有效—[而且]在高速下则坚挺起来，使巡航变得更加轻松。”

“正确的审校后，金属结构耐用且抗外部环境侵蚀”，公司补充道。“更好的消息是，一切仅需不到90,000美元！”

无人机控制着美国空军在一架塞斯纳206上安装的设备。

美国空军照片

我们不知道乌克兰人确切地如何将A-22改装成杀伤性无人机，但猜测也不难。请记住，2019年，美国空军从一架1968年款塞斯纳206轻型飞机上拆除了座椅和控制系统，并安装了一套计算机驱动的伺服系统。

“该系统‘抓住’了操纵杆，按下了舵和刹车，控制了油门，翻转了适当的开关，并像飞行员一样读取了仪表盘仪表，”空军研究实验室[解释道](https://www.wpafb.af.mil/News/Article-Display/Article/1935442/air-force-research-laboratory-successfully-conducts-first-flight-of-robopilot-u/ "https://www.wpafb.af.mil/News/Article-Display/Article/1935442/air-force-research-laboratory-successfully-conducts-first-flight-of-robopilot-u/")。“同时，该系统使用GPS和惯性测量单元等传感器进行情况感知和信息收集。计算机分析这些细节，以便做出最佳飞行控制决策。”

<fbs-ad position="topx" progressive="" ad-id="article-0-topx-2">“想象一下，能够快速且经济地将类似塞斯纳或派珀等通用航空飞机转变为无人机，让它自主执行任务，然后将其重新转回原有的有人驾驶配置”，AFRL科学家Alok Das说道。“所有这些都可以在不对飞机进行永久性修改的情况下实现。”

在A-22无人机的情况下，乌克兰人显然不担心卸载自主控制系统。事实上，这架无人机是一种缓慢的巡航导弹。它不打算返回基地。

作为一种巡航导弹，A-22真是物超所值。考虑到新控制系统和爆炸性的有效载荷的价格，一架A-22无人机可能要花费几十万美元。

比起乌克兰本地制造的约50万美元的“海神”巡航导弹，它的价格要便宜得多。而且，它*远*比乌克兰从英国获得的300万美元的“风暴影子”巡航导弹便宜得多。

作为一个额外的优势，A-22的航程约为600英里，超过了“海神”和“风暴影子”两者的最大航程均不超过200英里。

这款运动飞机巡航导弹的主要缺点是速度较低：最高只有每小时126英里，而“风暴影子”巡航导弹可以维持每小时600英里的速度。

<fbs-ad position="topx" progressive="" ad-id="article-0-topx-3">理论上，这使得基于A-22的无人机容易受到俄罗斯的防空系统威胁。但实际上，俄罗斯境内的防空系统覆盖范围很广。乌克兰的这种新型廉价运动飞机无人机有足够的空间自由飞行和打击。</fbs-ad></fbs-ad></fbs-ad>
