如果你在工作中追求卓越，在快节奏环境中蓬勃发展，并且是一位教育背景和职业背景非常强大的后端工程师，我们很乐意听取你的意见！

**责任**

+   设计、构建和维护高效、可重用、可靠的Python代码，开发与我们的Artisans本地集成的工具。

+   构建超越我们产品团队和用户期望的功能。

+   与前端开发人员合作，确保系统整体性和良好集成，确保平台各部分功能无缝衔接。

+   使用Docker开发容器化应用程序，确保流畅的部署和测试流程。

+   编写文档完善、清晰的代码，并参与代码审查，以维护高质量的代码库。

+   确保后端基础设施的安全性、可伸缩性和性能，识别问题并创建解决方案以改善系统性能。

+   精通Python、FastAPI、Docker、PostgreSQL、Celery、Redis。

+   5年以上Python经验。

+   对系统设计有很强的直觉。

+   对JavaScript和前端技术略有了解。

+   具备设置和维护CI/CD流水线的丰富经验。

+   极致的细节关注。

+   追求在工作中的完美。
