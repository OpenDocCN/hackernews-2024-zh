# 楽天Kobo宣布推出其首款彩色电子阅读器 - The Verge

> 来源：[https://www.theverge.com/2024/4/10/24124411/kobo-libra-colour-clara-colour-e-reader-kindle-e-ink](https://www.theverge.com/2024/4/10/24124411/kobo-libra-colour-clara-colour-e-reader-kindle-e-ink)

楽天Kobo将推出其首款彩色电子阅读器，[Kobo Libra Colour](https://go.skimresources.com/?id=1025X1701640&xs=1&url=https%3A%2F%2Fus.kobobooks.com%2Fproducts%2Fkobo-libra-colour)和[Kobo Clara Colour](https://go.skimresources.com/?id=1025X1701640&xs=1&url=https%3A%2F%2Fus.kobobooks.com%2Fcollections%2Fereaders%2Fproducts%2Fkobo-clara-colour)。两者均采用E Ink的最新Kaledio彩色屏幕技术，具有细腻的柔和色调，在彩色显示内容时从300ppi灰度分辨率降至150ppi。

我将很快测试这两款电子阅读器，但到目前为止，它们似乎只是Kobo现有电子阅读器的小升级。尽管如此，这并不是件坏事！七英寸的Kobo Libra 2是我在亚马逊生态系统之外的[最喜欢的电子阅读器](/23769068/best-ebook-readers)，提供与[Kindle Paperwhite](/22745670/amazon-kindle-paperwhite-2021-review-usb-c-signature-edition-display-e-reader-books)相同的IPX8防水设计，但额外增加了物理翻页按钮、无锁屏广告以及更多存储空间。

价格为$219.99的Kobo Libra Colour保留了所有这些功能，但现在也兼容[Kobo Stylus 2](https://help.kobo.com/hc/en-us/articles/10733885674391-About-Kobo-Stylus-2#:~:text=Note%3A%20Kobo%20Stylus%202%20only,Elipsa%2C%20and%20Kobo%20Elipsa%202E.)，就像[Kobo Elipsa 2E](/2023/4/5/23669572/kobo-elipsa-2e-e-reader-tablet-kindle-scribe-onyx-remarkable)一样。然而，它比Kobo Libra 2贵$30，还需要另外购买售价$69.99的触控笔。

价格为$149.99的Kobo Clara Colour与其最接近的兄弟$139.99的[Kobo Clara 2E](/23542918/kobo-clara-2e-ereader-review-amazon-kindle)有些不同。它提供相同的六英寸显示屏和IPX8防水设计，但现在配备了16GB存储空间，以及改进的处理器。希望如此；Kobo Clara 2E的运行速度缓慢曾是我最大的抱怨之一。

楽天Kobo还推出了升级版的黑白Kobo Clara BW，具有相同的存储和处理器升级，售价为$129.99。

所有设备均可从今天起预订，并将于4月30日发货。
