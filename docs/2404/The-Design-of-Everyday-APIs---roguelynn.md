# 每日 API 设计 - roguelynn

> 来源：[https://www.roguelynn.com/talks/everyday-apis/](https://www.roguelynn.com/talks/everyday-apis/)

一个图书馆的好 API 是什么？或者更重要的是，是什么让它变糟？

实现一个 API 是一门艺术。它是用户和库本身之间的连接。我们如何优化这种连接以使体验更加愉悦？是什么让用户选择一个库而不是另一个？一个人性化的 API 需要包含哪些要素？

This talk will first discuss what makes an API good: documentation, simplicity, consistency, completeness, and flexibility. We will apply those elements by looking at examples in the wild of good and poorly designed APIs.
