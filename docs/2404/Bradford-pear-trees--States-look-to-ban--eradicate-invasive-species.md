# 布拉德福梨树：各州寻求禁止、根除入侵物种

> 来源：[https://www.usatoday.com/story/news/nation/2024/03/25/bradford-pear-trees-smell-invasive-species-banned/73040722007/](https://www.usatoday.com/story/news/nation/2024/03/25/bradford-pear-trees-smell-invasive-species-banned/73040722007/)

[布拉德福梨树](/story/news/nation/2024/03/23/bradford-pear-trees-smell-invasive/73000817007/)，以其蓬松的白色早春花朵，给人留下深刻印象。即使在全盛时期散发出的[树木的腐臭气味](/story/news/environment/2024/02/29/callery-pear-trees-bloom-in-the-spring-heres-why-thats-bad-invasive-species-indiana/72786605007/)也没能阻止它成为开发商和园艺设计师几十年来的首选装饰植物。

但是现在这棵树已经[种植在美国境内日益严重的问题中心](/story/news/local/2024/03/18/states-are-banning-bradford-pear-trees-should-tennessee-do-the-same/72984756007/)：它正在促使相关入侵树木扩散，这些树木正在占领一些城市绿地和牧场，并侵占森林。

"只需驾车沿路行驶，就能很容易看到它的扩展情况，" 弗吉尼亚州林业部的森林健康项目经理Lori Chamberlin告诉今日美国报。"它在受干扰的地区真的表现得很出色，比如路边、田野和森林边缘。在某些情况下，我们甚至看到它实际上侵入了森林。它正在取代本地树种，并阻止本地树木的生长。"

[弗吉尼亚州林](https://dof.virginia.gov/forest-management-health/forest-health/financial-assistance-program/callery-pear-exchange-program/)业部已将梨树列为入侵植物之一 - 其他几个州也是如此。四月份，该部门将举办一次树木交换活动，为土地所有者提供本地树木，以替换他们移除的高山梨树最多三棵。

这是弗吉尼亚州的首次树木交换 - 所有替换树木都已预订，但该部门希望明年扩展这一活动，Chamberlin说。"但该计划的另一个重要部分就是让公众意识到高山梨树（树木）实际上是一种入侵物种，"她说。

## 为什么布拉德福梨树和高山梨树是有害的？

"《纽约时报》在1964年写道："很少有树木具备所有理想属性，"“但布拉德福装饰梨树与理想非常接近。”

一个看似完美适合前院和街道的树木如何分支出来，成为许多州保护主义者和林业官员的公敌第一？奇怪的科学。

最初的布拉德福梨树之所以适合种植，是因为它被认为是不育的，不能繁殖。

植物育种者继续调整梨树后代 - 布拉德福梨树和其他栽培种（或克隆）如贵族、公鸡、克利夫兰选择和圣克拉拉都属于[梨树](https://www.clemson.edu/extension/bradford-pear/flowering-pear-trees.html)（梨树）物种。

植物育种者尝试新类型的梨树的一个原因是：布拉德福树可能长到20至30英尺高，尤其是在风大的天气下易分裂和折断。

但在某个时刻，其他梨树和布拉德福梨树进行了杂交，一些开始结出可生长的果实。鸟类和哺乳动物吃果实，然后在离树很远的地方排泄种子。

所以你院子里或主干道旁的布拉德福梨树看似无害 - 尽管它的花朵散发出刺鼻的香气以吸引传粉者。但这些种子可能会被啃食果实的生物遍布广泛（不适合人类食用）。

这些后代正在作为入侵物种制造更大问题，压倒本地植物并且难以根除。"当它引入这里时它表现得很好，有点太好了，" Chamberlin说。"我们现在意识到它正在侵略我们的自然区域。"

## 为什么各州在禁止种植梨树？

梨树生长迅速，一旦定居某地，它们就会从本地植物那里窃取养分、阳光和水分。与布拉德福梨树不同的是，这些梨树有坚硬的刺，并且在扩展时形成密集丛林。

"它们会弄爆轮胎，这使得在采伐木材或进行管理活动时变得棘手，" 克莱姆森大学林业与环境保护系助理教授大卫·科伊尔说道。

"所以你必须用金属履带的机器，这些东西也会很快占领牧场和旧田地。所以如果你养牲畜的话，情况会变得非常困难。"

Coyle在2000年初与美国农业部森林服务进行研究后，注意到南方树木的扩展，直到2018年成为克莱姆森大学教授。

他把阻止梨树传播和继续销售布拉德福德梨树放在了他的“头等大事”上，并帮助组织了2020年克莱姆森大学延伸部门和南卡罗来纳林业委员会的布拉德福德梨树悬赏计划。

[南卡罗来纳州把梨树](https://news.clemson.edu/invasive-bradford-pear-3-other-species-to-be-banned-for-sale-in-sc/)列入了其州级植物害虫清单，并通过了对布拉德福德梨树及使用[Pyrus calleryana](https://www.clemson.edu/extension/bradford-pear/flowering-pear-trees.html)砧木生长的任何树木的销售禁令，[自2024年10月1日起生效](/story/news/nation/2021/11/29/bradford-pear-south-carolina-sets-2024-ban-bounties-tree/8793351002/)。

## 梨树：堪萨斯、密苏里州和北卡罗来纳州等其他州正在采取行动

梨树可以在“美国东部，从新泽西到伊利诺伊州，南至得克萨斯州”，根据[入侵.org](https://www.invasive.org/alien/pubs/midatlantic/pyca.htm)，一个由乔治亚大学入侵物种和生态系统健康中心与美国农业部联合开展的项目，该项目跟踪入侵植物、昆虫及其他病原体。

各州面临着来自该物种的不同挑战。“我们还没有到无法控制的临界点，以致于我们无能为力来应对这种情况，”堪萨斯州林业服务的森林健康协调员Ryan Rastok说。

看到其他一些州“情况更糟，我们正利用这一点来表明，也许我们需要对此采取些行动，” Rastok说。“一旦它建立起来，要将其根除非常困难。”

下面是一些州为了对抗梨树的扩散所做的努力。

+   **阿拉巴马州：** [阿拉巴马合作延伸系统](https://www.aces.edu/blog/topics/forestry-wildlife/callery-pear-history-identification-and-control/)认为梨树是一种入侵物种，因为它们是“快速生长的树木，能形成密集的灌木丛，与本地植物竞争光线、水分和营养。”

+   **阿肯色州：** [阿肯色大学合作](https://www.uaex.uada.edu/environment-nature/ar-invasives/invasive-plants/bradford-pear.aspx)[扩服务](https://www.uaex.uada.edu/environment-nature/ar-invasives/invasive-plants/bradford-pear.aspx)认为布拉德福德梨树是一种入侵物种。“虽然花朵可爱，但树木随着时间不久便无法适应，被认为是入侵物种。”

+   **乔治亚州：** 乔治亚州林业委员会在其[入侵植物控制计划](https://gatrees.org/forest-management-conservation/invasive-plant-control-program/#:~:text=Callery%20pear%20is%20one%20of,%2C%20roadsides%2C%20or%20forested%20areas.)中将梨树列为罪魁祸首，该计划由美国森林局管理。治理土地以清除入侵物种的土地所有者，可以获得每英亩60美元的补偿。

+   **印第安纳州：** 该州已将[梨树列入入侵植物名单](https://www.entm.purdue.edu/iisc/invasiveplants.html)。州自然资源部建议[将较大的树砍到根部](/story/news/environment/2024/02/29/callery-pear-trees-bloom-in-the-spring-heres-why-thats-bad-invasive-species-indiana/72786605007/)，然后用除草剂处理树根。

+   **堪萨斯州：** 2024年2月，[堪萨斯州农业部门颁布了针对该树种的检疫令](https://agriculture.ks.gov/docs/default-source/statutes-ppwc/callery-pear-quarantine-02-05-2024.pdf?sfvrsn=56d39cc1_4)，从2027年1月1日起生效，禁止该树种在州内移动或带入。堪萨斯森林服务与其他几个组织合作，在东堪萨斯和堪萨斯城地区的四月和五月举办[梨树回购活动](https://deeproots.org/callery-pear-events/)。

+   **肯塔基州：** 肯塔基大学农业、食品与环境学院与其他当地团体合作，于2022年和2023年举办了[布拉德福德梨树悬赏计划](https://news.ca.uky.edu/article/bradford-pear-bounty-program-combats-invasive-kentucky-trees)。

+   **明尼苏达州：** 该州将梨树指定为有害杂草，并已开始[为期三年的逐步停产期](https://www.mda.state.mn.us/callery-pear)。据[该州农业部门](https://www.mda.state.mn.us/callery-pear)称，2026年将停止销售该树种。

+   **密苏里州：** 该州众议院的一项法案[将在2026年停止销售梨树](https://moinvasives.org/2024/03/08/hb-2412-sb-1281/)和其他五种入侵物种。密苏里入侵植物委员会将[梨树列为入侵植物](https://moinvasives.org/2021/02/18/2021-top-invasive-plants-expanding-in-missouri/)之一，2019年与密苏里保护部门合作开始了回购计划。他们计划于2024年4月23日在全州各地举行[梨树回购活动](https://moinvasives.org/2024/01/12/callery-pear-buyback-2024/)（注册截止日期为4月15日）。

+   **北卡罗来纳州：** 该州设有[布拉德福德梨赏金计划](http://www.treebountync.com/)，春季活动分别在4月6日和5月4日举行。

+   **俄亥俄州：** [俄亥俄州在2023年禁止销售和种植梨树](https://ohiodnr.gov/discover-and-learn/plants-trees/invasive-plants/callery-pear)。

+   **宾夕法尼亚州：** 该州[禁止梨树](https://www.agriculture.pa.gov/Plants_Land_Water/PlantIndustry/NIPPP/Pages/Callery-Pear.aspx)自2022年2月生效。

+   **田纳西州：** [田纳西入侵植物委员会](https://www.tnipc.org/invasive-plants/plant-details/?id=86)认为布拉德福德和梨树种是“已建立”的入侵威胁。

+   **弗吉尼亚州：** [栽培梨树](https://www.invasivespeciesva.org/species/callery-pear#:~:text=In%20Virginia%2C%20Callery%20pear%20is,also%20be%20submitted%20here%20%3E%3E.) 在 [弗吉尼亚州自然遗产部的侵入植物名单](https://www.dcr.virginia.gov/natural-heritage/ip) 上。该州还有一个 [栽培梨树交换计划](https://dof.virginia.gov/forest-management-health/forest-health/financial-assistance-program/callery-pear-exchange-program/)，树木将在4月20日被领取。（不幸的是，所有替换树木都已分配完毕。林业部希望在2025年扩展其计划。）

## 你能对抗侵入性的梨树做些什么？

即使你所在的州没有积极对抗梨树和栽培梨树，你也可以尽自己的一份力。在选择种植树木时，请选择本地物种。

“选择非侵入性植物可以美化庭院，而不会对自然栖息地和依赖它们的食物网造成负面影响，”密苏里州保护部侵入物种协调员安吉拉·索科洛夫斯基说。“原生物种对自然最有益，因为这些植物作为传粉者、鸟类和其他野生动物的自然食物链的关键组成部分而进化。”

大多数州都有在线延伸服务，提供有关本地植物和树木以及侵入物种的信息。

与此同时，科伊尔和其他专家正在研究最佳的灭除刺梨入侵者的方法。

小树苗可以从地上拔出。但更大的树木和群体需要砍树并用除草剂处理，以防再次发芽 - 在某些情况下需要烧毁它们。

“所以一旦它们长成木质和多刺，过程就从潜在的一步过程变成了两步，成本更高，”科伊尔说。“我们正试图找出时机、有效性和管理实践。进展需要时间。”
