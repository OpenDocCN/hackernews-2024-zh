# 独家消息：Xbox 总裁Sarah Bond成立了一个专门的团队，致力于游戏保存和向前兼容 | Windows Central

> 来源：[https://www.windowscentral.com/gaming/xbox/exclusive-xbox-president-sarah-bond-has-set-up-a-new-team-dedicated-to-game-preservation-and-forward-compatibility](https://www.windowscentral.com/gaming/xbox/exclusive-xbox-president-sarah-bond-has-set-up-a-new-team-dedicated-to-game-preservation-and-forward-compatibility)

## 你需要了解的事情

+   根据Windows Central收到的邮件，经过微软确认为真实，Xbox 总裁Sarah Bond 最近向她的团队介绍了各种话题。

+   在邮件中，Sarah Bond 重申了微软计划建造新的Xbox硬件，重点是实现"一代人中史上最大的技术飞跃"。

+   Sarah Bond 还透露，微软已经成立了一个专门团队，以未来硬件范式为基础，为我们的数字Xbox游戏库提供了未来保障，以进一步"建立在 [Xbox] 提供向后兼容性的强大历史"之上。

+   此外，有消息透露，自《暗黑破坏神4》加入Xbox Game Pass以来，Xbox 已成为其最主要的平台。

* * *

[Sarah Bond 被提升为Xbox 总裁](https://www.windowscentral.com/gaming/xbox/microsoft-reorganizes-gaming-leadership-matt-booty-now-overseeing-xbox-game-studios-and-zenimax-media) 几个月前，她启动了微软游戏组织的重新组织，以为未来做好准备。随着 [Activision-Blizzard 现在已经被收购](https://www.windowscentral.com/gaming/xbox/microsofts-activision-blizzard-deal-for-xbox-is-now-officially-done)，Xbox 已经成为一个游戏业务强大的力量，跨越Xbox主机和云，PC游戏，移动游戏以及订阅服务。即使不考虑数以百万计的Xbox Game Pass订户，微软在这一领域还包括多种其他订阅服务游戏业务，从Fallout 1st到Minecraft Realms和魔兽世界。

Xbox 业务的复杂性无疑是一个巨大的挑战，但公司前方同样有巨大的机遇。为此，Sarah Bond 最近发出了一封电子邮件，鼓舞她的队伍，并在此过程中分享了一些有趣的消息。微软向我们确认这些通讯是真实的。

*"组织成立将近六个月了。在这段时间里，我们取得了巨大的集体成就。每个人都应该为我们所取得的成就感到无比自豪，并对前景充满期待，"* Sarah 说道，*"我们正在全力推进下一代硬件，致力于在一代人中实现史上最大的技术飞跃。"*

Xbox 编程总监Tina Amini加入了Xbox CEO Phil Spencer、Xbox 总裁Sarah Bond以及Xbox Game Studios 负责人Matt Booty。（图片来源：Official Xbox Podcast）

萨拉还提到了微软在游戏人工智能方面的创新，这被认为是微软对Xbox未来愿景的重要组成部分。最近我们得知，即将离开XEmTech的领导和人工智能创新者卡瑞姆·乔德里在组织内以玩家和开发者为先的理念而闻名。看来这种影响将继续推动微软在技术上的思考过程，“我们正在游戏人工智能领域进行创新，专注于提供玩家优先、开发者优先的价值，用于发现、参与和创作者速度。”

接下来，萨拉描述了微软继续将动视暴雪游戏和Battle.net整合到Xbox Game Pass和PC Game Pass中的工作。不久前，[微软将《暗黑破坏神4》带入了Xbox Game Pass](https://www.windowscentral.com/gaming/diablo-4-campfire-chat)，与Windows平台团队在Microsoft Store上合作。邦德透露，由于《暗黑破坏神》登陆Game Pass，Xbox现在成为了游戏的首选平台。“我们正在将动视暴雪国王的作品整合到我们的服务中——我们推出了《暗黑破坏神4》进入Game Pass，Xbox迅速成为D4玩家的首选平台。”邦德重申了Xbox未来即将推出的其他游戏，“我们正在与Battle.net整合，同时推出《使命召唤：战区》手机版，并准备迎接即将推出的《地狱之刃II》、《誓约》以及《印第安纳琼斯与大圆圈》。”

## Xbox为游戏保护和未来建立了新团队。

微软的Xbox 360向后兼容计划仍然是该平台最出色的特性之一。

最近，游戏的保护已成为一个日益关注的问题，因为向数字许可模式和在线服务的转变使得游戏很容易永久下线。这在移动游戏领域尤为严重，像是苹果和谷歌等发行商和公司可以轻松地消除游戏的存在。甚至像任天堂这样的核心平台也因关闭商店而使许多游戏下线。微软今年正在关闭Xbox 360商店的部分新购买功能。然而，数百款最受欢迎的[Xbox向后兼容游戏](https://www.windowscentral.com/tag/xbox-backwards-compatibility)成功逃过了Xbox 360的命运，继续在Xbox One和Xbox Series X|S平台上存活。现在看来，它们将继续存在下去。

在团队的邮件中，Sarah Bond透露，微软现已设立了一支专门的团队确保当前Xbox游戏库在未来硬件范式转变中得以防护，确保我们的游戏在未来长期保持可访问性。“我们组建了一支专注于游戏保存的新团队，这对Xbox的所有员工和整个行业都至关重要，”Bond说道，“我们正在利用强大的历史背景，为玩家提供后向兼容性，并承诺为未来的玩家带来Xbox游戏的惊人库藏以供享用。”

> 我们组建了一支专注于游戏保存的新团队，对Xbox的所有员工和整个行业都至关重要。我们正在利用强大的历史背景，为玩家提供后向兼容性，并承诺为未来的玩家带来Xbox游戏的惊人库藏以供享用。

据来源透露，微软可能在即将到来的年度Xbox展示大会上分享更多关于此领域的信息，该活动预计于2024年6月9日举办（根据The Verge网站信息）。

Sarah Bond还向[ karim Choudhry致以敬意，他是微软工作了26年的资深员工，即将退休](https://www.windowscentral.com/gaming/xbox/exclusive-xbox-cvp-kareem-choudhry-is-leaving-microsoft-amid-a-large-xbox-leadership-shake-up)，“本周，在特邀karim Choudhry告别的派对上，我们分享了我们克服的障碍以及这个团队所做的了不起的事情，”Sarah继续说道，“听着这些故事，我意识到Xbox团队拥有同样的三个品质。”

我最近在一篇详细文章中阐明了一些微软在未来必须克服的痛点（[原文链接](https://www.windowscentral.com/gaming/xbox/on-xboxs-growing-pains-and-its-strange-future)）。其中一项担忧直接与出版商在牺牲游戏机发展的前提下寻求增长的宇宙世界有关。不仅听上去微软并没有计划放弃游戏硬件，而且听上去微软正在准备确保不管未来发生什么，我们都能够长期维持访问Xbox游戏的能力。我知道微软对于Windows的后向兼容性能力非常自豪，因为你能够无需问题地运行从80年代通过MS-DOS运行的游戏。我也知悉微软希望确保Xbox对于类似致力于保存的奉献精神同样而闻名。

萨拉·邦德提到了面对游戏行业的挑战，随着游戏时间越来越集中在少数几款所谓的“黑洞游戏”上，比如Fortnite、Roblox和[使命召唤](https://www.windowscentral.com/tag/call-duty)，玩家们变得越来越不愿意尝试新的游戏。自2008年达到高峰以来，游戏主机市场总体也在萎缩，这让平台持有者们纳闷未来的增长将从何而来，尤其是在成本不断增加的情况下。

*"在游戏行业增长趋于平缓之际，我们的增长依然持续，"* 萨拉强调道，突显了微软在多样化和转型方面的能力。微软多样化的技巧帮助其超越了苹果，重新夺回了全球最有价值公司的宝座。Xbox现在已经比Windows本身更大，这在十年前可能看起来是不可能的事实。巨大的挑战依然存在，但正如我所说，同样巨大的机遇也在出现。
