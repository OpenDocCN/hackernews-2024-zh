# 台积电预计客户将为在海外制造的芯片支付更高的费用 • The Register

> 来源：[https://www.theregister.com/2024/04/18/tsmc_overseas_costs/](https://www.theregister.com/2024/04/18/tsmc_overseas_costs/)

台积电总裁魏哲家表示，想在该芯片巨头的非台湾设施进行制造的客户将需要分担费用，即通过支付更高的价格来分担成本。

在[TSMC Q1收益电话会议](https://www.theregister.com/2024/04/10/tsmc_sees_semiconductor_bounce_with/)的问答环节中，分析师[问到](https://www.fool.com/earnings/call-transcripts/2024/04/18/taiwan-semiconductor-manufacturing-tsm-q1-2024-ear/)首席执行官，作为全球最大芯片合同制造商，如何应对日益上涨的开支，例如电力开支，以实现2025年目标的53%的毛利润。

分析师建议，如果业务成本上升并可能侵蚀毛利率，唯一抵消的方式就是提高价格。

魏先生表示，由于这类信息属于机密，他无法透露定价策略，但他详细说明了在运营费用较高的晶圆厂的定价方式。“我们确实遇到了海外或最近的通货膨胀和电力等一些较高成本问题。”

“我们期望客户与我们共担部分更高的费用，并且我们已经开始与我们的客户进行讨论”，他继续说道，确认更高的生产成本至少将部分由更高的价格资助。

然而，魏先生声称，客户并不会为所付出的更多费用而得不到任何回报。

“正如我所说，对于海外晶圆厂，我们希望分享我们的价值，这也包括地缘政治位置的灵活性或类似的事物”，他说道。“如果我的客户要求在某些特定地区，那么毫无疑问，台积电和客户必须共同分担额外的成本。”

地缘政治是台积电首次在美国设立工厂的主要原因，尽管美国的晶圆厂更昂贵。自1949年中国内战结束以来，中国一直声称台湾合法归属于其控制之下，当时中国共产党在大陆建立了中华人民共和国，而国民党领导的中华民国则流亡到了台湾。

尽管中国从未试图全面夺取台湾并彻底解体其对手，但近年来对中国入侵的担忧增加，尤其是自从台积电（TSMC）的尖端晶圆厂使台湾成为极为重要的战略资产以来。一次成功的入侵可能会使西方无法获得TSMC的芯片，但即使是一次失败的尝试，也可能会造成严重损害。([详细内容](https://www.theregister.com/2023/01/12/china_taiwan_wargame_scenarios/))

那基本上就是魏所指的海外晶圆厂的地缘政治价值，因为它们很可能不会很快成为军事行动的目标。当然，由于在美国等地建设晶圆厂在财务上不是最佳选择，台积电已经获得了[数十亿美元的补贴](https://www.theregister.com/2024/04/08/tsmc_116b_arizona_funding/)。

Wei没有具体说明其海外晶圆厂会更昂贵多少，但考虑到台积电可能不会在没有获得110亿美元的补助和贷款的情况下建设这些厂房，我们可以猜测差距可能相当可观。®
