# 法官支持科技亿万富翁在加利福尼亚永久公司对抗家庭牧场主的案件中。

> 来源：[https://gizmodo.com/judge-sides-tech-billionaires-california-forever-case-1851381794](https://gizmodo.com/judge-sides-tech-billionaires-california-forever-case-1851381794)

[加利福尼亚永久公司](https://gizmodo.com/tech-billionaires-california-forever-local-government-1851250361)，这是由科技亿万富翁支持的努力，旨在在海湾地区数千英亩的农田上建立一个新城市，面临着一系列挑战，包括项目支持者与想要在开发者希望建设的地区长期居住的家庭之间的法律纠纷。

去年五月，加利福尼亚永久公司的母公司弗兰纳利联合公司，[起诉了其中一些家庭](https://gizmodo.com/california-forever-flannery-associates-marc-andreessen-1850981424)，指控他们参与了“价格固定”阴谋，并要求索赔五亿美元。农民们指责加利福尼亚永久公司使用“强硬手段”来强迫他们放弃土地。

现在，加利福尼亚法官似乎站在科技亿万富翁一边。萨克拉门托的特洛伊·纳尔利法官拒绝了土地所有者要求将加利福尼亚永久公司的诉讼驳回的努力，据《水银新闻》[报道](https://www.mercurynews.com/2024/04/01/silicon-valley-billionaires-planning-solano-county-california-forever-utopia-score-big-win-in-510-million-fight-against-farmers/)，这是周一的事情。纳尔利称，两年前土地所有者之间的信息交换提供了“直接证据”，证明存在价格固定的行为。

这些信息已在法庭上分享，包括一位土地所有者发给另一位土地所有者的短信，称他们不应该让加利福尼亚永久公司开发者“欺凌最后一批土地所有者”。同一位土地所有者声称，“剩余的土地所有者应就我们愿意出售我们的土地达成一致”，以便加利福尼亚永久公司无法“玩弄土地所有者之间的矛盾”。

纳尔利还引用了其他例子。一封电子邮件交换涉及一位土地所有者称，加利福尼亚永久公司的“过于激进行为”表明我们目前处于非常有利的位置，最好现在不要与他们交涉。”同一位所有者还说：“没有人建议我们不卖，问题是何时以及以什么价格出售。该地区的其他几位主要土地所有者也在采取同样的策略，不急于与其交涉。”

Nunely 拒绝驳回此案并不一定意味着加州永恒公司会在法律诉讼中获胜。加州永恒公司试图逼迫长期居民离开其土地的指控导致了一系列[争议性市政大会](https://gizmodo.com/california-forever-solano-county-billionaires-silicon-v-1851078305)，居民在会上公开对项目CEO 雅恩·斯拉梅克喊出了不满之声。居民指责项目开发者不可信，并试图从当地社区谋取利润。

Gizmodo 就加州永恒公司诉讼更新事宜进行了询问，并在其回复后更新了本故事。

如果最近的法院裁决对项目开发者的法律案件是好消息，那么目前还不清楚它是否能帮助他们实现创建一个新城市的长期目标，该城市仅依赖于海湾地区数百英亩的农田。CF 项目目前正试图说服索拉诺县居民[支持创建一个选民提案](https://gizmodo.com/tech-billionaires-project-to-build-a-new-city-accused-o-1851355578)，允许当地人在十一月投票。有指控称该项目欺骗居民签署支持该提案的请愿书，但开发者否认了这些指控。
