# pyinfra 将 Python 代码转换为 shell 命令并在您的服务器上运行

> 来源：[https://pyinfra.com/](https://pyinfra.com/)

pyinfra 将 Python 代码转换为 shell 命令并在您的服务器上运行。执行临时命令并编写声明性操作。目标为 SSH 服务器、本地机器和 Docker 容器。快速且可扩展，从单个服务器扩展到数千个。

想象一下，使用 Python 而不是 YAML，且速度更快的 `ansible`。
