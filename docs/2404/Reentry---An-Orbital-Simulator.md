# 《重返：轨道模拟器》

> 来源：[https://reentrygame.com/](https://reentrygame.com/)

## 学院

面对现实吧。驾驶和操作太空船并不是一件简单的事情。考虑到众多的子系统、程序和控制系统，即使是在安全可控的情况下，将火箭安全地从发射台上送入空中也是一个学习曲线陡峭的过程。点火后，唯一的紧急情况逃生方案是通过紧急终止程序。成千上万加仑的燃料被消耗掉，以产生足够的推力使火箭突破大气层的初始阻力，这时动压力正推动着发射器结构极限。自动导航逻辑谨慎地将航天器引导进入太空。与此同时，宇航员忙于监控可用数据并完成检查单，而驾驶舱在剧烈震动。

一旦进入轨道，接下来的操作是计划好的，同时不断验证确保维持生命所需的基本系统功能正常，并且有回到地球的方式。

《重返：轨道模拟器》使用逼真的轨道力学，对航天器施加的每一种力都会影响其姿态和轨迹。幸运的是，你不必一开始就全部了解。游戏中的学院为每种可用的航天器提供多个课程。每个课程侧重于一到两个程序，并以基础水平进行解释。逐步地，你将能够将这些知识点串联起来，并通过考试尝试获得某种航天器的认证。
