# 新的竞争规则带来了权衡。

> 来源：[https://blog.google/around-the-globe/google-europe/new-competition-rules-come-with-trade-offs/](https://blog.google/around-the-globe/google-europe/new-competition-rules-come-with-trade-offs/)

在一些国家，关于是否采纳单独针对少数主要技术服务的新规则存在激烈辩论。尽管这些规定性法律旨在促进竞争性数字市场，但它们可能涉及权衡，提高消费者和企业的价格并限制选择。我们想分享一些我们在遵守欧洲最近通过的数字市场法案（DMA）后听到的关切。

### 帮助企业和消费者

根据DMA，我们不得不移除有用的Google搜索功能，例如航班、酒店和当地企业。这意味着如果你在欧洲搜索航班，我们不能再显示关于航空公司、飞行时间和价格的全面信息。这对少数在线旅行聚合器有利，但却对航空公司、酒店运营商和小型企业造成了更难直接接触客户的影响。

我们引入了这些类型的Google搜索功能来帮助消费者，使人们更容易访问准确的信息。在欧洲，当你在Google上搜索航班时，现在会更加费力。而且，当你在一个旅行网站上搜索低价票价时，可能会在购买前涨价，出现“此票价已不再有效”的消息。

### 对DMA合规的早期反应报告

以下是我们在实施DMA变更的国家中观察到的其他一些情况：

+   我们看到一些成功中介服务的流量增加，而航空公司、酒店、当地商家和餐馆等各种企业的参与明显减少。

+   酒店担心自从我们实施合规变更以来，直接预订点击率下降了多达30%。这些企业现在不得不通过几个通常收取高额佣金的中介与客户联系，而来自Google的流量是免费的。欧洲航空公司和当地企业[表示](https://www.reuters.com/technology/airlines-hotels-warn-google-changes-may-benefit-large-intermediaries-2024-03-06/)类似担忧。

+   消费者对于在搜索中显示谷歌地图的变更表达了负面观点。许多欧洲用户在我们的[信息板块](https://www.reddit.com/r/GoogleMaps/comments/19ahfx2/why_doesnt_maps_show_up_under_google_searches/?rdt=48192)和我们的[帮助论坛](https://support.google.com/websearch/thread/258720758)中[提出了关切](https://www.liberation.fr/economie/pourquoi-google-maps-ne-fonctionne-plus-directement-dans-la-recherche-google-et-comment-y-remedier-20240304_2WCOEUZ5IJADFMSTFPQXY2KBTA/)，他们不再在搜索结果页面上看到有用的谷歌地图选项卡。

### 平衡利益，关注消费者

我们一直致力于改进谷歌搜索，帮助人们快速轻松地找到他们需要的信息。我们开发了[Google Images](https://blog.google/products/search/18-years-after-google-images-versace-jungle-print-dress-back/)以展示照片，而不仅仅是链接到照片。我们推出了[Google Maps](https://blog.google/products/maps/look-back-15-years-mapping-world/)来帮助您找到当地的商家，而不仅仅是提到其地址的网站。我们还开发了与航空公司、酒店和商家直接联系的方式，节省时间和金钱。这些功能还帮助了许多独立公司，包括小企业，与大型成功的网站竞争。

撤销部分这些进展的规则代表了竞争政策的根本转变。我们鼓励其他考虑实施这些规则的国家考虑潜在的负面后果，包括那些在监管过程中没有发言权的小企业。

我们将继续对我们的[DMA合规义务](https://blog.google/around-the-globe/google-europe/complying-with-the-digital-markets-act/)和过于严格的产品规定的影响保持透明。在我们看来，最佳方法应该是确保消费者可以继续选择他们想要使用的服务，而不是要求我们为了少数几家公司的利益重新设计搜索。
