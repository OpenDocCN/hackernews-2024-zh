# 虽然Affordable Connectivity Program即将用尽，但低收入美国人仍然需要互联网 - The Verge

> 来源：[https://www.theverge.com/2024/4/5/24121022/universal-broadband-access-fcc-usf-acp](https://www.theverge.com/2024/4/5/24121022/universal-broadband-access-fcc-usf-acp)

在2020年8月，当新冠肺炎大流行期间许多学校关闭时，社交媒体和新闻媒体充斥着两名小学生坐在塔可贝尔外面做作业的[一张照片](https://www.ktvu.com/news/photo-of-girls-using-taco-bell-wifi-becomes-symbol-of-digital-divide)。这张照片改变了数百万人对美国所有人，无论经济状况或地理位置如何，都需要有经济实惠的宽带互联网连接的看法。

毫不夸张地说，这张照片是促使国会在2021年两党基础设施法案中包括142亿美元用于Affordable Connectivity Program (ACP)的动力之一。ACP目前每个月为超过[2300万](https://www.whitehouse.gov/briefing-room/statements-releases/2024/02/06/fact-sheet-as-affordable-connectivity-program-hits-milestone-of-providing-affordable-high-speed-internet-to-23-million-households-nationwide-biden-harris-administration-calls-on-congress-t/)低收入家庭提供30美元，以更好地支付宽带服务（居住在部落地区和非常偏远地区的家庭每月获得75美元）。参与ACP的家庭数量甚至超过了参与Supplemental Nutrition Assistance Program (SNAP)的家庭数量。

不幸的是，除非国会迅速采取行动，否则低收入美国人可能会再次被迫坐在快餐店外获取互联网接入。

ACP得到了广泛和多样化的支持，跨越了各州州长、市长、州立法者和国会成员的政治光谱。ACP的创建，以及其他资助美国全境宽带部署的计划，均坚定地认识到普遍和经济实惠的连接对于一个健康的社会和经济至关重要。

尽管ACP备受欢迎，但除非国会拨款数十亿美元的额外资金，否则该计划将于4月底用完。有[两党联手](https://www.cnn.com/2024/01/09/tech/us-lawmakers-propose-usd7-billion-extension-for-expiring-federal-internet-discounts/index.html)推动延长该计划，但时间紧迫，联邦通信委员会（FCC），负责管理ACP的机构，已经[发布指南](https://www.fcc.gov/fcc-taking-steps-wind-down-affordable-connectivity-program)给宽带提供商，告知接收者他们可能会失去互联网服务。

过去一年来，包括公民社会、联邦和地方官员以及宽带提供商在内的广泛利益相关者进行了激烈但迄今未能成功的努力，以确保为ACP（普遍通信计划）获得资金，这清楚地表明国会拨款程序不适合为像普及负担得起的宽带这样重要的电信优先事项提供资金。

幸运的是，还有更好的选择。

国会授权FCC承担重要工作，确保“美国所有人民”都能够获得日益发展的通信服务水平。这一目标，称为“普遍服务”，主要通过1996年电信法案的一部分——通信法，实现。美国通信基金的资金来源是对传统电话服务的某些收入征收的费用。购买这类服务的消费者通常会在账单上看到“通用服务费”的项目。正如国会设想的那样，美国通信基金已从支持一个基本的通信网络——电话服务，发展为支持我们时代的基本通信网络——宽带互联网。

由于美国通信基金的影响，每个州和领土都拥有了更广泛的连接。

美国通信基金每年大约花费80亿美元，用于帮助弥合数字鸿沟，资助连接被认为“不经济”的农村、部落和偏远地区。它为学童和图书馆用户提供了接入机会，其中许多人否则无法负担接入费用，因此将在这个数字经济中落后。它为农村健康诊所提供了连接，帮助确保远程医疗可以送达他们所在地，而不是迫使他们开车数百英里或无法获得医疗照顾。它还帮助数百万低收入家庭接入并保持与有线和移动网络的连接。由于美国通信基金的作用，每个州和领土都拥有了更广泛的连接。

这些重要使命仍然至关重要，因为工作尚未完成。通信法赋予联邦通信委员会（FCC）更新和扩展美国通信基金的资金方式的权力，以应对当今的需求，包括确保ACP为低收入家庭提供补贴的永久资金机制。目前，通信法规定，只有电话公司通过其客户向美国通信基金缴费。随着越来越多公司停止提供电话服务，他们停止缴费，这增加了少数公司和消费者的负担。这种情况是不可持续的。

为了防止通用服务基金崩溃并拯救宽带补贴，FCC应立即启动程序，通过额外的通信网络的贡献来维持和扩展通用服务基金。 虽然其他人敦促国会赋予FCC权力考虑将费用应用于像云服务或Google和Meta等在线公司这样使用通信网络的其他公司，但几乎可以毫不费力地说，宽带公司应该为该基金做出贡献。 扩大参与通用服务基金的公司基础后，FCC可以复制甚至改进ACP提供的补贴。

FCC必须不论国会是否延长ACP，都必须现代化通用服务基金。 国会和白宫提议的资金水平最多只能提供一年的支持，如果我们现在通过的这个延长过程是任何迹象，那么寻求超过今年的资金将会更加艰难。

当通过两党基础设施法案时，国会拨款了650亿美元用于宽带部署、经济实惠和采纳。 法案文本还有意促成了美国通用服务基金的现代化，明确邀请FCC向国会提出扩大该基金的建议。 不幸的是，该机构在讨论各种选择时却没有进一步行动。

我们正面临另一个“孩子们在塔可钟前完成学校作业以使用WiFi”的时刻，超过2300万个美国家庭面临失去经济实惠的连接的危险。 FCC有机会和权力拯救它建立和培育的计划。 它应该紧急采取行动。
