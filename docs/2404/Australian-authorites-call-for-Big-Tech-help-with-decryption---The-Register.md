# 澳大利亚当局呼吁大科技公司帮助解密 • The Register

> 来源：[https://www.theregister.com/2024/04/25/asio_afp_accountable_encryption/](https://www.theregister.com/2024/04/25/asio_afp_accountable_encryption/)

昨天，澳大利亚主要情报机构的总监和其联邦警察局局长都呼吁社交网络提供更多帮助，以协助他们处理涉及恐怖主义、儿童剥削和种族主义民族主义的案件。

昨天，这两位头目一同出现在澳大利亚国家新闻俱乐部。

第一个登台发言的是澳大利亚安全情报组织局长迈克·伯吉斯，他开场说：“互联网是一个变革性的信息源…*而且*是世界上最强大的极端主义温床。”

在[详细阐述](https://www.asio.gov.au/director-generals-national-press-club-address)安全和技术之间存在动态紧张关系的论述中，伯吉斯补充道：“加密保护我们的隐私并促进我们的经济…*还*为暴力极端分子提供安全空间，让他们运作、建立网络和招募。”

Burgess将加密称为“显然是一件好事，对我们的民主和经济都是积极的”，因为它“保护隐私，使通讯和交易成为可能。”

但他指出，加密还为犯罪分子提供了匿名性，这就是为什么澳大利亚制定了法律来获取加密消息的原因。伯吉斯说，这些法律并没有起到作用，因为科技公司没有提供帮助。

“但即使授权允许我们合法拦截加密通信，没有拥有和运营该应用程序的公司的帮助，我们实际上无法阅读它，”他说。“公司必须愿意并能够执行我们的授权。”

间谍头目明确表示，他不希望获得更多权力，也不希望终结端到端加密。

“我要求技术公司做更多。我要求他们执行现有的权力并遵守现行法律，”他说。

Burgess辩称：“在非常有限和严格控制的情况下，如果没有他们的帮助，加密是无法问责的。” “实际上，无法问责的加密就像为恐怖分子和间谍建造一个安全房间，一个他们可以策划和计划的安全场所。”

他通过讲述最近一次行动的故事来支持“问责制加密”的案例，这次行动涉及一名澳大利亚人在线共享极端内容，据称他已经联系恐怖分子讨论进行暴力行动的计划。伯吉斯称，经过一次“困难、危险、耗时和资源密集的”行动后，ASIO确定该个体“具备进行袭击的意图和能力”。

间谍头目认为，有问责制的加密意味着ASIO可以更快速、更轻松地得出结论。

“对其通讯的快速和有针对性的访问可能是生死攸关的差别”，伯吉斯辩称。

澳大利亚联邦警察局局长基尔肖（Reece Kershaw）接下来发言，并引用了32名欧洲警察局长于4月21日发布的[声明](https://www.nationalcrimeagency.gov.uk/news/european-police-chiefs-call-for-end-to-end-encryption-roll-out-to-include-public-protection-measures)，其中他们表达了对端到端加密被滥用的深切关注，认为这损害了他们调查犯罪和保护公众安全的能力。

基尔肖支持了这一观点，并像他的欧洲同事一样，将其放在对儿童剥削调查的背景下进行了阐述。

“如果司法官员决定存在严重犯罪的合理怀疑，并且有必要让执法机构访问信息来调查该严重犯罪，技术公司应尊重法治和法院或独立司法机构的命令，提供该信息”，基尔肖说。

“我的大门向所有相关科技公司的CEO和董事长敞开，包括埃隆·马斯克和马克·扎克伯格”，这位警察负责人说。“我知道我们可以找到共同点，因为简单来说，科技应该让我们的生活更轻松和更安全，而不是相反。”

或许很难找到共同点，特别是因为埃隆·马斯克目前正在反对一项要求撤下描绘恐怖行为视频的澳大利亚法律。几名涉嫌在视频中被描绘为行为者的同伙于周三[被逮捕](https://www.afp.gov.au/news-centre/media-release/seven-arrested-following-joint-counter-terrorism-team-operation)，因其分享暴力意识形态，因此被认为对澳大利亚社会构成不可接受的风险。

### 幽灵幽灵

ASIO（澳大利亚安全情报组织）负责人伯吉斯（Burgess）还讨论了人工智能，他称这项技术“既是炒作，又是机遇，也是威胁”。

“炒作是因为当前现实与科技传道者和营销专家所宣称的之间存在巨大鸿沟。他们所称之为AI的很多东西并非如此”，伯吉斯在预测AI提供了“向社会各个部门，从经济到医疗，提供红利的机会”之前说道。

但他哀叹AI也能提高“那些可能用它威胁澳大利亚安全”的人的生产力，并表示一些恶意人士已经在利用AI进行恶行。

“我们知道已经有离岸极端分子向某商业可用的AI程序寻求建造武器和攻击计划的建议”，伯吉斯说。“如果程序拒绝提供请求的信息，极端分子会试图绕过道德限制。”

伯吉斯还预测“人工智能很可能会使激进化变得更加容易和更加快速”，并且“将增加间谍活动的数量”，因为它给国家提供了一个很好的理由，“收集个人数据来支持他们自己的编程，并且更有动力窃取关于竞争对手人工智能技术的信息。”

他还准备好应对外国情报机构进行的“更加频繁、更加可信和更加有效的虚假信息宣传活动”。

他说，ASIO将以人工智能对抗人工智能，因为这是“最明显而可能是最好的防御手段。”伯吉斯确认，“ASIO多年来一直在使用人工智能。”

“它并不是取代我们的人员 - 它是在增强和协助他们，”他说，通过使ASIO员工更容易筛选信息。

“找到一条关键情报就不再像是在一堆稻草中找针那么困难，而是像是在一片稻草田中找针，”他说。“人工智能使这一过程更加简单和快速；它可以在几分钟或几小时内识别出令人担忧的模式和关系，而不是几周或几个月。”

但前提是它所处理的数据没有加密。
