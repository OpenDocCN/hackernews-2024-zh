# Kodi 21.0 "Omega" - 发布 | 新闻 | Kodi

> 来源：[https://kodi.tv/article/kodi-21-0-omega-release/](https://kodi.tv/article/kodi-21-0-omega-release/)

**时机到了！** 我们非常高兴地呈现 Kodi 的最新和最伟大的版本，v21.0 "Omega"。

自从 2023 年 1 月 15 日发布 v20.0 "Nexus" 以来，我们已经看到了超过 3,750 次提交。一如既往，这是许多贡献者、测试人员和支持者的巨大努力。感谢我们的团队成员，以及所有为社区贡献过拉取请求、测试和提供反馈或在论坛上为用户提供支持的人。我们一直感激您为使 Kodi 更加完善所做的贡献。

让我们来回顾一些您可以期待的变化：

### 主要特性

**FFmpeg 6**

Kodi 在很大程度上依赖于 FFmpeg 来为我们完成大量繁重的工作。在过去的 15 个月里，许多开发人员为了将项目升级到使用更新的 FFmpeg 版本做出了贡献，最初的目标是 FFmpeg 5，但后来更新到了 FFmpeg 6。我们始终非常感激所有我们依赖的开源软件项目的工作，使得 Kodi 可以呈现给您。

**DolbyVision 实时配置转换**

对于我们的 Android 用户，**quietvoid** 做出了极好的贡献，使用户可以将一些不太受支持的 DV 档案类型转换为更受支持的档案类型。

**macOS 现在使用 "本地" 窗口系统**

另一个经过多年努力的变化。许多年前由一些了不起的开发人员开始的。我们现在终于能够彻底移除 Kodi 中剩余的 SDL 库使用的痕迹。在 Apple macOS 平台上，我们现在使用本地实现来显示窗口。

**一个新平台：webOS**

一个新的平台现在可以本地运行 Kodi：一些了不起的开发人员已经逆向工程了大量的 webOS 媒体管道，成功地将 Kodi 移植到了 LG webOS 电视上。我们最新的团队成员之一，**sundermann**，已经帮助引导我们最新支持的平台，扩展了 Kodi 的覆盖范围，覆盖更多设备。

**游戏内播放器查看器**

在 v21 中，游戏的控制器配置稍微有所改善：增加了一个游戏内窗口，可以查看每个玩家的控制器当前连接到哪个游戏端口。

**幕后故事**

绝大多数变化都是 "幕后" 的，对用户来说是看不见的，但提高了 Kodi 的稳定性、性能和安全性：对 Python 和二进制插件进行了 API 更改，并带来了新的换肤功能；几乎所有 Kodi 平台上的 Kodi 依赖项都进行了更新；通过常规使用的代码静态分析工具进行修复，数据库迁移修复以实现更平滑的更新…… 还有很多其他内容。

有太多单独的变更以详细列出所有内容。如果您想了解更多，请回顾 Omega Alpha 到 RC2 的[发布公告](https://kodi.tv/blog/tag/release)并查看 GitHub 的变更日志[在此](https://github.com/xbmc/xbmc/compare/20.0-Nexus...21.0-Omega)。

时间不会停滞，我们已经为开发分支了 Kodi v22 "P*"。再次感谢您的持续支持，我们将继续努力让 Kodi 更加出色！

**寻求帮助！**

如果您遇到任何问题或发现任何未解决的错误，请在我们论坛的[总体支持](https://forum.kodi.tv/forumdisplay.php?fid=111)部分发帖。如果您对问题有解决方案，请通过提交到我们的[GitHub 主分支](https://github.com/xbmc/xbmc)的拉取请求来提供您的更改。我们也欢迎愿意在论坛回答问题或为维基撰写文章的用户。

**捐赠**

为了支持和赞赏 Kodi，请考虑[捐赠](https://kodi.tv/contribute/donate)或[购买商品](https://kodi.tv/store)，如 T 恤或树莓派外壳。 您的捐款将极大地帮助我们运作，包括支付运营费用、开发者的硬件和软件许可证以及法律费用，以及支付团队成员参加行业/自由/开源软件事件和我们的年度会议。

**可用性**

现在，Kodi 21.0 "Omega"正被推送到常规渠道，包括我们自己的[下载页面](https://kodi.tv/download)和各种应用商店/仓库，因此很快就会到达您的设备。 值得注意的是，这是一个很长一段时间以来的第一次，我们已经解决了所有已知的阻碍问题，因此所有支持的平台都将在各种应用商店处理完毕后更新。

如果您是安卓用户，并希望尽快更新到 Kodi 21.0，您可以通过侧载或加入[Google Play - Kodi 测试通道](https://play.google.com/apps/testing/org.xbmc.kodi)，这将更快地提交更新版本。 "开放"版本将在一两周内从测试通道推广。我们强烈建议使用替代皮肤的安卓用户在此期间与其皮肤开发者跟进，或在此时间之前禁用自动更新，因为并非所有皮肤目前都已更新。
