# The justice system’s role in non disclosure of Horizon material – The Justice Gap

> 来源：[https://www.thejusticegap.com/the-justice-systems-role-in-non-disclosure-of-horizon-material/](https://www.thejusticegap.com/the-justice-systems-role-in-non-disclosure-of-horizon-material/)

### 安德鲁·格林博士认为，与计算机生成材料相关的证据规则被“以一种威胁被告的方式误解”，而与Horizon计算机系统和富士通在销售有缺陷系统中的角色有关的材料被隐藏在秘密法庭听证会上，邮政局的律师辩称，披露这些材料不符合公众利益。

邮政局副邮政局长是英国刑事司法系统中发生过的最严重的司法错误的受害者之一，他们部分责怪与计算机生成证据的可证明性相关的法律。 1984年，《警察与刑事证据法》（PACE）引入了必须遵循的严格规则，以验证此类证据的准确性，然后才能在法庭审判中提交。 第69（1）（a）节要求‘没有理由相信由于计算机的不当使用而使陈述不准确’，第69（1）（b）节要求‘在所有相关时间内计算机都正常运行’。

1999年，这些条款被废除。媒体报道称，法律引入了一种假设，规定法院应如何考虑电子证据。1999年引入的新规则是根据法律委员会的建议，法院应假定计算机系统在没有明确证据反对的情况下操作正确（Karl Flinders，《计算机周刊》，2021年1月20日）。

亚历克斯·赫恩在《卫报》（2024年1月12日）中引述了电子证据专家和大律师斯蒂芬·梅森的话：“对于声称‘这台计算机有问题’的人来说，他们必须证明。即使提供信息的人是指控者。” 这是对法律的正确解释吗？ 英国皇家检察署的在线法律指导（总是权威和有用的来源）指出，自1999年法律变更以来，计算机证据必须遵循普通法规则，即假定生成证据记录的计算机在相关时间段内正常运行，因此这些记录是可作为真实证据提交的。

就像任何不受特殊准入规则约束的其他证据一样。但普通法准入规则包括反驳条款，英国检察署提醒我们。然而，如果提出相反的证据（“可反驳推定”），这种推定可以被反驳。在这种情况下，将由寻求提供计算机记录作为证据的一方向法院证明在材料时段内计算机正常工作（有关法律详细指导，请参阅《Archbold》9-11 9-15）。

换句话说，如果控方在刑事诉讼中引入了计算机生成的材料，辩方有权提出证据，证明该材料明显不同于类似情况下常规产生的材料，并且没有证据支持计算机生成的材料的准确性（例如个别交易记录）。被告（全部具有良好品行）有权在庭审中反驳欺诈或虚假会计的指控。

根据英国检察署（CPS）的披露手册，如果材料与

控方针对被告的案件性质，

+   所指控的犯罪的基本要素，

+   控方依赖的证据，

+   被告提出的任何解释[反驳]，无论是正式采访还是其他情况。

因此，“寻求提供计算机记录作为证据的一方”需要“向法院证明计算机当时正常运行”。而非被告。举证责任不应被颠倒。法律不应特权化计算机生成的证据，超过其他类型的证据。例如，如果有人因基于识别证据而被起诉犯有暴力犯罪，他们有权要求证明闭路电视设备正常运行，证明他们确实是图像中显示的人物，证人目击者没有犯错等等。

关于计算机生成证据法律的变更使邮政局更容易起诉支行长，但并未改变普通法证据规则。使某种类型的起诉证据成为可证明的并非与证明其真实性和准确性相同。

那么，出了什么问题？为什么代表这些被告的律师未能成功挑战现在普遍认为是完全错误的指控和严重不公正的问题？为什么可以挑战的证据未被挑战？为什么律师对这个问题如此沉默？

当*《计算机周刊》*声称“IT丑闻揭露了一项法律规则，使得邮局可以起诉无辜者”时，他们是正确的——起诉变得更容易，因为不必证明计算机的正确功能就可以根据计算机生成的证据起诉嫌疑人。但是他们确实必须提供证据表明所涉系统的正常运行。

所以，记者和律师是在指责真正有错的规则吗——或者存在其他更令人不安的规则，隐藏了这些滥用？因此允许并鼓励不公正的起诉？

***披露基础*** 当警方调查案件时，他们有法律义务跟进所有合理的调查线索，无论这些线索指向或背离嫌疑人的有罪。他们拥有或获取的任何证据（称为材料），在一般情况下，必须在嫌疑人被指控犯罪并成为被告时以及在刑事诉讼结束之前的任何时候披露给嫌疑人。

披露官应该：

+   复制该材料以供检察官使用，包括：

+   在披露官的意见下，符合披露测试的材料（第7.3和7.4款的法典）……

+   任何属于适用反驳推定类别（法典第7.5款）的材料，和；

+   允许检察官检查材料。

披露官的认证

操作披露过程时，调查官必须确保已保留的所有相关材料要么向披露官揭示，要么在特殊情况下，直接向检察官高度敏感地安排揭示。

披露官员在披露过程中必须提供不同的认证，以覆盖：

+   揭示所有相关保留材料；

+   确认没有材料属于适用反驳推定的类别（如适用的话）；

+   判断材料是否符合披露测试；

+   在作为持续义务的一部分之后，判断材料是否符合披露测试。

实际的披露程序是复杂的。实际上，这完全受检察官的控制。该程序的一部分可以包括向某些材料授予公共利益豁免，这是披露程序的秘密部分，在某些案件中使用。这个程序被多频繁地使用？我们不知道。这是秘密的。（请参阅下文关于标准警察披露职责的说明。尽管邮局的起诉是私人起诉，披露要求与适用于警察和CPS的规定相同——所有邮局调查人员以前都是警察，知道刑事司法系统的规则和程序。）

***公共利益豁免（PII）***

根据刑事诉讼规则（CPR；此处引用的是最新的2020年版本），规定了授予个人身份信息（PII）以证据的规定，这可能会导致隐藏相关证据材料并排除其出现在审判中的效果。

刑事诉讼规则具有次级立法的地位，是议会辩论和表决通过的法令的次等替代品。任何更新都会向议会提出供成员检阅和评论。它们完整地在线发布。尚未熟悉此法律的读者可能想知道：

-   （1）这一程序规则的首要目标是公正处理刑事案件。

-   （2）公正处理刑事案件包括——

-   （a）无罪释放无辜者和定罪有罪者；

-   （b）公正对待控方和辩护方；

-   （c）承认被告的权利，特别是欧洲人权公约第6条下的权利。

一切顺利。CPR第15.3节涵盖了授予证据个人身份信息的程序。

本规则适用于在没有法庭命令的情况下，检察官必须披露材料的情况；并且检察官希望法庭决定是否披露此类材料符合公众利益。

‘必须披露材料’指的是有益于辩护的材料，因此应在正常披露规则下披露。当控方不希望辩方了解此类材料时，就会使用此程序，因为这样一来，它就会为公众所知。控方希望法庭裁定控方无需披露相关材料。

申请授予个人身份信息给证据的目的是在听证会上决定（除非法庭另有指示）（如果法庭有所指示）可以在被告缺席的情况下完全或部分进行。

未经法庭另行指示，法庭人员不得通知除了检察官以外的任何人参加根据本规则提出的申请的听证会……或者法庭对申请的决定。

*总结一下：* 个人可辨认信息（PII）可能授予由检方持有的材料，这些材料将有利于辩护案件，而辩方对此一无所知——不知道材料的存在，不知道有关其进行的法庭听证会，也无法对裁决进行挑战；也就是说，完全秘密进行。主持PII听证会的法官必须完全根据检方提供的证据做出决定。他们可以考虑辩方陈述，但在陈述被告案情时，辩护律师对将提交给PII申请的法庭的秘密证据一无所知，也不知道可能会进行PII听证会的可能性。检察官和调查人员可以自由地向法官陈述任何他们喜欢的事情，而无需担心反驳。他们可以向法官保证Horizon程序是“强大”的——在他们的用法中，这个词似乎是不可错误的同义词——它经过了试验和审查，并且由邮政或富士通员工无法访问其持有的数据。

有关听证会的记录和决定保存在法院大楼的某个安全锁定的柜子中。在哪里？好吧，这是个秘密。谁可以看到记录？我们不知道。但是上诉法院刑事部门（CACD）可以看到它——只要法院遵循PII规则。

由于PII听证会的秘密性及其所有内容均由法院决定，违反保密规则可能构成藐视法庭，这是受到严厉制裁的。谁受到法院规则的约束？CPR说：“与刑事案件的任何方式有关的人员都是其行为的参与者…”。因此，这包括辩护律师和被告本人。但不包括我们其他人。

何种材料受PII裁决的约束？根据《1996年刑事诉讼和调查法》（第23(1)条）2021版颁布的实施准则规定：

“[检察官]相信其披露会对重要公共利益构成严重损害的任何材料，以及这种信念的原因。

这些“敏感材料”通常包括“涉及国家安全的材料，... 线人、卧底警察或证人的身份或活动，或者其他向警方提供信息可能会因身份泄露而处于危险中的人员。”

但是还存在其他类别，可以直接涵盖Horizon材料。直接或间接揭示警察在刑事调查过程中依赖的技术和方法的材料。在这一标题下可以包括他们如何收集和分析（或未能分析）Horizon材料。

其披露可能促使其他犯罪的材料或阻碍犯罪的预防和侦查。

如果这些邮局分局长确切地知道Horizon是如何运作的，他们可能会利用这样的知识来犯罪，或者掩盖他们已经在犯的罪行，邮政局及其调查人员的观点如此。如果我们的假设是正确的，并且确实获得了PII裁定，那么这可能是其主要原因。

不披露Horizon材料的另一个原因可能是商业保密性（对富士通来说价值连城，他们为计算机系统及其维护支付了大笔费用，并且如果他们的无能暴露出来，他们可能会失去合同。政府也非常重视使用富士通的计算机系统，这被视为对重要公共利益构成“真正严重损害风险”。英国政府最高层（首相托尼·布莱尔）对将Horizon引入使用表示关注，并向富士通支付了巨额款项。（另见罗文娜·梅森和拉塞尔·斯科特，《Kemi Badenoch在与富士通在达沃斯会晤时未提及Horizon丑闻》，2024年2月11日《观察家报》：“独家报道：信息自由法请求显示（商务大臣）受到公务员的“风险”警告，但集中精力于科技公司在英国的投资。”）

但是CPR也明确规定了

法院只能在确信已经充分考虑了被告享有公平审判权的情况下决定（即授予）申请。

被告在未披露Horizon材料的情况下是否能够公平审判？法官将裁定，由于Horizon材料不会对辩护案件产生任何影响，因为他们已经得到坚定的保证Horizon是“健全的”。检察官将提出这样的论点，即辩护方有责任证明否定的情况，而辩护方并未提供任何证据表明Horizon存在重大和相关的故障。辩护方被排除在PII听证会之外，因此他们无法反驳这一主张。因此，这一观点得到了广泛接受，并且邮政局一再拒绝披露Horizon材料。

我们如何推断PII听证会已经举行？

+   1\. 邮政局的论点逻辑是，许多邮局分局长都不诚实，这意味着披露Horizon的详细信息将有助于未来犯罪的发生，这可能会使犯罪行为更难被检测到。PII裁定将阻止欺诈和盗窃，邮政局的律师们将这样辩称。

+   2\. 如果Horizon表现出揭露欺诈和盗窃的迹象，那么它仍然是一个有价值的系统。PII裁定将保护富士通的有价值资产，也保护英国国家及其与日本的关系。

+   3\. 审判邮政局案件的法官似乎对Horizon的“坚固性”（或不可靠性）比在公开法庭中声明的更有信心，这是一个问题，他们可以在不用担心可能在公开法庭中提出的辩护挑战的情况下得到保证。

+   4\. 如果审判这些案件的法官似乎了解更多有关个别被告（例如用于判决目的）的信息，而这些信息未在公开法庭或已达成的证据中提到，则这应归因于在被告不在场时提供的信息，例如PII听证会（需要分析判决意见以确认此命题）。

+   5\. 辩护律师似乎普遍对PII保密材料的问题保持沉默，这表明他们受到PII程序所需保密的约束。

+   6\. 上诉法院可能可以处理推翻错误判决的负担，考虑到所有案件的相似性以及起诉的崩溃。但是上诉程序可能会揭露PII听证会被滥用来掩盖Horizon系统的缺陷和未能解决这些问题。拟议的特殊立法将消除向国家、邮政局和富士通暴露尴尬的风险。这将解决法院被迫承认其他法院隐瞒的证据所引起的法律问题。毫无疑问，立法解决方案将采纳“程序有误”的解释，从而排除了调查其他解释的可能性，例如ITV剧集中特色的而非虚构的Horizon数据篡改问题。

+   7\. 也许更重要的是，许多（可能是所有）嫌疑人在其辩护中提出了关于Horizon的问题。这些是关于明显系统缺陷和有机会更改Horizon数据的可能性的合理解释，这些数据可以被有权访问Horizon数据的邮政局和富士通的工作人员操控。在遵循CPIA 1996及其相关实践守则规定的调查中，这些解释构成了合理的调查线索，调查人员有法律义务要跟进。邮政局自己的调查人员是前警察官员，他们了解刑事调查员的责任。

私人侦探或许没有费心调查被提出的辩护，或者他们遵循了实践守则并对嫌疑人的反驳证据进行了调查。在后一种情况下，他们本应获得相关材料，并应向被告披露，但因PII裁决而免除了披露义务。

[披露官员报告MG6E](http://www.cps.gov.uk/legal-guidance/disclosure-manual-chapter-10-disclosure-officers-report)

更新日期：2021年10月21日|法律指导。MG6E表格是由警方调查员填写的标准表格，用于准备所有刑事诉讼。填写完毕的表格将发送给CPS检察官。以下文字来源于CPS网站，但已添加所有强调部分。

披露官员应使用MG6E表格向检察官引起注意任何可能合理认为可能削弱对被告的起诉案件或有助于被告案件的材料。这同样适用于敏感材料。

披露官员还应在MG6E表格上解释（通过参考日程表上相关项目的编号）为何他或她持有这种观点。MG6E表格应根据国家披露标准填写完成。

任何支持或与在审讯或提起诉前提出的辩护一致的材料，或者从起诉文件中显而易见的材料，都应包括在内。还包括任何指向被告的信息，例如可能的不在场证明。单独查看的材料可能不满足测试；然而，几项材料一起可能有这种效果。在应用披露测试时，披露官员和检察官不应对辩护的优点进行评判，并且任何疑问都应有利于披露的解决。

…披露官员应在必要时与检察官协商，帮助确定可能需要披露的材料，并且在披露官员对可能满足披露测试的材料存有任何疑问时，必须明确向检察官指出这些材料.…

[被列为‘敏感’类别的材料，可能被视为需要公共利益豁免]

为帮助检察官决定如何处理调查员认为可能满足披露测试的敏感材料，调查员应提供详细信息，涉及以下问题：

+   材料被视为敏感的原因；

+   所谓的材料敏感程度，换句话说，为何认为披露将对重要公共利益造成实际损害风险；

+   揭示给辩护7的后果

1\. 材料本身

2\. 材料的类别

3\. 可能会提出申请的事实

+   材料对审判问题的显著重要性；

+   任何第三方介入将材料带给警方的情况；

+   若材料可能成为披露命令的对象，警方对继续起诉的看法，以及；

+   不论材料或信息是否可能在不损害其敏感性的情况下披露，无论是经过删节还是以其他形式。

为了帮助确定上述敏感程度，应考虑到可能会直接或间接通过逐步或累积的伤害损害公共利益的事实。直接伤害的例子包括：...

+   •暴露一种秘密调查技术。

逐步或累积伤害的例子包括：

+   暴露一种调查技术，使犯罪社区更加警觉，从而更能够避免被发现；

+   揭露出于信任或情报目的而提供的材料，可能会使材料的来源或其他人在未来不愿合作；和

+   活跃地否认在当前案件中使用过某来源，导致未来使用该来源的案件无法否认，因此暗示了使用来源的情况。皇冠既不应确认也不应否认使用某来源。

***将材料揭示给检察官*** 将材料揭示给检察官并不意味着自动向辩护揭示。只有当材料满足披露测试时，检察官才会向辩护揭示材料。如果材料敏感，并且满足披露测试，则检察官将在与警方磋商后披露材料，或者向法院申请裁定是否需要披露公共利益，或撤销起诉。

披露官应：

将这些材料复制给检察官以包括：

+   披露官认为符合披露测试（第7.3和7.4段的准则）的材料...

+   任何符合可反驳推定的类别的材料（准则第7.5段），以及；

+   允许检察官检查材料。

披露官的认证

主办官必须确保已保存的所有相关材料要么向披露官揭示，要么在特殊情况下，根据高度敏感的时间表直接向检察官披露。如果披露官不确定是否已经向其披露了所有相关保留的材料，则应向主办官询问以解决此事。

披露官在披露过程中必须提供不同的认证，以覆盖：

+   揭示所有相关保留的材料；

+   确认没有材料属于适用于可反驳推定的类别（如果适用）；

+   材料是否满足披露测试；和

+   材料是否在作为持续职责的一部分的辩护声明后满足披露测试。
