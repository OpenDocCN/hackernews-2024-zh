# 稳定扩散 3 API 现已推出 — Stability AI

> 来源：[https://stability.ai/news/stable-diffusion-3-api](https://stability.ai/news/stable-diffusion-3-api)

### **性能**

我们与市场上速度最快、最可靠的 API 平台 [Fireworks AI](https://fireworks.ai/models) 合作，以提供稳定扩散 3 和稳定扩散 3 Turbo。

通过 Fireworks AI，我们还将提供一个企业级 API 解决方案，确保 99.9% 的服务可用性，这对于依赖我们开发平台进行最重要生成 AI 工作负载的企业至关重要。我们的客户可以放心地利用我们的服务进行他们的关键业务操作。

### **安全**

我们坚信安全、负责任的 AI 实践。这意味着我们已经并将继续采取合理措施，防止不良行为者滥用稳定扩散 3。安全工作从我们开始训练模型时就开始，并贯穿于测试、评估和部署的整个过程。通过与研究人员、专家和我们的社区持续合作，我们期望在继续改进模型的同时，以诚信创新。

稳定扩散 3 旨在提供灵活的解决方案，帮助个人、开发者和企业释放他们的创造力，与我们激活人类潜力的使命相一致。
