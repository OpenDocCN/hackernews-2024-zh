# Ubuntu 24.04（诺布尔袋獴）

> 来源：[https://releases.ubuntu.com/noble/](https://releases.ubuntu.com/noble/)

## 选择一个镜像

Ubuntu分发在下面描述的三种镜像中。

### 桌面镜像

桌面镜像允许您在不更改计算机的情况下尝试Ubuntu，并且您可以选择稍后永久安装它。这种类型的镜像是大多数人想要使用的。安装此镜像需要至少1024MiB的RAM。

[64位PC（AMD64）桌面镜像](ubuntu-24.04-desktop-amd64.iso)

如果您的计算机基于AMD64或EM64T架构（例如Athlon64、Opteron、EM64T Xeon、Core 2），请选择此项。如果您不确定，请选择此项。

### 服务器安装镜像

服务器安装镜像允许您永久安装Ubuntu以供作为服务器使用。它不会安装图形用户界面。

[64位PC（AMD64）服务器安装镜像](ubuntu-24.04-live-server-amd64.iso)

如果您的计算机基于AMD64或EM64T架构（例如Athlon64、Opteron、EM64T Xeon、Core 2），请选择此项。如果您不确定，请选择此项。

### 网络引导tarball

网络引导tarball包含启动Ubuntu安装程序所需的文件。

[64位PC（AMD64）网络引导tarball](ubuntu-24.04-netboot-amd64.tar.gz)

如果您的计算机基于AMD64或EM64T架构（例如Athlon64、Opteron、EM64T Xeon、Core 2），请选择此项。如果您不确定，请选择此项。

包括[BitTorrent](https://help.ubuntu.com/community/BitTorrent)文件在内的所有可用文件的完整列表，请参见下文。

如果您需要帮助将这些镜像刻录到光盘，请参阅[Image Burning Guide](https://help.ubuntu.com/community/BurningIsoHowto)。
