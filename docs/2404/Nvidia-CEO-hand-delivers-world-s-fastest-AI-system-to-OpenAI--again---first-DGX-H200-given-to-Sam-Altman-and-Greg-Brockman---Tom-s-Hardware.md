# Nvidia CEO将世界最快的AI系统再次交付给OpenAI — 首个DGX H200交付给Sam Altman和Greg Brockman | Tom's Hardware

> 来源：[https://www.tomshardware.com/tech-industry/artificial-intelligence/nvidia-ceo-hand-delivers-worlds-fastest-ai-system-to-openai-again-first-dgx-h200-given-to-sam-altman-and-greg-brockman](https://www.tomshardware.com/tech-industry/artificial-intelligence/nvidia-ceo-hand-delivers-worlds-fastest-ai-system-to-openai-again-first-dgx-h200-given-to-sam-altman-and-greg-brockman)

Nvidia CEO黄仁勋承担了一个新的兼职角色，作为快递员，将世界上首台[DGX H200](https://www.tomshardware.com/news/nvidia-unveils-dgx-gh200-supercomputer-and-mgx-systems-grace-hopper-superchips-in-production)送到了OpenAI的Sam Altman和Greg Brockman手中。这三人在[Twitter/X上分享了他们的会面照片，](https://twitter.com/gdb/status/1783234941842518414)以纪念黄仁勋成功将DGX H200交付给OpenAI。

DGX H200是Nvidia推出的全新、速度极快的基于GPU的服务器，内置强大的[H200张量核](https://www.tomshardware.com/tag/enterprise)-级服务器外壳。基于霍珀架构的H200 GPU代表了与其小弟H100相比的一代跨越，新卡片现在拥有141GB内存，速度为4.8TB/s，而H100只有80GB内存，速度为3.3TB/s。Nvidia将其H200称为“用于超级AI工作负载的全球最强GPU”，这一说法几乎无人能辩驳。

黄仁勋在超级计算机上签署了题词“推动AI、计算和人类的进步”。这个签名和合影让人想起2016年的一个非常相似的场景，当时黄仁勋为OpenAI送去了[一个非常类似的交付](https://www.tomshardware.com/tech-industry/artificial-intelligence/elon-musk-reminisces-about-the-time-jensen-huang-donated-a-dgx-1-to-openai-shares-photo-gallery) — 第一台DGX-1服务器交付给了激动的埃隆·马斯克。当马斯克还是OpenAI的自豪成员和联合创始人时，他高兴地接收了DGX-1，也是由黄仁勋签名，上面写着“致计算和人类未来的祝酒”。DGX-1的赠送被埃隆和许多OpenAI团队成员视为加速研究几周的福利，而性能大幅提升至DGX H200可能会产生类似的影响。

但那些是更快乐的时光；如今，马斯克正在[起诉 OpenAI](https://www.tomshardware.com/tech-industry/artificial-intelligence/elon-musk-sues-openai-alleging-breaches-of-the-founding-agreement)。他声称 OpenAI 违反了创始协议，将其从非营利组织转变为侵略性营利公司，并加剧了人工通用智能（AGI）的潜在危险，称 AGI 的前景是“对人类的严重威胁”。今天，马斯克和黄之间似乎没有什么深情，这对喜欢皮夹克/Nvidia CEO 和那位让 Twitter 沦陷的人的粉丝来说是一种温暖的解脱。时间将告诉我们，马斯克是否会为训练他的宠物 Grok 的[下一代](https://www.tomshardware.com/tech-industry/artificial-intelligence/elon-musk-says-the-next-generation-grok-3-model-will-require-100000-nvidia-h100-gpus-to-train)获得自己的免费 AI 超级计算机。
