# Lawmakers Are Kicking Warrantless Wiretapping Into Overdrive

> 来源：[https://gizmodo.com/lawmakers-are-kicking-warrantless-wiretapping-into-over-1851417659](https://gizmodo.com/lawmakers-are-kicking-warrantless-wiretapping-into-over-1851417659)

本周，国会预计将对一项法案进行投票，该法案将更新关键的联邦监视权力。法案的支持者将其包装为一项改革法案，但隐私倡导者表示，实际上它将大大扩展政府的监视能力。

H.R. 7888，或称[情报改革与美国安全法案](https://rules.house.gov/bill/118/hr-PIH-RISA)，将更新美国情报社区广泛称为第702条的关键监视权力。第702条于2008年作为[FISA修正法案](https://www.eff.org/702-spying)的一部分而创建，授予政府与国外人士通信的美国人进行无需令状的监视的法律授权。该法允许美国情报社区（尤其是[NSA](https://www.wired.com/story/section-702-nsa-abuses-reauthorization/)）在[政府法院](https://www.progressivecaucuscenter.org/the-foreign-intelligence-surveillance-act-and-the-warrantless-collection-of-personal-data)认为出于国家安全原因必要时，利用电子通信服务提供商的强制协助进行“有针对性的监视”。

尽管政府表示这种监视活动专门针对非美国人，但是根据[Brennan Center](https://www.brennancenter.org/our-work/research-reports/pclob-report-reveals-new-abuses-fisa-section-702)，该法律已经多次允许大量美国人的通信陷入政府的网络。作为与纽约大学相关联的法律非营利组织，Brennan Center指出，第702条款已成为“常用的国内间谍工具”，用于每年进行数十万次无需令状的“后门”搜索，以获取美国人的私人通信。

在过去，政府大多通过窃取[在线通信提供商](https://cdt.org/wp-content/uploads/2017/02/Section-702.pdf)的信息流来进行监视。然而，现在，新的更新将大大扩展政府委托美国企业进行通信监视的能力。问题在于，法案中模糊的措辞使人难以理解哪些组织将被要求为政府进行监视，哪些将被豁免。

“这个定义非常广泛，”ACLU的高级联邦政策顾问Kia Hamadanchy告诉Gizmodo。“去年12月发布的版本存在着巨大的问题。”他说，这周发布的新版本现在免除了“某些企业——比如酒店”。然而，这些豁免之所以必要，似乎表明法案法律范围过于广泛。

在关于立法的最新描述中，法律专家Marc Zwillinger在一篇文章中描述了提议法律的广泛范围：[一篇描述](https://www.zwillgen.com/law-enforcement/fisa-702-reauthorization-amendments-second-time-not-charm/)。

> 新修正案将允许政府强制要求广泛范围的其他实体和个人，在FISA 702条款下进行监视，尽管有这些例外情况。从新定义的广泛性来看，起草者感到有必要排除普通场所，如老年中心、酒店和咖啡店，这表明修正案的范围显而易见。但除了这些特定例外情况外，新定义的范围将覆盖它们以及许多未受特定豁免的企业。
> 
> 即使有了这些特定例外情况，定义也可能包括例如仅具有对其物理空间内通信设备访问权限的数据中心和商业房东拥有的建筑物的所有者和运营者。还可能包括其他具有对这类设施和设备访问权限的人，包括投递人员、清洁承包商和公用事业提供商。

换句话说，政府可以征用私营部门的不明确数量，使其充当自己的个人情报资产。的确，Hamadanchy列举了一些可能属于这一类别的公司，它们数量众多：“商业房东、杂货店、理发店、五金店、健身中心”，或者如他所说，“任何有WiFi连接”的人。

尽管不清楚这些公司一旦卷入间谍工作将被要求做什么。例如，对于商业房东来说，如果要求他们监视政府的目标，这种业主可能会扮演什么角色尚不清楚。不过，可以进行推测。“我认为他们会去找”房东“，然后侵入该建筑的WiFi网络，”Hamadanchy提出。

如果所有这些内容对你来说似乎并不令人感到轻微的恐惧，那可能应该让你感到恐惧。“这是一部糟糕的法案，”Hamadanchy说。“我们不应该通过糟糕的法案。我们应该花时间做好。”他说，参议院规则可能允许政策进程一直拖到周日，并且他预计可能会有人试图剥夺法案中更具争议性的政策。

尽管如此，拜登政府一直在努力游说通过这项法案。周三，白宫国家安全顾问杰克·沙利文 [发布了一份声明](https://www.whitehouse.gov/briefing-room/statements-releases/2024/04/17/statement-by-national-security-advisor-jake-sullivan-on-the-upcoming-vote-on-reauthorization-and-reform-of-fisa-section-702/)，声称新立法将确保“美国政府拥有保护国家安全所需的工具，同时大幅增强隐私和公民自由的保护。”随后，他呼吁参议院“迅速将法案送到总统的办公桌上”。
