# deco.cx: 开源、无头前端平台

> 来源：[https://deco.cx](https://deco.cx)

## 极速UI组件库

加速您的构建过程。借助我们直观的可视化编辑器，采用预构建的通用适配UI组件，实现快速设计和部署。

## 原生设计系统构建器

瞬间更新您网站的样式。轻松调整颜色、按钮样式和排版，通过几次点击创建独特的品牌外观。

## 实时拖放网站编辑器

业务用户和营销人员可以实时协作，通过无代码CMS重复使用品牌化组件，创建令人惊叹的数字体验。

## 浏览器中的完全代码控制

利用deco.cx为Preact、Tailwind和Typescript定制的集成代码编辑器，加速网页开发。直接在浏览器中编辑，与任何Git同步，并利用我们的AI合作伙伴实现无缝编码。

## TypeScript优先内容建模

我们基于类型的CMS直接从您的代码生成，提供了无限的个性化可能性，带来流畅的开发体验。

## 一键安装应用和集成

通过我们的deco Hub轻松增强您的站点。访问与任何后端API连接的大量主题、应用和模板库。

## 为特定受众个性化内容

超越A/B测试。创建基于数据驱动洞察力支持的实验、活动和定向体验。

## 深入的Web分析与可观察性

解锁高级受众洞察和全面系统可观察性，实现最佳性能。
