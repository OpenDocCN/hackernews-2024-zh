# YouTube在桌面上有了全新的设计，而每个人都对此感到愤怒

> 来源：[https://www.androidpolice.com/youtube-new-desktop-design-nobody-likes/](https://www.androidpolice.com/youtube-new-desktop-design-nobody-likes/)

### 摘要

+   YouTube的新视频播放器布局因设计狭窄和注重内容参与而引发用户的争议。

+   全新的网络用户界面将视频标题、上传者信息和评论移到右侧的小列中，而视频推荐则占据了主要位置。

+   面对压倒性的负面用户反馈，YouTube可能需要重新思考这一设计。

YouTube是[最佳娱乐应用程序之一](https://www.androidpolice.com/best-entertainment-apps-android/)，特别是如果你寻找用户生成的视频内容而不是最新的好莱坞大片。然而，在内容呈现部门，包括在视频开始前播放[马拉松广告片段](https://www.androidpolice.com/youtube-30-second-unskippable-tv-ads/)和将移动优先的[短片内容推送到次优平台](https://www.androidpolice.com/youtube-shorts-tv-launch/)，它做出了几个值得质疑的决定。其最新的决定是对YouTube播放器周围元素进行全面改造，网络用户并不高兴。

[](/youtube-testing-new-shorts-carousel-placement-web/)相关在网络上的视频推荐边栏中引入新的走廊

最近，我们发现YouTube在网络上进行了[新的短片货架放置的a/b测试](https://www.androidpolice.com/youtube-testing-new-shorts-carousel-placement-web/)，放在视频播放器右侧的建议视频列中。然而，现在，整个视频标题、上传者信息和评论都向右移动，与视频推荐交换位置。YouTube将所有椭圆形按钮、文字和评论压缩到右侧纤细的列中，而我们刚提到的推荐视频和短片货架现在则在播放器下方占据了显著位置。

因此，即使像视频字幕和描述这样专注于无障碍功能的特性也被挤在那么小的空间中。在播放器控制中切换到剧院模式也没用 —— 它只是将视频信息和评论列移到播放器下方，但仍然与推荐内容右对齐，占据了大部分屏幕空间。

## YouTube观众并不满意

*全新的网络用户界面*

新的Web用户界面与旧界面相去甚远，Twitter上的看法出现了分歧。一方面，[一些用户赞赏](https://x.com/CherriFireLive/status/1778090193871917233)能够在观看视频时阅读并参与评论，而无需向下滚动和隐藏视频播放器。另一方面，一些用户发现这种变化[令人不适](https://www.reddit.com/r/youtube/comments/1c0ayew/why_just_why/)，感觉过于密集，两侧有大量的[浪费屏幕空间](https://x.com/chkdg8/status/1777935840766722367)。

新的用户界面与智能电视应用程序的[最近更新](https://www.androidpolice.com/youtube-app-video-streaming-upgrade/)非常相似，但我们认为新设计也代表了公司的意识形态转变，优先考虑内容参与度和用户留存，而不是通过评论进行社区互动。我们联系了YouTube寻求评论，得知新布局是一个有限测试，旨在使在桌面观看视频时更容易阅读评论。

YouTube帮助文档中关于[测试功能和实验](https://support.google.com/youtube/thread/18138167/youtube-test-features-and-experiments?hl=en#:~:text=different%20layouts%20to%20the%20desktop%20watch%20experience)揭示了自今年2月20日以来YouTube尝试过的众多布局之一。幸运的是，这仍然是一个测试，鉴于Twitter用户的压倒性反馈，UI设计师可能需要重新考虑这个布局。
