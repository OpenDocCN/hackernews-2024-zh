# DongleHider+是一款精巧的新Framework笔记本扩展卡，可隐藏多达三个小型设备 | Tom's Hardware

> 来源：[https://www.tomshardware.com/laptops/donglehider-is-a-cunning-new-framework-laptop-expansion-card-for-hiding-up-to-three-dongles](https://www.tomshardware.com/laptops/donglehider-is-a-cunning-new-framework-laptop-expansion-card-for-hiding-up-to-three-dongles)

开发者和制造商LeoDJ分享了他的Framework笔记本扩展卡设计，这款[DongleHider+](https://community.frame.work/t/donglehider-multiple-dongles-and-a-normal-usb-a-socket/47187)肯定会受到模块化PC机主的广泛赞赏。它可以容纳多达三个小型设备，并提供一个外置USB Type-A连接器。

使用这张卡片确实需要一些DIY技能，因为它并非出售。要成功隐藏您的Logitech Unifying型号设备，需要先移除其外壳。

安装后与框架完美贴合，并提供一个实用的USB Type-A端口。 （图片来源：LeoDJ）

LeoDJ不喜欢他的[无线鼠标](https://www.tomshardware.com/best-picks/best-wireless-mouse)的蓝牙延迟，因此希望将Logitech Unifying型号接收器永久插入笔记本电脑，而不影响设备的整洁外观。DongleHider+项目出色地实现了这些目标，并通过为多达三个（小型）设备提供连接以及提供一个便捷的[USB Type-A](https://www.tomshardware.com/news/usb-3-2-explained)端口，达到了更多效果。

想要追随LeoDJ脚步的人可以在Framework社区帖子中找到足够的指导。从[社区帖子](https://community.frame.work/t/donglehider-multiple-dongles-and-a-normal-usb-a-socket/47187)，你可以前往制造者的[GitHub仓库](https://github.com/LeoDJ/FW-EC-DongleHiderPlus?tab=readme-ov-file)和[Mastodon主题](https://chaos.social/@LeoDJ/112108367751183162)，这些地方都包含了制作PCB和Framework笔记本扩展卡的有用源文件。特别是Mastodon主题，提供了图解的组装指南。

从左至右：一个标准USB A扩展卡，一个裸露的DongleHider+ PCB，两个组装好的DongleHider+ PCB，一个标准的Logitech Unifying型号接收器，一个完整安装了设备的DongleHider+扩展卡，还有一个3D打印的外壳。（图片来源：LeoDJ）

有些人可能对拆卸和[焊接](https://www.tomshardware.com/best-picks/best-soldering-irons)他们想要永久安装到DongleHider+的设备感到紧张，但LeoDJ表示在二手市场上它们相当便宜。提供了拆卸Logitech Unifying Gen1和Gen2接收器的清晰指示。此外，将设备焊接到PCB上不应太具挑战性，尤其是如果您只打算内部安装一个设备。

准备焊接一个设备到位。（图片来源：LeoDJ）

LeoDJ 提供了几个兼容性说明：使用 [Framework 13](https://www.tomshardware.com/reviews/framework-laptop-13-intel-2023)（AMD 版本）和 16（AMD 版本）时，他注意到当连接到支持 USB4 的扩展卡槽时，外部端口连接了 USB 3 设备，第二个内部 USB 适配器就会停止工作。
