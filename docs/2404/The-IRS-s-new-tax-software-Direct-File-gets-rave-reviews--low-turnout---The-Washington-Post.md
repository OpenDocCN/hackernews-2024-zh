# 美国国内税务局的新税务软件 Direct File 受到了好评，但参与率较低 - 华盛顿邮报

> 来源：[https://www.washingtonpost.com/business/2024/04/15/irs-direct-file-reviews/](https://www.washingtonpost.com/business/2024/04/15/irs-direct-file-reviews/)

拜登政府在周一宣布，他们已经达到了一个小目标，至少有10万纳税人通过美国国内税务局的新税务软件 Direct File 提交了申报 —— 这是商业税务准备服务的一种替代方案。

尽管政府将直接文件（Direct File）宣传为一个小规模试点项目，但它仍代表了数十年来在税务申报领域最重要的实验之一 —— 一个免费平台，让美国人可以直接向政府在线申报税务。
