# 焦虑和拖延：它们之间的关系及如何处理——解决拖延

> 来源：[https://solvingprocrastination.com/anxiety/](https://solvingprocrastination.com/anxiety/)

*焦虑*是人们在各种情况下经历的一种[担心](https://web.archive.org/web/20211221211118/https:/www.nhs.uk/mental-health/conditions/generalised-anxiety-disorder/overview/)和不安感，比如[在](https://web.archive.org/web/20211222155737/https:/www.nimh.nih.gov/health/topics/anxiety-disorders)参加考试、面试工作或做重大决定之前。偶尔焦虑是生活中的一部分，尽管一些人[患有](https://web.archive.org/web/20211222155803/https:/www.mayoclinic.org/diseases-conditions/anxiety/symptoms-causes/syc-20350961)焦虑症，通常[涉及](https://web.archive.org/web/20211222155737/https:/www.nimh.nih.gov/health/topics/anxiety-disorders)比正常更强烈、持续和频繁的焦虑发作。

焦虑症——无论是一般形式还是极端形式——与[*拖延*](https://solvingprocrastination.com/procrastination/)倾向相关，即不必要地推迟决定或行动。例如，如果一个学生对一项家庭作业感到焦虑，他们可能会推迟开始工作，以[避免处理](https://doi.org/10.1111/spc3.12011)与这项作业相关的消极情绪。

焦虑和拖延之间的关联复杂而且可能具有严重的影响，因此理解它至关重要。因此，在以下文章中，您将更多了解焦虑和拖延之间的关联，并看看在实践中您可以做些什么。

## 焦虑和拖延之间的联系

焦虑可能导致人们拖延：

+   增加对任务的[厌恶感](https://doi.org/10.1080/0260293042000160384)，例如通过增加他们与任务[相关联的](https://doi.org/10.1002/pits.10045)负面情绪。

+   使他们更[担心](https://doi.org/10.1023/A:1026474715384)一项任务，这可能会导致问题，如过度思考、感到不知所措，并且不确定如何开始，尤其是面对[不确定性](https://doi.org/10.1016/j.janxdis.2022.102633)等问题。

+   例如，通过让他们[反复思索](https://doi.org/10.1007/s10942-017-0271-5)，例如让他们专注于过去的痛苦回忆而让他们分心，远离未来。

因此，人们在焦虑时往往更容易拖延，或者拖延那些让他们感到焦虑的任务。即使因为这个原因拖延会导致更多的焦虑和负面情绪，而不是完成任务，这种行为也可能导致其他各种问题，例如表现不佳、错失机会以及增加人际冲突。

此外，焦虑不仅可能是拖延的潜在原因，还可能是其潜在后果，这意味着拖延可能使人们感到焦虑或加剧现有的焦虑。这意味着人们可能会进入焦虑-拖延循环，其中焦虑促使人们拖延，进而使他们更加焦虑，导致继续拖延，如此循环往复。

然而，关于焦虑与拖延之间的关系，也有一些重要的警告：

+   **焦虑与拖延之间的关系是复杂的**。这种复杂性意味着，例如，各种因素如人们的*自我效能*和*正念*可以调节焦虑对拖延的影响（例如，高自我效能可以作为一种抵御焦虑影响拖延的韧性因素）。此外，还有各种类型的拖延（例如焦虑性和享乐主义性）以及焦虑的各种形式（例如作为稳定的人格特征和暂时的心理状态），它们可以以不同的方式相关联。

+   **焦虑并不总是导致拖延**。具体来说，一些人尽管感到焦虑，但仍能及时行动，例如如果他们的焦虑程度足够轻微，而动机足够强烈的话。此外，也有一些情况下，焦虑并不会增加拖延的可能性，甚至会降低这种可能性，例如让人们对某项任务感到担忧，从而更有可能早日开始完成。这也意味着，并非每个焦虑的人都会拖延。

+   **除了焦虑之外，其他事物也会导致拖延。** 例如，[其他导致拖延的原因](https://solvingprocrastination.com/why-people-procrastinate/)包括抽象目标、远期奖励、[抑郁症](https://solvingprocrastination.com/depression/)、[完美主义](https://solvingprocrastination.com/perfectionism/)和[ADHD](https://solvingprocrastination.com/adhd/)。这意味着，并非所有拖延的人一定焦虑，而且即使焦虑的人也不一定是因为他们的焦虑或仅因为他们的焦虑而拖延。

**总结起来**，焦虑可能通过多种方式导致拖延，例如增加人们对任务的厌恶并使其更多地反复思考。此外，拖延有时会增加焦虑，从而导致焦虑-拖延循环。然而，这些现象之间的关系是复杂的，例如焦虑并不总是导致拖延，有时甚至可能减少拖延。

## 焦虑性拖延

术语*焦虑性拖延*可以用两种主要方式。首先，它可以用来指由焦虑驱动的拖延。其次，它还可以用来指尽管有意愿工作，尽管意识到这种延迟是非理性和自我毁灭的拖延行为。

第二种类型的拖延有时也被称为[传统拖延](https://doi.org/10.22215/etd/2015-11051)或*非理性延迟*。它与*享乐拖延*（或*享乐延迟*）相对比，后者是由于优先考虑愉快活动或缺乏关注而自愿推迟事务。

焦虑性拖延与享乐性拖延之间的区别类似于*回避性拖延*，后者由于恐惧和焦虑而推迟，以及*兴奋性拖延*，后者是在截止日期前等待使任务更加刺激。然而，这些类型拖延的区别已经受到批评，例如[被动和主动拖延](https://solvingprocrastination.com/active-passive-procrastination/)之间的区别。

请注意，通常从实际角度来看，不同类型的拖延分类的确切方式并不重要。重要的是要理解焦虑和类似问题可以引起拖延，但其他类型的问题也可以引起拖延，例如将短期享乐优先于长期成就。

## 有关焦虑的其他信息

下面的子部分包含有关焦虑的进一步信息，这些信息通常对处理基于焦虑的拖延并不重要，但在某些情况下仍然可能是有用的。

### 焦虑的形式

焦虑可以表现为许多不同的形式。例如，它可以根据焦虑的强度（例如，从轻度到严重）而有所不同，这取决于因素，例如焦虑的强度，它发生的频率以及它对经历它的人的影响。同样，焦虑可以全天候持续存在，或者它只能在[特定时刻](https://doi.org/10.1080/0260293042000160384)发生，例如某人参与某项特定任务时。

此外，人们可能会因许多事物而感到焦虑，例如：

+   特定类型的任务，例如学生在[学校](https://doi.org/10.1016/j.lindif.2021.102013)中参加的*考试焦虑*或作为工作一部分进行的公开演讲的*表现焦虑*。

+   特定主题，如数学和[统计](https://doi.org/10.1007/s10755-013-9256-1)，无论人们与该主题相关的活动是什么（例如，参加考试，做作业或写论文）。

+   整个领域，如[学校](https://solvingprocrastination.com/academic-procrastination/)或[工作场所](https://solvingprocrastination.com/workplace-procrastination/)。

+   在任务执行上的[后果](https://doi.org/10.1348/000709999157761)，例如成功相关的奖励或失败相关的惩罚。

+   他人对他们表现的评估（也称为*评估焦虑*）。

+   与[他人](https://doi.org/10.1177/0033294118755111)的一般互动（也称为*社交焦虑*）。

### 焦虑与恐惧的区别

与焦虑类似，对各种事物的*恐惧*，如失败或负面反馈，也可能导致[拖延或由此引起](https://solvingprocrastination.com/fear/)。

焦虑和恐惧密切相关，因为它们在很多方面高度 [相似](https://doi.org/10.1016/j.cpr.2010.08.004)，甚至 [重叠](https://doi.org/10.1111/psyp.13474)，无论是在描述方式上还是对人们的影响方式上。一个区分这些概念的可能方式是把恐惧视为对具体和即时威胁的情感反应，而把焦虑视为对更加模糊和不确定的未来威胁的反应。在这种区分下，恐惧更可能 [引发](https://doi.org/10.1111/psyp.13474) 即时和积极的反应，而焦虑更可能促使谨慎和观察。然而，恐惧和焦虑这两个术语通常可以 [互换使用](https://doi.org/10.1016/j.cpr.2010.08.004)，它们的确切定义以及它们之间的任何区别，严重依赖于使用这些术语的人、方式以及上下文。

从实际的角度来看，在克服基于焦虑的拖延症的背景下，理解这两个概念之间的任何区别通常不是至关重要的，只要你在一般情况下理解这些概念，并且理解它们都可能导致拖延。

### 焦虑和神经质

增加的焦虑与*神经质*这一人格特质 [相关](https://doi.org/10.1111/ap.12173)，反映出倾向于负面情绪和 [压力](https://solvingprocrastination.com/stress/)。然而，整体上神经质与整体上拖延之间的联系 [相对较弱](https://doi.org/10.1037/0033-2909.133.1.65)，可能是由于它们之间关系的复杂性，例如神经质的不同方面可以增加或减少拖延，类似于焦虑的不同形式对拖延的影响。

## 处理基于焦虑的拖延症

如果你患有严重的焦虑或焦虑症，那么你通常应该 [寻求帮助](https://web.archive.org/web/20211222155803/https:/www.mayoclinic.org/diseases-conditions/anxiety/symptoms-causes/syc-20350961)，从持有许可的专业人士（例如治疗师、心理学家或精神科医生）那里，他们将能够 [为你提供建议](https://web.archive.org/web/20211221211118/https:/www.nhs.uk/mental-health/conditions/generalised-anxiety-disorder/overview/)，包括疗法、药物和生活方式变化等 [适当的治疗方案](https://web.archive.org/web/20211222155737/https:/www.nimh.nih.gov/health/topics/anxiety-disorders)，这对减少焦虑以及导致的问题，如拖延，都有益处。

如果你正在尝试自己解决基于焦虑的拖延问题，那么一个关键的技巧是[*基于探究的压力减少法*](https://solvingprocrastination.com/study-inquiry-based-stress-reduction/)。要做到这一点，首先要确定一个你有的紧张想法，然后考虑以下问题：

+   这个想法是真实的吗？

+   你能绝对确定这个想法是真实的吗？

随后，你应该反思这个令人紧张的想法，考虑其原因和影响等因素，以更好地理解它。你可以通过思考以下问题来帮助自己做到这一点：

+   当你有这个想法时，你会如何反应？

+   这个想法给你的生活带来平静还是压力？

+   当你考虑这个想法时，你看到了什么形象，过去或现在？

+   由于有这个想法，会引起什么身体感觉？

+   当你有这个想法时，会引发什么样的情绪？

+   当你有这个想法时，是否会出现任何困扰或成瘾？

+   当你有这个想法时，你如何对待他人？

+   当你有这个想法时，你如何对待自己？

随后，你应该试着不带有因紧张想法而造成的扭曲去感知现实，并思考如果没有那个想法，你会如何感受，通过考虑以下问题：

+   没有这个想法，你将会是谁？

最后，你应该找出支持相反的紧张想法有效性的具体证据，并探讨这个相反想法是否可能成立。例如，如果你最初的想法是“我*不能*好好准备考试”，你可以将其转变为相反的想法“我*会*好好准备考试”，然后找到支持你生成的新积极想法的证据。

此外，还有许多其他[抗拖延技巧](https://solvingprocrastination.com/how-to-stop-procrastinating/)可以帮助你应对基于焦虑的拖延，其中包括以下最显著的：

+   **质疑和解决你的恐惧**。例如，如果你意识到自己[害怕](https://solvingprocrastination.com/fear/)有人因为你的工作不完美而批评你，你可以问自己一些问题，比如“他们实际上会关心我犯一些小错误的可能性有多大”或“如果这个人批评我的工作，那又怎样”。在做这些时，可以通过使用[自我距离技术](https://effectiviology.com/self-distancing-rational-decisions/)来帮助，例如问自己“为什么*你*要如此担心这些微不足道的问题？”或者考虑如果朋友处于你的处境，你会给出什么建议。

+   **为未来的偶发事件做好准备**。例如，弄清楚哪些分散注意力的事物可能会引诱你拖延，并计划如何处理它们。

+   **允许自己犯错**。例如，如果你在写论文，接受你的工作[不会完美](https://solvingprocrastination.com/perfectionism/)，尤其是在初稿阶段。

+   **将工作分解为小而可管理的步骤。** 例如，如果您有一个看起来令人不知所措的大项目，比如写一篇研究论文，您可以将其分解为一系列小步骤，如制定大纲，查找相关资源和撰写引言。

+   **从一个微小的步骤开始。** 例如，承诺只写一句话或者只锻炼两分钟，同时允许自己在迈出那个微小的第一步后停止，以减少开始时的压力。

+   **进行起始仪式。** 例如，您可以进行一种令人平静的呼吸例程，每次在做让您焦虑的任务之前快速完成，以便[放松](https://doi.org/10.1016/j.obhdp.2016.07.004)。

+   **在不同任务之间切换。** 例如，如果您在一项任务上遇到困难无法取得进展，请切换到[另一项任务](https://solvingprocrastination.com/productive-procrastination/)，直到您准备好回到第一个任务。

+   **根据您的生产力周期安排工作。** 例如，如果您发现早上更容易集中精力完成创造性任务，那么尽可能在那个时间段安排这些任务。

+   **改善您的工作环境。** 例如，如果您当前的工作环境有很多令人讨厌的背景噪音，可以购买降噪耳机或者找一个更安静的地方。

+   **改善您的社会支持网络。** 例如，您可以找到一个角色模型来模仿或一个权威人士来追责，或者您可以与那些激励您取得进展的人交往，同时尽量减少与让您感到压力的人的接触。

+   **充分休息。** 例如，如果您需要集中精力进行深度集中的任务，请确保有足够的休息时间，以免过度劳累。为了鼓励自己这样做，您可以提醒自己，即使休息可能会在短期内降低您的生产力，但从长远来看，无论是在生产力还是在您的健康方面，通常都会更好。

+   **发展自我效能。** 具体来说，这是您对实现目标所需行动的[信念](https://solvingprocrastination.com/self-efficacy/)。您可以通过[各种方式](https://solvingprocrastination.com/self-efficacy/)来发展它，比如确定您可以用来实现目标的策略，然后考虑如何成功执行这些策略。

+   **原谅自己以前的拖延行为。** 例如，如果您需要开始一个长期推迟的任务，您可以说“我不应该一开始就推迟这个任务，但那已经是过去，现在重要的是继续前进，只是把它完成”。

+   **培养自我怜悯。** 具体来说，您应该培养[自我怜悯](https://solvingprocrastination.com/self-compassion/)的三个组成部分：*自善性*，这涉及对自己友善；*共同人性*，这涉及认识到每个人都会面临挑战；以及[*正念*](https://solvingprocrastination.com/procrastination-study-mindfulness/)，这涉及以非评判的方式接受您的情绪。

+   **承认并奖励您的进步。** 例如，一旦您连续一周成功实现学习目标，您可以奖励自己一些愉快的享受。

在选择使用哪些技巧时，可以帮助您首先弄清楚[为什么会拖延](https://solvingprocrastination.com/why-people-procrastinate/)，以及您何时以及如何拖延，因为这将帮助您确定在您特定情况下使用最佳的[抗拖延技巧](https://solvingprocrastination.com/how-to-stop-procrastinating/)。具体来说，这可以帮助您弄清楚您的焦虑如何导致您拖延，还可以帮助您识别除焦虑外其他导致您拖延的问题，如[抑郁症](https://solvingprocrastination.com/depression/)、[完美主义](https://solvingprocrastination.com/perfectionism/)和抽象目标。

但是，如果您意识到您拖延的主要原因是因为您患有焦虑症，通常应该首先治疗焦虑症，如果必要的话，寻求经过许可的专业帮助，因为这对于处理您的拖延和减少焦虑可能导致的其他严重问题都非常重要。
