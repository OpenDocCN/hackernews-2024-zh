# 这是一个青少年

> 来源：[https://pudding.cool/2024/03/teenagers/](https://pudding.cool/2024/03/teenagers/)

这个世界对孩子们有很多同情。

当我们年轻时，我们对生活的控制很少。我们玩耍，搞砸事情，惹上麻烦。我们也忍受着家庭功能失调，家庭混乱，暴力，欺凌和其他一切。但我们告诉自己，最终，我们将能够塑造自己的生活。

然后我们到了18岁，被期望要成为“成年人”并且自己搞明白。

如果我们失败了，我们就会受到惩罚。

我们因为不上大学、因为身体不健康、因为贫穷、因为负担不起医疗保健和食品和住房而受到责备。
