# 斯蒂芬·列维特和约翰·多纽捍卫了《奇葩经济学》所著名的发现

> 来源：[https://www.economist.com/by-invitation/2024/04/08/steven-levitt-and-john-donohue-defend-a-finding-made-famous-by-freakonomics](https://www.economist.com/by-invitation/2024/04/08/steven-levitt-and-john-donohue-defend-a-finding-made-famous-by-freakonomics)

从我们发表一篇学术论文将堕胎合法化与1990年代以来美国犯罪率的大幅下降联系起来，已经过去了二十多年。其基本理论很简单。出生时不被期望的孩子面临一系列不利的生活结果，并且在后来的生活中犯罪率更高。合法化的堕胎大大减少了不期望的出生数量。因此，合法化的堕胎将减少犯罪，尽管会有相当大的滞后效应。

我们的论文引起了很多争议，这在由我们之一与斯蒂芬·杜伯纳合著的畅销书《奇葩经济学》中的一章进一步加剧。该书于2005年出版。对于许多人来说，更重要的是对我们的假设进行政治性回应，而不是评估其是否正确。
