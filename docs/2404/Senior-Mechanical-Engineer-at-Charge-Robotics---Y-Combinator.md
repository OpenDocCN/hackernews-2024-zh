# Charge Robotics高级机械工程师 | Y Combinator

> 来源：[https://www.ycombinator.com/companies/charge-robotics/jobs/ml4f9l4-senior-mechanical-engineer](https://www.ycombinator.com/companies/charge-robotics/jobs/ml4f9l4-senior-mechanical-engineer)

作为Charge Robotics的高级机械工程师，你将负责开发一个新的机器人系统，用于组装太阳能场。

我们需要一位经验丰富的机械设计工程师来帮助设计下一代的送货系统，该系统可将太阳能场的40英尺长、1吨重的部件运输到场地周围。

你将会与非常大的工业设备合作，成为太阳能建设的内行专家，并直接帮助加速全球向可再生能源的过渡。

## 你将要做的工作

+   设计、建造、测试和迭代用于太阳能装配的机械系统。

+   跟踪、调查和解决机器人的硬件问题。

+   与其他工程师沟通，以整合新设计。

+   操控一些非常大的机制

## 你

+   熟悉行业标准的CAD软件（我们使用Onshape）

+   知道如何制造你的零件，同时也知道什么时候将图纸送到店里。

+   喜欢在快节奏的创业环境中工作。

+   拥有优秀的书面沟通技能。

+   位于或能够搬迁至旧金山湾区。

+   从个人角度对减缓气候变化过程中发挥作用感到满足。

+   对看到你的工作部署到实际的太阳能场感到兴奋！

## 如果你还能

+   有与工业机器人臂、末端执行器和自动化系统一起工作的经验。

+   拥有机械工程或相关领域的四年制学位。

Charge Robotics是一家由Y Combinator支持的创业公司，致力于建造大规模太阳能场的机器人。新太阳能项目的需求正在迅速增长（美国所有太阳能装机量的五分之一是去年安装的！），但今天的建筑公司由于劳动力资源有限而无法跟上。

我们认为这是疯狂的，所以我们开始研发机器人，直接解决这一瓶颈，加快世界向可再生能源过渡的速度。

我们是一家小而快速发展的公司，这意味着不断学习和成长的机会。你将对公司和产品的发展方向产生重大影响，这将反映在可观的股权补偿中。而且你还可以和🤖巨型机器人🤖一起玩耍。

如果你对解决具有直接气候影响的有趣技术问题感兴趣，你会很合适。
