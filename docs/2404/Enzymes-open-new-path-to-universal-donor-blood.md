# 酶开辟了通往万能供体血液的新路径

> 来源：[https://www.dtu.dk/english/newsarchive/2024/04/enzymes-open-new-path-to-universal-donor-blood](https://www.dtu.dk/english/newsarchive/2024/04/enzymes-open-new-path-to-universal-donor-blood)

寻求开发万能供体血液的探索迈出了重要一步。丹麦技术大学（DTU）和隆德大学的研究人员发现，当与红细胞混合时，某些酶能够去除构成人类ABO血型A和B抗原的特定糖类。结果已经在[科学期刊](https://www.nature.com/articles/s41564-024-01663-4) *[自然微生物学](https://www.nature.com/articles/s41564-024-01663-4)* 上发表。

*"这些新酶组合首次不仅去除了广为描述的A和B抗原，还能够去除先前未被认为对输血安全构成问题的扩展变体。我们即将能够从B型供体中制造万能血液，但将更复杂的A型血液转变仍需努力。我们现在的重点是详细调查是否存在额外的障碍，以及如何改进我们的酶以达到生产万能血液的最终目标，" DTU研究领导人之一、这一发现的高级科学家之一Maher Abou Hachem教授如是说。*

*他指出，这一发现是DTU研究人员在人类肠道微生物群酶方面的专业知识与隆德大学研究人员在基于碳水化合物的血型和输血医学方面结合的结果。*

## **对供体血液的高需求**

**人类红细胞携带特定复杂糖类结构（抗原），定义了ABO血型的四种类型：A、B、AB和0。这些抗原控制了供体和受体之间的兼容性，以实现安全的血液输注和器官移植。供体血液经过疾病标记物和主要血型的筛查后，可以在冰箱中保存长达42天。**

**由于老年人口比例增加以及更多患者接受需大量输血的医疗程序，对供体血液的需求很高。成功将A型或B型血液转化为ABO万能供体血液，可以显著减少当前与存储四种不同血型相关的后勤和成本。此外，万能供体血液的开发将通过减少接近过期日期的血液浪费，增加供体血液的供应。

之所以需要去除A和B抗原以创造万能供体血液，是因为当输注到不匹配的受体时，它们可能引发危及生命的免疫反应。

使用酶来产生万能供血者的概念自40年前提出，之后，研究人员发现了更高效的酶来移除A和B抗原，但仍然无法完全解释或解决与血液相关的所有免疫反应问题，因此这些酶仍未在临床实践中被使用。

## **肠道中的酶**

**丹麦技术大学（DTU）和隆德大学的研究小组开发了新的方式，即寻找既能清除A和B型血型抗原又能分解阻止它们的糖类的酶。研究团队发现了来自人类肠道细菌Akkermansia muciniphila的新酶混合物，这种细菌以分解 mucus 的方式获取养分，而 mucus 包裹着肠道表面。大家发现，这些酶具有极其高效性，因为肠道粘膜表面的复杂糖分子与血细胞表面发现的糖类结构相似。**

**“膜层的独特之处在于，能够在这种材料上存活的细菌往往有专门用来分解膜糖结构的酶，而这种结构包括血型AB0抗原。这一假设被证明是正确的,”Maher Abou Hachem说。**

本研究中，研究人员测试了24种酶，用它们处理了数百份血样样本。

“万能血液将提高捐赠血液的利用效率，并避免误输A型、B型、O型或AB型血液，从而预防潜在致命不良反应于接收者身上。一旦我们能够生成A型、B型、O型、或AB型万能的捐赠血液，这将简化安全血液产品运输和分配的后勤工作，同时最大限度地减少了血液浪费”隆德大学研究负责人马丁·L·奥尔森教授如是说。

DTU 和隆德大学的研究人员已经申请了新酶及其酶处理方法的专利，并期待在未来三年半内不断发展这项新技术，在未来共同的项目中进行生物试验研究。如果成功，需要在受控患者试验中进行验证，才能考虑进行商用和临床应用。

该初始研究项目由丹麦独立研究基金（Technology and Production Sciences, FTP），瑞典研究委员会，瑞典政府及其县委员会提供的ALF拨款，以及Knut和Alice Wallenberg基金会和丹麦研究基金（Natural Sciences, FNU）提供资金支持。新持续项目则由诺和诺德基金会资助，以及跨学科协同计划。
