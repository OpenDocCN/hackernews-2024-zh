# 微软投资15亿美元加速阿布扎比G42的人工智能发展和全球扩展 - 故事

> 来源：[https://news.microsoft.com/2024/04/15/microsoft-invests-1-5-billion-in-abu-dhabis-g42-to-accelerate-ai-development-and-global-expansion/](https://news.microsoft.com/2024/04/15/microsoft-invests-1-5-billion-in-abu-dhabis-g42-to-accelerate-ai-development-and-global-expansion/)

+   *这笔战略投资将增强阿联酋作为全球人工智能中心的地位，并为合作伙伴和客户提供进一步创新和增长的机会。*

+   *G42和微软将合作，确保安全人工智能技术和云能力的好处能够在全球新兴经济体中得到负责任的分享。*

+   *微软副主席兼总裁布拉德·史密斯将加入G42董事会。*

+   *两家公司将支持成立10亿美元的开发者基金。*

+   *该合作包括首个其类的政府间保证协议，双方承诺遵守人工智能安全和安全领域的世界领先标准。*

**2024年4月16日，阿布扎比，阿联酋** – G42，阿联酋领先的人工智能技术控股公司，以及微软公司今日宣布，微软将向G42进行15亿美元的战略投资。此举将加强两家公司在将最新的微软人工智能技术和技能培训项目引入阿联酋及全球其他国家方面的合作。作为这一扩展合作的一部分，微软的副主席兼总裁布拉德·史密斯将加入G42董事会。

*这一扩展合作将赋予各种规模的组织在新市场中利用人工智能和云的优势的能力，同时确保它们采用符合世界领先标准的人工智能。*

建立在两家组织在人工智能和数字转型项目上的长期合作基础上，微软的投资加深了对这一战略合作伙伴关系的相互承诺。G42将在微软Azure上运行其人工智能应用和服务，并与其合作向全球公共部门客户和大型企业提供先进的人工智能解决方案。G42和微软还将共同努力将先进的人工智能和数字基础设施引入中东、中亚和非洲国家，为这些国家提供公平获得服务的机会，以解决重要的政府和企业关切，并确保最高标准的安全和隐私。

**G42主席阿联酋谢赫·塔努恩·本·扎耶德·阿尔·纳哈扬**表示：“微软对G42的投资标志着我们公司成长和创新旅程中的一个关键时刻，象征着两家组织在愿景和执行上的战略一致性。这一合作是对共同价值和进步愿景的证明，促进全球范围内更大的合作和协同。”

该合作还将支持一个熟练且多样化的人工智能人才队伍和人才库的发展，通过在开发者基金中投资10亿美元，推动阿联酋和更广泛地区的创新和竞争力。

“我们两家公司不仅将在阿联酋合作，还将把人工智能和数字基础设施及服务带给被忽视的国家。”— **布拉德·史密斯，微软副董事长兼总裁**。“我们将结合世界一流的技术与安全、可信任和负责任的人工智能标准，与阿联酋和美国政府密切合作。”

商业合作由双方政府的保证支持，通过首次类似协议，应用世界一流的最佳实践来确保人工智能的安全、可信和负责任的开发和部署。微软和G42将密切合作，提升其联合国际基础设施的安全和合规框架。双方公司将致力于遵守美国和国际贸易、安全、负责任人工智能和商业诚信法律法规。这些工作受《G42与微软政府间保证协议》的详细规定监管，该协议是在与阿联酋和美国政府的密切磋商中制定的。

**彭晓，G42集团首席执行官**，表示：“通过微软的战略投资，我们正在推进规模化交付先进的人工智能技术的使命。这一合作显著增强了我们的国际市场存在，将G42独特的人工智能能力与微软强大的全球基础设施相结合。我们不仅扩展了运营范围，还制定了创新的新行业标准。”

**萨默·阿布-尔泰夫，微软公司副总裁兼总裁，中东和非洲中东欧地区**，补充道：“我们对G42的投资证明了阿联酋和更广泛地区蓬勃发展和充满活力的技术景观。这种战略合作伙伴关系定位良好，有望为我们的客户和合作伙伴提供机会，加速创新，并促进经济增长。与G42合作，我们将引入尖端技术，通过利用云和人工智能的力量，推动国家和市场推进其数字议程。”

G42与微软的合作在过去一年中取得了数个重要里程碑。包括2023年4月宣布的联合计划，旨在开发为公共部门和工业量身定制的AI解决方案，充分利用微软广泛的合作伙伴生态系统和云能力。2023年9月，两家公司达成协议，推出主权云服务，并合作开发在Azure公共云平台上释放先进AI能力的潜力。最后，在2023年11月，微软宣布在新的Azure AI云模型即服务提供中推出了G42的Jais阿拉伯大语言模型。
