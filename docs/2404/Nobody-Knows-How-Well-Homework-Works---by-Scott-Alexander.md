# 标题：没有人知道家庭作业是否有效——斯科特·亚历山大

> 来源：[https://www.astralcodexten.com/p/nobody-knows-how-well-homework-works](https://www.astralcodexten.com/p/nobody-knows-how-well-homework-works)

前天，我讨论了**学习瓶颈**([文章链接](https://astralcodexten.substack.com/p/study-ritalin-works-but-school-isnt))。我想要探讨的是家庭作业的有效性。如果它运作良好，就意味着学生在例子和重复训练方面感到瓶颈。如果它运作不佳，则可能是其他原因。

这种方法的一个小问题是，我们实际上并不知道是否有任何人准确报告了他们进行的家庭作业时间。Couper引用了一些研究，表明学生报告的作业时间与测试成绩的相关性为相当不错的r = 0.25。但在同一样本中，家长报告的作业时间相关性接近于零。Couper猜想学生估计的数值比家长的好，我认为这种说法有道理——增加噪声的反向效果比减少噪声的反向效果更容易——但最终我们并不知道。根据[华盛顿邮报文章](https://www.washingtonpost.com/news/answer-sheet/wp/2012/11/26/homework-an-unnecessary-evil-surprising-findings-from-new-research/)的记录，两个非常相似的数据集内，学生报告的作业时间数量差异很大——这可能是因为提问方式的问题？我不知道，从学校的儿童收集的自我报告似乎存在很多问题。

那些两边引用的研究主要采用“完成家庭作业的时间”作为自变量，然后与测试成绩或成绩进行相关分析。如果做更多家庭作业的学生得到更高的测试成绩，则结论是家庭作业有效；否则，则认为无效。

原文：

但这只是我们面临的问题中最轻的。这种方法假设完成家庭作业的时间是一个安全的数据代理，表示教师分配的家庭作业数量。它并非如此。学生可能因为聪明、发现作业容易、能够迅速完成而花更少的时间在作业上。也可能因为热爱学习并对学习内容的重视程度很大而花更多时间在作业上。或者因为是亚裔第二代移民，有严格父母要求完美完成，而花更多时间。他们也可能因为处于某种极其不利的生活环境而不适合安静坐在书桌前，所以花更少的时间在作业上。所有这些因素都使“完成家庭作业的时间”成为“教师分配的家庭作业量”的不良代理，直接使作业与测试成绩的相关性混杂不清。大多数研究都没有注意调整这些因素。那些尝试调整的也只是随意选择，对要使用什么模型猜测不当，然后得到基本上随机的结果。

作业的支持者（[哈里斯·库珀](https://sci-hub.st/https://www.jstor.org/stable/3700582)）和反对者（[阿尔菲·科恩](https://classtap.pbworks.com/f/Homework+May+Not+Be+A+Good+Thing.pdf)）都简要地提到了这个问题，然后还是认真地对待了这些研究。如果你这么做了，你会发现，也许小学时作业并不有益，但可能在高中时有用（尽管有些人对这种说法的一半持不同意见）。但你为什么要认真对待这些呢？

是否有真正的随机研究？库珀在他的评论文章中找到了六篇，没有一篇是发表或同行评审的。只有一篇是按学生随机分组的，并且关于其实际随机性的描述存在矛盾；另外五篇是按班级进行的集群随机化（这意味着它们具有非常低的有效样本量）。有几篇在混乱的方式中搞砸了。尽管如此，这些研究相当一致地显示出作业具有中到高的效果大小。那篇可能是由学生随机分组的（因此可能还可以接受）的一篇的效果大小为0.39。一些集群随机的研究，没有太糟糕的，其效果大小在0.9范围内；集群随机化使得难以称其为显著，但非官方地看来令人印象深刻。

自从库珀写下他的2006年评论以来，我找到了一项确实很好的、个体随机化的作业研究，[纳瓦兹和威尔本](https://sparxmaths.com/pdf/Homework-length-trials-2018.pdf)。他们选取了368名使用数字平台上代数课程的学生，并随机分配了他们的普通作业负载的0%、50%、100%或150%（相当于每晚0、15、30或45分钟）。结果：

作业更多的学生表现更好，p < 0.0001。看起来很可靠。也许九年级的代数作业是有用的。但是每个人都已经预料到高中的作业比小学的更有用，数学作业比其他科目更有用。因此，不清楚例如四年级的阅读作业是否会遵循相同的模式。

但是，这是我们在数百万儿童的作业中唯一确定的事实。至于其他一切，我猜还是听听你的直觉吧。
