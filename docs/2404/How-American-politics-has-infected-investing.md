# 美国政治如何影响投资

> 来源：[https://www.economist.com/finance-and-economics/2024/04/21/how-american-politics-has-infected-investing](https://www.economist.com/finance-and-economics/2024/04/21/how-american-politics-has-infected-investing)

对于这只对冲基金，它的品牌名称已经透露了一些端倪。1789资本去年成立，以美国《权利法案》提出的年份命名。它为投资者提供投资机会，专注于三大关键主题：一个平行的保守经济体系，迎合那些希望避免被自由主义思想轰炸的消费者；远离自由贸易；以及受到环境、社会和治理（ESG）投资趋势惩罚的公司。其创始人奥米德·马利克是一名前银行家，曾为反疫苗、未来不明的总统候选人[罗伯特·肯尼迪（junior）](https://www.economist.com/1843/2024/04/10/robert-f-kennedy-junior-doesnt-care-if-he-condemns-america-to-trump)举办过筹款活动。
