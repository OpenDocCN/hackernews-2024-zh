# 在Bright | Y Combinator担任Ops & Customer Service - General Manager。

> 来源：[https://www.ycombinator.com/companies/bright/jobs/cd8fV1s-ops-customer-service-general-manager](https://www.ycombinator.com/companies/bright/jobs/cd8fV1s-ops-customer-service-general-manager)

☀️ **关于Bright：**

Bright是墨西哥领先的屋顶太阳能平台 - 我们提供分布式太阳能操作系统，使数千家庭和企业可以以零预付成本提供更便宜的电力，我们与项目融资商、本地安装商和分销商网络合作，以满足由此产生的需求。我们的第一个市场是墨西哥，该市场比美国拥有更多的阳光、更高的电费和更低的劳动成本。Bright的投资者包括[IFU.DK](http://IFU.DK)、First Round Capital、Y Combinator、莱昂纳多·迪卡普里奥、丹尼尔·塞维特耶、Sunrun和Tesla的创始人以及其他顶级硅谷公司。

💼 **关于职位：**

Bright，墨西哥领先的分布式太阳能公司，正在寻找一位积极进取、使命驱动的个人加入我们的团队，担任Ops And Customer Service General Manager，并有可能晋升为首席运营官（COO）。理想的候选人将具备在100到300-500名员工范围内扩展公司的强大记录，并热衷于推动我们的使命。虽然有客户服务扩展经验更佳，但客户至上的心态和提升客户体验的承诺至关重要。

🥇 **我们正在寻找这样的人：**

+   在高增长公司环境中至少拥有6年的领导经验。

+   在高增长公司100+人规模阶段，具有COO或类似职位的经验。

+   拥有强大的战略思维和解决问题的能力。

+   出色的沟通和人际交往能力，能够在组织的各个层级建立和维护良好关系。

+   对Bright使命充满激情，并致力于产生积极影响。

+   出色的沟通、人际交往和组织能力。

+   热衷于构建和扩展成功的团队。

+   精通西班牙语和英语的多语种能力。

**✅ 关于您：**

您拥有将公司规模从100名员工扩展到300-500名员工的成功经验。

+   您对运营最佳实践有深刻理解，并具备识别和实施改进措施的敏锐眼光。

+   您希望在世界留下自己的遗产，并关心产生比自己更大的影响。您是一位强大的领导者，能够激励和启发高绩效团队。

+   您是一位出色的沟通者，能够有效跨越组织各个层级进行协作。（如果具备以客户为中心方法构建和扩展客户服务团队的经验，将获得额外加分！）

**💻 您的任务和责任将包括：**

+   培养客户至上的文化：

    +   强调整个组织的客户至上思维，教育人员关于提供卓越服务的高效益的重要性。建立相关的奖励和激励措施来支持这一努力。

    +   领导扩展客服团队的努力，并实施旨在提升客户满意度的倡议。

+   运营效率：

    +   监督日常运营和客户服务团队，优化流程以支持每年超过100%的客户增长。

    +   优化内部流程和程序，以增强效率和可扩展性。

    +   分析关键绩效指标并进行数据驱动的决策，以提高运营效率和推动增长。

+   团队领导与发展：

    +   建立和指导一个高效团队，培养责任感、创新精神和持续改进的文化。

    +   发展人才管道和接班计划，以支持未来的增长和领导需求。

+   风险缓解

    +   识别并减轻与快速扩展相关的潜在客户风险和挑战。
