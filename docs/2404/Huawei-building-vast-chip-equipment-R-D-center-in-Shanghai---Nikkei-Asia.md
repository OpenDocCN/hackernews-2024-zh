# 华为在上海建设大规模芯片设备研发中心 - 日经亚洲

> 来源：[https://asia.nikkei.com/Business/Tech/Semiconductors/Huawei-building-vast-chip-equipment-R-D-center-in-Shanghai](https://asia.nikkei.com/Business/Tech/Semiconductors/Huawei-building-vast-chip-equipment-R-D-center-in-Shanghai)

台北 -- 华为技术正在上海建设一个庞大的半导体设备研发中心，作为这家中国科技巨头继续加强其芯片供应链以应对美国的打压的一部分。

该中心的使命包括建造光刻机，这是生产尖端芯片所必需的关键设备。华盛顿的出口管制大幅削减了华为获取这些设备的渠道，而这些设备的生产主要由荷兰的ASML以及日本的尼康和佳能这三家公司垄断。
