## Claude 模型简介

Claude 3 Haiku、Sonnet 和 Opus 是下一代 Claude 模型。它们是最先进的文本和视觉模型，可以基于图像和文本输入生成类似人类的文本输出。具有强大的视觉能力，它们为计算机视觉和图像理解应用程序打开了令人兴奋的可能性。Opus 是最强大的模型，在各种任务上提供无与伦比的智能、推理和创造力，而 Haiku 以非常高的速度和低成本提供了丰富的功能。Sonnet 在两者之间取得了很好的平衡。
