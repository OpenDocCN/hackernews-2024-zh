# ESA卫星将测试锋利的编队飞行技术 - IEEE Spectrum

> 来源：[https://spectrum.ieee.org/satellite-constellation-formation-flying-esa](https://spectrum.ieee.org/satellite-constellation-formation-flying-esa)

欧洲空间局将于今年年底发射一项任务，以展示轨道上的精确编队飞行技术，创建人造日食。在上周的新闻发布会上，该机构宣布了任务的详细信息以及轨道器将使用的技术来执行其精心编排的操作。

ESA的[Proba-3](https://www.esa.int/Enabling_Support/Space_Engineering_Technology/Proba_Missions/About_Proba-3)（自主机载项目）由一对航天器组成：质量为300公斤的日冕望远镜航天器和质量为250公斤的[遮光器](https://www.esa.int/Enabling_Support/Space_Engineering_Technology/Proba_Missions/Face_to_face_with_Sun-eclipsing_Proba-3)。这对航天器现在计划于9月搭乘印度[PSLV火箭](https://en.wikipedia.org/wiki/Polar_Satellite_Launch_Vehicle)发射，并最终进入高度椭圆形的600×60,530公里轨道。该机构称，目标是将这两个分开的航天器移动到大约144米的距离，遮光器将作为一个盘状物体阻挡太阳。

实现这种编队将允许日冕望远镜研究我们恒星的高度电离、极端高温的大气层，同时也演示了技术作为更雄心勃勃的未来编队飞行计划的前导。

Proba-3的盘状遮光器航天器旨在为日冕望远镜航天器遮挡太阳。P. Sebirot/ESA

日食代表天体间壮观的对齐以及[惊人美丽的时刻](https://www.livescience.com/space/the-best-photos-and-videos-of-the-april-8-total-solar-eclipse-over-north-america)。它们也是进行科学研究的短暂机会。元素氦是由一位法国天文学家在印度进行的1868年日食中首次在太阳色球层中发现的。五十年后，英国天文学家弗兰克·戴森和阿瑟·爱丁顿[进行了测量](https://en.wikipedia.org/wiki/Eddington_experiment)，通过1919年日食期间恒星视位置的显著移动作为爱因斯坦广义相对论的早期测试。

ESA对Proba-3有科学目标，利用在空间中进行的观测研究太阳天体物理学，不受地球大气的干扰。该机构的太阳冠观测协会（[ASPIICS](https://www.mps.mpg.de/6884329/proba-3)）望远镜将有助于分辨太阳冠为何显著比太阳本身更热。这可能进一步增进我们对太阳的理解，并协助太阳天气预测。

然而，Proba-3 旨在展示的精密形成飞行可能有助于解锁未来的突破。

Proba-3 的日冕航天器承载了该任务的ASPIICS仪器，用于观测太阳日冕。P. Sebirot/ESA

这两个航天器将对齐以每轨道约六小时的时间创建日食，从接近其距地球最远点（远地点）开始。两个航天器将使用无线星间链路进行通信，并且两者都将使用星追踪器确定它们的姿态。

全球导航卫星系统（GNSS）接收器将在近地点或最接近地球的点附近接收GPS信号，当与专用的相对导航算法结合使用时，将允许定期确定它们的相对位置。遮光器上的光学传感器将观察日冕上的脉冲LED，为更精细的测量提供数据。但是，更高的毫米级精度仍需要更多的技术妙法。

### Proba-3 如何实现毫米级轨道精度

为此，遮光器将向安装在日冕航天器上的角反射器发射激光，该反射器将反弹激光束。这种[计量](https://en.wikipedia.org/wiki/Metrology)的行为将允许精确跟踪两个航天器的相对位置和方向。利用收集到的数据，ESA表示，航天器可以使用安装在遮光器上的10毫牛顿级冷气推进器控制和维持毫米级精度。

“导航、控制和控制经历了很多发展，这正是我们想要展示的，” [达米安·加拉诺](https://ieeexplore.ieee.org/author/37088738884)，Proba-3 项目经理，在4月3日的 [新闻发布会](https://www.youtube.com/watch?v=nEztx11vC8o) 上表示。

“我们有一个实际的应用，那就是观察日冕。因此，通过对日冕进行非常好的观测，我们肯定会证明所有这些设备都在工作，并且技术正在提供实际的科学数据，” 他说，补充道，Proba-3 的形成飞行控制算法和计量系统可能适用于未来的任务。

精确控制的遮光器航天器可以与空间望远镜配合使用，阻挡来自恒星的光线，以直接探测潜在的轨道行星，而一组航天器则可以通过干涉法创建大型观测站，实现比单一大型卫星更大的口径和更长的焦距。

进一步的应用包括地球观测、基于空间的引力波检测以及两个或多个航天器需要互动的一系列任务，如会合、对接和轨道服务。

在进行这类复杂项目之前，Proba-3 需要证明它能够掌握基础。"这显然是操作上的挑战，" ESA 技术、工程和质量总监[Dietmar Pilz](https://www.esa.int/About_Us/Corporate_news/Dietmar_Pilz_Director_of_Technology_Engineering_and_Quality)表示。"要看我们能在形成飞行中达到什么程度，我们可以实现什么距离。这需要来自项目中所有合作伙伴的大量操作经验和软件。"

如果成功，Proba-3 可能会超越以往的空间日冕仪的工作，并为未来复杂的空间操作铺平道路。
