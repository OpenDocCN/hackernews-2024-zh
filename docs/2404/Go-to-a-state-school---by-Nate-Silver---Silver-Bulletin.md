# 去一个州立大学 - 由**内特·银**撰写 - 银公告

> 来源：[https://www.natesilver.net/p/go-to-a-state-school](https://www.natesilver.net/p/go-to-a-state-school)

这些天，我用Twitter有两种方式：首先，公然为这份通讯和我正在推广的其他事物打广告。其次，作为一种观点的临时区域。有时这些观点是[故意挑衅的](https://x.com/NateSilver538/status/1781779307150430254)，而其他时候则包含了更严肃的种子。[这里](https://x.com/NateSilver538/status/1782113251846709629)是最近的一个例子：

等等，我对这个真心的吗？是的，多多少少。如果我在为朋友的儿子或女儿提供建议，面对[决策日](https://www.insidehighered.com/news/admissions/traditional-age/2024/02/05/how-fafsa-delay-throwing-admission-timelines#:~:text=The%20vast%20majority%20of%20colleges,for%20accepted%20students%20to%20commit.)，我会告诉他们放弃常春藤联盟，而去[一所高质量的州立大学](https://www.bestcolleges.com/blog/public-ivy-schools-and-little-ivies/)，在某些条件下。让我阐述一些例外情况：

+   如果学生的身份与成为普林斯顿人或康奈尔女士或其他什么有着深刻联系，那么我会觉得有点奇怪 —— 但无论如何，我会告诉他们去，我不是来这里嘲笑任何人的特殊癖好的。

+   如果（i）他们对自己的学位有很高的信心，并且（ii）它是像法律这样的领域，[认为证书尤为宝贵](https://x.com/dilanesper/status/1782113752990773702)，我也会告诉他们选择精英私立学院。

+   而且，如果他们来自经济困难的背景，并且被提供了一个加入精英的黄金门票，我会告诉他们强烈考虑去。我对选择性大学工资溢价的文献不是很了解，但在这些经济困难学生中，好处似乎是[集中的](https://www.nber.org/papers/w17159)。

但如果这个学生只是去学校“找自己” —— 而且她或她的父母大部分支付账单？是的，可能会选择一流的州立大学 —— 尤其是如果她所在的州有非常好的[本州公立学校，节省成本的幅度更大](https://educationdata.org/private-vs-public-college-tuition#:~:text=The%20average%20annual%20cost%20of,%2435%2C852%20for%20one%20academic%20year.)。这比毕业时背负巨额债务并且毕业院校可能被视为高度极端化要好。高等教育的公众认知急剧下降，我预计问题会变得更糟。

在我们进一步进行之前，有一个不可避免的问题：我是不是个伪君子？尽管我来自一个大十城镇——我父亲是密歇根州立大学的教授——我去了芝加哥大学。如今芝加哥被视为一所精英私立学校，但当我1996年首次入学时，它并不是[这样的学校](https://www.sparkadmissions.com/blog/ivy-league-acceptance-rates/)，当时的录取率约为40%。不过，这有点误导人——芝加哥大学的学生群体以“自我选择”而著称，这是一个被戏称为“快乐之地”的学校的环境。与今天相比，虽然我当时的学费看起来很划算，但按照90年代后期的标准来看，并不便宜。

但芝加哥大学一直是个例外。至少在我的时代，它倾向于吸引那些被学校怪癖声誉吸引的怪胎书呆子。芝加哥大学也一直是[政治多元化的学校](https://www.insidehighered.com/views/2018/12/21/chicago-principles-are-gold-standard-freedom-expression-campuses-opinion)，有[强烈的言论自由制度承诺](https://www.thefire.org/colleges/university-chicago/free-speech-rankings)，强调[核心课程](https://college.uchicago.edu/sites/default/files/documents/CoreRequirements.pdf)，[与其他类似学校相比少量的成绩通胀](https://ripplematch.com/insights/the-top-20-universities-with-the-highest-average-gpas-84ef5edf/)，以及[对传统录取偏爱较少](https://www.koppelmangroup.com/blog/2022/9/26/legacy-admission-guide-for-u-chicago)。毫无疑问，我是有偏见的，但我认为如果其他学校像芝加哥这样经营，精英高等教育今天就会比现在少些问题。

即使我曾经申请过哈佛——顺便说一句，没有被录取——我也不认为这会让我成为一个伪君子，因为我的论点是，这种变化是比较近期的，而许多下游影响还在未来。

即使在 [几乎所有美国机构都失去了公众信任](https://news.gallup.com/poll/508169/historically-low-faith-institutions-continues.aspx) 的时期，高等教育的信心下降尤为显著。据 Gallup 的民意调查，2015 年，57% 的美国人对高等教育有“极大信任”或“相当多信任”，[夏季2023](https://news.gallup.com/poll/508352/americans-confidence-higher-education-down-sharply.aspx)年，这一数字下降到36%。共和党人中的信心下降尤为严重，尽管只有32%的独立人士对高等教育表示信任，但民主党人中的数字也有所下降：

关于*精英们*如何看待高等教育的具体数据更难找到。在这里，我指的是广义上拥有权力、财富和影响力的人；我并不仅仅是用这个词作为“知识分子”的同义词。实际上，在我即将出版的书[《在边缘》](https://www.natesilver.net/p/officially-announcing-on-the-edge)中，我探讨了两个精英阶层之间的冲突 — 一个是分析性、风险承担职业，比如金融和科技[4]，我称之为“河流”，另一个是更加知识性的职业（学术界、媒体和政府），我称之为“村庄”。在书中，我认为这些群体是自然的竞争对手 — 他们的冲突最近变得更加明显，例如华尔街和硅谷帮助主导反对大学校长的行动。

因此，在某种程度上，我请求你们相信我对一本尚未公开的书籍的报道以及对话题的普遍接触，因为我可以确定自己符合我所定义的“精英” — 我预测科技和金融将对他们认为来自学术界的胡说八道变得不再宽容。例如，本周，尽管谷歌可能是[最进步或硅谷公司](https://www.natesilver.net/p/google-abandoned-dont-be-evil-and)，还是[解雇了数十名](https://www.washingtonpost.com/technology/2024/04/22/google-nimbus-israel-protest-fired-workers/?utm_source=twitter&utm_medium=social&utm_campaign=wp_main)参与亲巴勒斯坦/反以色列抗议活动的员工，这类活动类似于大学校园中的抗议。

显然，科技和金融并不是唯一的职业。而且，即使你降低哈佛、耶鲁或哥伦比亚学位的溢价30%，它们仍然具有极高的价值。现代大学 [服务多种不同的功能](https://twitter.com/pmarca/status/1745979447390486644)，其中许多不会受到影响。但我预计声望的下降足以对这些学位的价值产生实质性影响，足以使一些边缘学生认为它们不再“值得”。

一旦一个机构的看法变得政治上两极分化，想要改变这种情况是非常困难的。但我预计精英高等教育将会遇到特别大的问题。

这是因为学术界是一个进展缓慢的机构。学术教职员工 — 尤其是如果他们获得终身教职 — 往往会在一两个机构度过他们的整个职业生涯。对新建筑或项目的投资预期将会持续几十年。这里有大量的官僚主义存在。即使是学生，从他们第一次申请到大学，到最终毕业，几乎需要五年的承诺，而且这还是在他们按时完成的情况下，[很多学生并不会](https://www.nbcnews.com/news/us-news/college-students-dont-graduate-4-years-government-counts-6-years-succe-rcna2776)。

尽管有一些迹象表明学校意识到他们问题的严重性 — 特别是在[恢复标准化测试要求](https://www.insidehighered.com/news/admissions/traditional-age/2024/04/12/sudden-move-harvard-caltech-return-test-requirements)方面 — 我认为精英私立学院的问题在中期内可能会加剧。一些年轻教职员工，其中一些是在需要提供类似[掩盖政治立场的多样性声明](https://www.nytimes.com/2023/09/08/us/ucla-dei-statement.html)的政策下聘用的，更加支持“觉醒”（或我称之为[社会正义左派主义](https://www.natesilver.net/p/why-liberalism-and-leftism-are-increasingly)），而不像老一辈教职员工那样支持传统自由主义价值观，如言论自由。这些问题可能会自我强化。如果保守派学生 — 或者只想上课和朋友们一起闹着玩的政治中立学生 — 越来越对精英私立学院的政治环境失去兴趣，他们将不会选择这些学院，这反过来会使这些学院变得更加封闭，有较少数量相对心理健康的普通学生。

虽然到目前为止，我主要引用了来自右派的批评，但精英私立学院也容易受到来自左派的批评。正如传统录取系统所显示的，很少有像哈佛、斯坦福或耶鲁这样的地方在社会中任何地方比这些地方更加强化现有的等级制度。它们也是极其富有的机构，其[捐赠基金实际上充当对冲基金](https://www.nytimes.com/2021/10/15/business/dealbook/college-endowment-funds-harvard.html)。它们的招生过程也非常愚蠢，有时[招收他们知道自己不想要的学生](https://www.pbs.org/newshour/education/colleges-ratchet-recruiting-applicants-just-turn)，以降低录取率和提高排名。

这也许不是你期待的类比，但在某些方面，公众对精英学院看法的急剧转变让我想起了2010年代对[同性婚姻支持的急剧转变](https://news.gallup.com/poll/350486/record-high-support-same-sex-marriage.aspx)。同性婚姻可以从进步的角度来进行辩护，因为它为历史上被边缘化的群体创造了更多权利，也可以从保守的角度来进行辩护，因为婚姻通常被视为一种稳定的、保守的制度。精英私立学院既是左翼激进主义的温床，又是极度财富和特权的地方，也放弃了许多使它们首次跻身于此的[严格学术标准](https://www.slowboring.com/p/college-students-should-study-more)和[像言论自由这样的核心原则](https://www.natesilver.net/p/free-speech-is-in-trouble)。[左派和自由派和保守派](https://www.natesilver.net/p/why-liberalism-and-leftism-are-increasingly) *都* 有理由怀疑它们，即使它们有点尴尬。

因此，去他妈的哈佛和这些学校的其余部分 —— 修正看法的时机已经迫在眉睫。如果符合你的最佳利益，那就尽管去吧。但州立学院和机构对社会来说是更好的机构之一 —— 这是真正[让美国伟大的](https://www.theatlantic.com/magazine/archive/1891/03/the-state-university-in-america/633818/)事物之一 —— 并且通常以更低的价格提供更全面的体验和可比较的教育。
