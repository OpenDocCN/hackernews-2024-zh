## **比特币——一项计算机科学发明——是世界首创，永不重复**

[比特币](https://www.kraken.com/learn/bitcoin)并非凭空创造。几十年来，先前的电子现金项目失败了，但每一个都在前人的基础上取得了增量进展。比特币是这一过程的最终成果，是整个科学界的胜利。

一些前辈项目，如DigiCash，过于依赖可信的权威，因此从未获得市场认可。其他项目，如HashCash，通过计算机网络创造了工作货币，但无法长期保值。

最后，还有像自由储备（Liberty Reserve）这样的恐怖故事，那些运作工作电子货币的人员因其工作而被直接逮捕和监禁。

所有这些项目都有一个共同的目标——颠覆政府的货币垄断，创建一个免于中央控制的可行互联网货币。

## **以下是比特币成功而这些项目失败的5种方式**

+   在没有中央发行者的情况下，以公平透明的方式发行其资产BTC

+   允许用户加入并从其网络的运作中受益

+   通过密码学提供强大的财产权保障

+   采用不可更改的固定货币政策

+   给予用户工具持续改进比特币

许多数字资产领域的人认为比特币已经实现了上述所有目标，这使得比特币极不可能被政府或私人市场替代。

综上所述，这些成就代表了一个超越其各个部分之和的价值主张。即使在成千上万种加密货币中，比特币仍然独树一帜。

## **公平启动**

Satoshi的天才之处在于决定使用一种称为工作证明（PoW）的先前发明，将比特币分发给愿意用计算能力保护网络的任何用户。

要发行新的比特币，比特币用户必须使用计算机设备解决数学难题，通过消耗电力和资源来验证他们的工作。作为回报，他们将获得新铸造的BTC，这一过程被称为挖矿。

这一分布创建了一个公平竞争的环境，并促进了全球社区。

至关重要的是，这个系统意味着中本聪不需要出售、发行或市场化比特币。在2011年，他甚至将比特币软件的运作移交给一个开源开发者社区，其中没有人直接受到他的支付或任何形式的金融补偿。

用户通过向协议提供服务、交易能源以换取所有权，或者直接与他人交易来赚取比特币。这种设计确保了要获得比特币必须付出努力。

比特币的成功不仅仅是关于创造一种新货币；它关于创造一种分发价值的系统，这种系统不能被操纵，也不会不公平地优势*任何*用户。就像每个人一样，中本聪挖掘了他获得的所有比特币。

今天比特币的发行仍然是一个公平竞赛，但对于许多流通的替代加密货币来说并非如此，它们仍在寻找一种替代PoW的方式。

其中许多通过内部销售不成比例地分配其稀缺数据。这使得这些有利用户能够积累更多的货币或直接影响网络的发展和经济政策。

比特币摆脱了公平性和操纵的顾虑。

## **一个开放的网络 **

在其核心，比特币是一套规则系统，用于管理全球分布式数据库，跟踪其经济内数据的所有权。

为了网络能够有效运行，许多参与者必须保留和同步其数据库的副本，并同意这些副本没有差异。否则，就像旧电子货币一样，存在用户可能能够分配他们不拥有或未赚取的数据的风险，从而欺诈性地创建新的硬币并将其投入流通。

每个比特币竞争者都面临一个问题：数据库的大小与网络用户维护其自己副本的能力之间存在直接关系。

比特币做出了深思熟虑的权衡，以保持这种关键功能的可访问性。你可以将每个区块链网络视为由三类角色组成：

+   **矿工**，他们通过发现新区块并将其链接到之前的区块（构建区块链）来帮助确保网络安全并获得奖励。

+   **节点**，通过跟踪交易历史和验证新交易来保持过程的诚实性。

+   **用户**，基于对这些检查和平衡的信任进行交易。

与任何加密货币一样，这些基本功能存在进入门槛。但至关重要的是，比特币的障碍不是协议的产物，而是市场力量的结果。任何想要保护数据库的用户都可以通过获取电力和计算能力来实现。任何想要验证数据库的用户都可以通过下载和存储其分类账来实现。

这两种活动只受计算资源市场的影响。

其他加密货币增加了执行这些功能的成本的功能。有些将确定它们成本的能力分配给特定的用户，允许确保数据库的用户决定他们的同行在持有某种加密货币时必须具备的一定数量，或者满足他们制定的其他标准。

这些牺牲往往奖励财富和影响力，类似于政府管理的经济体系，其中货币的供应和分配不受市场力量的控制，而是由少数人决定。再次强调，比特币摆脱了这些妥协。

## **强大的产权**

产权被定义为[个人或组织的专有权](https://www.econlib.org/library/Topics/College/propertyrights.html)，以自己的决定管理和处置通过他们的劳动赚取的资源。

尽管这对于生活在保护这些权利的国家的任何人来说可能是直觉性的，但并非全球所有地区的人都享有这些权利。在一些国家，即使是民主国家，政府也可以通过（或滥用）法律体系冻结个人的银行账户。

这是其他加密货币常见的另一个困境。可以向任何加密货币添加功能，或更改规则，通过强制用户下载新的不兼容软件来改变所有权的分配。

比特币依赖于对其软件进行向后兼容的升级。这意味着其开发者优先考虑不强制用户进行升级的更改。用户可以运行与比特币网络兼容的任何软件，而不会牺牲功能（尽管这可能会以安全性为代价）。

其他加密货币通常会对其软件引入不兼容的更改，那些反对这种更改的人可能无法再享受与他人相同的利益。如果选择拒绝升级，你的硬币可能无法在经济中被接受。

当提出不兼容软件时，开发者可以衡量用户意见，但最终，每个用户都取决于其他用户的大多数。

对于比特币来说，少数群体可以继续使用旧版本，保持他们的比特币及其价值完好无损，尽管他们会面临安全性的权衡。这种允许不同意见存在的做法使比特币成为产权的捍卫者。

只要你持有比特币的私钥，你就有权拥有这些硬币。只要你运行任何与比特币兼容的软件，你就可以确保能够在比特币经济中使用这些密钥进行交易。同样，你可以确保比特币的总量永远不会超过2100万个。

## **固定的货币政策**

所有的货币都基于社会契约。用户同意用他们的劳动交换一个媒介，以便稍后自由获取产品和服务。

货币历史主要由两种类型的系统主导，这两种系统有不同类型的社会契约。

+   基于市场的货币，如黄金，基于无法由人类创造的有限资产

+   基于政府的货币，因其可以随意印刷而易于通胀，因为政府用它们支付开支

比特币是基于市场的货币，具备决定货币的所有特征：

+   它是**耐用的**：只要有互联网和电力，就会有比特币存在

+   它是**便携的**：你可以在世界任何地方访问你的资金

+   它是**稀缺的**：所有用户都可以确信，比特币只会有2100万枚

由于其公平的启动、开放的网络和强大的产权保护，比特币的货币政策不仅是固定的，而且是可信的。用户可以放心，它将保持不变，除非其数百万用户都同意更改，尽管这种可能性极小。

其他加密货币则提供具有较低可信度的可变货币政策。

有些货币变动频繁，与政府管理货币别无二致，其价值可能受政治干预影响。像中央银行一样，它们控制货币供应，并采取旨在维持价格稳定和促进经济增长的行动。

其他货币的发行没有限制，损害了它们的可信度。

同样，全球中央银行利用货币政策工具来控制其国家货币的供应。正如美联储所展示的那样，这些机构在何时以及为何调整利率方面常常含糊其辞。通常只有内部人员参与决策。

那些使用稳定币、以美元支持的加密资产或某种形式的中央银行数字货币（CBDC）的人，同样只是选择加入到现有系统中。

## **无限的改进**

尽管上述特性为比特币奠定了坚实的基础，但仅凭这些特性还不足以确保它永远不会被替代。这就是为什么它的最后一个属性可能是最重要的：比特币的改变和改进能力。

比特币似乎难以扩展到为全球80亿人提供其好处，需要开发额外的交易层，这些层可以扩展比特币的基础能力，同时不牺牲其核心价值主张。

仅在过去一年，比特币开发人员已经实现了以前认为不可能的壮举，而没有改变核心代码，解锁了[Turing-complete智能合约](https://www.kraken.com/learn/what-is-bitcoin-bitvm)，以及将比特币转换为[不可替代的代币](https://www.kraken.com/learn/what-are-bitcoin-ordinals-nft)的新方法。

比特币用户成功实施引人注目的新功能的能力，使得现有提供类似功能的加密网络变得多余。

在不断扩展的竞争加密货币和政府管理货币的海洋中，政策多样且不断变化，比特币独树一帜。

*投资加密资产存在风险，每种代币可能都有其独特的风险。以下是通常适用于所有加密资产的风险列表：*

*波动性：加密资产的表现可能非常波动，其价值可能会迅速下跌，就像它能上涨一样快。您应该做好准备，可能会失去您投资的所有资金。*

*缺乏保护：加密资产投资没有受到监管，无论是金融服务补偿计划（FSCS）还是金融调解员服务（FOS），都不会在您的加密资产投资出现问题时提供帮助或保护。*

*流动性：某些加密资产市场可能存在流动性不足的问题，这可能会阻止您以您希望或预期的价格买入或卖出您的加密资产。*

*复杂性：特定的加密资产可能伴随着难以理解的复杂风险。进行您自己的研究，如果某事听起来好得过分，那可能确实是如此。*

*不要孤注一掷：将所有资金投入单一类型的投资是有风险的。将资金分散投资于不同的项目，可以降低您对任何一个项目表现的依赖。*
