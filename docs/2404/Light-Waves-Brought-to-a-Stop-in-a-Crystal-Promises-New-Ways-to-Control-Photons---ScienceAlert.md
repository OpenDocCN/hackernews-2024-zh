# 光波在晶体中停下来，为控制光子提供了新的方法 : ScienceAlert

> 来源：[https://www.sciencealert.com/light-waves-brought-to-a-stop-in-a-crystal-promises-new-ways-to-control-photons](https://www.sciencealert.com/light-waves-brought-to-a-stop-in-a-crystal-promises-new-ways-to-control-photons)

发现新的方法来减慢或甚至停止光波的流动，可能会带来更先进的光子器件，如激光器、LED显示器、光纤和传感器。

在一个通过调整硅晶体以使其表现得像被扭曲的狡猾陷阱中，科学家们发现了一种灵活的全新方法来使光波完全静止。

光可以通过几种不同的方式停下来，例如通过冷却原子云，甚至可以[编织光波在一起](https://www.sciencealert.com/using-exceptional-points-coalescing-waves-stopping-light)。来自荷兰AMOLF和代尔夫特理工大学的这种新方法具有可以将新技术应用实现的优势。

"这个原理为减慢光场的速度提供了一种新的途径，从而增强它们的强度," [物理学家Ewold Verhagen说道](https://amolf.nl/news/light-stands-still-in-a-deformed-crystal)，[AMOLF](https://en.wikipedia.org/wiki/AMOLF)的。"在芯片上实现这一点对许多应用尤为重要。"

该团队的工作基于使用二维材料如[石墨烯](https://www.sciencealert.com/graphene)操纵电子。在导电材料中，电子可以自由移动，就像沿着微小高速公路飞驰一样。然而，施加磁场可以限制电子在某些能量上的运动，称为[朗道能级](https://en.wikipedia.org/wiki/Landau_levels)。

不仅磁铁可以将电子推向朗道能级。由单层原子组成的二维石墨烯也可以做到。通常情况下，石墨烯是导电的；但是如果你扭曲或变形石墨烯，比如拉伸它，你可以将电子限制在朗道能级内，将通常的导电材料变成绝缘体。

与AMOLF的René Barczyk和Delft大学的Kobus Kuipers合作，Verhagen试图找到一种材料，使光子产生类似于扭曲石墨烯对电子的影响。

现在，可以使用类似于石墨烯的材料——光子晶体——来操控光。研究人员发现他们可以以类似的方式延缓光波。

"光子晶体通常由硅层中的一种规则的——二维——孔洞模式组成。光可以在这种材料中自由传播，就像石墨烯中的电子一样," [Barczyk 解释道](https://amolf.nl/news/light-stands-still-in-a-deformed-crystal)。

"以恰当的方式打破这种规律将会扭曲这个阵列，从而锁定光子。这就是我们如何为光子创建朗道能级的方式。"

团队设计的蜂窝光子晶体能够利用一种过程将光限制在朗道能级上，该过程代表了不同类型的变形，比如弯曲或扭曲。他们甚至能够在同一材料的不同位置诱导不同类型的扭曲，从而形成一个光子晶体，在某些部分光能自由流动，但在其他部分会受到限制。

这一发现需要进一步的发展，但确实使科学家们在对光在非常小尺度上的精细控制方面迈出了一步。

"这使得芯片应用更加紧密，" [Verhagen 表示](https://amolf.nl/news/light-stands-still-in-a-deformed-crystal)。

"如果我们能够在纳米尺度上限制光并使其停止，其强度将得到极大增强。而且不仅仅在一个位置，而是在整个晶体表面上。这种光集中在纳米光子器件中非常重要，例如对于高效激光器或量子光源的开发。"

团队的研究已经发表在*[《自然光子学》](https://www.nature.com/articles/s41566-024-01412-3)*上。
