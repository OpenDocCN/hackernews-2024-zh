# 多款微软产品明年退出支持 • The Register

> 来源：[https://www.theregister.com/2024/04/18/2025_support_end_microsoft/](https://www.theregister.com/2024/04/18/2025_support_end_microsoft/)

Windows 10 不是唯一在明年即将被停用的微软产品—Office 2016、2019以及一系列生产力服务器也将面临支持结束的命运。

2025年10月14日是关键日期。大多数 Windows 10 版本计划到期，微软及其硬件合作伙伴也急于将用户迁移到 Windows 11。然而，大量生产力应用程序也将[退出支持](https://techcommunity.microsoft.com/t5/office-end-of-support-blog/key-end-of-support-dates-for-office-2016-2019-apps-amp/ba-p/4095876)，这意味着管理员在处理升级时将面临双重或三重打击。

虽然没有立即停止运作的风险，但这位 Windows 供应商警告称，在此结束日期后，"微软将不再提供安全补丁、错误修复或技术支持"。

"使用产品在支持结束后会让您的组织容易受到潜在的安全威胁，生产力损失以及合规问题的影响。"

微软还希望客户使用其云服务。其首选选项是注册类似其 E3 订阅以获得持续的支持和更新。

然而，并非所有人都有这个选项。在现场运行 Exchange Server 的勇士们特别受到影响。Exchange 2019 将在明年支持结束，但仍未有替代品。

在2022年，微软[表示](https://techcommunity.microsoft.com/t5/exchange-team-blog/exchange-server-roadmap-update/ba-p/3421389)服务器的下一个版本将在2025年下半年推出。原计划于2021年发布，但微软称："不幸的是，2021年对于 Exchange Server 有其他计划。在2021年3月，我们面临了一个严峻的现实：国家支持的威胁行为者正在攻击本地 Exchange 服务器。"

自那时起，微软专注于[修补 Exchange Server 中的漏洞](https://www.theregister.com/2023/01/28/microsoft_patch_exchange_servers/)，但坚决拒绝更改支持结束日期。但积极的一面是，当下一个版本发布时，微软承诺将延长支持至2025年10月14日之后。

Skype for Business Server 2019 也将于2025年10月14日退役，尽管在2022年，微软[表示](https://techcommunity.microsoft.com/t5/skype-for-business-blog/skype-for-business-server-support-skype-for-business-app-sdk-amp/ba-p/3599779)将"继续支持 Skype for Business Server"直至该日期之后。

微软还在 2020 年的 Ignite 活动上宣布，计划推出一款无版本的本地解决方案，名为 vNext，称从 Skype for Business 2019 升级到 vNext 将会很顺利，新服务器将是“未来将用户迁移到 Teams 的最简单路径”。

2023 年，分析机构 Directions on Microsoft [询问](https://www.directionsonmicrosoft.com/blog/where-is-skype-for-business-server-vnext/) 了一个简单的问题：“Skype for Business vNext 在哪里？”

这是一个很好的问题，一些组织已经着手回答，通过迁移到不同的平台。例如，瑞典当局正在[寻找替代方案](https://www.theregister.com/2024/04/08/tele2_collaborate/)。

我们询问了微软关于 Skype for Business vNext 的行踪，如果公司有任何信息分享，我们将进行更新。

所有这些都突显了微软面临的风险，因为许多产品即将结束支持。尽管公司希望并预计大多数客户将转向订阅模式和持续支持，但同样存在这样的可能性，即这些客户可能会借此机会考虑其他选择。 ®
