# 持续的社交媒体滚动对眼动影响不利 - 视光医疗顾问

> 来源：[https://www.optometryadvisor.com/home/meetings/aaopt-2021/social-media-scrolling-detrimental-to-ocular-health/](https://www.optometryadvisor.com/home/meetings/aaopt-2021/social-media-scrolling-detrimental-to-ocular-health/)

| 本文是视光医疗顾问关于2021年美国视光学院年会的会议报道的一部分，该会议于2021年11月3日至6日在波士顿举行。*视光医疗顾问*团队将报道AAO主要眼护理专家在该会议上展示的多项研究成果。请关注更多来自[AAO 2021视光学会议](https://www.optometryadvisor.com/home/meetings/aaopt-2021/)的内容。 |
| --- |

据在波士顿举行的2021年美国视光学院年会上的研究结果显示，持续的社交媒体滚动对眼动有害。此外，他们还探讨了有氧运动后由中枢神经系统引起的眼动疲劳和肌肉骨骼疲劳之间的潜在联系。

研究人员进行了一项交叉研究，调查社交媒体上的垂直滚动如何影响垂直和水平扫视。此外，它还探讨了有氧运动后是否存在眼动疲劳或肌肉骨骼疲劳的潜在联系。

研究要求12名年龄在18至55岁之间的参与者在两个会话期间在社交媒体平台上滚动15分钟。研究人员使用内置应用程序测量了滚动速度，并在其中一个会话期间，参与者在静止自行车上骑行15分钟，心率为最大心率的60%。使用基于视频的眼动追踪器测量了上/下和左/右半场的1°、3°、5°、7°和10°幅度的垂直和水平扫视。

结果显示，垂直扫视峰值速度减少（21%至34%），当靠近任务前进行骑行时进一步减少（33%至41%）。此外，扫视不准确性增加（15%至19%），当靠近任务前进行骑行时进一步增加（20%至28%）。在完成近任务后，水平扫视峰值速度也显示类似但较小的减少（14%至29%），水平扫视增益也减少（12%至16%）。当将近任务与骑行结合时，这种效应加剧（峰值速度：19%至31%，不准确性：16%至21%），但在任何实验条件下，扫视潜伏期的变异性都没有。

研究人员解释说，虽然我们知道人类的[视觉](https://www.optometryadvisor.com/systemic-disease/vf-defects-worse-in-odms-with-incomplete-bvn/)系统在近距工作需求下受到挑战，但由于在浏览社交媒体内容时涉及水平和垂直眼动，此类滚动对眼动系统的影响仍不明确。

“总体来说，我们的研究结果表明，持续的社交媒体浏览对眼动有不利影响，”报告称。
