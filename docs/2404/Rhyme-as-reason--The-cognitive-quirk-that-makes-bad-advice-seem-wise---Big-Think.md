# 押韵即理由：让坏建议看起来聪明的认知怪癖 - Big Think

> 来源：[https://bigthink.com/neuropsych/rhyme-as-reason/](https://bigthink.com/neuropsych/rhyme-as-reason/)

很多年以前，当我还是个年轻人时，我喝了点酒，醉了，第二天早上，我告诉我爸爸早餐时发生的事情。“我们在餐馆喝了一些酒，”我呻吟着说，“然后在马克家喝了一些啤酒。这似乎不足以让我感觉这么糟。”我爸爸笑了笑，那种心知肚明的笑声。他接着说了几句至今让我记忆犹新的话：“啤酒在酒前，感觉就好；酒在啤酒前，感觉怪。” 

多年以后，我相当确定我爸爸是在玩弄格言式的伪科学，但关键是，在他多年传授的许多建议中，只有一些在我的记忆中留下了印象。而那些正是那些*押韵*的建议。有一个记忆的重量在于一句精妙的短语中，而押韵的句子比整本书的学习内容更容易被记住。

更重要的是，人类倾向于认为押韵的智慧比非押韵的更有信誉。如果我爸爸说：“不要在啤酒之前喝酒，”我会哼哼唧唧地把它抛之脑后。然而，由于他的言辞*押韵*，我却信以为真。

这种现象被称为“押韵即理由效应”，是一种具有很多现实生活和实际应用的认知现象。

## Too many names

有   有一个奇怪且常常令人沮丧的事实：你或许能记住你童年时某首 obscure 歌曲的所有六句歌词，但却记不住上周的事情。原因在于人类的大脑更容易将事物记住为押韵的诗歌、歌曲或诗篇，而不是平凡的散文。早期文学的主要例子——如《吉尔伽美什史诗》、《伊利亚特》和《诗篇》——都是以诗歌形式写成的。这不仅使得朗诵变得精彩，还意味着人们可以轻松地记住大量内容。

1995年，认知心理学家大卫·鲁宾写了一本名为《口头传统中的记忆》的书。在书中，他探讨了文学文化和口头文化之间的差异，并列出了允许后者存在的必要因素。其中一个因素就是押韵的使用。为了证明这一观点，鲁宾做了一个实验。他让他的学生试着用两个押韵的词记住一首民谣。然后，他将他们的表现与使用非押韵词的另一组进行比较。押韵组的表现远远超过了非押韵组。

为什么会这样？鲁宾假设这一切都取决于我们记忆事物的方式。假设你正试图记住曾经见过一次的某人的名字。在没有任何提示或指针的情况下，你的大脑有几乎无限的选择来迷失方向。但是，有了押韵的帮助，选择就变少了。路径只会指向一个或少数几个地方。例如，“那个人是如此严肃，他的名字是[空白]。” 只有少数几个记忆辅助选项可以填入这个空白处。

更近期的研究似乎显示[鲁宾是对的](http://sofferpsychmemory.weebly.com/semantic-acoustic-and-visual-levels-of-encoding.html)。当我们使用押韵时，我们将信息分解为更易于存储和检索的形式 —— 这被称为[语义编码](http://sofferpsychmemory.weebly.com/semantic-acoustic-and-visual-levels-of-encoding.html)。

## 记忆深刻并不意味着更好。

押韵作为理由效应的第二个方面是，我们不仅更容易保留和回忆押韵的内容，而且更倾向于对其评价更高。弗里德里希·尼采非常喜欢谚语。事实上，他写了一系列完全由谚语组成的书籍，因为他知道一个精心制作的词语转折比散文更快地切入你的理解。在《欢乐的科学》中，他写道，“即使到现在...我们中最聪明的人有时仍会被节奏所迷惑 —— 如果我们有时因为一个想法有韵律的形式而认为它更真实，那只是因为它跳跃出来并呈现得非常神圣。” 如果你能雄辩或诗意地表达，就更容易操控并吸引他人。人们经常将冗长误认为是权威。

两位心理学家，马修·S·麦克隆和杰西卡·托菲巴赫什，调查了这一现象，并在名为“[济慈启发式：谚语解释中的押韵作为理由](https://www.sciencedirect.com/science/article/abs/pii/S0304422X99000030)”的论文中详细阐述了他们的发现。他们向参与者展示了一系列谚语。一些参与者得到了押韵的谚语；其他人则得到了用非抒情同义词重写的谚语。例如：

（押韵）：小小的笔触将推倒巨大的橡树。

（同义词）：小小的笔触将推倒巨大的树木。

他们发现，参与者不仅在审美上更偏爱押韵的句子，而且他们往往认为这些句子更准确。尽管参与者认为美学特质并不能赋予真理，但他们无法避免自己的偏见。这导致团队将其称为“济慈启发式”（Keats heuristic），以诗人[约翰·济慈](https://www.poetryfoundation.org/poems/44477/ode-on-a-grecian-urn)的句子“美即真，真即美”为名。事实证明，我们不仅更倾向于更好地记住押韵的内容，而且更高度地评价它们。

这里有两个重要的教训需要从中学到。第一个是押韵是一个很好的记忆方法。如果你想记住它，就押韵吧。第二个是要明白，仅仅因为你用押韵记住了某件事，并不意味着它是准确的。我爸爸的智慧，“啤酒在前，葡萄酒在后，你会感觉良好；葡萄酒在前，啤酒在后，你会感觉古怪”，听起来可能很好，但基于不足和可疑的证据。更好的是像“喝多了酒，头痛欲裂”。请记住，一个优美的措辞未必比一个笨拙的措辞更好。
