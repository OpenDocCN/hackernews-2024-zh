# 安妮·贝桑特（Annie Besant）和查尔斯·里德比特（Charles Leadbeater）的《*Occult Chemistry*》（1908年） — 公共领域审查

> 来源：[https://publicdomainreview.org/collection/occult-chemistry/](https://publicdomainreview.org/collection/occult-chemistry/)

[](#p-1-0)

Besant 和 Leadbeater 在《*Occult Chemistry*》中的研究始于“空气中的四种气体” — 氢、氧、氮和“一种原子量为3的第四种气体，迄今为止化学家尚未发现。” 这些是最简单的元素。随着原子序数的增加，它们的原子的几何复杂性也增加。例如，钠“由一个上部分组成，可分为一个球体和12个漏斗，一个下部分，同样划分；以及一个连接棒。” 原子们被合理地分组，而不是按照周期表的方式（按原子序数排列，显示不同元素的相关化学性质），而是根据整体形状：尖刺、哑铃、四面体、立方体、八面体、交叉基座和星形。因此，硼、氮和钒等元素 — 尽管在化学上几乎没有共同之处 — 都“有六个漏斗，开口在一个立方体的六个面上。”
