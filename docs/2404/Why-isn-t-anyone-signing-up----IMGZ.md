# 为什么没有人注册？ - IMGZ

> 来源：[https://imgz.org/blog/2024/02/25/why-isnt-anyone-signing-up/](https://imgz.org/blog/2024/02/25/why-isnt-anyone-signing-up/)

四年前，我们就在这个博客上写了一篇文章，说明[我们的增长是指数级的](https://imgz.org/blog/2020/12/23/haha-suckers/)。事实上，当时情况看起来很乐观，当时我们对IMGZ的增长前景充满了无限的乐观。毕竟，一天之内从零增加到四名用户（从零开始）只能让你合理地推断一个月内会有数万亿的用户。

然而，事情却一直停滞不前。四年来，我们设法保留了微不足道的30名用户，其中5名是朋友和家人，其余的我甚至不确定他们是否支付了任何费用，因为IMGZ每年的费用是10美元，所以我确实需要仔细看看银行账户中是否有任何钱。

与此同时，服务器成本不断攀升。我们摇摇欲坠的基础设施维护费用昂贵，最近我们又遭遇了额外的灾难：垃圾邮件机器人决定滥用我们的注册系统，并为发送的注册邮件增加了电子邮件提供商的账单。这使我们的托管成本增加了十倍，从每月2美元增加到每月20美元。

## 财务状况

现在，我不是会计巫师，但如果你每月赚5美元，却要支付20美元，这听起来不像第一个数字比第二个数字大，所以情况听起来很严峻。当然，我们都可以去一些高薪的FAANG公司找工作，但那样我们就必须妥协，要*隐藏*我们对用户的敌意。至少对他们的敌意程度不会改变，但这只是一种小小的安慰。

因此，我们需要采取极端措施，并进行一些令人害怕的*产品开发*。看来那个黑暗的日子已经来临，我们需要与用户交流了。

我们不是在寻找一些使用IMGZ的人，而是要找出*不*使用IMGZ的用户，并问问他们为什么不用。这背后的理由是，使用IMGZ的用户已经支付费用，所以他们已经没有更多的钱可以为我们提取，而不使用IMGZ的用户则是一个真正未开发利用潜力的泉源，可以通过增加我们的估值来增加100美元。

## 产 品

所以，我们正在尝试一种开发者零社交技能可以做到的最好的事情：写一篇博客文章。

亲爱的非客户，为什么你们不使用IMGZ并为我们的服务付费呢？是因为告诉你们滚蛋并永远不要注册这项服务的文字吗？因为我们确实是这么想的，但显然有些人已经坚持到了足够长的时间，患上了斯德哥尔摩综合症，并且现在相当享受这种经历。是因为图像传递效果不佳吗？我们可以做些改进，但这听起来太麻烦了。

请给我们发送电子邮件，嗯，[email protected]，因为我从来没有费心设置过imgz.org的电子邮件地址，请告诉我们为什么你不使用这项服务。我相信，在阅读这篇文章的三个人中，至少有一个会费心回复的。

我不知道我们是否打算对反馈做些什么，但如果听起来不错，为什么不呢？可能会很有趣添加一些不错的新功能。
