# 一份声明

## Jon Pretty, 26 April 2024

我是一名Scala开发者和演讲者，三年前被取消。昨天，我在伦敦高等法院听取了几名Scala社区的知名成员就他们在2021年4月27日对我的不实言论道歉。我因诽谤起诉他们，他们承认错误并达成了和解，支付了我的费用和赔偿金。

他们的指控耸人听闻且卑劣，但毫无根据。它们的来源是一名女性对2018年一段关系的怨恨，而我则违背了她的意愿结束了这段关系。她捏造或接受了一种替代叙述，这发展成了一种行为模式的指控，并最终导致被告发布了一封公开信，现在他们也同意这封信是诽谤的。

在两年的法律诉讼中，被告从未提供任何证据来支持他们的指控，并在法庭上承认他们没有适当的理由提出这些指控。他们已向法院作出承诺，不再发布类似的诽谤声明，或者让他人代表他们这样做。

未在出版前有签署人与我联系。我没有收到任何警告，对指控的实质也毫无所知。我只是在被永久排除在社区之外的同时，才发现自己被指控的内容，就像其他人一样。我没有机会为自己辩护。由于缺乏正当程序，这不是偶然导致了严重的不公正。

经历取消和忍受在线的骚乱是一种创伤。我选择了退出我所知道的生活。其后果伤害了我和身边的人，并且让我们陷入了困境。

我的就业机会被彻底毁掉了。我的慈善和教育项目，以及我的小企业，都无法继续下去。尽管我具备可转移的技能，这些指控成为程序界和各行业中公认的可转移的红旗，我自那以后几乎无法谋生。经过两年的法律诉讼，终于在一个依赖事实的论坛中得到公正的审查。这一结果终于为我平反。

许多熟悉这封公开信的人会对这一结果感到惊讶，因为一些支持者表达了如此确定的看法。但他们应该质疑为什么从未提供我的诽谤索赔的任何辩护。他们应该寻找答案，了解为什么在没有证据的情况下如此多的人会如此确信。而被告的共同签署人应重新评估他们支持这封公开信时的一切的可信度。

尽管如此，我相信这封公开信的作者们坚信这样做是正确的。我相信他们希望在对他们震惊的故事做出善意回应时展示他们的善良。我认识其中许多人多年，知道他们是善良、道德的人，只是犯了错误，而且不希望再犯错。

善良的人可能会犯错误，但并不会变成坏人。因此，我向他们提供我认为他们应得的尊重和同情。我承认他们的良好意图，并慈悲地解释其后果。我们彼此应该以礼相待，让彼此有足够的时间和空间在公开的询问中反思，没有任何强迫或紧急情况，这样每个人都能更好地理解这一结果，也更好地理解彼此。

我在这里避免了提及任何第三方，并敦促其他人也这样做。我深知成为网络骚扰的对象是什么感觉，我不希望任何人有同样的经历。我拒绝责备和升级的文化。

自2021年以来，我并没有闲着。我大部分时间都在编写Scala 3，我认为这是一个非常优秀的软件开发平台，值得获得更大的成功。我被参与其中的成功所激励，并将很快开始分享我的工作。
