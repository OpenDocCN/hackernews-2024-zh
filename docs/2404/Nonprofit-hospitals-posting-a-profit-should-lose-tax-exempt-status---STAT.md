# 非营利医院如果盈利应该失去免税地位 - STAT

> 来源：[https://www.statnews.com/2024/04/14/nonprofit-hospitals-turn-profit-charity-care-tax-exempt-status/](https://www.statnews.com/2024/04/14/nonprofit-hospitals-turn-profit-charity-care-tax-exempt-status/)

尽管勤劳的纳税人因为又一年的税表提交麻烦已经结束而松了一口气，但美国内部税务局继续让大多数美国医院[免交](https://kffhealthnews.org/news/article/nonprofit-hospitals-tax-breaks-community-benefit/)联邦税款。现在是国会审视税务局在医疗保健中作用的时候了。

机构使用模糊的“社区利益”标准慷慨地向所谓的非营利医院授予免税地位，即使其中许多医院在经济上正在通过高额医疗账单利用病人。我和几位约翰·霍普金斯大学的同事[发表](https://jamanetwork.com/journals/jama/fullarticle/2737183)了一篇在《美国医学协会杂志》上的研究，显示一些非营利医院会起诉和扣押低收入病人的工资，这些病人无力支付他们的医疗账单。这样做哪里有社区利益？非营利医院应该是有同情心和怜悯心的，而不是掠夺性和无情的。

许多医院强迫那些急需治疗和真诚寻求帮助的人在接受治疗的条件下签署他们的财务生活的文件。有些使得[争议账单](https://www.nytimes.com/2022/06/30/well/live/surprise-medical-bills.html)变得困难，而其他一些则[不遵守](https://thehill.com/policy/healthcare/4107837-just-over-a-third-of-hospitals-are-complying-with-price-transparency-rules-report/)联邦[医院透明度规则](https://www.cms.gov/priorities/key-initiatives/hospital-price-transparency/hospitals)，该规则要求医院公布常见可购服务的价格。最近非营利医院正在增加数十亿美元的[“设施费”](https://www.wsj.com/health/healthcare/hidden-hospital-fees-cost-patients-hundreds-of-dollars-0024cd95)用于常规护理，这些账单花招正在欺诈普通美国人。如果国会通过一项法律，使使用这些费用的医院失去得到的税收豁免权益，这些账单花招就能在一夜之间消失。

许多免税医院辩称他们提供数百万美元的免费护理。但实际情况是这样的：他们称之为免费护理的很多内容实际上是法律要求的急救护理（1986年的[急诊医疗援助和劳工法案](https://www.cms.gov/medicare/regulations-guidance/legislation/emergency-medical-treatment-labor-act)要求医院对任何因急症而来的人提供护理）。但在提供护理后，患者经常被追讨付款，通常是以人为高价。然后，医院报告他们的高标价与实际收取的差额作为慈善护理。

但这不是慈善。具有讽刺意味的是，2021年发表在《健康事务》上的一项研究发现，营利性医院比非营利性医院提供更多65%的慈善护理。

获得免税地位的标准不应仅仅是遵守法律。

医院领导人告诉我，他们需要避免缴纳联邦税款才能生存。但是税收工作的方式是，企业只对其利润征税。一些医院通过成本转移会计和贷款再融资来隐藏他们的利润，甚至参与积极的房地产和风险投资。根据[公共税务记录](https://projects.propublica.org/nonprofits/organizations/951644600)，比佛利山的西达医院在2022年有7.5亿美元的盈余，并且未缴纳联邦税款。纽约的斯隆·凯特琳癌症中心也有4亿美元的盈余，同样未缴纳联邦税款。该医院的前三名管理人员年薪近2000万美元。

尝试在这些或任何非营利性医院预约，看看是否能在没有保险或提前现金的情况下做到。几乎是不可能的。大多数非营利性医院会拒绝安排预约，甚至癌症治疗，除非你通过展示支付能力来符合经济资格。

如果医院只为经济上有资格的人安排约会和手术，称他们是一家企业，他们应该像其他美国企业一样支付他们公平的税款。凯撒家庭基金会的一项[研究](https://www.kff.org/health-costs/issue-brief/the-estimated-value-of-tax-exemption-for-nonprofit-hospitals-was-about-28-billion-in-2020/)估计，2020年非营利性医院的免税价值总额约为280亿美元。根据我的计算，这相当于每年提交报税的美国人获得107美元的税收抵免。

在停止这种系统性作弊的势头正在增长。今年八月，包括比尔·卡西迪和伊丽莎白·沃伦在内的美国参议院两党议员要求IRS在此问题上更多透明度，此前，[美国政府问责办公室的一份报告](https://www.gao.gov/assets/gao-23-106777.pdf)发现，该机构“没有很好地记录流程”，以确保医院符合免税标准。

十月，参议院健康、教育、劳工和养老金委员会主席伯尼·桑德斯强调了解决问题的必要性。他在[一份声明中](https://www.help.senate.gov/chair/newsroom/press/news-chairman-sanders-releases-new-report-showing-major-nonprofit-hospital-systems-exploiting-tax-breaks-and-prioritizing-ceo-pay-over-helping-patients-afford-medical-care)表示：“全国许多非营利医院系统未能为低收入美国人提供他们非营利地位所要求的负担得起的医疗服务，尽管他们获得数十亿美元的税收优惠，并向其高级执行人员提供极高的薪酬套餐。”

2023年，宾夕法尼亚州法官吊销了一家医院的财产税豁免权，这标志着政府官员对医院不纳税的不满。此举是在波茨敦学区起诉该医院之后采取的行动，学区认为该医院的特殊税收待遇导致学校每年减少[$900,000的收入](https://whyy.org/articles/pa-hospitals-revoked-property-tax-exemption/)。

不仅令人难以置信的是IRS对医院的漫不经心态度，还有该机构对医生制定的直接初级护理和其他创造性策略的敌意，以解决慢性病问题。IRS不允许订阅直接初级护理（一种以预防为重点的密集初级护理，需要额外时间）的个人或家庭将其费用作为可减税的健康储蓄账户支出。[2019年白宫行政命令](https://trumpwhitehouse.archives.gov/presidential-actions/executive-order-improving-price-quality-transparency-american-healthcare-put-patients-first/#:~:text=potentially%20including%20direct%20primary%20care%20arrangements%20and%20healthcare%20sharing%20ministries%2C%20as%20eligible%20medical%20expenses)对此立场进行了挑战，但没有成功。上个月，IRS严厉[警告](https://www.irs.gov/newsroom/irs-alert-beware-of-companies-misrepresenting-nutrition-wellness-and-general-health-expenses-as-medical-care-for-fsas-hsas-hras-and-msas)美国人，不能将他们自己的健康储蓄账户用于购买医生指导下的健康食品。

IRS继续推行对疾病的反应性而不是健康的主动性方法。慢性病是美国的主要死亡原因，并消耗了美国人在[医疗保健上支出的大部分4.5万亿美元](https://www.cms.gov/data-research/statistics-trends-and-reports/national-health-expenditure-data/nhe-fact-sheet)。当前的打地鼠系统行不通。大多数美国成年人定期服用[4种或更多处方药](https://www.consumerreports.org/prescription-drugs/too-many-meds-americas-love-affair-with-prescription-medication/)，使美国成为全球服药最多的国家。该国需要尝试[新的健康方法](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC7023959/)。

IRS不应该责备那些选择直接初级医疗或想要吃更健康食物的美国人，而应该让医院对他们应该缴纳的税款负责任。

*Marty Makary, M.D., M.P.H., 是约翰霍普金斯医学院和凯里商学院的教授，也是即将出版的书籍* [*“盲点：当医学出错时及其对我们健康的影响”*](https://www.bloomsbury.com/us/blind-spots-9781639735310/) *(Bloomsbury出版社，2024年9月)*。
