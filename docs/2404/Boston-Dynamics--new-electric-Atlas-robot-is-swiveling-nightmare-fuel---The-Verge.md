# 波士顿动力的新电动Atlas机器人正在旋转的噩梦燃料 - The Verge

> 来源：[https://www.theverge.com/2024/4/17/24133145/boston-dynamics-resurrects-atlas-humanoid-robot-electric-new](https://www.theverge.com/2024/4/17/24133145/boston-dynamics-resurrects-atlas-humanoid-robot-electric-new)

它还活着！在宣布[退役液压机器人Atlas](/24132451/boston-dynamics-atlas-robot-retirement)一天后，波士顿动力推出了[新的全电动版本](https://bostondynamics.com/blog/electric-new-era-for-atlas/)的人形机器人。下一代Atlas机器人的设计旨在比其前身具有更广泛的运动范围。

波士顿动力希望新版本能够展示Atlas在保持人形的同时，不限制“两足机器人的运动方式”。新版本经过重新设计，配备了可旋转的关节，公司声称这使其“独特地能够处理乏味、肮脏和危险的任务”。

“我们设计了Atlas的电动版本，使其更强壮、更灵活”，公司在其新闻稿中表示。“Atlas可能看起来像人类形态，但我们装备了这款机器人，使其能够以最有效的方式完成任务，而不受人类运动范围的限制。Atlas将以超越人类能力的方式移动。”

展示这款新机器人能力的预告片既令人不安又戏剧化。[视频](https://go.skimresources.com/?id=1025X1701640&xs=1&url=https%3A%2F%2Fwww.youtube.com%2Fwatch%3Fv%3D29ECwExc-_M)从Atlas以类似尸体的方式躺在地板上开始，然后迅速将其腿部向后折叠至身体上，以一种适合某种Cronenberg式身体恐怖电影的方式站立起来。其弯曲的、发光的头部确实增添了一些皮克斯灯的魅力，但Atlas然后在腰部旋转并朝摄像机前进的方式确实感觉相当令人震惊。

设计本身也更接近人类形态。类似于[特斯拉的Optimus](/2023/12/13/23999559/teslas-new-optimus-gen-2-robot)，新的Atlas现在拥有更长的四肢，更直的背部，以及一个可以根据需要旋转的明显“头部”。视野中没有电缆，其“面部”包括内置的环形灯光。这是对其前身的显著改进，现在还配备了一些波士顿动力的新人工智能和机器学习工具。

公司还表示，它正在继续构建上一代液压Atlas机器人已实现的能力，例如提升和操作物体，同时探索“几种新的夹持器变体”，以使机器人适应各种不同的行业环境。

波士顿动力公司表示，新的[Atlas](https://www.bostondynamics.com/atlas)将在未来几年内与一小部分客户进行测试，首先是现代汽车公司。其他来自竞争对手如[Figure](/2024/1/18/24043065/bmw-figure-robotics-humanoid-robot-workers-deployment)和[Apptronik](/2024/3/15/24101791/mercedes-robot-humanoid-apptronik-apollo-manufacturing)的人形机器人也在为宝马和奔驰的生产线进行类似的测试。
