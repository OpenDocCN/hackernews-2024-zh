# 特斯拉不是下一个福特，而是下一个康埃德。- 大西洋月刊

> 来源：[https://www.theatlantic.com/technology/archive/2024/04/tesla-cars-batteries-power-company/678168/](https://www.theatlantic.com/technology/archive/2024/04/tesla-cars-batteries-power-company/678168/)

近来，特斯拉的汽车似乎有些危险。它们的自动驾驶功能与数百起事故和超过十几起死亡事件有关。然后，在本月早些时候，该公司召回了其整个Cybertruck车队。正如*InsideEVs*所说，一个困扰其油门踏板的机械问题，“可能会把这个不锈钢梯形车体变成一枚重达6,800磅的地面导弹。”

在此过程中，特斯拉——未回复多次请求评论——[辩称](https://www.theverge.com/2023/10/31/23940693/tesla-jury-autopilot-win-liable-micah-lee)其汽车和自动驾驶软件。截至上周，该公司告诉联邦监管机构，Cybertruck的故障[尚未](https://static.nhtsa.gov/odi/rcl/2024/RCLRPT-24V276-7026.PDF)与任何事故或伤害相关联。但即使解决了所有安全问题，也可能无法阻止特斯拉整个电动汽车业务的危险性。昨天下午，全球市值最高的汽车公司发布了其2024年第一季度的[财报](https://digitalassets.tesla.com/tesla-contents/image/upload/IR/TSLA-Q1-2024-Update.pdf)，宣布其净收入较一年前下降了55%。在随后不久的投资者电话会议上，埃隆·马斯克只能模棱两可地描述成为一个特别灾难性月份的车辆巨无霸：“应对了几个意料之外的挑战。”仅仅在四月份，特斯拉宣布了自2020年以来的首次销量下降，召回了一条生产线的车辆，并据称取消了另一条生产线的计划，并开始大规模[裁员](https://www.bloomberg.com/news/articles/2024-04-15/tesla-to-cut-over-10-of-its-global-workforce-electrek-reports)。对于这个月来说，还有六天时间，情况可能会变得更糟。

马斯克能否维持其电动车帝国现在成为疑问。他告诉投资者，特斯拉现在的主要重点是人工智能和自动驾驶汽车。但即使这种转型失败，公司也已经将自己定位在绿色转型的另一个、也许更为关键的领域：提供和储存美国的能源。特斯拉的电动车充电站正在崛起，如果不是主导地位的话，其巨大的电池则用于家庭甚至整个社区的可再生能源储存。与去年同期相比，特斯拉能源业务的利润增长了140%，马斯克昨天还断言这一部门的增长速度将继续“显著快于汽车业务”。因此，特斯拉的未来可能不在于追随福特的步伐，而更像是杜克能源和康埃迪森公司。换句话说，特斯拉正在转型成为一个公用事业公司。

特斯拉的核心问题在于其汽车正落后于行业发展。尽管销量下滑，该公司仍然是美国最大的电动车制造商，其汽车销售收入仍远远超过来自能源存储的收入。但特斯拉的车型，曾经无可否认的高科技和酷感，正在老化。

Cybertruck于11月首次亮相，但特斯拉仅售出大约4,000辆，比福特F系卡车[平均两天销售](https://media.ford.com/content/dam/fordmedia/North%20America/US/2024/01/04/Q4%202023%20Sales%20Final.pdf)的数量还少。此外，特斯拉在四年多内未发布全新的乘用车型。其竞争对手利用这段时间赶上了。中国品牌比亚迪正在生产[便宜时尚的汽车](https://www.theatlantic.com/technology/archive/2024/01/electric-cars-cheaper-china-than-america/677290/)，并最近[超过了](https://www.theatlantic.com/ideas/archive/2024/04/biden-trump-chinese-cars/678093/)特斯拉成为全球销量最高的电动车销售商。虽然比亚迪的汽车在美国不可用，但Rivian、现代和福特等汽车制造商正在销售高科技电动汽车。如今，美国人想要的不仅仅是高科技的电动车型，而是价格实惠的电动车型——即使特斯拉逐步[降低](https://www.reuters.com/business/autos-transportation/tesla-slashes-prices-model-3-models-y-vehicles-us-2023-10-06/) [售价](https://www.nytimes.com/2024/04/22/business/dealbook/tesla-earnings-price-cuts-recall.html)，也未能实现这一点。在四月的又一次失败中，*路透社* [报道](https://www.reuters.com/business/autos-transportation/tesla-scraps-low-cost-car-plans-amid-fierce-chinese-ev-competition-2024-04-05/) 称公司取消了一款备受期待的更实惠车型，售价仅为25,000美元。马斯克昨天告诉投资者，公司正在加快更实惠车型的时间表，这些车型将“在我们当前车型生产线上建造”。但他没有具体说明价格，并拒绝直接回答是否为全新车型或[对现有车型进行调整](https://www.reuters.com/technology/tesla-jumps-musks-promise-more-affordable-cars-eases-growth-fears-2024-04-24/)的问题。

公司在电动车领域仍然拥有一个巨大优势。无论它们的制造商是谁，几乎所有未来在美国销售的电动车都将依赖于特斯拉。就像加油站对高速公路系统的可用性至关重要一样，电动充电站是推广电动车使用的关键障碍。特斯拉的超级充电站比许多竞争对手的充电站速度更快、更可靠，这就是为什么大多数主要汽车制造商已宣布他们将在未来的车辆中采用特斯拉的专有充电接口的原因。全国范围内的超级充电站数量已经稳步增长，[了解更多](https://digitalassets.tesla.com/tesla-contents/image/upload/IR/TSLA-Q1-2023-Update)，而且预计在本十年内将迎来飞速增长。

在未来几年，特斯拉的超级充电站可能也将全部依靠特斯拉的电池供电，这是公司转型成为能源供应商的鲜为人知的核心。随着美国继续转向清洁能源，储能将变得至关重要：太阳能和风能[现在是并将继续是](https://www.eia.gov/todayinenergy/detail.php?id=61242)该国增长最快的可再生能源，但能源网格不能在夜晚、阴天或微风停止时停止运转。就像特斯拉在十多年前领先于电动车采用曲线一样，它现在正准备成为电池热潮的霸主。

自2019年以来，该公司一直在向[纽约](https://www.tesmanian.com/blogs/tesmanian-blog/tesla-megapack-units-to-power-the-largest-battery-system-in-new-york-city)、[马萨诸塞州](https://carboncredits.com/tesla-413m-megapacks-revolutionize-energy-grid/)、[加利福尼亚](https://electrek.co/2022/04/18/tesla-megapack-project-730-mwh-pge/)、[迪拜](https://driveteslacanada.ca/energy/tesla-megapack-facility-in-dubai-goes-online/)、[澳大利亚](https://www.teslarati.com/edify-tesla-megapack-150mw-300mwh-system/)、[英国](https://www.datacenterdynamics.com/en/news/uk-builds-europes-biggest-battery-out-of-tesla-megapacks/)等地的电网运营商以及包括[Apple](https://www.cnbc.com/2021/04/01/apple-will-use-tesla-megapack-batteries-at-its-solar-farm-facility.html)在内的私人客户销售“Megapacks”——这些巨大的电池能够临时为数千户家庭供电。特斯拉正在继续扩建加利福尼亚的工厂，以生产这些电池，并在上海建设另一个工厂。直到最近，[竞争并不激烈](https://electrek.co/2024/04/12/catl-unveils-tesla-megapack-competitor-claims-zero-degradation-tener/#:~:text=CATL%20unveils%20Tesla%20Megapack%20competitor%2C%20claims%20zero%20degradation%20and%20more%20capacity,-Fred%20Lambert%20%7C%20Apr&text=CATL%20has%20unveiled%20Tener%2C%20a,zero%20degradation%20after%205%20years.)，一些分析师甚至[预测](https://www.investing.com/news/stock-market-news/tesla-megapack-business-could-be-worth-substantially-more-than-car-business--rbc-capital-3320046)未来Megapack业务有望价值“大大超过”特斯拉的汽车业务。

[阅读：特斯拉的魔力已被压缩到其充电站](https://www.theatlantic.com/technology/archive/2023/06/tesla-charging-stations-ford-gm-nacs/674423/)

特斯拉还销售Powerwall，这是专为家庭安装设计的大型电池。 自2018年以来，Powerwall已占据了所有家庭电池安装的大约一半，并且需求预计将会激增。 自2023年以来，公司部署的能量存储量是前一年的两倍以上。 特斯拉还拥有一系列太阳能电池板，尽管这一业务已被证明是不稳定的，但这仍然是公司提供所需的原始电力的另一种方式。 通过其充电器和电池，特斯拉的主要产品正变得基础设施化，远离消费者，但同样重要。 特斯拉的首席财务官Vaibhav Taneja昨天表示，今年的能量存储部署预计将增长至少75%，并开始“显著贡献我们的整体盈利能力。”

当然，这种未来远非注定。 特斯拉的汽车业务仍然是该国为数不多的盈利电动车运营之一； 福特和通用正在EV上输掉数十亿美元，因为它们正重新调整公司远离内燃机。 而且，可以说，马斯克几乎不是一个可预测的执行者。 昨天的收益电话表明，他对自动驾驶的“机器人出租车”更感兴趣而不是电气化网络：他在三个月内将特斯拉的AI培训资源翻了一番。 但自动驾驶汽车恰恰是一个不安全的赌注，而且半自动驾驶车辆，这已经成为行业标准，将不再使特斯拉脱颖而出。 清洁能源是一个竞争激烈、资本密集且迅速变化的行业。 就像特斯拉在EV领域的巨大领先优势一样，其电池和充电优势也将无法自我维持。

但除非发生更为灾难性的崩溃，否则特斯拉似乎成功地从清洁能源革命的一波跃升到另一波——从提供汽车到提供不仅是汽车，还包括家庭、办公室以及几乎所有其他东西所需的电力。 十年后，即使特斯拉车辆的流行度下降，公司的影响力可能比以往任何时候都更强大。
