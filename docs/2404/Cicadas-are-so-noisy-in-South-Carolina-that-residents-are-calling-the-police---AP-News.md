# 南卡罗来纳州蝉鸣声如此惊人，居民竟向警局报警询问声源|AP新闻

> 来源：[https://apnews.com/article/noisy-cicadas-south-carolina-police-39f2bd760a51c8c1d28d88b12473840e](https://apnews.com/article/noisy-cicadas-south-carolina-police-39f2bd760a51c8c1d28d88b12473840e)

新伯里郡，SC（AP）— 在南卡罗来纳州的某个县，不断出现的[蝉叫声如此之大](https://apnews.com/article/periodical-cicada-invasion-bugs-everywhere-locusts-evolution-e57f708cdcf144a20fe8f694e95b5d44)，以至于当地居民致电警局询问为何能听到警笛声或巨大的轰鸣声。

新伯里郡警长办公室在周二的一则Facebook消息中通知人们，他们所听到的尖叫声只是雄性蝉在经历了十多年休眠后用以吸引配偶的鸣唱声音。

有几个人甚至叫停了副手，询问这噪音到底是怎么回事，新伯里郡警长李·福斯特说道。

在38,000人口，距离哥伦比亚西北约40英里（65公里）的新伯里县内，蝉群最为活跃，随着周二的进行，来自不同地区的呼叫不断增多。

美国东部长达数万的[红眼周期性蝉](https://apnews.com/article/cicadas-fungus-zombies-pee-urination-pumps-7bf34c2d7a3955efc13cfa05e92f27cb)本月从地下破土而出。这些蝉群具有周期为13或17年的循环。

它们群体的歌声能与喷气发动机的音量相匹敌，研究它们的科学家们通常会携带耳塞来保护听力。

在周二之后，福斯特明白了原因。

“尽管对某些人来说，噪音令人恼火，但它对人类或宠物并无危险”，福斯特在对县内居民的声明中写道，“遗憾的是，这正是大自然的声音。”
