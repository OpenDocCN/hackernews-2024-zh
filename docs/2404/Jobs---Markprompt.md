# 职位 | Markprompt

> 来源：[https://markprompt.com/jobs](https://markprompt.com/jobs)

我们是一个紧密团结的团队，位于旧金山Dogpatch。我们的客户群体从快速增长的初创公司到已建立的企业公司不等，我们致力于倾听每一个客户并帮助他们成功。我们的开发支柱是安全性、可靠性和性能，结合务实主义，始终寻找可行的解决方案，并对客户反馈和请求做出快速响应。我们在基础设施和产品层面工作，努力构建正确、未来可靠的解决方案，品味出众。

我们得到了Y Combinator的支持，以及Vercel、Stripe、GitHub、Dropbox和Algolia等公司的创始人和领导的支持。

### 职位：创始工程师

作为创始工程师，您将直接与我们的客户互动，并将与我们当前的技术栈一起工作：TypeScript、Supabase、Next.js、Vercel和[Effect](https://effect.website/)。主题将包括：并发系统、模块化服务基础设施、文本处理、RAG、数据库优化、LLM工程、日志和遥测、CI/CD、与第三方系统的集成、前端。

### 要求

+   在Web技术方面有丰富的经验，特别是JavaScript/TypeScript

+   有将代码推向生产的强大经验

+   热爱从零开始构建产品

+   渴望成为创始团队的一部分

+   坐标旧金山或愿意搬迁

### 加分项

+   熟悉Effect

+   有部署LLM本地应用到生产环境的经验

+   曾经创办过公司或建立过高影响力项目
