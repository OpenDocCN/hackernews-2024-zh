# 协议 - 无用之事杂志

> 来源：[https://www.futilitycloset.com/2024/04/19/protocol-3/](https://www.futilitycloset.com/2024/04/19/protocol-3/)

1995年，新墨西哥州参议院提出了一个令人吃惊的修正案——要求心理学家在为被告的能力作证时穿着巫师服装：

> 当心理学家或精神科医生在被告的能力听证会上作证时，应当戴上不少于两英尺高的锥形帽子。帽子表面应印有星星和闪电。此外，心理学家或精神科医生应当戴上不少于18英寸长的白色胡须，并在关键时刻用魔杖刺空气。每当心理学家或精神科医生就被告的能力作证时，法警应即时调暗法庭灯光，并敲击一下中国式的锣。

参议院全体一致通过了该措施，准备提交众议院审议，此时发起人邓肯·斯科特解释说，他原本是想写个讽刺——他觉得太多心理健康从业者在充当专家证人。这项提案最终被撤回，未能成为法律。
