# Ronda Rousey：“我从未想过要谈论脑震荡。那感觉像是一种软弱” | Ronda Rousey | 卫报

> 来源：[https://www.theguardian.com/sport/2024/mar/31/ronda-rousey-i-never-wanted-to-talk-about-concussion-it-felt-like-a-weakness](https://www.theguardian.com/sport/2024/mar/31/ronda-rousey-i-never-wanted-to-talk-about-concussion-it-felt-like-a-weakness)

‘我担心这个问题，因为我们家族中已经有阿尔茨海默病和痴呆症患者，而那些家庭成员并没有频繁被打在头部，” [朗达·劳西](https://www.theguardian.com/sport/ronda-rousey) 在考虑到由脑震荡所带来的后果和她为女性打破的众多障碍之前以及一次毁灭性的陨落之后说道。

2015年，Rousey在顶峰时被《*体育画报*》称为“[世界上最具统治力的运动员](https://www.si.com/mma/2015/05/12/ronda-rousey-ufc-mma-fighter-armbar.)”。她改变了一项残酷的运动，成为[UFC的代表面孔](https://www.theguardian.com/sport/blog/2018/jan/30/ufc-ronda-rousey-wwe-mixed-martial-arts)，这个亿万美元的巨头推动了混合武术的流行。

除了成为 2012 年第一位被深具保守色彩的[UFC](https://www.theguardian.com/sport/ufc)签约的女性，Rousey 还建立了令人敬畏的 15-0 记录，其比赛平均持续时间为 34 秒。但她的凶猛建立在一个隐藏的脆弱基础上。在柔道比赛中，Rousey 遭受了多次脑震荡，她知道自己的大脑无法承受多次头部打击。在她遭受太多惩罚之前，她必须在 UFC 的战斗中以暴力收场。

现在，Rousey 可以分享她的秘密了，她回顾了多次脑震荡的后果，成为了动人而有趣的伴侣。“每次忘记了钥匙或弄丢了手机，我就会说：‘我要死了！完了！”她以滑稽的方式大声说道。

她刚刚过了 37 岁，Rousey 又开始思考起来。“我有部分能力下降了，有时候我会给女儿唱摇篮曲时说错一个词。我会想：‘天哪！这是 [痴呆的开始]！’今天早上送女儿去上学后，驾车回家的路上，经过了我曾经数百次经过的拐角处，突然间我想‘我在哪儿？’然后我想起来了。”

Ronda Rousey 在 2012 年的 Strikeforce 活动中对 Sarah Kaufman 发起了攻击。摄影：Esther Lin/Forza LLC/Getty Images

我们都会有大脑突然空白的时刻，但对于 Rousey 来说，这带着复杂的潜在影响。她与姐姐 Maria Burns Ortiz 合著的[新书](https://www.penguin.co.uk/books/456155/our-fight-by-rousey-ronda/9781529912388)常常令人震撼，并在最好的时候展现了脑震荡的真实个人历史。她 11 岁开始练习柔道，并追求赢得奥林匹克金牌的目标，但 Rousey 尝试逃避事实，“多年来我一直在遭受连续的脑震荡。”

当我问她年轻时可能遭受了多少次脑震荡时，她耸了耸肩。“很难说，因为我得到脑震荡时不会休息，我会继续训练并不断恢复它。所以我不是几天就能好，可能会持续几周甚至几个月。一年大部分时间我都在经历脑震荡的症状。虽然有不同程度的严重性，但我最严重的一次是在阿根廷举行的泛美柔道锦标赛上被摔到头部。我完全失去意识，直到第二天早上才醒来。”

Rousey 的担忧被忽视了。“人们会觉得我在抱怨头痛。有人会说：‘你头疼了？忍一忍。如果奥运会期间你头疼怎么办？’从小我就是这样处理的。这成了一种生活方式。”

她的母亲安·玛丽亚·伯恩斯成为1984年世界柔道锦标赛的首位美国冠军。然后，她失去了丈夫，罗恩·罗琳自杀身亡，罗琳当时八岁。在如此逆境中，安·玛丽亚开始指导罗琳，并帮助她在2004年世界青年柔道锦标赛上赢得金牌，并在2008年奥运会上获得铜牌。

罗琳小时候，公众对脑震荡几乎没有科学知识。“我妈妈根本不了解脑震荡，”她说，“因为在我柔道生涯的末期才开始有研究出来。我害怕它，试图压抑它。在我10年的柔道生涯中，我脑震荡次数比其他人多得多，所以当我开始做[MMA](https://www.theguardian.com/sport/mma)时，我不想让任何人知道。他们已经有足够的理由阻止我进入MMA和UFC。我不想让他们知道更多关于脑震荡的事情，而且我很幸运有足够的技能来快速赢得大部分比赛。”罗琳对那些笼中之魔和拳击运动中的无知 machismo 表示极度鄙视：“人们谈论你的‘下巴’时总是带着一种敬畏之情。它被扔来扔去，好像它是一种个性特征或吸收打击意志的标志。这也是为什么我从不想谈论脑震荡的另一个原因。它让我觉得这是一种个人弱点，而不是我从小就开始经历的神经退化。”

罗琳·罗琳（蓝）在2008年奥运会女子-70公斤柔道铜牌比赛中击败德国选手安妮特·波姆途中。摄影：Olivier Morin/AFP/Getty Images

她把眼镜往鼻梁上提了提。“真糟糕，因为你看看这些拳击手的下场。[穆罕默德·阿里](https://www.theguardian.com/sport/muhammad-ali?page=2)是我的英雄之一，他有着最出色的下巴。但看看发生了什么。我不会评判任何人，因为我也愿意接受生活在轮椅上，如果那是我为了取得所有成就必须付出的代价。我尊重阿里愿意过那种生活，因为那正是我也试图做的。

“我希望我不会以那种方式结束，但你永远不知道。可能几十年后你会明白你已经经受了太多打击。当你有了孩子和家庭，就更难去赌博你的未来。我从成为地球上最理想的单身女士一夜间成为了立即成家立业的人，这完全改变了你的优先事项。”

罗琳体验过名利的漩涡，现在她很高兴地与丈夫特拉维斯·布朗（前UFC选手）一起经营一家再生农场，那里有两个十几岁的男孩。这对夫妇有两个年幼的孩子，罗琳在家人和动物的环绕下，找到了一种方法来治愈自己，在她的UFC统治悲惨结束后。

劳西书中最有力的篇章记录了她的 [第一次惨败](https://www.youtube.com/watch?v=vGhdcHWdW4o)，当前拳击手霍利·霍姆在2015年11月在墨尔本UFC史上最大的5.6万名球迷和超过一百万付费观众面前将她击倒。霍姆的第一击使劳西脑震荡，也使冠军的下唇裂开。

在回合结束时，劳西咬下一小块外突的肉，“像咬苹果一样咬下我的内唇”，并把它吐了出来。她至今仍感到她内唇的缺失部分，并记得在第二回合被击倒后更衣室里的绝望。

罗恩达·劳西被霍利·霍姆在2015年的UFC女子雏量级冠军赛中击败后倒在了擂台上。摄影：斯科特·巴博尔 / Zuffa LLC / Getty Images

劳西“独自坐在冷酷、灰色的水泥地板上”，“眼泪顺着我的脸颊流下来”。她赤脚、无声而发抖。“我能尝到嘴里的血，我的舌头触及一块曾经是我的内下唇内侧肌肉的开放的肉洞。”

她听到外面的人们在她毁灭性的失败中狂欢。“那是我一生中最糟糕的时刻。那是我曾经感受到的最剧烈的痛苦、悲伤、尴尬和羞耻。我想自杀。我想吞下一瓶止痛药，闭上眼睛，结束一切。”

通讯推广后

只有一个男人能够正确地与她交谈。特拉维斯在她抱头痛哭时找到了合适的话语：“你不仅仅是一名斗士。”

劳西长期以来一直受到崇拜，并被迫频繁为UFC战斗。但在那场失败和2016年12月对阿曼达·努涅斯的另一场 [提前停赛失利](https://www.theguardian.com/sport/blog/2016/dec/31/ronda-rousey-amanda-nunes-ufc) 后， [劳西被无情嘲笑](https://www.theguardian.com/sport/2015/nov/17/holly-holm-defends-ronda-rousey-people-can-be-pretty-brutal) ，成为社交媒体欲望摧毁一个名人的典型例子。她终于找到了摆脱这种困境的方法。劳西知道她必须保护自己的大脑，不再冒头部被击打的风险。她还告诉我，特拉维斯如何以耐心和幽默，向她展示如何重新正常生活。

罗恩达·劳西与丈夫特拉维斯·布朗合影。摄影：埃里克·威廉姆斯

“他是为数不多的几个把我看作不仅仅是Ronda Rousey，UFC冠军的人之一。以下是一种完美的方式来描述Travis。当我们刚开始在一起时，我告诉他我绝不会为男人做饭。所以一年以来，他每顿饭都为我们做饭，因为他爱我。然后有一天我说：‘我可以做非常好吃的煎饼。我想给你做些煎饼。’”

Rousey开心地笑了。“所以我开始做煎饼，然后做了更多其他的饭菜。我想通过为他做饭来表达我对他的爱，就像他为我所做的一样。然后他做了一件非常聪明的事情。他把GPS上的语音改成了澳大利亚口音。这是因为他不希望我与我在墨尔本失败的经历有任何不好的联系。直到今天，我们在GPS上仍然听到澳大利亚口音。”

在加利福尼亚州，他们有“我们的再生牧场，我们从一粒种子开始，现在拥有数百英亩的草原”。“我们正在研究如何利用我们的动物和自然过程使这个生态系统发挥其最大潜力。我们现在有群羚羊，梅花鹿和候鸟经过。我们把这片被忽视和滥用的土地变成了真正的野生动物避难所，并人道养殖我们的动物，使它们能展示出它们所有的自然行为。”

“我们本可以把我们赚的钱投资到房地产，成为房东。但我不想留给我们的孩子一堆因为世界在燃烧而在燃烧的钱。再生农业是应对气候变化最具可扩展性的解决方案之一。我真的相信它。”

Rousey在WWE当摔跤手有一段时间，她很快发现，即使在那样的马戏团里，女性也遭到了恶劣对待。但她为改变格斗运动的方式并使女性选手成为UFC业务的核心感到自豪。“我试图尽快获胜，零伤害，我对我所能够取得的成就感到非常自豪，尤其是在我的局限性下。”

即使她无法确定自己大脑的未来健康，她也找到了内心的平静。“我需要享受当下，快乐于我所处的位置，”在一小时的沉思和喧闹的笑声之后，Rousey说。“我不希望我的身体在被埋入地下时是完美的。”

“我没有遗憾，如果有一天我可以被停在海洋前，只能坐下来看鲸鱼，我应该会很快乐。我会再做一次，但我希望在行动中多一些科学和知识。”
