# HTML5 遗传算法 2D 汽车游戏 - 建议使用 Chrome 浏览器

> 来源：[https://rednuht.org/genetic_cars_2/](https://rednuht.org/genetic_cars_2/)

### 但这是什么？

该程序使用简单的遗传算法，通过多代随机生成的双轮形状演变成汽车。与 [BoxCar2D](http://boxcar2d.com/) 有些类似，但完全自主编写，仅使用相同的物理引擎（[box2d](http://box2d.org/)）。

由 [David Bau](http://davidbau.com/) 编写的 seedrandom.js（感谢！）

### 控制

| 保存种群 | 将当前种群保存到本地。 |
| --- | --- |
| 恢复保存的种群 | 恢复先前保存的种群。 |
| 惊喜 | 切换绘图，加快模拟速度。 |
| 新种群 | 保留生成的赛道并重新开始整个车辆种群。 |
| 使用种子创建新世界 | 同一个种子始终会创建相同的赛道，因此您可以与朋友协商一个种子并进行比赛。 :) |
| 突变率 | 每个个体的每个基因在新一代诞生时突变为随机值的几率。 |
| 突变大小 | 每个基因可以突变到的范围。较小的数字意味着基因的值会更接近原始值。 |
| 精英克隆 | 将复制到下一代的前 n 辆车。 |
| 查看顶级回放 | 暂停当前模拟并显示表现最佳的车辆。再次点击以恢复模拟。 |

### 图表

### 基因组

基因组包括：

+   形状（8个基因，每个顶点一个）

+   轮胎尺寸（2个基因，每个轮胎一个）

+   轮胎位置（2个基因，每个轮胎一个）

+   轮胎密度（2个基因，每个轮胎一个），颜色较深表示轮胎密度较大

+   底盘密度（1个基因），颜色较深表示底盘密度较大

### 简介

这不像应该那样确定性地运行，因此您的最佳车辆可能不如以前表现得那么好。随着距离的增加，地形变得更加复杂。
