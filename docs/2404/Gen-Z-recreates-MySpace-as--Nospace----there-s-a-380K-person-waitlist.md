# Z世代将MySpace重现为‘Nospace’ — 现在有38万人等待名单

> 来源：[https://nypost.com/2024/04/23/lifestyle/gen-z-recreates-myspace-as-nospace-theres-a-380k-person-waitlist/](https://nypost.com/2024/04/23/lifestyle/gen-z-recreates-myspace-as-nospace-theres-a-380k-person-waitlist/)

Z世代正在扭转数字时代的时钟。

蒂凡尼·钟在过去十年中研究了社交媒体习惯，并利用这些信息创建了名为Nospace的Z世代版Myspace。

在疫情期间几乎全程在线上学习的Zoomer一代，成长于社交媒体的巅峰时期，算法决定了他们看到的内容以及他们互动的对象。

现在，他们正在转向非标准社交媒体，寻找那些无法在TikTok上找到的更真实的连接。

“Z世代简直受够了传统社交媒体的老调重弹”，品牌营销策略师卡特亚·瓦尔巴诺娃告诉Bustle。

27岁的钟希望Nospace能给Z世代提供一个类似千禧一代的社交媒体体验 — 不管是好是坏。

像MySpace和Facebook早期那样，Nospace用户将拥有个性化可定制的个人资料页面，可以在上面分享他们正在做的事情的随意更新。

“我现在看到的是所有社交媒体只不过是媒体 — 不再是社交了”，钟告诉Bustle。“每个人都在观看彼此的生活和个性，但没有人在真正参与。”

“这就是我们要解决的问题：与他人的连接和自我表达”，她补充道。

Nospace用户将能够分享他们“正在观看、吃、读、听的真实生活情况”，Nospace

Nospace用户将能够分享他们“正在观看、吃、读、听的真实生活情况”，并且“星标”他们的兴趣，以便找到具有相似兴趣的人。

是的，这款新应用将列出用户的最亲密朋友，就像MySpace的Top 8一样。

“这些天的其他应用只是算法控制的。如果你没有得到足够的互动，很多帖子就会被推到后面去”，钟解释道。

那些测试过该应用的人尤其将Nospace比作一个巨大的全球群聊，特别是将“仅限朋友”与你自己策划的朋友群和实时全球时间轴分开。

“他们渴望新的体验，但主流应用无法提供，”瓦尔巴诺娃说。“信息过载加剧了对社交恐惧的焦虑，导致了焦虑症、饮食障碍和心理健康问题。难怪Z世代渴望能让他们感觉更好而不是更糟的地方。”

Nospace将不再使用“赞”功能，而是采用“加速”——类似于Facebook或Instagram的赞，但不会推动算法或排名内容——这只是为了“乐趣”而设。

“感觉像是一个游戏。人们喜欢升级。你可以获得无限的加速……每个人都可以感觉到特别。”

目前Nospace的等待列表上有超过380,000人。Nospace

钟的目标是，她的应用将帮助人们找到兴趣相似的朋友，并鼓励更多的自我表达。

“由于各种原因，孤独感正在增加，”她说。“我们与现实生活中的朋友没什么可聊的了，因为每个人都在看不同的内容。不像是我们有着相同的周日节目，或者每周放映相同的剧集。”

“在一个只看自己内容或个性化内容和视频流的世界里感到孤独，与他人建立联系反而更为重要。”她说。“社交比以往任何时候都更为重要。这种联系已经不复存在。”

据[应用网站](https://nospace.app/)称，目前有超过380,000人在等待Nospace的上线，计划于2024年夏季启动。

这并不是第一个引起千禧一代兴趣的“老式”社交媒体功能。

今年早些时候，[Facebook重新推出](https://nypost.com/2024/03/22/lifestyle/facebook-revived-the-poke-feature-and-gen-z-loves-it/)了它的“戳一戳”功能，年轻一代非常喜欢。
