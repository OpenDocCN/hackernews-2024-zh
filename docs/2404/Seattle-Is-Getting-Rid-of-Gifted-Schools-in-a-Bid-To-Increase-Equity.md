# 西雅图为增加公平性而取消了天才学校

> 来源：[https://reason.com/2024/04/04/seattle-is-getting-rid-of-gifted-schools-in-a-bid-to-increase-equity/](https://reason.com/2024/04/04/seattle-is-getting-rid-of-gifted-schools-in-a-bid-to-increase-equity/)

为了增加种族公平，西雅图正在废除其专门的公立学校。讽刺的是，这一决定可能会伤害这项政策旨在帮助的学生。

2021年，西雅图公立学校（SPS）开始逐步淘汰其“高能力同学学校”。该学区有三所小学、五所中学和三所高中专门为加快学习进度的学生提供教育。学区计划在2027-28学年完成淘汰这些特殊学校。这一变革背后的原因根植于该项目中白人和亚裔学生比例失调。

“西雅图社区和我们的家庭开始表现出对班级和现在与高级学校有关的学校中的种族差距不适的迹象，” 2020年SPS任务小组报告[指出](https://www.seattleschools.org/wp-content/uploads/2021/07/ALTF-Final-Report-ADA.pdf)。“我们当前关于接受服务的学生的数据显示，被识别为高能力的学生人数与每天上学的学生人数不成比例...当前的做法必须被打断，我们必须对我们的承诺和优先事项进行真正的审查。”

学校官员表示，有天赋的学生将通过“高能力邻里”模式获得专门的教学指导，计划在下个学年开始实施。然而，《西雅图时报》最近的[报道](https://www.seattletimes.com/education-lab/why-seattle-public-schools-is-closing-its-highly-capable-cohort-program/)对西雅图是否能兑现这一承诺表示怀疑。

“SPS 提供整个班级的模式，所有学生都在同一个教室，教师为每个学生量身定制学习计划，” 记者克莱尔·布莱恩写道。“教师不一定在教室里有额外的工作人员；学区正在为教师提供课程和教学指导，以便使其运作。”

在一个班级中，教师有额外时间为能力水平广泛的学生制定个性化教学计划，这个想法显然是不现实的。

“你只能做那么多的差异化教学，” 卡伦·斯托科夫斯基，一个在高能力同学学校有三个孩子的家长，告诉《时报》。“她还补充道，一位校长告诉她，‘你有些孩子几乎无法阅读，还有些孩子在一年级或幼儿园就读《哈利·波特》。你如何不仅将这些孩子提升到年级水平，还挑战那些已经远远超出年级水平的孩子？

约翰斯·霍普金斯大学的教育教授乔纳森·普鲁克告诉*泰晤士报*，新计划“几乎是在要求人们做不可能的事情，这是非常困难的。”

布莱恩的文章有帮助地指出，SPS正在采取几种措施来增加高才能力学生群体学校的多样性——这些解决方案并不需要放弃该计划。SPS已经开始要求所有儿童从一年级到八年级每年进行高级学术能力测试。此前，这种测试并不普遍，且通常在周六进行——这使得父母可能在周六工作的低收入孩子更难被识别出来。

当学区取消[高级课程](https://reason.com/2023/02/21/to-increase-equity-this-california-high-school-is-eliminating-honors-courses/)以减少种族不平等时，他们最终做的与他们声称的意图恰恰相反。富裕家庭可以搬到更好的学区或者将孩子送进私立学校，而聪明但贫穷的孩子最终陷入了“公平”的课堂，这些课堂让他们缺乏刺激且被忽视。
