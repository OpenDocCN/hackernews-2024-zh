# Tridge returns to rsync

[jzb于2024年4月6日发布]

Wayne Davison已经[宣布](https://lists.samba.org/archive/rsync-announce/2024/000119.html)发布了[rsync](https://rsync.samba.org/)的[版本3.3.0](https://rsync.samba.org/ftp/rsync/NEWS#3.3.0)，其中包含了一些bug修复和小的增强功能。Davison还宣布了维护者的变更，并将项目移至一个新的GitHub项目：

> GitHub仓库已经迁移到一个新的RsyncProject组织。因为各种生活事件一直占据着我的时间，我联系了Tridge [Andrew Tridgell]（原作者），他很慷慨地同意重新参与rsync的工作，还有早期贡献者Paul Mackerras。这个新团队主要将致力于维护任务，而不是新功能。如果您想参与进来，请随时在新的discord RsyncProject频道上联系。

GitHub组织现在在[这里](https://github.com/RsyncProject/rsync)。
