# 罗技鼠标和键盘用户将获得免费的AI升级 | ZDNET

> 来源：[https://www.zdnet.com/article/logitech-mouse-and-keyboard-users-you-are-about-to-get-a-free-ai-upgrade/](https://www.zdnet.com/article/logitech-mouse-and-keyboard-users-you-are-about-to-get-a-free-ai-upgrade/)

罗技

[生成式AI](https://www.zdnet.com/article/what-is-generative-ai-and-why-is-it-so-popular-heres-everything-you-need-to-know/)助手已经变得如此广泛使用，以至于访问这些工具的快捷方式被物理集成到[许多新的笔记本键盘](https://www.zdnet.com/article/microsoft-is-adding-a-new-copilot-key-to-your-windows-pc/)中。罗技采取了不同的方法，在其键盘和鼠标上添加了访问[ChatGPT](https://www.zdnet.com/article/what-is-chatgpt-and-why-does-it-matter-heres-everything-you-need-to-know/)的快捷方式，而无需添加新的物理键。

周三，罗技推出了新的Logi AI Prompt Builder，这是一个软件窗口，让用户在不中断工作流程的情况下访问聊天机器人的帮助。该公司还推出了[Signature AI Edition of the M750 mouse](https://logitech.cfzu.net/c/159047/499820/8585?&sharedid=zdnet&partnerpropertyid=1980086&u=https%3A%2F%2Fwww.logitech.com%2Fen-us%2Fproducts%2Fmice%2Fsignature-ai-edition-m750-wireless-mouse.910-007396.html&subId1=subid_value)。

[在Logitech官网查看](https://logitech.cfzu.net/c/159047/499820/8585?&sharedid=zdnet&partnerpropertyid=1980086&u=https%3A%2F%2Fwww.logitech.com%2Fen-us%2Fproducts%2Fmice%2Fsignature-ai-edition-m750-wireless-mouse.910-007396.html&subId1=subid_value)

在Logi + Option应用程序中，用户现在可以免费访问Low AI Prompt Builder，并在需要ChatGPT时在鼠标或键盘上创建快捷键，自动打开Logi AI Prompt Builder软件窗口。

**同时：[无限公司的99美元AI可穿戴设备承诺记住您的会议](https://www.zdnet.com/article/limitless-99-ai-wearable-to-promises-to-remember-your-meetings-and-well-everything-else/)**

该窗口根据用户选择要处理的文本，提供基于预定义的常见查询配方（如重述或总结）的查询建议。除了帮助用户通过绕过手动将文本复制粘贴到浏览器来更轻松地访问ChatGPT外，Low AI Prompt Builder还通过其预定的“配方”和自定义下拉菜单使提示写作更加容易。

Logitech

用户还可以通过选项自定义查询，包括更改语调、长度和风格，如上所示。

"新的Logi AI Prompt Builder是对具有与Logi Options+软件兼容的罗技鼠标或键盘的任何人的AI流畅度的捷径，他们希望轻松访问AI的无限潜力，"罗技个人工作空间解决方案总经理Delphine Donné在一份声明中说。

用户今天可以免费访问 Logi AI Prompt Builder，只要他们使用支持 Logi + Option 应用的英文版 Logitech 键盘或鼠标。根据新闻稿，这包括一些公司最受欢迎的产品线，如 Logitech MX、Ergo、Signature 和 Studio 系列。

简而言之，这款免费软件将用户的键盘或鼠标转变为一个工具，通过快捷方式访问 ChatGPT，而无需升级硬件。如果您计划升级鼠标，Logitech 推出了 Logitech [Signature AI Edition Mouse](https://logitech.cfzu.net/c/159047/499820/8585?&sharedid=zdnet&partnerpropertyid=1980086&u=https%3A%2F%2Fwww.logitech.com%2Fen-us%2Fproducts%2Fmice%2Fsignature-ai-edition-m750-wireless-mouse.910-007396.html&subId1=subid_value)。

这款新鼠标配备了一个 AI 提示按钮，提供快捷方式访问 Logi AI Prompt Builder。Logitech 表示，该鼠标今天可以预订，并将在本月在美国的 Logitech 网站上以 50 美元的价格，以及在英国上市。
