# 亚马逊HQ2去年原计划增加就业机会。但事实上，它却减少了就业机会。- 华盛顿邮报

> 来源：[https://www.washingtonpost.com/dc-md-va/2024/04/16/amazon-hq2-jobs-arlington-virginia/](https://www.washingtonpost.com/dc-md-va/2024/04/16/amazon-hq2-jobs-arlington-virginia/)

亚马逊在其北弗吉尼亚总部新建岗位的进度严重落后于计划，去年在这些办公室的工作人员数量减少，该公司证实，这表明了其原本宣扬的经济刺激项目实际上[陷入了停滞期](https://www.washingtonpost.com/dc-md-va/2023/04/19/amazon-headquarters-arlington-virginia-hybrid/?itid=lk_inline_manual_2)。

在整个北美广泛宣传之后，这家科技巨头在2018年与弗吉尼亚州官员达成协议，将其第二总部的一半设在华盛顿特区郊外的阿灵顿：作为弗吉尼亚州多达7.5亿美元的纳税人补贴的交换条件，它同意在五角大楼附近建立一个巨大的新校园，并填补成千上万的新员工。
