# Meta 确认其 Llama 3 开源LLM 将在下个月发布 | TechCrunch

> 来源：[https://techcrunch.com/2024/04/09/meta-confirms-that-its-llama-3-open-source-llm-is-coming-in-the-next-month/](https://techcrunch.com/2024/04/09/meta-confirms-that-its-llama-3-open-source-llm-is-coming-in-the-next-month/)

在伦敦的一次活动上，Meta 确认计划在接下来的一个月内首次发布 Llama 3 —— 用于驱动生成式AI助手的其大型语言模型的下一代。

这证实了《The Information》周一发布的一篇报道，称 [Meta 正接近推出](https://www.theinformation.com/articles/meta-platforms-to-launch-small-versions-of-llama-3-next-week)。

“在接下来的一个月内，实际上更短，希望在很短的时间内，我们希望开始推出我们的新一代基础模型套件 Llama 3，”Meta 全球事务总裁尼克·克莱格说道。他描述了听起来像是发布了几个不同版本或变种的产品。“在今年内将会有多个不同的模型，具有不同的能力和多样性，从现在开始就真的非常快了。”

Meta 首席产品官 Chris Cox 补充道，计划是将 Llama 3 应用到 Meta 的多个产品中。

Meta 一直在紧急赶超 OpenAI，后者在一年多前推出 ChatGPT 并在应用程序大受欢迎，将生成式AI问答转化为日常主流体验，这让其及其他大科技公司如 Google 都大吃一惊。

Meta 在人工智能方面一直采取非常谨慎的态度，但这并未得到公众的好评，此前的 Llama 版本被批评为功能过于有限。（Llama 2 于 [2023年7月公开发布](https://techcrunch.com/2023/07/18/meta-releases-llama-2-a-more-helpful-set-of-text-generating-models/)。第一个版本的 Llama 并未对外发布，但却在网上泄露。）

比其前身范围更广的 Llama 3 预计将解决这一问题，其能力不仅仅是更准确地回答问题，还能处理更广泛的问题，可能包括更有争议性的话题。公司希望这将使该产品受到用户的欢迎。

“我们的目标是，未来使 Meta AI 助手成为全球最有用的助手，”副总裁 AI 研究 Joelle Pineau 表示。“还有很多工作要做才能达到这个目标。”公司未透露 Llama 3 使用的参数规模，也未展示其工作原理的任何演示。预计它将拥有约 1400亿个参数，而最大的 Llama 2 模型只有 700亿个参数。

尤其值得注意的是，Meta 的Llama家族作为开源产品，代表了一种不同的哲学方法，即人工智能在更广泛技术发展中的应用。通过这样做，Meta 希望获得更多开发者的青睐，而不是采用更加专有的模型。

但是，Meta 似乎在其他生成式人工智能领域，尤其是超出文本生成方面，更加谨慎。公司尚未发布其图像生成工具Emu，派诺表示。

“延迟问题非常重要，安全性以及易用性，这些都是生成您引以为傲且表现出您的创造性背景的图像所需的条件。”考克斯说道。

具有讽刺意味的是，即使在Meta努力推出Llama 3的同时，该公司内部也有一些对生成式人工智能持怀疑态度的人。

[杨·勒库恩](https://ai.meta.com/blog/v-jepa-yann-lecun-ai-model-video-joint-embedding-predictive-architecture/)，这位备受尊敬的人工智能学术泰斗，也是 Meta 的首席人工智能科学家，对生成式人工智能的局限性进行了批评，并表示他的赌注在于其后续发展。他预测下一个大潮将是联合嵌入预测架构（JEPA），这是一种不同于传统训练模型和生成结果的方法，Meta 正在利用它来构建更精确的预测人工智能，特别是在[图像生成](https://ai.meta.com/blog/v-jepa-yann-lecun-ai-model-video-joint-embedding-predictive-architecture/)领域。

“人工智能的未来在于JEPA，而不是生成式人工智能”，他说道。“我们将不得不改变克里斯的产品部门的名称。”
