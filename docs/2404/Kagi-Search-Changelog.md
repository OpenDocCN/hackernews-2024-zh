# Kagi搜索变更日志

> 来源：[https://kagi.com/changelog#3750](https://kagi.com/changelog#3750)

### 公告

**从3月26日开始**，我们将为所有在此日期后订阅的新客户**包含本地化销售税**。

**从4月9日开始，所有符合销售税/增值税资格的现有客户将在其每月USD发票上看到增加的本地化销售税/增值税**，此税额将额外添加到他们当前的订阅费用之上。此调整将于4月9日后的下一张发票上显示，不会立即额外收费。

如果您选择使用**PayPal、OpenNode或其他非Stripe提供商的替代支付方式**：下次购买积分时，您将被提示选择一个销售税区域。

如果您对此变更有任何疑问，请随时通过[Discord](https://discord.gg/5V7qcs3X88)或[支持邮箱：support@kagi.com](mailto:support@kagi.com)联系我们的团队。我们将建立一个常见问题解答页面，回答有关迁移过程的任何问题。

具体来说，这意味着受影响的会员（如果适用于您的国家/州，销售税/增值税将自动添加到Kagi价格之上），这是由于Kagi变得足够大而具有法定的销售税/增值税义务所致。此外，Kagi将不得不补缴过去近两年未收取的所有销售税/增值税。我们选择替客户吸收这些成本。
