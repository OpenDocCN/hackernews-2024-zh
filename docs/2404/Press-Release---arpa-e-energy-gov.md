# 新闻发布 | arpa-e.energy.gov

> 来源：[https://arpa-e.energy.gov/news-and-media/press-releases/us-department-energy-announces-10-million-explore-using-plants](https://arpa-e.energy.gov/news-and-media/press-releases/us-department-energy-announces-10-million-explore-using-plants)

**华盛顿特区 —** 美国能源部今天宣布拨款高达1000万美元，探索使用植物从土壤中提取镍的过程，即被称为植物采矿的过程，旨在建立竞争力强的国内供应链，补充传统采矿方法，并减少镍的进口。由能源高级研究项目署（ARPA-E）管理，此举支持拜登总统的“投资于美国”议程，加强国内关键材料供应链，增强我们的经济和国家安全，并满足对确保美国在新兴清洁能源经济中领先的关键材料日益增长的需求。

“为了实现拜登总统设定的达到清洁能源目标，并支持我们的经济和国家安全，需要采取全员参与的方法和创新解决方案”，**ARPA-E主任伊夫琳·N·王说**。“通过探索植物采矿来作为首个关键材料的镍提取目标，ARPA-E旨在实现成本竞争力和低碳足迹的提取方法，以支持能源过渡。”

在美国能源部的[**关键材料评估**](https://www.energy.gov/sites/default/files/2023-07/doe-critical-material-assessment_07312023.pdf)（CMA）中列出的关键材料中，镍由于有大量文献记录的镍超积累（HA）植物而成为验证植物采矿在美国可行性的理想目标。镍用于锂离子电池的阴极，存在于电动汽车、消费电子、静态存储、不锈钢、冶金、涂层、电镀和其他合金中。镍对全球清洁能源技术供应链至关重要，未来需求预计将增长。

今天宣布的新的ARPA-E探索性课题，[**植物超积累植物开采富镍土壤（PHYTOMINES）**](/technologies/exploratory-topics/phytomining)，旨在激励植物采矿技术在美国的技术发展，这可能会补充当前和未来的国内镍来源，并催化超越镍的关键矿物的植物采矿。

PHYTOMINES 的目标成果包括：

+   开发优化生物系统技术，调节超积累植物对镍的可用性和吸收。

    +   技术可以是土壤或植物微生物组的干预，或是开发使镍在植物体内以增强速率积累的植物特性。ARPA-E将这些项目视为早期阶段的概念验证，可能在封闭或露天实验室、温室或受限田野进行，其中光照、湿度和温度条件可以完全编程。

+   理解影响植物采矿潜力的地质、生态和经济因素之间的相互关系。

    +   可能的项目包括绘制感兴趣的HA物种分布图、土壤中的矿物特性以及适合植物采矿的自然栖息地和相邻区域的土地所有权数据，扩展机会以及植物采矿项目的技术经济和生命周期分析。

PHYTOMINES鼓励农民、科学家、电池制造商、钢铁和采矿行业等之间的合作伙伴关系。您可以在[**ARPA-E Exchange**](https://arpa-e-foa.energy.gov/Default.aspx#FoaId521a7aa4-b255-4c3b-a211-b128d2a4a0e4)上获取更多信息。

ARPA-E推进高潜力、高影响力的清洁能源技术，涵盖战略性对美国能源安全至关重要的技术领域。了解更多关于这些努力以及ARPA-E致力于确保美国继续在开发和部署先进清洁能源技术方面领先全球的信息。
