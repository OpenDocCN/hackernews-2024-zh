# 美国航空公司一直将101岁的乘客误认为婴儿

> 来源：[https://www.bbc.com/news/articles/c9wz7pvvjypo](https://www.bbc.com/news/articles/c9wz7pvvjypo)

一位101岁的女士因为航空公司预订系统的错误而一直被误认为婴儿。

问题出现是因为美国航空公司的系统显然无法计算出Patricia，这位不愿透露姓氏的人，出生于1922年而不是2022年。

BBC目睹了最新的混乱，她和机组人员能够一笑了之。

“他们认为我只是个小孩子而不是一位老太太，这真是太有趣了！”她说。

但这位百岁老人表示她希望这个故障能够修复，因为过去它曾给她造成一些问题。

例如，在一次情况下，机场工作人员没有在航站楼内准备好她的交通工具，因为他们在等待一个可以抱的婴儿。

BBC目击的故障发生在Patricia飞行时在芝加哥和密歇根州马奎特之间的航班上，这位记者也在同一班飞机上。

Patricia与她的女儿Kris一起飞行。

“我女儿在网上为票预定了票，而机场的电脑认为我的生日是2022年而不是1922年”，她告诉我。

“去年也发生了同样的事情，他们也在等待一个孩子而不是我。”

Patricia的座位被预订为成人票。

但看来机场的电脑系统无法处理如此久远的出生日期 - 所以它默认为100年后的日期。

这位前护士每年都会乘飞机去看家人，逃避寒冷的冬天，尽管混乱，美国航空公司的员工在两次情况下都非常友好和乐于助人。

美国航空公司未对置评请求做出回应。

这位百岁老人表示她希望能够修复这个问题。在一次先前的飞行中，当其他乘客离开后，Patricia和她的女儿等在飞机内，因为机场工作人员没有为她安排轮椅。

她表示，承认她真实年龄也对克里斯有好处。

“我希望他们能修复电脑系统，因为我的可怜女儿几乎要从一个门走到另一个门，扛着所有的行李和衣服”，她说。

Patricia单独旅行到97岁，但自那时起便依赖家人的帮助。

“我现在的视力有些问题，所以我不想一个人去做这件事”，她说。

但她坚决表示，这些IT问题不会让她放弃飞行，并且她期待秋天的下一次飞行。

到那时她将是102岁了 - 或许到那时航空公司的电脑系统会意识到她的真实年龄。
