# 华盛顿州彩票AI网站将用户照片转化为色情内容 | Rantz - MyNorthwest.com

> 来源：[https://mynorthwest.com/3956403/rantz-washingtons-lottery-ai-porn-user/](https://mynorthwest.com/3956403/rantz-washingtons-lottery-ai-porn-user/)

华盛顿州彩票的新AI手机网站将用户的照片转化为软色情内容，出于谨慎起见关闭了该网站。

当梅根，一位居住在坦沃特的50岁母亲，于3月30日访问华盛顿州彩票的新AI手机网站时，她以为自己将要体验一些轻松愉快的时光。[Test Drive A Win](https://testdriveawin.com/) 让用户可以将自己的照片上传到一个装有梦幻度假地图的飞镖板上，你可以用彩票赢得的钱支付梦幻度假地图。根据飞镖掷中的位置，你可以上传一个自拍或使用手机拍照上传，AI将你的图像与度假地点合成。

梅根选择了“与鲨鱼共游”的梦幻度假选项。她对华盛顿州彩票生成的一张人工智能图片感到震惊。那是软色情内容。

## 华盛顿州彩票网站如何推出色情内容？

梅根说她使用了网站内的选项拍了一张自己脸的照片上传。生成的照片显示一个微笑的AI版本的梅根，但几乎是完全裸体的。

在这张图片中，梅根坐在床上，下身穿着泳裤，但上身却没有穿。她的裸露胸部显现出来。图片的背景似乎是一个水族馆里的卧室，周围有鱼在游动。右下角有一个华盛顿州彩票的Logo。

“我们的纳税人的钱用在这上面！我完全震惊了。至少可以说这令人不安，” 梅根向KTTH的Jason Rantz Show解释道。

梅根担心这张图片可能会泄露出去，并可能会发生在其他网站访问者身上。她告诉一个朋友，这个朋友碰巧认识华盛顿州彩票的某位员工，告诉了他发生的事情。

“我还认为，对此负责的人应该被解雇，” 她补充道。

### 华盛顿州彩票关闭了该网站

AI创建这张特定图片的方式和原因尚不清楚，考虑到规定内容的规则。KTTH的Jason Rantz Show当然无法验证这张图片。不过，华盛顿州彩票的发言人确认他们收到了有关AI系统创建该图片的报告。

在KTTH的Jason Rantz Show询问三个小时后，华盛顿州彩票关闭了该网站，并提供了一份声明。

“我们了解到AI平台的单个用户据称获得了一张不符合开发人员设定的内置参数的图片。在发布前，我们已经同意了一套全面的规则来管理图像的创建，包括要求图像中的人物穿着完整。我们尚未看到AI提供给此用户的图片，但出于谨慎起见，我们暂时将网站下线，” 发言人向KTTN的杰森·兰茨秀确认。

发言人没有确认他们何时首次听说此问题。他也没有解释为什么华盛顿州彩票在我的询问之后才将网站下线。

-   梅根很高兴网站已经移除。她表示希望调整后端设置的工作已经完成。

### UPDATE: 4/2/24, 7:38 p.m.

华盛顿州彩票发言人周二晚间向KTTN的杰森·兰茨秀发送了以下声明：

> 本周早些时候我们就已经听说了这件事。在得知后，我们立即让开发人员检查了平台的所有参数，并对设置感到满意，但在今天的进一步内部讨论后，我们选择出于谨慎起见将网站下线，因为我们不希望类似这样的事件再次发生。
