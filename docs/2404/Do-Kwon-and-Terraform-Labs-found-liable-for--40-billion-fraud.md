# Do Kwon 和 Terraform Labs 被判定涉及 400 亿美元诈骗。

> 来源：[https://www.web3isgoinggreat.com/?id=do-kwon-and-terraform-labs-found-liable-for-40-billion-fraud](https://www.web3isgoinggreat.com/?id=do-kwon-and-terraform-labs-found-liable-for-40-billion-fraud)

Solana 生态系统正在应对一系列空账户问题。尽管尚未最终确定原因，但一些盗窃与像 Solareum 这样的交易机器人有关。Solareum 推测这些攻击可能与被篡改的 Telegram 机器人令牌有关，这些令牌可能使攻击者能够从消息历史中获取私钥。

Solareum 后来表示他们将关闭项目，并删除了他们的网站。这引起了一些用户的批评，指责他们没有做任何调查来查明黑客的身份，甚至可能是责任方。该项目在 Twitter 上写道：“我们在 #SOLAREUM 团队可以澄清，我们并没有盗取资金。”啊，好吧，在这种情况下。

其他机器人可能也参与了这起盗窃案，尽管目前尚不清楚。尽管有些人猜测一个名为 BonkBot 的交易机器人可能是罪魁祸首，但这似乎是毫无根据的。

盗窃总金额尚不清楚，但超过了 500,000 美元。
