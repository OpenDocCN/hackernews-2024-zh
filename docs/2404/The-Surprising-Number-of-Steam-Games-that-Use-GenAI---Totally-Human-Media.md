# Steam 上使用 GenAI 的游戏数量之惊人 — Totally Human Media

> 来源：[https://www.totallyhuman.io/blog/the-surprising-number-of-steam-games-that-use-genai](https://www.totallyhuman.io/blog/the-surprising-number-of-steam-games-that-use-genai)

[**Atman: Rebel Flame**](https://store.steampowered.com/app/2879000/AtmanRebel_Flame/)（188 条评价中有90%为正面评价）：

> 在游戏设计过程中，我们使用了先进的AI工具来生成一些图片作为创作参考和灵感来源，并在此基础上由专业人员进行艺术处理，展现了《Atman: Rebel Flame》独特的艺术风格。同时，我们也非常重视玩家的反馈和建议。如果您对游戏的艺术表现有任何建议，请通过社区与我们进行沟通。

这里还有更多的例子：

> **Girls of The Tower:** “为了更好地专注于游戏的核心设计，我们通过艺术家的草稿、AI辅助渲染和艺术家的调整过程生成了一些物品图标和少数NPC肖像（不包括主角）。此外，为了确保最终的艺术呈现，所有AI生成的资产都经过了艺术家的手工精炼。”
> 
> **ARC TCG:** “所有卡片的概念都是由Leonardo.Ai通过Leonardo Diffusion XL模型创作的。然而，这些内容的生成是通过复杂的提示来提供更独特的图像，此外，每张图像都经过进一步审查，以提供比基础模型更详细和更多选择的内容。这种AI方法最初被用来降低开发成本，否则这个游戏无法推出。根据经济游戏的表现，未来我们将逐步用专业艺术家创作的图像替换AI生成的图像。”
> 
> **RetroMaze:** “生成AI（DALL·E 3）被用于Steam社区的资源中，具体来说是用于交易卡片和表情艺术。”
