# 如何修复美国的航空系统 | On Point

> 来源：[https://www.wbur.org/onpoint/2023/11/15/how-to-fix-americas-aviation-system](https://www.wbur.org/onpoint/2023/11/15/how-to-fix-americas-aviation-system)

自上次美国航空公司致命飞机坠毁已经将近15年。但近期差点意外事件却在增加，过去十年增长了25%。

这是一个巨大的跃升 — 由几个长期潜伏的航空问题突然汇合引起的。

首先，FAA要求的地面雷达和控制系统已经过时。

"它们都建立在非常老旧的计算机上。有些仍在使用软盘，我开玩笑都不敢相信，" 美国国家航空交通管制协会名誉主席保罗·里纳尔迪说。

其次，航空交通管制员逐渐老龄化，仍在工作的人员已经筋疲力尽。

"大多数控制员每周工作60小时，" 里纳尔迪补充道。

它可以修复吗？

今天，*On Point:* 如何修复美国的航空系统。
