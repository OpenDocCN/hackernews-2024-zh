# NEJM研究链接：微塑料在血管中与心脏问题相关

> 来源：[https://www.statnews.com/2024/03/06/nejm-microplastics-blood-vessels-plaque-heart-attack-stroke-death/](https://www.statnews.com/2024/03/06/nejm-microplastics-blood-vessels-plaque-heart-attack-stroke-death/)

微塑料和纳米塑料，即环境中广泛分布的小块塑料，越来越被发现可以进入人体，引发了关于它们最终停留位置及其对人体健康影响的问题。在一项新研究中，研究人员首次发现这些塑料颗粒存在于血管中积聚的脂肪斑块内，并将其与心脏问题的增加率联系起来。

研究人员发现，在其斑块中存在这些塑料颗粒的人群，心脏病发作、中风或死亡等重大并发症的风险增加了4.5倍，根据这项研究，[发表在](http://www.nejm.org/doi/full/10.1056/NEJMoa2309822)《新英格兰医学杂志》周三。

具体而言，在约三年的时间内，150名病人中有30名（占总数的20%）在其斑块中检测到塑料的病例出现了并发症，而107名病人中有8名（占总数的7.5%）未检测到塑料的病人则出现了并发症。

他们发现的塑料类型通常用于日常生活中：聚乙烯，用于塑料袋和瓶子，以及聚氯乙烯，用于管道、绝缘材料和医疗设备。

作者警告称，这项研究并未明确证明塑料颗粒确实导致了更高的问题风险，但确实发现了微塑料与心血管并发症之间的重要联系，这需要在未来的研究中进一步探讨。

该论文出现在更大范围内对微塑料的研究焦点之中，这些微塑料已经被发现存在于从食物和水源到大气的环境中。随着气候变化加剧，塑料产品的降解加速，这一问题预计将会加剧。到目前为止的研究发现，微塑料在小鼠身上具有危害，并且还在人体的不同部位——如胎盘、肝脏和肺部——检测到它们的存在，但对微塑料实际影响人体健康的研究较少。

“人们已经广泛意识到微塑料无处不在，分布于各个地理区域，”该研究的作者之一、克利夫兰大学医疗中心心血管医学主任桑贾伊·拉贾戈帕兰说道。“我们不清楚的是它们是否对健康有重大影响，以及我们是否真的应该担心它们？长期影响是什么？”

研究包括意大利约300人进行颈动脉内膜切除术，这是一种用于清除血管内脂肪斑块的手术。随着时间的推移，这些斑块的积累可能会阻塞血管，导致心脏病发作或中风。

手术收集的斑块被冷冻，然后由研究人员分析。他们使用显微技术和化学分析发现斑块中嵌入的塑料碎片。

由于先前的研究表明微塑料可能会增加炎症反应，研究人员观察了炎症标志物的标记，包括白介素-18、白介素-1β、白介素-6和TNF-α，并发现斑块中的塑料颗粒量与这些炎症标志物的水平相关。

Changcheng Zhou，加州大学河滨分校的教授，未参与此研究，他表示，研究的下一步可能是了解微塑料如何增加炎症并导致心脏问题的机制。周教授建议，一个研究方向是调查塑料中的化学物质是否渗入到斑块中，他研究塑料产品内部的化学物质。

尽管该研究远未证明微塑料会引起心脏问题，路易斯维尔大学的医学副教授Tim O’Toole表示：“这是一项非常必要的分析”，他也未参与此研究。

“由于全球气候变化，微塑料污染问题可能会继续恶化，甚至变得更加严重，” O’Toole说道。O’Toole一直在研究污染物对心脏健康的影响。“气温的升高将增加这些污染物的分解，它们在我们的水源和食物链中将会成为更加严重的问题。”
