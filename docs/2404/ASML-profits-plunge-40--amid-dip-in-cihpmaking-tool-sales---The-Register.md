# ASML利润暴跌40％，伴随芯片制造工具销售下降 • The Register

> 来源：[https://www.theregister.com/2024/04/17/asml_profits_slump_40_percent_q1_2024/](https://www.theregister.com/2024/04/17/asml_profits_slump_40_percent_q1_2024/)

芯片制造设备大师ASML在日历第一季度几乎一半的销售额来自中国，而整体订单下降和利润暴跌。

光刻巨头报告称，截至3月31日的三个月内，净利润暴跌40％，至12亿欧元[$13亿]，营收下降25％至39.66亿欧元[$42.2亿]。

收入约为ASML早期指导的中点，然而，新订单订购价值36亿欧元[$38亿]未能达到分析师的预期，远低于去年Q4创纪录的91亿美元[$96亿]，有人认为这要归因于中国芯片制造商在新出口限制出台前“急于下单”[https://www.theregister.com/2023/11/08/china_stockpiling_asml_kit/]。

中国占据了ASML第一季度销售额的惊人49％，该公司确认。

最新的限制措施，由荷兰政府应美国政府要求实施，有效地禁止了ASML自一月起向中国销售部分深紫外（DUV）光刻机。

"关于ASML的订单接收量，已经有相当多的猜测，"首席财务官Roger Dassen在视频采访中说道。“我认为重要的是要认识到，在过去六个月里，如果把Q4和Q1的订单接收量合起来看，你会看到接近130亿欧元[$138.3亿]，这是一个相当大的数字。”他声称。

达森补充说：“现在人们已经认识到订单接收过程通常是相当波动的。”

但作为目前唯一能够为制造最先进半导体提供极紫外（EUV）光刻设备的公司，ASML应该能够度过订单预订低谷。由于拜登政府“CHIPS法案”[https://www.theregister.com/2024/03/15/biden_intel_chips_funds/]的资金支持，全球正在建设新一波芯片制造厂，尤其是在美国。

达森预测这将是一个“过渡年”，在这12个月里势头会逐渐增强。

“这意味着下半年将比上半年更强劲。而且我们确实正在加速迎接我们认为将是2025年强劲一年的状态。这是真正由我们今天所看到的一些行业趋势支持的。这对我们来说意味着建立能力并为这种增长做好准备，”他说道。

### 谁是神秘客户？

Dassen表示，ASML预计会看到其High NA（数值孔径）EUV设备的销量增加，该设备在最先进的生产工艺中为更小的特征提供更好的分辨率。英特尔是今年早些时候在其俄勒冈州希尔斯伯勒设施中接收到TWINSCAN EXE:5200系统的[首个客户](https://www.reuters.com/technology/asml-ships-first-high-na-lithography-system-intel-statement-2023-12-21/)。

“业界对High NA真的很兴奋，理由非常充分，” Dassen声称。“如果你看看High NA设备的规格，它将提供比低NA设备高3倍的晶体管密度。所以这是一个真正的大进步，”他说。

对于第二季度，ASML预计净销售额将较高，介于57亿欧元（60.6亿美元）和62亿欧元（65.9亿美元）之间。即将离任的总裁兼首席执行官彼得·文因克（Peter Wennink） - 本月离任 - 表示2024年全年的展望保持不变，预计下半年将比上半年强，符合行业从低谷中持续复苏的趋势。

完全没有提到进一步出口限制的可能性，但这很可能发生，因为美国试图遏制中国的半导体进展，并敦促其盟友跟随。

然而，随着所有新的芯片工厂在中国境外的建设，ASML似乎持乐观态度，并仍然预计到2025年的年收入将在300亿欧元（319亿美元）至400亿欧元（423亿美元）之间。

“我认为到2025年，我们将处于复苏的高峰期中，” Dassen说。“如果你只是看看我们的客户所指示的所有晶圆厂开工的消息。CHIPS法案资金分配积极结果的最新消息。所有这些都非常强大，非常支持全球新晶圆厂的开工。” ®
