# 亚马逊终止 iRobot 合并，伊莉莎白·沃伦反对，导致 iRobot 解雇了350名员工

> 来源：[https://reason.com/2024/01/29/irobot-lays-off-350-employees-as-amazon-kills-merger-elizabeth-warren-opposed/](https://reason.com/2024/01/29/irobot-lays-off-350-employees-as-amazon-kills-merger-elizabeth-warren-opposed/)

今天，亚马逊[终止](https://media.irobot.com/2024-01-29-Amazon-and-iRobot-agree-to-terminate-pending-acquisition)其计划中收购Roomba机器人吸尘器制造商 iRobot 的行动，因为两家公司看到"没有获得监管批准的路径"。随后 iRobot [宣布](https://investor.irobot.com/news-releases/news-release-details/irobot-announces-operational-restructuring-plan-position-company)将裁员近三分之一。

尽管公司归咎于欧盟监管机构的终止，但多管闲事的美国议员也对此事起了作用，导致了这笔交易的失败。

2022年8月，[亚马逊宣布](https://press.aboutamazon.com/2022/8/amazon-and-irobot-sign-an-agreement-for-amazon-to-acquire-irobot)拟以17亿美元收购 iRobot。这一收购将[补充](https://apnews.com/article/colin-angle-d4de06cdcf36beeec71166ff44abdf68)亚马逊日益增长的智能家居产品系列，如[Echo Hub](https://press.aboutamazon.com/2023/9/amazon-unveils-next-generation-echo-show-8-all-new-echo-hub-and-new-echo-frames)控制面板和[Ring视频门铃](https://reason.com/2024/01/25/amazons-ring-will-stop-giving-police-your-doorbell-footage-without-a-warrant/)。

第二个月，联邦贸易委员会（FTC）[开始调查](https://apnews.com/article/technology-federal-trade-commission-65d4e0de892f0e1b2f8ee20b6f45832e)这一合并，并且不久后，议员们[发表了自己的意见](https://www.warren.senate.gov/newsroom/press-releases/warren-jones-lawmakers-call-on-ftc-to-oppose-amazons-proposed-165-billion-acquisition-of-irobot)。在给FTC主席莉娜·卡恩的[信函](https://www.warren.senate.gov/imo/media/doc/Letter%20to%20FTC%20on%20Amazon%20-%20iRobot%20Merger.pdf)中，马萨诸塞州民主党参议员伊丽莎白·沃伦和五名民主党代表建议"FTC应该行使其权力反对亚马逊- iRobot 交易"，因为这一收购"可能会损害消费者，在家庭机器人市场上减少竞争和创新"。

该信引用了[*Bloomberg*文章](https://www.bloomberg.com/news/articles/2022-08-05/amazon-s-irobot-deal-seen-facing-tough-ftc-antitrust-review)，文章指出"iRobot 的Roomba在美国智能吸尘器市场以75%的市场份额（按收入计算）占据主导地位"。议员们引用了FTC的反垄断权威和"亚马逊的反竞争行为历史"作为反对此交易的理由。

2023年11月，欧洲委员会在自己的审查中[发布了](https://ec.europa.eu/commission/presscorner/detail/en/ip_23_5990)一份反对声明。委员会担心这笔交易“可能限制机器人吸尘器市场上的竞争”。它指出，“因为亚马逊的在线市场对于销售机器人吸尘器尤为重要”，亚马逊“可能有能力和动机排斥 iRobot 的竞争对手”，无论是通过排斥他们出现在市场上，还是“削弱它们的接入渠道”。

英国，这个已经不再是欧盟一员的国家，在2023年6月[批准](https://www.gov.uk/government/news/amazon-s-purchase-of-roomba-maker-cleared-by-cma)了这一并购。当时，该国的竞争与市场管理局“得出结论称该交易不会在英国引发竞争问题。”

本月早些时候，《政治家》[报道](https://www.politico.eu/article/amazon-dodges-concessions-to-appease-eu-on-irobot-deal/)称，亚马逊在1月10日之前有机会提出让欧洲委员会满意的让步，但亚马逊让期限失效且未提出任何让步。

今天，这些公司[共同宣布](https://media.irobot.com/2024-01-29-Amazon-and-iRobot-agree-to-terminate-pending-acquisition)终止收购协议，并表示在欧盟没有获得监管批准的途径。随后，iRobot [宣布](https://investor.irobot.com/news-releases/news-release-details/irobot-announces-operational-restructuring-plan-position-company)“一项运营重组计划”，计划裁员350名员工，占其员工总数的31%。预期按并购协议继续担任公司联合创始人兼首席执行官的科林·安格尔将会辞职。

尽管欧洲的担忧可能是导致这笔交易波折的原因，看来美国议员要求联邦贸易委员会反对这笔交易也实现了他们的愿望。

然而，欧盟及伊丽莎白·沃伦等议员的抱怨被夸大了。收购 iRobot 并不会剥夺消费者购买机器人吸尘器的能力：虽然 Roomba 可能是畅销品牌，但像 Shark 和 Eufy 这样的公司也推出了自己的竞争版本，甚至在[更低的价格](https://www.rollingstone.com/product-recommendations/electronics/cheap-roomba-alternatives-989418/)销售。在其[2024年最佳机器人吸尘器清单](https://www.nytimes.com/wirecutter/reviews/best-robot-vacuum/)中，《Wirecutter》推荐了非 iRobot 产品，指出与竞争对手相比表现不佳。

与此同时，随着公司等待监管机构的决定，iRobot正在亏损：该公司在2023年7月取得了一笔[2亿美元的过桥贷款](https://www.reuters.com/markets/deals/amazon-amends-irobot-deal-lower-offer-price-by-15-2023-07-25/)以支持其在交易完成前的运营（届时，亚马逊[降低了其收购报价](https://investor.irobot.com/news-releases/news-release-details/amazon-and-irobot-modify-merger-price)以考虑这笔新债务）。虽然交易被搁浅，亚马逊[现在将支付](https://investor.irobot.com/news-releases/news-release-details/irobot-announces-operational-restructuring-plan-position-company)9400万美元的终止费用，但iRobot预计2023年将报告高达2.85亿美元的营运亏损。

那么，是否这正是像沃伦这样的立法者心中所想的呢？FTC的信件担忧这次并购“可能会损害消费者，减少家庭机器人市场上的竞争和创新。”但如果没有这次并购，iRobot很可能会面临破产，将近三分之一的员工将失去工作——考虑到这家公司是[位于马萨诸塞州](https://about.irobot.com/)，其中相当一部分可能是沃伦的选民。
