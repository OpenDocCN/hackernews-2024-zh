# 数以万计的人们在加那利群岛上进行抗议，呼吁紧急重新考虑西班牙群岛的旅游战略，并冻结游客数量，认为几十年来的模式使居民生活难以负担，环境不可持续。

> 来源：[https://www.theguardian.com/world/2024/apr/20/thousands-protest-canary-islands-unsustainable-tourism](https://www.theguardian.com/world/2024/apr/20/thousands-protest-canary-islands-unsustainable-tourism)

数以万计的人们正在加那利群岛各地抗议，旨在要求西班牙群岛的旅游战略进行紧急反思，并冻结游客数量。他们认为，几十年来的旧模式使居民的生活变得难以负担，并且对环境不可持续。

这些抗议活动以“Canarias tiene un límite”（加那利群岛有限度）为口号，得到包括绿色和平、世界自然基金会、生态行动、地球之友和SEO/Birdlife等环保组织的支持。

“我们已经到了资源利用和本地居民福祉之间平衡崩溃的地步，尤其是过去一年里，”集体“Canarias se Agota”（加那利群岛已经忍受不了）的发言人维克多·马丁说，该集体协助在周六全群岛范围内协调了抗议活动。

“Canarias se Agota”已有11名成员因抗议南特内里费岛南部两个大型豪华开发项目的建设而进行了一周的绝食，他们称这些项目非法且完全不必要。

警方称，有2万人参加了示威活动，但组织者称人数接近5万人，西班牙TVE公共电视台报道。

“我们不反对旅游业，”抗议者之一Rosario Correo告诉TVE。“我们要求改变允许旅游业无限增长的模式。”

一张摄于格兰加纳利岛的海滩照片。摄影：Greg Balfour Evans/Alamy

公共电视称，马德里和巴塞罗那的抗议者也聚集起来，支持加那利群岛的抗议活动。

去年，有1390万人次访问了这些岛屿，岛上居住着220万人口。旅游业占该群岛GDP的约[35%](https://www3.gobiernodecanarias.org/noticias/canarias-lidero-la-recuperacion-turistica-nacional-en-2022-con-16-961-millones-de-euros-un-355-del-pib-regional/)，仅2022年就带来了169亿欧元的收入，但当地人表示，该行业正压力自然资源，并导致租房市场价格过高。

西班牙国家统计研究所的数据显示，[33.8%的加那利群岛居民面临贫困或社会排斥的风险](https://www.ine.es/dyngs/Prensa/es/ECV2023.htm)，除了安达卢西亚外，这是任何地区中比例最高的。

马丁表示，在气候紧急情况下，当地政府继续专注于旅游业毫无道理。“在有更多游客的城市地区，需水量正在上升，”他说。“我们经历了非常干旱的冬季，特内里费已经宣布水资源紧急状态。”

“如果本月降雨不足，将会出现限制措施，但目前这里的温度已经达到36摄氏度。这一切都是不可持续的，意味着我们甚至无法维持正常水平的旅游业。尽管如此，这里的当局和企业仍试图坚持这种模式。”

马丁说，由于高物价、低工资、缺乏公共住房和持续的生活成本危机，该群岛许多地区的住房状况也非常严峻。“当我看到一些做酒店女佣或服务员的人住在棚屋里时，我意识到我们已经到了极限。”

“工资太低，以至于它们无法支付基本生活成本，尤其是在当前全球性危机中，尽管这种危机在加那利群岛尤为明显，因为我们几乎不得不进口所有东西。”

他坚称抗议运动并非反对游客，指出加那利群岛许多人都认识并喜欢来自英国和德国等国家的几代家庭。

“问题不在于游客，”他说。“问题在于建立在商业阶层周围、并与其共谋的模式，他们不愿意倾听应该做些什么，以及一个政治阶层为商业阶层服务而非为所有公民服务。”

他说，对加那利群岛的旅游模式进行彻底反思不能再等了。“我们提出的要求非常简单。鉴于旅游业是主要经济活动并导致所有这些问题的根源，我们要立即停止这两个大型项目，”他谈到特内里费岛的开发计划时如是说。

“我们还希望实行旅游停摆，以便研究每个岛屿可以承载的负荷，并确定我们是否已经超过临界点。在负载过重的地区，我们希望看到经济活动的衰退阶段，以惠及自然资源。否则，你只会得到一个仅有少数人受益的现有模式。”

马丁表示，对加那利群岛所遭受的问题进行适当研究可能会产生全球影响。“这种旅游模式的重新思考可以使加那利群岛成为可持续旅游发展的典范，”他说。“我们可以因为一些积极的事物而为人所知，而不是因为一些负面的事物。”

加那利群岛地区总统费尔南多·克拉维霍表示，他的政府已经采取行动。“这届政府采取的所有行动都基于对这一模式的重新审视，” [他本周告诉记者们](https://www.europapress.es/islas-canarias/noticia-presidente-canarias-asegura-todas-acciones-iniciadas-estan-orientadas-revision-modelo-turistico-20240415115153.html)。“加那利群岛的旅游模式一直很成功，但显然，像任何事物一样，总有可以改进的地方。”

过度旅游已经成为 [西班牙许多城市和地区的主要问题](https://www.theguardian.com/travel/2017/aug/10/anti-tourism-marches-spread-across-europe-venice-barcelona)，在巴塞罗那引发了抗议和反弹，导致塞维利亚当局 [考虑向游客收费，以限制人数进入该城市著名的西班牙广场](https://www.theguardian.com/world/2024/feb/26/seville-to-charge-tourists-to-visit-plaza-de-espana-to-limit-numbers)。
