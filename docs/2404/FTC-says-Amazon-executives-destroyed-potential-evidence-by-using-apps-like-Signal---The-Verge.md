# FTC 表示，亚马逊高管通过使用 Signal 等应用程序销毁了潜在证据 - The Verge

> 来源：[https://www.theverge.com/2024/4/26/24141801/ftc-amazon-antitrust-signal-ephemeral-messaging-evidence](https://www.theverge.com/2024/4/26/24141801/ftc-amazon-antitrust-signal-ephemeral-messaging-evidence)

FTC 律师们 [在星期四提交了一份文件](https://www.documentcloud.org/documents/24623188-ftc-vs-amazon-signal?responsive=1&title=1)，声称亚马逊的高层使用 Signal 的自动删除消息功能销毁了与 [该机构的大规模反垄断诉讼](/2023/9/26/23809817/amazon-ftc-antitrust-lawsuit-monopoly) 相关的证据。 （你还记得那个吗？FTC 指控亚马逊创建了一个名为“尼斯湖项目”的秘密定价算法，可能产生了 [超过10亿美元的额外利润](/2023/11/2/23943724/amazon-project-nessie-algorithm-ftc-antitrust-case)。）

现在，[*华盛顿邮报*](https://www.washingtonpost.com/technology/2024/04/26/amazon-ftc-messages-deleted-bezos/)（由亚马逊创始人兼前首席执行官杰夫·贝索斯拥有）报道称，亚马逊只是最近被指控使用像 Signal 这样的加密消息应用，可以自动永久删除消息的几家公司之一。

您可能还记得政府在 [Sam Bankman-Fried 在使用 Signal 时的类似论点](/2023/10/26/23934195/sam-bankman-fried-self-testimony-deleted-signal)，在其欺诈审判期间，[该判决最终如何做出](/23894366/ftx-sam-bankman-fried-trial-updates-news)。删除的聊天记录也成为了 [至少一名陪审员](/2023/12/12/23999333/epic-juror-confirms-googles-deleted-chats-were-a-factor-in-the-verdict) 在谷歌最近的法庭败诉中的争议点，以及在 [美国司法部反垄断审判中对谷歌的影响](/2023/10/30/23939043/us-v-google-sundar-pichai-testimony)。

在 [公司的“竞争”页面发布的声明中](https://www.aboutamazon.com/news/company-news/amazon-response-to-ftc-motion-signal-app)，亚马逊称 FTC 的指控“毫无根据”，称“亚马逊多年前自愿向 FTC 报告了员工有限的 Signal 使用情况，全面收集了员工手机上的 Signal 对话，并允许机构工作人员检查这些对话，即使它们与 FTC 的调查无关。”

本周的文件包括两名亚马逊高管之间 Signal 聊天的截图，其中一人说：“你觉得加密了吗？” 然后打开了自动删除消息。

FTC的律师称，贝索斯、现任CEO安迪·贾西、总法律顾问大卫·扎波尔斯基、前全球运营CEO戴夫·克拉克以及其他高管都是Signal的用户。文档中将贝索斯标识为“重度Signal用户”，指示他人使用该应用，尽管[2018年他个人手机遭到黑客攻击](/2020/1/23/21078828/report-saudi-arabia-hack-jeff-bezos-phone-fti-consulting)可能是其中的一部分原因。

由于亚马逊在被通知调查后超过15个月才指示员工保存应用中的消息，FTC认为，“由于亚马逊的行动和不作为，高度可能已经销毁了相关信息。”

FTC的律师正在调查亚马逊保留文件的努力，以确定可能丢失了多少信息。尽管去年秋季请求提供关于亚马逊向员工提供有关瞬时应用的建议的相关文件，FTC声称亚马逊迄今拒绝提供大部分请求的内容。如果法官发现亚马逊在未能保留与案件相关的数据方面存在过失，可能会面临制裁，如果法官发现这些失误是[故意的](/2023/12/1/23984902/judge-james-donato-investigate-google)，情况可能会更糟。
