# Webb 捕捉到了标志性的马头星云，以前所未有的细节

2024年4月29日，201600次观看，158个赞

美国宇航局（NASA）/ 欧洲空间局（ESA）/ 加拿大航天局（CSA）的詹姆斯·韦伯空间望远镜捕捉到了迄今为止我们天空中最独特的物体之一，即马头星云的最锐利的红外图像。这些观测以前所未有的空间分辨率展示了这个标志性星云的复杂性。

[Webb 捕捉到了标志性的马头星云，以前所未有的细节](/ESA_Multimedia/Images/2024/04/Webb_captures_iconic_Horsehead_Nebula_in_unprecedented_detail)

[Webb 的](/Science_Exploration/Space_Science/Webb) 新图像展示了猎户座（猎户座）西侧的猎户座 B 分子云中一部分天空。从尘埃和气体的湍流波浪中升起的是马头星云，又名巴纳德33号，位于大约1300光年远的位置。

该星云形成于一个坍缩的星际云团中，并因附近的炽热恒星的照射而发光。环绕马头星云的气体云已经消散，但突出的柱状结构由难以侵蚀的厚实物质组成。天文学家估计马头星云还有大约五百万年的时间才会分解。Webb 的新视图聚焦在星云独特的尘埃和气体结构顶部被照亮的边缘。

放大到马头星云

[马头星云（NIRCam 图像）](/ESA_Multimedia/Images/2024/04/Horsehead_Nebula_NIRCam_image)

马头星云是一个众所周知的光子主导区域（PDR）。在这样的区域中，来自年轻的大质量恒星的紫外光形成了一个主要是中性暖气体和尘埃的区域，处于完全电离的围绕大质量恒星的气体和它们诞生的云层之间。这种紫外辐射强烈影响这些区域的气体化学反应，并作为加热的最重要来源。

这些区域发生在星际气体足够密集以保持中性的地方，但又不足够密集以阻止来自大质量恒星的远紫外光的穿透。从这些PDR发出的光提供了一个独特的工具，用来研究驱动我们银河系和整个宇宙中星际物质演化的物理和化学过程，从繁星形成的早期时代到今天。

由于其接近并且几乎边缘朝向的几何结构，马头星云是天文学家研究PDR物理结构以及其各自环境内的气体和尘埃化学特性演变以及它们之间过渡区的理想目标。被认为是研究辐射如何与星际物质相互作用的最佳天体之一。

[马头星云（MIRI图像）](/ESA_Multimedia/Images/2024/04/Horsehead_Nebula_MIRI_image)

多亏了Webb的 [MIRI](/Science_Exploration/Space_Science/Webb/MIRI_factsheet) 和 [NIRCam](/Science_Exploration/Space_Science/Webb/Webb_s_instruments) 仪器，国际天文学家团队首次揭示了马头座边缘照明的小尺度结构。他们还探测到了与PDR前沿垂直延伸的条纹特征网络，其中包含在星光蒸发流中携带的尘埃粒子和电离气体。这些观测还使天文学家能够研究尘埃衰减和发射效应，更好地理解星云的多维形态。

接下来，天文学家打算研究已经获得的关于星云的 [光谱数据](/ESA_Multimedia/Images/2021/06/Spectroscopy_with_Webb)，以证明跨星云观察到的物质的物理和化学特性的演变。

这些观测是在Webb GTO程序 #1192（PI: K. Misselt）中进行的，结果已被 *天文学与天体物理学*（Abergel等人，2024）接受发表。

**更多信息**

[Webb](/Science_Exploration/Space_Science/Webb) 是迄今为止发射到太空的最大、最强大的望远镜。根据国际合作协议，欧空局提供了望远镜的发射服务，使用了阿丽亚娜5号运载火箭。与合作伙伴一起，欧空局负责开发和适应阿丽亚娜5号的Webb任务，并通过阿利安斯宇宙公司采购发射服务。欧空局还提供了工作马达光谱仪 [NIRSpec](/Science_Exploration/Space_Science/Webb/Webb_s_instruments) 和中红外仪器 [MIRI](/Science_Exploration/Space_Science/Webb/MIRI_factsheet) 的50%，后者由欧洲国家资助的欧洲研究所联合体（MIRI欧洲联盟）与JPL和亚利桑那大学合作设计和建造。

Webb 是NASA、ESA和加拿大空间局（CSA）的国际合作伙伴关系。
