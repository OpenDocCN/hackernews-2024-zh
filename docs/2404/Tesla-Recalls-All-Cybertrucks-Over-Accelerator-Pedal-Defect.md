# 特斯拉因油门踏板缺陷召回所有 Cybertruck

> 来源：[https://gizmodo.com/tesla-recalls-all-cybertrucks-over-accelerator-pedal-de-1851421654](https://gizmodo.com/tesla-recalls-all-cybertrucks-over-accelerator-pedal-de-1851421654)

照片：Mike Mareen（Shutterstock）

[特斯拉的 Cybertruck](https://gizmodo.com/posts-about-broken-cybertrucks-fill-tesla-owners-forums-1851389945) 一直以来都因为其外观问题而备受尴尬，不过问题不仅止于此。这款看起来像是由小孩设计的电动车已经出现了[生锈问题](https://gizmodo.com/cybertruck-owners-say-their-stainless-steel-trucks-are-1851257880)，[在积水中表现不佳](https://jalopnik.com/watch-this-tesla-cybertruck-sputter-out-after-driving-t-1851347369)，甚至[有些连车洗都挺不过](https://gizmodo.com/drive-through-car-wash-was-too-much-for-tesla-cybertruc-1851420356)，而造成油门卡滞的问题最终促使了召回。

特斯拉投资者对特斯拉 Cybertruck 召回表示不满。

<track kind="captions" label="English" src="https://kinja.com/api/videoupload/caption/19013.vtt" srclang="en">

根据美国国家公路交通安全管理局（NHTSA）的[报告](https://static.nhtsa.gov/odi/rcl/2024/RCLRPT-24V276-7026.PDF)，特斯拉将于周五召回 3,878 辆 Cybertruck。这些车辆存在问题，即踏板顶部的垫子可能会脱落并卡进内饰，导致意外加速。这个缺陷是由于在组装过程中使用的润滑剂，旨在使垫子更容易连接到踏板上。不幸的是，这种临时解决方案似乎也使得垫子更容易从踏板上脱落。

特斯拉暂未就召回问题回应置评请求。

Cybertruck 受影响的车主将免费更换或重新制作他们的踏板。特斯拉已经开始解决生产中的缺陷，并将修复方案应用到已经送达的卡车上。

油门卡滞的问题在三月底开始引起车主们在社交媒体上的[发帖](https://gizmodo.com/tesla-cybertruck-hit-with-stop-sale-because-of-unintend-1851410669)。上周，特斯拉因此问题暂停了 Cybertruck 的销售。据美国国家公路交通安全管理局（NHTSA）称，并没有因该缺陷而导致死亡或伤害的报告。

Cybertruck 花了数年时间，从一个[尴尬的发布阶段](https://www.youtube.com/watch?v=udxR5rBq_Vg&t=1s)到去年十二月份[进入人们的车库](https://gizmodo.com/elon-musk-makes-first-deliveries-cybertruck-price-range-1851062357)。其“独特”的设计，起步价超过$80,000，导致了[一个又一个尴尬时刻](https://gizmodo.com/cybertruck-drivers-humiliating-themselves-are-the-bigge-1851320392)。而马斯克则更专注于从股东那里获得[$55亿的补偿方案](https://jalopnik.com/tesla-creates-website-to-convince-shareholders-to-make-1851421233)。
