# 汤姆·赛德曼·弗洛伊德的《兔子故事书》（1924） — 公共领域评论

> 来源：[https://publicdomainreview.org/collection/tom-seidmann-freud-hare-tales/](https://publicdomainreview.org/collection/tom-seidmann-freud-hare-tales/)

[](#p-1-0)

出生于维也纳，汤姆（原名玛莎）·赛德曼·弗洛伊德（1892–1930）是西格蒙德的姐姐玛丽亚·弗洛伊德的女儿。大约在15岁时，她选择了男性化的名字汤姆，并出国到伦敦和柏林学习艺术，最终定居在慕尼黑。在这里，她结识了学者格尔绍姆·施洛姆，他会[回忆](https://archive.org/details/fromberlintojeru00scho/page/124/mode/2up)她是一个“真正的波西米亚人”，“可以说是靠吸烟为生”。因其父亲罗马尼亚公民身份被巴伐利亚驱逐出境后，她返回柏林，在那里她创作了儿童图书并与即将成为丈夫的扬克夫·赛德曼创办了佩雷格林出版社 — [以](https://www.documenta14.de/en/artists/22765/tom-seidmann-freud)拉丁文*peregrinus*，即外国人，命名 —。在仅存的少数私人信件之一中，我们可以[听到](https://opus.bibliothek.uni-augsburg.de/opus4/files/95878/95878.pdf)汤姆自己对这段时期的兴奋：“Jakele 很好很聪明，我非常爱他。春天，我们所有的罪过都会得到赦免。这对我真的很重要！”正是通过佩雷格林，她出版了《兔子故事书》，以及1923年的[*鱼的旅程*](https://leopard.tu-braunschweig.de/receive/dbbs_mods_00000177)（The Fish’s Journey），讲述了一个年轻男孩在波光粼粼的海底发现金色王国的故事，该书刚刚在她的十几岁的弟弟溺亡于梅克尔湖后出版。她曾一度与诗人海姆·纳曼·比亚利克合作为希伯来语的童话插图 — 旨在供巴勒斯坦的犹太复国主义学校使用 — 后来转向编写教育文本。这些启蒙读物和抄写本的质量让瓦尔特·本雅明在*法兰克福时报*的一篇赞美文章中感慨地[建议](https://www.textlog.de/benjamin/kritik/chichleuchlauchra-fibel)，“在孩子们游戏的地方，埋藏着一个秘密”。
