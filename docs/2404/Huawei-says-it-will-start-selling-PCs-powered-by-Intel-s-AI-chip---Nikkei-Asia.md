# 华为表示将开始销售由英特尔人工智能芯片驱动的个人电脑 - 日经亚洲

> 来源：[https://asia.nikkei.com/Business/Technology/Huawei-says-it-will-start-selling-PCs-powered-by-Intel-s-AI-chip](https://asia.nikkei.com/Business/Technology/Huawei-says-it-will-start-selling-PCs-powered-by-Intel-s-AI-chip)

台北——中国华为技术公司周四推出了其首款人工智能驱动的个人电脑，宣称该电脑将配备英特尔最新的芯片组，并运行其自家的操作系统。

尽管美国对该公司实施了严格的限制措施，大大限制了其获取先进的美国技术，这一宣布仍然如期而至。
