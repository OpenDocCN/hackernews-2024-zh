# Annin Robotics - 开源的六轴机器人，您可以自己组装

> 来源：[https://www.anninrobotics.com](https://www.anninrobotics.com)

Annin Robotics的AR4机器人是一个六自由度桌面尺寸的工业机器人，是一个自由、开放、低成本的计划。任何人都可以制作这款机器人，可以使用铝合金零件，也可以自行3D打印零件。

​

这些机器人被广泛应用于小型自动化过程，并在数百所中小学、技校和大学的各种教育项目及爱好者社区中使用。

AR系列机器人的目标是打造成本最低的机器人，以便任何人都可以为了娱乐、教育目的或小规模生产操作而组装六轴机械臂。

​

AR4机器人臂使用基于Arduino的Teensy 4.1控制板，并采用免费开源的Python控制软件。
