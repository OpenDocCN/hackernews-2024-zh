# 博斯特罗姆的《深度乌托邦》- 罗宾·汉森撰写 - 克服偏见

> 来源：[https://www.overcomingbias.com/p/bostroms-deep-utopia](https://www.overcomingbias.com/p/bostroms-deep-utopia)
> 
> 尼克·博斯特罗姆的新著……封面精美，提出了许多有趣的问题，副标题暗示可能探讨在人工智能和机器人能够做所有事情的未来中生命的意义。但可惜的是，在经过多次预热和期待后，他在第427页上留下了一个未答之谜，突然间没时间了……他试图探讨像是，生活如何保持有趣？当挑战消失后，我们的目的和意义是什么？实现感是否可以充实？但在每种情况下，教学方法更多是所有可能答案的调查，而不是更为困难的任务，即作出具体预测。([更多信息](https://twitter.com/FutureJurvetson/status/1773054722901180596))
> 
> 人工智能解决了你所有问题，生活会怎样？你就不必再工作了。只需抬一抬眉毛，想要什么奢侈品都能得到……在《深度乌托邦》中，博斯特罗姆聪明地选择了想象……人工智能会精确执行我们要求的任务。虽然他说我们不适合追求完美，博斯特罗姆确实探讨了乌托邦生活如何变得可接受。博斯特罗姆……将这本书视为探索人工智能未来的一部作品，而非对其结论的阐述。他将这本书比作粒子加速器，将原子撞击在一起以研究它们的组成部分，如夸克。在《深度乌托邦》中，他说，他将价值观相互碰撞，以研究它们的构成。([更多信息](https://www.nytimes.com/2024/04/05/opinion/ai-jobs-nick-bostrom.html))

在他的新书*[深度乌托邦](https://nickbostrom.com/deep-utopia/)*中，尼克·博斯特罗姆问道，如果与他非常相似的生物可以在永恒中生活，拥有完全和平、广阔的财富和完全控制极其高效的人工智能，他们会如何选择生活。他主要试图列举尽可能多的合理可能性。

波士顿的写作引人入胜，尽管不够简洁。他没有克服人们通常的印象，即想象引人注目的乌托邦相当困难，尽管我认为他确实有助于澄清一些他的生活目标。

但我发现很难被他的关键问题所激发。在比我们更能干的生物的未来中，我更感兴趣的是那些更好的生物会做什么，而不是像我现在这样的生物会在那里做什么。而且我觉得像我这样的生物长期富裕、和平，并完全控制这样一个世界的想法相当[不太可能](https://twitter.com/robinhanson/status/1779571114403876877)。

此外，在无论哪种实际未来场景最接近他的假设中，我只是不相信在那里我们能力水平的生物会有像波斯特罗姆今天的价值观。波斯特罗姆提出他的问题，关注的是他身边的左翼学者和富裕西方社会。在我们今天的世界中，或者来自过去文明的人们，会发现他在他假设情景中如何实现*他们*的愿望的书籍中很少提到荣誉，享受征服者的哀叹，或者“后代繁多，如天上的星星和海边的沙粒”。

显然，苏联曾经想找外星人，因为他们假定外星人一定非常聪明，所以他们会是共产主义者。但期望一个不到一个世纪的立场对亿万年更先进的外星人有重要意义似乎很疯狂。同样，许多未来学家试图在接近自己文化价值观的背景下想象疯狂先进的技术和社会制度，即使这些价值观在几个世纪前的现实中几乎不存在。

未来将会很奇怪。尽管很难想象技术、制度和宇宙的变化，但更难想象未来的文化。但我可以向你保证，它们将会有很大变化，*很多*。
