Kerberos是一种网络认证协议。它旨在通过使用秘密密钥加密技术为客户端/服务器应用程序提供强大的身份验证。这一协议的免费实现可在[麻省理工学院](https://web.mit.edu/)获取。Kerberos也作为商业产品提供。

**互联网是一个不安全的地方**。互联网中使用的许多协议都不提供任何安全性。恶意黑客普遍使用“嗅探”工具从网络中截取密码。因此，将未加密密码发送到网络的应用程序极其容易受到攻击。更糟糕的是，其他客户端/服务器应用程序依赖于客户端程序“诚实”地声明正在使用它的用户身份。其他应用程序依赖于客户端仅限于其被允许执行的活动，服务器不作任何其他强制执行。

一些网站尝试使用[防火墙](firewalls.html)来解决其网络安全问题。不幸的是，防火墙假设“坏人”在外部，这通常是一个非常错误的假设。大多数真正有害的计算机犯罪事件都是由内部人员实施的。防火墙在这样做的同时也有重大缺点，即它们限制了用户如何使用互联网。 （毕竟，防火墙只是不连接网络且关闭电源的计算机的一个不那么极端的例子！）在许多地方，这些限制根本不现实，也是不可接受的。

Kerberos是MIT为解决这些网络安全问题而创建的**解决方案**。Kerberos协议使用**强加密**，使得客户端可以在不安全的网络连接中向服务器证明其身份（反之亦然）。客户端和服务器使用Kerberos证明其身份后，还可以加密所有通信，以确保隐私和数据完整性。

Kerberos可以在MIT免费获取，其版权许可与BSD操作系统和X Window系统非常相似。MIT以源代码形式提供Kerberos，以便任何希望使用它的人可以检查代码，确保代码可信。此外，对于那些更倾向于依赖专业支持产品的人，Kerberos也可以从许多不同的供应商处获得。

总之，Kerberos 是解决您网络安全问题的方案。它提供了身份验证工具和强大的网络加密，帮助您保护整个企业的信息系统。我们希望您能像我们一样发现 Kerberos 的实用性。在 MIT，Kerberos 对我们的信息技术架构至关重要。
