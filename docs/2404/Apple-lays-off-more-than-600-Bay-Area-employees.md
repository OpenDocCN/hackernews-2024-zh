# 苹果公司裁员超过600名旧金山湾区的员工

> 来源：[https://sfstandard.com/2024/04/05/apple-layoffs/](https://sfstandard.com/2024/04/05/apple-layoffs/)

科技巨头苹果公司根据一份州政府的文件，在圣克拉拉县的八个办公室裁员614人。

这一裁员行动是继今年早些时候决定停止电动汽车项目后的举措。这一举动与CEO [蒂姆·库克的保证](https://www.cnbc.com/2023/05/04/apple-layoffs-are-a-last-resort-tim-cook-says.html)相矛盾，去年他曾表示公司并未考虑大规模裁员。

尽管[人工智能行业蓬勃发展](https://sfstandard.com/2024/03/26/san-francisco-economy-after-edd-report/)，但根据该州就业发展部最新修订的报告，旧金山和圣马特奥县在2023年2月至2024年2月间总计失去了16,700个工作岗位。其中约有14,300个工作岗位位于包括科技在内的信息产业。

苹果公司自疫情爆发以来首次进行大规模裁员，裁员将于5月27日生效。
