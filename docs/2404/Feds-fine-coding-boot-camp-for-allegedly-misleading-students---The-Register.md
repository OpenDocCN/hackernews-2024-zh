# 美联储因涉嫌误导学生对编码训练营处以罚款 • The Register

> 来源：[https://www.theregister.com/2024/04/18/feds_say_coding_boot_camp/](https://www.theregister.com/2024/04/18/feds_say_coding_boot_camp/)

美国消费者金融保护局（CFPB）因涉嫌欺诈商业行为对编码训练营BloomTech进行了多项处罚。

该企业在其[网站](https://www.bloomtech.com/)上声称将帮助学生在亚马逊、思科和谷歌等公司获得“梦想工作”，接受了未承认或否认任何不当行为的和解命令。

昨天，在一份[公告](https://www.consumerfinance.gov/about-us/newsroom/cfpb-takes-action-against-coding-boot-camp-bloomtech-and-ceo-austen-allred-for-deceiving-students-and-hiding-loan-costs/)中，美国消费者金融保护局（CFPB）表示已对BloomTech及其首席执行官奥斯汀·奥尔雷德采取行动，指控其未向学生披露贷款的真实成本，并声称BloomTech毕业生的就业率过于乐观。BloomTech，前身为Lambda School，自2017年以来一直运营，并提供为期六至九个月的科学和工程职业项目，重点是计算机技术。

“BloomTech及其首席执行官试图推动学生选择被宣传为无风险的收入份额贷款，但实际上带有显著的财务费用，且与其他信用产品具有相同的风险，”CFPB主任罗希特·乔普拉表示。通过收入份额贷款或收入份额协议，BloomTech允许学生稍后支付学费，但作为交换，必须支付其未来收入的一部分，CFPB声称。

该机构声称BloomTech明确告诉学生，其收入份额贷款（平均需要支付约$4k的“财务费用”）实际上根本不是贷款。CFPB在[和解](https://files.consumerfinance.gov/f/documents/cfpb_bloomtech-inc-consent-order_2024-04.pdf) [PDF]中声称，“绝大多数”学生使用这些贷款来支付教育费用，并声称每名学生最终可能需要支付高达$30k的收入给BloomTech来清偿贷款。

另外，据称BloomTech还将部分贷款出售给投资者，违背了“只有当你（学生）赚钱时才赚钱”的承诺。此外，CFPB声称BloomTech违反了持有人规则，即在将贷款出售给投资者时未给予学生某些权利，这在消费者保护监督机构看来是非法的。

根据命令，编程学院被指称在其毕业生间的虚构招聘率为71%至86%，而实际情况是在大约50%左右，有时候甚至低至30%。这项指控声称，BloomTech在董事会中披露的“非公开”报告中透露了这一实际招聘率。文件还声称，贝姆科技的CEO奥斯特·奥尔德尔在一次场合中称其中一个课程的招聘率为100%，但后来澄清这只是针对一个单独学生的课程。

当撰写此文本时，BloomTech的网站正在宣传一个86%的招聘率。

"CFPB发现BloomTech和奥尔德尔使用误导性声明，并且在消费者的合理信赖BeamTech会以他们的利益行事这一点上获得了非正当的优势，"该机构表示。

奥尔德尔就CFPB的声明在X上进行回应，表示虽然他接受罚款，但否认指控。"我们决定和解是因为很清楚地持续的法律诉讼将极为耗时，无比昂贵，并且会让我们分心，无法专注于我们的核心使命，"这位CEO说,"我们进行和解并没有同意或否认执煾示范的任何指控。"

他还提到，BloomTech在2021年“主要停止”使用收入共享贷款，并不再提供服务。然而，公司的FAQ下仍有一项问题提到收入共享贷款（或称ISAs）。

### 学生可以撕毁他们的收入共享协议。

BloomTech面临于严重的制裁。它不再能够从在过去一年内未能获得工作的毕业生处收取收入共享贷款的款项，还需取消超过18个月前毕业的、年薪低于70,000美元的学生的数千美元贷款费用。此外，当前学生必须能够解除贷款并退学，或在继续就读的前提下能使用第三方贷款。

此外，BloomTech和奥尔德尔个人必须向CFPB的受害者救济基金缴纳罚款，公司罚款64,000美元，CEO罚款100,000美元。奥尔德尔被禁止从事学生贷款业务达十年之久。给单个学生的一笔收入共享贷款可能要高达30,000美元，所以罚款对编码机构的财务影响可能不是最大的。
