# HiFive Premier P550 - SiFive 开发板

> 来源：[https://www.sifive.com/boards/hifive-premier-p550](https://www.sifive.com/boards/hifive-premier-p550)

## HiFive Premier P550 *特点与规格*

机器的核心是 SiFive Performance™ P550 核心复杂系统，采用 12nm 工艺制造；这是一个四核应用处理器，具有十三级、三重发射、乱序流水线，支持 RISC-V RV64GBC 指令集，内置 LPDDR5、eMMC 和 PCIe Gen3。板载功能包括 16GB/32GB LPDDR5、1x PCIe 扩展槽、双 10/100/1000 以太网口、SATA3、5 个 USB 3.0 端口、板载 GPU、NPU、2D/3D 加速器以及支持最高 8K@50fps 或 28 通道 1080P@30fps 视频编码/解码器。这是一款高级软件开发系统，采用模块化设计，使用系统级模块 (SOM)，拥有 13.3 TOPS AI 计算能力，专为机器视觉、智能视频分析、AI PC 和服务器解决方案等应用设计。RISC-V 无所不能。

搭载 Eswin EIC7700 SoC，配备四核 SiFive Performance P550 核心复杂系统

16GB 或 32GB DDR5，128GB eMMC

双千兆以太网口 + 远程调试以太网口

x16 PCIe® Gen 3 扩展槽

M.2 E-Key 插槽（PCIe Gen 3），用于 Wi-Fi/蓝牙模块

SATA3 连接器（6Gb/s）

行业标准 miniDTX
