# 并且有Digitron

> 来源：[https://www.retrobit-izlozba-digitrona.com/digitron-and-buje/and-there-was-digitron](https://www.retrobit-izlozba-digitrona.com/digitron-and-buje/and-there-was-digitron)

双方都要求更多领土，威胁不断，武器随之响起。

在这样的氛围中，生活是困难的，因此在1954年至1964年期间，几乎有10,000名居民离开布伊什蒂纳，寻求更安全的生活和就业机会，留下了10,000公顷的耕地。
