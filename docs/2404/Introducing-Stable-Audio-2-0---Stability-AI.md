# 介绍稳定音频 2.0 — Stability AI

> 来源：[https://stability.ai/news/stable-audio-2-0](https://stability.ai/news/stable-audio-2-0)

今天，我们很高兴推出[稳定音频 2.0](https://stableaudio.com/)。这款模型能够通过单一的自然语言提示生成高质量、全长达三分钟、44.1 kHz立体声的完整曲目。

这款新模型不仅限于文本转音频，还包括音频到音频的能力。用户现在可以上传音频样本，并通过自然语言提示将这些样本转换为各种声音。此更新还扩展了声效生成和风格转换功能，为艺术家和音乐家提供了更多的灵活性、控制和提升的创作过程。

稳定音频 2.0在[稳定音频 1.0](https://stability.ai/news/stable-audio-using-ai-to-generate-music?__hstc=21858660.f6b5ccdfb81bac2a567ea0a5d95bab83.1711472661937.1711657025410.1711659307340.6&__hssc=21858660.1.1711659307340&__hsfp=851621926)的基础上进行了构建，后者于2023年9月首次商业化推出，是第一个能够生成高质量44.1kHz音乐的AI工具，利用了潜在扩散技术。此后，它被评为[TIME 2023年最佳发明](https://stability.ai/news/stable-audio-times-best-inventions-of-2023?__hstc=21858660.f6b5ccdfb81bac2a567ea0a5d95bab83.1711472661937.1711657025410.1711659307340.6&__hssc=21858660.1.1711659307340&__hsfp=851621926)之一。

这款新模型今天可以在稳定音频[网站](https://stableaudio.com/)免费使用，即将在稳定音频API上线。

**新功能**

我们迄今最先进的音频模型扩展了艺术家和音乐家的创作工具包，增加了新的功能。使用文本转音频和音频到音频的提示，用户可以制作旋律、伴奏曲、音轨和音效，从而提升创作过程。

**完整长度曲目**

稳定音频 2.0以其可以生成长达三分钟的歌曲，包括开头、发展和结尾的结构化作品以及立体声音效，使自己脱颖而出，超越其他最先进的模型。
