## 开发开始

StreetComplete 是一个使用游戏化方式让用户参与现实任务的应用程序，旨在通过游戏化贡献于广泛使用的 OpenStreetMap 项目的开发。该团队最近启动了他们流行的安卓应用的 iOS 版本的开发。根据他们的 GitHub 页面，贡献者寻求志愿者熟悉代码并帮助开发这个应用。

> ## 你可以如何帮助
> ## 
> +   你可以接管项目板上的任务！或者，你可以熟悉提到的技术/框架，以便以后做出有意义的贡献。特别是：Jetpack Compose / [Compose Multiplatform](https://www.jetbrains.com/lp/compose-multiplatform/)。这对我来说也很新，所以我也需要找到自己的位置。
> +   
> +   你可以赞助开发！请看侧边栏上的“赞助此项目”标题。更多资金让我可以在总体上更多时间投入到这个应用的开发中。如果你知道任何适合的大规模资金来源，请与我联系！
> +   
> +   你可以一般性地帮助维护和问题分类。在这方面节省的时间，我可以用来进一步开发这个项目！
