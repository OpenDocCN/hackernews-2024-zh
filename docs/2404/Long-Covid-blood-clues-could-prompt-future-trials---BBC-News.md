# 长期Covid的血液线索可能促使未来的试验 - BBC News

> 来源：[https://www.bbc.co.uk/news/health-68762171](https://www.bbc.co.uk/news/health-68762171)

在《自然免疫学》期刊上的研究人员还发现，长期Covid患者血液中的一些蛋白质可能与其特定症状相关联。

例如，那些有胃肠症状的人体内SCG3标记物水平增加，此前与肠道与大脑之间的通讯障碍有关。

研究人员表示，这可以帮助将长期Covid患者分为不同的亚组，并对设计临床试验特别有用，尤其是针对免疫反应和炎症的治疗。

伦敦帝国学院临床研究员Felicity Liew博士表示，研究结果表明，炎症“可能是长期Covid在住院后的共同特征，无论症状类型如何”。

这可能会打开试验现有药物对抗长期Covid的大门，例如用于类风湿性关节炎的药物，这是一种自身免疫条件，导致关节炎炎症。

主导研究的Peter Openshaw教授说：“这项工作提供了强有力的证据，表明长期Covid是由病毒后炎症引起的，但显示了复杂的层面。

"我们希望我们的工作为开发针对各种长期Covid的特定测试和治疗方法打开道路，并认为‘一刀切’的治疗方法可能行不通。"
