# 加州理工学院恢复本科生入学标准化考试要求 - www.caltech.edu

> 来源：[https://www.caltech.edu/about/news/caltech-restores-standardized-test-requirement-for-undergraduate-admission](https://www.caltech.edu/about/news/caltech-restores-standardized-test-requirement-for-undergraduate-admission)

加州理工学院恢复了对申请本科学习的潜在学生提交SAT或ACT考试成绩的要求。该测试要求立即生效，这意味着从2024年秋季开始申请并在2025年秋季入学的所有学生都必须在申请中报告考试成绩。

这一决定是根据本科招生政策顾问委员会的建议作出的，认为标准化考试提供给招生官和教员有关学术准备情况的有用信息，作为对所有潜在学生进行全面考虑的一部分的判断。加州理工学院恢复标准化考试要求，重申作为科学家和工程师社区的承诺，利用所有相关数据进行决策过程。

委员会发现，即使在实施了考试禁令的情况下，每年仍有越来越多的申请者完成了标准化测试，最近一届入学的学生中超过95%的学生参加了标准化考试。但是考试成绩直到最终录取决定后才对招生办公室可见。在即将到来的招生周期中，考试成绩将作为学生独特情况和经历的一个额外数据点。顾问委员会指出：“我们认为我们的招生办公室和审阅申请者的教员必须获得所有可能影响他们对潜在学生准备情况理解的信息，这一点至关重要。”

该学院致力于吸引和教育世界上最有才华和前途的学生——那些对科学、工程和数学有深刻兴趣，并具备在面对学术挑战时蓬勃发展的能力、活力和适应力的学生，他们为我们的社区带来了多样化的经历和观点。“我们为过去几年招收的学生感到自豪，并对我们为提升识别能力所采取的步骤充满信心，这些能力将使学生在加州理工学院及其之后的生活中取得成功，”Provost David Tirrell说道。

该学院在2020年实施停止SAT和ACT考试要求时，由于COVID-19大流行，暂停了标准化考试对全球学生的访问。该禁令在2021年延长，然后在2022年再次延长。该学院的决定终止了最初定于2025年到期的禁令。
