# Cascade解决了这个争论：在把碟子放入洗碗机之前是否需要漂洗？

> 来源：[https://www.allrecipes.com/should-you-rinse-plates-before-loading-dishwasher-8623561](https://www.allrecipes.com/should-you-rinse-plates-before-loading-dishwasher-8623561)

我要坦白一件事：我是个预漂洗洁碗的人。在把[脏碟子放进洗碗机](https://www.allrecipes.com/article/how-to-load-a-dishwasher/)之前，我会彻底冲洗它们，让它们几乎干净了再放入洗碟架。但我不确定为什么要这样做。也许是因为我在成长和做家务时习惯这么做，或者可能是因为我担心我们1940年代的老房子的水压和管道问题。

结果证明，我在预漂洗的习惯上并不孤单。Procter & Gamble委托进行的[最新调查](https://news.pg.com/news-releases/news-details/2023/New-Cascade-Platinum-Plus-Raises-the-Standard-for-Clean-Dishes-with-Breakthrough-Innovation/default.aspx)显示，我们中有86%的人在洗碗的例行程序中会预漂洗或预清洗。

我开始想，我在[把碟子放入洗碗机](https://www.allrecipes.com/best-way-to-load-a-dishwasher-according-to-bosch-8422641)之前做预漂洗，这样做有什么好处吗？还是说像水这样的时间和资源是浪费？我联系了[洗碗机洗涤剂巨头Cascade](https://www.allrecipes.com/article/tiktok-was-wrong-about-dishwasher-pods/)寻找答案。

## 你需要在把碟子放入洗碗机之前预清洗吗？

根据Cascade的高级科学家摩根·艾伯哈德的说法，答案是否定的。但有一个例外情况。

“你需要确保刮掉任何可能[堵塞或阻塞你的机器](https://www.allrecipes.com/article/how-to-clean-dishwasher-filter/)的残留食物块，但是可以留下碟子上最后一层的食物残渣、酱汁和油脂。”Eberhard建议说，“完全不需要打开水龙头。”

她还说，我们可以通过使用洗碗机跳过预漂洗来节约水和能源，更不用说时间了。

“能源之星认证的洗碗机在一个循环中使用不到四加仑的水；而开着的水槽在仅仅两分钟内就可以使用相同的量。”Eberhard告诉Allrecipes，“每天通过跳过水槽，[使用你的洗碗机](https://www.allrecipes.com/article/things-you-didnt-know-your-dishwasher-could-do/)可以节省一周100加仑的水。”

“能源之星认证的洗碗机比在水槽中洗碗节约能源，所以你也可以通过使用洗碗机来节省水费。”她补充道。

考虑到我家老化且可能脆弱的管道，我问Eberhard像Cascade这样的[洗碗机清洁剂](https://www.allrecipes.com/article/baking-soda-dish-detergent/)是如何分解洗碗机中的食物、酱汁和油脂的。

“像Cascade Platinum Plus这样的超强装洗碗机洗涤剂有强大的清洁成分组合，可以水合并擦洗难以去除的顽固食物，使餐具几乎无斑点，即使放入橱柜也能直接使用。” 她解释道。“双效清洁配方由抗淀粉和抗蛋白酶组成，能比以往更有效地分解难以清洗的食物。”

## 装载洗碗机的有效技巧

现在我们已经得到确凿的答案，不必在装载洗碗机之前冲洗或预洗餐具，我请埃伯哈德提供她如何确保我们的餐具每次都能洗得干干净净的建议。

### Cascade高级科学家摩根·埃伯哈德

“装载洗碗机时最重要的事情是记住，来自喷臂的水必须能够到达盘子的脏部分。如果碗在一起嵌套或盘子叠放在一起，水就无法到达它们，这意味着洗涤剂无法分解盘子上残留的食物和油脂。”

— Cascade高级科学家摩根·埃伯哈德

“确保每个盘子在每一次清洗中都有通向水源的清晰路径，以获得最佳清洁效果”，Eberhard继续说道。“餐具也是如此。按设计，勺子喜欢，嗯，挨在一起。尽量混合每个餐具篮的各种餐具类型，以免它们最终都挤在一起。”

她补充道，确保叉子和勺子在洗碗机出来时最干净的方法是立起来装载它们，“这样最脏的部分更容易接触到水，而不是被压在篮子底部。” 要确保刀子朝下装载以确保安全，“这样在卸载时不会意外割伤自己。”

Cascade提供的额外[洗碗机装载技巧](https://www.allrecipes.com/article/how-to-load-a-dishwasher-the-right-way/)包括：

+   **双重检查物品是否适合洗碗机：** 洗碗机可能对[某些珍贵的厨房物品](https://www.allrecipes.com/article/what-is-not-dishwasher-safe/)过于粗糙，或者对木头、铝和铸铁等材料不适合。如果不确定，一定要再次确认所有物品是否适合洗碗机。

+   **装载底部架：** 将盘子和沾有食物的锅碗瓢盆放在底部架上，朝向齿的方向，面向水源。通常，水源位于机器中心，你希望喷水和脏盘之间有尽可能清晰的路径。

+   **上层货架的装载：** 将杯子、马克杯和碗倒置放在上层货架的叉齿间，以防止它们发出嘈杂声音。这样做不仅可以更好地清洗它们，还可以避免积水，从而避免留下不雅的沉积物。对于[酒杯](https://www.allrecipes.com/article/dishwasher-top-rack-hack/)和其他易碎玻璃器皿，将它们在上层货架上留出一指宽的间隔，并利用杯杆架（如果您的洗碗机有的话）来固定它们，以防它们倒下。塑料器皿也应放在上层货架上。由于上层货架的温度通常较低，这可以防止塑料熔化。
