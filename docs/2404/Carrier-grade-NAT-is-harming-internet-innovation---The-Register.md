# Carrier-grade NAT 正在损害互联网的创新 • The Register

> 来源：[https://www.theregister.com/2022/01/20/ipv4_nats_slow_ipv6_transition/](https://www.theregister.com/2022/01/20/ipv4_nats_slow_ipv6_transition/)

运营商和大科技公司乐意继续使用网络地址转换（NAT）和 IPv4 来保护其投资，结果导致 IPv6 的过渡进展缓慢，整个互联网的形象正受到现任者的影响。

这是 Geoff Huston 的观点，他是亚太网络信息中心（APNIC）的首席科学家。

Huston 的观点发表在一篇标题为《2021 年 IP 地址分配》的[长篇文章](https://blog.apnic.net/2022/01/19/ip-addressing-in-2021/)的结论部分中，该文报告了去年 IPv4 和 IPv6 的使用情况。

文章深入详细讨论了 2021 年两种协议的采用情况。主要新闻是可用 IPv4 地址的小池子继续缩减，即使地址交易有所增加。与此同时，IPv6 的采用率仅增长了 3%，更有意思的是，去年没有任何主要参与者迁移到这一更新的协议。

> 在 IPv4 中广泛使用 NAT 限制了互联网的技术基础，将其限制在一个非常简单的客户端/服务器交互模型中。

但 Huston 认为原始数据并未说明实际发生了什么。他认为，IPv4 中 NAT 的广泛使用掩盖了互联网大多数仍停留在 IPv4 上的事实，因为现任者们并未看到改变的必要性。

Huston 的文章思考了这一切对互联网未来的意义，而他并不认同眼前的景象。

"我们正在目睹一个不再将技术创新、开放性和多样化作为推动力的行业，" 他写道。

"在 IPv4 中广泛使用 NAT 限制了互联网的技术基础，将其限制在一个非常简单的客户端/服务器交互模型中，使用 TCP 和 UDP。NAT 的使用迫使交互进入客户端发起的交易，而今天的网络中已不再维持一个具有相当通信灵活性的开放网络模型。"

"现任者正在巩固其地位，而在这个漫长的 IPv4/IPv6 过渡期间，创新和企业家精神却被放在了次要位置。"

那些现任者规模庞大，行动符合自身利益。

"今天的互联网运营服务由少数几家大型运营商提供，每家都似乎在各自的市场中占据非常强大的位置，" Huston 观察到。"这些大型运营商的驱动因素倾向于风险规避、保守主义，并在其运营范围内增加控制水平。"

APNIC 在 2021 年 12 月表达了类似的担忧，当时一份由拉丁美洲互联网注册机构 LATNIC 联合委托的报告 [建议](https://www.theregister.com/2021/12/09/study_on_the_internets_technical_success_factors/)，大科技公司的内部网络承载了全球流量的很大一部分，使得像谷歌和 Facebook 这样的公司对互联网架构具有巨大影响力。

在他的文章中，Huston 进一步探讨了这一主题。"互联网行业的演变对网络中立性、运输与服务提供功能的分离、投资配置、基础设施投资的风险和回报预期以及互联网本身的开放性都产生了深远的影响。

"考虑到这一行业的规模经济，要维持一个高效、完全开放和竞争激烈的行业一直是具有挑战性的，但当基础平台耗尽 IP 地址这一基本货币时，这一议程的挑战程度成倍增加，" 他写道。

"如今，我们似乎越来越倾向于寻找一个能够挑战新成立的主导者日益显露的自满情绪的监管和治理框架，" 他悲叹道，并以或许是令人沮丧的观察结尾："目前还不清楚我们在这一探索中会有多大成功。" ®
