# 1build的营销主管 | Y Combinator

> 来源：[https://www.ycombinator.com/companies/1build/jobs/oW1x2mB-marketing-lead](https://www.ycombinator.com/companies/1build/jobs/oW1x2mB-marketing-lead)

[Handoff](http://handoffai.com)是美国5000亿美元翻新承包商市场的新移动和网络应用程序。我们利用生成AI技术为翻新承包商提供快速、准确和直观的方式，仅需几分钟即可创建施工成本估算。美国有60万以上的承包商可能成为Handoff的用户，我们正在寻找与我们一样充满激情的团队成员，帮助我们赢得这个市场。

Handoff由风险投资支持的科技公司[1build](http://1build.com)开发，其起源于硅谷。1build已在建筑定价数据和与顶级建筑材料供应商的集成中确立了市场领导地位。1build团队遍布全球，成员来自苹果、特斯拉和WeWork等顶级科技公司。

## 作为营销团队的成员，您将：

+   成为我们的市场开拓先锋，帮助Handoff发现最有利可图的获取渠道，并开发系统以优化这些渠道。

+   在理想客户配置文件（住宅翻新承包商）中实现潜在客户快速增长，同时降低获取成本。

+   发展和扩展未付费渠道，包括SEO、联盟/影响者营销和社交媒体。

+   发展和管理我们的Facebook社群，打造成为一个增长渠道。

+   展示在市场技术工具方面的专业知识，如Hubspot、Meta/Google广告、Google Analytics、Google Search Console和Ahrefs（或类似工具）。

+   以英语撰写出色的内容，吸引读者，并有效传达我们产品的价值。

+   利用社交媒体进行客户获取，并优化内容以符合SEO要求（理解关键词、回链、索引等）。

+   运行快速实验，了解有效和无效的策略，并相应优化营销工作。

+   测量、量化并优化营销倡议，以在有限资源下实现最大影响。

+   根据需要协助产品和销售团队进行营销支持、运营活动和其他支持。

## 关于您：

+   以产品为中心，具有与以产品为导向的SaaS合作和销售经验。

+   精通技术，始终寻找使您的流程更加高效和有效的方法。

+   为开发和优化从0到1的获取渠道的挑战感到兴奋。

+   积极解决问题，找到解决方案并消除障碍。

+   热衷于成为创业环境的一部分，并肩投入工作以完成任务。
