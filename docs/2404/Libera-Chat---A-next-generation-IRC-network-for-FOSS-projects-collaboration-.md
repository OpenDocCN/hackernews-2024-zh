# 三周年！

[2024年5月19日](/news/three) 由 el

大家好！

当我们三年前开放了Libera Chat时，我们并不确定事情会如何发展。当时确实有人怀疑我们今天是否还会在这里，努力为大家提供稳定的协作平台。但是我们在这里！

除了在眨眼之间建立了一个完整的网络外，我们还为过去36个月取得的各种成就感到非常自豪。我们已经：

我们还在积极开发我们的下一代IRCd，名为Sable。您可以[查看代码库](https://github.com/Libera-Chat/sable)或加入`#libera-dev`获取更多信息。

最后，从*下个月*开始，我们的[免费商店](https://www.freewear.org/Libera.Chat)将开始销售今年四月愚人节玩笑中的libera.cat标志设计！请关注我们的[bluesky](https://bsky.app/profile/libera.chat)或[fosstodon](https://fosstodon.org/@liberachat)社交账号以获取相关公告。

与此同时，请加入我们的`#libera-birthday`庆祝活动！
