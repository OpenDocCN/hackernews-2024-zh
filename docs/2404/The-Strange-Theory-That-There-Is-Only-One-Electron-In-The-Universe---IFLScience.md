# 宇宙中只有一个电子的奇怪理论 | IFLScience

> 来源：[https://www.iflscience.com/the-strange-theory-that-there-is-only-one-electron-in-the-universe-73818](https://www.iflscience.com/the-strange-theory-that-there-is-only-one-electron-in-the-universe-73818)

据估计，可观测宇宙中大约有10^(82)个原子。由于周期表中的每个元素至少含有一个电子，因此你可以安全地假设可观测宇宙中至少有10,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000,000个电子。

或者你可以？根据理论物理学家约翰·惠勒提出的理论，他在与同行物理学家[理查德·费曼](https://www.iflscience.com/feynmans-reversed-sprinkler-puzzle-finally-has-a-solution-72685)的对话中阐述了他的想法，宇宙中只有一个电子——只是看起来有很多是因为它在时间的前后移动。

尽管这听起来很奇怪，但这本身是对电子极端奇异性的回应。电子像其他基本粒子一样，彼此无法区分。它们具有相同的负电荷、相同的质量和[相同的自旋](https://www.theguardian.com/science/life-and-physics/2011/mar/17/1#:~:text=In%20contrast%2C%20the%20mass%20of,it%20has%20this%20particular%20mass.)。将一个电子换成另一个，你[无法分辨](https://phys.libretexts.org/Bookshelves/Quantum_Mechanics/Advanced_Quantum_Mechanics_(Kok)/12%3A_Multiple_Particle_States/12.1%3A_Indistinguishable_Particles)它们。

它的反粒子——正电子——在电荷、质量和自旋上也无法区分。奇怪的是，它们与电子完全相同，除了它们的正电荷。正是这些因素导致惠勒建议电子和正电子实际上只是一个粒子，它在向前的时间中带有负电荷，在向后的时间中带有正电荷。

“有一天，在普林斯顿研究院，我接到了惠勒教授的电话，他说，‘费曼，我知道为什么所有电子都有相同的电荷和相同的质量’”，费曼在他1965年的[Nobel Lecture](https://www.nobelprize.org/prizes/physics/1965/feynman/lecture/)中说道。

"'因为，它们都是同一个电子！' 然后他在电话上解释道，'假设我们通常在时间和空间中考虑的世界线，而不是只在时间上升，是一个巨大的结，然后，当我们通过与固定时间对应的平面切割结，我们将看到许多许多世界线，这些世界线代表许多电子，除了一件事。如果在一个部分这是一个普通的电子世界线，在它倒转并从未来回来的部分中，我们对适当的时间-适当的四速度有错误的符号 - 那相当于改变电荷的符号，因此，路径的这一部分将表现得像一个正电子'。”

这将使一个古老的粒子，通过时间来回弹了一个真正不可理解的次数。虽然这将是一个有趣的解释为什么电子和正电子共享性质的方式，但是几乎不可能是正确的。正如费曼指出的那样，在宇宙中，正电子并不像电子那样多，[更多](https://theconversation.com/cern-discovery-sheds-light-on-the-great-mystery-of-why-the-universe-has-less-antimatter-than-matter-147226#:~:text=Scientists%20believe%20that%20in%20the,preference%20to%20matter%20over%20antimatter.)物质比反物质多。如果正电子和电子是同一个前向和后向移动的基本粒子，你会期望它们是偶数。

"嗯，也许[缺失的正电子]藏在质子或其他什么东西里面，" 威勒的解释，不太令人信服。虽然这只是一个思想实验，很可能不是[应该认真对待的](https://interestingengineering.com/science/could-every-electron-in-the-universe-be-the-same-one)，但这通话对费曼产生了深远的影响，写了一篇[论文](https://www.taylorfrancis.com/chapters/edit/10.1201/9780429493249-10/theory-positrons-feynman)讨论正电子可以描述成像电子倒退移动时间。

"我没有像我对待正电子可以简单地被表示为电子在时间上倒退的观察那样认真对待他所有的电子都是相同的想法，他补充道。"那，我抄袭了！"
