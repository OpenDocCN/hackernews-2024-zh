# 洛杉矶正在使用AI试点计划试图预测无家可归问题

> 来源：[https://www.cnbc.com/2024/04/19/los-angeles-is-using-an-ai-pilot-program-to-try-to-predict-homelessness.html](https://www.cnbc.com/2024/04/19/los-angeles-is-using-an-ai-pilot-program-to-try-to-predict-homelessness.html)

2021年2月1日，一名女性经过洛杉矶加利福尼亚州街头排列的无家可归人士的帐篷。

FREDERIC J. BROWN | AFP | Getty Images

原文：去年十二月，单身母亲Courtney Peterson因为一家现已关闭的住院过渡居住项目的工作而被裁员。除了可以带着她7岁的儿子上班外，这份工作还能支付足够租金，在洛杉矶Van Nuys社区的一间工作室公寓里生活了一年半。

她被裁员后，Peterson表示她立即开始担心如何支付1月的房租，并开始寻找可能的帮助途径。她说，在她儿子还是婴儿时，他们住在一辆旅行拖车里，这是她不想再重复的情况。

“我开始联系当地的教堂或声称提供租金援助的地方，”Peterson告诉CNBC。“但许多人要求我必须有积极的驱逐通知才能给我援助。我觉得我已经没有更多选择了。我几乎和所有我能想到的人联系过，但都没有运气。”

Peterson没有收到驱逐通知，反而收到了洛杉矶县卫生服务部门内的无家可归预防单位的一封信，提供了一线希望。这项试点计划使用预测人工智能识别潜在无家可归的个人和家庭，提供援助帮助他们稳定并保持住所。

根据住房和城市发展部的数据，2023年加州有超过181,000名无家可归个体，比2007年增长超过30%。加州州审计员办公室的一份报告发现，2018年至2023年，该州在无家可归问题上支出了240亿美元。

该计划于2021年启动，帮助部门服务了近800名潜在无家可归的个人和家庭，根据洛杉矶县卫生服务部门无家可归预防部门副主任Dana Vanderford的说法，86%的参与者离开时保留了永久住房。

她说，个人和家庭可以获得4000到8000美元的资助，其中大部分资金来自美国救援计划法案。追踪到需要帮助的个人并说服他们这一提议是真实的而非骗局可能是一个挑战，但一旦建立联系，援助将迅速启动。

"我们经常在客户失去住房或者发生医疗紧急情况后的几天内见到他们。我们与人们见面的时机感觉非常关键，" Vanderford 说道。"我们能够从无处可寻出现、冷调一个人、提供资源，并预防我们与之合作的人中 86% 即将失去住房的情况，这种成就感非常了不起。"

Peterson 说她和儿子收到了大约$8,000 来支付房租、水电费和基本生活需要，这让她能够继续留在公寓里找新工作。该项目与客户合作四个月，然后在六个月、十二个月以及出院后的十八个月进行跟进。像 Amber Lung 这样的案件工作者表示，他们能够亲眼看到预防工作的重要性。

"一旦人们失去住房，感觉好像有很多障碍要重新找回住房，所以如果我们可以填补一点帮助他们保持住房的空白，我认为稳定情况要比他们最终进入庇护所或者街头更容易。" Lung 说道。

## 预测风险

这款 AI 模型是由加州大学洛杉矶分校的加州政策实验室在数年内开发的，使用了洛杉矶县首席信息官办公室提供的数据。按照加密的方式进行隐私保护，数据来自包括急诊室访问、行为健康护理和从食品券到收入支持及无家可归服务等大型公共福利项目在内的七个不同的县部门，据加州政策实验室的执行主任 Janey Rountree 介绍。该程序还从刑事司法系统中提取数据。

这些数据在多年的链接后，会被用来预测谁会经历无家可归。

一旦模型识别出谁会经历无家可归的模式，实验室就会用它来尝试预测未来，并创建一个按最高风险到最低风险排名的匿名人员列表。实验室向县提供了这个列表，以便能够在事情发生之前联系可能面临失去住房风险的人。

然而，[过去的研究](https://crypto.stanford.edu/~pgolle/papers/census.pdf) 发现匿名数据可以根据人口统计信息追溯到个人身上。基于1990年美国人口普查数据的一项广泛研究发现，87% 的美国人可以通过邮政编码、出生日期和性别来识别。

"加州存在着长达数十年的深层次住房短缺问题，住房成本不断上涨，这是我们人们经历无家可归的根本原因，" Rountree 表示。"最大的误解是认为无家可归是由个人风险因素引起的，而事实上很明显，根本原因是结构性经济问题。"

Rountree表示，政策实验室免费提供了软件给该县，并且没有计划对其进行货币化。她说，与具有相关专业知识的人（从教师到社会工作者）密切合作，可以帮助促进积极的社会结果。

“我只想强调，每个经历无家可归的社区都很重要，需要测试和创新预防措施。” 她说。“在无家可归服务的生命周期内，这是一个相对新的策略。我们需要更多的证据。我们需要在如何找到处于风险中的人方面做更多的实验。我认为这只是其中一种方法。”

全国终结无家可归联盟在2017年发现，慢性无家可归者每年平均为纳税人造成35578美元的费用，当他们被安置在支持性住房中时，这些成本平均减少了将近一半。

洛杉矶县已经与圣克拉拉县就该计划进行了初步对话，而圣地亚哥县也在探索类似的方法，Vanderford说。

## 政府对人工智能的使用

政府机构使用AI技术受到审查，因为可能带来的后果。警方对AI技术的依赖导致了[错误逮捕](https://apnews.com/article/artificial-intelligence-algorithm-technology-police-crime-7e3345485aa668c97606d4b54f9b6220)，而在加利福尼亚州，选民们在2020年拒绝了废除州内保释系统并用算法取而代之以确定个人风险的计划，因为他们担心这会在司法系统中[增加偏见](https://www.nbcnews.com/news/us-news/california-may-replace-cash-bail-algorithms-some-worry-will-be-n1243750)。

AI初创公司Hugging Face的首席伦理科学家玛格丽特·米切尔广泛表示，围绕政府使用AI的伦理关注点取决于使用情境和可识别信息的安全性，即使是匿名化的情况下也是如此。米切尔还指出，从寻求政府帮助的人们那里获得知情同意的重要性。

米切尔说：“人们是否意识到正在收集的所有信号以及它们可能被关联到他们身上的风险，以及对恶意使用的双重担忧？” “还有数据保留多长时间以及最终可能看到它的人。”

尽管这项技术旨在在洛杉矶县在房屋丧失之前为有需要的人提供援助，米切尔称从“德行伦理”角度来看这是一件积极的事情，但从功利主义的角度来看，还存在更广泛的问题。

“这些可能是像‘对纳税人的成本是多少，以及这个系统实际上避免房屋无家可归的可能性有多大？’这样的担忧。” 她说。

至于彼得森，她正在寻找工作，并希望能找到一个允许她灵活工作的远程职位。她希望最终获得执业护理证书，并且有朝一日能购买一套房子，让她的儿子有自己的房间。

"这意味着很多，因为你知道，我的儿子并不总是有稳定的生活。我也没有一直有这样的稳定，"她谈到了来自该项目的援助。"能够称这里为家，知道我不会明天就搬走，我儿子也不必马上找新朋友... 对我和我儿子来说都意味着很多。"
