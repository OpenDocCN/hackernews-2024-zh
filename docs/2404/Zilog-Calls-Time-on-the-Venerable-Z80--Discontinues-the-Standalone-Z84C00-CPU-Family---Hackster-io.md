# Zilog 宣布停产备受推崇的 Z80，终止独立的 Z84C00 CPU 家族 - Hackster.io

> 来源：[https://www.hackster.io/news/zilog-calls-time-on-the-venerable-z80-discontinues-the-standalone-z84c00-cpu-family-723594464754](https://www.hackster.io/news/zilog-calls-time-on-the-venerable-z80-discontinues-the-standalone-z84c00-cpu-family-723594464754)

Zilog 已宣布停止生产独立封装的 DIP Z80 CPU 型号 — 这一举动可能给那些希望构建基于 Z80 的系统或修复现有机器的复古计算机爱好者带来麻烦。

Zilog 在 1976 年推出了首款产品八位 Z80 CPU，由前英特尔工程师和公司联合创始人费德里科·法吉尼设计，以及早期雇员岛正俊设计。旨在改进英特尔的 8080 并专注于嵌入式计算，Z80 迅速成为微型计算机中最流行的 CPU 之一 — 与一年前推出的 MOS Technology 的竞争对手 6502 并驾齐驱，并出现在包括辛克莱 ZX Spectrum、Tandy TRS-80 和南梦宫的 Pac-Man 游戏机柜中。

尽管 Z80 CPU 属于早期设计，但仍然是现代嵌入式系统和怀旧时代爱好者设备的热门选择 — 但是 Zilog 终于决定结束这条产品线，尽管曾经一度是最受欢迎的第二来源 CPU 设计之一。

自 1976 年以来持续生产的 Zilog Z80 CPU，至少在独立 CPU 形式上，终于走到了寿终正寝的时候 — （📷：Zilog/Electronics Magazine）

在本周由 Zilog 母公司 Littelfuse 发布的产品变更通知中，公司警告用户 "我们的晶圆代工厂将停止支持 Z80 产品和其他产品线" — 最后采购日期为 2024 年 6 月 14 日，之后将不再提供这些零件。

通知列出了 13 种 Z80 零件，均属于 Z84C00 家族，正式宣布终止生命周期 — 包括通常用于替换老化芯片或构建受其启发的现代 Z80 机器的零件。然而，停产似乎不包括 eZ80 嵌入式核心系列 — 只涵盖独立 CPU 部件。

*更新*：另一份产品变更通知确认，新的 Z180 核心系列也受到影响，最后下单日期为 2024 年 6 月 14 日。

受零件生命周期结束影响的用户可在 [产品通知 PDF](https://www.mouser.com/PCN/Littelfuse_PCN_Z84C00.pdf) 中找到联系邮箱；Zilog 警告说，“根据总体最后购买需求，可能会实施最小/最大订购数量。”
