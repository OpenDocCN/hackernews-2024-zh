# Icon Painter

> 来源：[https://www.iconpainter.io/](https://www.iconpainter.io/)

### Can I use Icon Painter on a Mac?

No, Icon Painter only works with Windows PC's. We would love to offer if for Apple Mac's but the OSX operating system does forbids accessing and changing application icons.

### Can I use Icon Painter on my phone or tablet?

Icon Painter is a desktop application for Windows PC's and does not work with phones. If you tablet runs Windows, you might be able to use Icon Painter on it. Please test compatibility with the Free version first.

### Why are the system icons not showing in Icon Painter?

The system icons Computer, Control Panel, Network and Recycle Bin cannot be changed programatically on Windows 11 and hence are excluded from Icon Painter. However, you can manually load and export them from Icon Painter. Thereafter, you can change them in the Windows Settings under the Personalization -> Themes -> Desktop Icon Settings tab.

### After restarting my computer all the icons are now blank. What can I do?

This will happen if you removed or changed files from the Icon Painter app data folder. To fix this, you can simply re-paint the icons in Icon Painter.

### After restarting my computer one of the icons is now blank. What can I do?

This can happen when you re-install or upgrade an application whose icon you previously painted. To fix this, you can simply re-paint the icons in Icon Painter.

### Why are documents and folders from my desktop not showing in Icon Painter?

Curently, only application shortcut icons can be painted with Icon Painter. However, we might add the painting of documents and folders on the desktop in a future version of Icon Painter.

### Does Icon Painter care about my privacy?

Yes, except for the taskbar and desktop icons, Icon Painter will not touch anything on your computer. In particular, Icon Painter will not collect any personal data.

### What is the difference between the Free and the Pro version of Icon Painter?

Both versions show a preview of icons that have been customized by moving the settings sliders. However, only the Pro version allows the resulting icons to be actually applied.

### Do I need an internet connection to use Icon Painter?

No, you do not need an active internet connection to use Icon Painter.

### What are the system requirements?

Your computer should be running Windows 11 or 10, though the latter is untested. It must have a 64-bit processor, which is standard in most modern PCs. Try installing and running the Free version before purchasing the Pro version.

### Do you offer refunds?

No, we do not offer refunds due the fact that you can try Icon Painter in the Free version before any purchase.
