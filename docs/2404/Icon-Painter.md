# Icon Painter

> 来源：[https://www.iconpainter.io/](https://www.iconpainter.io/)

### 我可以在 Mac 上使用 Icon Painter 吗？

不，Icon Painter 只适用于 Windows PC。我们很想为苹果 Mac 提供支持，但是OSX操作系统禁止访问和更改应用程序图标。

### 我可以在手机或平板电脑上使用 Icon Painter 吗？

Icon Painter 是一款适用于 Windows PC 的桌面应用程序，不支持手机使用。如果您的平板电脑运行 Windows，可能可以在其上使用 Icon Painter。请先使用免费版本测试兼容性。

### 为什么系统图标在 Icon Painter 中没有显示？

系统图标“计算机”、“控制面板”、“网络”和“回收站”在 Windows 11 上不能通过程序更改，因此在 Icon Painter 中被排除在外。但是，您可以手动从 Icon Painter 加载和导出它们。之后，您可以在 Windows 设置中的个性化 -> 主题 -> 桌面图标设置选项卡中更改它们。

### 重新启动计算机后所有图标都变成空白了。我该怎么办？

如果您从 Icon Painter 的应用数据文件夹中移除或更改文件，将会发生这种情况。要解决此问题，您可以简单地在 Icon Painter 中重新绘制图标。

### 重新启动计算机后，其中一个图标现在是空白的。我该怎么办？

当您重新安装或升级之前已经绘制过图标的应用程序时，可能会发生这种情况。要解决此问题，您可以简单地在 Icon Painter 中重新绘制图标。

### 为什么我的桌面上的文档和文件夹没有显示在 Icon Painter 中？

目前，只能使用 Icon Painter 绘制应用程序快捷图标。然而，我们可能会在将来的版本中添加绘制桌面上文档和文件夹的功能。

### Icon Painter 是否关心我的隐私？

是的，除了任务栏和桌面图标外，Icon Painter 不会触及您计算机上的任何内容。特别是，Icon Painter 不会收集任何个人数据。

### 免费版和专业版的 Icon Painter 有什么区别？

两个版本都可以通过移动设置滑块来预览已自定义的图标。然而，只有专业版允许实际应用生成的图标。

### 我需要互联网连接才能使用 Icon Painter 吗？

不，您不需要活动的互联网连接来使用 Icon Painter。

### 系统要求是什么？

您的计算机应运行 Windows 11 或 10，尽管后者未经测试。它必须配备64位处理器，这在大多数现代PC中是标准配置。在购买专业版之前，请尝试安装和运行免费版本。

### 是否提供退款？

不，由于您可以在购买之前在免费版本中尝试 Icon Painter，我们不提供退款。
