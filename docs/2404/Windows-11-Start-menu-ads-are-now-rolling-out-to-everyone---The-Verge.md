# Windows 11 的开始菜单广告现在正在向所有人推出 - The Verge

> 来源：[https://www.theverge.com/2024/4/24/24138949/microsoft-windows-11-start-menu-ads-recommendations-setting-disable](https://www.theverge.com/2024/4/24/24138949/microsoft-windows-11-start-menu-ads-recommendations-setting-disable)

Microsoft 正在开始为所有用户在 Windows 11 的开始菜单内部启用广告。经过 [本月早些时候与 Windows Insiders 的测试](/2024/4/12/24128640/microsoft-windows-11-start-menu-ads-app-recommendations) 后，Microsoft 开始向 Windows 11 用户分发本周的更新 [KB5036980](https://support.microsoft.com/en-us/topic/april-23-2024-kb5036980-os-builds-22621-3527-and-22631-3527-preview-5a0d6c49-e42e-4eb4-8541-33a7139281ed)，其中包括“来自 Microsoft Store 的应用的推荐”。

“在最新的公开发布的 Windows 11 更新说明中，Microsoft 表示：‘开始菜单的推荐部分将显示一些来自 Microsoft Store 的应用。’‘这些应用来自一小部分经过策划的开发者。’这些广告旨在帮助 Windows 11 用户发现更多应用，但主要会给 Microsoft 尝试诱使其构建更多 Windows 应用的开发者带来利益。”

Microsoft 这些广告仅在两周前开始测试，因此看到这个“功能”如此短的时间内从测试渠道进入发布阶段令人惊讶。[在最初的测试时](/2024/4/12/24128640/microsoft-windows-11-start-menu-ads-app-recommendations)，我提到如果反馈表明这些广告不受欢迎，Microsoft“可能会决定放弃这些广告”，但两周的反馈显然不足以确定这一点。

幸运的是，你可以禁用这些广告，或者正如 Microsoft 所称的“推荐”。如果你安装了最新的 KB5036980 更新，那么请进入“设置” > “个性化” > “开始”，然后关闭“显示提示、应用推广和更多推荐”的开关。虽然 KB5036980 目前是可选的，但 Microsoft 将在未来几周内推送给所有 Windows 11 设备。

Microsoft 在 Windows 11 的开始菜单启用广告的举措，跟随了在 [Windows 10 锁屏](http://www.theverge.com/2015/4/29/8514345/windows-spotlight-lock-screen-with-ads) 和 [开始菜单](http://www.theverge.com/2016/5/16/11682164/microsoft-windows-10-promoted-apps-start-menu-anniversary-update) 中类似的促销位。Microsoft 还在去年开始测试 [在 Windows 11 的文件资源管理器内部放置广告](/2022/3/15/22979251/microsoft-file-explorer-ads-windows-11-testing)，然后停止了这个实验，并称该测试“并非意在向外发布”。希望那个实验仍然保持实验的状态。
