# 我们能够击败巨人

> 来源：[https://www.thehandbasket.co/p/meta-the-handbasket-censorship-slay-giants](https://www.thehandbasket.co/p/meta-the-handbasket-censorship-slay-giants)

我不会说我和全球影响力最大的公司之一的 Meta 纠缠不清。但我认为可以公正地说，我采取了有针对性的行动，以证明他们审查工作的恶意。

你可能还记得，[我重新发布了一篇专栏](https://www.thehandbasket.co/p/kansas-reflector-meta-facebook-column-censored)，内容批评了 Facebook 在气候变化广告政策上的做法，结果链接在周四被跨Meta平台屏蔽了。

当我在 Threads（Meta版的 Twitter）上发布了重新发表的专栏链接后，它被下架并被标记为违反网络安全社区标准。随后不久，*所有*指向我的网站的链接在*所有*Meta平台上都被屏蔽，任何曾发布过我的文章的人都收到了警报，指出链接因同样的网络安全违规行为而被移除。前一天，*Reflector*也遭到了同样的对待，[如我所报道](https://www.thehandbasket.co/p/facebook-kansas-reflector-links-banned)。

在意识到所有指向 *The Handbasket* 的链接被屏蔽发布后的几分钟内，我发现我不能再向 Threads 发布内容了。尽管只是短短一个小时，这导致我无法参与直接影响我的工作和声誉的对话。作为一个整个成年生活都在利用社交媒体放大我的声音和工作的人来说，这感觉有些不寻常。有点像《黑镜》中的情节，这不是我喜欢的感觉。

但随着舆论对 Meta 的压力增加，我的链接于周五下午晚些时候恢复访问，发言人安迪·斯通终于在[推特](https://twitter.com/andymstone/status/1776366040844296268?utm_source=www.thehandbasket.co&utm_medium=referral&utm_campaign=we-can-slay-giants)上就此问题发表了声明。然而，这则声明充斥着很多形式虚无的措辞：

**由于安全错误，链接到** [@KansasReflector](https://twitter.com/KansasReflector?utm_source=www.thehandbasket.co&utm_medium=referral&utm_campaign=we-can-slay-giants) **一度被屏蔽。同样的安全错误导致了对** [@statesnewsroom](https://twitter.com/statesnewsroom?utm_source=www.thehandbasket.co&utm_medium=referral&utm_campaign=we-can-slay-giants) **和 The Handbasket/**[@marisakabas2](https://twitter.com/marisakabas2?utm_source=www.thehandbasket.co&utm_medium=referral&utm_campaign=we-can-slay-giants)** 的链接也被屏蔽。**

**现在错误应用的封锁已经从所有三个域名中解除，但我们的系统需要时间来完全恢复所有链接。**

**这无疑令人沮丧，我们对所有受到影响的人表示诚挚的道歉。我们将继续监控这一情况。**

安全错误是什么？我们不知道。导致链接被阻止的原因是什么？我们也不知道。这个道歉有多诚恳？答案是一点也不，因为尽管被CNN和其他人点名，我仍未亲自听到Meta的代表的回应，尽管我是他们的主要目标之一。

我可以容忍很多事情，但被当作傻瓜对待不是其中之一。发生了什么事情，没有人愿意说出真相。由于Meta的傲慢，一个地方新闻媒体的专栏现在仅在[The Handbasket](https://www.thehandbasket.co/p/kansas-reflector-meta-facebook-column-censored)上就被观看了将近14,000次。

在一次[星期五电话](https://kansasreflector.com/2024/04/05/facebook-apologizes-for-blocking-kansas-reflector-then-expands-crackdown-to-other-news-sites/?utm_source=www.thehandbasket.co&utm_medium=referral&utm_campaign=we-can-slay-giants)中，与《反射镜》的总编辑谢尔曼·史密斯通话时，斯通“没有详细说明错误发生的原因，并表示不会再有进一步的解释。” 史密斯告诉我，当他问斯通公司是否计划向所有被告知我们的链接被标记为恶意的Facebook用户发布后续警报时，斯通说没有。但他确实鼓励史密斯写下这件事。

尽管早期的链接已经恢复，对我、史密斯和《反射镜》团队以及[States Newsroom](https://statesnewsroom.com/?utm_source=www.thehandbasket.co&utm_medium=referral&utm_campaign=we-can-slay-giants)来说，遭受的损害已成定局。我们收到了许多担心我们内容安全的电子邮件。在人们本已对新闻缺乏信任的时刻，我们的信任受到了损害。

我从这次经历中得出了一些结论：

**1\. 现在拥有多样化的社交媒体平台变得更加重要。** 随着Meta掌控三大平台的关键，并且有[一个白人至上主义疯子](https://www.thehandbasket.co/p/elon-musk-crosses-the-border-to-nazi)掌舵另一个平台，投资于提供从薄皮亿万富翁幻想中解脱的新兴平台至关重要。

如果不是因为[Bluesky](https://bsky.app/?utm_source=www.thehandbasket.co&utm_medium=referral&utm_campaign=we-can-slay-giants)，我的故事不会像现在这样火起来。相比前辈，这是一个微小的社交平台，但它却强大。正是因为该社区高度重视自由新闻，他们扩大了对Meta封锁的报道，甚至引起了国家级新闻的关注。他们给我发来了在他们尝试将我的作品发布到Meta平台时发生的情况截图，并在整个持续发生的令人困惑的局势中保持参与。

我不仅感到Bluesky网络的支持，还因一个新的媒体生态系统从Twitter的废墟中崛起而感到振奋。

**2\. 独立而非营利的媒体机构从未如此重要。** *Reflector* 能够发布批评Facebook广告政策的专栏，因为他们不受企业霸主的控制。他们无畏地报道堪萨斯州的地方政府和政治日复一日，是当地新闻机构被大公司收购和腐化之际的不可或缺的资源。正如去年夏天我们在[Marion County Record](https://www.thehandbasket.co/p/a-conversation-with-the-newspaper)上看到的那样，科技公司并非是唯一意图使地方媒体沉默的势力。

作为一名独立记者，我感觉比以往任何时候都更加自由。在我洗澡的时候（在纽约地区的异常地震期间），我想到了转载*Reflector*的专栏，因为我只需对我的订阅者负责，所以我能够直接去做。没有繁文缛节——只有我自己的编辑能力的极限。

尽管冒险将自己置于危险之中有些冒险和紧张，但最终感觉是正确的。无论是[Substack的创始人](https://www.thehandbasket.co/p/we-got-substack-to-admit-its-nazi)还是[Ronna McDaniel](https://www.thehandbasket.co/p/ronna-mcdaniel-msnbc-marisa-kabas)，我已经习惯了采取大动作。这种类型的新闻使我孤立，并经常使我与那些对此感到不适的人疏远，但我意识到在过去十年中建立了自己的平台之后，我一定会好好利用它。

**3\. 我们能够击败巨人。** 我知道所谓的人工智能威胁、Google能力的减弱以及亚马逊在我们生活的各个方面加速渗透，这使得情况看起来已经太晚了。但事实并非如此。

几天前，大多数人还没有听说过*The Handbasket*；很多人甚至都不知道*Kansas Reflector*。但是现在，我们正面对着Meta。有权势的人在秘密会议中谈论我们。

如果我们无名无姓，毫无意义，如果我们对他们的霸权构不成任何威胁，他们根本不会感到有必要与我们对话。我只是一个坐在布鲁克林家里用笔记本电脑的女人，而他们是一个拥有数十亿美元资产的国际公司，拥有各种权力的杠杆。然而，我们在这里，对峙着。

这场争斗会导致 Meta 解体吗？远非如此。但是在科技巨头中普遍存在这样的心态，认为如果他们继续削弱新闻的权力和声望，并与大型媒体建立起依赖关系，他们就可以免受批评，并摆脱对做得更好的责任。然而，我们继续看到，情况并非如此。
