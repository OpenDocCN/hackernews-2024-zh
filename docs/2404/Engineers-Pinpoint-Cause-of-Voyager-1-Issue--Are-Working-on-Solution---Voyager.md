# 工程师们准确找到了旅行者1号问题的原因，并正在解决方案中努力 - 旅行者

> 来源：[https://blogs.nasa.gov/voyager/2024/04/04/engineers-pinpoint-cause-of-voyager-1-issue-are-working-on-solution/](https://blogs.nasa.gov/voyager/2024/04/04/engineers-pinpoint-cause-of-voyager-1-issue-are-working-on-solution/)

NASA的工程师们确认，自去年11月以来，NASA的旅行者1号航天器由于一台计算机上的少量损坏内存导致向地球发送不可读的科学和工程数据。该计算机称为飞行数据子系统（FDS），负责在遥测调制单元（TMU）和无线电发射机发送数据到地球之前打包探测器的科学和工程数据。

[三月初](https://blogs.nasa.gov/voyager/2024/03/15/nasa-engineers-make-progress-toward-understanding-voyager-1-issue/)，团队发布了一个“poke”命令，促使航天器发送FDS内存的读数，其中包括计算机软件代码以及变量（根据命令或航天器状态变化的代码中使用的值）。使用这些读数，团队确认FDS内存约3%已损坏，导致计算机无法执行正常操作。

团队怀疑，一个负责存储FDS内受影响部分的单个芯片不起作用。工程师无法确定导致问题的确切原因。两种可能性是，该芯片可能被太空中的高能粒子击中，或者在46年后可能只是磨损了。

尽管可能需要几周或几个月的时间，但工程师们乐观地认为他们可以找到一种方法，使FDS在没有不可用内存硬件的情况下正常运行，从而使旅行者1号重新开始返回科学和工程数据。

[1977年发射](https://voyager.jpl.nasa.gov/mission/)的双子航天器飞越了土星和木星，而航天器2号则飞越了天王星和海王星。它们目前正探索太阳系外的星际空间，即太阳创建的粒子和磁场泡泡外的空间，称为太阳圈。航天器2号继续正常运行。
