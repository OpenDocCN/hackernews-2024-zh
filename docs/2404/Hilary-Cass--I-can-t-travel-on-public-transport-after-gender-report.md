# 希拉里·卡斯（Hilary Cass）：发布性别报告后，我不能乘坐公共交通工具了

> 来源：[https://www.thetimes.co.uk/article/hilary-cass-i-cant-travel-on-public-transport-any-more-35pt0mvnh](https://www.thetimes.co.uk/article/hilary-cass-i-cant-travel-on-public-transport-any-more-35pt0mvnh)

在接受《泰晤士报》采访时，上周发布关于治疗跨性别儿童的里程碑审查的儿科医生表示，虚假信息的传播正在使年轻人处于“风险”之中。

发布她的388页报告后，包括工党议员黎明巴特勒在内的人士重复声称，卡斯在报告中没有包括100项跨性别研究。

卡斯称呼这一说法“完全错误”，并表示人们通过传播“彻头彻尾的虚假信息”来破坏她的报告是“不可饶恕”的。

医生66岁，曾谈及围绕此问题的激烈辩论，还透露她收到了“恶毒”的辱骂邮件，并得到了安全建议以保护自己的安全。

对于她的批评者，卡斯说：“我对批评感到非常沮丧，因为这是彻头彻尾的虚假信息。完全不准确。

“这一切始于报告公布前一天，一个影响力者张贴了一张论文清单的图片，声称这些论文因未进行随机对照试验而被拒绝。

黎明巴特勒（Dawn Butler）在议会上声称，审查未包括100篇论文，卡斯称这一说法“完全错误”。

JANE BARLOW/PA

“那份清单与我们的报告或任何论文完全无关。

“如果你故意试图破坏一份审视儿童医疗证据的报告，那是不可饶恕的。你这样做会让儿童处于风险之中。”

在卡斯审查发布后的几天里，激进分子在社交媒体上声称，报告中只包括了100篇研究中的两篇。

巴特勒告诉下议院：“大约有100项研究未包括在这份卡斯报告中，我们需要知道为什么。”

卡斯解释说，研究人员评估了每一篇论文，但只从高质量和中等质量的论文中提取了结果，共60篇，这是103篇论文中的一部分。

对于巴特勒，她补充道：“你不会在议会上散布误导性信息的意图……但我感到沮丧的是，她对报告的理解。”

卡斯（Cass）的NHS审查发现，一个旨在让儿童改变性别的医学领域建立在不牢固的基础之上。她发现没有充分的证据支持全球临床实践，在18岁以下的人群中处方荷尔蒙以暂停青春期或过渡到异性别。

Tavistock和Portman NHS基金会信托的性别认同发展服务（Gids）在她的临时报告中被要求关闭，因为发现它“不是一个安全或可行的长期选择”。

卡西的临时报告公布后，塔维斯托克中心的GIDS诊所被命令关闭。

亨利·尼科尔斯/法新社

她的最终报告——全球最大规模的关于跨性别医疗争议领域的审查，涉及跨性别患者、家庭、学术界和医生。

Cass说，她很高兴看到，在对待儿童性别症的治疗问题上，报告大部分时间内双方并没有“武器化”她的报告。但她仍然不得不面对一些人，尤其是那些活动人士团体中的人，给出的“相当激进”的回应。她也远离了Twitter/X。

Cass说：“目前有一些非常恶毒的电子邮件进来。其中大部分被我的团队挡在外面，所以我没有看到它们。”她说，其中一些邮件包含了“我不会登报纸上的词语”。

她补充道：“令我沮丧的是辩论变得多么幼稚。如果我不同意某人，那么我就会被称为反跨性别或者TERF（排斥跨性别的激进女权主义者）。”

Cass表示，每次审查发表了一些“人们不喜欢的观点”，虐待行为就会激增。

她说，在2022年的临时报告后，在线讨论变得更加激烈，尤其是在利兹·特拉斯被选为保守党领袖和首相之后，“这就是辩论变得更加激烈、人们更加陷入壕沟的时候，然后网络上的愤怒升级”。

尽管被卷入文化战争的中心，但前英国皇家儿科学院主席卡西仍然坚定不移。

当被问及虐待是否对她产生影响时，她说：“不会……这是私人问题，但这些人不了解我。”

“我对所有这些错误信息感到非常非常的沮丧和恼火，比起虐待，这才是让我愤怒的事情。”

她补充道：“目前我不乘坐公共交通工具，遵循安全建议，这有些不便。”

上周，有消息称NHS成人性别诊所最终同意分享有关9000名在塔维斯托克诊所治疗过的儿童的后续数据。

• [**“秘密”性别诊所因青春期阻滞剂数据而让步**](https://www.thetimes.co.uk/article/secretive-gender-clinics-back-down-over-puberty-blocker-data-vvmxp65c2)

Cass透露，六家诊所通过拒绝合作阻挠了她关于给青春期阻滞剂和性激素处方长期影响研究的努力。她称，他们不分享数据是“协调的”和“意识形态驱动的”。

她告诉《泰晤士报》，在审查期间，她与诊所进行了一次“非常困难”的会议。她声称，他们指责审查团队浪费了他们的“宝贵时间”。

她补充道：“当我们接近时，他们对我们并不友好。”

Cass还透露，塔维斯托克诊所拒绝合作，未交出被精神科医生检查过的去转换者的数据。

审查团队希望评估患者历史中可能与去性别化相关的风险因素。一位进行了对Gids患者信息审计的顾问同意将其提供给Cass团队。

但Cass说：“我们要求Tavistock进行审查，但他们不肯给我们。这令人非常失望。”

在她的报告发布后，平等事务部长凯米·巴德诺克告诉《星期日泰晤士报》，机构（包括英国国民保健服务NHS）已经被意识形态控制，如果要全面实施Cass的建议，需要进行纠正。

Cass昨天回应称，显然可能存在关于“几乎任何事情，特别是关于这件事”的意识形态控制。

关于Tavistock，她告诉泰晤士报称，她已经得出结论：“在工作人员中肯定有一两个人…我会称他们为活动分子。”

然而，她补充道：“我认为大多数工作人员相信他们所做的是正确的。”

针对性别不安的儿童治疗长时间等待名单的问题，最近出现了几家私人诊所来填补这一空白。Cass表示担忧，并声称年轻人将无法得到NHS诊所提供的全面关怀水平。

海伦·韦伯利博士的公司GenderGP继续提供青春期阻滞注射。

ADRIAN SHERRATT 为泰晤士报报道

她单独指出了GenderGP，这是由英国医生海伦·韦伯利创立的位于新加坡的公司，继续广告宣传青春期阻滞注射。

• [**可能会被取消注册的给孩子使用青春期阻滞剂的私人医生**](https://www.thetimes.co.uk/article/private-doctors-who-give-children-puberty-blockers-may-be-struck-off-nf97p73l0)

Cass声称Webberley提供的关怀“显然远远不及任何一个正常评估和探索认可的标准”。

在她的报告发布后，她保持低调，但坚决表示她希望协助实施她审查的32条建议。

与此同时，英格兰国民保健服务NHS宣布将进行类似Cass风格的成人性别诊所审查。

“你在这里听到了：我不会做成人性别诊所审查。”Cass说。
