# 作为服务的监控：以色列“国防”技术对隐私和人权的全球影响 | Tor项目

> 来源：[https://blog.torproject.org/surveillance-as-a-service-global-impact-of-israeli-defense-technologies-on-privacy-human-rights/](https://blog.torproject.org/surveillance-as-a-service-global-impact-of-israeli-defense-technologies-on-privacy-human-rights/)

强调、揭示并积极抵制监控技术的传播和正常化对于保护全球人权至关重要。在Tor项目中，我们知道，通过集体意识和行动，我们可以共同构建和贡献于旨在保护人们免受监控和压迫普遍影响的隐私技术。同样重要的是，要追究公司的责任，并认识监控技术的来源和促使者，特别是在我们看到这些技术在加沙持续种族灭绝中被积极利用的现在。

本文探讨了以色列监控技术在巴勒斯坦的影响，展示了其使用的地方性实例如何产生广泛的影响，从而为此类压制性实践的普遍接受和全球采纳铺平了道路。

在抵制技术压制方面，全球立场的需求正在增长。技术工作者和广大国际社会被敦促将诚信置于利润之上，以保护隐私并防止普遍监控对我们生活的有害影响。面对由AI增强和恶化的监控解决方案的不断增长的需求，解决这些问题变得更加紧迫。

## 全球监控与压迫

下面的例子展示了监控资本主义的惊人复杂性及其日益扩展的全球足迹。它们还探讨了以色列间谍软件和监控公司如何通过重新品牌化并在全球设立办事处来应对全球审查，同时一系列风险投资公司促进它们的运作并帮助它们避免急需的问责。

+   Elbit Systems是以色列的[领先军事技术出口商](https://www.aljazeera.com/features/2023/11/17/israels-weapons-industry-is-the-gaza-war-its-latest-test-lab)，专门从事各种先进的监控技术，在太空、空中、海上和地面行动中可怕地部署。其监控系统、无人机和其他高科技工具被广泛用于实施暴力占领和种族隔离，在全球需求旺盛。

+   这种普遍的监控技术并不止于巴勒斯坦，但通常是[从这里开始的。](https://theintercept.com/2019/08/25/border-patrol-israel-elbit-surveillance/)

+   自至少2014年以来，Elbit Systems在[美国南部边境](https://visualizingpalestine.org/visuals/watched)以及[阿联酋](https://elbitsystems.com/pr-new/elbit-systems-establishes-a-company-in-the-united-arab-emirates/)和[英国](https://www.aljazeera.com/economy/2019/9/11/meeting-the-arms-dealers-with-killer-sales-targets)部署这些工具。其中一些交易通过Elbit Systems的美国子公司Kollsman Inc进行，后者参与了一名员工在前往沙特阿拉伯出差期间[引发的争议死亡事件](https://www.wickedlocal.com/story/hudson-sun/2015/03/07/autopsy-on-contractor-suggests-foul/35046157007/)。

+   这些关系通过风险投资进一步加强。阿联酋至少投资了[1亿美元](https://www.wsj.com/articles/u-a-e-sovereign-wealth-fund-invests-100-million-in-israel-venture-capital-firms-11642164356)在以色列的风险投资公司，考虑到它是[长期使用其间谍软件工具](https://web.archive.org/web/20240131160116/https://www.washingtonpost.com/nation/interactive/2021/hanan-elatr-phone-pegasus/)的顾客，这并不令人意外。例如，以色列公司负责[阿布扎比的大规模监控系统](https://www.middleeasteye.net/news/abu-dhabi-announces-launch-israeli-installed-mass-surveillance-system)，合同金额为6亿美元。迪拜还通过名为[黑墙环球](https://circuit.news/2022/09/04/uae-israel-cyber-intelligence-firm-grows-with-its-perch-in-the-gulf/)的当地实体，由以色列情报退伍军人领导，为希望部署以色列监控技术的阿拉伯国家提供后渠道。

+   在类似结构的交易中，以色列公司如IntuView与沙特阿拉伯签有合同，包括扫描沙特公民的数据。其[顾问委员会](https://web.archive.org/web/20240102123821/https://www.intuview.com/team)由前摩萨德局长和前中央情报局局长组成。

+   这项技术正变得越来越先进，并受到威权政府的青睐。在美国公司的协助下，以色列正在[部署配备多个传感器和摄像头的机器人](https://www.haaretz.com/israel-news/2024-03-03/ty-article-magazine/.premium/gaza-becomes-israels-testing-ground-for-remote-control-military-robots/0000018e-03ed-def2-a98e-cfff1e640000)，用于监视建筑物和开放空间。

+   AI-based systems like "[福音](https://www.theguardian.com/world/2023/dec/01/the-gospel-how-israel-uses-ai-to-select-bombing-targets)" 主要由无人机录像、拦截通信、监视数据以及通过监测个人和大群体的移动和行为模式获取的信息组成，用于制定轰炸目标。

+   以类似方式，通过“[Lavender](https://www.972mag.com/lavender-ai-israeli-army-gaza/)”，以色列正在利用人工智能通过对加沙居民进行大规模监视收集信息来检测暗杀目标，几乎没有人为监督。报告强调了许多这些攻击导致大规模屠杀平民和整个家庭，通常是在他们自己的家中。

+   间谍软件一直是以色列一个非常赚钱且日益显著的业务。[公民实验室](https://citizenlab.ca/about/)多年来揭露了以色列的NSO等公司的复杂间谍软件，这些公司与长期监禁、谋杀和沉默异见者以及监视民间社会组织的政府有合同，包括沙特阿拉伯、巴林和阿联酋。这些政府主要是为了[便利获取](https://gulfstateanalytics.com/pegasus-as-a-case-study-of-evolving-ties-between-the-united-arab-emirates-and-israel/)这类技术而与以色列正常化关系，然后将其部署以压制自己的公民。

+   以色列的[Voyager Labs](https://www.voyager-labs.com/)积极推广和销售监视工具，这些工具已被用来对记者和活动人士进行个人资料和恐吓，包括[哥伦比亚军事情报](https://www.haaretz.com/israel-news/security-aviation/2023-02-28/ty-article/fake-friends-leak-reveals-israeli-firms-turning-social-media-into-spy-tech/00000186-7f75-d079-ade7-ff7507970000)。他们最近将[总部迁至美国](https://www.prnewswire.com/il/news-releases/voyager-labs-announces-relocation-to-us-to-support-accelerated-growth-301653722.html)，以继续与一支“世界级人工智能研究团队”共同开发这类工具。

+   美以联合企业[Verint](https://www.verint.com/our-company/)，其产品包括监视摄像头和分析软件，用于监视和分析大量语音和视频数据集，曾将他们的技术销售给阿塞拜疆、印尼、南苏丹、乌兹别克斯坦和哈萨克斯坦的专制政权。他们的工具之一的[用途](https://www.haaretz.com/israel-news/2018-10-20/ty-article-magazine/.premium/israels-cyber-spy-industry-aids-dictators-hunt-dissidents-and-gays/0000017f-e9a9-dc91-a17f-fdadde240000)包括对LGBTQ+社区和人权活动人士的暴力镇压。

+   这些公司并不是孤立操作。面对审查或曝光时，它们会分拆出新的实体，通常由同样参与的风险投资公司支持。例如，Verint将一个名为Cognyte的独立公司[分拆](https://www.verint.com/press-room/2021-press-releases/verint-celebrates-day-one-as-a-company-focused-on-enabling-brands-to-achieve-boundless-customer-engagement-following-completion-of-cognyte-software-spin-off/)，以继续其利润丰厚的交易。一位以色列人权律师[说](https://www.aljazeera.com/news/2023/1/15/israels-cognyte-won-myanmar-spyware-tender-before-coup)这些行为“协助和煽动反人类罪行”。纽约的对冲基金Edenbrook Capital是其最大的[股东之一](https://finance.yahoo.com/quote/CGNT/holders/)。

+   另一个更近期的例子是来自前Pegasus软件CEO的[Dream Security](https://theintercept.com/2024/01/18/israel-nso-group-shalev-hulio-dream-security/)。他们最新一轮融资由洛杉矶风险投资公司[Group 11](https://group11.vc/)共同领导。这显示了一个令人不安的逃避循环。

+   Corsight，一家以色列[公司](https://www.corsight.ai/company/)，利用面部识别和Google照片[进行大规模监视](https://web.archive.org/web/20240404075749/https://www.nytimes.com/2024/03/27/technology/israel-facial-recognition-gaza.html)，监视巴勒斯坦人民，而他们并不知情或同意。到目前为止，Google已经[拒绝](https://theintercept.com/2024/04/05/google-photos-israel-gaza-facial-recognition/)回答，尽管禁止他们的技术用于“立即危害”之中。除了严重侵犯隐私外，这项技术还[误认人员](https://www.theverge.com/2024/3/27/24114043/israel-facial-recognition-gaza-strip-corsight)，使他们的生命处于危险之中。加拿大-以色列风险投资公司[AWZ ventures](https://www.awzventures.com/our-portfolio)是其主要投资者之一。他们的工具还被美国国土安全部[用于面部识别](https://www.prnewswire.com/news-releases/corsight-ai-receives-top-rankings-in-2020-biometric-technology-rally-conducted-by-the-department-of-homeland-security-301225577.html)。

+   以色列公司AnyVision在国际范围内拥有存在，其背后是用于监控[巴勒斯坦人](https://www.nbcnews.com/news/all/why-did-microsoft-fund-israeli-firm-surveils-west-bank-palestinians-n1072116)动态的"高级战术监视"软件。其CEO曾是[SoftBank投资部门的操作合伙人](https://theorg.com/org/anyvision/org-chart/avi-golan)，该部门与美国康涅狄格州总部的持股公司Eldridge一同为该公司进行了大额融资[（白案新闻发布）](https://www.whitecase.com/news/press-release/white-case-advises-softbank-and-eldridge-us235-million-funding-anyvision)。几个月后，他们在美国学校中的监视系统[被曝光](https://themarkup.org/privacy/2021/07/06/this-manual-for-a-popular-facial-recognition-tool-shows-just-how-much-the-software-tracks-people)，随后[更名为Oosto](https://oosto.com/press/anyvision-now-oosto/)并恢复了运营。

+   这些合作中最显著的例子之一是Palantir，继续赢得与[英国政府](https://www.nytimes.com/2023/11/21/business/palantir-nhs-uk-health-contract-thiel.html#:~:text=Palantir%2C%20the%20software%20company%20founded,secure%20such%20a%20sensitive%20role)和各种美国机构的重要合同，[最近与美国军方签署](https://www.businessinsider.in/stock-market/news/palantir-stock-spikes-11-as-the-ai-firm-inks-178-million-contract-with-the-us-army/articleshow/108276459.cms)一项AI驱动的目标定位系统合同，并将再次向以色列提供其监视产品，参与新的"[战略伙伴关系](https://origin.www.bloomberg.com/news/articles/2024-01-12/palantir-israel-agree-to-strategic-partnership-for-battle-tech)"。

+   全球市场对于"边境安全技术"（[https://theintercept.com/2024/03/12/dhs-border-towers-ai/](https://theintercept.com/2024/03/12/dhs-border-towers-ai/)）预计到2027年将超过700亿美元，从2022年的480亿美元增长而来。这一市场显著增长部分归因于采用以色列公司开发的AI集成监视塔技术，这些公司在[巴勒斯坦人](https://therealnews.com/the-weapons-israel-tests-on-palestinians-will-be-used-against-all-of-us)身上测试压制技术。这一做法使得以色列能够在真实场景中改进并展示其产品的效果，使其更具吸引力于国际买家，这些买家往往同样是有着可怕人权记录的压制政权。
