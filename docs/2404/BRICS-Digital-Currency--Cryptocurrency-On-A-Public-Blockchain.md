> 来源：[https://illya.sh/blog/posts/brics-cryptocurrency-blockchain/](https://illya.sh/blog/posts/brics-cryptocurrency-blockchain/)

随着我们将生活的各个方面进一步扩展到数字领域，价值交换系统也将如此。尽管大部分金融世界依赖技术运作，但其基础实施仍基于古老的物理货币系统。这也是为什么跨境和跨货币交易在2024年仍然是一个挑战的原因之一。更不用说银行仍然在关门后以批处理方式处理大部分交易。

大量资源被用于专用系统、专业硬件和专有协议，以实现如数字支付等简单操作。即使在您的地区这不是问题，全球范围内仍然存在问题。您可以通过在不同国家间旅行来验证这一点，而且您无需走得太远：只需尝试穿越欧盟（EU）成员国。目前存在的解决方案对许多企业来说仍然令人不满意，特别是中小企业（SMEs），因为涉及的费用问题。因此，它们通常更倾向于使用实物货币或区域特定的替代支付方式。尽管存在所有这些挑战，全球和互操作的支付系统是一种非常理想的工具。

## BRICS数字货币

BRICS数字货币可以利用区块链、智能合约和零知识证明（ZKP）来实现其目标，并在透明性、信任度、可升级性、互操作性和隐私保护方面提供巨大的价值主张。这种货币的主要目标是为BRICS国家提供统一的交换媒介，促进无缝的跨境交易和经济合作。虽然最初的用例将用于成员国之间的国家级贸易和交易，但合理地期望其在更大范围内扩展至个人和企业的普遍公众。

## 从成员经济体中获取价值

提议的BRICS货币的核心原则是，其价值来源于成员国和其本地货币的集体经济。这需要一个动态的BRICS货币供应，根据每个参与国家的经济表现进行调整。因此，**BRICS货币的经济价值是成员国经济价值的衍生物**。这种**经济价值**由一个公式定义，我们将在下一节中探讨。

考虑以下情景：如果某个BRICS成员国的经济价值增加，那么BRICS货币的供应量将成比例增加。相反，如果经济价值下降，供应量将减少。从本质上讲，BRICS货币可以被视为其成员国联合经济状况的衍生物。正如我之前在[DeFiDe, 我的ETHLisbon 2023黑客马拉松项目](https://github.com/iluxonchik/eth-lisbon-hackathon-23)中探讨的那样，区块链技术极大地简化了衍生品的创建。通过将BRICS货币的价值与成员国经济表现捆绑在一起，它成为了反映BRICS集团的集体经济实力的衍生工具。术语*bloc*指的是具有共同利益的国家组成的集团，他们形成联盟。因此，*BRICS集团*指的是为了共同利益而联合行动的BRICS成员国。

这种方法相比现有的法定货币带来了许多好处，其中最明显的好处之一是货币供应的去中心化，因为它由多个实体管理，而不是单一实体。这也允许在比现有法定货币更大的规模上对冲风险和减少波动性。一个国家经济的衰退可以被另一个国家的增长抵消。这与地方货币形成对比，其价值和供应通常由单一实体控制，无论是中央银行还是政府。

## BRICS货币供应公式

BRICS数字货币采用一种算法和动态供应机制，根据成员国经济表现进行调整。BRICS货币总供应量由以下公式表示，该公式考虑了各种经济因素：

`M = ForT + FDI + PI + ForEx + G&EX + InterSec + Der + ForDeposit + EuroCurrency + MTransfer + Etc.`

此公式中的每个组成部分代表关键的经济指标：

+   `ForT`: 外贸，包括BRICS国家与世界其他地区之间进出口总值。

+   `FDI`: 外国直接投资，捕捉投资资本的流入和流出。

+   `PI`: 投资组合投资，包括股票、债券和其他金融工具的投资。

+   `ForEx`: 外汇交易，代表货币交换的交易量。

+   `G&EX`: 黄金和外汇储备，部分支持BRICS货币价值的关键组成部分，其中黄金是具有内在价值的资产持有。

+   `InterSec`: 国际证券，例如BRICS国家在国际市场上发行的债券。

+   `Der`: 衍生品，从基础资产或基准衍生其价值的金融合同。

+   `ForDeposit`: 外国存款，指在外国账户中持有的BRICS货币总值。

+   `EuroCurrency`: 欧元市场运作，捕捉BRICS国家在其他货币市场上的活动。

+   `MTransfer`：国际货币转移，资金在金砖国家与其他国家之间的流动。

+   `Etc.`：根据需要的其他相关经济指标。文章并未详细展开，但你可以想象其中的一个因素是国家的比特币储备。

该公式旨在创建一个金砖国家货币供应，以动态响应成员国经济状态。当我们在文本中提到**经济价值**时，我们指的是上述公式定义的经济价值。我们将探讨几种情景，以说明这一机制在实践中的运作，但首先让我们放大视角，看看**价值**是什么以及如何定义**价值**的过程。

### 供应、需求、价值与橙子 🍊

在经济谈论和理论上，我们可以变得很复杂，但这并不会改变一个核心事实：某物品的价值是由**供应和需求的比例**定义的。假设你住在一个偏远的小镇，你是第一个种植橙子树的人。你的镇上的人们喜欢这种水果，而且树产生的橙子多于你自己消费的量，所以你决定出售你的橙子。橙子的供应是有限的，因为你的树每个季节只能生产有限数量的橙子。决定你橙子价值的是镇上对橙子的需求。如果镇上对橙子的需求增加，那么这种需求将推动人们愿意为你的橙子交换的价值上升。换句话说，他们愿意为你的橙子支付更多。因此，需求增加而供应不增加将导致橙子价值的增加。

现在，假设在看到你的橙子生意蓬勃发展后，你的邻居也决定种植一棵橙子树。现在，镇上有两个人拥有橙子树。橙子的供应增加了，但需求保持不变。即使之前的客户仍愿意以同样的价格购买橙子，这些客户的数量是有限的，而且他们将购买有限数量的橙子。有些人不愿意为橙子支付高于某一金额的价格。因此，这将使你和你的邻居处于这样一种境地：要么你不从剩余的橙子中获利，要么你降低橙子的价格，从而能够应对市场的新部门需求：只愿意以较低价格购买橙子的部分。因此，供应增加而需求不增加将导致橙子价值的下降。

然而，请考虑第三种情况，即橙子的供应和需求均成比例增加的情况。在这种情况下，橙子的价值将保持不变。BRICS 货币供应公式旨在保持货币的稳定或增值，并具有动态供应。通过根据成员国经济表现调整供应来实现这一目标，确保 BRICS 货币的价值反映 BRICS 集团的基础经济价值。

### BRICS 经济繁荣期间的供应

想象一下 BRICS 国家经历强劲经济增长期。随着出口增加，外贸激增，外国直接投资大量涌入，以及投资组合在 BRICS 金融市场上的飙升，外汇储备，包括其黄金持有量，显著扩展。在这种情况下，BRICS 货币供应公式将计算 M 的较高值，导致数字货币的流通供应增加。这种供应扩展旨在反映 BRICS 集团的经济价值增加，从而增加对 BRICS 货币的需求，因为该集团既以该货币运作，又定义其价值。

### BRICS 经济衰退期间的供应

现在考虑相反的情况，即 BRICS 经济面临挑战的情况。贸易量收缩，外国投资减少，资本外流增加。中央银行储备被动用以支持本地货币。在这种情况下，货币供应公式将导致 M 的较低值，触发 BRICS 数字货币供应的收缩。这种减少有助于抵消需求减少和经济活动的下降，从而减少或避免 BRICS 货币的贬值。

### 黄金支持的 BRICS 货币

如果您仔细观察上述公式，您会注意到**黄金**在 G&EX 组件中的包含。不像*法定货币*，其价值由意志决定（这是拉丁文中 *fiat* 的意思），**黄金**并非货币，而是货币，因为它具有内在价值。虽然称 BRICS 货币由黄金支持可能是不正确的，但称 BRICS 货币部分由黄金支持是正确的。因此，这使得提议的 BRICS 数字货币至少部分成为货币，正如 **货币** 这个词的定义。

黄金作为货币形式已有5000多年的历史，是人类历史上最悠久的货币形式。其内在价值、稀缺性和耐久性使其成为可靠的价值存储和交换媒介。当一种货币以黄金作为支撑时，意味着该货币的供应量直接反映了黄金的基础抵押物。实际上，这意味着该货币的一个单位可以兑换固定数量的黄金。这不仅为货币提供了额外的稳定性，还作为对抗通货膨胀的避险工具。货币的价值与黄金的价值挂钩，而黄金是一种具有内在价值的有形资产。

#### 以黄金支持的美元（USD）

金本位货币并非新概念。许多国家过去曾使用过金本位货币，包括美国。在金本位制度下，美元可以直接按固定比率兑换成黄金。这一制度在二战后的布雷顿森林体系协议中得到正式确认。

布雷顿森林体系协议建立了一个固定汇率体系，各国将其货币与美元挂钩，而美元则以每盎司35美元的比例与黄金挂钩。这意味着外国政府可以以固定汇率将其美元储备兑换成黄金。然而，1971年停止了美元直接可兑换成黄金，这实际上将美元转变为一种**法定货币**。

通过允许成员国使用其本地货币进行贸易，或者使用部分以该国本地货币价值为基础的货币进行贸易，而不是强制使用大部分由第三方控制的货币进行贸易，BRICS数字货币对新兴市场和发达市场都是一个吸引人的选择。除了使成员国至少部分使用其本地货币外，还使这些国家能够通过利用BRICS中的本地流动性池来增加其货币的价值。

这样的货币体系是朝向全球范围内去中心化迈出的一步。它允许BRICS成员国拥有一种不受单一实体控制而是由一组实体控制的货币。这是朝向更加去中心化金融系统的一步，其中货币的控制权分散在成员国之间。这与大多数现有金融基础设施形成鲜明对比，其中货币的控制权集中在单一实体（如中央银行或政府）手中。

## 如何在公共区块链上将BRICS货币实现为加密货币

在本节中，我们将探讨**如何在公共区块链上将BRICS数字货币实现为加密货币**。您可能已经注意到，本节内容较短，我邀请您将其与前文关于实现其作为加密货币的论述联系起来，说明相比于其他方法，这种实现更为简单、高效和不那么复杂。

货币本质的关键是其价值。价值由供需比例定义。供应由我们上面探讨的货币供应公式定义：`M = ForT + FDI + PI + ForEx + G&EX + InterSec + Der + ForDeposit + EuroCurrency + MTransfer + Etc.`。供应可以通过智能合约进行算法控制，而像zkSNARKs和zkSTARKs这样的ZKP协议可以用于可验证且私密地从任何数字源桥接数据。需求来自于参与BRICS经济网络所带来的激励。为了参与，人们通常必须遵守某些要求并按照网络的规则行事。所有这些要求都可以通过自执行智能合约实现。

如何在公共区块链上将BRICS数字货币实现为加密货币的解释已经在上述段落中概括说明。接下来，我们将更详细地探讨其实现方式。

### BRICS加密货币的动态供应机制

正如我们已经看到的，BRICS数字货币的供应遵循一个明确定义的公式。算法稳定币和自动做市商（AMM）采用类似的方法。您可能熟悉Uniswap V2的`x*y=k`公式，这种算法确定了池中资产的价格。这类去中心化货币在去中心化交易所（DEX）上的日交易量价值以数十亿美元计算。这些经过实证验证的机制同样可以用于BRICS数字货币。

在区块链上实现这种动态供应机制将涉及从每个成员国的可信数据源获取经济数据的智能合约。这些数据点可以涵盖文章提议的BRICS货币供应公式中概述的因素：

`M = ForT + FDI + PI + ForEx + G&EX + InterSec + Der + ForDeposit + EuroCurrency + MTransfer + Etc.`

公共区块链上的一组智能合约将负责控制BRICS货币的供应。零知识证明极大简化了整合过程，允许从各种数字来源（如API、数据库、预言机，甚至其他区块链）可验证地获取数据。

### BRICS货币，zkLocus和DeFi

之前在本文中我们已探讨了[zkLocus](https://zklocus.dev/)如何通过将地理位置数据与每笔交易关联，同时保护终端用户隐私，从而增强DeFi的能力。zkLocus有效地将地理位置转化为可编程的现实世界资产（RWA），并允许其在智能合约中使用。本节详细探讨了此集成及其如何与BRICS和ERC-20代币集成。

zkLocus 在 [Mina 协议区块链](https://minaprotocol.com) 上实现原生，同时兼容其他区块链，如以太坊、ICP 和 Cardano。Mina 的一个价值主张是您可以轻松地将其数据桥接到其他区块链上。zkLocus 地理位置证明是原始的 zkSNARK 证明，因此它们本身可以直接被其他区块链使用。因此，可以通过桥接 Mina 区块链的数据或直接在选择的区块链上验证 zkLocus 证明来集成 zkLocus。通过 Mina 进行集成的好处在于利用链上的协议功能，例如其本地代币 **$ZKL** 和赏金，这些赏金可以激励在链上提交 zkLocus 证明，从而外包链上的数据压缩和提交给第三方。这样做的巨大好处是通过让大型节点池执行所有计算，而无需管理复杂的基础设施。
