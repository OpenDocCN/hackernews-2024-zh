# 治疗蜘蛛恐惧的暴露疗法可以改善无关恐惧，研究发现

> 来源：[https://www.psypost.org/exposure-therapy-for-arachnophobia-can-benefit-unrelated-fears-study-finds/](https://www.psypost.org/exposure-therapy-for-arachnophobia-can-benefit-unrelated-fears-study-finds/)

研究人员发现，害怕某一物体的人往往也会害怕类似的事物，这一过程被称为恐惧泛化。暴露疗法是我们目前最有效的治疗方法，通过逐渐控制性地暴露人们害怕的事物，教导大脑认知，让害怕的事物不再构成威胁。这种疗法不仅能够减少对特定事物的恐惧，还可以减轻对其他类似事物的恐惧。例如，治疗对蜘蛛的恐惧也可能减轻对蟑螂的恐惧。

Iris Kodzaga及其同事研究了这一效果是否可以应用于完全不同的恐惧，比如治疗蜘蛛恐惧以帮助克服对高处的恐惧，以了解这种疗法的广泛性和有效性。这项研究发表在[*翻译精神病学*](https://doi.org/10.1038/s41398-023-02698-7)上。

电话筛选过程确定了18至40岁之间报告害怕蜘蛛和高处的个体，排除了急性心理状况或正在接受治疗的个体。这一过程将参与者池缩小到最终的样本群体，分为暴露组和对照组。

研究所使用的材料包括非毒性的家蜘蛛和一座高72米的教堂塔楼作为恐惧诱导刺激，以及各种诊断和评估工具，如Mini-DIPS、贝克抑郁量表-II和蜘蛛恐惧问卷。行为途径测试（BATs）评估了对蜘蛛和高处的恐惧和回避行为。

研究从初步筛选和招募开始，随后进行预评估阶段，参与者完成诊断性面谈和问卷调查，并进行蜘蛛和高处的BATs，以建立基础恐惧水平。暴露组然后根据Öst的协议接受治疗，涉及逐步接近蜘蛛的14个步骤，由主观痛苦单位量表监测。

对于这一组的后评估包括重复进行BATs和问卷调查，以评估恐惧和回避行为的变化。对照组经历了类似的过程，但没有暴露疗法，以评估仅时间和任务重复的影响。

暴露疗法显著减少了对蜘蛛恐惧的个体的恐惧和回避行为，在蜘蛛BATs期间观察到了主观恐惧水平和行为分数的改善。这些效果不仅限于治疗的蜘蛛恐惧，还泛化到未治疗的高处恐惧。

曝露组和对照组都表现出对高处增加接近行为，但曝露组显示出更为显著的恐惧减少，表现为主观恐惧评分和恐高问卷得分的降低。这表明曝露疗法不仅直接应对特定的恐惧，还在减少相关未处理恐惧方面产生更广泛的影响，展示了其在处理恐惧泛化方面的有效性。

未来的研究应当复制这些发现，并在各种焦虑障碍中进行长期跟踪，以评估这些泛化效应的持久性。

论文“[对未处理的刺激从另一个恐惧类别的有益暴露效应的泛化](https://doi.org/10.1038/s41398-023-02698-7)”的作者是Iris Kodzaga、Ekrem Dere和Armin Zlomuzica。
