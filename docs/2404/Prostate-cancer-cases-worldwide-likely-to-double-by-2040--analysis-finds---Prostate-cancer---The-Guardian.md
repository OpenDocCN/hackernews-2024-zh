# Prostate cancer cases worldwide likely to double by 2040, analysis finds | Prostate cancer | The Guardian

> 来源：[卫报](https://www.theguardian.com/society/2024/apr/04/prostate-cancer-cases-worldwide-diagnoses-deaths-2040-study)

-   到2040年，全球被诊断患有前列腺癌的男性人数预计将翻倍，达到每年290万例，而预计每年的死亡人数将增加85%。这是迄今为止规模最大的研究。

-   前列腺癌已经是导致死亡和残疾的主要原因，在100多个国家中是[最常见的男性癌症形式](https://www.theguardian.com/society/2024/feb/29/study-offers-hope-in-identifying-high-risk-prostate-cancer-patients)。但随着全球人口老龄化和预期寿命的延长，一项新的分析预测，在未来15年内，病例和死亡人数将大幅增加。

预计到2040年，诊断案例将从2020年的每年140万例增加到290万例，这意味着每小时将有大约330名男性被告知他们患有该疾病。

预计全球死亡人数在20年期间将增加85%，从2020年的37.5万例增加到2040年的近70万例。专家们表示，由于低收入和中等收入国家的低诊断率和数据缺失，真实的死亡人数可能更高。

这些发现发表在《柳叶刀》上，作为其前列腺癌重要委员会的一部分，并将在巴黎举行的欧洲泌尿学年会上周六展示。

人口老龄化和预期寿命增加意味着全球范围内老年男性的数量正在上升。由于前列腺癌的主要风险因素，如年龄超过50岁和家族病史，无法避免，专家们表示仅通过生活方式改变或公共卫生干预将无法防止病例的激增。

然而，根据这份长达40页的报告的作者们称，对该疾病症状的广泛认识、测试倡议的获取、早期诊断以及治疗技术的进展仍然可以帮助减轻负担并拯救生命。

“随着全球越来越多的男性进入中年和老年阶段，前列腺癌症病例数量不可避免地将上升”，研究的主要作者尼克·詹姆斯教授说道。“我们知道这种病例激增即将到来，因此我们需要立即开始规划并采取行动。”

“基于证据的干预措施，如改善早期检测和教育项目，将有助于拯救生命，并在未来几年内预防前列腺癌引发的疾病”，伦敦癌症研究所的前列腺癌研究教授兼皇家马斯登NHS基金会信托的顾问临床肿瘤学家詹姆斯补充道。

詹姆斯称，全球需要新的和改进的检测方法，以减少过度诊断和过度治疗的同时，早期检测可能致命的肿瘤。

报告还指出，增强男性及其家庭对寻找迹象的知识也是关键。

前列腺癌的症状可能包括需要更频繁地排尿，尤其是在夜间；需要赶紧去上厕所；开始排尿困难；感觉膀胱未完全排空；以及尿液或精液中出现血液。

这些症状并不总是意味着你患有前列腺癌。许多男性的前列腺随着年龄增长会变大，因为一种称为良性前列腺增生的情况。前列腺癌可能已经扩散的迹象包括睾丸、背部或骨疼痛，食欲减退和非故意体重减轻。

阿尔弗雷德·塞缪尔斯表示，在临床试验中记录种族应该成为强制性要求。照片：阿尔弗雷德·塞缪尔斯

这项研究还突显了需要更多研究来更好地了解黑人男性的前列腺癌问题，因为大多数研究集中在白人男性身上。

阿尔弗雷德·塞缪尔斯在2012年得知自己患有晚期前列腺癌时已经54岁。他曾在娱乐业工作了三十年，为包括碧昂丝和鲍勃·迪伦在内的名人提供安全保护，但是这一震惊的诊断一夜之间结束了他的职业生涯。当检查显示他的癌症已经扩散时，医生排除了手术的可能性，这位六个孩子的父亲开始失去希望。

然而，他随后加入了一项临床试验，并开始接受一种如今正在延长全球成千上万男性生命的药物治疗。十二年后，来自伦敦西北部哈罗的塞缪尔斯欢迎了六个孙辈的到来，并开始第二职业，增强人们对癌症研究的认识。

“由于我的前列腺癌被晚期诊断，如果我没有参加临床试验，今天我可能已经不在了。这是我的生命线，”现年66岁的塞缪尔斯说道。“这个报告出来已经有很长时间了。现在有必要强制在临床试验中记录种族，并且试验必须反映人口的种族多样性，这样我们才能找到更好的治疗方法，适合像我这样的人。”

爱米·赖兰斯，慈善组织前列腺癌英国部门的改进护理主管，称《柳叶刀》的报告是“一个及时的行动呼吁”。她补充说，医疗系统必须更好地识别那些处于最高风险的人群：黑人男性和有家族前列腺癌史或遗传风险因素如[BRCA变异](https://www.theguardian.com/society/2023/sep/15/consumer-genetic-test-results-causing-unnecessary-breast-cancer-alarm)的男性。
