# Overture Maps Foundation Releases Beta of Its First Open Map Dataset – Overture Maps Foundation

> 来源：[https://overturemaps.org/overture-maps-foundation-releases-beta-of-its-first-open-map-dataset/](https://overturemaps.org/overture-maps-foundation-releases-beta-of-its-first-open-map-dataset/)

*预计生产就绪的1.0版本将释放无数的地图服务和产品创新*

**加利福尼亚州旧金山 — 2024年4月16日** — [Overture Maps Foundation](https://overturemaps.org/)（Overture），一个协同努力的组织，旨在实现当前和下一代可互操作的开放地图服务和产品，今天宣布其全球开放地图数据集的Beta版发布，标志着Overture的基础数据即将达到生产就绪的形式和格式。

映射服务和应用提供商可以开始使用Overture的数据和模式进行广泛的商业地图应用和地理空间分析的Beta测试。在Beta阶段，Overture将收集来自这些评估的反馈，并朝着一般发布迈进。尽管预计会有增量变化，但Overture预计数据和模式将基本稳定。预计生产发布将于夏季开始。

“Overture正在实现其承诺，构建和提供高质量的地图数据以支持地图应用程序，”Overture Maps Foundation执行董事Marc Prioleau说道。“这次Beta版发布集合了多个开放数据源，经过了多次验证测试，采用了新的模式，并具有允许附加其他空间数据的实体参考系统。这对于通过提供可以用于应用程序的数据来推动开放地图数据的进步是一个重要的步骤。它还为数据集的未来改进和扩展奠定了基础。我们的成员公司已经开始使用Overture，我们期待更多的用户和反馈，以便我们能够迅速进入生产并提供全面的市场级开放地图数据集。”

领先的Overture成员已开始将Overture基础地图整合到其服务和产品中。例如，Meta正在基于Overture数据提供其地图解决方案。微软正在全球范围内采用Overture以增强Bing Maps的覆盖范围。Esri正在开始使用Overture数据在ArcGIS中发布新的2D和3D地图图层，而TomTom将把Overture数据整合到其Orbis Maps中，为最广泛的使用案例提供最全面的地图。通过Beta版发布，整个地图社区可以开始测试Overture数据并将其纳入其服务管道中。

Overture 独特之处在于允许企业通过 Overture 的[全球实体参考系统或 GERS](https://docs.overturemaps.org/guides/gers/)将外部数据附加到 Overture 基础地图上。在 Beta 版中，GERS ID 或唯一标识符附加到数据中的实体。这使得数据能够一致、正确并更快地添加到地图中。Beta 版还包含了 Overture 地图数据架构的最新改进，这使得地图服务开发人员能够以标准且有文档记录的方式摄取和使用地图数据，并具备互操作性，从而使用数据变得简单和可预测。

Beta 版包括五个独特的基础层，这些层从最佳可用的开放数据源中组装而成，格式化为 Overture 架构并分配了 GERS ID：

+   **兴趣点**。这包括全球近 5400 万个地点的数据，附带置信度评分，允许用户构建基于地图的本地发现工具或驱动本地搜索引擎。

+   **建筑物**。全球拥有 23 亿个独特建筑物轮廓。任何地图服务提供商或开发者都可以在建筑信息之上添加他们的数据，创建从物业管理、风险评估、经济发展到三维可视化等新的应用案例。

+   **交通**。代表着全球范围内的道路、人行道和其他交通基础设施网络，已经以标准化架构提供，方便地在地图应用程序中使用。

+   **行政边界**。作为一个全球开放数据集，包含国家和地区行政边界的边界数据，其中包括已翻译成超过 40 种不同语言以支持国际使用的地区名称。

+   **基础层**。这些上下文层包括陆地和水域数据，并在需要时帮助完善显示地图。

**Overture 的数据配方**

Overture 数据来源于最优秀的开放地图数据源，并汇总成单一数据集。数据包括社区构建的数据，包括 OpenStreetMap、卫星/航空影像推导的 AI 数据、政府发布的权威数据以及新开放的商业数据源。然后对数据进行验证和质量检查，以确保其安全可用。持续的挑战是保持并添加数据以反映变化。为此，Overture 正在促进广泛合作，建立和维护最新的、全面的数据库。

**关于 Overture 地图基金会**

成立于2022年，Overture Maps Foundation 是全球领先的家园，致力于可靠、易于使用和可互操作的开放地图数据开发协作。这种互操作地图是可扩展性的基础，使公司能够贡献自己的数据。会员们汇集资源，构建完整、准确且随着现实世界变化而更新的地图数据。地图数据将在开放数据许可下对所有人开放和扩展。更多信息，请访问 overturemaps.org。
