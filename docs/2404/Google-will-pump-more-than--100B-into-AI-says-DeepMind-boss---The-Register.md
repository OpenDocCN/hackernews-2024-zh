# 谷歌将向AI投入超过1000亿美元，DeepMind老板表示 • The Register

> 来源：[https://www.theregister.com/2024/04/17/google_deepmind_funding/](https://www.theregister.com/2024/04/17/google_deepmind_funding/)

根据DeepMind首席执行官Demis Hassabis的说法，谷歌最终将投资1000亿美元到人工智能领域。

Google在2014年收购的DeepMind首席执行官于周一在温哥华的TED大会上进行了预测，据[Bloomberg](https://www.bloomberg.com/news/articles/2024-04-16/deepmind-ceo-says-google-will-spend-more-than-100-billion-on-ai)报道。有人问Hassabis关于谷歌竞争对手微软和OpenAI据称正在计划的[传言中的1000亿美元星门超级计算机](https://www.theregister.com/2024/04/01/microsoft_openai_5gw_dc/)。

Hassabis简单地回答说：“我们不会讨论具体的数字，但我认为随着时间推移，我们的投资会超过这个数字。” 他没有具体说明谷歌将如何花费这1000亿美元或更多，也没有说明花费速度如何，但他坚称巧克力工厂的处理能力超过了微软或任何其他公司。

支出所有这些现金的一个途径很可能是硬件 - 例如基于Arm架构的谷歌Axion CPU。这些即将推出的芯片内部有什么还不清楚，但谷歌声称它们比“最快的通用Arm实例”（可能是与Ampere或亚马逊最新CPU的比较）快30%，比Intel和AMD的基于x86的处理器更快50%，效率提高60%。当然，这些只是谷歌的声明，尚未经过独立测试。

这笔1000亿美元预算的一部分可能会分配给DeepMind，后者涉足AI的软件方面。

DeepMind最近的一些活动并不算太引人注目。关于AI发现材料的研究[似乎并不那么有用](https://www.theregister.com/2024/04/11/google_deepmind_material_study/)，其天气预测模型[无法替代传统预测方法](https://www.theregister.com/2023/11/15/google_deepmind_graphcast/)，DeepMind本身也承认。

然而，该组织在推动AI获得[社交技能](https://www.theregister.com/2023/11/28/ai_agents_can_copy_humans/)方面取得了进展，这是使AI更加智能的重要一步。

但是，为了实现这一切，Hassabis解释说他的团队需要“大量计算资源”，这也是他们2014年与谷歌合作的“一个原因”之一。

虽然谷歌可能不会把1000亿美元投入到单一的超级计算机中，但它肯定会增加其计算能力组合，例如即将到来的[$10亿美元堪萨斯城数据中心](https://www.theregister.com/2024/03/20/googles_kansas_city_datacenter/)。®
