# Anduril - Anduril被选中参与美国空军协同作战飞行器CCA项目

> 来源：[https://www.anduril.com/article/anduril-air-force-collaborative-combat-aircraft-CCA/](https://www.anduril.com/article/anduril-air-force-collaborative-combat-aircraft-CCA/)

今天，美国空军宣布Anduril已被选为协同作战飞行器（CCA）项目的两家供应商之一，未来阶段，Anduril将设计、制造和测试生产代表性的CCA。

“现在没有时间继续以往的惯例做法了。通过CCA计划，肯德尔部长和空军已经接受了快速前瞻的态度，以快速和规模化地部署自主系统，”Anduril的CEO兼联合创始人Brian Schimpf说。“我们很荣幸能被选中参与这一前所未有的机会，这预示着对国防工业基地持续扩展的需求。Anduril自豪地为其他非传统国防公司在大型项目上竞争和交付铺平了道路。”

“Anduril在这个项目上的工作刚刚开始，”Anduril空中优势与打击部门高级副总裁Jason Levin说道。“未来，美国和盟国的成功需要以超过对手速度的速度、成本和规模交付CCA。我们期待继续与美国空军合作，尽快将这一关键能力交付给我们的空军人员。”

Anduril自2017年成立以来，自主性和可负担的大规模一直是其核心理念。Anduril致力于通过将现代软件专业知识与快速、独特的硬件开发和制造方法结合，改变美国和盟国的防务能力。从尖端的反无人机系统到超大型自主水下车辆，Anduril已经证明其能够按时、按规模提供高性能、下一代、软件定义的能力。
