# 特斯拉因油门踏板故障召回所有3,878辆Cybertruck - The Verge

> 来源：[https://www.theverge.com/2024/4/19/24134753/tesla-recall-cybertruck-faulty-accelerator-pedal-nhtsa-defect](https://www.theverge.com/2024/4/19/24134753/tesla-recall-cybertruck-faulty-accelerator-pedal-nhtsa-defect)

特斯拉因故障导致车辆油门踏板卡住，已经为实际交付给客户的每一辆Cybertruck发布了召回通知。

[根据国家公路交通安全管理局](https://static.nhtsa.gov/odi/rcl/2024/RCLRPT-24V276-7026.PDF)（NHTSA）在周三的消息显示，当施加“高力量”时，该缺陷可能导致踏板垫脱落并困在车辆内饰中。

故障是由于在油门踏板组装过程中引入了“未经批准的更改”，导致“润滑剂（肥皂）”减少了垫子的保持力，召回通知指出。如果油门踏板被卡住，卡车的制动仍将正常工作，尽管这显然不是一个理想的解决方法。

召回影响“所有2024年款（'MY'）Cybertruck车辆，从2023年11月13日到2024年4月4日制造”，估计在总计3,878辆车中的100％存在故障。这基本上是自其[去年的发布活动](/2023/11/30/23982569/tesla-cybertruck-delivery-price-specs-features-elon-musk)以来交付给客户的每一辆Cybertruck。

[一次召回似乎是不可避免的](/2024/4/15/24131237/tesla-delivery-delay-accelerator-pedal-stuck)，在此之前，Cybertruck客户据报本周早些时候收到通知，他们的交付正在延迟，至少有一位车主被其车辆经销商告知由于油门踏板而被召回。另一位Cybertruck车主在TikTok上也突显了这个问题，展示了故障“将油门踏板100％卡住，全开油门”的情况。

根据NHTSA文件中报告的时间线，特斯拉首次在3月31日收到有关缺陷油门踏板的通知，随后在4月3日收到第二份报告。公司在4月12日完成了内部评估以找出原因，随后自愿发布了召回通知。截至本周一，特斯拉表示并不知道与油门踏板故障有关的任何“碰撞、伤害或死亡”事件。

特斯拉正在通知其门店和服务中心，关于“大约”在4月19日左右的问题，并承诺无偿为Cybertruck车主更换或重新加工召回车辆的油门踏板。从4月17日起生产的所有卡车还将配备新的油门踏板组件和零件编号。

这是特斯拉的多次召回之一，影响了Cybertruck，但这次是最为重要的。公司在今年2月份曾因警示灯面板上的字体过小，未达到安全标准而在美国召回了200万辆特斯拉车辆（[查看详情](/2024/2/2/24059114/tesla-recall-brake-system-font-size-power-steering)），尽管后来通过软件更新解决了这个问题。

特斯拉的粉丝们过去对“召回”一词曾提出异议，因为该公司已经证明通过空中升级软件更新能够有效修复问题。但在这种情况下，他们可能不得不承认，这个术语确实适用。
