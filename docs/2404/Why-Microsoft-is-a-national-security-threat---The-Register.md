# 为什么微软是一个国家安全威胁 • The Register

> 来源：[https://www.theregister.com/2024/04/21/microsoft_national_security_risk/](https://www.theregister.com/2024/04/21/microsoft_national_security_risk/)

采访 微软在美国联邦政府内部拥有令人震惊的IT控制力度-前白宫高级网络政策主任AJ Grotto 认为，称其最近的安全失败为国家安全问题是公平的。

本周，Grotto 在一次与 *The Register* 的采访中谈到了这些问题，你可以在下面的视频中观看这次采访，他告诉我们，即使微小的让步也是联邦政府的一场重大斗争。

[Youtube 视频](https://www.youtube.com/watch?v=FxDsYy-c7Bo)

“如果你回顾一下几年前的 [SolarWinds 事件](https://www.theregister.com/2023/10/31/sec_charges_solarwinds_sunburst_fraud/) … [微软] 实际上是在向联邦机构推销日志记录能力”，而不是将其设为默认选项，Grotto 说道。“因此，政府机构要识别自己在 SolarWinds 泄露中的风险是非常困难的。”

Grotto 告诉我们，微软必须“被迫勉强地”默认为政府提供日志记录能力，考虑到这家大公司去年从安全服务中获得了大约 200 亿美元的收入，这种让步至多可以说是微不足道的。

Grotto 表示：“这表明，他们 [微软] 拥有大量的影响力，而且他们毫不畏惧地使用它。”

此外，还有关于中国监听者对 Exchange Online 的 [入侵](https://www.theregister.com/2024/04/03/cisa_microsoft_exchange_online_china_report/)，以及俄罗斯网络操作人员的 [另一次微软安全漏洞](https://www.theregister.com/2024/04/12/microsoft_cisa_order/)，这两者都允许间谍获取美国政府的电子邮件访问权限，因此 Grotto 表示，将微软及其产品列为国家安全关注重点是合理的。

但是，在 Grotto 看来，85%的美国政府生产软件，甚至更多的操作系统份额属于 Redmond，那么如何解决这个问题呢？

“政府需要专注于鼓励和促进竞争，” Grotto 说。他认为政府还需要公开审视微软，并确保每个人都知道它何时搞砸了。

“归根结底，微软或者任何公司，最终都会直接回应市场激励，” Grotto 告诉我们。“除非这种审查能够改变其客户的行为，使他们可能希望转向其他选择，否则微软改变的激励就不会像他们本应有的那样强烈。” ®
