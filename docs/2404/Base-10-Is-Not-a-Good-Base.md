# 十进制不是一个好的进制

2024年2月17日

我想到了我不喜欢米制的另一个原因：十进制不是一个好的进制。你希望你的进制要么是2的幂，这样你可以一直除以2（就像一加仑有128液体盎司），要么有很多有用的因数（就像一英尺有12英寸，所以你可以很容易地得到⅓或¼；和一分钟有60秒）。十进制的因数并不是很好；你很少想要除以5。

某些外星种族有八根手指，当他们来访时，他们理解 *为什么* 我们使用十进制，但对他们来说，这一定是一个多么糟糕的进制。就像你有100克，你可以除以2两次，然后只能再除以5。谁需要除以5？尤其是在烹饪中。

这就好像我们发现了一种在所有事情上都使用14进制的外星物种一样。一个单位，然后14，然后196？！如果你有196克，你可以除以2两次，然后你只能再除以7？糟糕的进制！使用12或8之类的。

米制的整个基础都是一个糟糕的进制。
