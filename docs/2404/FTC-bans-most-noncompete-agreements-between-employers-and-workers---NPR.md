# FTC禁止大多数雇主与工人之间的竞业禁止协议：NPR

> 来源：[https://www.npr.org/2024/04/23/1246655366/ftc-bans-noncompete-agreements-lina-khan](https://www.npr.org/2024/04/23/1246655366/ftc-bans-noncompete-agreements-lina-khan)

联邦贸易委员会主席琳娜·卡恩（Lina Khan）表示，竞业禁止协议阻止了工人转换工作，即使他们可以赚更多钱或享有更好的工作条件。

Drew Angerer/Getty Images

周二，联邦贸易委员会以三对二的投票结果，几乎全面禁止了竞业禁止协议，这些协议通常阻止工人加入竞争企业或自己创业。

在投票前几个月，FTC收到了超过[26,000条公众评论](https://www.npr.org/2024/04/23/1246430110/noncompete-agreements-ftc-ban-lina-khan)。主席琳娜·卡恩在周二引用了一些工人的故事。

“我们听到一些员工因为竞业禁止条款而陷入虐待工作环境的困境，”她说。“有人指出，当雇主与一个宗教原则与自己相冲突的组织合并时，竞业禁止条款阻止了工人自由选择不与他们宗教实践冲突的工作。”

这些账户，她说，“指出了剥夺人们经济自由如何同时剥夺他们各种其他自由的基本现实。”

FTC估计，从最低工资收入者到CEO，约有3000万人，即五分之一的美国工人，受到[竞业禁止协议的约束](https://www.npr.org/2023/01/13/1148446019/ftc-rule-ban-noncompetes-low-wage-workers-trade-secrets)。它表示，这一政策变化可能通过鼓励人们自由交换工作岗位，导致近3000亿美元的工资增长。

这项禁令将于今年晚些时候生效，但为现有的竞业禁止协议中给予公司高级主管的一项例外，理由是这些协议更可能已经经过协商。FTC表示，雇主不应强制执行其他现有的竞业禁止协议。

投票结果是3比2，沿党派路线分歧。反对派委员梅利莎·霍利奥克（Melissa Holyoak）和安德鲁·弗格森（Andrew Ferguson）认为FTC越过了其权力边界。霍利奥克预测这一禁令将在法庭上受到挑战，并最终被撤销。

投票后不久，美国商会表示将起诉FTC以阻止该规则，并称其是不必要、非法且明显的权力扩张。

一年多来，该团体一直积极反对这项禁令，称非竞争条款对公司至关重要，因为它们能够更好地保护商业机密，同时也对雇员有利，因为能够鼓励雇主更多地投资于员工培训和发展。

"这一决定为政府对商业的微观管理设立了危险的先例，可能会损害雇主、工人和我们的经济，" 美国商会总裁兼首席执行官苏珊·P·克拉克在一份声明中写道。
