# AWS停止销售雪橇车用于云迁移

> 来源：[https://www.cnbc.com/2024/04/17/aws-stops-selling-snowmobile-truck-for-cloud-migrations.html](https://www.cnbc.com/2024/04/17/aws-stops-selling-snowmobile-truck-for-cloud-migrations.html)

亚马逊网络服务雪橇车

CNBC

在[亚马逊](/quotes/AMZN/)2016年的年度云计算会议上，该公司通过在舞台上展示一辆18轮卡车吸引了观众的注意力。现任亚马逊首席执行官安迪·贾斯（Andy Jassy）称之为“雪橇车”，并表示公司将使用这辆卡车帮助客户快速将数据转移到亚马逊网络服务设施。

不到8年后，这辆卡车已经停用。

截至三月，AWS已经从其网站上移除了雪橇车，而亚马逊单位已经停止提供该服务，CNBC已证实。AWS“[雪系列](https://aws.amazon.com/snow/)”产品的网页现在将用户引导至其他数据传输服务，包括Snowball Edge，一个重50磅、大小像手提箱的设备，可以装备快速固态驱动器，以及更小的Snowcone。

一位AWS发言人在一封电子邮件声明中表示，公司已经推出了更具成本效益的数据移动选项。当客户使用雪橇车服务时，他们必须处理电力、制冷、网络、停车和安全等问题，发言人说道。

“自2016年推出雪橇车以来，我们推出了许多新服务和功能，这些服务和功能使得向AWS迁移数据变得更加快速和简便，对于我们的客户而言。”发言人写道。

2019年，一辆AWS雪橇车出现在西雅图的停车场。

安德鲁·埃弗斯 | CNBC

根据AWS网站上曾经发布的页面显示，雪橇车的价格为每月$0.005/GB，不包括其他成本。对于有100PB数据容量（雪橇车的容量）的公司来说，一个转移作业每月大约需要花费$500,000。

亚马逊决定取消雪橇车，这是贾斯为应对销售增长乏力而实施的成本削减措施的一部分。自2022年末以来，亚马逊已经[裁减](https://www.cnbc.com/2023/03/20/amazon-layoffs-company-to-cut-off-9000-more-workers.html)了超过27,000个工作岗位，并中止了设备和零售部门的项目。今年以来，亚马逊在AWS部门[裁减了数百个](https://www.cnbc.com/2024/04/03/amazon-layoffs-hundreds-of-jobs-cut-in-cloud-computing-unit.html)工作岗位。

尽管AWS及其竞争对手[微软](/quotes/MSFT/)Azure和[谷歌](/quotes/GOOGL/)云平台通常会停用产品和服务，但由于雪橇车是如何在2016年底的亚马逊重生大会上引入的方式而显得格外引人注目。

贾斯当时领导AWS，在成千上万人面前发表主题演讲，18轮卡车与他一同登台。

“我们需要一个更大的箱子”，Jassy 说道，观众们纷纷举起手机来拍摄这一壮观景象。

Jassy 向观众解释了为什么这款卡车具有开创性。他说，通过 10 Gbps 的连接，将一艘艘比特，或者说 1 百万 TB 的数据迁移到云端需要 26 年。AWS 的客户可以用 10 辆 Snowmobile 在不到六个月内完成这项工作。他说，每个 Snowmobile 在硬盘驱动器上的容量为 100 PB。

在与 2016 年 11 月 30 日发布的一篇[博客文章](https://aws.amazon.com/blogs/aws/aws-snowmobile-move-exabytes-of-data-to-the-cloud-in-weeks/)同时，亚马逊云传道人 Jeff Barr 描述 Snowmobile 为“一个坚固、防篡改的船运集装箱，长 45 英尺，高 9.6 英尺，宽 8 英尺”，“可以停放在现有数据中心旁的有盖或无盖区域”。

Barr 帮助通过一组乐高积木建造的 Snowmobile 照片传达了该过程的简单性，这些积木正在连接到公司的数据中心。

“我们打算确保 Snowmobile 既更快又比使用基于网络的数据传输模型更便宜”，Barr 写道。

但是这款产品并没有获得成功。

卫星运营商 Maxar 的发言人表示，公司曾在 2017 年使用 Snowmobile 一次，从自己的服务器向 AWS 迁移了超过 100 PB 的数据。发言人称，那时的 Snowmobile 是“那个时刻的理想解决方案”。

“从那时起，我们一直将我们的图像和相关数据直接上传到云端”，发言人补充道。

AWS 仍然领导着庞大的云基础设施市场，去年产生了 908 亿美元的[收入](https://www.cnbc.com/2024/02/01/aws-q4-earnings-report-2023.html)，占亚马逊总销售额的 16%。公司发言人表示，AWS 的 Snowball Edge 设备比 Snowmobile 车辆更小，成本更低，周转时间更短，客户可以通过邮寄将其归还给亚马逊，设备用完后。

还有 AWS 数据同步服务用于数据传输，该服务在 2018 年宣布推出。据公司称，客户普遍发现将数据在线发送到 AWS 比使用 Snowmobile 更经济。

“我们为 Snowmobile 带给客户的价值感到非常自豪，我们很高兴看到他们选择了更新、更高效的技术”，发言人写道。
