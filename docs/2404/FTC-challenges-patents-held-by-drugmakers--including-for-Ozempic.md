# FTC挑战制药公司持有的专利，包括Ozempic

> 来源：[https://www.cnbc.com/2024/04/30/ftc-challenges-patents-held-by-drugmakers-including-for-ozempic.html](https://www.cnbc.com/2024/04/30/ftc-challenges-patents-held-by-drugmakers-including-for-ozempic.html)

2023年10月31日，一盒Ozempic及其内容摆放在英国泰恩赛德郡达德利的桌子上。

李·史密斯 | 路透社

联邦贸易委员会周二表示，正在挑战数百项制药公司持有的涉嫌“垃圾”专利，涵盖20种品牌药物，包括[诺和诺德](/quotes/NVO/)的畅销药物Ozempic、Saxenda和Victoza。

FTC向10家公司发出了信函，警告他们某些药物专利的列示方式不当。这些公司包括诺和诺德、[阿斯利康](/quotes/AZN/)、百灵格、Covis Pharma、[葛兰素史克](/quotes/GSK/)、[诺华](/quotes/NVS/)、[特瓦制药](/quotes/TEVA/)和[安发斯达制药](/quotes/AMPH/)，以及它们的部分子公司。

许多药物专利适用于2型糖尿病，还有支气管哮喘和慢性阻塞性肺病（COPD）的吸入器。

大多数畅销药物由数十项专利保护，涵盖各种成分、制造过程和知识产权。只有在专利到期或在法庭上成功挑战后，仿制药制造商才能推出更便宜的品牌药物版本。

“通过提交虚假专利列表，制药公司阻碍了竞争，提高了处方药的成本，迫使美国人支付高昂的药物价格，他们依赖这些药物，”FTC主席丽娜·汗在一份声明中说道。“通过挑战垃圾专利申请，FTC正在打击这些非法手段，确保美国人能够及时获得创新且价格合理的药物版本。”

FTC还通知了食品药品监督管理局面临的挑战。FDA在一份名为橙皮书的文件中管理批准药物的专利列表。

FTC去年秋天首次挑战了数十个品牌药物专利，导致三家制药公司配合并在FDA撤销了其专利。另外五家公司则未予配合。

星期二的公告扩展了拜登政府对制药行业涉嫌专利滥用的打击力度。FTC认为，制药公司不必要地列出数十个额外专利，用于保持其品牌药物的高价并阻碍仿制药进入美国市场。

专利争议增加了拜登政府努力使医疗更加负担得起的更广泛努力的一部分 —— 这是总统乔·拜登2024年连任竞选的关键支柱。

"我们赞赏联邦贸易委员会与食品药品监督管理局合作打击大型制药公司的专利游戏，并降低处方药费用，包括减肥药和糖尿病药物，" 美国国家经济委员会副主任乔恩·多南伯格在一份声明中告诉 CNBC。
