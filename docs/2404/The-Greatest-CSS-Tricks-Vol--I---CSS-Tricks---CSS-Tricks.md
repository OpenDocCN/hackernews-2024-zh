## 关于本书

作者：Chris Coyier

这是我在运营这个网站期间遇到的我最喜欢的技巧汇编。其中大部分并非我自己的创意，而是由比我聪明得多的人提出的。在这里，我展示它们并按照我理解的方式解释这些技巧。也许其中一些比其他技巧更“有用”，但无论您是否直接使用这些技术，都可以从中学到很多。
