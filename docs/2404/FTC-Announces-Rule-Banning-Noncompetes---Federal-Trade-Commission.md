# 联邦贸易委员会宣布禁止非竞争规则 | 联邦贸易委员会

> 来源：[https://www.ftc.gov/news-events/news/press-releases/2024/04/ftc-announces-rule-banning-noncompetes](https://www.ftc.gov/news-events/news/press-releases/2024/04/ftc-announces-rule-banning-noncompetes)

今天，联邦贸易委员会（[发布了最终规则](/system/files/ftc_gov/pdf/noncompete-rule.pdf)）以促进竞争为目的，在全国范围内禁止非竞争条款，保护工人换工作的基本自由，增加创新，促进新业务的形成。

“非竞争条款使工资保持较低水平，抑制新想法，并且从禁止非竞争条款后将创立的8,500多家新创企业中包括的动态性中剥夺了美国经济的活力，”联邦贸易委员会主席莉娜·M·卡恩说道。“联邦贸易委员会禁止非竞争的最终规则将确保美国人民有自由去追求新工作，创业或将新想法推向市场。”

联邦贸易委员会估计，禁止非竞争最终规则将导致新业务的形成年增长率达到2.7%，每年新增超过8,500家新企业。最终规则预计将增加工人的收入，预计平均每年每位工人的收入额外增加524美元，并预计在未来十年内降低高达1,940亿美元的医疗保健成本。此外，最终规则预计将推动创新，导致在最终规则下未来十年每年平均增加17,000至29,000项专利。

[](/node/85151)

非竞争条款是一种普遍且经常是剥削性的做法，强加合同条件，阻止工人换工作或创业。非竞争条款常常迫使工人要么留在他们想要离开的工作中，要么承受其他重大伤害和成本，比如被迫转向薪酬较低的领域，被迫搬迁，被迫离开劳动力市场，或者被迫应对昂贵的诉讼。据估计，约有3000万工人——近五分之一的美国人——受到非竞争条款的约束。

根据联邦贸易委员会的新规定，大多数工人现有的非竞争条款在规定生效日期后将不再具有可执行性。对于占工人人数不到0.75%的高级执行人员，他们现有的非竞争条款可以根据联邦贸易委员会的最终规定继续有效，但雇主被禁止与他们签订或试图执行任何新的非竞争条款，即使涉及到高级执行人员。雇主将被要求通知除高级执行人员外的其他受现有非竞争条款约束的工人，他们不会对他们执行任何非竞争条款。

2023年1月，FTC发布了一份[拟议规则](https://www.ftc.gov/news-events/news/press-releases/2023/01/ftc-proposes-rule-ban-noncompete-clauses-which-hurt-workers-harm-competition)，经过90天的公众评论期。FTC收到了超过26,000条关于拟议规则的评论，其中有超过25,000条评论支持FTC对非竞争条款的禁令。这些评论为FTC的最终规则制定过程提供了信息，FTC仔细审查每条评论，并对拟议规则作出了修改以回应公众的反馈。

在最终规则中，委员会已确定，对雇主与工人签订非竞争协议并执行某些非竞争协议构成不公平的竞争手段，并且违反了FTC法案第5条的规定。

委员会发现，非竞争条款往往会通过抑制工人与雇主之间的有效匹配而对劳动力市场的竞争条件产生负面影响。委员会还发现，非竞争条款往往会对产品和服务市场的竞争条件产生负面影响，抑制新业务的形成和创新。还有证据表明，非竞争条款导致市场集中度增加，并且导致消费者价格上涨。

## 非竞争条款的替代方案

委员会发现，雇主有几种替代非竞争条款的选择，这些选择仍然能使企业保护其投资，而无需执行非竞争条款。

商业机密法和保密协议（NDAs）都为雇主提供了保护专有和其他敏感信息的成熟手段。研究人员估计，超过95%拥有非竞争条款的工人已经签署了保密协议。

委员会还发现，雇主如果希望留住员工，可以通过提高工资和工作条件，基于工人的劳务供应优势与工人竞争，而不是使用非竞争条款锁定工人。

## 从NPRM（提前通知条约）的变化

在最终规则下，高级主管的现有非竞争协议仍然有效。然而，雇主禁止与高级主管签订或执行新的非竞争协议。最终规则将高级主管定义为年薪超过151,164美元且履行决策职位的工人。

此外，委员会取消了拟议规则中的一项规定，该规定要求雇主法律上修改现有非竞争协议并正式废除它们。这一变化将有助于简化合规流程。

反之，在最终规则下，雇主只需通知受制于现有非竞争协议的工人，将不会在未来对他们执行非竞争协议。为了帮助雇主遵守这一要求，委员会在最终规则中包含了[范本语言](/legal-library/browse/rules/noncompete-rule)，这些语言可供雇主用来与工人沟通。

Ferguson）委员。[丽贝卡·凯莉·斯劳特](/news-events/news/speeches/remarks-commissioner-rebecca-kelly-slaughter-supporting-final-rule-banning-noncompete-agreements)、[阿尔瓦罗·贝多亚](/news-events/news/speeches/oral-statement-commissioner-alvaro-m-bedoya-ftcs-issuance-non-compete-clause-rule)、[梅丽莎·霍利奥克](/news-events/news/speeches/oral-statement-commissioner-melissa-holyoak-matter-non-compete-clause-rule) 和 [安德鲁·N·弗格森](/news-events/news/speeches/oral-statement-commissioner-andrew-n-ferguson-matter-non-compete-clause-rule) 委员各自发布了单独的声明。主席丽娜·M·卡恩将发布单独声明。

最终规则将在《联邦公报》发布后120天生效。

规则生效后，市场参与者可以通过电邮向竞争局报告涉嫌违反该规则的信息，电邮地址为 [noncompete@ftc.gov](mailto:noncompete@ftc.gov)。
