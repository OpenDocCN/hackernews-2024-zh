# 罗克特斯塔测试了太空推进过程中的聚变增强 | 航空航天测试国际

> 来源：[https://www.aerospacetestinginternational.com/news/space/rocketstar-tests-fusion-enhanced-in-space-propulsion-process.html](https://www.aerospacetestinginternational.com/news/space/rocketstar-tests-fusion-enhanced-in-space-propulsion-process.html)

美国初创企业罗克特斯塔已成功展示了一种利用核聚变增强的脉冲等离子推进器用于航天器的电推进单元。

该公司的FireStar驱动器使用的是水燃料脉冲等离子推进器，它利用一种非中性核聚变来增强其性能。

据罗克特斯塔称，其推进系统的基础推进器通过水蒸气的电离生成高速质子。当这些质子与硼原子核碰撞时，原子核发生聚变，转化为一个高能量形式的碳，迅速分解成三个α粒子。

通过在推进器的排气中引入硼，FireStar驱动器使得这种聚变过程成为可能。类似于喷气发动机中的加力燃烧室通过在排气中引入燃料来增加推力的方式，推进器排气中发生的聚变显著提高了其性能。

这一聚变过程最初是在美国空军的AFWERX（空军工作项目）倡议的研发计划期间设计的，其中在脉冲等离子推进器的排气喷流中引入了硼化水，从而产生了α粒子和γ射线，明确表明了核聚变的存在。

在美国亚特兰大的乔治亚理工学院高功率电推进实验室（HPEPL）的后续测试中，这一过程已经得到进一步验证。罗克特斯塔表示，在测试中，该过程生成了电离辐射，并将基础推进单元的推力提高了50%。

新墨西哥大学核工程教授亚当·赫希特（Adam Hecht）说：“罗克特斯塔不仅仅是逐步改进了推进系统，而是通过应用一种新颖的概念，实现了排气中的核聚变-裂变反应，迈出了一大步。这是技术发展的激动人心时刻，我期待他们未来的创新。”

罗克特斯塔的首席执行官克里斯·克拉多克（Chris Craddock）表示：“我们对我们团队多时以来一直在探索的一个想法的初步测试结果感到非常激动。在佛罗里达的一次会议上，我在餐巾纸上勾勒了这个想法，并向迈尔斯航天的创始人韦斯·费勒（Wes Faler）描述了这个想法。他在开发基础推进器和聚变增强方面非常聪明。

“我们收购了迈尔斯航天，费勒现在是我们的首席技术官。因此，我很兴奋地将我们已经出色的推进器与聚变增强结合起来，以显著提高性能。感谢AFWERX和美国空军航空航天司令部（USSF）相信这是可能的！”

今年计划测试FireStar推进器，包括进一步的地面测试，以及计划于2025年2月在太空机器人公司Rogue Space Systems的Barry-2号航天器上作为载荷进行太空演示。“我们非常兴奋地为RocketStar测试FireStar推进器”，Rogue Space Systems的首席运营官Brent Abbott说。“我们期待考虑将其用于未来的Rogue任务。”
