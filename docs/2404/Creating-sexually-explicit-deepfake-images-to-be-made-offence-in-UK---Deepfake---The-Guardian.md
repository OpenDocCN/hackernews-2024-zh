# 创建英国的性露骨deepfake图像将被定为犯罪行为 | Deepfake | 卫报

> 来源：[英国法律将创作性露骨deepfake图像定为犯罪行为](https://www.theguardian.com/technology/2024/apr/16/creating-sexually-explicit-deepfake-images-to-be-made-offence-in-uk)

创建性露骨的“deepfake”图像将成为新法律下的犯罪行为，司法部宣布。

根据该法律，未经同意创建此类图像的人将面临刑事记录和无限罚款。如果图像被广泛传播，他们还可能面临监禁。

-   这一罪行将通过对刑事司法法案的修正引入，该法案正在议会中审议中。负责受害者和保护工作的部长劳拉·法里斯（Laura Farris）表示，制作深假性图像“无论是否分享”都是不可接受的。

-   “这是某些人试图贬低和无人性化他人的又一例子，尤其是女性。如果这些材料被广泛传播，它有可能引发灾难性后果。本届政府将不容忍这种行为。

-   “这项新罪行向外界传达了一个清晰的信息，即制作这种材料是不道德的，往往是对女性的歧视，也是一种犯罪。”

-   身为影子内政大臣的伊芙特·库珀（Yvette Cooper）支持了这一声明：“政府接受了工党呼吁将制作深假色情刑事化的决定是受欢迎的。将某人的图像叠加在性暗示的照片和视频上是对其自主权和隐私的严重侵犯，可能造成巨大的伤害，这是不可容忍的。

-   “警方和检察官必须配备严格执行这些法律所需的培训和工具，以防止肇事者免于惩罚。”

-   “Glamour UK”的总编辑黛博拉·约瑟夫（Deborah Joseph）对计划的修正表示欢迎。

“在最近的《Glamour》调查中，我们发现91%的读者认为深度伪造技术对妇女的安全构成威胁。从受害者的个人故事中听到，我们也知道其影响有多严重，”她说。

“虽然这是重要的第一步，但在妇女真正感到安全免受这种可怕活动影响之前，还有很长的路要走。”
