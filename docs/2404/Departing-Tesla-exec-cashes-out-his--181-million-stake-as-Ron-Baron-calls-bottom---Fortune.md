# 离任的特斯拉高管清仓售出其价值1.81亿美元的股份，而Ron Baron则认为股价已触底 | Fortune

> 来源：[https://fortune.com/2024/04/26/tesla-elon-musk-drew-baglino-insider-sale-ron-baron/](https://fortune.com/2024/04/26/tesla-elon-musk-drew-baglino-insider-sale-ron-baron/)

特斯拉的前动力总成工程高级副总裁Drew Baglino在上周离职后几天就清仓了。他在[Fortune](https://fortune.com/company/tesla/)报道中称，他几乎完全清仓了自己在公司中的所有股份，价值1.81亿美元。

在一份[监管披露](https://ir.tesla.com/_flysystem/s3/sec/000197236224000520/000197236224000520-gen.pdf)中，特斯拉表示，Baglino行使了其大部分已经获得的股票期权，转换为114万股。他在周三将这些股票迅速抛售到市场上，这是特斯拉提交其10-Q报告后的第一时间，也是公司内部人士的最早可能机会。

这些期权在他[离职](https://fortune.com/2024/04/23/tesla-elon-musk-ceo-executive-resign-earnings-call-electric-vehicle/?utm_source=search&utm_medium=suggested_search&utm_campaign=search_link_clicks)后90天即将到期，但Baglino本可以选择保留部分股份。

曾经是取代Elon Musk成为特斯拉王座上领先竞争者的Baglino，帮助打造了公司领先的商业能源存储业务，以其工业规模的Megapack电池为主导，可能是公司最高利润的制造产品。

但他负责的太阳能屋顶业务仅剩下少量数据，特斯拉本季度停止发布部署数据，这是其唯一公开的指标。更糟的是，一度被视为确保特斯拉在竞争对手面前成本竞争力的基石的4680电池，遇到了重复的制造问题，现在实际上不再是其股票故事的核心。

“我认为至少在短期内这并不是非常重要的”，Musk在周二告诉投资者，他辩称，Battery Day在2020年的主要揭示实际上不过是对电池供应商材料成本飙升的一种保护措施。

“我们进行了电池项目，以应对我们的供应商由于每千瓦时成本的急剧增长而采取的巨大订单”，Musk继续说道。现在，随着像福特这样的老牌车企退出电动车市场后，电池生产商陷入了过剩产能，特斯拉的价格变得更有利。

## Baron对低成本电动车的新战略转型充满信心。

Baglino的股票销售与特斯拉资深投资者Ron Baron在周四的CNBC乐观态度相吻合。Baron Capital的负责人，其最大的头寸是特斯拉，称其股票已触底。

“它将大幅上涨。现在是谷底，”他说，辩称其[新战略转变](https://fortune.com/2024/04/24/elon-musk-tesla-first-quarter-earnings-low-cost-car-new-models/)将填补能年产300万辆汽车的生产网络中的一个[120万辆的空缺](https://fortune.com/2024/03/23/elon-musk-tesla-q1-production-deliveries-earnings-growth/)，而且无需额外的资本支出。

但他对股票的不满显而易见，自去年此时以来仅上涨了1%，并且从2021年初以来总体下跌。

“当市场如此强劲时，一年仅上涨1%并不令人激动，”他说。“我一直四处看，看到每个人都变得富有，而我并不贫穷，但在过去三年里我并没有取得很大的进展，因此我认为这就像一根橡皮筋——我将再次赶上。”
