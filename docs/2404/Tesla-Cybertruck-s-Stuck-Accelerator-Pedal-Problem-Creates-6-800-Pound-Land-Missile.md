# 特斯拉Cybertruck的卡踏板问题造成了6800磅的陆地导弹

> 来源：[https://insideevs.com/news/716031/tesla-cybertruck-stuck-accelerator-pedal/](https://insideevs.com/news/716031/tesla-cybertruck-stuck-accelerator-pedal/)

特斯拉Cybertruck的所有者们出来公开发表了一个可能会使不锈钢梯形变成6800磅陆地导弹的问题。至少一名车主发布了一段视频，概述了他们在Cybertruck加速踏板设计缺陷方面的经历。

如下视频所示，踏板的盖子可能会脱落并卡在驾驶员侧脚底板的接缝中，导致踏板保持按下状态。

Cybertruck的粗糙发布

特斯拉在Cybertruck发布过程中遭遇重重问题。从安装不当的车门把手到糟糕的面板间隙，以及大量关于动力丧失的报告，评论家们对其质量控制进行了合理的抨击。如果广泛存在，这个加速踏板问题可能是特斯拉迄今为止Cybertruck的最大安全隐患。

幸运的是，遇到这个问题的车主能通过踩刹车来停止加速，切断驱动电机的动力。然而，当驾驶员松开刹车时，据报道卡车立即全速加速。考虑到[6800磅的卡车能在不到2.7秒内从0加速到60英里](https://insideevs.com/news/708502/tesla-cybertruck-cyberbeast-quarter-mile-0-60-mph/)，这种情况有些可怕。

在撰写本文时，特斯拉尚未公开评论延迟原因，也未确认停售是否因加速踏板问题。同样，国家公路交通安全管理局未发布任何投诉，表明踏板存在缺陷或当前召回。

美国各地的Cybertruck购买者，包括加利福尼亚、北卡罗来纳、密苏里和新泽西的购买者，最近被告知他们的卡车交付时间已延迟。受影响的人们询问了延迟背后的原因，但据称未得到回应。

有意购买者推测，最近关于加速踏板的病毒性帖子可能是延迟的原因。尽管目前尚未得到官方确认，但广受欢迎的特斯拉支持者Omar Qazi，也被称为WholeMarsCatalog，发帖表示交付因"[加速踏板问题](an%20issue%20with%20the%20accelerator%20pedal)"而暂停。

[*Drive Tesla*](https://driveteslacanada.ca/news/tesla-cybertruck-deliveries-paused-for-accelerator-pedal-cover-issue/) 报道，博客上分享的信息将踏板问题归因于生产过程中存在的“润滑剂残留”。然而，也值得注意的是，[踏板上似乎没有任何紧固件固定](https://service.tesla.com/docs/Cybertruck/ServiceManual/en-us/GUID-18A00B65-4539-4123-BAAD-663990B8B98E.html)，这意味着踏板可能是摩擦固定或者用粘合剂固定的，可能是成本节约的产物。*Drive Tesla* 还报道称，特斯拉预计将在 [4月20日](https://insideevs.com/news/650999/tesla-elon-musk-not-guilty-420-private-case/) 之前准备好修复方案以恢复交付。

另一位 X 用户补充说，特斯拉正在尽快采购和运送新版本的这种踏板至服务中心和交付地点。没有更多的电动卡车会交付，直到新的踏板被安装完毕，该用户引述多个特斯拉服务中心的说法。

几位车主似乎从特斯拉收到了 [以下信息](https://twitter.com/bfreshwa/status/1779123695082000608)：

> 我们刚刚收到关于您车辆准备方面的意外延误通知。我们需要取消您的交付预约，并在能够重新安排时再次联系您。对于给您带来的不便，我们深感抱歉，并期待尽快安排您的 Cybertruck 交付。

有趣的是，一个车主指出他们的[**Cybertruck**](https://www.cybertruckownersclub.com/forum/threads/brake-malfunction-hit-light-pole.14130/)存在问题，导致这辆卡车在四月初撞上了灯杆。车主在 [*Cybertruck Owners Club*](https://www.cybertruckownersclub.com/forum/threads/brake-malfunction-hit-light-pole.14130/) 论坛上发布了帖子，在他们发布事故照片之前，论坛成员们对此持怀疑态度。即便如此，论坛的成员们仍然认为事情并不简单。

如果这确实是一个更为普遍的问题，几乎肯定会导致官方召回。整个情况让人想起了2010年丰田因加速踏板可能机械卡住而进行的[230万辆车召回](https://www.cars.com/articles/toyota-recalls-2-3-million-vehicles-over-sticking-accelerator-pedal-1420663220496/)，尽管很多指责被夸大，问题往往是由于误用和不正确的地毯放置，但此事对丰田的声誉和财务状况造成了沉重打击。

与此同时，[Cybertruck 发布会遭遇了一些问题](https://insideevs.com/news/715096/tesla-cybertruck-early-breakdowns/)。车主们报告了一个又一个问题，许多人甚至发现他们的卡车被拖车拖走，因为无法驾驶。

从2023年11月的众包数据显示，Cybertruck 预订已超过200万。新订单的等待期约为5年。

特斯拉此前表示，计划在其德克萨斯千兆工厂每年生产125,000辆Cybertruck，并计划到2025年扩展至250,000辆。Cox Automotive的数据显示，特斯拉在2024年Q1仅交付了约2,803辆Cybertruck。

[本月早些时候的一项InsideEVs调查](https://insideevs.com/news/715096/tesla-cybertruck-early-breakdowns/)还发现，Cybertruck车主面临其车辆的起步问题。他们提到了与电气线束、车载功率转换系统（PCS）故障、断裂的驱动单元以及转向和制动问题等相关的问题。特斯拉服务中心迅速解决了一些问题，而其他问题则花费了更长的时间来解决。

新车发布可能很艰难—[只需问问福特](https://www.motortrend.com/news/2020-ford-explorer-launch-problems-lessons-shareholder-meeting/)。但是，如果不能[支持](https://insideevs.com/news/713343/first-tesla-cybertruck-teardown-caresoft-global/)一辆价值10万美元的卡车的[质量控制](https://insideevs.com/news/713112/cybertruck-panel-gap-mkbhd-video/)，一些问题正在标志着特斯拉的车辆不可靠。

Cybertruck交付的障碍出现在特斯拉品牌试图在2024年特别困难的道路上前行之时。

[$25,000的廉价特斯拉据报道已被取消](https://insideevs.com/news/715134/affordable-tesla-canceled/)，现在的重点是将于8月8日揭晓的特斯拉Robotaxi。[特斯拉的Q1销售](https://insideevs.com/news/715296/tesla-ev-production-sales-difference-2024q1/)显著下降，而来自竞争对手品牌的电动汽车，如凯迪拉克Lyriq、现代Ioniq 5和Ioniq 6、起亚EV6和福特野马Mach-E等，现在正在获得越来越多的关注。

特斯拉希望看到的最后一件事情就是Cybertruck出现更多问题，尤其是考虑到去年其发布时的所有华丽和炫耀，以及其在其开发中投入的数十亿美元资源。

如果这个问题导致召回，特斯拉可能会面临更多问题，因为它[已经经历了一个艰难的销售年度](https://insideevs.com/news/714873/tesla-q1-deliveries-sales/)。
