# 两种生物融合成亿万年一遇的进化事件

> 来源：[https://newatlas.com/biology/life-merger-evolution-symbiosis-organelle/](https://newatlas.com/biology/life-merger-evolution-symbiosis-organelle/)

科学家们捕捉到了一次亿万年来的进化事件正在进行中，两种生物融合成一个生物体，拥有其同类羡慕的能力。上次这种情况发生时，地球上出现了植物。

这种现象称为[原生共生](https://newatlas.com/biology/synthetic-hybrids-yeast-bacteria-evolution-symbiosis/)，当一个微生物吞噬另一个，并开始像内部器官一样使用它时发生。作为交换，宿主细胞提供营养、能量、保护和其他利益给共生体，直到最终后者无法独立生存，本质上成为宿主的器官——或者在微生物细胞中称为细胞器。

想象一下，如果肾实际上是小动物在身体里跑来跑去，人类不得不通过透析机手动过滤血液。然后有一天，某个家伙不知怎么地把这些肾小动物之一卡住了……内部（我们能评判什么呢？）——然后意识到他再也不需要透析机了。他的孩子们也是如此，直到最终我们所有人都在体内拥有这些有用的小家伙。这就是这里正在发生的事情。

地球上的生命有40亿多年的历史，我们认为原生共生只发生过两次，而每次都是进化的重大突破。第一次发生在大约22亿年前，一个古菌吞噬了一种细菌，后者成为了线粒体。这种专门的能量产生细胞器使得基本上所有复杂的生命形式得以进化。直到今天，它仍然是备受赞誉的“细胞动力站”。

第二次发生在大约16亿年前，当一些更先进的细胞吞并了能够从阳光中获取能量的蓝细菌。这些成为了叫做叶绿体的细胞器，赋予了光合能力，以及一种迷人的绿色，给你可能听过的一些生物群体——植物。

显微镜下显示的活苔藓细胞，展示其叶绿体（绿色圆圈）

现在，科学家们发现这种情况正在再次发生。一种名为*Braarudosphaera bigelowii*的藻类发现吞噬了一种能让它们做一些通常情况下藻类和植物无法做到的事情的蓝细菌——直接从空气中“固定”氮，并将其与其他元素结合以创建更有用的化合物。

氮素是关键的营养物质，通常植物和藻类通过与保持独立的细菌的共生关系获取它们。起初人们认为*大型B. bigelowii*与名为UCYN-A的细菌建立了这种关系，但经过更仔细的检查，科学家们发现这两者之间的关系更为密切。

在最近的一项研究中，一个团队发现了藻类与UCYN-A之间的大小比例在不同相关物种之间保持相似。它们的生长似乎受到养分交换的控制，导致了联合的新陈代谢。

“这正是细胞器的运行方式，”研究的作者之一乔纳森·泽尔说。“如果你看线粒体和叶绿体，情况是一样的：它们与细胞一起扩展。”

在一项后续研究中，团队和其他合作者利用强大的X射线成像技术观察了活体藻类细胞的内部。这揭示出寄主和共生体之间的复制和细胞分裂是同步进行的，这是原初内共生作用的更多证据。

*Braarudosphaera bigelowii* 不同细胞分裂阶段的X射线图像。新发现的硝酸盐体用青色突出显示，藻类细胞核为蓝色，线粒体为绿色，叶绿体为紫色。

Valentina Loconte/伯克利实验室

最后，团队比较了分离的UCYN-A细菌和藻类细胞内部的蛋白质。他们发现，分离的细菌只能产生大约它所需蛋白质的一半，依赖于藻类寄主提供其余部分。

“这是从内共生体向细胞器发展的典型特征之一，”泽尔说道。“它们开始丢弃DNA片段，基因组变得越来越小，并且开始依赖母细胞来运输这些基因产物或蛋白质本身进入细胞。”

总体而言，团队表示这表明UCYN-A是一个完整的细胞器，被命名为硝酸盐体。它似乎大约在1亿年前开始进化，这听起来是一个非常长的时间，但与线粒体和叶绿体相比只是一瞬间。

研究人员计划继续研究硝酸盐体，以了解它们是否存在于其他细胞中，并探索其可能产生的影响。一个可能的好处是，它可能为科学家提供一个新的途径，将固氮能力引入植物中，以培育更好的作物。

这项研究发表在期刊 *[Cell](https://www.cell.com/cell/pdf/S0092-8674(24)00182-X.pdf)* 和 *[Science](https://www.science.org/doi/10.1126/science.adk1075)* 中。
