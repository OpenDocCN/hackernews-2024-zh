# ego | Y Combinator 的高级游戏开发人员（Unreal Engine）

> 来源：[https://www.ycombinator.com/companies/ego/jobs/2myUIDW-senior-game-developer-unreal-engine](https://www.ycombinator.com/companies/ego/jobs/2myUIDW-senior-game-developer-unreal-engine)

# **关于我们**

Ego 正在构建一个 AI-native 游戏引擎和平台，用户可以使用自然语言创建和共享 3D 动画角色、世界、游戏和脚本交互。我们首先建立了一个类似于 Valheim 和 Minecraft 的生存游戏，并带有 Animal Crossing/Sims 风格的村庄。所有 NPC/村民将由 LLM 代理驱动，他们将拥有自己的生活、个性和日常活动。该游戏将允许您在玩游戏时使用自然语言进行游戏修改，而无需编码。

我们正在寻找对玩游戏和制作游戏充满激情的人，能快速交付，喜欢尝试新技术，并在游戏开发团队中（包括游戏开发人员、设计师和技术艺术家）表现出色的人。如果您喜欢制作游戏的感觉，推动游戏技术的边界，并且在快节奏的创业环境中蓬勃发展，我们希望能听到您的声音！

如果您想了解更多关于 ego 的信息，请随时查看我们的 YC 启动页面：[https://www.ycombinator.com/launches/KeD-ego-an-ai-native-3d-simulation-engine-getting-us-closer-to-the-matrix-one-pixel-at-a-time](https://www.ycombinator.com/launches/KeD-ego-an-ai-native-3d-simulation-engine-getting-us-closer-to-the-matrix-one-pixel-at-a-time)
