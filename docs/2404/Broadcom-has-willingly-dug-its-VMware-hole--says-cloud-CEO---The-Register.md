# Broadcom CEO表示，Broadcom自愿挖了VMware的坑。 • The Register

> 来源：[https://www.theregister.com/2024/04/08/broadcom_vmware_civo/](https://www.theregister.com/2024/04/08/broadcom_vmware_civo/)

Broadcom收购VMware后的方向引起了很多争议 - 大部分发生的事情都印证了Virtzilla客户在交易完成前表达的担忧。

总部位于英国的云计算公司Civo的CEO Mark Boost在接受*The Register*采访时表示，Broadcom的策略似乎是将大多数VMware客户抛诸脑后，而试图保留最大的客户以获取利润。您可以在下方观看完整的采访。

"非常清楚……重点是只关注600个账户，我记得。另外提到还有6,000个客户希望维持下去，" Boost告诉我们说。"特别提到会有十万客户随着时间流逝逐渐流失。"

许多客户对[强制捆绑定价](https://www.theregister.com/2024/04/02/vmware_cloud_foundation_updates_coming/)表示担忧，Boost说这导致了显著的价格上涨。

"我们已经听说价格上涨了10倍，" Boost指出。"通常我们听到的是，从三到五倍的平均价格涨幅。"

"时间会证明" Broadcom能否摆脱这个困境，Boost表示，"但其中很多都是有意为之。" ®
