# 美国宇航局的旅行者1号重新开始向地球发送工程更新

> 来源：[https://www.jpl.nasa.gov/news/nasas-voyager-1-resumes-sending-engineering-updates-to-earth](https://www.jpl.nasa.gov/news/nasas-voyager-1-resumes-sending-engineering-updates-to-earth)

所以，他们设计了一个计划，将受影响的代码分成几个部分，并将这些部分存储在FDS的不同位置。为了使计划成功，他们还需要调整这些代码部分，以确保它们仍然作为一个整体运行。还需要更新FDS内其他部分对该代码位置的引用。

团队首先确定了负责打包航天器工程数据的代码。他们在4月18日将其发送到FDS内的新位置。从地球到旅行者1号的无线电信号需要大约22个半小时，它距离地球超过150亿英里（240亿公里）。信号返回地球也需要22个半小时。当任务团队在4月20日收到航天器的回应时，他们发现修改成功了：五个月来首次能够检查航天器的健康状态。
