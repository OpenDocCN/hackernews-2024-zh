# 两名旧金山湾区铁路工人因在火车站内建造秘密公寓而被指控

> 来源：[https://therealdeal.com/sanfrancisco/2024/04/06/two-bay-area-railway-workers-charged-for-building-secret-apartments-inside-train-stations/](https://therealdeal.com/sanfrancisco/2024/04/06/two-bay-area-railway-workers-charged-for-building-secret-apartments-inside-train-stations/)

两名旧金山湾区的铁路工人在火车站内建造两个非法公寓，给“交通导向开发”带来了新的含义。

圣马特奥检察官声称，两名前加州火车官员在旧金山南部的半岛火车站内建造了两个秘密的办公室转公寓，[据圣何塞水星报报道。](https://www.mercurynews.com/2024/03/30/crime-or-peak-urbanism-yimbys-cant-help-but-applaud-the-caltrain-officials-who-built-secret-apartments-inside-stations/)

前加州火车副总监约瑟夫·纳瓦罗和承包商塞思·安德鲁·沃登[面临重罪指控](https://www.mercurynews.com/2024/03/28/ex-caltrain-employee-contractor-charged-with-building-secret-homes-at-train-stations/)，他们滥用公共资金在历史悠久的伯灵格姆和米尔布雷站内建造了迷你公寓。两个设施均未对公众开放。

据法院记录显示，61岁的沃登是圣地亚哥县奥申塞德居民，于3月27日被提审并保释。据媒体报道，他声称自己无罪。

66岁的纳瓦罗，曾是旧金山湾区居民，现居住于宾夕法尼亚州的纽镇，计划于4月29日被提审。

运输工人在2020年发现了沃登的米尔布雷公寓，但直到2022年匿名举报后才发现了纳瓦罗在伯灵格姆站内的秘密藏身之处。两人均被解雇。

在开发者可能轻松花费超过$900,000来建造一单元经济适用房的地区，他们所犯的罪行在一定程度上得到了赞赏。

一些社交媒体上的人表示，在像伯灵格姆这样建于1894年并被加利福尼亚州指定为历史地标的历史悠久的火车站内居住有其独特的吸引力。

“如果这不是在交通旁边建造更多住房的案例，那我不知道还有什么。”社交媒体网站X的一位用户写道，以前称为Twitter。

许多人对沃登和纳瓦罗印象深刻，他们分别在米尔布雷单位花费了$8,000，在伯灵格姆单位花费了$42,000，成功将办公室廉价改造为住房。

“雇佣那个人来建造住房”，东帕洛阿尔托的招聘人员马克·迪南写道。“公寓要价$42k！”

纳瓦罗据称仅花费了$42,000安装了小厨房、淋浴、水暖和安全摄像头，检察官说。

这对被指控试图低调行事，确保所有发票都未超过$3,000，否则将触发Caltrain和雇佣Worden的TransAmerica Services公司的进一步授权。

在加利福尼亚州建房成本**昂贵至极**，除了高昂的建筑成本外，影响费用和许可流程也可能高达数千美元，这些费用根据《水星新闻》报道，Worden 和 Navarro 不会支付。

此次犯罪也突显了旧金山湾区[超级通勤者的困境](https://bayareaequityatlas.org/indicators/extreme-commuting)。

地方检察官史蒂夫·瓦格斯塔夫告诉《旧金山标准报》称，这些公寓对于两名前铁路工人来说是“方便之举”。他说：“他们觉得旧金山湾区的[通勤]确实很糟糕。”
