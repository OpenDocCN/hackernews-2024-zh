# 雀巢在贫困国家销售的婴儿奶粉中添加糖报告发现 | 全球发展 | 卫报

> 来源：[https://www.theguardian.com/global-development/2024/apr/17/nestle-adds-sugar-to-infant-milk-sold-in-poorer-countries-report-finds](https://www.theguardian.com/global-development/2024/apr/17/nestle-adds-sugar-to-infant-milk-sold-in-poorer-countries-report-finds)

世界上最大的消费品公司雀巢在许多贫困国家销售的婴儿奶粉和谷物产品中添加了糖和蜂蜜，这与旨在预防肥胖和慢性疾病的国际准则相抵触，一份报告指出。

瑞士调查组织公众眼镜的活动人士将瑞士跨国公司在亚洲、[非洲](https://www.theguardian.com/world/africa)和拉丁美洲销售的婴儿食品样品送往比利时实验室进行测试。

结果显示，在Nido（一款面向一岁及以上婴儿的后续奶粉品牌）和Cerelac（一款面向六个月到两岁儿童的谷类食品）的样本中，发现添加了蔗糖或蜂蜜。

在雀巢主要的欧洲市场，包括英国，婴幼儿配方食品中不含添加糖。尽管一些针对较大幼儿的谷类食品中含有添加糖，但是针对六个月到一岁宝宝的产品却没有添加糖。

公共眼组织的农业和营养专家Laurent Gaberell表示：“雀巢必须结束这种危险的双重标准，停止在全球范围内所有针对三岁以下儿童的产品中添加糖。”

肥胖问题在[低收入和中等收入国家](https://www.who.int/news-room/fact-sheets/detail/obesity-and-overweight)日益严重。根据世界卫生组织的数据，自2000年以来，非洲五岁以下超重儿童的数量增加了近23%。全球有超过10亿人口肥胖问题。

在任何国家，消费者要判断产品是否含有添加糖，以及含量多少，仅依靠包装上印刷的营养信息并不总是容易。标签通常将牛奶和水果中天然存在的糖分与任何添加糖一并列出。

欧洲地区的[世界卫生组织指南](https://iris.who.int/bitstream/handle/10665/364678/WHO-EURO-2022-6681-46447-67287-eng.pdf?sequence=1&isAllowed=y)指出，三岁以下儿童食品不得添加糖或甜味剂。尽管没有为其他地区专门制定指南，但研究人员称欧洲文件[同样适用于其他地区](https://accesstonutrition.org/app/uploads/2023/12/COMMITs-Adapted-WHO-Europe-Nutrient-Profile-Model-for-Commercially-Produced-Complementary-Foods-1-1.pdf)。

在英国超市销售的Cerelac产品。这款婴儿谷类食品的全球销售额超过12亿美元。照片：秀宝/Alamy

英国建议儿童四岁以下避免食用添加糖的食物，因为可能导致体重增加和牙齿蛀牙，详情请见[此处链接](https://www.nhs.uk/live-well/eat-well/food-types/how-does-sugar-in-our-diet-affect-our-health/#:~:text=Children%20aged%204%20to%206,what%20to%20feed%20young%20children.)。美国政府指南建议两岁以下儿童避免摄入含有添加糖的食物和饮料，详情请见[此处链接](https://www.cdc.gov/nutrition/infantandtoddlernutrition/foods-and-drinks/foods-and-drinks-to-limit.html)。

-   [在其报告中](https://stories.publiceye.ch/nestle-babies)，与国际婴儿食品行动网络合作撰写的Public Eye表示，市场研究公司Euromonitor International的数据显示，Cerelac的全球零售销售额超过10亿美元（8亿英镑）。低收入和中等收入国家的销售额最高，其中40%的销售额来自巴西和印度。

世界卫生组织的医务官员尼格尔·罗林斯博士表示，这些发现代表了“无法证明的双重标准”。

研究人员发现，面向六个月及以上婴儿的饼干味谷物早餐在塞内加尔和南非每份含6克添加糖。但同一产品在[瑞士](https://www.theguardian.com/world/switzerland)却没有添加。

对在印度销售的Cerelac产品的测试显示，平均每份含添加糖超过2.7克。

在巴西，Cerelac以Mucilon的名字广为人知，八种产品中有两种无添加糖，但其他六种每份含近4克。在尼日利亚，一种测试产品每份含高达6.8克。

与此同时，对全球零售销售额超过10亿美元的Nido品牌产品进行的测试显示糖含量存在显著差异。

在菲律宾，针对幼儿的产品不含添加糖。然而，在印度尼西亚，以Dancow销售的Nido婴儿食品产品中，每100克产品含约2克蜂蜜形式的添加糖，或每份0.8克。

在墨西哥，三种适合幼儿的Nido产品中有两种不含添加糖，但第三种每份含1.7克。报告称，南非、尼日利亚和塞内加尔销售的Nido Kinder 1+产品每份几乎含1克。

雀巢发言人表示：“我们相信我们早期童年产品的营养质量，并优先使用适合儿童生长和发育的高质量成分。”

她说，在婴儿食品的“高度监管”类别中，雀巢始终遵守“包括标签要求和碳水化合物含量阈值在内的当地法规或国际标准”，并声明其产品中的总糖分，包括来自蜂蜜的糖分。

食谱的变化取决于包括法规和当地原料供应在内的因素，她说。

她说，公司在过去十年中全球范围内将婴儿谷物组合中的添加糖总量减少了11％，并继续改良产品以进一步减少糖分。

她补充说，蔗糖和葡萄糖浆正在全球面向幼儿的“成长牛奶”中逐步淘汰。

本文经过2024年4月17日修改，以更正由公众之眼提供的销售数据。尼多的全球销售额超过10亿美元，而不是之前版本中所说的41亿美元，而且Cerelac为10亿美元，而不是之前版本中所说的12亿美元。
