# HashiCorp加入IBM以加速多云自动化

> 来源：[https://www.hashicorp.com/blog/hashicorp-joins-ibm](https://www.hashicorp.com/blog/hashicorp-joins-ibm)

今天，[我们宣布](https://www.prnewswire.com/news-releases/ibm-to-acquire-hashicorp-inc-creating-a-comprehensive-end-to-end-hybrid-cloud-platform-302126646.html) HashiCorp已签署协议，被IBM收购，以加速我们几乎12年前开始的多云自动化旅程。我对这一消息非常兴奋，并相信这是一个机会，通过IBM的支持进一步推动HashiCorp的使命并扩展到更广泛的受众。

当我们2012年创办公司时，云计算领域与今天大不相同。Mitchell和我最初作为业余爱好者接触公共云，尝试初创公司的想法，后来作为专业开发人员构建关键任务应用程序。这一经历清楚地表明，自动化对于规模化管理云基础设施是绝对必要的。公共云的变革性影响也清楚地表明，我们最终会生活在一个多云世界中。最后，很明显，这项技术的采纳将由我们的同行实践者推动，他们正在重新构想基础设施的格局。

我们创立了HashiCorp，旨在为实践者社区在多云世界中实现云自动化。今天，我为我们共同取得的一切感到非常自豪。我们的产品每年被我们充满热情的用户社区下载数亿次。每年，我们认证数以万计的新用户使用我们的产品，他们每天都在使用我们的工具来管理他们的应用程序和基础设施。

我们与成千上万的客户合作，包括全球数百大组织，支持他们在多云环境中的旅程。他们信任我们处理关键任务应用程序和核心基础设施。基础设施最令人满意的一点是在全球范围内悄然支持令人难以置信的应用程序。我们以能够[让数百万玩家一起游戏](https://www.hashicorp.com/resources/how-we-used-the-hashistack-to-transform-the-world-of-roblox)，[为咖啡订购提供忠诚积分](https://www.hashicorp.com/resources/starbucks-secrets-at-the-retail-edge-with-hashicorp-vault)，连接自动驾驶汽车，并每天保护数万亿美元的交易而自豪。这就是为什么我们始终相信基础设施推动创新。

自我们成立以来，HashiCorp 产品组合已显著增长。我们继续与社区和客户合作，识别他们在采用多云基础设施和过渡到零信任安全方法中面临的挑战。这些挑战反过来成为我们在HashiCorp Cloud Platform上构建新产品和服务的机会。

这就是我为今天的公告感到兴奋的原因所在。我们将继续作为HashiCorp构建产品和服务，并作为IBM软件部门的一部分运营。通过加入IBM，HashiCorp产品可以面向更广泛的受众，使我们能够服务更多用户和客户。对于我们的客户和合作伙伴来说，这种结合将使我们能够比作为独立公司更进一步。

围绕HashiCorp的社区是我们成功的关键。我们将继续深度投入到今天与HashiCorp合作的用户和合作伙伴社区中。此外，通过IBM和Red Hat社区的规模，我们计划显著扩展我们的影响力和覆盖范围。

尽管我们已经有超过十年的HashiCorp历史，我们相信云采用的早期阶段才刚刚开始。通过与IBM的合作，我们有机会帮助更多客户更快地实现这一目标，加速产品创新，并继续扩大我们的实践者社区。

我非常感激我们的用户、客户、员工和合作伙伴的支持。建设HashiCorp到今天这一点是一段非常有回报的旅程，我期待着迎接下一个篇章。
