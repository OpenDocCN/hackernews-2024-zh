# 媒体正在消失。乔丹·梅赫纳正在努力拯救它。 - 华盛顿邮报

> 来源：[https://www.washingtonpost.com/entertainment/video-games/2024/04/01/jordan-mechner-prince-persia-karateka/](https://www.washingtonpost.com/entertainment/video-games/2024/04/01/jordan-mechner-prince-persia-karateka/)

弗朗西斯·梅赫纳记得一切，包括他的家人如何利用希特勒的水彩画来拯救他们的生命。他坚持让儿子乔丹把所有的事情都写下来。

“我是唯一一个还记得这些人和地方是谁的人了”，父亲在2015年说道，指着一幅家庭肖像，上面有叔叔“乔治”的画像，他多年前购买了希特勒上台前的画作。这些画作被用来换取纳粹占领下的奥地利的签证。“没有背景的图片毫无意义。”
