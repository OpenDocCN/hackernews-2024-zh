我们的目标是创建一个包含所有索尼 VAIO 型号文档、提供驱动程序和恢复光盘下载的 Wiki。要了解更多信息，请访问我们的 [关于页面](/index.php/VAIO_Library:About "VAIO Library:About")。希望您在本站访问愉快！

**请注意，本站仍在开发中。请查看 [开发状态](/index.php/Development_Status "Development Status")。**

恢复媒体光盘可在 [恢复光盘](/index.php/Recovery_Discs "Recovery Discs") 页面获取。

欢迎支持 VAIO 图书馆，请查看我们的 [捐赠页面](/index.php/Donate "Donate")！即使是小额捐款也将不胜感激。
