# 欢迎来到用户敌意的黄金时代 - The Atlantic

> 来源：[https://www.theatlantic.com/technology/archive/2024/04/roku-tv-ads-patent/678041/](https://www.theatlantic.com/technology/archive/2024/04/roku-tv-ads-patent/678041/)

智能电视变得过于智能会发生什么？答案似乎是更多的侵入式广告。

上周，技术和娱乐记者Janko Roettgers揭露了Roku去年八月提交的一个[反乌托邦专利](https://www.lowpass.cc/p/roku-hdmi-ad-insertion-patent?utm_source=www.lowpass.cc&utm_medium=newsletter&utm_campaign=roku-explores-taking-over-hdmi-feeds-with-ads)，该公司是全球数千万人使用的电视和流媒体设备制造商。文件详细介绍了“HDMI定制广告插入”的计划，这将允许Roku制造的电视通过HDMI端口监视视频信号——用户可能会连接游戏机、蓝光播放器、有线电视盒或甚至另一个流媒体设备——然后在内容暂停时注入定向广告。这将是Roku监控潜力的极端扩展：目前该公司无法看到用户在从其专有流媒体平台切换到其他操作时可能正在做什么。显然，这是一个问题，因为Roku错失了赚钱的机会！

尽管专利可能永远不会实现（Roku的一位发言人告诉我，公司目前没有计划在任何产品中加入HDMI广告插入），但它揭示了消费者硬件中一种令人沮丧的最新趋势。互联网连接的产品在购买后可以以让用户感到侵入或甚至敌对的方式进行转变。来自Roku的另一个例子：就在上个月，该公司向用户展示了其服务条款的更新，要求他们进入一项预仲裁流程，这将使起诉公司变得更加困难。一方面，这并不算不寻常——应用程序经常强迫用户接受服务条款的更新才能继续使用。但另一方面，由于法律协议的原因，完全不能使用电视机似乎令人不悦：“在我按下‘同意’之前，我的电视实际上被扣押并变得无用，”一位Roku客户在Reddit上发帖说道。 “我甚至无法更改HDMI输入。”

Roku的一位发言人确认，用户必须同意最新的条款才能使用该公司的服务，但表示客户可以选择通过信件寄送到公司总法律顾问处退订（尽管这个时间窗口在3月21日已关闭）。发言人告诉我：“和许多公司一样，Roku定期更新其服务条款。在更新时，我们会采取措施确保客户知晓变更。”

以前，电视是电视，广告是广告，计算机是计算机。现在它们被互联网和机会主义公司混合在一起，形成了一种不祥的混合物，这些公司开发了[“自动内容识别”](https://themarkup.org/privacy/2023/12/12/your-smart-tv-knows-what-youre-watching)系统。这些系统收集关于个体观看习惯的详细数据，并记录到数据库中，然后用于服务广告或出售给感兴趣的方。比如[政治家](https://www.theatlantic.com/technology/archive/2019/11/smart-tvs-collect-data-political-advertising-use/601381/)。日常电子设备的慢速监视殖民化被免费互联网服务所规范，这些服务让人们适应了我们的个人信息实际上是业务成本的心态：电视变得更便宜了，而[现在我们用数据来支付](https://www.theatlantic.com/technology/archive/2023/01/smart-tvs-sony-lg-cheap/672614/)。这不仅仅是一笔糟糕的交易；从根本上讲，这不应该适用于*人们用金钱购买的硬件和软件*。一个Roku客户最近在X上恰如其分地总结了这种沮丧：“我们放弃了上帝的光（阴极射线和磷光）换来了这些。”

这种现象与另一个现代关注点发生了碰撞——作家和活动家科里·多克托罗生动地称为“恶化化”（[链接](https://www.theatlantic.com/ideas/archive/2023/02/netflix-streaming-password-sharing-family-household/672967/)）。这个术语反映了一种普遍的文化感觉，即事情变得越来越糟，我们使用的数字产品实际上被用来反对我们。例如：除了其广告植入的流媒体设备外，Roku还为智能手机提供了一个遥控应用。在一个Reddit [帖子](https://www.reddit.com/r/Roku/comments/1bg956e/ads_in_roku_remote_app/?utm_source=share&utm_medium=web3x&utm_name=web3xcss&utm_term=1&utm_content=share_button)中，一个用户附上了公司在发布后不久插入应用程序中的微妙广告模块的截图，轻微地使您在电视屏幕上导航成为“恶化化”。一个评论员写道：“等着看我们在使用遥控器之前不得不观看1分钟的视频广告。”另一个回答说：“别给他们任何主意。”

[科里·多克托罗：这就是Netflix认为你的家庭是什么](https://www.theatlantic.com/ideas/archive/2023/02/netflix-streaming-password-sharing-family-household/672967/)

医生的放狗等理论涉及到企业的商业模式欺骗性的转变，企业在吸引了人们用以提供漂亮、免费功能后，然后开启网络广告喷涌。Roku符合这一模式框架。该公司在过去一年里在物理设备上亏损4.4亿美元，但在广告和产品的服务产品上赚了近16亿美元。事实上，Roku就是一个广告公司，这与像谷歌和Facebook等公司类似。市场营销依赖于受限制的观众：广告插播、高速公路上你无法避免看到的广告牌等。

其他公司已经在他们的设备中加入了“数字版权管理”或DRM限制，这阻止人们尝试修改自己拥有的设备。我去年写过关于我使用信用卡购买墨盒订阅后，HP远程打坏了我的墨盒打印机的事情。我的打印机**原本**有墨水（我为此付费了），但我无法使用它。它感觉就像勒索一样。限制性权利使用发生在各个方面——只在特定平台上播放的歌曲、电影和有声读物，以及大型、昂贵的实体技术产品，如汽车。如今，所有权的概念现在让人感到疑惑。如果HP可以禁用我的打印机，如果Roku可以关掉我的电视，如果特斯拉可以通过远程方式改变我的汽车电池的生命，那么我所拥有的设备实际上**是我的**吗？

[阅读：我的打印机勒索我]((https://www.theatlantic.com/technology/archive/2023/02/home-printer-digital-rights-management-hp-instant-ink-subscription/672913/)

答案是：不完全是这样，或者不如以前那样了。对设备的实际控制权的丧失，以及我们每天使用的商品质量下降，共同造成了消费者普遍的消极情绪，这种情绪已经开始超越数码领域。大量生产并亚马逊化的低价消费品与波音因服务于股东利益提高航空制造质量而减少的产品质量不同，也与电视在展示广告时会产生**震惊**效果（原始链接在Reddit上，链接指向[不再购买Roku电视](https://www.reddit.com/r/Roku/comments/l5xl5l/pretty_sure_this_is_the_last_roku_tv_i_buy_after/?utm_source=share&utm_medium=web3x&utm_name=web3xcss&utm_term=1&utm_content=share_button)）有所不同；然而，这些看似无关的事物开始给人一种*联系*的感觉——这个问题可以从铺天盖地的、企业实体日益不名誉的行为描述一般特性中定义，它是一种衰败感、一种不被尊重的感觉。

在某些领域，这意味着服务质量下降，以换取更高的利润；在其他领域，这感觉就像是被迫期望和接受无论可以怎样变现都会变现，而不管消费者体验是否受到影响。人们在各个地方都能感受到这一点。他们在好莱坞感受到，正如记者Richard Rushfield [最近](https://theankler.com/p/congrats-jay-penske-you-made-the)所说，娱乐行业充斥着那些认为交易比观众更重要的高管——最终认为消费者“别无选择，只能购买最新的*碟中谍*或*速度与激情*电影票——因为他们一直这样，我们拥有他们，所以他们会看我们要他们看的东西。”人们在意想不到的地方感受到，比如职业高尔夫球：最近，我惊讶地读到[Fried Egg Golf通讯](https://mailchi.mp/bbe4a80b96b8/scotties-private-island-nbc-puts-on-a-show)比较了NBC Sports疲弱的PGA巡回赛广播和波音公司的持续混乱。“目前人们普遍缺乏士气吗？”作者写道。“还有人对自己的工作感到自豪吗？或者我们只是让所有领域的质量因为成本削减而遭受损失？”

这最后两个例子本质上并非属于Doctorow式的：它们只是人们认为因为公司认为消费者会接受劣质产品或者他们别无选择而使事情变得更糟的例子。在这个意义上，Doctorow的*恶化*可能超越了其最初的数字意义。像*末日浏览*一样，它为时代精神提供了语言。去年在我访问他关于我的打印机敲诈事件时，Doctorow告诉我：“问题在于这一切都在恶化，而不是改善。”他说的是公司将消费者锁定在令人沮丧的生态系统中，同时也涉及到消费者的沮丧感。“我们最不希望的是一切都变成喷墨打印机化，”他说。

我意识到Doctorow的观察实际上是我和许多[其他人](https://twitter.com/search?q=https%3A%2F%2Fwww.lowpass.cc%2Fp%2Froku-hdmi-ad-insertion-patent&src=recent_search_click)在线上对一个可能不会实现的理论专利感到如此激动的*实际*原因。与你的电视进行人质谈判很烦人——甚至是令人愤怒的——但这只是一个小小的羞辱。更大的是一种逐渐增长的感觉，即通过微妙的技术背叛，我们购买的东西变得不再可靠已成为标准做法。一点监视在这里，一点强制仲裁在那里。总结起来，真正的问题变得存在主义。看起来墨盒式打印机正在获胜。
