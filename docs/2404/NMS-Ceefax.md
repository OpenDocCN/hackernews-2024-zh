# NMS Ceefax

> 来源：[https://www.nathanmediaservices.co.uk/ceefax/](https://www.nathanmediaservices.co.uk/ceefax/)

由天才**[Alistair Cree](https://zxnet.co.uk/)**创作的互动查看器，又名**[ZXGuesser](https://twitter.com/ZXGuesser)**。

## 如何使用

还记得电视文本吗？这完全相同。

每一页都分配了一个三位数的编号 - 你会看到导航行，上面列有页面描述，后面跟着一个数字（例如，“体育头条 302”）

使用屏幕遥控器上的数字键（或者如果您使用PC，则使用键盘）输入一个数字。等待页面加载时，顶部行变为绿色。

您还可以使用“频道上/下”按钮一次移动一页。
