# 亚马逊如何在仅两年内建立了无与伦比的电动车充电网络 | 进步手册

> 来源：[https://theprogressplaybook.com/2024/04/15/how-amazon-built-a-massive-ev-charging-network-in-just-two-years/](https://theprogressplaybook.com/2024/04/15/how-amazon-built-a-massive-ev-charging-network-in-just-two-years/)

*作者：马特·戴，彭博社*

亚马逊位于华盛顿州枫谷（Maple Valley）的仓库专为提速而建。夜间，大型卡车驶入一端卸下箱子和填充信封——有些来自附近更大的仓库，经过短途驾驶，有些则是从货机货舱中下来。等待的员工扫描、分类并装载它们到滚动货架上。

每天早晨7点之前，许多货架被推出到四条油漆线划定的车道上的数十辆货车前。这就像是一场一级方程式比赛的起跑线，但对于每小时22美元的送货司机来说，他们要将洗发水瓶和电池包送到西雅图郊区的门前。

它们的路线是一个长途旅行的最后一步，可以将产品运送数千英里，是亚马逊承诺在未来几十年内消除大部分碳排放的重要来源。

解决方案就在街对面的停车场：309个西门子电动车（EV）充电器，为Rivian Automotive制造的送货车辆提供动力。无尾气排放进行交付，并增加电动车队的规模，是亚马逊可以从其运营中清除碳的最直接方式之一。

仅仅两年多的时间里，亚马逊在美国约120个仓库安装了超过17,000个充电器，使这家零售巨头成为美国最大的私人电动车充电基础设施运营商。“我们已经找到了前进的路径，”负责亚马逊末端交付车辆车队的汤姆·切姆帕南尼卡尔（Tom Chempananical）说道。

联合包裹服务公司和联邦快递等物流公司制定了自己雄心勃勃的电动车目标，尽管许多公司未能有效地减少电子商务交付带来的增长排放。与此同时，亚马逊放弃了到2030年将一半交付物品零碳排放的承诺，称这一倡议被更广泛的气候目标所取代。但该公司在向电动车的过渡方面比大多数竞争对手走得更远、更快。了解亚马逊面临的挑战，这家以不遗余力地满足紧迫交付期限而闻名的公司，有助于为其他行业的公司制定消除自身碳成本的路线图。

“亚马逊的规模至关重要，”爱迪生电气学会（Edison Electric Institute）交通总监凯伦·谢夫特（Kellen Schefter）说道，该协会是一家投资者拥有的公用事业贸易团体，致力于将亚马逊与电力公司连接起来。“如果亚马逊能够证明他们既实现了气候目标，又实现了包裹交付目标，我们可以展示这一切确实可行。”

Amazon 还有很长的路要走。这家总部位于西雅图的公司表示，其2022年的运营排放约为71百万公吨二氧化碳当量，比杰夫·贝佐斯2019年承诺公司最终将停止对温室气体排放做出贡献时增加了近40%。亚马逊的许多排放来自各种活动 - 空运、海运、建筑和电子制造等等 - 这些活动目前或将来都没有明确的无碳替代方案。公司在减少长途卡车碳排放方面进展不大，这些排放往往集中在工业和郊区，而不是亚马逊电动送货车推出的大城市背景。

但亚马逊计划到明年通过购买太阳能、风能和其他无碳来源生产的电力量，来满足其运营所需的电力量。此外，亚马逊还支持了Rivian，并订购了10万辆定制送货车，其中已交付了13500辆，公司表示可以大幅减少其末端送货业务相关的排放。

要达到这一目标，亚马逊必须学会拿起电话打给电力公司。政府电力使用估计显示，一个位于工业区的10万平方英尺仓库可能需要大约50千瓦的电力，主要用于照明和空气循环。在停车场设置100个充电器可能需要10到20倍的电力。

“这里的不同之处在于这是一种新型的电力使用”，EEI的谢菲特说。“这在停车场需要非常大的电力需求。”如果该地区的输电线路有多余的容量，这种需求可以比较快速地得到满足。在电网的其他地区，升级可能需要数年时间。

亚马逊还学会了如何灵活和等待。该公司更喜欢可以像生产线一样运行的标准化流程。但在物理世界中，这种标准化流程在实际中的运作会有所崩溃，因为亚马逊的数百个末端送货仓库设计不同，停车场布局也不同，受当地不同的公用事业协议影响。

“我们需要为基础设施的准备时间准备多长时间，这有点令人惊讶”，领导亚马逊物流可持续团队的克里斯·阿特金斯说。

2020年，亚马逊与美国一些大型公用事业公司会面，这些公司询问亚马逊需要多少电力以及在哪里需要。伊利诺伊州最大的电力供应商Commonwealth Edison的代表也参加了会议。在疫情期间，这家公用事业公司在努力保证某些类型的新设备供应时，选择重新利用旧的变压器为亚马逊提供支持。埃克森公司旗下的这位大客户负责人Diana Sharpe表示：“为了确保我们有所需的设备并能够满足他们的时间要求，我们在内部做了一些非常有创意的工作。”

当Rivian在2022年春季开始大量投放货车时，ComEd已经为芝加哥普尔曼社区的一家亚马逊仓库输送了额外的电力。Rivian的首席执行官于7月份前来参加了一场剪彩仪式，宣布这些货车已上路。如今，ComEd为芝加哥地区的四个亚马逊仓库提供约1100个充电器的电力支持。

亚马逊学到的另一个教训是一个公司不太愿意谈论的：在初期阶段，绿色发展可能成本高昂。根据亚马逊部署的充电器类型（行业内几乎全部为称为Level 2的中端充电器），根据美国国家可再生能源实验室提供的成本估算，硬件成本可能在5000万到9000万美元之间。考虑到除插座和相关硬件之外的费用，例如在停车场挖掘铺设电线或设置电力面板和电柜，这些费用可能会翻倍。亚马逊拒绝就其电动车充电推动项目花费多少发表评论。

除了充电器的费用外，电动车车队运营商通常还需承担电力升级的费用。当公司像亚马逊那样请求增加电力容量时（Maple Valley仓库为其充电器提供了3兆瓦的电力），它们往往会为此付费，使公用事业公司能够为单一客户完成工作并获得全额赔偿。亚马逊表示，它支付的升级成本由公用事业公司确定，但在一些地方，这些升级符合公用事业公司将自行处理的标准服务电力。

当亚马逊下订单给Rivian时，参与该项目的人员预计，运行电动交付车队最终会比公司常规燃料车队更经济，后者由福特、奔驰集团和斯泰兰蒂斯等公司批量定购。目前尚不清楚亚马逊是否已经达到这一点，不过Chempananical表示，亚马逊对Rivian车辆的价格标签感到满意。“所有这些成本继续下降，”他说。“随着使用量的增加，需求增加，供应增加，效率提高，并继续推动到更好的位置。”

亚马逊还需要解决充电器共享的物流问题。在Maple Valley仓库，77辆电动货车可以从307个2级充电器中进行选择。但其他地点的充电器数量少于可用的货车。充电一辆货车可能需要几个小时，最初这造成了一些头痛。亚马逊最初要求管理货车车队和司机的分包商在夜间保持自己的员工轮换使用充电器。去年秋天，亚马逊将这项工作内部化，使得分包工人可以开车而不是看管充电器。“他们最终还是需要司机的，”Chempananical说道。

亚马逊的合同司机表示他们喜欢这些货车，这些车辆专为公司有时严苛的、每90秒包裹一次的路线和频繁的停车设计。雇佣司机的人——亚马逊的交付服务合作伙伴——则有一些抱怨。两个西海岸的交付服务提供商（要求匿名以保护他们与亚马逊的关系）表示，由于少数修车厂有授权进行Rivian货车的维修工作，车身工作的成本可能是传统车辆的两到三倍。备件也可能难以获得。

卡车运输对亚马逊和整个行业来说都是一个更大的挑战。汽车制造商在电动化轿车和轻型卡车方面的进展要远远超过从港口到仓库之间运输集装箱的拖车（参见特斯拉的半挂卡车，尽管数年前发布，但仍处于试生产阶段）。大眼睛的Rivian电动货车在西雅图、洛杉矶、纽约及其周边地区频繁穿梭，已经成为常见景象。但是，通常是在像宾夕法尼亚东北部或加利福尼亚内陆帝国等地贫困区承载大型仓库的社区，却尚未享受到电气化带来的同样好处。根据知情人士透露，在公司疫情后的成本削减中，亚马逊已推迟和搁置了一些与卡车运输和其他所谓的“中间里程”可持续性投资有关的计划。

Atkins驳回了这一批评，列举了亚马逊在这个领域所做的举措：购买使用压缩天然气的卡车，并投资于绿色氢能源制造商等其他燃料来源。他说：“我们做对事情很重要，而不是只是与错误的合作伙伴扩展规模。”

“它会到达那里的，”Chempananical谈到了亚马逊在“中间里程”投资方面的未来。“只是一个时间和方式的问题。”

在枫谷仓库，负责亚马逊送货服务的运输供应商太平洋交付与物流的贾斯汀·谢尔（Justin Shearer）向我们展示了他在这个设施内的工作区域。像许多亚马逊庞大的送货承包商一样，他还有另外一份工作。谢尔是一名商业渔民，早年间他还销售矿产和木材设备。他并不认为自己是环保主义者。但他说，不用上路运输石油产品的好处显而易见。

“建设这样的基础设施成本很高，”他说。“这种规模的建设还没有完成，但除非从某个地方开始，否则你永远不会变得更好。”
