# 互联网如何变得糟糕

*2024年4月19日*

最近，我更换了iPhone 8的电池（这是一款完美的智能手机；要和我争论吗？），希望能延长其使用寿命几年。然而，iStore服务中心要求在进行任何服务工作之前，按照[POPIA法案](https://popia.co.za)的要求擦除手机。这让人很恼火，但由于我备份良好，只是有点不方便，不是什么大问题。

一旦我把手机重新安装了所有的应用程序，我发现自己在浏览Hacker News，并跟随链接阅读了一个大型新闻网站上的一篇文章。到处都是广告。我忘记重新安装广告拦截器了。

在过去的十多年里，我一直使用广告拦截器上网，因此这次体验让我大开眼界。由于各种弹窗广告、动画横幅、内容中的巨大分隔线、附近美女等等，我简直无法看完文章。而这种情况并非个例。我通过Hacker News访问的大多数网站都从头到脚都是广告。

"Enshittification"是由[Cory Doctorow](https://en.wikipedia.org/wiki/Cory_Doctorow)创造的一个术语，准确描述了这里发生的情况。在线服务质量的下降是为了获利。这个词还用来描述那些提供免费服务但在锁定了客户基础后开始收费的公司（我在看你，[Heroku](https://devcenter.heroku.com/changelog-items/2461)）。

如果一个产品的商业模式要求他们在吸引用户后降低产品质量，我宁愿这种产品从一开始就不要存在。因为不幸的是，一个补贴的产品会排挤那些基础却更加可持续的产品。创造一个占据市场份额然后改变规则的做法，实际上损害了他们的用户和整个市场。

另一方面，我喜爱互联网。这是一个人们分享想法和数字聚集的地方。我将其视为散布在网络黑暗中的篝火。目前，我常去一个位于南非的开发者Slack频道，那里我可以畅所欲言。我也花了相当多的时间阅读有关人们搬迁到新城市、有趣的副业项目、思考和沉思、生活片段等内容，以及更多的内容，都可以在[Bear博客发现频道](https://bearblog.dev/discover)上找到。

我觉得这些小众社区在互联网广告商业化普及之后要有机地偶然发现变得更加困难。任何主要搜索引擎的第一页，无论搜索什么，都会被SEO塞满，这可能是由AI编写的，或者与AI编写的内容几乎无法区分。

现在很多人选择在Reddit上搜索而不是Google，原因有其道理。由于在Reddit上更难以变现内容，因此减少了填充SEO乱七八糟内容以捕捉关键词排名的动机。尽管如此，即使是Reddit的使用体验也在恶化。每次页面加载都试图让我安装应用，他们杀死了大多数第三方应用，而且在移动设备上几乎无法使用old.reddit.com。因此，即使是传统搜索引擎的“替代品”也变得越来越糟糕。

这里是我试图用一个漂亮的小结来结束这篇文章的部分，然后说点像“我们所需要做的就是X”这样的话（该死，X现在是一种东西……所以也许是Y）。但我做不到。我没有解决方案。互联网是分散的自由竞技场，而且有明显的经济激励来被利用。我预见社区会变得更加封闭，我预测未来内容网站和信息流会因策展和管理而生死攸关。我将一直坚持使用RSS直到最后。
