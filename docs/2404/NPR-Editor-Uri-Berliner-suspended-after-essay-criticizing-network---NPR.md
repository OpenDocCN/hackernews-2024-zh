# NPR编辑Uri Berliner因批评网络而被停职：NPR

> 来源：[https://www.npr.org/2024/04/16/1244962042/npr-editor-uri-berliner-suspended-essay](https://www.npr.org/2024/04/16/1244962042/npr-editor-uri-berliner-suspended-essay)

****切换说明****Uri Berliner

NPR在Uri Berliner发表一篇批评网络失去公众信任的文章后，暂停他五天不发薪酬，并在一档播客中解释了他的观点。

Uri Berliner

NPR已经正式惩罚了Uri Berliner，这位高级编辑一个星期前公开辩称网络通过以刚性进步思维方式处理新闻故事而“失去了美国的信任”。

Berliner的五天无薪停职，从上周五开始，此前未曾报道。

然而，这家公共广播网络在其他方面也在应对[Berliner为在线新闻网站The Free Press撰写的文章](https://www.thefp.com/p/npr-editor-how-npr-lost-americas-trust)所带来的后果。这激怒了他的许多同事，导致NPR领导宣布对网络报道进行每月内部审查，并为包括前总统唐纳德·特朗普在内的保守派和共和党批评者提供了新的弹药。

保守派活动人士[Christopher Rufo](https://www.politico.com/news/magazine/2024/01/03/christopher-rufo-claudine-gay-harvard-resignation-00133618)现在也在瞄准NPR的新任首席执行官Katherine Maher，因为她加入该网络前几年在社交媒体上发布的信息。其中包括[2020年一条称特朗普为种族主义者的推文](https://twitter.com/krmaher/status/1260960520586145797)，以及在那一年社会正义抗议期间试图淡化骚乱的另一条推文。Maher在上个月加入了NPR这个新闻机构 —— 这是她的[第一个新闻机构职位](https://www.npr.org/2024/01/24/1226035539/npr-ceo-katherine-maher-wikimedia)。

周一关于她发表的信息声明中，Maher赞扬了NPR记者的诚信，并强调了他们报道的独立性。

“在美国，每个人作为私人公民都有言论自由，”她说。“重要的是NPR的工作和我的承诺作为其CEO：公共服务，编辑独立性以及服务所有美国公众的使命。NPR是独立的，不受任何政党控制，也没有商业利益。”

网络指出，“CEO不参与编辑决策。”

周一后晚些时候在采访中，Berliner说，社交媒体的帖子表明Maher几乎不可能成为最适合领导该组织的人。

在正式谴责信中，NPR没有提到柏林上周二晚上出现在克里斯·库默的《新闻国家》节目中，NPR事先同意了这一点。（NPR的首席传播官告诉柏林，要专注于自己的经历，不要分享专有信息。）NPR的信函也没有提到[他对《纽约时报》的言论](https://www.nytimes.com/2024/04/11/business/media/npr-criticism-liberal-bias.html)，该文章于周四午后发布，谴责信发出不久。

柏林说，他没有得到NPR的许可来接受我这篇报道的采访，但他并不担心后果：“与NPR记者交谈并因此而被解雇将是非同寻常的，我想。”

柏林是NPR商业台的成员，我也是，他曾帮助编辑了我的许多报道。他没有参与本文的准备工作，也没有在文章发布之前看过。

在谴责柏林的时候，NPR表示他还公开发布了关于听众人群特征的专有信息，这些信息是他们认为机密的。他称那些数字“本质上是营销材料。如果它们真的很好，他们可能会分发并向世界发送。”

### 新闻室内部的愤怒和背叛感情

他的文章和后续公开言论引起了NPR内部的深深愤怒和失望。同事们认为柏林选择了符合他论点的例子，并质疑他的描述准确性。他们还指出，他并没有征求涉及到的记者的评论。

*早间版*主持人米歇尔·马丁告诉我，网络中一些同事与柏林分享了他们的担忧，认为报道经常通过意识形态或理想主义的棱镜呈现，可能会疏远听众。

“解决这个问题的方法是通过培训和指导”，Martin说道，她本身在这家网络工作了将近二十年，还曾为《华尔街日报》和ABC新闻报道。“不是通过毁掉这个地方，糟蹋你的同事，在那些实际上并不关心的人面前全面公开。”

几位NPR记者告诉我，他们不再愿意与柏林一起工作，因为他们不再相信他会保密他们在工作中关于报道的内部想法。

“新闻编辑室建立在信任基础上”，[NPR政治记者丹尼尔·库尔兹勒本（Danielle Kurtzleben）在上周推特上发表推文](https://twitter.com/titonka/status/1778461456410800339)，没有提到柏林的名字。“如果你通过去另一个渠道糟蹋你的同事（在记者工作上表现糟糕，从那个角度来看），违背了所有人的信任，我不知道你现在怎么做你的工作。”

柏林否认了这种批评，称他的文章或后续言论没有泄露关于报道的私人观察或争论。

周一下午晚些时候，Chapin宣布将由执行编辑Eva Rodriguez主持每月审查报道的会议。

"每个月我们将自问一些问题：我们是否以各种形式捕捉到了这个国家的多样性 — 包括种族、民族、宗教、经济、政治和地理等等 — 并且在复杂性中，以一种方式帮助听众和读者认识到他们自己和他们的社区？" Chapin在备忘录中写道。"我们是否提供了有助于他们理解的报道 — 即使只是稍微好一点 — 那些与他们共同点不多的邻居们？"

Berliner表示他对这一宣布表示欢迎，但会在会议进行后再做评判。

在本文中，Chapin说道，自从Lansing在去年担任领导角色统一了新闻和节目部门以来，这类会议已经讨论过了。

"如果我们要做的话，现在看起来是时候交付了，" Chapin说道。"健康的讨论是我们更需要的东西。"
