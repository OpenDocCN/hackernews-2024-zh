# JetBrains 的 Inspectopedia

> 来源：[https://www.jetbrains.com/pages/inspectopedia-by-jetbrains/](https://www.jetbrains.com/pages/inspectopedia-by-jetbrains/)

使用 Inspectopedia，我们透明地展示了所有 JetBrains 代码检查，您可以在我们的 IDE 或您的 CI/CD 流水线中本地使用。我们公开分享静态分析规则、代码示例、修复指南和解释，以帮助您和您的团队交付最佳代码，并保持一致。
