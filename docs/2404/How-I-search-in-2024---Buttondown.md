# 2024年我的搜索方式

我们现在处于信息检索领域的一个非常奇怪的边缘空间，特别是那些关注搜索趋势并在LLM的前沿工作的消费者。

另一方面，我们有旧公司的衰落。以广播为基础的集中式[社交媒体](https://newsletter.vickiboykis.com/archive/where-do-we-go-from-here/?utm_source=vicki&utm_medium=email&utm_campaign=how-i-search-in-2024)，曾经稳定地为少数热心人提供新闻源和实时搜索，现在基本上已经死了，或者濒临消亡。搜索，特别是Google，现在基本上是一堆无用的广告和SEO游戏化，还有Reddit搜索结果的终点。每个人都已经写过它并且[广泛报道了这一点。](https://www.404media.co/google-search-really-has-gotten-worse-researchers-find/?utm_source=vicki&utm_medium=email&utm_campaign=how-i-search-in-2024)

另一方面，我们有这些奇怪的LLM事物的迅猛崛起，它们实际上只是[统计文本生成器](https://arxiv.org/abs/2212.03551?utm_source=vicki&utm_medium=email&utm_campaign=how-i-search-in-2024)，但人们正在通过像RAG这样的范式进行信息检索，这有点像用锤子去钓鱼。

第三方面（对生成艺术来说是一个非常恰当的隐喻），在过去15年大公司衰落的背景下，我们看到[新兴的独立搜索引擎场景正在崛起](https://news.ycombinator.com/item?id=30783391&utm_source=vicki&utm_medium=email&utm_campaign=how-i-search-in-2024)，它们渴望用AI工具重新夺回网络的质量。

我注意到自己越来越多地使用这些新工具，无论是出于工作需要还是个人对信息检索的兴趣，以及[在网上找到好东西](https://vickiboykis.com/essays/2016-11-20-fix-the-internet/?utm_source=vicki&utm_medium=email&utm_campaign=how-i-search-in-2024)，所以我想分享一下我最近在搜索方面做的事情：

**Kagi**: 我至少在过去8年中一直将[DuckDuckGo](https://web.archive.org/web/20131006201326/https://dukgo.com/help/en_US/company/history)作为我的默认搜索引擎，并且我想说可能是10年了，我喜欢他们最初的使命，但结果从来不怎么好，我总是不得不追加g!，这意味着我还是在做Google搜索。

我一直听说很多关于Kagi的事情，最后在一月份决定试用一下。结果非常好，我立刻购买了订阅并将我的默认搜索引擎在Web和移动浏览器中切换到了Kagi。他们最近因为其产品的多样性而引起了[一些争议](https://news.ycombinator.com/item?id=40011314&utm_source=vicki&utm_medium=email&utm_campaign=how-i-search-in-2024)，但搜索结果非常出色，[商业模式确实是面向用户的](https://blog.kagi.com/safe-round?utm_source=vicki&utm_medium=email&utm_campaign=how-i-search-in-2024)，我觉得自己不会再回到Google，除非发生非常糟糕的事情（请继续保持，Kagi）。

我也非常喜欢他们的[总结工具](https://help.kagi.com/kagi/api/summarizer.html?utm_source=vicki&utm_medium=email&utm_campaign=how-i-search-in-2024)，以及[Small Web](https://kagi.com/smallweb?utm_source=vicki&utm_medium=email&utm_campaign=how-i-search-in-2024)。我过去几天一直在Small Web上随意浏览，非常惊讶和耳目一新地发现真正的人在写真正的内容，而不是机器人、商业广告、SEO垃圾或夸大其词的内容。有一次，我看到一篇关于一个人在墨西哥海岸航行他的船的帖子，还有一篇关于Unix包装历史的帖子。这是一种愉快和放松的体验。

到目前为止，Kagi唯一做得还不够好的事情是本地结果，找出餐馆或公园的开放时间以及路线，对此我感觉还好，因为本地化意味着个性化，而这是我不希望从这一批新的搜索引擎中得到的，至少不是以我之前在产品中想要的方式。

[**MarginNotes**](https://www.marginalia.nu/?utm_source=vicki&utm_medium=email&utm_campaign=how-i-search-in-2024) - 我喜欢MarginNotes的一切，这是一个具有自己索引的搜索引擎，由一个单独的家伙运行，最初是在他公寓的[服务器上](https://www.marginalia.nu/log/90-new-server-design/?utm_source=vicki&utm_medium=email&utm_campaign=how-i-search-in-2024)。MarginNotes只显示非商业内容，正如这个搜索引擎所说：“这个搜索引擎并不特别适合回答像问题一样提出的查询，相反，试着想象一些可能出现在你正在寻找的网站中的文本，并搜索这些内容。” 思路是你不会找到完全符合你要找的内容，而是找到一些相关的内容，这些内容由于无情的PageRank已经被从相关性中删除。

我搜索过的例子包括“变压器架构”，并不是因为我想了解变压器，而是想看看除了Medium文章或通常的主流来源外，还有什么来源可以说这些内容。我对MarginNotes的看法是一种找到主流以外内容的方式，但这些内容是真实而深刻的。就像在真正的互联网知识池中畅快地游泳一样。

Viktor 也有[令人惊叹的技术写作](https://www.marginalia.nu/log/a_103_query_parsing/?utm_source=vicki&utm_medium=email&utm_campaign=how-i-search-in-2024)，内容涵盖运行自己的搜索引擎并优化其规模，以及[所有代码均为开源](https://www.marginalia.nu/log/58-marginalia-open-source/?utm_source=vicki&utm_medium=email&utm_campaign=how-i-search-in-2024)。

[**困惑度**](https://www.perplexity.ai/?utm_source=vicki&utm_medium=email&utm_campaign=how-i-search-in-2024) - 这个产品一直备受关注，我一直不愿对任何新的[LLM炒作周期产品过多评论](https://techcrunch.com/2024/04/23/perplexity-is-raising-250m-at-2-point-5-3b-valuation-ai-search-sources-say/?utm_source=vicki&utm_medium=email&utm_campaign=how-i-search-in-2024)，但困惑度在技术查找时表现出乎意料的出色，同时我也需要其引文和链接，因为我不相信LLM能仅仅为我生成答案。例如，“太阳食是如何工作的？” 我还用它来总结Arxiv论文的PDF。

简而言之，我们现在正处于将搜索引擎和信息检索在某种程度上解构为任务特定元素的阶段，我对大搜索时代的余波中会出现什么其他新事物感到兴奋。
