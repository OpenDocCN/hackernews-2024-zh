# OMG！我们来到了四十！（宣布发布 Fedora Linux 40） - Fedora Magazine

> 来源：[https://fedoramagazine.org/announcing-fedora-linux-40/](https://fedoramagazine.org/announcing-fedora-linux-40/)

哦，哇。这感觉像一个很大的数字！我很自豪地宣布 Fedora Linux 的第 40 个发布版，这是一个由社区构建和维护的操作系统，属于*我们所有人*。我也很高兴地指出我们重新按时发布了。感谢所有使这成为可能的 Fedora 贡献者们，再次使这成为我们迄今为止最好的一个版本。

这个对我来说也是一个*特别*激动人心的数字，因为这标志着我担任 Fedora 项目领导的第*20*个发布版本。在过去的十年里，我们经历了很多，看到我们的社区蓬勃发展我感到非常高兴。除了许多长期熟悉的名字和面孔外，看到新一代带来新的活力和思想也让人兴奋。在某些情况下，这甚至是*字面上*的新一代，因为你们中的许多人是在 Fedora 中长大的。但无论年龄多大，我为我们建立了一个如此友好和包容的社区感到骄傲，我们继续努力改善我们的包容性、多样性和可访问性。

不管怎样！足够了。是时候看看 Fedora Linux 40 为您带来了什么！如果您已经有系统，[升级 Fedora 到新版本](https://docs.fedoraproject.org/en-US/quick-docs/upgrading-fedora-new-release/) 很容易。如果您是新手，或者只是好奇，请转到 [Get Fedora](https://fedoraproject.org) 获取安装选项。

## Desktop news

Fedora Workstation Edition 特色是 GNOME 桌面环境，现已更新至版本 46\. 查看 [Fedora Workstation 40 有什么新功能？](https://fedoramagazine.org/whats-new-fedora-workstation-40) 获取亮点！

KDE Spin 现在包括 KDE Plasma 6，并且默认运行 Wayland。更多关于此和其他 KDE Spin 更新的信息请查看 [Fedora KDE 40 有什么新功能？](https://fedoramagazine.org/whats-new-in-fedora-kde-40)

我们也正式恢复了“Fedora Atomic Desktop”品牌，适用于所有使用 ostree 或基于镜像的 provisioning 的变体。我们的技术并非真正“不可变”，因此这提供了更好的分类。在 [介绍 Fedora Atomic Desktops](https://fedoramagazine.org/introducing-fedora-atomic-desktops/) 中可以了解更多详情 — 但简而言之，Fedora Silverblue 和 Fedora Kinoite 将保留，而其他桌面变体将成为 Fedora Sway Atomic 和 Fedora Budgie Atomic。

## AI 开发工具

Fedora Linux 40搭载了我们有史以来第一个PyTorch软件包。PyTorch是一个流行的深度学习框架，但是要可靠地安装正确版本的驱动程序和库等是困难的。目前的软件包仅支持在CPU上运行，没有GPU或NPU加速，但这只是第一步。我们的目标是构建一个完整的PyTorch和其他流行工具的堆栈，可以直接在各种硬件上使用。

我们还将与ROCm 6一同发布 —— 这是为AMD显卡提供加速支持的开源软件。我们计划在未来的版本中为PyTorch启用这一功能。

## 到处都是更新！

一如既往，我们使用更新的编译器和库重新构建了整个发行版（当然，这些更新的工具已经准备好供开发者使用）。这些更新带来了错误修复、安全改进和性能提升。

当然，数百名Fedora的打包者和测试人员共同努力，整合了来自成千上万上游项目的最新开源软件版本。这些项目又由无数从事市场营销、设计、文档、代码、质量、翻译、沟通、活动、治理、基础设施、安全等工作的开发者和贡献者们共同构建。再次感谢所有使Fedora变得如此出色的人，以及所有建立这个整个自由开源软件宇宙的人。

## 说到更新...
