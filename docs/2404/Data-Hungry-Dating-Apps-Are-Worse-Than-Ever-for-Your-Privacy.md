# 数据贪婪的约会应用对你的隐私造成了更大的威胁

> 来源：[https://foundation.mozilla.org/privacynotincluded/articles/data-hungry-dating-apps-are-worse-than-ever-for-your-privacy/](https://foundation.mozilla.org/privacynotincluded/articles/data-hungry-dating-apps-are-worse-than-ever-for-your-privacy/)

+   **一个应用没有隐私政策**

尽管在英语国家如美国、加拿大和英国可以下载，我们找不到 [Lovoo](https://foundation.mozilla.org/privacynotincluded/lovoo/) 的英文隐私政策。如果你懂德语，那么有针对你的隐私政策。但如果英语是你唯一的语言，对不起，没有隐私政策给你！

+   **大约一半的应用由两家公司拥有**

[Tinder](https://foundation.mozilla.org/privacynotincluded/tinder/), [OkCupid](https://foundation.mozilla.org/privacynotincluded/okcupid/), [Match](https://foundation.mozilla.org/privacynotincluded/match/), [Hinge](https://foundation.mozilla.org/privacynotincluded/hinge/), [Plenty of Fish](https://foundation.mozilla.org/privacynotincluded/plentyoffish/), [BlackPeopleMeet](https://foundation.mozilla.org/privacynotincluded/blackpeoplemeet/), [OurTime](https://foundation.mozilla.org/privacynotincluded/ourtime/) 等多个约会应用由 Match Group 拥有。[Christian Mingle](https://foundation.mozilla.org/privacynotincluded/christianmingle/), [Jdate](https://foundation.mozilla.org/privacynotincluded/jdate/), [Elite Singles](https://foundation.mozilla.org/privacynotincluded/elitesingles/) 和 [Zoosk](https://foundation.mozilla.org/privacynotincluded/zoosk/) 由 Spark Network 所有。此外，[Bumble](https://foundation.mozilla.org/privacynotincluded/bumble/) 和 [Badoo](https://foundation.mozilla.org/privacynotincluded/badoo/) 也属于同一公司。这意味着你的个人数据可能从它们所属公司的任何应用中共享。这样的数据共享可能性很大，这并不好。

+   **诉讼称：爱情是一场失败的游戏**

在情人节（哎呀！）[Match Group](https://foundation.mozilla.org/privacynotincluded/match/) 因“锁定用户在优先考虑公司利润而不是市场承诺和客户关系目标的付费游戏循环”而被起诉。哦，不。OK-在隐私方面，[eHarmony](https://foundation.mozilla.org/privacynotincluded/eharmony/) 也被指控使用“欺骗性销售做法”，为用户设置了“订阅陷阱”。

+   **AI 在匹配人类方面并不聪明**

[Tinder](https://foundation.mozilla.org/privacynotincluded/tinder/)、[OkCupid](https://foundation.mozilla.org/privacynotincluded/okcupid/)、[Facebook Dating](https://foundation.mozilla.org/privacynotincluded/facebook-dating/)、[Happn](https://foundation.mozilla.org/privacynotincluded/happn/)、[CoffeeMeetsBagel](https://foundation.mozilla.org/privacynotincluded/coffeemeetsbagel/) 和 [Tantan](https://foundation.mozilla.org/privacynotincluded/tantan/) 在其匹配算法中都使用了人工智能。对于这些算法如何工作的信息很少，我们担心它们可能会加倍地强化类似于 [性别种族主义](https://foundation.mozilla.org/blog/dating-apps-racist-tinder-hinge/) 和对胖人的厌恶等现实世界的偏见 —— 这些偏见在真正重要的地方剥夺了用户的真实偏好。我们每个人远远不止我们划动的总和！

+   **骗子和浪漫骗子正在增加**

浪漫骗子利用假的约会应用程序档案来针对毫无戒备的应用程序用户，他们假装对你的内心感兴趣，而他们真正想要的是你的钱。它们是一个在全球范围内都很严重的问题，美国联邦贸易委员会报告称，消费者在2023年因这些骗局损失了数百万美元。虽然可能没有任何应用程序能够免疫，但来自 [Tinder](https://foundation.mozilla.org/privacynotincluded/tinder/)、[Tantan](https://foundation.mozilla.org/privacynotincluded/tantan/)、[Zoosk](https://foundation.mozilla.org/privacynotincluded/zoosk/)、[Facebook Dating](https://foundation.mozilla.org/privacynotincluded/facebook-dating/) 和 [Lovoo](https://foundation.mozilla.org/privacynotincluded/lovoo/) 的骗局最近成为新闻焦点。

[Muzz](https://foundation.mozilla.org/privacynotincluded/muzz/) 和 [Her](https://foundation.mozilla.org/privacynotincluded/her/) 在其隐私政策中表示，约会者可以在他们的应用程序上使用昵称或化名。这很酷！
