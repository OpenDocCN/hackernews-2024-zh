# 如何确定空气质量传感器的误差？

> 来源：[https://www.airgradient.com/blog/how-to-determine-error-of-sensor/](https://www.airgradient.com/blog/how-to-determine-error-of-sensor/)

了解传感器准确性的一个有用工具是其平均误差。它描述了传感器测量值偏离真实值的程度。例如，如果温度传感器的平均误差为2°C，并测量到15°C，则真实温度可能在13°C到17°C之间。平均误差越低，传感器性能越好。通常用两个参数来量化这一点：平均绝对误差和均方根误差。以下文章将通过例证解释这两个参数。

#### 平均绝对误差（MAE）

MAE 是传感器测量值与真实值（参考测量）之间的平均绝对差异。

更加科学地表达，方程式如下：

*c[sens]*：传感器测量的空气污染物浓度

*c[ref]*：参考测量测得的空气污染物浓度（真实值）

*n*：测量次数

#### 为什么要取绝对值？

方程中的两个竖线表示传感器测量值与真实值之间的绝对差异。这确保了无论传感器读数是高于还是低于真实值，差异始终为正。为什么这一点如此重要在图1中有所体现：传感器（黑色）有时会注册低于真实值（红色），有时会高于真实值。因此，两个测量之间的差异在图1的第二面板中交替显示为上下波动。

计算这些差异的平均值时，负值和正值互相抵消，导致一个非常小的误差（0.2 µg/m³），这可能会误导地表明传感器的准确性很高。如果我们取绝对差异，即将所有负差异转换为正值，我们将得到一个平均误差（MAE）为9 µg/m³，这更好地反映了实际情况。

*图1*：顶部：PM传感器（黑色）测量的（虚构的）示例与参考传感器测量的真实PM浓度（红色）比较。中部：两个测量之间的差异，虚线表示平均（=平均）差异。底部：两个测量之间的绝对差异，虚线表示平均绝对差异

#### 均方根误差（RMSE）

与取错误的绝对值不同，也可以对其进行平方以获得正值。如果对这些平方误差求平均并取其平均值的平方根，即可得到均方根误差（RMSE）：

如果你无法理解这个计算过程，也不用担心。RMSE与MAE大多数时候非常相似。

**RMSE与MAE的区别**

基本上，这两个值都提供了相同的信息：传感器的平均误差。只是平方差强调了较大的误差，因此 RMSE 对异常值比 MAE 更敏感。图2以图形方式说明了这一点。

*图2*：（虚构的）与（左）和无异常值的传感器测量比较（右）。底部面板显示了在没有异常值发生时，MAE 和 RMSE 非常相似（面板b）。如果传感器测量包含异常值，则 RMSE 比 MAE 显著增大（面板d）。

RMSE 总是与 MAE 相同或更高，因此是评估传感器性能的“更严格”的指标。因此，AirGradient 通常关注 RMSE 来描述其监测器。这也符合美国环境保护署的测试指南。

好传感器 vs. 坏传感器

看看图3的两个顶部面板。你会选择哪一个传感器？左边还是右边？大多数人可能会选择左边的传感器，因为其测量值非常接近参考值（红线），而右边的传感器则比较偏离红线。你直观地看出左边的传感器更准确。选择得好！

RMSE 反映了这一点：毫不奇怪，左侧传感器的 RMSE（即平均误差）低于右侧传感器的 RMSE。RMSE 越低，传感器的性能越好。一个完美的传感器（完全测量污染物的真实浓度）的 RMSE 为 0。

*图3*：相对准确（左）和较不准确的低成本传感器（右）的 PM[2.5] 测量。顶部面板（a）和（c）显示了传感器（黑色）测得的 PM[2.5] 浓度的时间图，与参考传感器（红色）测得的“真实”值进行比较。底部面板（b）和（d）显示了上述两个测量之间的差异时间图。虚线标记了 RMSE。

#### 标准化的均方根误差（nRMSE）

现在，请问在图4中的两个传感器中，你会选择哪一个？左边还是右边？就 RMSE 而言，两个传感器的误差几乎相同。但是，你可能会选择右边的传感器，对吗？那是因为右边的传感器具有更低的相对误差。

仔细观察图4中两个传感器暴露的污染水平。右侧的颗粒物（PM）浓度比左侧高出10倍。在污染较少的环境中，RMSE（参见下面的面板）约为7 µg/m³，对测量结果的影响更大。如果我们只看 RMSE，我们会误以为两个传感器的表现相同。这就是 nRMSE 发挥作用的地方。它对比较不同污染水平环境下传感器的性能很有用。对于温度和湿度传感器而言，它的用处较小。

nRMSE是通过对RMSE进行归一化计算得出的，这意味着通过在测试间隔期间将其除以平均（真实）测量值来得出。在我们的例子中，这将是在传感器测试时测得的平均PM浓度。在图4的底部，您可以看到在污染较少的环境和污染较多的环境中计算nRMSE。在污染较少环境中的传感器的nRMSE为66%，表明其测量平均偏离真实浓度66%。在污染环境中，传感器的测量平均偏离为6%。因此，相对而言，右侧传感器比左侧传感器更准确。

*图4*：比较处于污染较少（左）和污染较多环境（右；注意y轴的不同刻度）的传感器和参考测量。nRMSE在底部进行计算。

#### 什么是可接受的误差？

您可以在美国环保局的测试指南中找到低成本PM2.5空气传感器性能的目标值。对于24小时均值数据，他们建议RMSE最大为7 μg/m³，nRMSE最大为30%（两个标准中的任何一个都足以满足标准）。

因此，当传感器读数偏离真值30%时没问题。这不是太多了吗？

所有传感方法，包括高标准的参考级仪器，都有其局限性。如果比较两个参考仪器，它们理想情况下应该测量完全相同，即真正的污染物浓度。但它们并不是这样：美国环保局的一项研究共同放置两个常用的参考仪器（Met One BAM-1020和Teledyne API T640）并发现，在某些浓度范围内，它们相互偏离超过25%。

别误会我们；参考级仪器仍然是空气质量测量的金标准。在正确操作时，它们比低成本传感器更可靠和准确。然而，需要承认所有测量都存在一定程度的不确定性。低成本传感器相对参考级仪器有许多其他优势，如其小尺寸、易于设置和维护，当然它们的低成本。为了从这些优势中受益，可以容忍稍低的准确性。

#### 总结

+   所有参数，如MAE、RMSE和nRMSE，都提供了关于传感器平均误差的信息，即传感器测量值与真值的偏离程度。

+   这些值越接近0，传感器性能就越好。

+   RMSE是一个“严格”的标准，因为它对异常值更敏感。

+   相对误差（如nRMSE）对比不同污染水平的环境内传感器的性能是有用的，但对于温度和湿度传感器来说则不那么有用。

+   美国环保局建议使用RMSE和nRMSE来表征传感器的平均误差。

+   对于PM2.5，最多可接受RMSE = 7 μg/m³或nRMSE = 30%的测量偏差。

+   即使是参考级仪器的测量结果也存在不确定性。
