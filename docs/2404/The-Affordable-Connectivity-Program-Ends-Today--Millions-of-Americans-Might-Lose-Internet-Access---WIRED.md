# 《廉价连接计划》今日结束。数百万美国人可能会失去互联网接入 | WIRED

> 来源：[https://www.wired.com/story/affordable-connectivity-program-ending/](https://www.wired.com/story/affordable-connectivity-program-ending/)

今天标志着《廉价连接计划》的结束，这是美国政府立法的里程碑，旨在使人们更容易在家中负担得起互联网连接。该计划的终结标志着一个重大变化，停止福利将影响到可能最需要它们的数百万美国人。

## 什么是《廉价连接计划》？

2021年，美国国会通过了总值1.2万亿美元的[基础设施投资和就业法案](https://www.congress.gov/bill/117th-congress/house-bill/3684)，也被称为[两党基础设施协议](https://www.whitehouse.gov/briefing-room/statements-releases/2021/11/06/fact-sheet-the-bipartisan-infrastructure-deal/)。这是一项雄心勃勃的法案，旨在支持一系列陷入困境的行业，包括交通网络、能源系统和公共事业。《廉价连接计划》是该协议的一部分。它拨款142亿美元用于资助能够帮助低收入家庭负担得起高速互联网的抵押。

今天，所有这些都将成为过去。

这肯定不是一个理想的情况，考虑到互联网在现代社会中是一种必需品，特别是在后COVID时代，远程工作已成为常态化的情况下。不能上网会导致数字鸿沟加剧，没有互联网接入的人可能在社会和经济上落后。依赖折扣的美国人将看到他们的互联网账单飙升，并且可能被迫在增加的费用与支付食物、租金和其他必需品之间找平衡。更不用说许多《廉价连接计划》的受益者已经习惯于免费获得互联网；最初的[白宫敦促](https://www.whitehouse.gov/getinternet/)导致许多互联网提供商提供以每月30美元提供最低下载速度为100 Mbps的计划。将ACP福利应用于这些计划的客户免费获取互联网接入。从明天开始，他们将不得不开始支付费用。

## 为什么《廉价连接计划》到期？

问题在于资金。与任何联邦资助一样，一个项目能够在没有被国会续签的情况下运行多长时间是有限制的。拜登政府已经多次呼吁立法者拯救该项目，包括去年十月的一次请求，要求投资[$60亿美元](https://www.whitehouse.gov/briefing-room/statements-releases/2023/10/25/fact-sheet-white-house-calls-on-congress-to-support-critical-domestic-needs/)来继续该项目，以及其他国内支出。尽管如此，以及来自[倡导团体](https://www.digitalinclusion.org/blog/2023/10/25/white-house-asks-congress-for-6-billion-to-keep-acp-going/)和联邦通信委员会等组织的多次恳求，这些尝试都未能取得成效，该项目现已正式失效。

## 谁受影响？

据白宫称，自该项目实施以来，已有超过2300万家庭受益。这是一个惊人的数字——大约每六个家庭中就有一个，现在必须另寻其他资金来源来保持连接。

对于支持ACP的组织来说，ACP的终结并不完全是一个意外。为了预料到未来的资金短缺，ACP于2024年2月停止接受该项目的申请。根据美国联邦通信委员会的说法，[网络运营商有责任](https://www.fcc.gov/acp)通知用户该项目已经结束，因此受影响的家庭应该已经收到关于逐步减少的通知。

## 接下来会发生什么？

不幸的是，任何收到ACP补助金的家庭都将不得不开始为他们的互联网连接支付全额费用——前提是他们有能力负担得起。如果一个家庭的收入低于[联邦贫困线的200%](https://aspe.hhs.gov/topics/poverty-economic-mobility/poverty-guidelines)，或者如果家庭申请其他政府福利，如SNAP、医疗补助或社会保障，则可以通过其他方式获得类似的互联网费用减免，尽管金额要小得多。通用服务行政公司提供一项名为[Lifeline](https://www.lifelinesupport.org/)的服务，可以支付每月高达$9.25的连接费用（对于居住在符合条件的部落土地上的人士，最高可达每月$34.25）。

在昨天的[新闻发布会上](https://www.youtube.com/live/O_VbN_9G8cc?si=tsSNhtm7DVE7kTXt&t=2358)，白宫新闻秘书卡琳·让-皮埃尔重申了拜登政府的立场，敦促大型电信公司继续遵守ACP规则，尽管缺乏资金支持，她说：“本月早些时候，我们宣布鼓励供应商采取措施，在这个关键时刻通过提供低成本或免费计划来保持他们的消费者连接。”然而，目前还不清楚是否有任何供应商愿意配合。

如果您受到影响，您的互联网服务提供商应已经向您发送了两次有关您的服务的通知。如果情况不是这样，请联系您的服务提供商。最终，受影响的用户将不得不决定是否愿意以新价格继续他们的服务。FCC 在 [ACP 解除页面](https://www.fcc.gov/acp) 上提供了相关信息，还鼓励[提交投诉](https://consumercomplaints.fcc.gov/hc/en-us)。
