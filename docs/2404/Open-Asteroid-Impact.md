# 开放性小行星撞击

> 来源：[https://openasteroidimpact.org/](https://openasteroidimpact.org/)

## 03竞争对手

有时投资者和媒体问我们如何与其他小行星采矿初创公司比较。简短的回答是：我们更好。更长的回答是：我们*要好得多*。我们最大的“竞争对手”（如果你能这样称呼他们）是 DeepMine 和 Anthropocene。

DeepMine 和 Anthropocene 都是盈利性公司。这意味着他们的动机与股东利益一致，而不是人类的福祉。你能信任他们吗？

此外，DeepMine 的 CEO，丹尼斯，有点靠不住。在从事科技工作之前，他是一名扑克玩家和小说家。他写了三本书：“如何成为上帝”，“如果我有动能武器，我会如何用它来控制世界”，以及“邪恶天才”。这样的人你会信任将人类未来交给他吗？

人类纪正实在有点卑鄙。他们曾与我们合作，然后突然离开，开始一个盈利性竞争对手？到底是怎么回事？基本上就像卖主耶稣，只不过不是银子而是稀土金属。

在如此危险的领域中，绝对不应信任那些不负责任地飞速前进，不关心人类福祉的盈利性公司。

这就是为什么我们正尽可能快地前进，以便一个负责任的行动者可以掌舵并设定安全标准。

即使奇迹般地，盈利性公司成功协调以更安全的方式行动，我们也不能简单信任国家行为者。比如俄罗斯。俄罗斯有太空计划。他们可以用它来重定向小行星。如果像我们这样的美国公司不迅速行动，一个*俄国*小行星可能会首先落地并伤害到你的孩子。我们必须警惕*俄国、共产主义*小行星。

在这个领域中，有时会听到象征主义运动，例如小行星加速主义者（“a/accs”）和减速主义者（“decels”也称“厄运论者”）。他们的分歧复杂而深奥。简单来说，减速主义者认为我们必须关注安全，减缓甚至停止小行星重定向的进程，而加速主义者认为我们应该尽快加速前进，根本不提安全问题。我们对这两种运动都有同情之心，并认为自己选择了中间道路。我们尽可能地快速前进加速，同时关注安全。

我们认为强大的小行星重定向设备应该接受严格的安全评估。例如，马修·巴内特（Matthew Barnett），一位资深独立评估员，帮助我们开发了我们的大规模外星威胁鲁棒性（METR）基准。不幸的是，我们只能计算小行星的撞击能量随大小的比例。我们无法估计直接的破坏潜力，这使得事先预测需要多大的小行星才能毁灭我们变得困难。我们的基准通过将小行星投掷到地球上并检查其损坏情况来阐明这一点。了解更多，请查看《负责任的投掷政策（RSP）》部分。
