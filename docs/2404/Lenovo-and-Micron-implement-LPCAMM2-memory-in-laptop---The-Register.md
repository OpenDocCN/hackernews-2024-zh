# 联想和Micron在笔记本中实施LPCAMM2存储 • The Register

> 来源：[https://www.theregister.com/2024/04/24/lenovo_micron_lpcamm2_thinkpad/](https://www.theregister.com/2024/04/24/lenovo_micron_lpcamm2_thinkpad/)

联想最新的ThinkPad P1 Gen 7笔记本将是第一款使用新LPCAMM2存储形态的设备，这是SODIMM条的继任者。

虽然联想主要关注其配备英特尔Core Ultra CPU和Nvidia RTX 3000 Ada GPU的新笔记本的人工智能性能，但该公司还指出，他们的设备是全球首款使用LPCAMM2存储标准的设备。

根据[联想](https://news.lenovo.com/pressroom/press-releases/lenovo-unveils-its-new-ai-ready-thinkpad-p1-gen-7-mobile-workstation/)的说法，LPCAMM2与SODIMM相比使用空间少64%，主动功耗少61%。这归功于它基于LPDDR5X而非普通DDR5的设计。

LPCAMM2标准专为笔记本设计，其实源于戴尔开发的技术。最初被称为CAMM（压缩附加存储模块），2022年首次在戴尔Precision 7670上作为专有存储类型首次亮相。然而，在2023年，这款PC巨头将其知识产权捐赠给了[JEDEC](https://www.theregister.com/2023/01/17/jedec_to_formalize_dell_laptop/)，即负责标准化存储技术的组织。

当JEDEC最终确认其规范时，CAMM在2023年9月成为了LPCAMM2（低功耗压缩附加存储模块）。[三星迅速宣布](https://www.theregister.com/2023/09/26/samsungs_takes_camm_format_mainstream/)计划生产LPCAMM2存储条，并声称其性能比基于SODIMM的前身提升50%，效率提升70%。此外，LPCAMM2可以提供双通道内存，无需第二个模块。

除了性能和效率提升外，LPCAMM2还设计成像SODIMM一样可以用户升级。此外，它的物理尺寸较小可能使其成为可焊接LPDDR5X模块的替代品，这些模块不能在没有焊接铁的情况下从计算机中移除。

Micron似乎比三星更快推出了他们在CES期间首次[宣布的](https://investors.micron.com/news-releases/news-release-details/micron-first-market-lpddr5x-based-lpcamm2-memory-transforming)技术。Micron强调了LPCAMM2在人工智能方面的优势，但如果新标准的所有声称都能成真，那将对所有用户带来巨大改进。

目前还没有对ThinkPad P1 Gen 7中LPCAMM2进行深入分析或评测。Micron声称Idle功耗减少80%的说法特别引人注目，但尚不清楚LPCAMM2能否达到这一水平。®
