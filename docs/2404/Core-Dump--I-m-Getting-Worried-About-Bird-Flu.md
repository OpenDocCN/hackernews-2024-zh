# Core Dump：我对禽流感感到担忧

> 来源：[https://coredump3.blogspot.com/2024/04/im-getting-worried-about-bird-flu.html](https://coredump3.blogspot.com/2024/04/im-getting-worried-about-bird-flu.html)

我对禽流感（又称H5N1禽流感）感到担忧。现在称其为禽流感有点名不副实，因为最近的消息清楚表明它正在广泛传播于哺乳动物，现在包括牛。这种情况已经发生了一段时间，但现在已经广泛到足以引起主流媒体的关注。

[《纽约时报》](https://www.nytimes.com/2024/04/22/health/birdflu-marine-mammals.html?unlocked_article_code=1.nU0.3zh_.QQeDDYQdTTq-&smid=url-share)（礼品链接）特别关注最近大规模爆发，尤其是海狮，这已经严重破坏了北美和南美沿海的殖民地。

> 他们发现的情况令人震惊：病毒导致约17,400只海豹幼崽死亡，占殖民地年轻动物的95%以上。
> 
> 这场灾难是自2020年以来在全球肆虐的禽流感流行病的最新一幕，促使多个大陆的当局杀死了数百万只家禽和其他鸟类，这一无效尝试。
> 
> H5N1没有停止过。禽流感病毒通常对它们的宿主很挑剔，通常只依附于某一种野生鸟类。但这种病毒已迅速侵入了惊人广泛的鸟类和动物，从松鼠和臭鼬到瓶鼻海豚、北极熊，最近还包括奶牛。

当然，这里的担忧是病毒可能会发生变异，使其在人类中更易传播，其死亡率至少为33%。

在他的《地面真相》博客中，Eric Topol [提供了更详细的总结](https://erictopol.substack.com/p/a-bird-flu-h5n1-status-report)，介绍了我们对H5N1传播情况的了解。

> 确认了8个州的H5N1感染奶牛群。但FDA昨天的报告显示，商业牛奶PCR检测阳性强烈支持牛的传播远远超出这8个州。重要的是强调（PCR）是检测病毒残留物质，而不是活病毒，过去几乎不可能通过巴氏杀菌法找到活病毒。其他测试，评估任何活病毒的潜在证据（卵子活性和培养），FDA将继续报告。迄今为止，所有有限的培养试验均显示牛奶中没有任何活病毒。

[Gizomdo](https://gizmodo.com/bird-flu-detected-milk-dairy-cows-spread-fda-update-1851433417?utm_source=vip)提供了关于传播到牛群的更多细节，以及FDA的应对措施。

> 外部科学界周日发布的遗传证据表明，从鸟类到牛类的最初溢出事件可能早在2023年12月就发生了，比当地官员报告的第一个已知病例早几个月。再加上在购买的牛奶中发现H5N1，现在看来，爆发的规模可能比目前统计的要广泛得多。
> 
> “对奶牛的传播远比我们所知道的要广泛得多”，Scripps Research Translational Institute 创始人埃里克·托波尔（Eric Topol）周二告诉 CNN。
> 
> 诸如H5N1这类禽流感毒株的主要担忧是它们可能在某一天发展出适当的突变组合，使病毒能够在人类之间轻松传播，同时引起许多严重疾病。因此，它在奶牛中的存留时间越长，某些毒株适应和在哺乳动物之间传播的可能性就越大，包括人类在内。

我不担心在牛奶中找到病毒的遗传物质（巴氏杀菌会杀死任何活病毒），但我不会喝未经巴氏杀菌的牛奶（即使没有H5N1的威胁，这也从来不是一个好主意）。即使现在没有使用，也要储备一些好的N95口罩。
