# 在美国被禁止：健康灯泡 | 心理学今日

> 来源：[https://www.psychologytoday.com/us/blog/the-light-doctor/202404/banned-in-the-us-healthy-light-bulbs](https://www.psychologytoday.com/us/blog/the-light-doctor/202404/banned-in-the-us-healthy-light-bulbs)

美国能源部（DOE）最近发布了一项最终规定，从2028年6月25日起生效，将坚定不移地巩固不健康的蓝泵LED灯泡的主导地位。我们必须在为时已晚之前撤销这一决定。

在2024年4月19日的联邦公报110页通知中，存在许多错误陈述和缺陷分析，让人不知从何说起。以下是一些关键摘录：

**“在目前看来，美国能源部认为生物钟友好型集成LED灯具与其他（GSLs）没有独特的属性。”（即灯泡，他们称之为一般服务灯具）。（p. 28877）**

这是无稽之谈。生物钟友好型灯泡具有明显的光谱功率分布，即每种颜色波长的相对光量。白天，生物钟友好型灯具富含480 nm的蓝光，并在晚间和夜间几乎不发出480 nm的蓝光。可以用手持式分光仪轻松测量。

证据表明，生物钟友好型灯具在晚间和夜间440-495 nm蓝光应低于2%，而在白天应高于20%。

**“关于具有显著影响人类生物钟周期的具体灯具属性，没有共识。”（p. 28877）**

此声明无视自2000年以来逾10,000篇经过同行评审的科学论文，涉及生物钟与光线相互作用。它也未能承认多个共识报告和研究，例如最近248位领先科学家的共识研究，他们在[生物钟](https://www.psychologytoday.com/us/basics/circadian-rhythm "Psychology Today looks at circadian rhythms")和光线领域发表了超过2,500篇论文，并得出结论：

+   人类视网膜ipRGC黑色素感受器的敏感峰值大约在可见光谱的蓝色部分，约为480nm。

+   生物钟同步最有效的波长是接近ipRGC黑色素感受器敏感峰值的460-495 nm蓝光，详细信息请参见[同步](https://www.psychologytoday.com/us/basics/binaural-beats "Psychology Today looks at entrainment")。

+   晚上（临睡前三小时）的富含蓝光（460-495nm）比相同强度下的缺少蓝光更易干扰夜间睡眠。

+   晚上（临睡前三小时）的富含蓝光（460-495nm）会使生物钟系统的相位延迟，比相同强度下的缺少蓝光更显著。

+   晚上（临睡前三小时）的富含蓝光（460-495nm）比相同强度下的缺少蓝光更易扰乱生物钟。

+   夜间暴露于 460-495nm 蓝光会抑制褪黑激素的产生。

+   夜间暴露于 460-495nm 蓝光会扰乱昼夜节律。

+   晚上（临睡前三小时）使用的光应尽可能少含有蓝色成分。

+   增加灯具的能效是可取的，但不应增加引起昼夜节律紊乱和严重疾病风险的风险。

+   具有高 460-495nm 蓝色成分的 LED 灯应携带警告标签：“夜间使用可能有害。”

总之，具有高蓝色成分的灯泡与低蓝色成分的灯泡具有明显不同的属性。

“采纳的标准，表达为每瓦灯泡或灯具效率的最低流明（“lm”）输出（“W”）…… 这些标准适用于从 2028 年 7 月 25 日开始在美国境内制造或进口的所有产品。”（第 28857 页）

美国能源部错误地将流明视为灯泡提供的唯一价值，因此专注于最大化每瓦流明（或者最小化创造这些流明所需的功率）。

但流明主要衡量与视觉亮度相关的绿色和黄色波长，而不衡量管理昼夜节律健康所必需的蓝色波长或愈合所必需的红色波长。

因此，用于提供健康光波长的电能被视为浪费能源，根据美国能源部的说法。因此，当每瓦流明被用作监管标准时，健康灯泡的排名较低。这使得健康灯泡极其困难，甚至不可能达到这些任意高流明/瓦的标准。

“美国能源部在这项最终规则中不豁免昼夜友好型集成 LED 灯具免受 GSL 定义的影响，因此这些灯具将受到 GSL 的标准约束。”（第 28877 页）

当灯波长的光谱分布必须集中在达到高流明每瓦的目标时，蓝泵 LED 受益颇丰。如果这一新规则不豁免昼夜友好型灯泡，那么晚上使用的低蓝光和零蓝光昼夜友好型灯泡的使用可能会受到禁止风险。

昼夜节律基本阅读

美国能源部（DOE）目前仅从两种前体（SO 2 和 NO X ）对环境细颗粒物（“PM 2.5”）浓度和一种前体（对于 NO X ）对环境臭氧的变化中获得健康效益（第 28859-60 页）。

美国能源部倾向于通过考虑仅减少温室气体的益处来正当化其禁止健康灯具的严苛政策。他们未考虑在夜间使用富含蓝光 LED 灯泡的健康成本，包括睡眠障碍、[精神病](https://www.psychologytoday.com/us/basics/psychiatry "心理学今日关注精神病")障碍、肥胖症、糖尿病、心脏病、乳腺癌、前列腺癌、结肠直肠癌等等。

2028年后只允许使用蓝泵LED

昼夜节律光研究中心

**发生了什么？**

正如DOE所承认的：

**“在2023年1月的临时规则中，DOE提出了对‘昼夜节律友好的集成LED灯’的定义，并建议符合该定义的灯具被排除在GSL定义之外。DOE确认了市售的集成LED灯，它们被营销为通过改变光谱来帮助人类的睡眠-清醒（即昼夜节律）周期，同时观察到它们的效率范围从47.8 lm/W到85.7 lm/W。”**

然而，照明行业和气候游说团体敦促DOE取消所提出的豁免，认为这是照明制造商可能利用的一个漏洞，以规避监管标准。

不幸的是，DOE忽略了我和其他人提供的关于保护使用光谱工程设计的昼夜节律友好灯对健康有多重要的证据。

为了保护所有使用者的健康，DOE做出的这个有害且不合理的决定在2028年之前必须被撤销，以免昼夜节律友好灯泡消失在商店货架上。
