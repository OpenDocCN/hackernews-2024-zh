# 远程丈夫的崛起

> 来源：[https://www.economist.com/united-states/2024/04/04/the-rise-of-the-remote-husband](https://www.economist.com/united-states/2024/04/04/the-rise-of-the-remote-husband)

在加利福尼亚州富裕且海滨的橙县城市科斯塔·梅萨，她正逐步晋升为一家大型法律事务所当地办事处的合伙人；而他是一家总部位于湾区的科技初创公司的高管，距离超过400英里。在马萨诸塞州的剑桥，他在他们公寓里离校区不远处编写代码，而她则在哈佛法学院上课。她是一名产科医生，他远程为一家科技公司工作；她是常春藤盟校的学者，他为一家加密公司工作。在全国各地，富裕且受过良好教育的人群中，似乎正在出现一种新趋势。当妻子们早上出门去上班、上课或医院时，她们在家里与留守的丈夫挥手告别。

这绝非是一个上世纪50年代风格的性别对调复兴。毕竟，男性仍然在工作，而非主要做饭、清洁和照顾孩子。但它确实反映了远程工作兴起的一个未被重视的影响：远程丈夫的崛起。
