# 雀巢如何在低收入国家让儿童对糖上瘾

> 来源：[https://stories.publiceye.ch/nestle-babies/](https://stories.publiceye.ch/nestle-babies/)

我们的调查显示，尽管世界卫生组织的国际法律禁止商业促销此类产品，雀巢仍然采用激进的营销手段在低收入和中等收入国家推广Nido和Cerelac。该法典最初于1981年在“婴儿杀手”丑闻之后被采纳，并在后来的决议中进行了明确和强化，以禁止所有婴儿配方奶粉的促销以保护母乳喂养。该禁令还适用于像Cerelac这样不符合营养指南并含有“高糖分”婴儿食品。

雀巢回应称，“我们在全球范围内遵守世界卫生组织的法典和后续世界卫生大会决议，这些法典和决议由各国政府实施。” “在当地法律不如我们实施法典的严格政策时，我们遵守我们严格的政策，” 雀巢补充道。

实际上，尽管70%的国家采纳了基于该法典的法律，但许多国家存在漏洞，实施普遍存在弱点，特别是在低收入和中等收入国家，通常是由于婴儿食品行业和出口国的压力所致。此外，尽管这些都被该法典覆盖，但雀巢的政策既不适用于一岁以上婴儿的配方奶粉，也不适用于其他婴儿食品。

此外，雀巢将其Nido和Cerelac产品宣传为有益儿童发展的健康必需品，尽管它们含有添加糖，并对儿童健康和发育带来许多风险。“通常制造商的健康声明并未得到科学支持，” Rollins 在阅读我们的报告后说道。“如果你有一个药品产品，想要声称它能改善婴儿的大脑发育或促进婴儿生长，你必须通过非常高的证据标准，” 他解释道。“但因为它是食品，你不必通过这些标准。”

世界卫生组织在最近的报告中 [解释](https://iris.who.int/bitstream/handle/10665/260137/9789241513470-eng.pdf?sequence=1)，营养和健康声明“理想化产品，暗示其优于家庭食品，并掩盖了风险”。该组织警告说，这些做法误导消费者关于这些食品真实含量的认识，联合国机构认为不应该进行这些声明。然而，雀巢已将其作为营销策略的核心支柱。

## **“聪明成长”**

“聪明成长”。位于雅加达市中心和印尼其他主要城市的巨大广告牌上展示着这一口号。这是雀巢推广其 Nido 品牌的核心，该品牌在印尼被称为 Dancow，作为“父母为儿童成长和发展的伙伴”。“受到妈妈对宝宝营养的爱的启发，Dancow 是最健康的选择”，雀巢 [声称](https://www.facebook.com/DancowIndonesia/)，却没有透露其产品含有添加糖。

去年，雀巢在印尼推出了一项 [活动](https://www.dancow.co.id/dpc/akudankausukadancow)，旨在“支持1岁及以上儿童的潜力”。该活动成功吸引了200万名母亲在社交媒体上分享与孩子的“激动人心的时刻”，有效地将她们转变为未付费的广告商和品牌大使。“感谢 @dancow 陪伴我的孩子成长和发展”，一位母亲写道。

在巴西，雀巢采用同样精益求精的策略推广其 Cerelac（Mucilon）婴儿米粉品牌。其广告活动基于“由Mucilon强化营养，母亲首选”的概念，由负责的广告公司主管 Dani Ribeiro [思考](https://grandesnomesdapropaganda.com.br/anunciantes/nova-campanha-mostra-o-amor-dos-pais-e-a-nutricao-fortificada-por-tras-de-cada-colherada-de-mucilon/) 提出。广告依赖于父母对他们宝宝的爱，强调“有助于宝宝免疫力和大脑发育的营养益处”，她解释道。“家长们因为能为子女做出正确选择而感到满足”，Ribeiro 补充道。

在巴西，雀巢将 Cerelac（Mucilon）婴儿米粉推广为富含营养，有助于儿童免疫力和大脑发育的产品。

在南非，雀巢将Cerelac宣传为含有12种必需的维生素和矿物质的来源，主题是“小小身体需要大力支持”。多国公司大声宣称：“150多年来，代代父母信赖雀巢Cerelac，它为宝宝提供了所需的一切”，[吹嘘](https://www.facebook.com/CerelacSA/)。然而，在这个面临真正肥胖[流行病](https://extranet.who.int/ncdccs/Data/ZAF_B10_National%20Strategy%20for%20prevention%20and%20Control%20of%20Obesity%204%20August.pdf)的国家，所有销售的Cerelac产品都含有高含量的添加糖。

克里斯·范·图莱肯（Chris Van Tulleken）是伦敦大学教授，畅销书《超加工人群》的作者，该书探讨了超加工食品的普遍性和影响，对雀巢使用的营销策略非常担忧。他指出：“这些并不是健康的产品。它们并非必需品。它们比真正的食物差劲。”“这类产品是全球营养转型的一部分，转向超加工饮食，与体重增加和肥胖有关，但也伴随许多其他不良健康结果。”
