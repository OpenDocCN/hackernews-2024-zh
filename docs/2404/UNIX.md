# UNIX

<main>

*(这不是我写的，但我稍作了英文上的润色。)*

UNIX大师听到他学徒在电脑房间里发出苦恼的呼声，便前去查看。

他发现学徒显然在极度痛苦中，几乎要哭出来了。“出了什么问题？”他问道。“为什么喊出来了？”

“使用这个系统太糟糕了。我每天必须使用四个编辑器来完成我的学习，因为没有一个编辑器可以做到所有的事情。”

大师庄重地点点头，问道：“那么，你打算提出什么解决这个明显困境的建议呢？”

学生仔细思考了几分钟，脸上露出了喜悦的表情。他兴奋地说道：“嗯，很明显。我要写出史上最好的编辑器。它将做现有的四个编辑器所有的事情，但做得更好、更快。由于我的新编辑器，世界将变得更美好。”

大师迅速举起手，扇了学徒一巴掌。大师年事已高，身体虚弱，学徒并未受到身体上的伤害，但对所发生的事情感到震惊。他转过头面对大师。“我做错了什么？”他问道。

“愚蠢！”大师说道，“难道你以为我想要再学习另一个编辑器吗？”

立刻，学徒恍然大悟。
