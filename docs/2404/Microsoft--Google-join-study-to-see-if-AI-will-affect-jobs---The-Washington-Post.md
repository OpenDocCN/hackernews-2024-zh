# 微软、谷歌加入研究，看人工智能是否会影响工作 - 华盛顿邮报

> 来源：[https://www.washingtonpost.com/technology/2024/04/04/jobs-ai-replace-study-microsoft-google-cisco/](https://www.washingtonpost.com/technology/2024/04/04/jobs-ai-replace-study-microsoft-google-cisco/)

随着人工智能继续渗透科技行业，关于它如何影响工人和经济的讨论正在加剧。包括谷歌和微软在内的大科技公司正在组建一个研究小组，以探讨AI可能如何影响技术工作。

由电信设备制造商思科（Cisco）领导的团队，还包括IBM、芯片制造商英特尔（Intel）、企业软件公司SAP和技术顾问公司埃森哲（Accenture）。工会包括AFL-CIO和通信工人联合会（CWA）被列为该团队的“顾问”。
