# 加工食品之争 | OpenMind 杂志

> 来源：[https://www.openmindmag.org/articles/the-processed-food-fight](https://www.openmindmag.org/articles/the-processed-food-fight)

经过数十年的探索，许多科学家认为他们终于找到了现代饮食中的主要问题——导致肥胖、糖尿病、心脏病以及许多其他严重慢性病的因素不断升级。罪魁祸首不是饱和脂肪、反式脂肪或者你从未听说过的某种新杀手脂肪。也不是胆固醇、碳水化合物或者糖分；乳制品、麸质或者肉类。它根本不是某种具体的*东西*。

根据这一越来越流行的观点，问题在于我们消费大量的超加工食品或UPF。传统上，人们通过烹饪和腌制等有限的加工技术来保存食物或使其更加美味。现代食品公司通过挤压、成型等技术，大幅度地改变食物，添加实验室衍生的成分，包括调味料、乳化剂和防腐剂。他们利用这种额外的加工使食品更便宜、更耐久且更方便。

“有一个长篇、正式的科学定义，但可以概括为：如果食物包装在塑料中，并且至少有一种成分是你的厨房里找不到的，那它就是UPF，”作者、传染病医生克里斯·范图莱肯在他最近广受赞誉的书籍《超加工人群：不是食物的食物背后的科学》中写道。这些超加工食品已经成为人们饮食的主要组成部分——在美国、英国和加拿大等国家，占据了绝大多数。“我们开始食用由新型分子构成和使用以前从未遇到过的加工过程制成的物质，这些物质甚至不能真正被称为‘食物’。”

许多研究人员和营养学家称这种转变成本高昂。他们指出一系列研究表明超加工食品是[肥胖](https://pubmed.ncbi.nlm.nih.gov/36280730/)、[心脏病](https://bmcmedicine.biomedcentral.com/articles/10.1186/s12916-023-03111-2)甚至[像阿尔茨海默病这样的神经退行性疾病](https://www.frontiersin.org/articles/10.3389/fnut.2023.1288749/full)的主要驱动因素。随着科学证据的积累，一些国家已通过采取公共卫生政策来尝试减少人们食用超加工食品的数量。这一信息已经进入了大众媒体，并越来越多地以令人担忧的报道形式出现。“超加工食品不仅对健康有害，还会影响你的心理健康，”国家地理在11月报导称。“超加工食品为何对健康如此有害？”经济学人在8月探讨如是说。

尽管有良好的研究提出了对超加工食品的担忧，我们应该对将其宣称为现代饮食的主要祸害持谨慎态度。与超加工食品有关的案例并不像标题和公众呼声所暗示的那样坚实。与健康问题相关的这些加工食品的研究存在局限性，同时也存在相互矛盾的证据。这是研究不能自我说明的典型案例：它必须始终由人在其他时常矛盾的证据背景中解释。“是否真的有研究表明超加工食品不健康？这在提供案例时是推动非常薄弱的数据，”雷丁大学的营养学研究员冈特·库恩勒如是说。

营养学领域长期以来对当代研究趋势反应过度，我们现在冒着在处理问题上犯同样错误的风险。“我们过去也有这个问题，就像后来被证明是适得其反的低脂推荐，”英格兰阿斯顿大学的营养师杜安·梅洛说。“我们犯了太多次错误。我们需要更加谨慎地进行更智慧的改变。”关于超加工食品的证据，正如科学中常见的情况，拒绝给出简单明了的黑白答案。

现代独特食物导致现代健康问题的想法开始在大众媒体和大众想象中得到越来越多的认同。

“超加工食品”这一术语是15年前引入的，当时[圣保罗大学的医生、流行病学家和营养学家Carlos Monteiro](https://bv.fapesp.br/pt/pesquisador/498/carlos-augusto-monteiro/)在《公共健康营养》期刊上发表了[一篇简短评论](https://www.cambridge.org/core/journals/public-health-nutrition/article/nutrition-and-health-the-issue-is-not-food-nor-nutrients-so-much-as-processing/0C514FC9DB264538F83D5D34A81BB10A)，主张工业食品“几乎与生存不相容”，“那些包含大量超加工食品的饮食在本质上营养不平衡且对健康有害”。

Monteiro最后建议采取诸如提高酒精和烟草价格、减少可获得性等政策。他和几位同事[很快创建了一个食品分类系统](https://www.scielo.br/j/csp/a/fQWy8tBbJkMFhGq6gPzsGkb/?lang=en#ModalTutors)，名为NOVA，将未加工或轻加工食品归为第一组，将超加工食品归为第三组（后来该系统扩展为四组）。超加工食品包括各种甜食、咸食和快餐，如汽水、鸡块、冰淇淋、薯片和饼干。它还包括通常不被视为垃圾食品但为方便或使用非家庭烹饪原料制成的产品，如果味酸奶、切片面包、婴儿配方奶粉和低糖添加的早餐谷物。

其他研究人员开始使用NOVA分类系统来检查超加工食品的潜在健康影响，比较那些食用大量超加工食品的人与那些食用更自然食物的人。一系列研究发现，超加工食品一直与更糟糕的健康结果相关联。

[2011年在危地马拉进行的一项研究](https://onlinelibrary.wiley.com/doi/abs/10.1002/hec.1579)发现，食用高度加工食品的人群中体重指数和肥胖率较高——加工食品消费量每增加10个百分点，体重指数增加4.3%。[2019年](https://www.sciencedirect.com/science/article/abs/pii/S0091743519301720)美国小组的研究发现，食用超加工食品与代谢紊乱（如高血糖和高血压）有关，这些状况增加了肥胖、糖尿病和心脏病的风险。

随着更多研究的出现，独特的现代食物导致独特的现代健康问题的观念开始在大众媒体、大众想象以及许多医生中获得认同。“超加工食品助长了这种观念，即过去的一切都更好”，Kuhnle指出。“一方面，你有农民在田间劳作。另一方面，你有冒烟的工厂。而且你可以补充说，人们普遍对科学、工业和政府持有某种程度的不信任。”

但是，这种回顾性研究有一个关键而广为人知的局限性：相关性不等于因果关系。在这些研究中，食用大量超加工食品的人通常健康状况较差——虽然不像吸烟那样糟糕，但稍微差一些。这可能被解释为超加工食品导致健康问题，但也可能是其他因素或*混杂因素*的结果。食用大量加工食品的人可能总体生活方式不健康，这可能解释为什么他们的结果较差。那些听从公共健康建议食用新鲜食品的人，可能也选择步行而不是使用电梯，这种增加的身体活动在研究调查中可能不会清楚地显示出来。研究人员试图纠正这些混杂因素，但不同类别的人类比较复杂，无法完全统计测量和比较。

西班牙大学生的一项研究发现，那些食用最多超加工食品的人，在10年内发展抑郁的可能性比那些食用最少的人高出三分之一。

观察性研究也容易出现称为*逆因果性*的问题：是饮用健怡饮料导致人们体重增加，还是想减肥的超重人群开始饮用健怡饮料？此外，人们记住自己吃了什么食物以及在观察性研究中准确报告的能力也存在重大问题。一些研究人员表示，这类证据通常不值得像加粗显示的关注。“这些东西如何发表在*新英格兰医学杂志*上，”心理学家和[食品研究专家彼得·罗杰斯](https://www.bristol.ac.uk/people/person/Peter-Rogers-be40b6c8-36cf-4fa2-bbb9-90fbf3f447b0/)说，“对我来说很奇怪。我们为什么不更关心证据的水平。”

一些研究人员试图通过进行前瞻性研究来更好地了解超加工食品。这意味着他们收集一组受试者，监测他们的饮食习惯和生活方式一段时间，然后将他们的饮食与健康结果进行比较，试图纠正除了饮食以外的其他因素。[2019年法国研究](https://pubmed.ncbi.nlm.nih.gov/30742202/)发现，那些食用更多超加工食品的人有更高的死亡风险。每增加10%的超加工食品摄入量，死亡人数增加14%，其中大约一半来自癌症和心血管疾病。[同年的一项研究](https://link.springer.com/article/10.1007/s00394-019-01970-1)，研究了西班牙大学生，发现那些食用最多超加工食品的人，比食用最少的人在10年内发展抑郁的可能性高出三分之一。

前瞻性研究可以展示人们随时间变化的情况，并有助于减少逆向因果关系和食物摄入量测量不准确的问题，但并不能消除核心的混淆因素：遵循吃新鲜食物建议的人们很可能在许多复杂的方式上与那些光顾快餐店的人不同。绝大多数针对超加工食品的证据属于间接证据。然而，一些国家的政策制定者认为证据足够坚实，[开始实施措施](https://www.nyu.edu/about/news-publications/news/2023/july/ultra-processed-foods-policy.html)以引导公民远离超加工食品。最近在《英国医学杂志》上，一群研究人员撰文称，“超加工食品与心脑血管健康：公共健康政策需要减少消费，不能再等待。”

有一项有影响力的研究已超越观察性研究，提供了更具体的证据，说明超加工食品如何影响人们。2019年，美国国立卫生研究院（NIH）的研究员[凯文·霍尔](https://www.niddk.nih.gov/about-niddk/staff-directory/biography/hall-kevin)及其同事们[发表了一篇论文](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC7946062/)，基于一项实验：他们在医院设置中让20名受试者先吃为期两周的高超加工食品饮食，然后改为两周的最低加工食品饮食，每顿饭都可以随意进食。另一半受试者则先吃最低加工食品饮食。

研究人员精心安排了这两种饮食在营养素、能量密度、纤维和其他属性上的等价性；它们只在加工百分比上有所不同。在受试者吃超加工食品饮食期间，他们每天摄入的卡路里多了约500卡，仅仅两周时间他们就增重了2磅。在吃新鲜食品时，他们减轻了2磅。最终，直接证据表明超加工食品可能会导致肥胖。

然而，一篇论文并不能解决一个复杂的领域。“这是一项非常良好进行的研究，”库恩勒说，“但像所有研究一样，它也有局限性。”例如，研究中的超加工食品纤维含量较低，因此研究人员还给人们提供了溶于水的纤维补充剂，但这可能无法等同于食物中的纤维摄入。此外，没有任何单一的小型实验可以解决如此重要的营养问题。“我曾对20或50人进行过类黄酮（水果中发现的化合物，但也用于加工）的研究，”库恩勒说，“反应总是‘这是一项单一的研究’。我们需要更多的研究来理解这一问题。”

Hall乐意承认，这项单一而小规模的研究并不能对超加工食品做出最终结论，部分原因是它并没有代表现实生活。“这是一个非常人为的环境，我们可以完全控制他们的食物，”他说。他还指出，我们不能根据两周的数据来推断终身效果。“当然，（过度进食的水平）不会持续太久，”他说。

增加摄入更多超加工食品，如谷物、黑色或全麦面包和酸奶，与患II型糖尿病的几率稍有降低有关。

像许多其他研究人员一样，Hall说，紧迫的问题是找出加工食品到底有什么问题——即找出它们似乎影响人类健康的机制。他重新分析了2019年的研究数据，并发现支持其他研究人员提出的具体假设。也许超加工食品的问题在于其能量密度；或者是其质地，可能促使人们吃得太快；或者是其所谓的[“超级可口性”，](https://pubmed.ncbi.nlm.nih.gov/31689013/) 定义了特定的糖、盐和脂肪的组合。Hall和他的同事现在正在进行第二项类似的饮食试验，试图复制原始发现，并查看这些假设中是否有任何得到实验验证的。这类研究成本高昂，需要大约两年的时间来运行，还需进行准备和分析。

另一个挑战是，并非所有超加工食品都一样。在许多情况下，它们比整食并不更能量密集。例如，水果酸奶通常比自制饼干的能量密度低，主要是因为酸奶含有更多的水分。这是超加工食品概念的问题之一。冷冻和罐装蔬菜通常被归类为超加工食品，但它们通常比饼干和其他自制的碳水化合物丰富的食品更健康。

对许多研究人员来说，超加工食品的难题并不那么难以理解。他们说，加工食品的问题可以很好地通过多年来主流营养学家一直在强调的指导方针来解释。“许多超加工食品含有高盐、高脂肪和高糖分，” Kuhnle说。“对于大多数超加工食品，我们实际上不需要标签。它本来就是我们所谓的垃圾食品。”

最近的一些研究表明，尽管有垃圾食品和不健康的超加工食品，还有其他类别可能不那么糟糕，甚至可能带来好处。一项关于大约20万人的[前瞻性研究](https://diabetesjournals.org/care/article/46/7/1335/148548/Ultra-Processed-Food-Consumption-and-Risk-of-Type)（2023年2月发表）发现，多吃诸如谷类、黑色或全麦面包、酸奶和乳制品甜点等超加工食品，与发展2型糖尿病的几率略微降低有关。另一项关于大约27万人的[前瞻性研究](https://www.thelancet.com/journals/lanepe/article/PIIS2666-7762(23)00190-4/fulltext)（2023年11月发表）发现，多吃超加工面包、谷类和植物基肉类替代品的人与少吃这些食物的人相比，多病（患有两种或更多严重健康问题）的发病率相同或稍低。

这些研究像其他流行病学研究一样容易受到混杂因素的影响。有可能是早餐吃谷类食品的人在其他难以测量的方式上过着更健康的生活。尽管如此，它们引发了一个问题：超加工食品是否构成一个应该无论如何都要同等对待的单一类别。在实际的食品决策中，许多营养专家会对不同类型的超加工食品进行划分。“汽水在我家里是被禁止的，”多病研究的高级作者[海因茨·弗莱斯林](https://www.thelancet.com/journals/lanepe/article/PIIS2666-7762(23)00190-4/fulltext)说。“但是我女儿喜欢这种麦片东西。她吃它。我认为它可能成为健康饮食的一部分。”

凯文·霍尔在办公室度过一天时，通常会带来微波冷冻午餐，尽管这些午餐通常含有添加剂，以保持冷冻和重新加热时的风味和口感。“它含有大量蛋白质，大量纤维，大量豆类，”他说。“这是一个超加工的餐，但我认为它很健康。”

这些正当的区分有时会在将健康问题归咎于超加工食品魔怪的潮流中失落。“我觉得这是一种时尚。我在标题上看到：超加工食品与癌症或心脏病有关，”在帕多瓦大学研究食品化学的[弗朗切斯科·维西奥利](https://www.unipd.it/en/contatti/rubrica?detail=Y&ruolo=1&checkout=cerca&persona=unipd&key=BE0C498252BAF82CD68F69B047905FE3)说。“我不赞成追随潮流，我不赞成顺流而下。五年后，人们会说，‘哇，慢下来。’进步必须慢慢来。”

我们已经有很长一段历史在提前实施营养政策，然后在获得更好的证据后才仓促修改，造成许多人心中的困惑、冷漠和愤世嫉俗。

对于超加工食品所宣称的影响的关注导致了一个出于善意但可能误导的急切行动。通过谴责一种新的膳食恶棍来改善我们的健康应引起警惕：我们已经有很长一段时间在营养政策上过早采取行动，然后在我们获得更好的证据后匆忙修正，这些行动在许多人心中播下了混乱、冷漠和怀疑的种子。

关于超加工食品的研究突显了一些重要内容，但“超加工食品”这一类别是否能更好地帮助我们理解问题或改善问题还不清楚。一些高度加工的食品，如含糖饮料和加工肉类，可能会影响健康，但这些情况我们多年来已经通过其他营养研究了解到。大多数人已经接受到了一个信息：他们应该食用更健康的新鲜食物，而不是那些从密封袋中拿出来，标签上写满了难以念的成分的不自然食品。但这似乎收效甚微。

在许多国家，大部分人们食用的东西都属于NOVA的“UPF”类别。与大多数食品作战是愚蠢的；这也是昂贵、无效和对于没有能力购买和准备新鲜食物的人们不公平的。Kevin Hall 是那位对超加工食品提出最广为传播的证据的研究人员，他说：“我认为我可能无法过分强调这一点。你可能认为超加工食品是问题的根源，但超加工食品占到了食品供应的60%。这不像香烟和癌症，没有人需要吸烟，你可以简单地禁止香烟。人们需要吃食物。你不能简单地禁止食品供应的60%，你也不能简单地对食品供应的60%征税。”

英国阿斯顿大学的营养师杜安·梅勒尔补充道，如果政策反对超加工食品使人们远离那些帮助人们准备健康、大部分天然餐点的加工“入门产品”，例如耐储藏的沙拉酱和汤料块，那么这些政策可能会适得其反。

我们已经有一些很好的证据表明哪些食物特别不好，进一步的研究将会让我们了解更多。我们现在的主要目标是找出如何让人们更容易选择更健康的选择，避免真正的垃圾食品，这是一个无论我们是否称之为“超加工食品”，都面临的挑战。更好的加工食品可以并且应该在替换危险食品方面发挥关键作用。

“我认为市场上可能已经有一大批健康的超加工食品，” Hall 说。“我们应该多生产这些食品，并增加它们的供应。”
