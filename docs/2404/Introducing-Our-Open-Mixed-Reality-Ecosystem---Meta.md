# 推出我们的开放混合现实生态系统 | Meta

> 来源：[https://about.fb.com/news/2024/04/introducing-our-open-mixed-reality-ecosystem/](https://about.fb.com/news/2024/04/introducing-our-open-mixed-reality-ecosystem/)

今天，我们迈出了向更开放计算平台迈进的重要一步，用于元宇宙。我们向第三方硬件制造商开放了驱动我们 Meta Quest 设备的操作系统，为开发者提供了更大的生态系统来进行构建，最终为消费者创造更多选择。这个平台是十年投资于支持混合现实的基础技术的产物，开放它意味着更多人将受益于这一投资。我们正与全球领先的技术公司合作，创建一个新的混合现实设备生态系统，并且让开发者更容易构建混合现实应用。

我们相信更开放的生态系统是将混合现实的力量带给尽可能多人的最佳途径。有了更多设备，这个新生态系统将为全球的消费者和企业提供更多选择。开发者将有更广泛的硬件选择来运行他们的应用程序，更多设备制造商将扩展他们的市场到更广泛的用户，就像我们在个人电脑和智能手机中看到的那样。

## 推出 Meta Horizon OS

这个新的硬件生态系统将运行在**Meta Horizon OS**上，这是驱动我们 Meta Quest 头戴显示器的混合现实操作系统。Meta Horizon OS 结合了支持当今混合现实体验的核心技术，并在平台的核心设置了社交存在感的一套功能。

Meta Horizon OS 是 Meta 十年工作的成果，致力于构建下一代计算平台。为了开发独立头戴式显示器，我们开发了像内外跟踪和自主跟踪控制器等技术。为了实现更自然的交互系统和社交存在感，我们开创了手部、眼部、面部和身体跟踪技术。对于混合现实，我们构建了一整套技术，将数字世界与物理世界融合，包括高分辨率的透视传输、场景理解和空间锚点。这项从 Android 开源项目的移动优先基础开始的长期投资，已经生产出了一个由数百万人使用的完整混合现实操作系统。

开发者和创作者可以利用所有这些技术来创建混合现实体验。他们可以通过 Meta Horizon OS 内建的内容发现和货币化平台，包括我们将重命名为 Meta Horizon Store 的 Meta Quest Store，来触达他们的受众并扩展他们的业务。

Meta Horizon OS 的社交层意味着人们的身份、头像、社交图谱和朋友群组可以随着他们在虚拟空间中的移动而移动，开发者可以将这些社交功能整合到他们的应用中。而且因为这个社交层是为了连接多个平台而设计的，人们可以在跨越混合现实、移动和桌面设备的虚拟空间中共享时间。Meta Horizon OS 设备也将使用与现在Meta Quest所有者使用的相同移动伴侣应用，我们将其重新命名为Meta Horizon应用。

## 新一代硬件

领先的全球科技公司已经开始合作，通过在 Meta Horizon OS 上构建新设备，将这一新生态系统带入现实生活：

+   [**华硕共和国游戏者**](https://rog.asus.com/us/) 将利用其作为游戏解决方案领导者的专业知识，开发全新的高性能游戏耳机。

+   [**联想**](https://www.lenovo.com/) 将借助其与Oculus Rift S共同设计的经验，以及在工程领域领先设备如ThinkPad笔记本系列的深厚专业知识，开发面向生产力、学习和娱乐的混合现实设备。

+   [**Xbox**](https://www.xbox.com/) 和 Meta 去年联手推出了 [Xbox Cloud Gaming (Beta)](https://www.meta.com/blog/quest/xbox-cloud-gaming-beta-launch/) ，让人们可以在混合现实中的大型2D虚拟屏幕上玩Xbox游戏。现在，我们再次合作，共同打造受Xbox启发的限量版Meta Quest。

所有这些设备将从 Meta 与高通技术公司的长期合作中受益，后者开发了与我们的软件和硬件堆栈紧密集成的 Snapdragon® 处理器。

## 更开放的应用生态系统

随着我们开始向更多设备制造商开放Meta Horizon OS，我们也在扩展应用开发者触达其受众的方式。我们开始消除Meta Horizon商店和应用实验室之间的障碍，这使得任何符合基本技术和内容要求的开发者都能在平台上发布软件。应用实验室的标题很快将在我们所有设备的商店中设立专门的部分，使它们更易于被更多人发现。

我们还在开发一种新的空间应用框架，帮助移动开发者创建混合现实体验。开发者可以使用他们已经熟悉的工具，将他们的移动应用带到Meta Horizon OS，或者创建全新的混合现实应用。

消费者和开发者都将从一个多个硬件制造商在共同平台上构建的生态系统中受益。我们期待继续这一旅程，让更多人体验混合现实。
