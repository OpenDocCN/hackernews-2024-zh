# 巴西的 Twitter 文件：独家洞察审查制度和政治

> 来源：[https://www.twitterfilesbrazil.com/#Start](https://www.twitterfilesbrazil.com/#Start)

En: 转发：选举法院命令 - 巴西最高法院

我们在此调查的范围内收到了一份新的法院命令。简单回顾：这是一个旨在确定背后可能存在的攻击机构和选举系统的协调努力的调查。总统博尔索纳罗本人也在这个过程中接受调查，此案件引起了公众的极大关注，以及对民主制度完整性的合法关注。我们收到了来自此调查的多个不寻常的请求，最近的一个要求我们根据标签提及提供未确定数量的用户数据。这些标签涉及围绕选举的动员 - 大致翻译为 #PrintedVoteNO；#DemocraticAuditableVote 和 #BarrosoInJail - 巴罗索是前TSE主席。在会议后，我们能够缩小请求的范围，并且联邦警察明确表明了他们想要的内容。

现在，巴西最高选举法院命令 Twitter 在5天内披露有关发布带有 #BarrosoNaCadeia 标签的推文用户的数据，时间段为2021年8月3日00:00至15:59（巴西时间）；带有 #VotoDemocraticoAuditavel 标签的推文用户的数据，时间段为2021年8月3日14:00至19:59；以及带有 #VotoDemocraticoAuditavel 标签的推文用户的数据，时间段为2021年8月10日22:00至22:59。请求的数据包括：用户在发布带有这些标签的推文之前最后一次访问的IP地址记录（目前不可用），以及具体带有适用标签的推文。

在过去，站点完整性向我们提供了标签报告、初始数据和内容提取，并提供了基于此提取的结果表，这些内容与警察部门的会话为依据。

我们已经收到了发布于这些日期带有 #VotoDemocraticoAuditavel 标签的推文，并将要求 LP 根据情况生成数据，但是考虑到2021年8月3日带有 #BarrosoNaCadeia 标签的推文数量（如结果文档I-B表中提供的40,800条），站点完整性当时并未提供这些推文的副本。

然而，考虑到这一新的法院命令，我们需要这些内容以便遵守披露命令，并检查哪些用户是数据披露决定的对象。根据我们目前拥有的报告，兴趣期间内有182条推文：

我们急需内容、用户处理和相应的BSI数据，请尽快提供，但请告知团队能完成此提取需要多少时间，以及是否有关于此请求的任何担忧或困难。我们正式的回复截止期限是5天（下周一），但考虑到我们与他们有良好的关系，我们可以请求延期。

我们想要强调，正如你所知，这个案件是高优先级且敏感的，虽然我们理解这可能对团队造成影响，但我们也知道遵守规定或者至少有理由挑战决定的重要性。这是有关整个调查的背景文件 - 已更新至此新进展。
